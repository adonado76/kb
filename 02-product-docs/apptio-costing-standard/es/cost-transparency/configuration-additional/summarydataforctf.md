# Resumen de las fuentes de datos necesarias para la Costing Standard Foundation

Para configurar Costing Standard Foundation, necesitará los siguientes conjuntos de datos:

- Libro Mayor ( OpEx y CapEx )
- Presupuesto ( OpEx y CapEx )
- Lista de centros de coste
- Plan contable
- Registro de activos fijos
- Efectivos reales por centro de coste
- Plantilla prevista por centro de coste
- Lista de proveedores
- Recuentos de volumen de infraestructura por torre y subtorre de TI

Además, tendrás que crear algunas tablas de asignación, entre las que se incluyen:

- Asignación de códigos de cuenta a grupos de costes
- Asignación de activos fijos a torres de recursos informáticos
- Asignación de torre de recursos de mano de obra a TI
- Asignación de proveedores a torres de recursos informáticos
- Asignación de centros de costes a torres de recursos informáticos
