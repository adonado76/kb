# Cargue sus datos

El proceso general de cómo Apptio utiliza sus datos incluye:

1. Cargue sus datos en Apptio.
2. Añada y asigne sus datos al conjunto de datos maestros correspondiente.

![](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/1424_1.png)

Los conjuntos de datos maestros se utilizan en toda la aplicación Costing Standard para:

- Volver a los objetos del modelo
- Apoyar las estrategias de asignación
- Calcular métricas
- Visualizar datos en informes
