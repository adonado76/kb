# Amazon Web Services configuración

Para configurar la aplicación para Amazon Web Services, debe instalar los componentes CTF-Cloud Service y CTF- Amazon Web Services.

## Acerca de esta tarea

Para configurar la aplicación para Amazon Web Services, debe instalar los componentes CTF-Cloud Service y CTF- Amazon Web Services.

Al instalar el componente CTF- Amazon Web Services, se añaden dos conjuntos de datos al grupo de proveedores de servicios en nube:

- AWS Datos maestros de facturación de imputación de costes: se utilizan para asignar los datos de facturación que se cargan.
- AWS Datos maestros de compras de RI: se utilizan para asignar datos de instancias reservadas.

El componente no añade ningún objeto al modelo de costes y no añade ningún informe nuevo.

Hay ocho pasos para configurar la aplicación para Amazon Web Services.

## Paso 1: Instalar el componente CTF - Cloud

Instale el componente Nube del mismo modo que los demás componentes de CTF:

### Procedimiento

1. Abra el proyecto Costing Standard .
2. Haga clic en la pestaña **Proyectos**.
3. Haga clic en **Componentes**.
4. Haga clic en el componente **CTF - Cloud Service**.
5. Haga clic en **Instalar**.
