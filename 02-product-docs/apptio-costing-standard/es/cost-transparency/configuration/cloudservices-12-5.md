# Configuración de Nube para el cálculo de costes Estándar 12.1 a través de 12.5x

Utilice el componente CTF - Cloud Service Provider para importar datos de uso de servicios web. El componente es opcional.

## Introducción

Si utiliza Amazon Web Services ( AWS ), Microsoft Azure, SoftLayer, o CenturyLink, puede importar los datos de uso de los informes mensuales de uso y facturación. Los datos se utilizan en los informes CTF y en los informes de aplicaciones y servicios. Los datos pueden ayudarle a comparar el coste de los servicios en nube con el de los servicios internos. Para utilizar los datos de webservices, debe instalar el componente CTF - Cloud Service Provider.

## Dos procedimientos de configuración diferentes

Existen dos procedimientos de configuración diferentes:

- [Amazon Web Services configuración](aws.html "Para configurar la aplicación para Amazon Web Services, debe instalar los componentes CTF-Cloud Service y CTF- Amazon Web Services.")
- [Microsoft Azure y la configuración de otros proveedores](azure.html "Para configurar la aplicación para Microsoft Azure u otros proveedores, debe instalar el componente CTF-Cloud Service.")

Cada procedimiento se documenta por separado.

## Vinculado a proveedores

Para que el coste de la nube se calcule correctamente, los datos del proveedor deben introducirse en la aplicación Costing Standard . Los datos del proveedor identifican los costes procedentes de los servicios web y los asignan a los servicios en nube. El modelo que figura a continuación muestra la relación.
