# AWS Caducidad de los formatos de facturación

AWS ha indicado que dejará obsoletas versiones específicas de informes de facturación, incluido el Informe detallado de facturación con recursos y etiquetas, así como el Informe de asignación de costes, ambos fuentes de datos de facturación para Cloud Cost Management. AWS no ha facilitado una fecha concreta en la que dejará de dar soporte a esos formatos de facturación, pero ha mencionado que, cuando anuncie una fecha, ofrecerá al menos un periodo de gracia de 6 meses.

Apptio está trabajando actualmente en la integración con el informe de costes y uso de AWS (también conocido como CUR), que será el formato de facturación futuro de AWS. Apptiowill completará esa integración mucho antes de que AWS deje de admitir los demás formatos de facturación.
