# Costes de aplicación

![Coste de aplicación](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/application-cost-report.png)

| Ventajas claves | Detalles |
| --- | --- |
| - Ver costes de aplicación, costes unitarios y costes por usuario - Desglose del gasto mensual de cada aplicación - Vea el ahorro de costes que ha conseguido retirando aplicaciones - Identifique las áreas que pueden racionalizar su cartera de aplicaciones   **Preguntas contestadas**   - ¿Cuál es el coste medio por usuario de cada aplicación? - ¿Estamos pagando por aplicaciones redundantes o duplicadas que sirven para lo mismo? - El contrato de esta aplicación expira pronto. ¿Deberíamos renovarlo? | **Para** :  Líderes de soluciones, propietarios de aplicaciones, propietarios de servicios  **Cómo navegar** : Vaya a **Informes** > **Aplicación (Colección)** > **Extracción de costes de aplicación** |

**Ideas:**

El Informe de Costes de Aplicación ha sido diseñado para responder a las siguientes preguntas:

**Racionalización de Aplicaciones** :  
Muestra el Gasto Anual de Ejecución de Aplicaciones junto con los Propietarios de Negocio. La siguiente vista puede utilizarse para identificar solapamientos dentro de una función de aplicación determinada, permitiendo la eliminación de aplicaciones redundantes como parte de la actividad de Racionalización de la Cartera de Aplicaciones.

![Racionalización de aplicaciones](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/application-rationalization.png)

**Aplicaciones retiradas** :  
Muestra las aplicaciones marcadas para su retirada o eliminación, con los gastos de ejecución restantes y las fechas objetivo. Esta información se utiliza para programar el desmantelamiento y obtener ahorros.

![Solicitud de jubilación](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/retired-application.png)

**Contratos que expiran** :  
Muestra los contratos por aplicaciones y, en función de su fecha de finalización, nos permite decidir si renovar el contrato o retirarlo para reducir el gasto.

![Contratos que expiran](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/expiring-contract.png)

**Optimización de aplicaciones** :  
Muestra las métricas financieras y de uso consolidadas de las 10 aplicaciones principales. Esto se utiliza para comprender el coste total de propiedad y orientar las acciones de optimización de mayor valor.

![Optimización de aplicaciones](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/application-optimization.png)

Profundizando en cada aplicación, podemos comprender los diversos impulsores del gasto de OpEx, así como los detalles de mano de obra, proveedores y contratos para esa aplicación. Esta información detallada puede consultarse en el Informe detallado de retirada de costes de aplicación.

![Taladro de aplicación](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/application-drill.png)
