# Gestión de TI - Detalles de las torres de TI - Informe de análisis fijo/variable ( v103 )

◆ Se aplica a: Costing Standard 11.8.x que se ejecuta en TBM Studio v12 o TBM Studio v11.

## Introducción

Utilice este informe para identificar el gasto fijo relativo /var iable para cada subtorre de TI.

![](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/itmittowersfixedvariable103_1.png)

## Navegación

Gestión TI > Torres TI > Nombre de la Torre TI > Análisis Fijo/Variable

## Funciones

Este informe está destinado a:

- Gestión de TI
- Propietario de torre de TI

## Objetivos

Utilice este informe para:

Identificar el gasto fijo relativo /var iable para cada subtorre de TI.

## Preguntas contestadas

La información presentada en este informe puede utilizarse para responder a las siguientes preguntas:

- Si baja la demanda o el consumo, ¿bajarán también mis costes o son fundamentalmente fijos y se mantienen intactos?
- Si necesito reducir costes, ¿qué subtorres de TI podrían tener un impacto a corto plazo?
- ¿A qué se deben mis costes fijos?
- ¿Hay cambios arquitectónicos, estructurales o de aprovisionamiento que pueda hacer para crear una estructura de costes más variable?
- ¿Cuál es el ratio de gasto fijo adecuado para nuestra empresa y este servicio?

## Próximas acciones

- Para ver los gastos fijos de 13 meses de /var iable e identificar las tendencias a lo largo del tiempo, haga clic en Ver en la columna Tendencia.
- Consulte los informes IT Finance Fijo/Variable para ver qué cuentas son fijas o variables.
