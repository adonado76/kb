# IT Finanzas - Desviación de cuenta - Informe de detalles de transacción ( v103 )

Se aplica a: Costing Standard 11.8.x que se ejecuta en TBM Studio v12 o TBM Studio v11.

## Introducción

Utilice este informe para identificar las transacciones específicas que se produjeron en el período actual para una cuenta específica.

![imagen](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/itfaccountvariancetransactiondetails103_1.png)

## Navegación

IT Finanzas > Desviación de cuenta > Cuenta

## Funciones

Este informe está destinado al personal de Finanzas de TI.

## Objetivos

Utilice este informe para identificar las transacciones específicas que se produjeron en el período actual para una cuenta específica.

## Preguntas contestadas

Puede utilizar la información presentada en este informe para responder a las siguientes preguntas:

- ¿Reconozco para qué sirven estas transacciones?
- ¿Le parece anormal el importe de una transacción concreta?
- ¿Se ha introducido alguna transacción incorrectamente? ¿Pertenecen a una cuenta diferente?
- ¿Hay alguna transacción que parezca que debería haberse realizado en un periodo anterior y se haya procesado con retraso?

## Próximas acciones

- Si una transacción se ha introducido incorrectamente, póngase en contacto con el titular de la cuenta para corregirla o documentarla con una explicación.
- Extraiga más detalles de la transacción del libro auxiliar de cuentas a pagar o del sistema financiero de registro.
