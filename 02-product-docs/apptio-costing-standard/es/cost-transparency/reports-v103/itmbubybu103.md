# Gestión de TI - Unidades de negocio - Informe por unidad de negocio ( v103 )

Utilice este informe para revisar los gastos por unidad de negocio.

Se aplica a: Costing Standard 11.8.x que se ejecuta en TBM Studio v12 o TBM Studio v11.

![](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/itmbubybu103_1.png)

## Navegación

Gestión de TI > Unidades de negocio > Por unidad de negocio

## Funciones

Este informe está destinado a:

- Propietarios de unidades de negocio
- directores de informática
- directores financieros

## Objetivos

Utilice este informe para:

- Identificar al Responsable de Relaciones Comerciales de una unidad de negocio.
- Vea el gasto medio por miembro del personal por unidad de negocio.
- Ver gastos totales, porcentaje fijo y porcentaje variable.

## Preguntas contestadas

La información presentada en este informe puede utilizarse para responder a las siguientes preguntas:

- ¿Estamos gastando demasiado poco o demasiado por miembro del personal para una unidad de negocio concreta?
- ¿Tenemos la combinación adecuada de gastos fijos y variables en todas las unidades de negocio?
- ¿Es necesario tomar medidas para mitigar el riesgo?

## Próximas acciones

Haga clic en el nombre de una unidad de negocio para acceder al informe Unidad de negocio - Detalles de consumo individual de esa unidad de negocio.
