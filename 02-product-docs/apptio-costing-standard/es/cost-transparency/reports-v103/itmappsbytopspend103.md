# Gestión de TI - Aplicaciones - Informe por Top Spend ( v103 )

Utilice este informe para ver los gastos más elevados hasta la fecha por aplicación.

Se aplica a: Costing Standard 11.8.x que se ejecuta en TBM Studio v12 o TBM Studio v11.

![](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/itmappsbytopspend103_1.png)

## Navegación

Gestión TI > Aplicaciones > Por Top Spend

## Funciones

Este informe está destinado a:

- Propietarios de aplicaciones
- Propietarios de la cartera de aplicaciones / Vicepresidente de Desarrollo y Soporte de Aplicaciones
- Arquitectos de empresa

## Objetivos

Utilice este informe para:

- Vea rápidamente los gastos más importantes hasta la fecha por aplicación en el gráfico Gastos más importantes hasta la fecha por aplicación.
- Revisar el gasto operativo y de desarrollo por aplicación.
- Revise los porcentajes de gasto fijo y variable por aplicación.

## Preguntas contestadas

La información presentada en este informe puede utilizarse para responder a las siguientes preguntas:

- ¿Cuánto hemos gastado por solicitud?
- ¿Tenemos un equilibrio razonable de gasto para mantener las operaciones actuales y desarrollar aplicaciones para el futuro?
- ¿Es necesario tomar medidas para mitigar el riesgo?

## Próximas acciones

Haga clic en el nombre de una aplicación para obtener un informe Cartera de aplicaciones - Detalle.

Utilice los informes detallados para revisar:

- Un resumen de la solicitud.
- Desglose del gasto operativo de la aplicación por Torre de TI.
- El gasto de una subtorre de TI a lo largo del tiempo.
- El gasto con proveedores para la aplicación.
