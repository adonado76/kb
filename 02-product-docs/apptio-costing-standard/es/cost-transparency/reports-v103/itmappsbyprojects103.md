# Gestión informática - Aplicaciones - Informe por proyectos ( v103 )

Utilice este informe para identificar los gastos de aplicaciones por proyecto y familia de aplicaciones.

Se aplica a: Costing Standard 11.8.x que se ejecuta en TBM Studio v12 o TBM Studio v11.

![](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/itmappsbyprojects103_1.png)

## Navegación

Gestión informática > Aplicaciones > Por proyectos

## Funciones

Este informe está destinado a:

- Propietarios de aplicaciones
- Propietarios de la cartera de aplicaciones / Vicepresidente de Desarrollo y Soporte de Aplicaciones
- Arquitectos de empresa

## Objetivos

Utilice este informe para identificar los gastos de aplicaciones por proyecto y familia de aplicaciones.

## Preguntas contestadas

La información presentada en este informe puede utilizarse para responder a las siguientes preguntas:

- ¿Cuánto hemos gastado en proyectos relacionados con aplicaciones?
- ¿Se ajusta el gasto a los objetivos de inversión de las solicitudes?

## Próximas acciones

Utilice el informe Finanzas TI - Proyectos para hacer un seguimiento de los gestores de proyectos.
