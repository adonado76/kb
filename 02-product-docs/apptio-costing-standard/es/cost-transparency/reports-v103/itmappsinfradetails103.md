# Gestión informática - Aplicaciones - Infraestructura Informe detallado ( v103 )

Utilice este informe para revisar el consumo de torres informáticas por parte de las aplicaciones.

Se aplica a: Costing Standard 11.8.x que se ejecuta en TBM Studio v12 o TBM Studio v11.

![](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/itmappsinfrastructuredetails103_1.png)

## Navegación

Gestión TI > Aplicaciones > Detalles de la infraestructura

## Funciones

Este informe está destinado a:

Revise el consumo de la torre informática por parte de las aplicaciones.

## Objetivos

Utilice este informe para revisar el consumo de la torre Compute IT por parte de las aplicaciones.

## Preguntas contestadas

La información presentada en este informe puede utilizarse para responder a las siguientes preguntas:

- Para una aplicación determinada, ¿es adecuado el consumo de recursos informáticos (servidores)?
- ¿Hay que hacer algo para mitigar el riesgo?

## Próximas acciones

- Haga clic en las pestañas Almacenamiento y Service Desk para obtener informes similares.
- Haga clic en el nombre de una aplicación para obtener un informe detallado de la misma.
