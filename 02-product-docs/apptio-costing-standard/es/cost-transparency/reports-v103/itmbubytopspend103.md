# Gestión de TI - Unidades de negocio - Informe por Top Spend ( v103 )

Utilice este informe para ver los gastos más importantes por unidad de negocio.

Se aplica a: Costing Standard 11.8.x que se ejecuta en TBM Studio v12 o TBM Studio v11.

![imagen](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/itmbubytopspend103_1.png)

## Navegación

Gestión TI > Unidades de negocio > Por gasto máximo

## Funciones

Este informe está destinado a:

- Propietarios de unidades de negocio
- directores de informática
- directores financieros

## Objetivos

Utilice este informe para:

- Consulte los gastos más elevados por unidad de negocio.
- Para cada unidad, vea su propietario, responsable de relaciones comerciales y otros datos.

## Preguntas contestadas

La información presentada en este informe puede utilizarse para responder a las siguientes preguntas:

- ¿Qué unidades de negocio consumen más costes de TI?
- ¿Es necesario tomar medidas para mitigar el riesgo?

## Próximas acciones

- Haga clic en el nombre de una unidad de negocio para acceder a un informe Unidades de negocio - Detalles de consumo individual para esa unidad de negocio.
- Póngase en contacto con el gestor de relaciones comerciales asignado si necesita seguimiento.
