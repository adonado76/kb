# AWS Eliminação dos formatos de cobrança

AWS indicou que estará descontinuando versões específicas de relatórios de faturamento, incluindo o Relatório de faturamento detalhado com recursos e tags, bem como o Relatório de alocação de custos, ambos fontes de dados de faturamento para o Cloud Cost Management. AWS não forneceu uma data específica até a qual encerrará o suporte a esses formatos de faturamento, mas mencionou que, quando anunciar uma data, fornecerá pelo menos um período de carência de seis meses.

Apptio está trabalhando atualmente em uma integração com o relatório AWS Cost and Usage (também conhecido como CUR), que será o formato de faturamento futuro para AWS. A Apptiowill concluirá essa integração bem antes de o AWS encerrar o suporte aos outros formatos de faturamento.
