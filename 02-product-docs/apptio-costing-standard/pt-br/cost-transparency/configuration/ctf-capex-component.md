# CTF - CapEx componente: instalar e configurar

Use o componente opcional CTF - CapEx para separar o relatório CapEx do relatório OpEx das versões anteriores do Costing Standard. Se sua organização não carregar dados de CapEx em seus modelos, esse componente impedirá a exibição de campos vazios de CapEx em relatórios prontos para uso. Isso permite que você se concentre nos dados do OpEx se usar apenas o relatório OpEx.

Aplica-se a: TBM Studio 12.8 e posterior, com o modelo v107 e posterior

## Introdução

O componente CapEx fornece insights sobre as despesas de capital de TI nos níveis de conta, pool de custos e centro de custos no orçamento e nos últimos relatórios de variação de previsão.

O componente CTF - CapEx não é instalado automaticamente com o projeto Costing Standard padrão.

## Casos de uso

Os dados desse componente apóiam as seguintes decisões:

- Gerenciar as despesas de capital de acordo com o orçamento e a previsão
- Identificar mudanças nas despesas de capital mês a mês, trimestre a trimestre e outros períodos de tempo
- Identificar as principais despesas de capital
- Identificar as principais variações de orçamento e previsão

## Instalar o componente

Para instalar o componente:

1. Abra o **Costing Standard** projeto.
2. Na guia **Projeto**, clique em **Componentes**.
3. Clique no componente **CTF - CapEx**.
4. Clique em **Install (Instalar** ).

## Dados principais

Para obter uma descrição dos campos na tabela de dados mestre, consulte as informações na página do componente CTF - CapEx no produto. Para exibir a página:

1. Na guia **Projeto**, clique em **Componentes**.
2. Clique no componente **CTF - CapEx**.

## Campos obrigatórios e recomendados

Os campos obrigatórios e recomendados necessários para os relatórios de mão de obra padrão do site Costing Standard são os mesmos listados na [tabela Populate the Cost Source Master Data (Preencher os dados mestre da fonte de custos](populatecostsource.html "A tabela Dados mestre de origem de custos define os dados necessários para os relatórios relacionados a custos e orçamento. Para preencher a tabela, você deve carregar os valores de custo e de orçamento. Para preencher a tabela de dados mestre da fonte de custos, carregue dois conjuntos de dados: Cost Actuals (Custo real) e Cost Budget (Custo orçamentário). Mapeie os conjuntos de dados para a tabela de dados mestre da fonte de custos.") ).
