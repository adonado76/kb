# Procedimentos gerais de configuração

Os três procedimentos gerais de configuração são: carregar dados, mapear dados, validar e gerenciar alocações de modelos.

Ao configurar o aplicativo Costing Standard , você executará três procedimentos repetidamente.

- [Carregar dados no aplicativo Costing Standard](loaddata.html "As tabelas de dados mestre fornecem uma maneira de mapear seus dados para os dados exigidos pelo aplicativo CT. As informações abaixo descrevem como carregar dados no aplicativo.")
- [Mapear dados para uma tabela de dados mestre](maptomaster.html "Depois de fazer o upload de uma tabela de dados de origem e transformá-la, se necessário, você pode mapeá-la para uma tabela de dados mestre. Use as instruções a seguir para mapear dados para uma tabela de dados mestre.")
- [Validar alocações de modelos](validatemodel.html "Use a visualização da tabela de modelos para validar o fluxo de valor em um modelo. Depois de fazer o upload e mapear os dados de origem para a tabela de dados mestre apropriada, suas tabelas modeladas associadas devem começar a mostrar os drivers de unidade e as alocações.")
