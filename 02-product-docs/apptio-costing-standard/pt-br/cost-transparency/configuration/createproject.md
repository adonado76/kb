# Criar o projeto de cálculo de custos padrão

A primeira etapa do processo de configuração do Costing Standard é criar o projeto Costing Standard . Quando você cria o projeto Costing Standard , o aplicativo Costing Standard cria as métricas padrão, os modelos de cálculo e os relatórios.

## Procedimento

1. Faça login na instância Apptio.
2. Abra o menu Configurações e clique em **Novo projeto**.
3. Na caixa de diálogo Novo projeto, digite o nome do projeto.
4. Selecione **Costing Standard** para o tipo de projeto.
5. Clique em **OK**.
