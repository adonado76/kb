# Amazon Web Services configuração

Para configurar o aplicativo para Amazon Web Services, você deve instalar os componentes CTF-Cloud Service e CTF- Amazon Web Services.

## Sobre esta tarefa

Para configurar o aplicativo para Amazon Web Services, você deve instalar os componentes CTF-Cloud Service e CTF- Amazon Web Services.

Quando você instala o componente CTF- Amazon Web Services, ele adiciona dois conjuntos de dados ao grupo Cloud Service Provider:

- AWS Dados mestre de faturamento de alocação de custos: usados para mapear os dados de faturamento que você carrega.
- AWS Dados mestre de compras do RI: usados para mapear dados de instâncias reservadas.

O componente não adiciona um objeto ao modelo de custo e não adiciona nenhum relatório novo.

Há oito etapas para configurar o aplicativo para Amazon Web Services.

## Etapa 1: Instale o componente CTF - Cloud

Instale o componente Cloud da mesma forma que você instala os outros componentes do CTF:

### Procedimento

1. Abra o projeto Costing Standard .
2. Clique na guia **Projetos**.
3. Clique em **Components (Componentes** ).
4. Clique no componente **CTF - Cloud Service**.
5. Clique em **Install (Instalar** ).
