# Gerenciamento de TI - Aplicativos - Por relatório de despesas principais ( v103 )

Use este relatório para ver as principais despesas acumuladas até o momento por aplicativo.

Aplica-se a: Costing Standard 11.8.x em execução em TBM Studio v12 ou TBM Studio v11.

![](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/itmappsbytopspend103_1.png)

## Navegação

Gerenciamento de TI > Aplicativos > Por despesas principais

## Funções

Este relatório foi elaborado para:

- Proprietários de aplicativos
- Proprietários do portfólio de aplicativos / VP de desenvolvimento e suporte de aplicativos
- Arquitetos corporativos

## Objetivos

Use este relatório para:

- Veja rapidamente as principais despesas YTD por aplicativo no gráfico Top Spend YTD by Application.
- Analise os gastos operacionais e de desenvolvimento por aplicativo.
- Analise as porcentagens de gastos fixos e variáveis por aplicativo.

## Perguntas respondidas

As informações apresentadas neste relatório podem ser usadas para responder às seguintes perguntas:

- Quanto gastamos por aplicativo?
- Temos um equilíbrio sensato de gastos para manter as operações atuais e desenvolver aplicativos para o futuro?
- São necessárias ações para mitigar o risco?

## Próximas ações

Clique no nome de um aplicativo para obter um relatório Portfólio de aplicativos - Detalhes.

Use os relatórios detalhados para revisar:

- Um resumo do aplicativo.
- Um detalhamento dos gastos operacionais do aplicativo por torre de TI.
- Os gastos com uma subestrutura de TI ao longo do tempo.
- Os gastos com fornecedores para o aplicativo.
