# Gerenciamento de TI - Aplicativos - Relatório de detalhes do aplicativo ( v103 )

Use este relatório para analisar os gastos com aplicativos por nuvem privada, física e virtual.

Aplica-se a: Costing Standard 11.8.x em execução em TBM Studio v12 ou TBM Studio v11.

![](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/itmappsappdetails103.png)

## Navegação

Gerenciamento de TI > Aplicativos > Detalhes do aplicativo

## Funções

Este relatório foi elaborado para:

- Proprietários de aplicativos
- Proprietários do portfólio de aplicativos / VP de desenvolvimento e suporte de aplicativos
- Arquitetos corporativos

## Objetivos

Use este relatório para:

- Analise os gastos com aplicativos e o número de usuários.
- Obtenha o custo unitário acumulado no ano por usuário para cada aplicativo.

## Perguntas respondidas

As informações apresentadas neste relatório podem ser usadas para responder às seguintes perguntas:

- O custo total por usuário está alinhado com as expectativas ou referências?
- São necessárias ações para mitigar o risco?

## Próximas ações

Use as segmentações no relatório para analisar os diferentes custos associados ao aplicativo.
