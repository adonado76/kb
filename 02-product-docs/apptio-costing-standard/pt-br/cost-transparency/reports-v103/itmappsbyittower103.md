# Gerenciamento de TI - Aplicativos - Por relatório da IT Tower ( v103 )

Use este relatório para analisar os gastos com aplicativos nas torres de TI e por família de aplicativos.

Aplica-se a: Costing Standard 11.8.x em execução em TBM Studio v12 ou TBM Studio v11.

![imagem](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/itmappsbyittower103_1.png)

## Navegação

Gerenciamento de TI > Aplicativos > Por torre de TI

## Funções

Este relatório foi elaborado para:

- Proprietários de aplicativos
- Proprietários do portfólio de aplicativos / VP de desenvolvimento e suporte de aplicativos
- Arquitetos corporativos

## Objetivos

Use esse relatório para analisar os gastos com aplicativos em todas as torres de TI e por família de aplicativos.

## Perguntas respondidas

As informações apresentadas neste relatório podem ser usadas para responder às seguintes perguntas:

- Para um determinado aplicativo, o consumo de data centers, servidores e custos de armazenamento está correto?
- São necessárias medidas para reduzir o risco orçamentário?
