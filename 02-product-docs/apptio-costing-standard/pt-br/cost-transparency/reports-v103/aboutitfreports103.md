# Sobre os relatórios de finanças de TI ( v103 )

Aplica-se a: Costing Standard 11.8.x em execução em TBM Studio v12 ou TBM Studio v11.

Os relatórios do IT Finance fornecem dados sobre:

- O orçamento para o ano corrente.
- O gasto até o momento.
- Variações orçamentárias por pool de custos.
- As principais despesas e a variação por pools de custos.

Para acessar os relatórios do IT Finance, clique em **IT Finance** na página inicial. Use as guias na página do relatório do IT Finance para acessar dados específicos:

- Desvio de conta
- Pool de custos
- Fornecedores
- Conta do provedor de nuvem
- Mão de Obra
- Ativos fixos
- Valores discrepantes
