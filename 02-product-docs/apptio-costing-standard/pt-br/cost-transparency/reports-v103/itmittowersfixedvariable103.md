# Gerenciamento de TI - Detalhes das torres de TI - Relatório de análise fixo/variável ( v103 )

◆ Aplica-se a: Costing Standard 11.8.x em execução em TBM Studio v12 ou TBM Studio v11.

## Introdução

Use este relatório para identificar o gasto fixo relativo de /var iable para cada subesfera de TI.

![](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/itmittowersfixedvariable103_1.png)

## Navegação

Gerenciamento de TI > Torres de TI > Nome da torre de TI > Análise fixa/variável

## Funções

Este relatório foi elaborado para:

- Administração da TI
- Proprietário da torre de TI

## Objetivos

Use este relatório para:

Identifique o gasto fixo relativo de /var iable para cada subtorre de TI.

## Perguntas respondidas

As informações apresentadas neste relatório podem ser usadas para responder às seguintes perguntas:

- Se a demanda ou o consumo cair, meus custos também cairão ou eles são basicamente fixos e permanecem intactos?
- Se eu precisar reduzir custos, quais subárvores de TI podem ter um impacto em curto prazo?
- O que está determinando meus custos fixos?
- Há mudanças arquitetônicas, estruturais ou de fornecimento que eu poderia fazer para criar uma estrutura de custos mais variável?
- Qual é o índice de despesas fixas adequado para nossa empresa e esse serviço?
