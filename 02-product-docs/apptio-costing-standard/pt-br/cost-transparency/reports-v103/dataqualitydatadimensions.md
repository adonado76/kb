# Qualidade dos dados - Dimensões dos dados

Aplica-se a: Costing Standard 11.8.x em execução em TBM Studio v12 ou TBM Studio v11.

## Introdução

Use esse relatório para avaliar a integridade, a validade e a exclusividade dos dados de fontes de custo, mão de obra, fornecedores, projetos, data centers, servidores, armazenamento, tíquetes, aplicativos e unidades de negócios. Clique em uma dimensão para ver um relatório detalhado da dimensão.

![relatórios de dimensão de dados](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/dataqualitydatadimensions_1.png)

## Navegação

Qualidade de dados > Dimensões de dados

## Funções

Este relatório foi elaborado para administradores de TBM.

## Objetivos

Use este relatório para:

- Avaliar a integridade dos dados. A integridade é medida como o número de células nos conjuntos de dados de suporte que têm valores. Por exemplo, uma pontuação de completude de 80% indica que 80% das células têm valores.
- Avaliar a validade dos dados. A validade é determinada pela comparação dos valores nos dados com uma lista de valores aceitáveis que você carregou no aplicativo.
- Avalie a exclusividade dos dados. A exclusividade indica se os valores na coluna-chave de uma tabela são exclusivos ou se há duplicatas. As duplicatas não são necessariamente ruins, dependendo da natureza dos dados na tabela.

## Perguntas respondidas

As informações apresentadas neste relatório podem ser usadas para responder às seguintes perguntas:

- Meus dados estão completos o suficiente para tomar decisões informadas?
- Os dados estão corretos?

## Próximas ações

- Clique em uma fonte de dados (por exemplo, g.: Cost Source, Labor, etc.) para ver os relatórios de integridade, validade e exclusividade.
- Para as áreas que não têm dados suficientes, procure maneiras de coletar dados mais completos.
- Se houver dados incorretos, verifique as fontes de dados e determine a causa dos dados incorretos.
