# Gerenciamento de TI - Unidades de negócios - Por aplicativos para relatório de aposentadoria ( v103 )

Use esse relatório para obter economias de custo que poderiam ser realizadas se um aplicativo fosse desativado.

Aplica-se a: Costing Standard 11.8.x em execução em TBM Studio v12 ou TBM Studio v11.

![](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/itmbubyappstoretire103_1.png)

## Navegação

Gerenciamento de TI > Unidades de negócios > Por aplicativos a serem aposentados

## Funções

Este relatório foi elaborado para:

- Proprietários de unidades de negócios
- CIOs
- Diretores financeiros

## Objetivos

Use este relatório para:

- Analise os custos mensais e anuais dos aplicativos.
- Compare os custos por investimento em aplicativos, unidade de negócios e família de aplicativos.

## Perguntas respondidas

As informações apresentadas neste relatório podem ser usadas para responder às seguintes perguntas:

- Existem aplicativos cujo custo não justifica seu suporte contínuo?
- Quais aplicativos são os mais caros? Esse custo é justificado?

## Próximas ações

Faça um acompanhamento com os usuários do aplicativo para ver se eles ainda têm um motivo válido para continuar usando o aplicativo.
