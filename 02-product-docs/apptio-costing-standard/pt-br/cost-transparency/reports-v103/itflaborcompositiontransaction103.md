# IT Finance - Detalhes do trabalho - Composição - Relatório de transação ( v103 )

Aplica-se a: Costing Standard 11.8.x em execução em TBM Studio v12 ou TBM Studio v11.

## Introdução

Use esse relatório para visualizar as transações de conta do mês atual para uma conta específica.

![](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/itflaborcompositiontransation103_1.png)

## Navegação

Gerenciamento de TI > Mão de obra > Centro de custos > Composição da conta > Exibição Tx

## Funções

Este relatório foi elaborado para:

- Equipe de finanças de TI
- Proprietário do centro de custo

## Objetivos

Use esse relatório para visualizar as transações da conta do mês atual.

## Perguntas respondidas

Você pode usar as informações apresentadas neste relatório para responder às seguintes perguntas:

- Quais são as transações do mês atual?
- Alguma das transações é inesperada? Em caso afirmativo, eles exigem acompanhamento para verificar se há cobranças incorretas, alterações de preços ou um erro de codificação (por exemplo, g.: centro de custo errado)?

## Próximas ações

- Entre em contato com o IT Finance para obter os documentos de suporte para uma transação específica em questão.
- Determine se a transação em questão é um evento único, uma tendência que terá impacto contínuo ou um erro de codificação que talvez precise ser corrigido.
