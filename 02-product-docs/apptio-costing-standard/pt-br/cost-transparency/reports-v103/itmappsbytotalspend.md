# Gerenciamento de TI - Aplicativos - Por relatório de gastos totais com aplicativos

Use esse relatório para analisar os gastos com aplicativos para operações e desenvolvimento.

Aplica-se a: Costing Standard 11.8.x em execução em TBM Studio v12 ou TBM Studio v11.

![](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/itmappsbytotalspend_1.png)

## Navegação

Gerenciamento de TI > Aplicativos > Por gasto total com aplicativos

## Funções

Este relatório foi elaborado para:

- Proprietários de aplicativos
- Proprietários do portfólio de aplicativos / VP de desenvolvimento e suporte de aplicativos
- Arquitetos corporativos

## Objetivos

Use este relatório para:

- Analise os gastos com aplicativos para operações e desenvolvimento até o momento.
- Veja rapidamente o total relativo de gastos com aplicativos para operações e desenvolvimento nos meses anteriores ao início do ano fiscal.

## Perguntas respondidas

As informações apresentadas neste relatório podem ser usadas para responder às seguintes perguntas:

- Quanto gastamos em operações e desenvolvimento até o momento neste ano?
- Onde está o nosso maior gasto por tipo de aplicativo, tipo de usuário ou importância comercial?
- É necessário tomar medidas para reduzir o risco das operações e planos atuais ou futuros?
