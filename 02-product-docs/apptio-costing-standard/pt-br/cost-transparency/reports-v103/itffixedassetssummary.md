# IT Finance - Ativos fixos - Relatório resumido ( v103 )

Aplica-se a: Costing Standard 11.8.x em execução em [TBM Studio v12](https://community.apptio.com/community/apptio/product-central/tbm-studio/studio-v12 "(Abre em uma nova guia ou janela)") ou [TBM Studio v11](https://community.apptio.com/community/apptio/product-central/tbm-studio/studio-v11 "(Abre em uma nova guia ou janela)").

## Introdução

Use esse relatório para identificar a depreciação atual e projetada de ativos fixos.

![](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/itffixedassetssummary_1.png)

## Navegação

IT Finance > Ativos fixos > Resumo

## Funções

Este relatório foi elaborado para:

- Equipe de finanças de TI
- Proprietário do centro de custo

## Objetivos

Use esse relatório para identificar a depreciação atual e projetada de ativos fixos.

## Perguntas respondidas

Você pode usar as informações apresentadas neste relatório para responder às seguintes perguntas:

- Qual é a depreciação atual dos ativos?
- Quando os ativos serão totalmente depreciados?
- Onde será o meu maior gasto para substituir os ativos depreciados?

## Próximas ações

- Revise a vida útil restante dos ativos clicando na guia Depreciation LIM (LIM de depreciação).
- Revise os ativos que estão totalmente depreciados clicando na guia Fully Depreciated (Totalmente depreciado).
