# Como verificar se um relatório é de estoque ou personalizado

Na **guia Relatórios (Modo de edição/guia Autor)**, você pode verificar se um relatório está pronto para uso (estoque) ou se foi personalizado. Isso pode ser útil para determinar se foram feitas alterações sem verificar os registros de auditoria.

## Versão de estoque de um relatório

1. Selecione um relatório específico em mente.
2. Entre no modo de edição.
3. Clique na guia **Author (Autor** ).
4. Observe que **a versão do estoque** está marcada.
