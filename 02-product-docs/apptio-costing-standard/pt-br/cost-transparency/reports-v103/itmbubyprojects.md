# Gerenciamento de TI - Unidades de negócios - Relatório de projetos ( v103 )

Use este relatório para revisar as despesas por projeto.

Aplica-se a: Costing Standard 11.8.x em execução em TBM Studio v12 ou TBM Studio v11.

![](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/itmbubyprojects103_1.png)

## Navegação

Gerenciamento de TI > Unidades de negócios > Por projetos

## Funções

Este relatório foi elaborado para:

- Proprietários de unidades de negócios
- CIOs
- Diretores financeiros

## Objetivos

Use este relatório para:

- Acompanhe os custos do projeto no último ano.
- Comparar os custos do projeto entre as unidades de negócios.
- Compare os custos do projeto para RTB, GTB e TTB.

## Perguntas respondidas

As informações apresentadas neste relatório podem ser usadas para responder às seguintes perguntas:

- Temos a combinação certa de gastos em projetos de RTB, GTB e TTB?
- Estamos alocando os gastos nos projetos de maior prioridade?
- São necessárias ações para alterar as despesas?

## Próximas ações

Obtenha as seguintes informações sobre um projeto clicando no nome na coluna Nome do projeto.

- Resumo do projeto
- Motoristas de projeto
- Fornecedores de projetos
- Trabalho de projeto
- Aplicativos impactados
