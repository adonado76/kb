# IT Finance - Variação de conta - Relatório de detalhes da transação ( v103 )

Aplica-se a: Costing Standard 11.8.x em execução em TBM Studio v12 ou TBM Studio v11.

## Introdução

Use esse relatório para identificar transações específicas que foram incorridas no período atual para uma conta específica.

![imagem](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/itfaccountvariancetransactiondetails103_1.png)

## Navegação

IT Finance > Variação de conta > Conta

## Funções

Este relatório foi elaborado para a equipe de finanças de TI.

## Objetivos

Use esse relatório para identificar transações específicas que foram incorridas no período atual para uma conta específica.

## Perguntas respondidas

Você pode usar as informações apresentadas neste relatório para responder às seguintes perguntas:

- Eu sei para que servem essas transações?
- O valor de uma determinada transação parece anormal?
- Alguma transação foi inserida incorretamente? Eles pertencem a uma conta diferente?
- Alguma transação parece que deveria ter sido incorrida em um período anterior e foi processada com atraso?

## Próximas ações

- Se uma transação tiver sido inserida incorretamente, entre em contato com o proprietário da conta para corrigi-la ou para documentá-la com uma explicação.
- Obtenha mais detalhes da transação no sub-livro Contas a pagar ou no sistema financeiro de registro.
