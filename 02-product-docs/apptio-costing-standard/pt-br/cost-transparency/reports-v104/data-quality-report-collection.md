# Coleta de relatórios de qualidade de dados

Essa coleta permite que os TBMAs e os administradores de sistemas identifiquem e meçam as lacunas dentro e entre os conjuntos de dados e priorizem suas ações por impacto nas alocações de custos:

- Resumo
- Conjunto de dados
- Mapeamentos
- Dados
- Dimensões
- Finanças
