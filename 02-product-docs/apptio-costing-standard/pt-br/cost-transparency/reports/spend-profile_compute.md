# Perfil de despesas

Esse relatório apresenta um perfil de custo dos custos de computação.

Esse relatório tem os seguintes elementos:

![](../../resources/images/ct_images/ct%20report%20collections/spend%20profile_1.png)

(1) KPIs: Os KPIs deste relatório são os mesmos do Compute - Summary. Consulte o elemento 1 no relatório Compute Summary (Resumo da computação).

(2) Filtros: Os filtros desse relatório são os mesmos do Compute - Summary. Consulte o elemento 2 no relatório Compute Summary (Resumo da computação).

(3) Despesas do mês atual por pool de custos: Use este gráfico para visualizar as despesas atuais do servidor por pool de custos (serviços externos, hardware, etc.).

(4) Tendência de 13 meses dos servidores: Use este gráfico para visualizar a contagem mensal de servidores, o custo por servidor e o gasto médio anual com servidores, para que você possa observar anomalias e tendências que não se alinham à sua estratégia de negócios.

(5) Gastos do servidor por: Use os gráficos de pizza by Profile, by OS e by Environment para entender o equilíbrio dos custos de seu servidor em várias métricas.

Perguntas respondidas:

- Como a estratégia de servidor está evoluindo?
- Onde estamos incorrendo em custos de servidor?
- Há anomalias em nossos gastos com servidores?
