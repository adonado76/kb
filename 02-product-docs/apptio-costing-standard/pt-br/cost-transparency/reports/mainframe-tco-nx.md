# Custo total de propriedade (TCO) de mainframes

Utilize este relatório para compreender os custos do mainframe, avaliar os gastos por categoria e período e identificar tendências de custos, a fim de apoiar as decisões de orçamento e investimento. Aplique filtros e visualizações disponíveis para se concentrar nos dados relevantes para sua análise.

Este relatório foi elaborado para ser utilizado pelos seguintes perfis:

- Diretores Financeiros
- Gerentes de Finanças de TI
- Gerentes de Operações de Mainframe
- Arquitetos de Negócios

## Respostas às perguntas

- Qual é a estimativa do custo anual das operações de mainframe com base nos gastos atuais?
- Como se comparam os gastos atuais aos de períodos anteriores?
- Quais categorias de custos contribuem mais para as despesas totais com mainframe?
- Como têm evoluído os níveis de consumo da MSU ( GCP ) e da MSU ( zIIP ) ao longo do tempo?
- Qual é a composição dos custos por linha de produtos e por produto?
- Quais produtos ou categorias apresentam tendências de aumento ou redução de custos?
- Qual é a parcela do custo do mainframe atribuída a cada função de negócio?

## Ações recomendadas

- Analise o custo anual previsto e compare-o com a dotação orçamentária.
- Analise as tendências de custos por categoria para identificar áreas com aumentos inesperados.
- Filtre por linha de produtos ou nome do produto para se concentrar em áreas de custo específicas.
- Compare o consumo de MSU em GCP e zIIP para identificar oportunidades de otimização da carga de trabalho.
- Exporte os dados resumidos para compartilhar com as equipes de finanças e operações.
- Estabeleça um acompanhamento regular para monitorar as tendências de custos e os padrões de consumo.
- Analise os produtos com custos elevados para determinar se é necessária uma otimização ou modernização.
