# Coleção de relatórios das torres de TI

Essa coleção fornece o custo por unidade para que você possa ver o custo total e as tendências de eficiência de acordo com a taxonomia padrão do site ATUM, alinhada com os benchmarks de seus pares. As análises intuitivas começam com uma exibição resumida que permite que você se aprofunde nos detalhes quando desejar.

O modelo automatizado mostra como a variação do orçamento GL afeta as torres/funções:

- [Revisão das torres de TI](it-towers-review.html)
- [Desempenho das torres de TI](ittowersperformance.html)
- [Composição das torres de TI](ittowerscomposition.html)
- [Planejamento de torres de TI](ittowersplanning.html)
