# Revisão de projetos

## Principais projetos

![](../../resources/images/ct_images/ct%20report%20collections/project%20review.jpg)

| Elementos-chave | Perguntas respondidas |
| --- | --- |
| - Revisar os KPIs na análise de despesas do projeto - Revisar os gastos do projeto nos principais projetos com base no acumulado do ano - Revisar as despesas do projeto nos principais projetos do período atual - Analise as despesas de projeto acumuladas no ano por iniciativa de negócios, portfólio de projetos, gerente de projetos, patrocinador de negócios, proprietário do projeto de TI ou tipo de despesas | - Quanto estamos gastando em comparação com o total gasto? - Qual a porcentagem de projetos iniciados no prazo e concluídos dentro do orçamento? - Quais são nossos principais projetos? - Qual é o nosso gasto com projetos no acumulado do ano com base na iniciativa de negócios? |

## Detalhes do projeto

![](../../resources/images/ct_images/ct%20report%20collections/project%20review_1.jpg)

| Elementos-chave | Perguntas respondidas |
| --- | --- |
| - Revisar os principais detalhes do projeto - Entenda os detalhes dos principais projetos, incluindo o orçamento do projeto. | - Quanto estamos gastando nos principais projetos em comparação com o orçamento aprovado? - Qual é o status do projeto? - O projeto está sendo acompanhado até a data de conclusão? |

## Detalhes do projeto Drilldown

![](../../resources/images/ct_images/ct%20report%20collections/project%20review.png)

| Elementos-chave | Perguntas respondidas |
| --- | --- |
| - Revisar os KPIs para os detalhes do projeto - Revisar detalhes do projeto - Analisar os custos e o desempenho do projeto | - Qual é a tendência em relação ao gasto orçamentário? - Qual é o objetivo do projeto? - Qual é o desempenho do projeto com o status de saúde atual? |

## Detalhes do projeto Revisão do orçamento

![](../../resources/images/ct_images/ct%20report%20collections/project%20review_2.jpg)

| Elementos-chave | Perguntas respondidas |
| --- | --- |
| - Analise os principais projetos e a iniciativa comercial do projeto - Revisão do orçamento aprovado para o orçamento remanescente | - Como os custos foram comparados ao orçamento aprovado? - Qual é a iniciativa comercial do projeto? - Qual foi o impacto do projeto na empresa? |
