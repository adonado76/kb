# Visão financeira

Use esse relatório para obter uma visão financeira dos gastos de seu data center por pool de custos.

Esse relatório tem os seguintes elementos:

![](../../resources/images/ct_images/ct%20report%20collections/financial%20view_1.png)

(1) KPIs: Os KPIs deste relatório são os mesmos do relatório Resumo. Consulte o elemento 1 no relatório resumido.

(2) Filtros: Os filtros desse relatório são os mesmos do relatório Resumo. Consulte o elemento 2 no relatório resumido.

(3) Custo YTD por pool de custos: Esse gráfico exibe o custo YTD de seus data centers por pool de custos.

(4) Tabela de detalhes: Essa tabela exibe dados detalhados de custo mensal resumidos por nome, local, gerente, região e nível. Use a lista Summarized Costs By (Custos resumidos por) para selecionar uma métrica. Use o gráfico para visualizar as despesas atuais do data center com base na métrica que você selecionar.

Perguntas respondidas:

- Há anomalias em nossos gastos?
- Há oportunidades de redução de custos?
- Qual aspecto dos gastos do data center é mais eficiente ou ineficiente?
