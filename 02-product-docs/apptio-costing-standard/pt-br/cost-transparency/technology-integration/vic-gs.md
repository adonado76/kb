# Informações sobre fornecedores - Integração com o Coups Introdução

O Coupa é um software de compras e pagamentos que contém dados que podem ser usados para implantar Vendor Insights . Siga as etapas abaixo para implantar Vendor Insights usando os dados do Coupa do Conector Coupa Datalink (Classic).

1. [Configure o Coupa para o Apptio](config-coupa.html)
2. [Configurar o conector Coupa DataLink](configure_the_datalink_coupa_connector.html)
3. [Configurar os dados do Coupa para o Apptio](coupa-configure-data.html)
4. [Instale e configure Vendor
   Insights usando os dados da Coupa](coupa-install.html)

**Especialistas no assunto necessários**

Os seguintes especialistas no assunto são necessários para realizar as etapas listadas acima.

- Administrador   
    
   Coupa Esta pessoa precisará de acesso para fazer login nas configurações administrativas do Coupa para registrar Datalink (Classic) como um aplicativo aprovado. Durante esse processo de configuração, o administrador do Coupa precisará coletar várias definições de configuração e fornecê-las ao TBMA para concluir a configuração do conector do Datalink (Classic).
- Administrador de Gestão de Negócios de Tecnologia (TBMA)   
    
   A configuração Vendor Insights requer um TBMA que possa acessar TBM Studio para configurar o Vendor
  Insights componente. Além disso, o TBMA precisará fazer login no console do administrador Datalink (Classic), que foi registrado pelo administrador do Coupa como um aplicativo autorizado no console do administrador Azure. O TBM Connect fornece orientações para guiar o TBMA pelo processo de configuração.
