# Resumo das fontes de dados necessárias para o Costing Standard Foundation

Para configurar o Costing Standard Foundation, você precisará dos seguintes conjuntos de dados:

- Livro Razão ( OpEx e CapEx )
- Orçamento ( OpEx e CapEx )
- Lista de centros de custo
- Plano de contas
- Registro de ativos fixos
- Número real de funcionários por centro de custo
- Número de funcionários planejados por centro de custo
- Lista de fornecedores
- Contagens de volume de infraestrutura por torre e subtorre de TI

Além disso, você precisará criar algumas tabelas de mapeamento, incluindo:

- Código de conta para mapeamento de pool de custos
- Mapeamento de ativos fixos para torre de recursos de TI
- Mapeamento da torre de recursos de mão de obra para TI
- Mapeamento de fornecedores para torre de recursos de TI
- Mapeamento do centro de custos para a torre de recursos de TI
