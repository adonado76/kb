# ARC RRC RU Reconciliação

◆ Aplica-se a: Vendor Insights no TBM Studio 12.8 e posterior ( v107 )

Use o relatório **ARC RRC RU Reconciliar** para analisar termos contratuais específicos de cobrança adicional de recursos (ARC) e crédito reduzido de recursos (RRC) para uma unidade de recursos (RU). Você pode visualizar as quantidades e o preço da unidade de recurso.

Use este relatório para:

- Analisar os termos específicos do contrato ARC/RRC para uma unidade de recursos
- Verifique as quantidades e o preço da unidade de recurso

Este relatório foi elaborado para:

- CIO e liderança sênior de TI
- Proprietários de aplicativos
- Proprietários de serviços
- Gerentes de Finanças de TI
- Gerentes de fornecedores

**Exibir o relatório de reconciliação ARC RRC RU**

No menu Aplicativo, selecione **Vendor Insights** .

1. Navegue até Coleções de relatórios > ARC/RRC.
2. Selecione qualquer item na coluna Unidade de recurso de uma tabela no componente de relatório Detalhes do ARC RRC para abrir o relatório ARC RRC RU para essa unidade de recurso.

Perguntas respondidas

Use as informações apresentadas neste relatório para responder às seguintes perguntas:

- O que me foi cobrado pela unidade de recursos e qual era a minha linha de base ARC/RRC?
- Qual é a diferença entre o que o fornecedor faturou e as unidades de recursos reais rastreadas no meu ambiente?
