# Summary of data sources needed for Costing Standard Foundation

To configure Costing Standard Foundation, you'll need the following data sets:

- General Ledger (OpEx and CapEx)
- Budget (OpEx and CapEx)
- List of Cost Centers
- Chart of Accounts
- Fixed Asset Register
- Actual Headcount by Cost Center
- Planned Headcount by Cost Center
- List of Vendors
- Infrastructure Volume Counts by IT Tower and Sub-Tower

In addition, you'll need to create some mapping tables including:

- Account Code to Cost Pool Mapping
- Fixed Assets to IT Resource Tower Mapping
- Labor to IT Resource Tower Mapping
- Vendors to IT Resource Tower Mapping
- Cost Center to IT Resource Tower Mapping
