# What's new in Costing Standard 2023

This section describes all of the Costing Standard application content changes
only.

To learn about the TBM Studio Server and Client release changes, see [What’s New in TBM Studio](../../studio/whats-new.htm "(Opens in a new tab or window)").

## Server 12.11.2 & Application Template v110 -7949 release - November 24, 2023

**New Application features**

**Eval() removal in Template v109, v110**: To take advantage of the performance improvements,
the Eval() function needs to be removed from a project’s configuration. In this release, the Eval()
function has been removed from all out-of-the-box content in Template v109 and v110. This will not
impact the functionality of existing customer project configurations.
