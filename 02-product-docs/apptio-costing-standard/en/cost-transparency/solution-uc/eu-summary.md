# End User Summary

End User use cases focus on understanding, allocating, and optimizing the cost of
workforce devices such as PCs, laptops, smartphones, tablets, and other client computing
equipment. These use cases help organizations improve cost transparency, strengthen
accountability for device spend, and make informed decisions around budgeting, refresh planning,
and fleet optimization.

Apptio offers two End User solutions that address these needs at different levels of depth
and maturity.

**End User Devices**

End User Devices provides a foundational view of end user device total cost of ownership
(TCO). This is an integrated solution which focuses on capturing device costs and allocating
them to end user services and business services to improve financial transparency and
accountability. This solution is best suited for organizations that want a straightforward,
cost-focused understanding of workforce device spend without operational or lifecycle
analytics. There are no out of box reports with this solution.

For more details on the End User Devices solution, go [here](eud-overview.html).

**End User Insights**

End User Insights delivers comprehensive, analytics-driven visibility into the end user
device fleet by combining cost, inventory, lifecycle, compliance, and usage data. It supports
advanced use cases such as refresh planning, compliance monitoring, orphan and excess device
identification, spare capacity optimization, and long-term cost reduction. This solution is
ideal for organizations looking to actively manage, optimize, and govern their end user
computing environment.

For more details on the End User Insights solution, go [here](eui-overview.html).
