# ARC RRC RU Reconcile

◆ Applies to: Vendor Insights on TBM Studio 12.8 and later (v107)

Use the
**ARC RRC RU Reconcile**
report to analyze specific additional resource charge (ARC) and reduced resource credit (RRC) contract terms for a resource unit (RU). You can view the resource unit's quantities and price.

Use this report to:

- Analyze specific ARC/RRC contract terms for a resource unit
- Review the resource unit's quantities and price

This report is designed for:

- CIO and Senior IT leadership
- Application Owners
- Services Owners
- IT Finance Managers
- Vendor Managers

**Display the ARC RRC RU Reconcile Report**

In the
Application
menu, select
**Vendor Insights**
.

1. Navigate to
   Report Collections > ARC/RRC
   .
2. Select any item in the
   Resource Unit
   column of a table in the
   ARC RRC Details
   report component to open the
   ARC RRC RU
    report for that resource unit.

![image](../../../../../03-media/apptio-costing-standard/resources/images/vi_images/vi/vi-arcru.jpg)

Questions answered

Use the information presented in this report to answer the following questions:

- What was I invoiced for the resource unit and what was my ARC/RRC baseline?
- What is the difference between what the vendor invoiced vs. actual resource units tracked in my environment?
