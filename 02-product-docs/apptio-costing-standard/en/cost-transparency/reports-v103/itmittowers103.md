# IT Management - IT Towers report (v103)

◆ Applies to: Costing Standard 11.8.x running on either TBM Studio v12 or TBM Studio
v11.

## Introduction

This report shows relative IT tower cost and composition by cost pool.

![](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/itmittowers103_1.png)

## Navigation

IT Management > IT Towers

## Roles

This report is designed for:

- CIO
- IT Management

## Objectives

Use this report to:

- Quickly see relative IT tower cost and composition by cost pool using the IT Towers by Cost Pool
  chart.
- Identify spend by IT Tower for current month, quarter and YTD using the Select Date Grouping
  option.

## Questions answered

The information presented on this report can be used to answer the following questions:

- How much have I spent by IT tower so far this year?
- Where is my biggest spend?
- What is driving my spend?
- Where is my biggest variance, by amount?
- What amount is material compared to my overall IT spend?
- Is action required to mitigate budget risk?

## Next actions

To see spend, volumes and unit cost, for tower, click the tower name in the table.
