# IT Finance - Cost Pools report (v103)

Applies to: Costing Standard 11.8.x running on either TBM Studio v12
or TBM Studio v11.

## Introduction

Use this report to view the fixed/variable spend and the OpEx/CapEx spend by cost pool.

![](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/itfcostpools103_1.png)

## Navigation

IT Finance > Cost Pools

## Roles

This report is designed for IT Finance personnel.

## Objectives

Use this report to view fixed/variable spend and OpEx/CapEx spend by cost pool.

## Questions answered

You can use the information presented on this report to answer the following questions:

- If demand or consumption drops, do I have flexibility to reduce spend?
- Where could I consider sourcing changes that would create a more variable cost structure?
- What financial agility is appropriate for our company at this time? Should I set a percent
  variable target?
- How does capitalization vary by cost pool?
- Where is most of my spend capitalized?

## Next actions

- Assess the future demand and services required (outside of Apptio).
- Evaluate the external rates and contract terms (outside of Apptio).
- Select the Account option above the table to view fixed/variable and OpEx/CapEx spend by account.
- Evaluate the external rates and contract terms (outside of Apptio).
