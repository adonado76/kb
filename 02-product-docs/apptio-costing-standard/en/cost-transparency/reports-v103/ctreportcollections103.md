# Costing Standard report collections (v103)

Applies to:
[Costing Standard](https://community.apptio.com/community/apptio/product-central/cost-transparency "(Opens in a new tab or window)") running on TBM Studio R12+ with Template
v103.

Note: This is the parent article for all the Costing Standard report collections for Template v103.
To determine which component you are using, see [Compare v104 and v103 Costing Standard reports](../reports-v104/comparev103v104reports.html).

## Navigation

1. Log in to Apptio.
2. In the **Environments** page, click **Cost Transparency**.
3. In the **Home** page, click one of the following report collections, or click a link
   to jump directly to a specific report.

Following are links to specific report collection lists:

- [Benchmarking Summary reports (v103)](../reports-v104/benchmarkingsummaryreports103_psdeprecate.html)
- [IT Finance reports](itfreports103.html)
- [IT Management reports](itmreports103.html)
- [Data Quality reports](dataqualityreports.html)
- [IT Infrastructure and Operations reports](itinfraopsreports.html)
