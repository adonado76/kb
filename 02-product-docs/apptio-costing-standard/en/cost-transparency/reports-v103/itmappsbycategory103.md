# IT Management - Applications - By Category report (v103)

Use this report to review top application spending by Application Investment Objective,
Application Lifecycle, Business Criticality, Application Type, Application User Category,
Application Family, or Application Function.

Applies to: Costing Standard 11.8.x running on either TBM Studio v12
or TBM Studio v11.

![](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/itmappsbycategory103_1.png)

## Navigation

IT Management > Applications > By Category

## Roles

This report is designed for:

- Application Owners
- Application Portfolio Owners / VP of Application Dev & Support
- Enterprise Architects

## Objectives

Use this report to:

- Use the drop-down list to review top application spending by Application Investment Objective,
  Application Lifecycle, Business Criticality, Application Type, Application User Category,
  Application Family, or Application Function.
- Review the trend of application spending by the selected category.

## Questions answered

The information presented on this report can be used to answer the following questions:

- Is this the correct mix of spending with across investment categories?
- Are our investments in applications sufficient to support corporate goals?
- Must action be taken to mitigate risk?
