# IT Management - IT Towers Details - Current FY Plan Variance - Tx report (v103)

◆ Applies to: Costing Standard 11.8.x running on either TBM Studio v12 or TBM Studio
v11.

## Introduction

Use to view account transactions for the current month’s spend broken out by responsible cost
center.

![](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/itmittowersplanvariancetx103.png)

## Navigation

IT Management > IT Towers > IT Tower Name >Tx View

## Roles

This report is designed for:

- Financial Analyst
- IT Tower Owner

## Objectives

Use this report to:

View account transactions for the current month’s spend broken out by responsible cost
center.

## Questions answered

The information presented on this report can be used to answer the following questions:

- What are the current month’s account transactions that make up the specific IT sub-tower
  spend?
- Which department and cost center is responsible for these transactions?
- Are any of these transactions unexpected? If so, do they require follow up to check for
  unexpected charges, pricing changes or a coding error (e.g. wrong cost center)?

## Next actions

- Contact IT Finance to pull up the supporting documents for a specific transaction in
  question.
- Determine if the transaction in question is a one-time event, a trend that will have ongoing
  impact, or a coding error that may or may not need to be corrected.
