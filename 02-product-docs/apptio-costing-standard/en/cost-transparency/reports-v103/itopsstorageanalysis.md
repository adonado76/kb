# IT Operations - Storage Analysis report

◆ Applies to: Costing Standard 11.8.x running on either TBM Studio v12 or TBM Studio
v11.

## Introduction

Use this report to review the cost, volume count, available space, and utilization rates for
storage.

![](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/itoperationsstorageanalysis_1.png)

## Navigation

IT Infra & Operations > Storage Analysis report

## Roles

This report is designed for:

- Storage leaders/managers

## Objectives

Use this report to:

- Review storage cost, volume, availability, and utilization.
- Analyze the data by location, tier, environment, and specific device.

## Questions answered

The information presented on this report can be used to answer the following questions:

- Which data centers have the highest and lowest storage costs? What are their charactistics?
- How is my storage volume distributed across the data centers?
- Which data centers have the highest and lowest storage utilization rates?
