# IT Operations - Summary report

◆ Applies to: Costing Standard 11.8.x running on either TBM Studio v12 or TBM Studio
v11.

## Introduction

Use this report to get an overview of the counts and utilization rates for servers, storage, data
centers, and service desk activity.

![](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/itopssummary_1.png)

## Navigation

IT Infra & Operations > Summary

## Roles

This report is designed for:

Infrastructure & Operations leaders/managers

## Objectives

Use this report to:

- Review the server metrics such as counts and cost per server.
- Review the storage metrics such as cost and utilization rates.
- Review the data center metrics such as cost per square foot.
- Review the service desk metrics such as ticket count and cost.

## Questions answered

The information presented on this report can be used to answer the following questions:

- How does my cost per server compare across the data centers?
- Are there data centers that are more efficient in there use of storage?
- Which data centers have the highest and lowest cost per square foot?
- What is the monthly average of tickets handled by the service desk?

## Next actions

Dig deeper into the metrics by selecting one of the other IT Operations tabs.
