# IT Management - IT Towers Details - Cost Pool Composition report (v103)

◆ Applies to: Costing Standard 11.8.x running on either TBM Studio v12 or TBM Studio
v11.

## Introduction

Use this report to identify spend by cost pool for each sub-tower.

![](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/itmittowerscostpoolcomp103_1.png)

## Navigation

IT Management > IT Towers > IT Tower Name > Cost Pool Composition

## Roles

This report is designed for:

- IT Management
- IT Tower Owner

## Objectives

Use this report to:

Identify the spend for each sub-tower by cost pool.

## Questions answered

The information presented on this report can be used to answer the following questions:

- Where is my biggest spend by cost pool?
- Does the spend for this category seem reasonable compared to other sub-towers?
- Is action required to further investigate spend in a particular cost pool?

## Next actions

- Look at the cost pool spend for previous periods to identify trends (increasing, decreasing or
  staying consistent).
- Investigate cost center composition to understand who is responsible for managing the costs by
  clicking the Cost Center Composition tab.
- Look at account transactions for the period by clicking the Budget Variance tab.
- Contact IT Finance Analyst for further investigation.
