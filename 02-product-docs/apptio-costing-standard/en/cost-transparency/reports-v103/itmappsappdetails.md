# IT Management - Applications - Application Details report (v103)

Use this report to review application spending by private cloud, physical, and
virtual.

Applies to: Costing Standard 11.8.x running on either TBM Studio v12
or TBM Studio v11.

![](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/itmappsappdetails103.png)

## Navigation

IT Management > Applications > Application Details

## Roles

This report is designed for:

- Application Owners
- Application Portfolio Owners / VP of Application Dev & Support
- Enterprise Architects

## Objectives

Use this report to:

- Review application spending and the number of users.
- Obtain the year-to-date unit cost per user for each application.

## Questions answered

The information presented on this report can be used to answer the following questions:

- Is the total cost per user in line with expectations or benchmarks?
- Is action required to mitigate risk?

## Next actions

Use the slicers in the report to review the different costs associated with the application.
