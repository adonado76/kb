# IT Management - Applications - By Vendor report (v103)

Use this report to identify spend by vendor, application family, and
application.

Applies to: Costing Standard 11.8.x running on either TBM Studio v12
or TBM Studio v11.

![](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/itmappsbyvendor103.png)

## Navigation

IT Management > Applications > By Vendor

## Roles

This report is designed for:

- Application Owners
- Application Portfolio Owners / VP of Application Dev & Support
- Enterprise Architects

## Objectives

Use this report to identify spend by vendor, application family, and application.

## Questions answered

The information presented on this report can be used to answer the following questions:

- Is the spending with each vendor consistent with the budget?
- Is the mix of OpEx and CapEx suitable to support current operations and future plans?
- Is action required to mitigate risk?

## Next actions

Use the IT Finance – Vendors report to follow up with the Vendor Manager.
