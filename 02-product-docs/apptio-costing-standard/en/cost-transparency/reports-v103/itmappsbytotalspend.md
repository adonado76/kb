# IT Management - Applications - By Total Application Spend report

Use this report to review application spending for operations and
development.

Applies to: Costing Standard 11.8.x running on either TBM Studio v12
or TBM Studio v11.

![](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/itmappsbytotalspend_1.png)

## Navigation

IT Management > Applications > By Total Application Spend

## Roles

This report is designed for:

- Application Owners
- Application Portfolio Owners / VP of Application Dev & Support
- Enterprise Architects

## Objectives

Use this report to:

- Review application spending for operations and development year-to-date.
- Quickly see relative total application spend for operations and development over the months back to the beginning of the fiscal year.

## Questions answered

The information presented on this report can be used to answer the following questions:

- How much have we spent for operations and development so far this year?
- Where is our biggest spend by application type, user type, or business criticality?
- Is action required to mitigate risk to current or future operations and plans?

## Next actions

Use the tabs in the Application Portfolio report to view application spending from different
perspectives.
