# How To: Check if a Report is Stock or Customized

On the **Reporting Tab (Edit Mode/Author Tab)**, you can check if a
report is out of the box (stock) or if it has been customized. This might be helpful in determining
if changes have been made without checking the audit logs.

## Stock Version of a Report

1. Select a specific report in mind.
2. Go into edit mode.
3. Click the **Author** tab.
4. Notice **Stock Version** is checked.
