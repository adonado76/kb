# Resource Towers Reports

The **IT Towers report collection** provides visibility into IT spend across
technology resource towers and sub-towers, enabling organizations to understand cost
distribution, performance trends, composition, and forward-looking planning at a functional
level. These reports support operational and strategic reviews by helping IT Finance and IT
leaders analyze how costs evolve over time, what drives tower spend, and how spend aligns with
budget and forecast.

This collection includes:

- IT Towers Review
- IT Towers Performance
- IT Towers Composition
- IT Towers Planning

## IT Towers Review

The IT Towers Review report provides visibility into
IT spend across resource towers and sub-towers, helping organizations understand where costs
are concentrated and how they change over time. It also provides a high-level comparison of
spend against budget to support periodic reviews and early identification of deviations.

This report is designed for:

- IT Finance
- IT Managers / Resource Managers
- Service Owners

**Insights Provided**

- Understand IT spend distribution across resource towers and sub-towers.
- Identify towers and sub-towers with the highest costs and assess whether spend
  patterns align with expectations and budget.
- Analyze how cost pools and vendors contribute to tower and sub-tower costs and
  identify unexpected drivers.
- Detect period-over-period changes in tower and sub-tower spend to identify emerging
  trends or anomalies.
- Review quantity and unit cost trends to understand the factors driving cost changes.
- Identify applications with the highest year-to-date consumption by tower and
  sub-tower.

For more details on how to use the IT Towers Review report go [here](https://www.ibm.com/docs/en/apptio-commercial/costing-standard/saas?topic=reports-it-towers-review "(Opens in a new tab or window)")

## IT Tower Performance

The IT Towers Performance report provides a
comparative view of IT resource tower spend over time. It highlights changes in spend across
towers and indicates whether towers are improving efficiency or experiencing increasing cost
pressure, with light reference to budget alignment.

This report is designed for:

- IT Finance
- IT Managers / Resource Managers

**Insights Provided**

- Compare quarter-over-quarter spend changes across IT resource towers.
- Identify towers with the largest cost reductions or increases relative to recent
  periods.
- Highlight performance trends that indicate improving or declining cost efficiency.
- Support performance reviews by identifying towers where spend movement may warrant
  further investigation.

For more details on how to use the IT Towers Performance report go [here](https://www.ibm.com/docs/en/apptio-commercial/costing-standard/saas?topic=reports-it-towers-performance "(Opens in a new tab or window)")

## IT Tower Composition

The IT Towers Composition report provides a structural view of IT resource tower costs by
breaking down spend across cost pools and vendors. It also provides a high-level view of how
tower spend compares to budget, supporting understanding of both cost makeup and financial
alignment.

This report is designed for:

- IT Finance
- IT Managers / Resource Managers

**Insights Provided**

- Understand how IT tower costs are distributed across cost pools such as hardware,
  software, and labor.
- Identify the primary cost pools driving spend within each IT resource tower.
- Analyze vendor spend by IT tower to understand where external spend is concentrated.
- Review tower spend relative to budget to identify areas that may require further
  analysis.

For more details on how to use the IT Towers Composition report go [here](https://www.ibm.com/docs/en/apptio-commercial/costing-standard/saas?topic=reports-it-towers-composition "(Opens in a new tab or window)")

![](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/rt-comp.png)

## IT Towers Planning

The IT Towers Planning report provides a forward-looking view of IT resource tower costs to
support financial planning and forecasting. It helps users compare actual costs against
budget and forecast, understand remaining budget, and assess how spend trends are tracking
over time at the tower level.

This report is designed for:

- IT Finance
- IT Managers / Resource Managers

**Insights Provided**

- Review IT tower costs in relation to budget and forecast to assess financial
  alignment.
- Understand remaining budget and projected budget depletion based on current spend
  trends.
- Compare trends for cost, budget, and forecast across IT resource towers over time.
- Analyze year-to-date cost, budget, forecast, and variance by IT resource tower to
  support planning decisions.

For more details on how to use the IT Towers Composition report, go [here](https://www.ibm.com/docs/en/apptio-commercial/costing-standard/saas?topic=reports-it-towers-planning "(Opens in a new tab or window)").
