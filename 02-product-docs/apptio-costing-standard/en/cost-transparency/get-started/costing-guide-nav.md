# A Guide to IBM Apptio Costing Navigation

This topic provides an overview of how the IBM Apptio Costing navigation is organized,
based on the current TBM Report Studio. This navigation will be different if you’re using the
New Report Studio.

## Classic Reports

Note: The menu items shown depend on your user permissions and configuration.

1. **Home Page**  - When accessing IBM Apptio Costing, the home page gives access to the various
   report collections available within the IBM Apptio Costing application. The reports are organized by
   areas of interest, revealing financial intelligence insights for key target personas.

   ![](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/costing-home.png)

   Click a heading to open the
   high-level summary report for a report collection.
2. **Report Navigation Buttons** - These navigation buttons allow you to move quickly through
   reports. The Back icon returns you to the previous page, while the Home icon takes you directly to
   your main landing page.

   ![](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/rep-nav.png)

**Navigation Sidebar Menu**

The left sidebar menu serves as your primary navigation
hub. It groups your report collections and provides access to My Landing Page, Recents, and
Favorites, enabling you to efficiently locate and return to the information that matters most.

![](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/left-nav-bar.png)

**My Landing
Page** - My Landing Page provides you with a personalized starting point in the application.
Select your preferred Report and nominate it as your preferred landing page, this allows you to
quickly resume your work without navigating through multiple menus.

![](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/config-lp.png)

**Recents** - Recents displays a
list of the items you have opened most recently. This view enables you to easily return to your
latest, most accessed reports without needing to search for them again.

**Favorites** -
Favorites provides a centralized location for the items you have marked as important. By adding
reports to your favorites, you can access them quickly and consistently from a single location.

![](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/config-fav.png)

**Top Navigation Menu**

The top navigation bar provides quick access to key
settings, navigation tools, help center, and connectivity to other IBM Apptio solutions.

![](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/top-nvg-bar.png)

1. **IBM Apptio Logo** - When the navigation sidebar is collapsed, the IBM Apptio Logo gets
   revealed. The IBM Apptio Logo allows you to switch between IBM Apptio applications available in your
   organization. IBM Apptio Costing supports common login and user administration through Frontdoor.
2. **Chevron Icon** - Expand or collapse the left navigation pane.
3. **Report Search** - Use the looking glass icon to quickly search for the report you’re
   looking for.
4. **Date Range** - IBM Apptio Costing is fully time-aware. Select and set your Date Range when
   navigating your reports.
5. **Environment** - IBM Apptio Costing provides you with 3 environments: Production, Staging
   and Development. Within Development, each person will have their own workspace. When navigating
   ensure you’ve selected the correct environment. Note: Permissions can be set to disallow access to
   certain environments, such as Development.
6. **Release** - IBM Apptio Costing provides you with release management optionality. Where most
   of the work and releases would happen in the Trunk, work can also take place in a Branch, which can
   get merged back into the Trunk. Hotfix is a 3rd option within release management.
7. **Comments and Collaboration** - The Comments panel allows you to collaborate directly within
   a report by adding context, asking questions, or sharing feedback with other users. You can leave
   public comments visible to all viewers or private comments for specific individuals or teams. You
   can also reply to existing comments and use mentions to notify users and bring them into the
   conversation. This helps keep discussion tied to the data, improving clarity, traceability, and
   decision-making.
8. **Help** - Open the Help Center, send product feedback, view release notes, or access the IBM
   Community.
9. **Settings** - The Settings section provides administrators with access to key configuration
   options that control how the application behaves for your organization. From this menu, you can
   manage system preferences, adjust project settings, configure multi-currency, and access tools such
   as Data Advisor, Reference Applications, and Row-Level Security. These options allow you to tailor
   the environment to your operational, security, and reporting requirements, ensuring a consistent and
   governed user experience. The About page provides detailed information on your current client and
   server versions, helping you verify your environment configuration and ensure compatibility with
   product updates and features.
10. **Profile Settings** - Manage your user profile; Admins can impersonate users.
11. **Application Menu** - Switch between IBM Apptio applications available in your organization.
    IBM Apptio Costing supports common login and user administration through Frontdoor.
12. **Refresh**  - Select the refresh frequency. By default, set to 3 seconds but can be switched
    to manual as needed.
13. **Saved Views and Reset** - This menu allows you to manage how a report is displayed and
    quickly return to preferred settings. You can save your current configuration using Save View or
    Save As, preserving filters, slicers, and layout choices for future use. If you need to start fresh,
    you can reset all slicers, reset global filters, or restore the report to its original default
    state. These options help you personalize your analysis while maintaining an easy path back to the
    baseline report.
14. **Exporting and Sharing Reports** - These options allow you to export and distribute report
    insights in the format that best suits your needs. You can download the report as an Excel or PDF
    file for offline analysis or recordkeeping. The Send Email option lets you share the current report
    directly with other users, while Email Subscriptions enables you to schedule automated report
    deliveries at regular intervals. Together, these features support seamless collaboration and timely
    communication across your organization.

**Multi-Currency**  - If multi-currency is enabled, use the currency menu to change your
view to a different currency.

## New Report Studio

**Look for the blue “Open IBM Apptio Report Studio” button** – Once your instance is upgraded
to 12.11.21 and above, you will see a blue button at the top of your current report studio. Clicking
this will open a list of available projects.

Note: Customers on versions 12.11.19 should be
enrolled in the public preview to see the blue button.

![image of the new report studio button](../../../../../03-media/apptio-costing-standard/resources/images/studio_images/nrs-button.png)

**Select a project to open it in the new report studio** – Choose a project from the drop-down
to automatically launch it in the new experience.

The new Report Studio opens on the landing page, as shown below.

**Reports Home** 

![navigation of landing page in new report studio](../../../../../03-media/apptio-costing-standard/resources/images/studio_images/nrs-home.png)

1. New button – Click here to create a new report or a new report collection
2. Search – Search for an existing report
3. New Reports – Lists all reports created using the new Report Studio.
4. Report Collections – Displays groups of related reports organized into collections.
5. Report Overflow Menu (3 dots menu) - Click here to access a host of operations that can
   performed on a report, like deleting a report, or exporting the report’s definition and
   configuration.
6. Project Settings – Click here to check in one or more reports, revert changes to them or to
   import a report configuration into the project in context.

**Navigation Panel** 

![navigation of top panel in new report studio](../../../../../03-media/apptio-costing-standard/resources/images/studio_images/nrs-navpan.png)

1. App Switcher – Use this to switch to another IBM Apptio product
2. Profile Settings - Use this to manage your profile, impersonate another user (Note: Admins may
   want to do this to check a user's permissions or to troubleshoot problems), and Logout
3. Help – Use this to access the new Report Studio help documentation
4. Settings - Use this to change the customer access, update permissions, and access multi-currency
   settings
5. Environment Dropdown – Use this to switch between the environments and user workspaces.
6. Date Picker – Use this to switch between date periods
7. Take me to – Use this quick navigation menu to switch to the Classic TBM Studio and/or the New
   Report Viewer.

To know more, see [IBM Apptio New Report Studio](https://www.ibm.com/docs/en/apptio-commercial/tbm-studio/saas?topic=rn-recent-releases "(Opens in a new tab or window)").
