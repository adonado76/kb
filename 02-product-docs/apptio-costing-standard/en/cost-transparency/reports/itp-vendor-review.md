# ITP – Vendor Review

![](../../resources/images/ct_images/ct%20report%20collections/itp%20vendor%20review.jpg)

Key elements:

- Analyze vendors by Cost Center, Cost Center Name, Vender Name, Contract Type and Location.
- Analyze an original plan and latest plan by vendor.

Questions answered:

- What changes should we make to re-balance vendor spend?
- Do we have redundant vendors?
- What is the monthly Avg amount of the vendor?
