# Operational Summary

Use this report to view your monthly operational spending for data center resources, such
as power capacity and cost per square foot.

This report has the following elements:

![](../../resources/images/ct_images/ct%20report%20collections/operational%20summary_1.png)

(1) KPIs: The KPIs in this report are the same as in the Summary report. See element 1 in the
Summary report.

(2) Filters: The filters in this report are the same as in the Summary report. See element 2 in
the Summary report.

(3) Operational Metrics by Category: Use the options at the top of this chart to selectively
display various metrics. Additional metrics are available in the Summarized By list. Use the chart
to view the current Data Center spend based on the metric you select.

(4) Table: Use this table for a more detailed view of the data in Operational Metrics by
Category. The Summarized By list changes the data displayed in this table. Select any link in the
first column to see pivot details for that item.

Questions answered:

- Which of my data centers are most expensive, and why?
- Are there opportunities to reduce cost?
- Which aspect of data center spend is most efficient, or inefficient?
