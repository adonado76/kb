# ITP – Asset Review

This report allows you to view the details of your deprecated assets (such as class, age,
location, IT tower, vendor, asset type, and category) over time.

![](../../resources/images/ct_images/ct%20report%20collections/itp%20asset%20review.jpg)

| Key element | Description |
| --- | --- |
| (1) Report Collection | This report collection provides the details you need to review your resource (labor and IT tower) spend, trends, and usage:   - Fixed Assets report - ITP – Asset Review (described on this page) |
| (2) Chart and table | View your asset count and spend by age range across cost centers, roles, and locations.   - Original Asset Plan: Use this tab to view details of the original plan. The table shows the   purchase date, in-service date, quantity, and purchase price of each asset. Use the options above   the table to add columns for cost center name, vendor, location, and comments. - Latest Asset Plan: Use this tab to view the current plan, with changes since the original. The   functionality on this tab is the same as in the Original tab. |
| (3) Slicers | The slicers in this report let you see your asset data limited by owner, name, class, asset, and location.  The following roles can use the slicers in this report for a more personalized view:   - IT Financial Controller or CIO: Without setting any slicers, you can see the overview of the   asset spend by class and the purchase date and price of each asset class. - Cost Center Owner or CIO -1: Set the filters to see specific areas of responsibility and manage   your deprecation budget and timing. - Financial Analyst: Set the slicers for areas you support and see asset deprecation across the   company. |
