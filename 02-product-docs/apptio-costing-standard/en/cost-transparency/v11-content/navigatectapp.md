# Find your way around Costing Standard

## Costing Standard Toolbar

![](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/apex-toolbar-options.png)

Toolbar

| Icon | Name | Description |
|  | All products | Navigate to another Apptio product |
|  | Maximize / Minimize | Maximize / minimize navigation  From here, you can select the report collection you want to use. |
|  | Comment | You can collaborate with other users, set notifications and leave notes for yourself.  [Learn more about Collaboration](../../cnc/home.htm "(Opens in a new tab or window)"). |
|  | Help | You can:   - search the Help Center - send feedback |
|  | Application settings | You can:   - go to application administration - view details of the application, such as the version and environment. |
|  | Profile settings | You can:   - manage your profile - impersonate another user.   Administrators may want to do this to check a user's permissions or to troubleshoot problems.   - log out - switch to Classical Navigation. |
