# Step 13: Close the merge trunk

1. Select the branch for the new template version (such as Version 104 Upgrade).
2. On the **Project ribbon**, click **Close Branch**.

   The **Confirm Close** dialog
   opens.
3. Click **OK** to close the branch.

   ![](../../../../../../03-media/apptio-costing-standard/resources/images/ct_images/upgrade-from-v3-29.png)
4. Confirm that Trunk is no longer displayed in the navigation bar.

   ![](../../../../../../03-media/apptio-costing-standard/resources/images/ct_images/upgrade-from-v3-30.png)

   Tip: Close the upgrade branch as soon as possible. The branch consumes the same amount of resources as the main trunk project. Closing the upgrade branch will free up resources and improve the overall performance.
