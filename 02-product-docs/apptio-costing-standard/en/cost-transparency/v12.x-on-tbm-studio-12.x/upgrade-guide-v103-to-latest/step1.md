# Step 1: Create a branch

Perform the upgrade process in a separate branch, rather than in a personal
**Development** environment.

1. Before creating the branch, complete and check in any changes in your main project.
2. On the **Project tab**, click **Create Branch**. The
   Create Branch dialog box opens.
3. Enter a new branch name, for example, "Version 104 Upgrade."

   ![](../../../../../../03-media/apptio-costing-standard/resources/images/ct_images/upgrade-from-v3-1.png)
4. Click **OK**.

   The Calculation Queue dialog opens. Wait
   for the calculations to complete.

   ![](../../../../../../03-media/apptio-costing-standard/resources/images/ct_images/upgrade-from-v3-2.png)
