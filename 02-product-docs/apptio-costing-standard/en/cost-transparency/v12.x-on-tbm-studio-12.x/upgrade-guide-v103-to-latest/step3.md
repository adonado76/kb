# Step 3: Change component version

1. On the **Project** tab, click **Project Settings**.

   The
   Edit Project Settings dialog opens.
2. For **Component Version**, select the latest version, for example,
   **Version 104**.

   ![](../../../../../../03-media/apptio-costing-standard/resources/images/ct_images/upgrade-from-v3-5.png)
3. Click **Save**.
4. Check in the change and enter a description, such as "Project Settings: change to v104."
