# SAP LeanIX Reports

While no specific SAP LeanIX reports exist in IBM Apptio Costing today, the integration
significantly enhances existing IBM Apptio reports and capabilities. Users can leverage IBM
Apptio’s Costing reporting and cost-modeling capabilities to produce:

- **Application-level TCO reports**

  Run cost, development cost, maintenance, total cost over time; using SAP LeanIX data as
  the source for applications, dependencies, components, user groups, and business
  capabilities. Adding additional views to the out-of-the-box reporting, e.g.: Application
  Cost YearToDate by Technical Suitability, Functional Suitability and Compliance.
- **Business Unit TCO reports**

  Using SAP LeanIX relationships (Applications, Business Capabilities, User Groups,
  Providers) to allocate cost appropriately.
- **SAP LeanIX export report**

  IBM Apptio Costing has an integration component that
  installs a pre-defined report that the SAP LeanIX API uses to pull application TCO metrics
  directly from IBM Apptio Costing into SAP LeanIX, to enhance reporting and cost visibility
  in SAP LeanIX.
