# Create the Costing Standard project

The first step in the Costing Standard configuration process is to create the Costing Standard project. When you create the Costing Standard project, the Costing Standard application
creates the standard metrics, calculation models, and reports.

## Procedure

1. Log in to the Apptio instance.
2. Open the Settings menu and click **New Project**.
3. In the New Project dialog, enter the name of the project.
4. Select **Costing Standard** for the project type.
5. Click **OK**.
