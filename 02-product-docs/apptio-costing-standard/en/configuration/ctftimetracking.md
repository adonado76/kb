# CTF - Time Tracking component: install and configure

The Time Tracking component uses project and other time reporting information to allocate
costs and to provide more detailed Project Labor cost and effort analysis. The CTF - Time Tracking
component is not installed with the standard Costing Standard project.

## Before you begin

You must install the following components before you install the Time Tracking component:

- Cost Source
- Labor

## About this task

Applies to: Costing Standard on TBM Studio 12.0 and later

The CTF - Time Tracking component provides insights for IT management and IT finance in support
of monthly, quarterly and annual operational and strategic reviews with a focus on project labor
allocations and the broader allocation of labor to projects and other operational activities.

The CTF - Time Tracking component is not installed with the standard Costing Standard
project.

## Procedure

1. Open the Costing Standard project.
2. Click the Project tab.
3. Click Components in the Ribbon.
4. Click the CTF - Time Tracking component.
5. Click Install.
