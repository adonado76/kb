# Cloud Use Cases in Costing Standard

**Apptio writers**: To this shortdesc tag, add a short description, which is an intro
paragraph 1-2 sentences long,

Table 1.

| Use Case | Questions Answered |
| --- | --- |
| Public Cloud Transparency |  |
| Understand overall public cloud spend | What is my overall spend on public cloud, across IaaS/PaaS and SaaS? How does my public cloud spend break down by compute, storage, and network?  How is that spend trending?  What are the consumption metrics that are driving that spend? |
| Understand burdened public cloud spend | What are the basic cost drivers (beyond provider service costs) of public cloud services? What is the cost of labor required to maintain and manage public cloud services?  What roles in my organization are involved in maintaining and managing public cloud spend? |
| Analyze and explore IaaS/PaaS spend | How does my IaaS/PaaS spend breakdown by vendor? What are the nature of the cloud services my organization is consuming (e.g. compute, storage, etc)?  Who in my organization is driving the most cloud costs (and how is that trending)?  What applications are consuming the most cloud services?  What are the detailed instance types I am using the most and spending the most on? |
| Analyze and explore SaaS vendor spend | How does my SaaS spend break down by vendor? What vendor managers are responsible for the most SaaS spend in my organization? |
| Understand financial impact of cloud migration | What would be the financial impact of moving certain workloads to public cloud services? |
| Hybrid Cloud Transparency |  |
| Compare costs and unit costs of on-prem versus public cloud infrastructure | Across my entire compute portfolio, what are my fully burdened on-prem costs and unit costs versus those for public cloud? |
| Understand public cloud infra costs by application | What are the public cloud costs attributed to a particular application and how do they impact TCO? How do different types of infra (on-prem vs. cloud) affect an application's TCO? |
| Analyze Hybrid Cloud investment across app portfolio including private and public cloud deployments | Across my portfolio of applications what is the breakdown of spend by private cloud, public cloud, and on-prem resources? |
| Private Cloud Transparency |  |
| Analyze overall infrastructure & private cloud TCO | What is the fully burdened cost of the infrastructure supporting my private cloud services? |
| Analyze capacity and cost per unit capacity of private cloud server infrastructure | What is my per unit cost of capacity for my private cloud? How should I price private cloud services (based on what it costs to support those services)? |
