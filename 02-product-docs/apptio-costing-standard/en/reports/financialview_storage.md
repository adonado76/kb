# Financial View

Use this report for a financial view of your Data Center spending per cost
pool.

This report has the following elements:

![](../../resources/images/ct_images/ct%20report%20collections/financial%20view_1.png)

(1) KPIs: The KPIs in this report are the same as in the Summary report. See element 1 in the
Summary report.

(2) Filters: The filters in this report are the same as in the Summary report. See element 2 in
the Summary report.

(3) Cost YTD by Cost Pool: This chart displays the cost YTD of your data centers per cost
pool.

(4) Details Table: This table displays detailed monthly cost data summarized by name, location,
manager, region, and tier. Use the Summarized Costs By list to select a metric. Use the chart to
view the current data center spend based on the metric you select.

Questions answered:

- Are there anomalies in our spending?
- Are there opportunities to reduce cost?
- Which aspect of data center spend is most efficient, or inefficient?
