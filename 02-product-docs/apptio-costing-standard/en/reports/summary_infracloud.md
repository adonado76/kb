# Summary

The Infrastructure & Cloud Summary report contains the following elements:

![](../../resources/images/ct_images/ct%20report%20collections/summary_1.jpg)

| Key element | Description |
| --- | --- |
| (1) Report collection | This report collection provides the details you need to review your resource (labor and IT tower) spend, trends, and usage:   - Infrastructure & Cloud, Server TCO and Storage TCO (described in this page) |
| (2) & (3) YTD Cost Summary by Virtualization Profile | Select the Server TCO angle bracket to jump to the Server TCO tab.    Use the chart to view a summary of your YTD costs for public and private cloud usage, physical servers, and virtual servers.  The table provides details about YTD cost, percentage of change, instance hours, and cost per hour. Select an item in the Virtualization Profile column to open a dialog with more detail about the item you clicked.     - KPIs: provide a high-level view of your virtual asset spend. - Trend chart and table: The trend chart shows cost per instance hour vs. the 12-month average   month-by-month. The table shows the cost, server count, instance count, and memory capacity for each   environment and class. |
| (4) & (5) YTD Cost Summary by Type | Select the Storage TCO angled bracket to jump to the Storage TCO tab.    Use the chart to view a summary of your YTD costs broken out for each type of storage system (such as storage area networks [SAN], network attached storage [NAS], and tape systems).  The table provides details about YTD cost, percentage of change, space, percentage of GB change, and cost per GB. Click an item in the Type column to open a dialog with more detail     - KPIs: Provide a high-level view of your storage spend. - Trend chart and table: The trend chart shows costs per unallocated address month-over-month. The   table shows the location and manufacturer of each storage asset, and breaks out the cost, cost YTD,   installed space, addressable space, and allocated space for each location and manufacturer. |
| (6) & (7) YTD Public Cloud OpEx Summary by Service Category | Select the Public Cloud TCO angled bracket to jump to the Public Cloud landing page. |
