# Projects Review

## Top Projects

![](../../resources/images/ct_images/ct%20report%20collections/project%20review.jpg)

| Key Elements | Questions Answered |
| --- | --- |
| - Review KPIs on Project Spend Analysis - Review Project Spend on Top Projects on YTD basis - Review Project Spend on Top Projects for current period - Review YTD Project Spend by Business Initiative, Project Portfolio, Project Manager, Business   Sponsor, IT Project Owner, or Spend Type | - How much are we spending compared to total spend? - What percentage of projects started on time and completed on budget? - What are our top Projects? - What is our YTD project spend based on Business Initiative? |

## Project Detail

![](../../resources/images/ct_images/ct%20report%20collections/project%20review_1.jpg)

| Key Elements | Questions Answered |
| --- | --- |
| - Review Top Project Details - Understand details behind top projects including project Budget. | - How much are we spending on Top Projects compared to Approved Budget? - What is the status of the Project? - Is the project tracking to the completion date? |
