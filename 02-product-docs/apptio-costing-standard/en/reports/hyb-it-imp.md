# Hybrid IT TCO Impact – Insights

![](../../resources/images/ct_images/hytco-insght.jpg)

| Key Benefits | Details |
| --- | --- |
| - Identify the number of applications that have a TCO or unit cost higher than before migration - Understand the distribution of migrations by Migration Type, such as:   - On-Prem -> Public Cloud   - Public Cloud -> Private Cloud   - Private Cloud -> On-Prem   - Etc - Understand the distribution of migrations by Migration Strategy (Rehost, Refactor, Rebuild etc.) | For  : C-Suite, Application Owners, Finance teams  Use Case  :  Evaluate Migration Financial Benefit  Understand whether application migrations across your Hybrid estate are delivering a financial benefit or not. |
