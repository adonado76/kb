# Manage Your Plans

*The Plan listing page is the primary interface for creating, managing and viewing plan contents and analytics.*

The following table describes the index of numbers in the previous screen:

| # | Name | Description |
| --- | --- | --- |
| 1 | List of available plans and their properties | Table showing all plans associated with the currently selected view. Click a row to open that plan. You can filter and sort the table by plan properties. Refer to Overview to learn more. |
| 2 | View Selector | Choose the view you want to work with from the View Selector. Doing so refreshes the table to show only those plans associated with the selected view. To see all plans in the system, clear the view selection. |
| 3 | Create Plan | Choose this button to open the New Plan drawer. Refer to Creating Plans to learn more. |
| 4 | Table actions menu | Choose this button to select the action you want to apply to your selected plans. The options are: Open Plan - Open the plan so it can be edited by users. Close Plan - Close the plan to editing. Users can view the content but cannot edit it. Rename Plan - Rename the selected plan. You will be prompted to provide a new name for the plan. Plan names must be unique across all the plans in a given view. Duplicate Plan - Create a copy of the selected plan. Delete Plan - Permanently deletes the selected plan or plans. You will be prompted to confirm this action. |
