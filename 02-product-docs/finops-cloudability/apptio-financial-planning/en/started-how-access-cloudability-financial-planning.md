# How to access Cloudability Financial Planning

Cloudability Financial Planning is available under the Plan menu in the navigation pane for Cloudability . Tools for creating, managing and viewing plans and spend-to-plan performance are available from the Plans link. See Introduction to Plans to learn more.

## Plan Administrators

Users with Cloudability Admin or Cloudability Planning Admin permissions. They have access to additional functionality for configuring recommended dimensions for use by users while creating Plans. Tools for creating and maintaining planning dimensions to support your planning process needs are available from the Plan Dimensions link located under Organize menu in the Cloudability navigation pane. See Working with Plan Dimensions to learn more.
