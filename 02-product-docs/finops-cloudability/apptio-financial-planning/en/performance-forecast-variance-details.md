# Forecast Variance Details

Provides a period by period tabular breakdown of forecast variances. Each period shows values for actuals (if period is in the past, otherwise shows a system-generated Estimate ), Forecast , Variance , and Variance % . Variance is computed as Forecast - actuals/ Estimate , so negative values indicate that Forecast is over actuals (or Estimate) for the period.

Forecast variances can be analyzed using interactive "pivot table" like functionality allowing you to summarize and drill into variances by different dimensions. See Overview to learn more.
