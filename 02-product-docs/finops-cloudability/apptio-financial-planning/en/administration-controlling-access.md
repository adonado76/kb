# Controlling access

*Cloud Financial Planning uses Enhanced Access Administration to assign and manage user access and permissions. Enhanced Access Administration can be accessed via the Access Administration option under the Gear menu located in the top-level toolbar area for Cloudability .*

By default, only Cloudability Admin users can access CFP functionality; a Cloudability Admin must grant permissions to other Cloudability Users in order for them to access CFP functionality. For information on how to use Enhanced Access Administration to manage user permissions, see Manage user permissions and roles in the Enhanced Access Administration help.
