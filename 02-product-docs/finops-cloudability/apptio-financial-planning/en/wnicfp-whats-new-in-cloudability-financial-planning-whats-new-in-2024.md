# What's new in Cloudability Financial Planning : What's new in 2024

## CFP - QoL enhancements and API access - April 16, 2024

In today's release, we have added new plan creation options, enhanced our analytics with full page mode and drill support, and opened up API access to core Cloudability Financial Planning (CFP) functionality.

## CFP Alerting & Feature enhancements - February 26, 2024

Today, we are unveiling Plan Alerts , a new feature designed to boost user awareness and foster increased accountability for cloud spending. With Plan Alerts, plan owners can seamlessly configure budget and forecast notifications for themselves and others within the plan. Unlike Cloudability 's budget notifications which require individual setup by users, CFP plan alerts can be shared. This enables plan owners to create alerts for all cost owners within the plan, notifying them when spending exceeds or is projected to exceed their approved budget thus ensuring that cost owners stay informed about their spending performance relative to critical financial targets.
