# Finding your way around a Plan

*All plans have a standardized structure. The plan is a comprehensive document that includes all the necessary data and perspectives required for editing, managing, and analyzing it.*

| # | Name | Description |
| --- | --- | --- |
| 1 | Plan name | Name of the currently open plan. |
| 2 | View selector | Currently selected view: If you change the View selection while a plan is open, it will return you to the plan listing page, where you can choose a plan that is compatible with the newly selected view. |
| 3 | Plan section | A plan has multiple sections. Click the tab to switch to a different section. Dashboard - interactive analytics on the plan contents and current spend-to-plan performance. Forecast - View and edit forecast amounts. Budget - View and modify budget targets. Status - View change status. (e.g. last modified date /time and user) |
| 4 | Section views | Some plan sections have multiple sub-views. Click the tab to switch views. |
| 5 | Cost Ownership filter | Drop-down selector used to filter plan contents by cost ownership. Multiple cost owner values can be selected/ applied. Selection applies to all pages (sections) in the plan. |
| 6 | Plan Actions menu | Additional plan actions. Note: Some actions require Plan Owner access in order to use. Actions not available to available for the current user will appear grey. Permissions - Share plan with other users and manage their access permissions . This is for Plan Owners only. Dimensions - Modify the dimensions used in the plan and their properties. This is for Plan Owners only. See Plan Dimension Types to learn more. Alerts (not available yet) - Setup and manage alerts for yourself, and optionally share alerts with other users. Note: you must be a Plan Owner in order to create shared alerts. Export Plan - export the plan contents to a CSV file. |
