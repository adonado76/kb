# Making forecast adjustments

The following table describes the index of numbers in the previous screen:

| # | Name | Description |
| --- | --- | --- |
| 1 | Add new adjustment | This opens the Adjustment drawer, where you choose dimension values for the new line item. You must select a value for the cost ownership dimension, but only for the objects you have edit permissions for as granted by the Plan Owner when they shared the plan with you. The Plan Owner may have made other dimensions required. In such cases, you must specify values for these dimensions before adding the line item. If you do not, the system will prompt you to do so. After adding the line item, you will be able to enter values on the Adjustment Values tab. |
| 2 | Edit adjustment | Clicking the pencil icon opens the Adjustment drawer where you can edit the Dimension or Adjustment values for the selected line item. |
| 3 | Amount entry | Click on a cell to edit it. You may enter forecast values using amounts or as a percentage of baseline. Entering one calculates the other. Note: if your baseline amount for the period is 0, then the amount will be 0 regardless of the percentage entered. You may enter amounts at either the individual period or summary period level (quarter or year). Entering an amount in a summary period will even-spread the amount across all of its contained periods. Example: entering $12,000 in the FY23 period will enter $1,000 into every monthly period in FY23. |
