# Update baseline forecast

*In the baseline tab, you can view and analyze the generated baseline forecast. You can also update it if you wanted to pull in latest actuals or modify the forecast generation parameters. When you update the baseline forecast, Cloudability Financial Planning regenerates the forecast using the latest actuals and any changes to the forecast generation parameters.*

![The Forecast tab in Cloudability Financial Planning. The image includes annotations numbered 1 through 3, corresponding with the numbers in the Table 1, to describe the UI elelments.](../../resources/images/cloudability_images/cfp-updating-baseline-forecast.png)

The following table describes the index of numbers in the previous screen:

| # | Name | Description |
| --- | --- | --- |
| 1 | Cost Ownership | Use this dropdown menu to filter forecast contents by cost ownership. You can apply multiple cost owner values. This selection applies to all pages (sections) in the plan. |
| 2 | Baseline forecast line items | This table displays your quarterly spend so you can analyze your baseline forecast. You cannot edit these values. Instead, use the Forecast tab to make edits to baseline forecast line items. Visit Overview to learn more. |
| 3 | Plan actions menu | Only Plan Owners can use these actions. Import Plan - Import the baseline forecast from a CSV file. Export Plan - Export the baseline forecast to a CSV file. |
