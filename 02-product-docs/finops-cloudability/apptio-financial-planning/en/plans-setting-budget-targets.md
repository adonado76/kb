# Setting budget targets

*Each plan also includes a budget at the same level of cost ownership as the forecast. That is, it is automatically populated with budget line items, one for every cost object (value of the cost ownership dimension). This alignment enables you to relate budget amounts to the forecast and use them as spending limits for specific cost ownership levels.*

Budget permissions are controlled at the cost object level just like forecasts. However, there is a separate permission for Budget Edit vs. Forecast Edit so that you can separate who can set budget targets from who can enter forecasts.

The following table describes the index of numbers in the previous screen:

| # | Name | Description |
| --- | --- | --- |
| 1 | Edit budget | Clicking the pencil icon opens the Edit Budget drawer where you can edit budget amounts for the selected line item. |
| 2 | Amount entry | Click on a cell to edit it. You may enter budget values using amounts or as a percentage of plan (forecast). Entering one calculates the other. If your forecast amount for the period is 0, then the amount will be 0 regardless of the percentage entered. You may enter amounts at either the individual period or summary period level (quarter or year). Entering an amount in a summary period will even-spread the amount across all of its contained periods. For example: Entering $12,000 in the FY23 period will enter $1,000 into every monthly period in FY23. |
