# Budget Variance Details

Provides a tabular breakdown of budget variances by period. Each period shows values for actuals (if period is in the past, otherwise shows Forecast ), Budget , Variance , and Variance % . Variance is computed as Budget - actuals/ Forecast , so negative values indicate that actuals (or Forecast ) is over the Budget for the period.

Budget variances can be analyzed using interactive "pivot table" like functionality allowing you to summarize and drill into variances by different dimensions. See Overview to learn more.
