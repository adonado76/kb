# Plan Dimension Types

Navigate to Organize > Plan Dimensions.

In Cloudability Financial Planning, there are 2 possible types of plan dimensions:

- Cloudability Plan Dimension: These plan dimensions are used to summarize and align actuals to plan line items.
- Plan-only Plan Dimension: These plan dimensions are used to capture additional details on adjustments (new and changed forecast line items) and can be as simple as a Comments field or as structured as a list of Initiatives.
