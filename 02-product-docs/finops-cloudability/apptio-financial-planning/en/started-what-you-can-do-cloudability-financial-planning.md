# What you can do with Cloudability Financial Planning

With Cloudability Financial Planning , you can

- Create plans containing generated forecasts based upon analysis of your prior spending patterns and adjust by adding new spend items and changes.
- Share plans with others to collect and consolidate their inputs.
- Track changes and run the approval workflow.
- Analyze plans using the built-in analytics.
- Monitor actual spending performance relative to plan and drill in to understand variance drivers.
- Set up plan alerts to proactively notify users when spending has exceeded or is projected to exceed thresholds that you set.
