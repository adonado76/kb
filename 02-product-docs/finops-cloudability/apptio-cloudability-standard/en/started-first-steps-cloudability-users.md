# First steps for Cloudability users

## Overview

IBM Cloudability optimizes cloud resources and translates bills and tags into insights to provide real-time clarity and accountability for consumption. Ask your administrator to create the relevant Cloudability roles for you. As a first time user, here are the steps to configure your Cloudability that will set you up for success.

## Configuration Steps

1. Customize Cloudability Cloudability gives you the ability to create custom dashboards and reports which provide insights into your cloud usage and spend. To access the Reports dashboard, navigate to Home > Reports.
1. Insights Get insights to containers, spend-based commitments, anomaly detection, cloud sustainability reporting, etc. Learn about Containers Learn about Spend-based Commitments Learn about Anomaly Detection Learn about Cloud Sustainability Reporting
1. True Cost Explorer True Cost Explorer helps you to understand the structuring of cloud billing files. It offers a visually intuitive way to explore your billing data and answer questions about cost drivers. Learn about True Cost Explorer
1. Tag Explorer Tag Explorer lets you view the tags your organization is currently using. Learn about Tag Explorer
