# Find your way around Cloudability

## Cloudability Toolbar

Allows quick access to manage profile, access, preferences, and much more.

## Current View

Allows you to select app-wide filters for different users on your Cloudability account, to narrow Cloudability's data.

## Toolbar

Allows access to various help pages and support access, exported reports' status tracking, various application level settings and managing your profile and access. These options are presented based on your role

| Icon | Description | Utilities |
| --- | --- | --- |
|  | Help | You can: Search the help center Send feedback Submit a support ticket Access Cloudability API documentation Access the support desk See what's new in Cloudability |
|  | Report exports | You can: View the status of reports you have exported Re-download a report you have previously exported |
|  | Settings | You can: Manage automated data management tasks and track status Configure fiscal calendars Switch to various environments or Apptio products Manage access administration Configure multiple currencies View anomalies, configure anomaly alerts Configure metrics-agent host View subscribed reports Manage user preferences Manage data source permissions and configurations Manage your profile, preferences like daily email preferences, organization settings, currency and API key Manage budget notification preferences |
|  | Profile settings | You can: Manage your profile Impersonate another user Manage user accounts and access administration Administrators may want to do this to check user's permissions or to troubleshoot access issues. Logout |
|  | All products | Navigate to another Apptio/IBM product or another environment |

## Cloudability Navigation

Left menu or navigation in Cloudability allows you to access various capabilities offered by the product. The menu presented is based on your role.

| Icon | Description | Utilities |
| --- | --- | --- |
|  | Maximize / Minimize | Maximize / minimize navigation |
| Home |  | Tools to view/manage your dashboard and reports My Dashboard Reports |
| Insights |  | Tools for exploring your data visually, such as: TrueCost Explorer Tag Explorer Containers Resource Inventory Anomaly Detection Scorecards and more. Learn more about Insights |
| Optimize |  | Tools for optimizing your cloud resources and providing visibility into your opportunities and savings. These include: Commitment Portfolio Commitment Recs Commitment Manager Rightsizing Rightsizing ROI Turbonomic Learn more about Optimize |
| Plan |  | Tools for budgets and forecasting. See how spend will trend and set budgets to receive email alerts. Current Month Forecast Budgets Workload Planning Plans Learn more about Plan |
| Organize |  | Tools for organizing tags, accounts into groups, cloud infrastructure, as well as filtering your data. Cost Sharing Tags and Labels Account Groups Business Mappings Views Plan Dimensions Data Reprocess Learn more about Organize |
| Settings |  | Tools for managing the integration of cloud vendor data into Cloudability, user management and managing various preferences. Vendor Credentials Workload Planning Preferences Datalink Users and Groups Commitment Preferences Rightsizing Policies Rightsizing Preferences Learn more about Settings |
|  | Logout | Logout of the application |
