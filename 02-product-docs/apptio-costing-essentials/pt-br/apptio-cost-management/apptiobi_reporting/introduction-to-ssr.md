# Apptio BI visão geral dos relatórios

Apptio BI permite que os usuários criem relatórios que podem conter visualizações personalizadas dos dados financeiros de TI mais importantes para eles. Apptio BI os relatórios podem conter várias visualizações configuráveis. Os relatórios podem ser privados ou exibidos publicamente.

Você pode usar os relatórios e visualizações do site Apptio BI para fazer o seguinte:

- Crie um relatório com várias visualizações para apoiar uma história ou um processo de revisão, para responder a uma pergunta ou para uma reunião
- Compartilhe seus relatórios com as partes interessadas
- Revisar relatórios compartilhados por outras pessoas em uma cadência específica usando assinaturas

## Definição de visualização

A visualização de dados é a representação gráfica de informações e dados. Ao usar elementos visuais como tabelas, gráficos e quadros, a ferramenta de relatórios oferece uma maneira fácil de ver e entender tendências, exceções e padrões em seus dados financeiros de TI. A capacidade de analisar grandes quantidades de informações possibilita a tomada de decisões baseadas em dados.

As visualizações são os blocos de construção que compõem os relatórios. Um único relatório pode conter qualquer número de visualizações (gráficos de barras, gráficos de linhas, tabelas, etc.) que consultam individualmente os dados robustos de TI subjacentes ao conjunto de produtos da Apptio. Apptio recomenda que você limite as visualizações a 20 por relatório.
