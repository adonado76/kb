# Notas de lançamento de 2025

## Costing Essentials melhorias (modelos v200 ) - 15 de agosto de 2025

- Adicionados novos campos nos relatórios “Relatório [de mapeamento](../workbench/labor-mapping.html) de mão de obra ausente” e “Relatório [de mapeamento de fornecedores ausentes](../workbench/vendor-mapping.html) ”.
- Adicionados novos campos no Relatório do Workbench [de Mapeamento da Organização](../workbench/organization-mapping.html "Oferece a capacidade de realizar o enriquecimento de dados para suas organizações de TI (unidades de negócios).").
- A fórmula de contagem de linhas foi atualizada para TS Infrastructure e TS Platform.
- Adicionada a tag Aplicação aos relatórios [de TCO da nuvem pública](../../cost-transparency/reports/pub-cloud-tco-overview.html).
- Adicionado novo campo “Código” nas tabelas de dados mestre de integração de planejamento.
- Adicionadas métricas de depreciação ao Modelo de Serviços de Tecnologia.

Removemos a opção “Global” de [todos os filtros](https://www.ibm.com/docs/en/apptio-commercial/costing-essentials/saas?topic=reports-labor-review "(Abre em uma nova guia ou janela)") para aproveitar melhor as Visualizações Salvas, removemos alguns [relatórios](https://www.ibm.com/docs/en/apptio-commercial/costing-essentials/saas?topic=started-end-user-devices-reports "(Abre em uma nova guia ou janela)") avançados de Dispositivos do Usuário Final e adicionamos a estratégia de alocação de pessoal FTE.

- Suporte ao idioma japonês para Public Cloud TCO.
- Public Cloud TCO adicionado ao projeto Costing Essentials de referência

## Public Cloud Custo total de propriedade (TCO) para Costing Essentials ( v200 ) - 28 de fevereiro de 2025

A versão Public Cloud TCO para Costing Standard ( v120 ) e Costing Essentials ( v200 ) oferece vários benefícios para os usuários principais, principalmente equipes financeiras de TI. Esta versão oferece uma visão financeira simples e transparente dos custos mensais da nuvem pública, ajudando a evitar surpresas desagradáveis na conta. Também permite que as equipes promovam a responsabilidade e garantam a eficiência ideal dos serviços de nuvem pública, minimizando o desperdício.

A nova versão inclui a instalação de novos componentes, como TCOPublic Cloud em v120 e TCOPublic Cloud e Public Cloud TCO Reporting em v200. Os usuários podem se conectar aos dados na nuvem e utilizar recursos como Nome da conta, Unidade de negócios e Tabela de mapeamento da torre na nuvem para personalizar relatórios e obter insights sobre as práticas de nuvem de sua organização. Com este lançamento, os usuários podem avaliar se sua organização está aplicando as melhores práticas de nuvem, incluindo cobertura RI e taxas de utilização, para otimizar seus serviços em nuvem.

## Costing Essentials ( v200 ) - 28 de fevereiro de 2025

Os Relatórios de Visão Geral do Modelo são um novo recurso projetado para fornecer aos usuários uma visão abrangente dos fluxos de custos dentro do modelo. Esta versão inclui relatórios sobre mão de obra, fornecedores, soluções, consumidores e modelo de custos, e está disponível para todos os usuários com acesso a relatórios. Os relatórios foram concebidos para aumentar a transparência e a clareza para os usuários, fornecendo uma visão clara e detalhada dos fluxos de custos dentro do modelo. A Visualização do Modelo de Trabalho está incluída no componente Relatórios de Custos e Trabalho, enquanto a Visualização do Modelo de Custos se encontra na coleção Visualização do Modelo. Os relatórios sobre fornecedores, soluções e consumidores estão localizados em suas respectivas coleções.

Os usuários podem clicar nos campos para selecionar e ver como o custo está fluindo entre diferentes objetos, proporcionando uma maneira clara e intuitiva de navegar pelos relatórios. Os relatórios estão acessíveis a qualquer pessoa com acesso a relatórios, sem necessidade de TBM Studio acesso. Os principais usuários dos Relatórios de Visão Geral do Modelo são o Administrador e o Usuário Avançado, para refinar as alocações de custos, tomar decisões informadas e otimizar seus processos de gerenciamento de custos.

## Costing Essentials - 17 de janeiro de 2025

Esta versão oferece as seguintes melhorias para aprimorar a experiência do usuário, o alcance global e a funcionalidade dos recursos.

- Suporte ao idioma japonês : Adicionado suporte ao idioma japonês em dispositivos de usuários finais e relatórios de benchmarking para alcance global e melhoria da experiência do usuário para nossos clientes que falam japonês.
- Recurso de upload de tabelas : Um novo relatório para o recurso de upload de tabelas foi adicionado ao Workbench, oferecendo aos usuários um local centralizado para gerenciar e fazer upload de dados.

  ![Imagem](../../../../../03-media/apptio-costing-essentials/resources/images/acm_images/ce-tu1.jpg)

- Disponibilidade geral do benchmarking : O benchmarking foi promovido para disponibilidade geral (GA), marcando um marco significativo no desenvolvimento desse recurso.
- Atualizações do projeto de referência : Dispositivos de usuário final e benchmarking foram adicionados ao projeto de referência, oferecendo uma solução abrangente e integrada para nossos clientes.
