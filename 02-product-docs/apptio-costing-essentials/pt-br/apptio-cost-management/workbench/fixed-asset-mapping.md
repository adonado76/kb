# Mapeamento de ativos fixos

## Enriquecimento de dados de ativos fixos

Use esse ET para atualizar seus metadados de ativos fixos (se necessário):

- Categoria
- Descrição
- Centro de custo
- Local
- Data de aquisição
- Data de entrada em serviço
- Valor da depreciação

![mapeamento de dados de ativos fixos](../../../../../03-media/apptio-costing-essentials/resources/images/studio_images/fam-1_1043x629.png)

## Mapeamento de ativos fixos

Use esse ET para mapear seus IDs de ativos fixos para :

- Tipo de solução
- Categoria de solução
- ID da oferta
- Ponderação
