# Mapeamento financeiro

## Plano de contas

Os usuários podem mapear seu Plano de Contas para as seguintes dimensões, para ajudar na categorização de despesas e na análise de relatórios:

- Pool de custos
- Subgrupo de custos
- Endereçável
- Tipo de Despesas

![imagem](../../../../../03-media/apptio-costing-essentials/resources/images/studio_images/ce-fm-1_931x555.png)

## Razão

Os usuários podem mapear as transações do Razão para as dimensões a seguir, a fim de melhorar a precisão da alocação e os insights dos relatórios:

- ID do Fornecedor
- ID de Projeto
- ID da oferta
- ID da organização

![imagem](../../../../../03-media/apptio-costing-essentials/resources/images/studio_images/ce-fm-2_955x585.png)

## Outros mapeamentos

A tabela Outros mapeamentos permite mapear as despesas nos centros de custo designados diretamente para os tipos de solução e categorias de solução, juntamente com uma ponderação de alocação.

- Tipo de solução e categoria de solução
- Ponderação
- Talvez uma alocação de 100% (insira "1") ou dividida em vários tipos/categorias de solução
- Todas as despesas com os centros de custo de origem serão alocadas aos tipos/categorias de solução de destino, com base na(s) ponderação(ões) definida(s)
- O custo recebido por cada tipo/categoria de serviço, por sua vez, será distribuído igualmente em suas ofertas de serviços *para crianças*

![imagem](../../../../../03-media/apptio-costing-essentials/resources/images/studio_images/ce-fm-3_938x585.png)

## Contas a pagar

Fornece a capacidade de mapear as transações no subrazão de Contas a Pagar (AP) (ou seja, faturas) para:

- ID do Fornecedor
- ID de Projeto
- Número de OC

Esse mapeamento permite o relacionamento de pedidos de compra de fornecedores com faturas de contas a pagar.

![imagem](../../../../../03-media/apptio-costing-essentials/resources/images/studio_images/ce-fm-4.png)

## Ordens de Compra

Essa tabela oferece ao usuário a capacidade de mapear os pedidos de compra para:

- Centros de custo
- ID do Fornecedor
- ID de Projeto
- Número do contrato

Esse mapeamento permite o relacionamento de contratos de fornecedores com suas respectivas POs.

![imagem](../../../../../03-media/apptio-costing-essentials/resources/images/studio_images/ce-fm-5_998x592.png)
