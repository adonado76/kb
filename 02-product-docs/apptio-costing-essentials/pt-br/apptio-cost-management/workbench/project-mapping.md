# Mapeamento de projetos

## Enriquecimento de dados do projeto

Fornece a capacidade de completar o enriquecimento de dados para seus projetos de TI e permite uma análise mais rica de seus gastos em relação ao seu portfólio de projetos. Inclui metadados como :

- Portfólio de projetos
- Iniciativa empresarial
- Unidade de negócios do patrocinador e patrocinador do negócio
- Tipo de despesas (CTB, GTB, RTB)
- Centro de custo
- Gestor de Projeto
- Orçamento aprovado do projeto
