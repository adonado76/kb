# Resumo das fontes de dados necessárias para o Costing Essentials Foundation

Para configurar o Costing Essentials Foundation, você precisará dos seguintes conjuntos de dados:

- Número real de funcionários por centro de custo
- Número real de funcionários por centro de custo
- Orçamento ( OpEx e CapEx )
- Plano de contas
- Fatura na nuvem
- Detalhes do contrato
- Livro Razão ( OpEx e CapEx )
- Lista de unidades de negócios
- Lista de centros de custo
- Lista de ativos fixos
- Lista de projetos
- Lista de serviços, aplicativos ou produtos
- Lista de fornecedores
- Número de funcionários planejados por centro de custo
