# Upload de tabelas

O componente Table Upload permite que os usuários finais com as permissões apropriadas carreguem diretamente os dados de planilhas em tabelas sem a necessidade de acesso ao site TBM Studio . O relatório Uploads de tabela consiste em duas guias: Upload de tabela e Acesso ao upload de tabela.

Guia Uploads de tabelas

A guia Table Uploads ajudará os usuários a fazer upload de dados diretamente para o Consumption/General Ledger, Budget e Labor sem a necessidade de acesso ao site TBM Studio .

Navegue até Workbench > Uploads de tabela > Upload de tabela, conforme mostrado:

![imagem](../../../../../03-media/apptio-costing-essentials/resources/images/studio_images/be-tu1.jpg)

O componente de upload de tabela oferece duas opções de configuração: [Configuração simples](../../studio/reports/table-report-upload-component.html) e [Configuração avançada](../../studio/reports/table-report-upload-component.html). Neste exemplo, ele está equipado com Advanced Configuration, permitindo a correspondência de usuários na guia Table Upload Access.

![imagem](../../../../../03-media/apptio-costing-essentials/resources/images/studio_images/be-tu2.jpg)

Guia Acesso ao upload de tabelas

A guia Table Uploads Access ajudará os administradores a gerenciar o acesso a quem pode fazer o upload direto de dados para o Consumption/General Ledger, Budget e Labor sem a necessidade de acesso a TBM Studio .

Navegue até Workbench > Uploads de tabela > Upload de tabela > Acesso ao upload de tabela, conforme mostrado:

![imagem](../../../../../03-media/apptio-costing-essentials/resources/images/studio_images/be-tu3.jpg)

Conceda acesso de upload de tabela especificando o nome da tabela e os detalhes do usuário correspondente para permitir que eles façam upload de dados na tabela designada, conforme mostrado abaixo.

![imagem](../../../../../03-media/apptio-costing-essentials/resources/images/studio_images/be-tu4.jpg)

![imagem](../../../../../03-media/apptio-costing-essentials/resources/images/studio_images/be-tu5.jpg)

*\*\* A visibilidade da guia pode ser personalizada com base nos requisitos do cliente para controlar quais funções de usuário têm acesso à guia "Table Upload Access" no relatório Table Upload.*

Para obter mais informações, consulte [Tabela Upload](../../studio/reports/table-report-upload-component.html).
