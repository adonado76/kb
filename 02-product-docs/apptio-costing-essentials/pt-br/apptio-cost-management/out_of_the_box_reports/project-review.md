# Revisão de projetos

Essa coleção oferece uma visão única do projeto OpEx e CapEx,, discriminada por mão de obra, fornecedores e área de despesas de cada um. Você pode visualizar o investimento e as despesas do projeto juntamente com o status, a prioridade e o orçamento. O investimento em projetos está vinculado aos aplicativos e às unidades de negócios que eles suportam:

Importante: o conteúdo desta seção está em andamento.

## Casos de uso

Este relatório resolve os seguintes casos de uso:

- Analisar a iniciativa comercial do projeto e o impacto que ela teve nos negócios
- Revisar a lista, os detalhes, o status, o desempenho e as metas de todos os projetos

## Personagens

- Todos os usuários

## Perguntas respondidas

O relatório responde às seguintes perguntas:

- Qual é a tendência em relação ao gasto orçamentário?
- Quanto estamos gastando em comparação com o total gasto?
- Qual a porcentagem de projetos iniciados no prazo e concluídos dentro do orçamento?
- Qual é o nosso gasto com projetos no acumulado do ano com base na iniciativa de negócios?
- Quanto estamos gastando em todos os projetos em comparação com o orçamento aprovado?
- Como os custos foram comparados ao orçamento aprovado?

## Visualização

![Imagem](../../../../../03-media/apptio-costing-essentials/resources/images/acm_images/project-report.png)

| Elemento-chave | Descrição |
| --- | --- |
| (1) Coleta de relatórios | Esta coleção de relatórios fornece os seguintes detalhes do projeto:  - Revisão do projeto (visualização padrão) - Lista de projetos |
| (2) KPIs | Os KPIs fornecem uma visão de alto nível dos seguintes gastos do projeto:  Project Spend as % Total Spend YTD Este KPI mostra a porcentagem do total de gastos do projeto no ano atual, em comparação com a porcentagem do ano anterior.  - Projetos CapEx: Esse KPI mostra o gasto do seu projeto CapEx no ano atual, em comparação com o gasto do projeto CapEx YTD no ano anterior.  - Projetos OpEx: Esse KPI mostra o gasto do seu projeto OpEx no ano atual, em comparação com o gasto do projeto OpEx YTD no ano anterior.  - Project Change % YTD: Este KPI mostra a porcentagem de mudança de projeto YTD para o ano atual, em comparação com a porcentagem de execução do projeto para o ano anterior. |
| (3) Despesas do projeto (período atual) | Analise o seguinte:  - gastos do projeto, detalhes do projeto e a tendência para o período atual. |
| (4) Todas as despesas do projeto | Você pode fazer o seguinte:  - Revisar todos os detalhes do projeto  - Compreender os detalhes por trás de todos os projetos, inclusive o orçamento do projeto.  - Analisar todos os projetos e iniciativas de negócios do projeto  - Revisão do orçamento aprovado para o orçamento remanescente  - Analise a tendência de todos os projetos por iniciativa de negócios, portfólio de projetos, patrocinador de negócios, tipo de despesas e gerente de projetos. |
