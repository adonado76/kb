# Resumen de las fuentes de datos necesarias para Costing Essentials Foundation

Para configurar Costing Essentials Foundation, necesitará los siguientes conjuntos de datos:

- Efectivos reales por centro de coste
- Efectivos reales por centro de coste
- Presupuesto ( OpEx y CapEx )
- Plan contable
- Factura en la nube
- Información del contrato
- Libro Mayor ( OpEx y CapEx )
- Lista de unidades de negocio
- Lista de centros de coste
- Lista de activos fijos
- Lista de proyectos
- Lista de servicios, aplicaciones o productos
- Lista de proveedores
- Plantilla prevista por centro de costes
