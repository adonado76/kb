# Apptio BI reports overview

Apptio BI allows users to create reports that can contain personalized visualizations of
the IT financial data they care most about. Apptio BI reports can contain multiple configurable
visualizations. Reports can be private or publicly viewable.

You can use Apptio BI reports and visualizations to do the following:

- Create a report with multiple visualizations to support a story or review process, to answer a
  question, or for a meeting
- Share your reports with interested stakeholders
- Review reports shared by others on a specific cadence using subscriptions

## Definition of visualization

Data visualization is the graphical representation of information and data. By using visual
elements like charts, graphs, and tables, the reporting tool provides an easy way for you to see and
understand trends, outliers, and patterns in your IT financial data. The ability to analyze large
amounts of information empowers data-driven decisions.

Visualizations are the building blocks that make up reports. A single report can contain any
number of visualizations (bar charts, line charts, tables, etc.) that individually query the robust
IT data underlying Apptio's product suite. Apptio recommends that you limit visualizations to 20 per
report.
