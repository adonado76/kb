# Technology Services Review

This report reviews your cloud services and allows organizations to visualize what
apps and/or services (and service types) are consuming the most spend.

## Visualization

## Infra and Cloud

This report displays the infra and cloud spends for the following:

- Yearly trend charts and summary of On Prem and Public Cloud
  deployments
- KPIs for public cloud/on prem Infrastructure and Platforms

![Image](../../../../../03-media/apptio-costing-essentials/resources/images/acm_images/infra-cloud.png)

## Cloud Summary

This report displays the cloud summary spends for the following:

- Cloud spend and trend charts
- KPIs for OpEx YTD spend for IaaS & PaaS. cloud compute/storage/network
- Cost by Category details

![Image](../../../../../03-media/apptio-costing-essentials/resources/images/acm_images/cloud-summary.png)
