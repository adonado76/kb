# End User Devices

End User Devices provides insights for IT Asset manager/Desktop Support Manager to make data driven decisions around fleet optimization and consolidation to help decide overall fleet size and associated costs. It will also assist with the planning and forecasting of estimated refresh costs for end of life and non-compliant devices.
