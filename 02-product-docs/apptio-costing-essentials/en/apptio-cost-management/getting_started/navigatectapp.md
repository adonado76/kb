# A Guide to IBM Apptio Costing’s Navigation

This topic provides an overview of how the IBM Apptio Costing navigation is organized,
based on the current TBM Report Studio. This navigation will be different if you’re using the
New Report Studio.

Note: The menu items shown depend on your user permissions and configuration.

1. **Home Page**  - When accessing IBM Apptio Costing, the home page gives access to the
   various report collections available within the IBM Apptio Costing application. The reports
   are organized by areas of interest, revealing financial intelligence insights for key target
   personas.

   ![dashboard](../../../../../03-media/apptio-costing-essentials/resources/images/studio_images/acm-report.jpg)

   Click a heading to open the high-level summary
   report for a report collection.
2. **Report Navigation Buttons** - These navigation buttons allow you to move quickly
   through reports. The Back icon returns you to the previous page, while the Home icon takes
   you directly to your main landing page.

   ![](../../../../../03-media/apptio-costing-essentials/resources/images/ct_images/rep-nav.png)

## Navigation Sidebar Menu

The left sidebar menu serves as your primary navigation hub. It groups your report
collections and provides access to My Landing Page, Recents, and Favorites, enabling you to
efficiently locate and return to the information that matters most.

![](../../../../../03-media/apptio-costing-essentials/resources/images/ct_images/left-nav-bar.png)

**My Landing Page** - My Landing Page provides you with a personalized starting point in
the application. Select your preferred Report and nominate it as your preferred landing
page, this allows you to quickly resume your work without navigating through multiple menus.

![](../../../../../03-media/apptio-costing-essentials/resources/images/ct_images/config-lp.png)

**Recents** - Recents displays a list of the items you have opened most recently. This
view enables you to easily return to your latest, most accessed reports without needing to
search for them again.

**Favorites** - Favorites provides a centralized location for the items you have marked
as important. By adding reports to your favorites, you can access them quickly and
consistently from a single location.

![](../../../../../03-media/apptio-costing-essentials/resources/images/ct_images/config-fav.png)

## Top Navigation Menu

The top navigation bar provides quick access to key settings, navigation tools, help center,
and connectivity to other IBM Apptio solutions.

![](../../../../../03-media/apptio-costing-essentials/resources/images/ct_images/top-nvg-bar.png)

1. **IBM Apptio Logo** - When the navigation sidebar is collapsed, the IBM Apptio Logo
   gets revealed. The IBM Apptio Logo allows you to switch between IBM Apptio applications
   available in your organization. IBM Apptio Costing supports common login and user
   administration through Frontdoor.
2. **Chevron Icon** - Expand or collapse the left navigation pane.
3. **Report Search** - Use the looking glass icon to quickly search for the report you’re
   looking for.
4. **Date Range** - IBM Apptio Costing is fully time-aware. Select and set your Date Range
   when navigating your reports.
5. **Environment** - IBM Apptio Costing provides you with 3 environments: Production,
   Staging and Development. Within Development, each person will have their own workspace. When
   navigating ensure you’ve selected the correct environment. Note: Permissions can be set to
   disallow access to certain environments, such as Development.
6. **Release** - IBM Apptio Costing provides you with release management optionality.
   Where most of the work and releases would happen in the Trunk, work can also take place in a
   Branch, which can get merged back into the Trunk. Hotfix is a 3rd option within release
   management.
7. **Comments and Collaboration** - The Comments panel allows you to collaborate directly
   within a report by adding context, asking questions, or sharing feedback with other users.
   You can leave public comments visible to all viewers or private comments for specific
   individuals or teams. You can also reply to existing comments and use mentions to notify
   users and bring them into the conversation. This helps keep discussion tied to the data,
   improving clarity, traceability, and decision-making.
8. **Help** - Open the Help Center, send product feedback, view release notes, or access
   the IBM Community.
9. **Settings** - The Settings section provides administrators with access to key
   configuration options that control how the application behaves for your organization. From
   this menu, you can manage system preferences, adjust project settings, configure
   multi-currency, and access tools such as Data Advisor, Reference Applications, and Row-Level
   Security. These options allow you to tailor the environment to your operational, security,
   and reporting requirements, ensuring a consistent and governed user experience. The About
   page provides detailed information on your current client and server versions, helping you
   verify your environment configuration and ensure compatibility with product updates and
   features.
10. **Profile Settings** - Manage your user profile; Admins can impersonate users.
11. **Application Menu** - Switch between IBM Apptio applications available in your
    organization. IBM Apptio Costing supports common login and user administration through
    Frontdoor.
12. **Refresh**  - Select the refresh frequency. By default, set to 3 seconds but can be
    switched to manual as needed.
13. **Saved Views and Reset** - This menu allows you to manage how a report is displayed
    and quickly return to preferred settings. You can save your current configuration using Save
    View or Save As, preserving filters, slicers, and layout choices for future use. If you need
    to start fresh, you can reset all slicers, reset global filters, or restore the report to
    its original default state. These options help you personalize your analysis while
    maintaining an easy path back to the baseline report.
14. **Exporting and Sharing Reports** - These options allow you to export and distribute
    report insights in the format that best suits your needs. You can download the report as an
    Excel or PDF file for offline analysis or recordkeeping. The Send Email option lets you
    share the current report directly with other users, while Email Subscriptions enables you to
    schedule automated report deliveries at regular intervals. Together, these features support
    seamless collaboration and timely communication across your organization.

**Multi-Currency**  - If multi-currency is enabled, use the currency menu to change your
view to a different currency.
