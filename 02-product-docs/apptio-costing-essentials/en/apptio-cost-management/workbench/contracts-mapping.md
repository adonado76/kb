# Contracts Mapping

## Contracts Data Enrichment

Provides ability to
complete data enrichment for your IT Vendor Contracts. You can specify such metadata as:

- Contract Title & Description
- Contract Owner
- Contract Link – insert a link to your contracts system
- Cost Center
- Start & End Dates
- Renewal Plans
- Terms
- Status
- Annual Committed Spend & Contract Amount

![image](../../../../../03-media/apptio-costing-essentials/resources/images/studio_images/cm-1_969x563.png)

## Contracts Mappings

Provides ability to define the allocations of Contracts to Solution , at the level of Service
offering. Users can determine the mapping of Contracts to:

- Solution Type
- Solution Category
- Offering ID
- Project ID

![image](../../../../../03-media/apptio-costing-essentials/resources/images/studio_images/cm-2_995x582.png)
