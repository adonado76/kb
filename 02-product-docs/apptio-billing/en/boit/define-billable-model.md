# Define Billable Units model (metric) allocations

If the Billing Standard Units component is installed, you must define the allocations from
Consumption Feed to Business Services, and from the Business Services table to the Business Unit
Allocation Master Data table. Use the settings shown in the following images. Be sure to select the
Billable Units metric.

![img](../../../../03-media/apptio-billing/resources/images/boit_images/boit_define-billable-model1.jpg)

![img](../../../../03-media/apptio-billing/resources/images/boit_images/boit_define-billable-model2.jpg)

![img](../../../../03-media/apptio-billing/resources/images/boit_images/boit_define-billable-model3.jpg)

![img](../../../../03-media/apptio-billing/resources/images/boit_images/boit_define-billable-model4.jpg)

To establish the data relationship, map the Business Unit ID column in the All Business Services
table to the Ident column in the Business Unit Allocation Master Data table as shown below.

![img](../../../../03-media/apptio-billing/resources/images/boit_images/boit_define-billable-model5.jpg)
