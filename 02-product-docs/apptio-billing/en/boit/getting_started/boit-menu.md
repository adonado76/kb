# Find your way around

## Billing Standard toolbar

![img](../../../../../03-media/apptio-billing/resources/images/boit_images/boit-menu-1.png)

**Toolbar**

| Type | Description | Action |
| --- | --- | --- |
| img | Comment | You can collaborate with other users, set notifications and leave notes for yourself. |
| img | Help | You can:  - search the Help Center - send feedback |
| img | Application settings | You can:  - go to application administration - view details of the application, such as the version and environment |
| img | Profile settings | You can:  - manage your profile - impersonate another user  Administrators may want to do this to check a user's permissions or to troubleshoot problems.  - log out - switch to Classical Navigation |

## Billing Standard navigation

![img](../../../../../03-media/apptio-billing/resources/images/boit_images/boit-menu-2.jpg)

| Type | Description | Action |
| --- | --- | --- |
| img | All products | Navigate to another Apptio product |
| img img | Maximize / Minimize | Maximize / minimize navigation |
