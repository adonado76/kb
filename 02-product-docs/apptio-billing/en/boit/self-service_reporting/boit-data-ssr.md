# Billing Standard data available in Apptio BI

Apptio BI will report against your Billing Standard production environment.

The following minimum Billing Standard version requirements apply:

- Reporting for Billing Standard out-of-the-box data: Costing Standard version 12.8.3 or later
- Reporting for custom fields in out-of-the-box data, custom modeled data sets, custom modeled
  metrics: version 12.9.7
- Multi-currency support: Costing Standard version 12.9.1 or later
- Row level security support: version 12.9.1. or later
