# Lista de verificación administrativa de facturación

Esta lista de verificación resume las tareas recurrentes para cada período de facturación. Se puede adaptar para convertirlo en un manual operativo más detallado.

**Antes del cierre del período**

- Confirme el calendario de facturación y las fechas límite para el próximo período.
- Validar que los sistemas fuente estén listos para proporcionar datos de consumo a tiempo.
- Comprueba que las tablas de tarifas estén actualizadas para el próximo período.

**Después del cierre del período**

- Actualizar los datos de costes para el periodo.
- Cargar o actualizar los datos de consumo que requiere el departamento de facturación.
- Ejecute los modelos de facturación en el entorno adecuado (primero en el entorno de pruebas y, después, en el entorno de producción tras la promoción).

**Validación**

- Ejecutar informes de control de calidad estructurales, de volumen y de tasa.
- Conciliar los totales de facturación con los costes y, si procede, con el plan o el presupuesto.
- Resuelva cualquier problema relacionado con los datos o la configuración y vuelva a ejecutar los modelos si es necesario.

**Liberación de factura**

- Generar facturas e informes detallados para el período.
- Confirme que los informes se generan correctamente y que la seguridad funciona según lo previsto.
- Poner los informes a disposición de los destinatarios adecuados y notificárselo.

**Integración de diarios y finanzas**

- Preparar y validar las exportaciones del diario.
- Entregar o cargar diarios al sistema financiero.
- Confirme la publicación y registre cualquier ID de referencia.

**Servicio de limpieza**

- Archivo de facturas emitidas durante el periodo.
- Registre cualquier problema conocido, ajuste manual o tratamiento puntual.
- Actualice la documentación o las notas que serán relevantes en el próximo ciclo de planificación anual.
