# Generación y distribución de facturas

Una vez que los resultados de facturación superan la validación, el siguiente paso es generar vistas de facturas y distribuirlas.

1. **Preparar vistas de facturas**
   - Ejecute informes de estilo de factura que muestren:
     - Consumidor.
     - Oferta o servicio.
     - Unidades, tarifa y cargo total.
     - Periodo y atributos básicos como el entorno o la región, si es necesario.
2. **Ajustar el nivel de presentación**
   - Decida qué nivel de detalle necesita cada público:
     - Resumen de alto nivel por servicio para altos directivos.
     - Partidas detalladas para propietarios de servicios o productos.
     - Opiniones específicas para consumidores concretos.
3. **Publicar en la interfaz de usuario**
   - Hacer visibles los informes de facturación pertinentes para:
     - Propietarios y analistas de servicios o productos.
     - Consumidores y partes interesadas, según sea necesario.
4. **Exportar para uso sin conexión**
   - Exportar los informes seleccionados a:
     - Hoja de cálculo o CSV para análisis local.
     - PDF para archivo o distribución, según corresponda.

1. **Comunicar disponibilidad**
   - Notificar a los destinatarios cuando las facturas estén listas:
     - Periodo cubierto.
     - Dónde encontrar los informes.
     - A quién contactar si tiene preguntas.

Para muchas organizaciones, este paso de distribución es también donde surge una primera ronda de preguntas. Un proceso estándar para responder a esas preguntas forma parte del ciclo.
