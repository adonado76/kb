# Facturación Datos estándar disponibles en Apptio BI

Apptio BI informará con respecto a su entorno de producción Billing Standard .

Se aplican los siguientes requisitos mínimos de la versión Billing Standard :

- Informes para los datos de Billing Standard listos para usar: Costing Standard versión 12.8.3 o posterior
- Informes para campos personalizados en datos out-of-the-box, conjuntos de datos modelados personalizados, métricas modeladas personalizadas: versión 12.9.7
- Compatibilidad multidivisa: Costing Standard versión 12.9.1 o posterior
- Soporte de seguridad a nivel de fila: versión 12.9.1. o posterior
