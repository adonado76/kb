# Personalizar o conteúdo dos principais casos de uso

No relatório Fatura detalhada, clique em **Principais casos de uso** no canto superior direito para exibir informações sobre como analisar a fatura. Dois casos de uso padrão são fornecidos como parte do aplicativo. Você pode modificar o texto nos casos de uso padrão e adicionar até três outros casos de uso. Pode haver um total de cinco casos de uso. O conteúdo padrão do caso de uso é armazenado em uma tabela chamada Referência de caso de uso. Mantenha o conteúdo dos casos de uso em uma planilha do Excel, carregue a planilha no aplicativo e anexe a tabela à tabela de casos de uso.

Descrições de campo

A seguir, as descrições dos campos da tabela de referência de casos de uso. A tabela está localizada no grupo no Project Explorer.

- **Título -** O nome do caso de uso.
- **Descrição -** Uma descrição do caso de uso que é exibida ao lado do nome.
- **Etapas -** Etapas sugeridas a serem seguidas para analisar a fatura.
- **ID-** Um número entre 1 e 5 que determina a ordem dos casos de uso.  
  Por exemplo, se você digitar o número 2, o caso de uso será o segundo na janela pop-up Top Detail Use Cases.
- **Relatório -** O nome do(s) relatório(s) em que o conteúdo será usado. No momento, o único valor válido é "Invoice" (Fatura)
