# Configurar padrão de faturamento

Para configurar Billing Standard, você deve ter um projeto de Transparência de custos existente. As tarefas de configuração estão listadas abaixo.

- [Abra o projeto Transparência de custos](https://www.ibm.com/docs/en/apptio-gov/costing-standard/saas?topic=costing-standard-foundation-module-configuration "(Abre em uma nova guia ou janela)")
- [Instalar os componentes do site Billing Standard](boit-components.html)
- [Adicionar dados à tabela Biblioteca de serviços](service-library-table.html)
- [Adicionar dados da unidade de negócios](business-unit-data.html)
- [Adicione os dados de consumo para os cálculos de Preço x Quantidade](add-consumption-data.html "Use os dados de consumo quando quiser cobrar das unidades de negócios com base no consumo de serviços, em vez de alocações de custos baseadas em fatores como o número de funcionários. Se nenhum dado de consumo adicional for conhecido, use uma lista das IDs de serviço, que pode ser fornecida anexando os dados brutos da biblioteca de serviços. Não anexe a tabela da biblioteca de serviços em si; no entanto, é permitido anexar os dados anexados à biblioteca de serviços.")
- [Adicionar serviços e dados orçamentários da unidade de negócios](services-bu-data.html)
- [Definir alocações do modelo de custos](define-cost-model1.html)
- [Definir alocações de modelos de unidades faturáveis](define-billable-model.html)
- [Definir a relação Charge Price x Quantity](define-charge-price.html)
- [Personalize o conteúdo dos principais casos de uso](top-use-cases.html)

Para obter informações sobre o suporte ao navegador, consulte [Apptio application browser support](https://community.apptio.com/viewdocument/supported-browsers-screen-resolut "(Abre em uma nova guia ou janela)").
