# Definir alocações do modelo de unidades faturáveis (métricas)

Se o componente Billing Standard Units estiver instalado, você deverá definir as alocações do Feed de consumo para os Serviços comerciais e da tabela Serviços comerciais para a tabela Dados mestre de alocação de unidades comerciais. Use as configurações mostradas nas imagens a seguir. Certifique-se de selecionar a métrica Unidades faturáveis.

![img](../../../../03-media/apptio-billing/resources/images/boit_images/boit_define-billable-model1.jpg)

![img](../../../../03-media/apptio-billing/resources/images/boit_images/boit_define-billable-model2.jpg)

![img](../../../../03-media/apptio-billing/resources/images/boit_images/boit_define-billable-model3.jpg)

![img](../../../../03-media/apptio-billing/resources/images/boit_images/boit_define-billable-model4.jpg)

Para estabelecer o relacionamento de dados, mapeie a coluna ID da unidade de negócios na tabela All Business Services para a coluna Ident na tabela Business Unit Allocation Master Data, conforme mostrado abaixo.
