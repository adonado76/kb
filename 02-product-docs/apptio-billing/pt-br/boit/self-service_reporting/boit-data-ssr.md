# Dados padrão de faturamento disponíveis em Apptio BI

Apptio BI será reportado em seu ambiente de produção Billing Standard .

Aplicam-se os seguintes requisitos mínimos da versão Billing Standard :

- Relatórios para dados prontos para uso do site Billing Standard : versão Costing Standard 12.8.3 ou posterior
- Relatórios para campos personalizados em dados prontos para uso, conjuntos de dados modelados personalizados, métricas modeladas personalizadas: versão 12.9.7
- Suporte a várias moedas: Costing Standard versão 12.9.1 ou posterior
- Suporte à segurança em nível de linha: versão 12.9.1. ou posterior
