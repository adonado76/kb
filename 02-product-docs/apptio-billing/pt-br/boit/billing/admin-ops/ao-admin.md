# Visão geral

Esta seção fornece padrões concretos para estruturar o faturamento, para que você não precise inventar tudo do zero todas as vezes.

Cada padrão descreve:

- Que problema resolve
- Qual edição é mais adequada
- Como são os dados e a forma do modelo
- Como os relatórios são normalmente utilizados

Quando um padrão requer o Padrão de Faturamento, isso é explicitamente indicado.

## Visão geral do padrão

Você pode pensar nesses padrões como “formas iniciais” que podem ser combinadas:

- Demonstração simples do serviço
- Estorno com diários
- Design centrado no catálogo de serviços
- Design centrado na nuvem
- Design centrado no projeto
- Visão da entidade jurídica e dos preços de transferência

A maioria dos ambientes acaba com uma mistura de dois ou três.
