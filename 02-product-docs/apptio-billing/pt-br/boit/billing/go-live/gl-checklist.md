# Lista de verificação de preparação pré-lançamento

Isso é o mínimo que você deve ter pronto antes mesmo de marcar uma data para o lançamento.

**Preparação de dados e modelos**

- Modelo de cálculo de custos:
  - As alocações são estáveis por pelo menos um ou dois períodos anteriores.
  - Não há questões críticas em aberto que interrompam regularmente as execuções.
- Dados para faturamento:
  - O catálogo de serviços/ofertas é revisado e aprovado.
  - O mestre do consumidor está completo para todas as entidades a serem faturadas.
  - As coberturas de consumo abrangem todos os serviços e consumidores dentro do escopo.
  - As tabelas de taxas ou preços são carregadas para o período de entrada em operação e os períodos seguintes.

**Ambiente e prontidão para lançamento**

- Em desenvolvimento:
  - Todo o trabalho de configuração de faturamento é verificado.
  - Sem experiências pela metade.
- Encenação:
  - Contém uma compilação completa com todas as alterações de faturamento incluídas.
  - Montagem utilizada para o ensaio geral.
- Produção:
  - Sem qualquer conteúdo de teste ou POC que possa confundir os usuários.

**Relatório e acesso à preparação**

- Relatórios:
  - Relatórios de faturamento principais identificados para entrada em operação (não todos os relatórios).
  - Os filtros e avisos do relatório funcionam conforme o esperado.
  - As principais opiniões foram analisadas com as partes interessadas.
- Acesso:
  - Os administradores e analistas podem acessar o projeto/ponto final relevante.
  - Os proprietários de serviços ou produtos podem visualizar os relatórios de que necessitam.
  - O departamento financeiro pode acessar as exportações do diário, quando aplicável.
