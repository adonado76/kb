# Geração e distribuição de contas

Depois que os resultados da faturação forem validados, a próxima etapa é gerar visualizações das faturas e distribuí-las.

1. **Preparar visualizações de contas**
   - Execute relatórios no estilo de fatura que mostram:
     - Consumidor.
     - Oferta ou serviço.
     - Unidades, taxa e custo total.
     - Período e atributos básicos, como ambiente ou região, se necessário.
2. **Ajustar nível de apresentação**
   - Decida o nível de detalhe necessário para cada público:
     - Resumo de alto nível por serviço para líderes seniores.
     - Itens detalhados para proprietários de serviços ou produtos.
     - Visões focadas em consumidores específicos.
3. **Publicar no front-end**
   - Torne os relatórios de faturamento relevantes visíveis para:
     - Proprietários e analistas de serviços ou produtos.
     - Consumidores e partes interessadas, conforme necessário.
4. **Exportar para uso offline**
   - Exportar relatórios selecionados para:
     - Planilha ou CSV para análise local.
     - PDF para arquivo ou distribuição, quando apropriado.

1. **Comunicar disponibilidade**
   - Notificar os destinatários quando as contas estiverem prontas:
     - Período coberto.
     - Onde encontrar relatórios.
     - Quem contatar em caso de dúvidas.

Para muitas organizações, essa etapa de distribuição também é onde surge uma primeira série de perguntas. Um processo padrão para responder a essas perguntas faz parte da execução do ciclo.
