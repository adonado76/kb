# Planejado x Real x Preço

Aplica-se principalmente ao Padrão de Faturamento, mas os conceitos podem ser usados de forma simplificada com o Essenciais de Faturamento, se os dados do plano estiverem disponíveis.

## Dados necessários

Para comparar o planejado com o real e com o preço, você precisa de:

- **Custo real**
  - A partir dos resultados do cálculo de custos.
- **Plano ou orçamento**
  - A partir de ferramentas de planejamento, planilhas ou tabelas mantidas manualmente.
  - Alinhado aos mesmos serviços e consumidores usados no faturamento.
- **Preço**
  - A partir das tabelas de tarifas ou preços.

## Padrões de relatórios

Visualizações úteis:

- Para cada serviço e período:
  - Custo do plano por unidade.
  - Custo real por unidade.
  - Preço por unidade.
  - Colunas de variação:
    - Real vs planejado.
    - Preço versus custo.
- Para cada consumidor e período:
  - Taxas do plano.
  - Encargos reais.
  - Variação e percentagem de variação.

Use essas visualizações para apoiar:

- Revisões anuais e trimestrais das taxas.
- Discussões sobre recuperação insuficiente ou recuperação excessiva.
- Ajustes nas taxas futuras ou no desenho do serviço.
