# Primeiras etapas para usuários do Cloudability

## Visão geral

IBM Cloudability otimiza os recursos da nuvem e converte faturas e tags em insights para oferecer clareza e responsabilidade pelo consumo em tempo real. Peça ao seu administrador para criar as funções relevantes do Cloudability para você. Como usuário iniciante, aqui estão as etapas para configurar o Cloudability que o prepararão para o sucesso.

## Etapas de configuração

1. Personalize Cloudability

   Cloudability oferece a possibilidade de criar painéis e relatórios personalizados que fornecem informações sobre o uso e os gastos com a nuvem. Para acessar o painel Relatórios, navegue até Home > Relatórios.
2. Insights

   Obtenha insights sobre contêineres, compromissos baseados em gastos, detecção de anomalias, relatórios de sustentabilidade da nuvem, etc.

   Saiba mais sobre [contêineres](https://www.ibm.com/docs/en/cloudability-commercial/cloudability-enterprise/saas?topic=insights-container-cost-allocation "(Abre em uma nova guia ou janela)")

   Saiba mais sobre [os compromissos baseados em gastos](../product/analyze-data-for-your-aws-savings-plans.html)

   Saiba mais sobre [a detecção de anomalias](../product/identify-unusual-spending-patterns-with-anomaly-detection.html)

   Saiba mais sobre [relatórios de sustentabilidade na nuvem](../product/cld-sustainability-reporting.html)
3. **Explorador de custos reais**

   O True Cost Explorer ajuda você a entender a estruturação dos arquivos de faturamento da nuvem. Ele oferece uma maneira visualmente intuitiva de explorar seus dados de faturamento e responder a perguntas sobre os geradores de custos.

   Saiba mais sobre [o True Cost Explorer](../product/explore-cost-drivers-with-truecost-explorer.html)
4. **Explorador de tags**

   O Tag Explorer permite que você visualize as tags que sua organização está usando no momento.

   Saiba mais sobre [o Tag Explorer](../product/identify-tagged-and-untagged-spend-with-tag-explorer.html)
