# Conectar o Jira Cloud

Este artigo explica como configurar o Jira Cloud no Cloudability.

Antes de começar

Antes de começar, verifique o seguinte:

- Sua organização utiliza o Jira Cloud (e não o Jira Server).
- Você é um administrador do Cloudability.
- O administrador da sua instância do Jira estará disponível durante o processo de configuração.
- Você tem o nome do site da sua instância do Jira.

Etapas para a integração

Credenciais do Jira para Cloudability

1. No site Cloudability, acesse Configurações > Credenciais do fornecedor > Adicionar fonte de dados > JIRA.
2. O painel “Adicionar conta do JIRA” é exibido.
3. Digite o nome do “ URL ” da Base.
4. Selecione Salvar.

Cloudability para o Jira

1. Faça login na sua instância do Jira Cloud como administrador.
2. Selecione Configurações > Complementos.
3. Selecione Encontrar novos complementos.
4. Pesquise por “ Cloudability ”.
5. Selecione Instalar.
6. Selecione Aceitar e instalar. Aguarde até que a instalação seja concluída.

Como confirmar o sucesso

Verifique o status em Configurações > Credenciais do fornecedor.
