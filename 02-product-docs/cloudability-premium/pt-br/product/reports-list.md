# Lista de relatórios

Ao acessar “Página inicial” => “Relatórios”, os usuários visualizam uma lista de todos os relatórios aos quais têm acesso. A lista de relatórios pode ser ainda mais filtrada para se concentrar apenas nos relatórios de uma determinada categoria:

- Destacados — exibe apenas os relatórios marcados como “Destacados” pelo usuário
- Meus — exibe apenas os relatórios criados pelo usuário atual
- Compartilhado — exibe apenas os relatórios compartilhados com o usuário atual
- Inscrito — exibe apenas os relatórios nos quais o usuário atual está inscrito

![](../../../../03-media/cloudability-premium/images/reports-list.png)

A lista de relatórios no Cloudability é preenchida automaticamente com os relatórios padrão (prontos para uso), criados pelo Cloudability, que atendem aos casos de uso mais comuns e podem ser modificados para personalização, incluindo dimensões e filtros adicionais, além de outras alterações. Todos os relatórios padrão pertencem a “ Cloudability ” e não podem ser modificados diretamente. Em vez disso, os usuários podem criar sua própria versão e incluir as alterações necessárias.

![](../../../../03-media/cloudability-premium/images/ootb-reports.png)
