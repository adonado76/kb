# Cloudability Relatórios

Cloudability Relatórios é uma ferramenta self-service que permite aos usuários acessar seus dados de custo e utilização para responder a perguntas ad hoc, programar relatórios para serem enviados em uma determinada frequência ou exportar os dados brutos para um determinado conjunto de dimensões, métricas e filtros.
