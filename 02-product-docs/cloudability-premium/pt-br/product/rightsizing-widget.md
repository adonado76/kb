# Widget de redimensionamento

Cloudability Os painéis oferecem a opção de visualizar os dados da fonte de dados “Rightsizing”.

Atualmente, o Widget de Rightsizing só pode exibir as economias dos últimos 10 dias relacionadas ao Rightsizing.

O recurso de ajuste de tamanho nos painéis do Cloudability está disponível para os seguintes produtos:

- AWS EC2 (Elastic Compute Cloud)
- AWS EBS (Elastic Block Store)
- AWS RDS (Serviço de Banco de Dados Relacional)
- Computação do Azure
- Azure Disco gerenciado
- Azure SQL Database
