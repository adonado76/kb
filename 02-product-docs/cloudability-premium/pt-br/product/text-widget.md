# Widget de texto

O Widget de Texto pode ser usado para fornecer descrições, comentários ou explicações adicionais ao Painel. Entre os principais recursos estão:

- Opções básicas do editor integrado
- Suporte a Markdown
- Inserção de links para conteúdo externo
