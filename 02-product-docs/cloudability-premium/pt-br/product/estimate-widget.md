# Widget de estimativa

Cloudability Os painéis oferecem a opção de visualizar os dados da fonte de dados “Estimativa”.

Atualmente, o Widget de Estimativa só pode exibir a diferença do KPI entre “Este mês” e “Mês passado” utilizando uma das seguintes métricas de custo:

- Custo (total ponderado)
- Custo (amortizado)
- Custo (ajustado)
- Custo (amortizado ajustado)
- Custo (lista)

As cobranças que ainda não constam no arquivo de faturamento serão estimadas para fins de exibição do KPI do Widget de Estimativa.
