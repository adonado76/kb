# Guia complementar ao programa Spend CUD da GCP

**Instruções para preencher dados para clientes que migraram voluntariamente para o novo Programa Spend CUD antes de 1º de fevereiro de 2026**

Os clientes precisam que recuperemos os dados GCP desde o mês da sua migração. Existem três grupos:

1. **Migração forçada:** Os clientes que foram migrados à força pela GCP durante a primeira semana de fevereiro foram atendidos. Não é necessária nenhuma ação adicional por parte dos nossos clientes/equipe de campo para apoiar este grupo.
2. **Migração voluntária, comunicação do cliente:** Para os clientes que migraram e nos informaram, procedemos à recuperação dos seus dados em seu nome.
3. **Migração voluntária, o cliente não comunicou o conhecimento:** para os clientes que não nos notificaram, registre um caso de suporte com as seguintes informações:

<Seu nome> migrou voluntariamente para o novo programa CUD da GCP e está solicitando uma nova obtenção dos dados do GCP desde a data da migração. Fornecidas as informações solicitadas sobre a conta: nome do cliente, região do Frontdoor, nome do cliente do Frontdoor, nome do FrontdoorEnvironment, contato interno, data de migração, contas de cobrança e se as contas são credenciadas como cobrança detalhada ou padrão.

Observe que as informações da conta não são obrigatórias, mas são altamente recomendadas, pois há um custo significativo para executar essas novas buscas. Se você não sabe se a nova busca foi realizada em seu nome, entre em contato com sua equipe de contas. A data da sua migração pode ser encontrada no console GCP. Veja abaixo:
