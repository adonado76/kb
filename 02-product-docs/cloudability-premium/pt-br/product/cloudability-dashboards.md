# Cloudability Painéis

Cloudability O Dashboards é uma ferramenta simples e self-service que pode ser usada para criar painéis totalmente personalizáveis, utilizando os dados disponíveis em Cloudability. Os usuários podem escolher entre uma variedade de widgets para visualizar as informações e ajustar as opções de visualização para personalizar os painéis de acordo com as necessidades individuais, garantindo que as informações sejam exibidas rapidamente e no contexto certo.

![](../../../../03-media/cloudability-premium/images/cloudability-dashboards.png)

Os painéis podem ser compartilhados com outros membros da equipe para melhorar a colaboração e a visibilidade. As anotações do widget podem garantir um entendimento comum dos custos e gastos com a nuvem, sem a necessidade de ferramentas de colaboração externas. A aplicação de visualizações d Cloudability s em um painel pode ser usada para restringir o escopo de cada painel, a fim de aprofundar os detalhes e visualizar os dados de uma perspectiva mais focada.
