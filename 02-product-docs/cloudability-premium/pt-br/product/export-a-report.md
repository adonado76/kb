# Exportar um relatório

Para extrair dados brutos de um relatório, os usuários podem exportar os dados para um arquivo do tipo “ CSV ” acessando a opção “Exportar” no menu de pontos:

![](../../../../03-media/cloudability-premium/images/export-menu.png)

Ao clicar no item de menu “Exportar”, será iniciada a geração do relatório. A preparação do relatório para download pode levar vários minutos, dependendo do tamanho do conjunto de dados que está sendo exportado. A seguinte janela modal será exibida no canto inferior direito da página para confirmar que o relatório está sendo gerado:

![](../../../../03-media/cloudability-premium/images/export-modal.png)

Os relatórios que estão sendo gerados podem ser encontrados no menu na parte superior da página, ao lado do seletor “Exibir”:

![](../../../../03-media/cloudability-premium/images/generate-report.png)
