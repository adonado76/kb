# Widget de tabela

O Widget de Tabela permite uma visão geral detalhada dos dados e é utilizado para uma análise precisa e aprofundada dos custos. O widget de tabela pode ser criado utilizando os dados de “Custo e Uso” ou de “Utilização”.

Os usuários podem escolher até 15 dimensões e até 8 métricas para criar a tabela.

As tabelas podem ser ordenadas por qualquer uma das dimensões ou métricas selecionadas.

Por fim, para aumentar a clareza do Widget de Tabela, os usuários podem optar por exibir apenas os N valores mais altos ou os N valores mais baixos na tabela. Os valores restantes serão agrupados na categoria “Outros”.

Os widgets de tabela nos painéis do Cloudability somam automaticamente todas as linhas e exibem o valor total na parte inferior do widget.
