# Widget de gráfico

O Widget de Gráfico permite que os usuários do Cloudability visualizem seus dados na forma de um gráfico de linhas, gráfico de área, gráfico de barras ou gráfico de colunas. Esse tipo de widget é mais útil quando o que mais importa são as tendências gerais, os picos ou as diferenças entre os valores no gráfico, e não os valores exatos.

Ao configurar um widget de gráfico, os usuários podem escolher qualquer dimensão da fonte de dados “Custo e uso” ou “Utilização” para o eixo X do gráfico.

Qualquer métrica da fonte de dados “Custo e uso” ou “Utilização” pode ser usada como eixo Y no gráfico.

Os dados podem então ser divididos em séries de dados separadas por meio do recurso “Agrupar por”, que exibirá uma série de dados separada para cada dimensão selecionada.

Nos gráficos de “linha” ou “área”, os usuários podem selecionar a opção “Spline”, que criará uma linha mais suave no gráfico.

Ao preencher o campo “Nome da série”, será adicionado um prefixo a cada dimensão usada como legenda no gráfico.

A opção “Escalar a partir do valor mais baixo” permite decidir se a escala do gráfico deve começar a partir do valor mais baixo no gráfico ou permitir que os usuários definam manualmente o ponto inicial do eixo Y. O valor padrão, quando a opção “Escalar a partir do valor mais baixo” não estiver selecionada, é iniciar a escala em “0”.

Por fim, os usuários podem exibir vários gráficos em um único widget. Clique no botão “Adicionar camada”, na parte inferior da tela de configuração, para criar outro tipo de gráfico. As etapas de configuração para cada camada são as mesmas que para a camada principal.
