# Configure as credenciais IBM Cloud usando o aplicativo de arquitetura implantável (DA)

O **aplicativo Deployable Architecture (DA)**, também conhecido como **IBM Cloudability Enablement**, oferece uma maneira alternativa e simplificada de conectar sua conta IBM Cloud a Cloudability. Esse aplicativo simplifica o processo de configuração e é o **método recomendado** para um credenciamento mais rápido. O aplicativo DA está disponível no [Catálogo Público](https://cloud.ibm.com/catalog/architecture/deploy-arch-ibm-cloudability "(Abre em uma nova guia ou janela)") IBM Cloud. Para saber mais sobre sua arquitetura e como configurá-la e implantá-la, consulte a [documentação do aplicativo Cloudability Enablement DA](https://cloud.ibm.com/docs/track-spend-with-cloudability?topic=track-spend-with-cloudability-overview "(Abre em uma nova guia ou janela)").

Nota:

O uso desse método para credenciar sua conta IBM Cloud em Cloudability é totalmente opcional.

Para obter ajuda e suporte para IBM Cloudability Enablement usando o aplicativo DA, acesse: [Obter ajuda e suporte para IBM Cloudability Enablement](https://cloud.ibm.com/docs/track-spend-with-cloudability?topic=track-spend-with-cloudability-help-and-support "(Abre em uma nova guia ou janela)")
