# Criar ou editar um relatório

Novos relatórios podem ser criados clicando no botão “Novo relatório” na página Lista de relatórios. Os relatórios existentes podem ser editados acessando o Relatório e clicando no botão azul “Editar Relatório”. São necessárias permissões de "Editar" para modificar um relatório existente.

Ao criar um relatório, várias opções de configuração estão disponíveis:

- Nome do relatório - forneça um nome descritivo para um relatório, para que ele possa ser facilmente acessado posteriormente.
- Intervalo de datas - configure o intervalo de datas que será usado para preencher o relatório.
- Categoria - selecione uma categoria à qual um relatório será atribuído. A categoria pode ser usada posteriormente para classificar relatórios e atribuí-los a determinados grupos. Uma nova categoria também pode ser criada clicando no botão “Adicionar categoria”.
- Custo compartilhado - esta caixa de seleção permite que as regras de compartilhamento de custos sejam aplicadas para gerar o relatório.

Dimensões e métricas

Nesta seção, os usuários podem selecionar a fonte de dados para consultar os dados. Atualmente, existem duas fontes de dados disponíveis: dados de custos ou dados de utilização.

Os usuários podem selecionar quais dimensões e métricas de uma determinada fonte de dados visualizar no relatório.

Ao selecionar várias métricas, os números relevantes do KPI são exibidos na parte superior do relatório. Os usuários podem clicar em um dos números de KPI para se concentrar em uma determinada métrica no gráfico do relatório abaixo. Todas as métricas selecionadas são incluídas na exportação de dados brutos.

Ao selecionar várias dimensões, a tabela na parte inferior do relatório será dividida por cada dimensão e uma linha separada será exibida para cada combinação única de valores de dimensão. Essa seleção também é considerada para exportar dados brutos do Relatório.

Em seguida, os usuários podem decidir como classificar os dados na tabela na parte inferior do relatório. Qualquer dimensão ou métrica pode ser usada para classificação em ordem decrescente ou crescente.

Por fim, os usuários podem configurar um conjunto de filtros que serão usados para restringir o conjunto de dados visível aos usuários. Qualquer medida (métrica ou dimensão) pode ser usada para incluir ou excluir dados do relatório.
