# Copy a Report to a Dashboard

Existing Reports can also be saved as new Widgets in existing Dashboards by clicking the "Copy to
Dashboard" button in the dot menu:

![](../../../../03-media/cloudability-premium/images/copy-a-report-to-a-dashboard.png)

Then, users can select an existing Dashboard where a Report will be saved and choose which type
of widget to add to the Dashboard. Users can save the KPI, Chart or Table in a Dashboard.
