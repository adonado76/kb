# Number Widget

The Number Widget can be used in Cloudability Dashboards to see a summary of a single Metric in a
Dashboard. This Widget displays a single value of a Metric (KPI) over a given date range.

Any Metric from the “Cost and Usage” or the “Utilization” data source can be used to display the
Number Widget.

Users can also decide to display a small line chart next to the KPI on the Number Widget, to
understand how the Metric changed over time.
