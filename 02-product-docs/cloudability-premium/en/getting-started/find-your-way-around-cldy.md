# Find your way around Cloudability

## Cloudability Toolbar

Allows quick access to manage profile, access, preferences, and much more.

![](../../../../03-media/cloudability-premium/images/find-your-way-around-cldy-toolbar.png)

## Current View

Allows you to select app-wide filters for different users on your Cloudability account, to narrow Cloudability's data.

## Toolbar

Allows access to various help pages and support access, exported reports' status tracking, various application level settings and managing your profile and access.
These options are presented based on your role

| Icon | Description | Utilities |
| --- | --- | --- |
| help icon | Help | You can:  Search the help center  Send feedback  Submit a support ticket  Access Cloudability API documentation  Access the support desk  See what's new in Cloudability |
| download icon | Report exports | You can:  View the status of reports you have exported  Re-download a report you have previously exported |
| settings icon | Settings | You can:  Manage automated data management tasks and track status  Configure fiscal calendars  Switch to various environments or Apptio products  Manage access administration  Configure multiple currencies  View anomalies, configure anomaly alerts  Configure metrics-agent host  View subscribed reports  Manage user preferences  Manage data source permissions and configurations  Manage your profile, preferences like daily email preferences, organization settings, currency and API key  Manage budget notification preferences |
| user account icon | Profile settings | You can:  Manage your profile  Impersonate another user  Manage user accounts and access administration  Administrators may want to do this to check user's permissions or to troubleshoot access issues.  Logout |
| app menu icon | All products | Navigate to another Apptio/IBM product or another environment |
