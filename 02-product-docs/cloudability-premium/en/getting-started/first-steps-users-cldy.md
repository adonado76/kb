# First steps for Cloudability users

## Overview

IBM Cloudability optimizes cloud resources and translates bills and tags into insights to provide real-time clarity and accountability for
consumption. Ask your administrator to create the relevant Cloudability roles for you. As a first time user, here are the steps
to configure your Cloudability that will set you up for success.

## Configuration Steps

1. Customize Cloudability 

   Cloudability gives you the ability to create
   custom dashboards and reports which provide insights into your cloud usage and spend. To access the
   Reports dashboard, navigate to Home > Reports.
2. Insights 

   Get insights to containers, spend-based commitments, anomaly
   detection, cloud sustainability reporting, etc.

   Learn about [Containers](https://www.ibm.com/docs/en/cloudability-commercial/cloudability-enterprise/saas?topic=insights-container-cost-allocation "(Opens in a new tab or window)")

   Learn about
   [Spend-based Commitments](../product/analyze-data-for-your-aws-savings-plans.html)

   Learn about
   [Anomaly Detection](../product/identify-unusual-spending-patterns-with-anomaly-detection.html)

   Learn about
   [Cloud Sustainability Reporting](../product/cld-sustainability-reporting.html)
3. **True Cost Explorer** 

   True Cost Explorer helps you to understand the structuring of cloud
   billing files. It offers a visually intuitive way to explore your billing data and answer questions
   about cost drivers.

   Learn about
    [True Cost Explorer](../product/explore-cost-drivers-with-truecost-explorer.html)
4. **Tag Explorer**

   Tag Explorer lets you view the tags your organization is currently using.

   Learn about
    [Tag Explorer](../product/identify-tagged-and-untagged-spend-with-tag-explorer.html)
