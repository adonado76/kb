# Widget de números

El widget «Número» se puede utilizar en los paneles de control de Cloudability para ver un resumen de una única métrica en un panel de control. Este widget muestra un único valor de una métrica (KPI) durante un intervalo de fechas determinado.

Cualquier métrica de la fuente de datos «Coste y uso» o «Utilización» puede utilizarse para mostrar el widget «Número».

Los usuarios también pueden optar por mostrar un pequeño gráfico de líneas junto al KPI en el widget de números, para comprender cómo ha variado la métrica a lo largo del tiempo.
