# Widget de redimensionamiento

Cloudability Los paneles de control cuentan con una opción para visualizar los datos de la fuente de datos «Rightsizing».

Actualmente, el widget «Rightsizing» solo puede mostrar el ahorro obtenido con «Rightsizing» durante los últimos 10 días.

La función «Rightsizing» en los paneles de control de « Cloudability » está disponible para los siguientes productos:

- AWS EC2 (Elastic Compute Cloud)
- AWS EBS (Almacén de bloques elástico)
- AWS RDS (Servicio de bases de datos relacionales)
- Azure Compute
- Azure Disco gestionado
- Azure SQL Database
