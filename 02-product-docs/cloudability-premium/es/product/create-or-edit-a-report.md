# Crear o editar un informe

Se pueden crear nuevos informes haciendo clic en el botón «Nuevo informe» de la página Lista de informes. Los informes existentes se pueden editar accediendo al informe y haciendo clic en el botón azul «Editar informe». Se requieren permisos de «edición» para modificar un informe existente.

Al crear un informe, hay varias opciones de configuración disponibles:

- Nombre del informe: proporcione un nombre descriptivo para el informe, de modo que se pueda acceder fácilmente a él más adelante.
- Intervalo de fechas: configure el intervalo de fechas que se utilizará para completar el informe.
- Categoría: seleccione una categoría a la que se asignará un informe. La categoría se puede utilizar posteriormente para clasificar los informes y asignarlos a determinados grupos. También se puede crear una nueva categoría haciendo clic en el botón «Añadir categoría».
- Coste compartido: esta casilla de verificación permite aplicar las reglas de reparto de costes para generar el informe.

Dimensiones y métricas

En esta sección, los usuarios pueden seleccionar la fuente de datos para consultar los datos. Actualmente hay dos fuentes de datos disponibles: datos de costes o datos de utilización.

Los usuarios pueden seleccionar qué dimensiones y métricas de una fuente de datos determinada desean visualizar en el informe.

Al seleccionar varias métricas, los números de KPI relevantes se muestran en la parte superior del informe. Los usuarios pueden hacer clic en uno de los números de KPI para centrarse en una métrica determinada en el gráfico del informe que aparece a continuación. Todas las métricas seleccionadas se incluyen en la exportación de datos sin procesar.

Al seleccionar varias dimensiones, la tabla situada en la parte inferior del informe se desglosará por cada dimensión y se mostrará una fila independiente para cada combinación única de valores de dimensión. Esta selección también se tiene en cuenta para exportar datos sin procesar del informe.

A continuación, los usuarios pueden decidir cómo ordenar los datos de la tabla situada en la parte inferior del informe. Cualquier dimensión o métrica se puede utilizar para ordenar en orden descendente o ascendente.

Por último, los usuarios pueden configurar un conjunto de filtros que se utilizarán para reducir el conjunto de datos visible para los usuarios. Cualquier medida (ya sea métrica o dimensión) se puede utilizar para incluir o excluir datos del informe.
