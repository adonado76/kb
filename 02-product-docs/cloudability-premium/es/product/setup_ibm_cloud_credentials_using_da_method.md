# Configurar IBM Cloud Credenciales mediante la aplicación de arquitectura desplegable (DA)

La **aplicación de arquitectura desplegable (DA)**, también conocida como **IBM Cloudability Enablement**, ofrece una forma alternativa y simplificada de conectar su cuenta IBM Cloud a Cloudability. Esta aplicación simplifica el proceso de configuración y es el **método recomendado** para una acreditación más rápida. La aplicación DA está disponible en el [catálogo público](https://cloud.ibm.com/catalog/architecture/deploy-arch-ibm-cloudability "(se abre en una pestaña o una ventana nueva)") IBM Cloud. Para obtener más información sobre su arquitectura y cómo configurarla e implantarla, consulte la [documentación de Cloudability Enablement DA App](https://cloud.ibm.com/docs/track-spend-with-cloudability?topic=track-spend-with-cloudability-overview "(se abre en una pestaña o una ventana nueva)").

Nota:

El uso de este método para acreditar su cuenta IBM Cloud en Cloudability es completamente opcional.

Para obtener ayuda y soporte para IBM Cloudability Enablement utilizando DA App, visite: [Obtención de ayuda y soporte para IBM Cloudability Habilitación](https://cloud.ibm.com/docs/track-spend-with-cloudability?topic=track-spend-with-cloudability-help-and-support "(se abre en una pestaña o una ventana nueva)")
