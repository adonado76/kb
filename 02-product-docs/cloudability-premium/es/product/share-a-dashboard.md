# Compartir un panel de control

Para mejorar la colaboración entre varios usuarios y equipos, los paneles de control de Cloudability admiten la función de compartir paneles.

Todos los usuarios (tanto los que tienen permisos de visualización como los de edición) pueden compartir un panel de control mediante URL. Esta función permite generar un enlace URL que redirige directamente al panel de control que se está visualizando en ese momento.

Los usuarios pueden optar por incluir en el « URL » la vista aplicada actualmente y/o el intervalo de fechas del nivel del panel de control aplicado actualmente, con el fin de especificar con mayor detalle el contexto visible al seguir el « URL ».

![](../../../../03-media/cloudability-premium/images/share-a-dashboard.png)

Además, los usuarios con permiso de «Edición» para un panel concreto pueden optar por compartir dicho panel con otros usuarios de Cloudability o con toda la organización.

El menú desplegable «Compartir con el usuario» permite seleccionar la lista de usuarios con los que se desea compartir el panel de control. El menú desplegable «Permisos» permite a los usuarios decidir si el panel de control se compartirá con permisos de «Visualización» o de «Edición».

[NOTA] Los administradores pueden editar cualquier panel de control que se les haya compartido; los permisos de solo lectura no se aplican.
