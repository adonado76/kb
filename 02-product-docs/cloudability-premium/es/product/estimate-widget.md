# Widget de estimación

Cloudability Los paneles de control cuentan con una opción para visualizar los datos de la fuente de datos «Estimate».

Actualmente, el widget «Estimate» solo puede mostrar la diferencia en los KPI entre «Este mes» y «El mes pasado» utilizando una de las siguientes métricas de costes:

- Coste (total combinado)
- Coste (amortizado)
- Coste (ajustado)
- Coste (amortizado ajustado)
- Coste (Lista)

Los cargos que aún no figuren en el archivo de facturación se estimarán con el fin de mostrar el indicador clave de rendimiento (KPI) del widget de estimaciones.
