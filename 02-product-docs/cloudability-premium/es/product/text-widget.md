# Widget de texto

El widget de texto se puede utilizar para añadir descripciones, comentarios o explicaciones adicionales al panel de control. Entre sus características más importantes se incluyen:

- Opciones básicas del editor integrado
- Compatibilidad con Markdown
- Incrustación de enlaces a contenido externo
