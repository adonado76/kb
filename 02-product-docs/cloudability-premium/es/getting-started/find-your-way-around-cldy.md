# Oriéntese Cloudability

## Cloudability Barra de herramientas

Permite un acceso rápido para gestionar el perfil, el acceso, las preferencias y mucho más.

![](../../../../03-media/cloudability-premium/images/find-your-way-around-cldy-toolbar.png)

## Vista actual

Le permite seleccionar filtros para toda la aplicación para diferentes usuarios en su cuenta Cloudability, para restringir los datos de Cloudability.

## Barra de herramientas

Permite acceder a varias páginas de ayuda y acceso a soporte, seguimiento del estado de los informes exportados, varios ajustes a nivel de aplicación y gestión de su perfil y acceso. Estas opciones se presentan en función de su función

| Icono | Descripción | Servicios de suministro |
| --- | --- | --- |
| icono de ayuda | Ayuda | Puede:  Buscar en el centro de ayuda  Enviar comentarios  Enviar una solicitud de asistencia  Acceda a la documentación de la API Cloudability  Acceder al servicio de asistencia  Vea las novedades en Cloudability |
| icono de descarga | Exportaciones de informes | Puede:  Ver el estado de los informes exportados  Volver a descargar un informe exportado anteriormente |
| icono de configuración | Valores | Puede:  Gestión de tareas automatizadas de gestión de datos y seguimiento de su estado  Configurar calendarios fiscales  Cambiar a varios entornos o productos Apptio  Gestionar la administración de accesos  Configurar varias monedas  Ver anomalías, configurar alertas de anomalías  Configurar el host del agente de métricas  Ver informes suscritos  Gestionar las preferencias de los usuarios  Gestión de permisos y configuraciones de fuentes de datos  Gestiona tu perfil, preferencias como el correo electrónico diario, la configuración de la organización, la moneda y la clave API  Gestionar las preferencias de notificación del presupuesto |
| icono de cuenta de usuario | Valores de perfil | Puede:  Gestione su perfil  Hacerse pasar por otro usuario  Gestión de cuentas de usuario y administración de accesos  Los administradores pueden querer hacer esto para comprobar los permisos de los usuarios o para solucionar problemas de acceso.  Fin de sesión |
| icono del menú de la aplicación | Todos los productos | Navegue a otro producto Apptio / IBM o a otro entorno |
