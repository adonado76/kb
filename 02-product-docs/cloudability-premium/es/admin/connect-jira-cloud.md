# Conectar Jira Cloud

En este artículo se explica cómo configurar Jira Cloud en Cloudability.

Antes de empezar

Antes de empezar, asegúrate de lo siguiente:

- Tu organización utiliza Jira Cloud (y no Jira Server).
- Eres administrador de « Cloudability ».
- El administrador de tu instancia de Jira estará disponible durante el proceso de configuración.
- Ya tienes el nombre del sitio de tu instancia de Jira.

Pasos para la integración

Credenciales de Jira para Cloudability

1. En Cloudability, ve a Configuración > Credenciales de proveedor > Añadir fuente de datos > JIRA.
2. Se abre el panel «Añadir cuenta de JIRA ».
3. Introduce el nombre de «Base URL ».
4. Selecciona «Guardar ».

Cloudability a Jira

1. Inicia sesión en tu instancia de Jira Cloud como administrador.
2. Selecciona «Configuración» > «Complementos ».
3. Selecciona «Buscar nuevos complementos ».
4. Busca « Cloudability ».
5. Selecciona «Instalar ».
6. Selecciona «Aceptar e instalar ». Espera a que finalice la instalación.

Cómo comprobar que se ha realizado correctamente

Comprueba el estado en Configuración > Credenciales de proveedor.
