# Mapeie as subtorres

As subárvores padrão foram definidas para os dados de benchmark fornecidos pelo aplicativo Apptio Benchmarking Infrastructure. Para ativar os relatórios de Benchmarking, você deve mapear suas subárvores para as subárvores de aplicativos.

Ao criar o projeto Costing Standard , você definiu as subtorres em sua infraestrutura de TI. Se você aceitou as definições padrão de subtorres, elas corresponderão às subtorres padrão definidas nos dados de referência. Sempre que possível, você precisa fazer a correspondência entre as subárvores dos dados de benchmark e as subárvores da sua infraestrutura de TI. O mapeamento completo das subtorres fornece os dados de benchmarking mais completos nos relatórios. Você mapeia as subtorres usando as instruções do aplicativo, conforme mostrado abaixo.

![img](../../../../../03-media/apptio-benchmarking/resources/images/it_benchmarking_images/map-sub-towers.jpg)

**Pré-requisitos**

Antes de poder mapear as subtorres e os reservatórios de custos, você precisa ter:

Importou o benchmarking da AIB.

Instalou o componente CTF-Benchmarking (consulte ).

Anexou os dados de benchmarking da AIB aos dados mestre de Benchmarking.

**Para mapear as subtorres**

1. Selecione a guia Relatórios.
2. Na página inicial, selecione Benchmarking.
3. Na barra de ferramentas de navegação do Benchmarking, selecione o ícone Mapa destacado abaixo.

![img](../../../../../03-media/apptio-benchmarking/resources/images/it_benchmarking_images/map-sub-towers-map.jpg)

1. Selecione a guia Sub-Towers.
2. Siga as instruções na página de mapeamento.

**Para mapear os pools de custos**

Para mapear os pools de custos, selecione a guia Pools de custos e siga o mesmo procedimento usado para mapear as subtorres.

Ao mapear os pools de custos, cada pool de custos de benchmark deve ser mapeado para um único pool de custos do cliente.
