# Guia de configuração de benchmarking

A configuração do Benchmarking é um processo de duas etapas, ilustrado na série de vídeos a seguir.

**Etapa 1: Configuração interativa de benchmarking**

Insira seus dados reais no Interactive Benchmarking:

- Automaticamente, extraindo os dados reais de seu projeto de Transparência de Custos (CT):

  [Vídeo: Obter automaticamente os dados reais da Transparência de custos (8 min.)](https://community.ibm.com/community/user/viewdocument/about-fetching-actuals-from-cost-tr "(Abre em uma nova guia ou janela)")
- Ou manualmente:

  [Vídeo: Entrada manual ou ajuste de valores reais (5 min.)](https://community.ibm.com/community/user/viewdocument/video-configure-interactive-benchm "(Abre em uma nova guia ou janela)")  Este vídeo também explica como você pode substituir os valores reais extraídos do seu projeto de CT.

Se o seu projeto de TC estiver em uma moeda que não seja a dos EUA (a partir de abril de 2017, somente em euros ou libras esterlinas), os vídeos a seguir fornecem orientações sobre a conversão no Interactive Benchmarking:

- [Vídeo: Uso com moedas não americanas (Introdução)](https://community.ibm.com/community/user/viewdocument/video-configure-interactive-benchm-1 "(Abre em uma nova guia ou janela)")
- [Vídeo: Uso com moedas não americanas (detalhes)](https://community.ibm.com/community/user/viewdocument/configure-interactive-benchmarking "(Abre em uma nova guia ou janela)")

**Etapa 2: Configuração do Benchmarking de Transparência de Custos**

Faça o download dos benchmarks selecionados para o setor, TI OpEx, métricas de infraestrutura da Interactive Benchmarking como um CSV.

Carregue o arquivo.CSV do Benchmark no Cost Transparency:

- [Vídeo: Carregar e anexar os dados de referência na Transparência de custos (3 min.)](https://community.ibm.com/community/user/viewdocument/configure-cost-transparency-benchma-1 "(Abre em uma nova guia ou janela)")

Mapeie seus pools de custos para referências em transparência de custos:

- [Vídeo: Mapear pools de custos (2 min.)](https://community.ibm.com/community/user/viewdocument/configure-cost-transparency-benchma-2 "(Abre em uma nova guia ou janela)")
- [Mapear os pools de custos](map-cost-pools.html)

Mapeie suas torres, subtorres e unidades de medida para referências em transparência de custos:

- [Vídeo: Torres de mapas (1 min.)](https://community.ibm.com/community/user/viewdocument/configure-cost-transparency-benchma "(Abre em uma nova guia ou janela)")
- [Mapear as torres](map-towers.html)
- [Configurar o Benchmarking de Transparência de Custos: Mapear subníveis (2 min.)](https://community.ibm.com/community/user/viewdocument/configure-cost-transparency-benchma-4 "(Abre em uma nova guia ou janela)")
- [Mapeie as subtorres](map-sub-towers.html)
- [Vídeo: Configurar o Benchmarking de Transparência de Custos: Mapear unidades de medida (2 min.)](https://community.ibm.com/community/user/viewdocument/video-configure-cost-transparency "(Abre em uma nova guia ou janela)")
- [Mapear as unidades de medida](map-uom.html)
- [Escolha os benchmarks exibidos nos relatórios](report-benchmarks.html)
