# Introdução

Este documento fornece as etapas de configuração manual do Interactive Benchmarking . A abordagem preferida para configurar o Interactive Benchmarking é conectar-se a um projeto de transparência de custos e recuperar os dados de custos - consulte o site da comunidade Benchmarking para obter vídeos de configuração sobre isso. No entanto, se você não conseguir se conectar ao projeto Cost Transparency e quiser obter manualmente os dados do seu projeto Cost Transparency, este documento fornece as etapas.

A Interactive Benchmarking tem três níveis de métricas de benchmark:

- Setor - Essas métricas exigem dados organizacionais do cliente, juntamente com dados anuais de custo de TI
- TI OpEx - Essas métricas exigem dados de custo anual do cliente por pools de custo e torres de recursos de TI
- Infraestrutura - As métricas de infraestrutura exigem custos anuais por subtorre, juntamente com seus volumes

Você pode obter mais informações sobre o produto Benchmarking no site da comunidade Apptio.

Com o Interactive Benchmarking, você pode se iluminar e encontrar valor mesmo com um subconjunto de dados. Um bom ponto de partida é (1) as métricas do setor e a adição e (2) a TI OpEx,, se você conseguir obter ATUM pools e torres de custo. Quando essas duas estiverem ativadas, você poderá identificar algumas torres/subtorres para iluminar a (3) Infraestrutura.

Para configurar essas métricas de benchmark, o Interactive Benchmarking fornece uma tela de configurações com várias guias.
