# Preencha a guia Perfil organizacional

Os valores do perfil organizacional são valores usados nas métricas do relatório de referência do setor e no cálculo do relatório de referência de TI OpEx.

Esses valores não são importados do modelo de transparência de custos. É necessária a entrada manual.

Os dados para esta seção podem ser (1) estimados ou baseados nas informações encontradas on-line nos arquivos da SEC ou (2) ser os valores reais do cliente.

É importante que os dados organizacionais correspondam aos números de gastos de TI nos dados de custo. Por exemplo, se os gastos com TI abrangem apenas a região da América do Norte, os dados organizacionais precisam refletir apenas essa parte dos negócios do cliente.
