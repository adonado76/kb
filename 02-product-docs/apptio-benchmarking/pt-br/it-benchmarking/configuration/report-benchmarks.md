# Escolha os benchmarks exibidos nos relatórios

Os relatórios de benchmark exibem 13 benchmarks de subtorres diferentes. Cinco das subtorres têm duas ou mais unidades de medida que podem ser usadas. Você deve escolher a unidade de medida que deseja usar para essas cinco subtorres.

As subtorres são:

Área de trabalho

Windows

Unix

Central de serviços

Linux

**Para escolher os benchmarks de subtorre exibidos nos relatórios de benchmark**

Abra o conjunto de dados Benchmark Unit of Measure Reference List.

1. Selecione Yes (Sim ) ou No (Não) na coluna Enabled (Ativado ).

![apex](../../../../../03-media/apptio-benchmarking/resources/images/it_benchmarking_images/choose-uom.jpg)

Neste exemplo, há duas opções de unidade de medida para a subárvore Linux : Servidores lógicos e Servidores físicos. Servidores lógicos foi selecionado como a unidade de medida, conforme indicado pelo valor Sim na coluna Ativado.

Para facilitar a seleção das unidades de medida, classifique a tabela pela coluna Subnível de recursos de TI.
