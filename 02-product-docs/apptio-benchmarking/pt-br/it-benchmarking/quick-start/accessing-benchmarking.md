# Acessando o Benchmarking

Para abrir o Benchmarking d IBMApptio, siga uma das duas opções listadas abaixo:

1. Faça login em IBM Apptio Benchmarking diretamente navegando até este endereço da web: [https://itbmx.apptio.com](https://itbmx.apptio.com "(Abre em uma nova guia ou janela)")
2. Acesse IBM Apptio Benchmarking via Administração do Access por:
   1. Navegando até a Administração de Acesso no seu navegador da web [https://frontdoor.apptio.com](https://frontdoor.apptio.com "(Abre em uma nova guia ou janela)")
   2. Clique no botão **Benchmarking Interativo**

      ![Acessando o Benchmarking](../../resources/images/it%20benchmarking_images/accessing-benchmarking.png)

Após a abertura, você poderá ver

- Uma página inicial com pontos de entrada, como **Benchmarking Interativo**.
- Blocos para relatórios **de custos** com benchmark, se configurados.
- Um caminho para **Configurações** ou configuração para Benchmarking, visível para usuários com direitos de administrador.

Se você não encontrar o botão Benchmarking Interativo ou tiver alguma dificuldade para acessar o aplicativo

- Confirme com seu contato de IBM Apptio se o produto foi provisionado para você.
- Confirme se sua função de usuário inclui acesso ao Benchmarking Interativo. No mínimo, a função que lhe foi atribuída no Frontdoor deve incluir o *ViewInteractiveBenchmarksAndCostData* acesso ao Benchmarking Interativo. Para concluir todas as ações de configuração, a função deve incluir as permissões *ViewInteractiveBenchmarksAndCostData* *RetrieveInteractiveBenchmarkingCostData*, *ConfigureInteractiveBenchmarking* e.
- Confirme se você está no ambiente correto (para o principal).
