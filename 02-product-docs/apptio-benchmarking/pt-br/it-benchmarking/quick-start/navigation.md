# Navegando

Esta seção irá familiarizá-lo com os elementos de navegação encontrados no Benchmarking Interativo.

**Barra de navegação principal**

![Avaliação comparativa interativa Navegação principal](../../resources/images/it%20benchmarking_images/main-navigation-benchmarking.png)

1. **Apptio Logotipo** : Voltar à Administração do Acesso.
2. **Menu de aplicativos** : alterne para um aplicativo diferente dentro do seu ambiente de Administração de Acesso.
3. **Menu Coleção de relatórios** : alterne entre diferentes coleções de relatórios dentro do Benchmarking Interativo.
4. **Configurações do perfil** : gerencie seus perfis de usuário e faça logout.
5. **Configurações** : Acesse a configuração do Interactive Benchmarking e a administração do acesso Frontdoor.
6. **Ajuda** : Acesse a Ajuda e identifique a versão do Interactive Benchmarking.

**Coleção de referências do setor**

![Coleção de referências do setor](../../resources/images/it%20benchmarking_images/collection.png)

1. **Guia “Despesas com TI como % da receita** ”: x
2. **Despesas com TI como % do OpEx tab** : X
3. **% OpEx Guia Gastos** : X
4. **Guia Gastos com TI por funcionário** : X
5. **Guia “FTEs de TI como % dos funcionários** ”: x

**OpEx de TI Coleção de benchmarks**

![OpEx de TI Coleção de benchmarks](../../resources/images/it%20benchmarking_images/opex-benchmarking.png)

1. **Na guia “Cost Pool”** (Grupo de custos): Compare o “ OpEx ” (Custo real) por grupo de custos com a comunidade “ Apptio ” (Grupo de custos).
2. **Na guia “Resource Tower” (Torre de recursos)/“Application Sub-Tower” (Subtorre de aplicativos** ): Compare o “ OpEx ” (Desempenho do aplicativo) real por torre de recursos com a comunidade “ Apptio ” (Desempenho do aplicativo) (quando o menu “Resource Tower” (Torre de recursos) estiver definido como “Overall” (Geral)). Compare o uso real de recursos ( OpEx ) por subtorre de recursos dentro da torre de recursos “Aplicativo” com a comunidade de uso de recursos ( Apptio ) (quando o menu da torre de recursos estiver definido como “Aplicativo”).
3. **Menu da Torre de Recursos** : Filtre o conteúdo do relatório para “Geral” (todas as Torres de Recursos) ou restrinja o conteúdo à Torre de Recursos “Aplicativo”.

**Coleção de benchmarks de infraestrutura**

![Coleção de benchmarks de infraestrutura](../../resources/images/it%20benchmarking_images/infrastructure-benchmarking.png)

1. **Guia Custos unitários** : compare os custos unitários reais com as métricas de referência de infraestrutura.
2. **Guia Eficiência FTE** : Compare a eficiência FTE real com métricas de referência de infraestrutura.
3. **Menu Subtorre de recursos** : filtre o conteúdo do relatório para uma Subtorre de recursos específica.

**Configuração**

![Configuração](../../resources/images/it%20benchmarking_images/benchmarking-configuration.png)

1. **Guia Perfil da organização** : insira a receita anual da organização, OpEx, e o número médio de funcionários representados como FTEs (equivalentes a tempo integral).
2. **Apptio Guia Origem** : Defina o projeto de cálculo de custos de origem e valide sua conexão.
3. **Guia Dados de custos** : insira os volumes anuais de gastos com TI.
4. **Guia Volumes** : insira a contagem média de dispositivos, usuários, instâncias e outros volumes relacionados.
5. **Guia FTEs** : Insira os volumes médios de FTE para as subtorres de recursos.
6. **Guia Avançado** : Defina qual versão e moeda do TBM ( ATUM ) serão aplicadas.

**Configuração inicial: etapas**   
gerais Do ponto de vista do analista, a configuração inicial consiste em quatro etapas principais:

1. Confirme o ambiente e o acesso.
2. Configure o perfil organizacional.
3. Forneça dados anuais sobre custos de TI.
4. Habilite domínios de referência e execute uma comparação básica.
