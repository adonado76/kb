**Unidade de medida do usuário final**

| Subtorre de recursos de TI | Unidade de medida (padrão) | Diretrizes |
| --- | --- | --- |
| Área de trabalho  Software do usuário final  Suporte na mesa de trabalho | Computadores pessoais | Computador pessoal é definido como desktop, computadores portáteis/laptops, terminais burros/clientes finos, terminais X, terminais baseados em Windows  Um computador usado por uma pessoa de cada vez, que pode executar vários aplicativos diferentes.  Um computador pessoal típico é um dispositivo de usuário único, com pelo menos uma tela e um dispositivo de entrada. Eles também podem ser conhecidos como estações de trabalho, sistemas de desktop ou sistemas clientes em uma configuração cliente/servidor. Exemplos:  - PCs compatíveis com Intel que executam Microsoft Windows  - Estações de trabalho Unix  - Laptops ou computadores portáteis  Inclui computadores que são usados principalmente por uma pessoa, mas que oferecem algumas funções de servidor (por exemplo, compartilhamento de arquivos ou impressoras ponto a ponto).  Não inclua computadores que estejam atuando principalmente como servidores, mesmo que sejam construídos com hardware semelhante ao de computadores pessoais de mesa. Não inclua terminais burros, thin clients, impressoras de mesa, PDAs, smartphones, tablets ou dispositivos dedicados a um aplicativo comercial específico, como caixas eletrônicos ou dispositivos de ponto de venda. Inclui terminais burros e thin clients usados para computação pessoal (como VDI). |
| Conferências e A/V | Usuários | O número de usuários que consomem serviços de TI (incluindo funcionários em tempo integral, prestadores de serviços, consultores, fornecedores) e é usado para calcular o custo de comunicação por usuário. A maioria dos clientes coleta essa contagem reunindo uma contagem de IDs de usuário para logon na rede, mas outras estimativas também são aceitáveis. |
| Help Desk de TI | Contatos/Mês | A unidade de medida é contatos por mês para os 12 meses anteriores. Um contato é uma comunicação entre um cliente e a Central de Serviços. Os contatos podem ser chamadas telefônicas, e-mails, uma visita pessoal, o preenchimento de um formulário de solicitação ou qualquer outra solicitação de um cliente. Os tíquetes abertos contam automaticamente se for necessário um esforço de FTE para lidar com eles, ou seja, eles não são fechados automaticamente. |
| E-mail | Caixa de correio | O número de caixas de e-mail. Normalmente, inclui caixas de e-mail provisionadas para usuários que consomem serviços de TI, bem como caixas de e-mail para não usuários, como contas de serviço. Se essa contagem for difícil de obter, comece com o número de usuários. |
