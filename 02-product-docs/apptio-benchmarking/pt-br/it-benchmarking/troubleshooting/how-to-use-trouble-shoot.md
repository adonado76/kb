# Como usar a seção Solução de problemas e perguntas frequentes

A seção de solução de problemas e perguntas frequentes é para quando algo parece errado, está faltando ou é confuso, e para quando alguém mais experiente lhe diz “Não acredito nesse número”

Público:

- Analistas de TBMA/ITFM
- Administradores
- Qualquer pessoa que tenha que explicar ou defender os resultados da análise comparativa

## Como usar a seção Solução de problemas e perguntas frequentes

Use esta seção quando:

- A visualização está em branco ou parcialmente em branco
- Uma métrica parece muito errada
- Os grupos de colegas se comportam de maneira estranha
- A análise comparativa não reflete as alterações recentes nos custos
- Você precisa de uma explicação curta e reutilizável para as partes interessadas

Cada seção oferece:

- Sintoma
- Causas prováveis
- O que verificar
- Uma resposta curta que você pode adaptar para e-mails ou slides

Se a causa principal for a configuração, você geralmente voltará ao [Guia de Configuração](../home.html#benchmarking__config_guide).

Se a causa principal for as definições, você será direcionado para [Referência de dados e metodologia](../home.html#benchmarking__data-methodology).
