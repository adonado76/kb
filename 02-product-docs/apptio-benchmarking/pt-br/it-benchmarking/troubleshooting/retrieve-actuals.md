# Sobre a obtenção de dados reais a partir do cálculo de custos padrão 12.2 +

Se você for um usuário das versões do Cost Transparency anteriores a 12.2, poderá recuperar automaticamente os dados reais para o Interactive Benchmarking.

No entanto, se você for usuário da versão 12.2 e posterior da Transparência de custos, poderá inserir manualmente os dados reais para o site interativo Benchmarking. [Consulte Benchmarking interativo - Guia de configuração manual (PDF)](../configuration/interactive-benchmarking/access-settings.html)

A capacidade de recuperar automaticamente os dados reais para os usuários do 12.2 + está planejada para uma versão futura em Q3, 2017.
