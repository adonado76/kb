# Navigating

This section will familiarize you with the navigational elements found within Interactive
Benchmarking.

**Main Navigation Bar**

![Interactive Benchmarking Main Navigation](../../resources/images/it%20benchmarking_images/main-navigation-benchmarking.png)

1. **Apptio Logo**: Return to Access Admninistration.
2. **Application Menu**: Switch to a different application within your Access Admninistration
   environment.
3. **Report Collection Menu**: Switch between different report collections within Interactive
   Benchmarking.
4. **Profile Settings**: Manage your user profiles and log out.
5. **Settings**: Access Interactive Benchmarking’s Configuration and Frontdoor Access
   Administration.
6. **Help**: Access Help and identify the version of Interactive Benchmarking.

**Industry Benchmarks collection**

![Industry Benchmarks collection](../../resources/images/it%20benchmarking_images/collection.png)

1. **IT Spend as % of Revenue tab**: x
2. **IT Spend as % of OpEx tab**: X
3. **% OpEx Spend tab**: X
4. **IT Spend per Employee tab**: X
5. **IT FTEs as % of Employees tab**: x

**IT OpEx Benchmarks collection**

![IT OpEx Benchmarks collection](../../resources/images/it%20benchmarking_images/opex-benchmarking.png)

1. **By Cost Pool tab**: Compare actual OpEx by Cost Pool to the Apptio Community.
2. **By Resource Tower / By Application Sub-Tower tab**: Compare actual OpEx by Resource Tower
   to the Apptio Community (when Resource Tower menu is set to “Overall”). Compare actual OpEx by
   Resource Sub-Tower within the “Application” Resource Tower to the Apptio Community (when Resource
   Tower menu is set to “Application”).
3. **Resource Tower menu**: Filter contents of the report to “Overall” (all Resource Towers) or
   restrict content to the “Application” Resource Tower.

**Infrastructure Benchmarks collection**

![Infrastructure Benchmarks collection](../../resources/images/it%20benchmarking_images/infrastructure-benchmarking.png)

1. **Unit Costs tab**: Compare actual unit costs with infrastructure benchmark metrics.
2. **FTE Efficiency tab**: Compare actual FTE efficiency with infrastructure benchmark
   metrics.
3. **Resource Sub-Tower menu**: Filter contents of the report to a specific Resource
   Sub-Tower.

**Configuration**

![Configuration](../../resources/images/it%20benchmarking_images/benchmarking-configuration.png)

1. **Organization Profile tab**: Enter the organization’s annual revenue, OpEx, and average
   number of employees represented as FTEs (full-time equivalent).
2. **Apptio Source tab**: Define the source Costing project and validate its connection.
3. **Cost Data tab**: Enter annual IT spend volumes.
4. **Volumes tab**: Enter the average count of devices, users, instances, and other related
   volumes.
5. **FTEs tab**: Enter the average FTE volumes for Resource Sub-Towers.
6. **Advanced tab**: Define which ATUM (TBM) version and currency will be applied.

**First-time setup: high-level steps**  
From an analyst perspective, first-time
setup is four main steps:

1. Confirm environment and access.
2. Configure organizational profile.
3. Provide annual IT cost data.
4. Enable benchmark domains and run a basic comparison.
