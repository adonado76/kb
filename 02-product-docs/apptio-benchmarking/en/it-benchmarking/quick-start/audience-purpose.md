# Audience and purpose

This section is for TBM Analysts, IT Finance Analysts, and admins who need to get IBM Apptio
Benchmarking up and running to the point where you can show a credible comparison.

You are not doing deep configuration here, just enough to:

- Access the product
- Enter basic profile and cost data
- Run and sanity check your first benchmark views

The section assumes:

- You already have access to the Apptio platform.
- Your organization either uses Costing or can provide at least basic annual IT spend data.
- You are comfortable with TBM concepts such as Cost Pools, Resource Towers, and Resource
  Sub-Towers.

The goal is simple:  
Starting from “I can log in,” get to “I can show a benchmark view
that makes sense” in a short, repeatable path.
