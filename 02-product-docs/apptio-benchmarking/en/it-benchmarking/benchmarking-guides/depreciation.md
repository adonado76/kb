# Depreciation in benchmarks

This page describes how depreciation is included in benchmarks, making forecasts more accurate
for predicting future spending on IT assets.

Interactive Benchmarking includes depreciation of IT assets differently for the three types of
benchmark:

Industry benchmarks: you can see details of any depreciation calculated in a
metric by selecting the tab and expanding the  How we calculated this metric  section, as
in the following example for  % Opex Spend  :

![img](../../../../../03-media/apptio-benchmarking/resources/images/it_benchmarking_images/opex-spend-depreciation.jpg)

You can expand the  Company Info, Instance Configuration, and Cost Data
 section and hover over the hyperlinked items to view a description of what each item contains
and if it includes depreciation.

[Learn more about Industry Benchmark metrics definitions](metrics-definitions.html)

OpEx benchmarks: monthly depreciation is considered in OpEx but the initial
capital investment is not included.

Infrastructure benchmarks: this data includes both CapEx and OpEx. CapEx costs
assume a three year straight line depreciation.
