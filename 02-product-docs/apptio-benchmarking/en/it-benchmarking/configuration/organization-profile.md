# Organizational profile configuration

Organizational profile drives:

- Scaling metrics like IT spend as percentage of revenue and per employee.
- Eligibility for some benchmark datasets.

**Required fields**   
In the Configuration settings, you will see a profile
section.

![Interactive Benchmark’s Configuration – Organization Profile](../../resources/images/it%20benchmarking_images/benchmarking-organization.png)

Populate the following fields:

- Organization (Annual) Revenue
- Organization (Annual) OpEx
- Average FTE (full-time equivalent) count for the year

Attention: Scope must be consistent. If your IT org’s cost applies only to a region or
segment of the larger enterprise, use revenue and headcount for that same region or segment. In
other words, align the scope of the Organization details to the same group supported by the IT org.
Additionally, ensure the IT org itself is included in the details.

**Maintenance**   
Review the profile when:

- A major acquisition or divestiture changes revenue and headcount materially.
- Your primary industry classification changes.
- Your IT scope changes from regional to global or vice versa.

  
Small year-over-year changes are normal. You do not need to edit the profile for
every minor revenue movement.
