# Choose the benchmarks displayed on reports

The benchmark reports display 13 different sub-tower benchmarks. Five of the sub-towers have two or more units of measure that can be used. You must choose which unit of measure you want to use for those five sub-towers.

The sub-towers are:

Workspace

Windows

Unix

Service Desk

Linux

**To choose the sub-tower benchmarks displayed on the benchmark reports**

Open the
Benchmark Unit of Measure Reference List
data set.

1. Select
   Yes
   or
   No
   in the
   Enabled
   column.

![apex](../../../../../03-media/apptio-benchmarking/resources/images/it_benchmarking_images/choose-uom.jpg)

In this example, there are two unit of measure options for the Linux sub-tower: Logical Servers and Physical Servers. Logical Servers has been selected as the unit of measure as indicated by the
Yes
value in the
Enabled
column.

To make it easier to select the units of measure, sort the table by the
IT Resource Sub-Tower
column.
