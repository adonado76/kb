# Complete the Organizational Profile Tab

The Organizational Profile values are values used in the Industry Benchmark report metrics and
in the calculation for the IT OpEx Benchmark report.

These values are not imported from the Cost Transparency model. Manual entry is required.

The data for this section can either be (1) estimated or based on the information found online in SEC filings or (2) be the customer’s actual values.

It is important that the organizational data corresponds to the IT spend numbers in the cost data. For example, if the IT spend covers the North America region only, then the organizational data needs to reflect only that portion of the customer’s business.
