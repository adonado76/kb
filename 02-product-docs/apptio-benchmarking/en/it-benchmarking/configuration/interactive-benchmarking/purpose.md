# Introduction

This document provides steps on configuring Interactive Benchmarking manually. The preferred
approach to configure Interactive Benchmarking is to connect to a Cost Transparency project and
retrieve cost data – refer to the Benchmarking Community Site for Configuration videos on this.
However, if you are unable to connect to the Cost Transparency project and would like to manually
get data from your Cost Transparency project, this document provides the steps.

Interactive Benchmarking has three levels of benchmark metrics:

- Industry – These metrics require a customer’s organizational data along with annual IT cost data
- IT OpEx – These metrics require a customer’s annual cost data by cost pools & IT resource towers
- Infrastructure – Infrastructure metrics require annual costs by sub-tower along with their volumes

You can get more information on the Benchmarking product on the Apptio Community Site.

With Interactive Benchmarking, you can light up and find value even with a subset of data. A
good starting point is (1) the Industry metrics and adding, and (2) IT OpEx, if you are able to get
ATUM cost pools and towers. Once these two are enabled, you can identify a couple of
towers/sub-towers to light up (3) Infrastructure.

In order to configure these benchmark metrics, Interactive Benchmarking provides a settings screen with multiple tabs.
