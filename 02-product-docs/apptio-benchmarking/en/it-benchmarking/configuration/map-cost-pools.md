# Map the cost pools

Standard cost pools have been defined for the benchmark data provided by the Apptio Benchmarking
Infrastructure application. To light up the Benchmarking reports, you must map your cost pools to
the application cost pools.

The standard cost pools are:

External Labor

Facilities & Power

Hardware

Internal Labor

Outside Services

Software

Telecom

Other

When you created the Costing Standard project, you defined the cost pools in your IT
infrastructure. These may or may not match the standard cost pools defined in the benchmark data.
Where possible, you need to match the cost pools in the benchmark data to the cost pools in your IT
infrastructure. Complete cost pool mapping gives the most complete benchmarking data in the reports.
You map the cost pools using the instructions in the application, as shown .

![CP](../../../../../03-media/apptio-benchmarking/resources/images/it_benchmarking_images/cost-pool-mapping.jpg)

**Prerequisites**

Before you can map the cost pools, you must have:

Imported the AIB benchmarking .

Installed the CTF-Benchmarking component (see ).

Appended the AIB benchmarking data to the Benchmarking master data .

**To map the cost pools**

Select the
Reporting
tab.

1. On the Home page, select
   IT Benchmarks
   .
2. In the Benchmarking navigation toolbar, select the Map icon highlighted below.

![cp1](../../../../../03-media/apptio-benchmarking/resources/images/it_benchmarking_images/cost-pool-mapping-map.jpg)

1. Select the
   Cost Pools
   tab.
2. Follow the instructions on the
   Mapping
   page.

When mapping cost pools, each benchmark cost pool must be mapped to a single customer cost pool.
