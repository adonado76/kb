# Purpose and scope

This section covers:

- How data flows between Costing, Benchmarking, and external datasets.
- Which configuration decisions matter and who owns them.
- How to set up organizational profile, cost data, infra volumes, and peer groups.
- How to keep the configuration healthy as the TBM model evolves.

It does not explain how to interpret reports or address gaps. That is in [Using Benchmarking Reports](../home.html#benchmarking__benchmark_reports) section and
[Operations & Governance](../home.html#benchmarking__operations_gov) section.
