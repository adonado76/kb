# Data partners and Benchmark sources

IBM Apptio Benchmarking combines multiple sources of benchmark data, aligned to the TBM taxonomy.

- **Industry and IT economics data**  
  Sourced from Rubin Worldwide, a provider of
  technology economics research and benchmarking data. These datasets focus on enterprise-level IT
  spend and staffing metrics.
- **Infrastructure metrics**   
  Sourced from ISG for infrastructure cost and
  efficiency benchmarks at the Resource Tower and Sub-tower levels.
- **Anonymized Apptio Community data**   
  Sourced from aggregated, opted-in,
  anonymized customer data at the Cost Pool level.
  - Actual customer cost data is normalized to the TBM taxonomy
  - The result is a set of spend distributions (for example OpEx by Cost Pool, OpEx by Resource
    Tower) that represent real operating models in the Apptio community

All benchmark data is normalized to TBM Cost Pools and Resource Towers so that it can be compared
directly with your Costing model once your actuals are aligned to the same taxonomy.
