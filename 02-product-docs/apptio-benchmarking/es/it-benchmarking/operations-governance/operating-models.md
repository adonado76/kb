# Modelo operativo y funciones

## Funciones clave

Responsable del programa TBM/ITFM

- Sabe cómo encaja el benchmarking en el programa TBM/ITFM
- Decide cuándo y dónde se utilizan los puntos de referencia
- Aprueba cambios importantes en la configuración y en los grupos de pares

Propietario de la evaluación comparativa

- Configura y mantiene el benchmarking
- **NOTA:** Consulte las secciones **[Guía](../home.html#benchmarking__config_guide)** **[de inicio rápido](../home.html#benchmarking__quickstart)** y configuración para obtener más información.
- Valida los resultados y prepara los análisis
- Mantiene la documentación, actualiza el calendario y el registro de cambios

Apptio Administrador de la plataforma

- Gestiona el acceso a las aplicaciones
- Garantiza que los entornos y las conexiones se mantengan en buen estado
- Abre y realiza el seguimiento de los tickets de soporte con Apptio cuando es necesario

Propietarios de Resource Tower / Responsables de dominio

- Consumir vistas de referencia relevantes para sus torres de recursos o dominios
- Proporcionar contexto para las diferencias con respecto a los compañeros
- Acciones conjuntas para abordar las deficiencias materiales

Finanzas / FP&A

- Utiliza puntos de referencia como base para establecer objetivos y planificar
- Ayuda a alinear las opiniones basadas en puntos de referencia con las opiniones financieras internas
- Cuestiona el alcance y las hipótesis cuando es necesario
