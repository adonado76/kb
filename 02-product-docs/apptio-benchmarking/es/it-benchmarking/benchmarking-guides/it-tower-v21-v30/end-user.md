**Usuario final Unidad de medida**

| Subtorre de recursos informáticos | Unidad de medida (estándar) | Directrices |
| --- | --- | --- |
| Espacio de trabajo  Software para usuarios finales  Asistencia de escritorio | Ordenadores personales | Por ordenador personal se entienden los ordenadores de sobremesa, los portátiles, los terminales tontos/clientes ligeros, los terminales X y los terminales basados en Windows  Ordenador utilizado por una persona a la vez, que puede ejecutar muchas aplicaciones diferentes.  Un ordenador personal típico es un dispositivo de usuario único, que proporciona al menos una pantalla y un dispositivo de entrada. También pueden denominarse estaciones de trabajo, sistemas de sobremesa o sistemas cliente en una configuración cliente/servidor. Ejemplos:  - PCs compatibles con Intel que ejecuten Microsoft Windows  - Estaciones de trabajo Unix  - Ordenadores portátiles  Incluye ordenadores utilizados principalmente por una persona, pero que ofrecen algunas funciones de servidor (por ejemplo, compartir archivos o impresoras entre iguales).  No incluya ordenadores que actúen principalmente como servidores, aunque estén construidos con hardware similar al de los ordenadores personales de sobremesa. No incluya terminales tontos, clientes ligeros, impresoras de sobremesa, PDA, teléfonos inteligentes, tabletas o dispositivos dedicados a una aplicación empresarial concreta, como cajeros automáticos o dispositivos de punto de venta. Incluye terminales tontos y clientes ligeros que se utilizan para la informática personal (como VDI). |
| Conferencias y A/V | Usuarios | El número de usuarios que consumen servicios informáticos (incluidos empleados a tiempo completo, contratistas, consultores, proveedores) y se utiliza para calcular el coste de comunicación por usuario. La mayoría de los clientes recopilan este recuento mediante un recuento de ID de usuario para el inicio de sesión en la red, pero también son aceptables otras estimaciones. |
| Servicio de asistencia informática | Contactos/Mes | La unidad de medida son los contactos mensuales anteriores a los 12 meses. Un contacto es una comunicación entre un cliente y el Service Desk. Los contactos pueden ser llamadas telefónicas, correos electrónicos, una visita personal, la cumplimentación de un formulario de solicitud o cualquier otra petición de un cliente. Los tickets abiertos cuentan automáticamente si se requiere un esfuerzo ETC para tratarlos, es decir, no se cierran automáticamente. |
| Correo electrónico | Buzón | El número de buzones de correo electrónico. Normalmente incluye buzones de correo electrónico proporcionados a usuarios que consumen servicios de TI, así como buzones de correo electrónico para no usuarios, como las cuentas de servicio. Si este recuento es difícil de conseguir, empiece por el número de usuarios. |
