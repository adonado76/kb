# Acerca de la obtención de datos reales de Costing Standard 12.2 +

Si es usuario de versiones de Transparencia de Costes anteriores a 12.2, puede recuperar automáticamente los datos reales para Benchmarking interactivo.

Sin embargo, si es usuario de la versión de Transparencia de costes 12.2 y posteriores, puede introducir manualmente los datos reales para Benchmarking interactivo. [Consulte Interactive Benchmarking - Guía de configuración manual (PDF)](../configuration/interactive-benchmarking/access-settings.html)

La capacidad de recuperar automáticamente los datos reales para los usuarios de 12.2 + está prevista para una futura versión en Q3, 2017.
