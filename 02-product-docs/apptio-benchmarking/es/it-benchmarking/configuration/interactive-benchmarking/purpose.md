# Introducción

Este documento proporciona los pasos para configurar Interactive Benchmarking manualmente. El método preferido para configurar Interactive Benchmarking es conectarse a un proyecto de Transparencia de Costes y recuperar los datos de costes - consulte el sitio de la comunidad Benchmarking para ver los vídeos de configuración al respecto. Sin embargo, si no puede conectarse al proyecto de Transparencia de Costes y desea obtener manualmente los datos de su proyecto de Transparencia de Costes, en este documento se indican los pasos a seguir.

Interactive Benchmarking dispone de tres niveles de métricas de referencia:

- Industria - Estas métricas requieren los datos organizativos de un cliente junto con los datos de costes anuales de TI
- TI OpEx - Estas métricas requieren los datos de costes anuales de un cliente por grupos de costes y torres de recursos de TI
- Infraestructura - Las métricas de infraestructura requieren costes anuales por subtorre junto con sus volúmenes

Puede obtener más información sobre el producto Benchmarking en el sitio de la Comunidad Apptio.

Con Interactive Benchmarking, puede iluminar y encontrar valor incluso con un subconjunto de datos. Un buen punto de partida son (1) las métricas de la industria y la adición, y (2) IT OpEx, si usted es capaz de obtener ATUM grupos de costes y torres. Una vez habilitadas estas dos, puedes identificar un par de torres/subtorres para iluminar (3) Infraestructura.

Para configurar estas métricas de referencia, Interactive Benchmarking ofrece una pantalla de configuración con varias pestañas.
