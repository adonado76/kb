# Acceso a la evaluación comparativa

Para abrir IBM Apptio Benchmarking, siga una de las dos opciones que se indican a continuación:

1. Inicie sesión en IBM Apptio Benchmarking directamente navegando a esta dirección web: [https://itbmx.apptio.com](https://itbmx.apptio.com "(se abre en una pestaña o una ventana nueva)")
2. Acceso IBM Apptio Evaluación comparativa a través de la administración de Access por:
   1. Navegando a Administración de acceso en su navegador web [https://frontdoor.apptio.com](https://frontdoor.apptio.com "(se abre en una pestaña o una ventana nueva)")
   2. Haga clic en el botón **«Benchmarking interactivo».**

      ![Acceso a la evaluación comparativa](../../resources/images/it%20benchmarking_images/accessing-benchmarking.png)

Después de abrirlo, puedes ver

- Una página de destino con puntos de entrada como **Interactive Benchmarking**.
- Azulejos para informes **de costes** habilitados para puntos de referencia, si están configurados.
- Una ruta a **Ajustes** o configuración para Benchmarking, visible para usuarios con derechos de administrador.

Si no ve el botón «Interactive Benchmarking» (Evaluación comparativa interactiva) o tiene dificultades para acceder a la aplicación

- Confirme con su IBM Apptio que el producto se le ha proporcionado.
- Confirme que su rol de usuario incluye acceso a Interactive Benchmarking. Como mínimo, la función que se le asigne en Frontdoor debe incluir el *ViewInteractiveBenchmarksAndCostData* para acceder a Interactive Benchmarking. Para completar todas las acciones de configuración, el rol debe incluir los permisos *ViewInteractiveBenchmarksAndCostData* *RetrieveInteractiveBenchmarkingCostData*, *ConfigureInteractiveBenchmarking* y.
- Confirma que estás en el entorno correcto (para main).
