# Comentarios y colaboración Uso de tareas

Se aplica a : Costing Standard desde 12.11.5 y posteriores

La función de tareas permite a los usuarios mantener conversaciones prácticas al permitirles crear comentarios y asignar a usuarios concretos tareas que deben completar. Las tareas pueden asignarse a una persona a la vez y pueden utilizarse en hilos privados o públicos.

Asignar una tarea

Para iniciar una tarea, añada un comentario ![Comentario](../../../../03-media/apptio-tbm-studio/resources/images/cnc/comments_icon.jpg) y determine si desea que el comentario sea privado o público. Para más información, consulte [Uso de los comentarios](using-comments.html) y de [la función de comentarios privados](cnc-private-commenting.html).

Introduce tu comentario y escribe @ para añadir un usuario.

Aparece una casilla para asignar un usuario y convertir el comentario en una tarea.

Seleccione la casilla de verificación Asignar a @ usuario.

Seleccione Asignar.

El usuario recibe una notificación por correo electrónico informándole de que se le ha asignado una tarea.

Reasignar una tarea

1. Para reasignar una tarea, seleccione Responder ![flecha](../../../../03-media/apptio-tbm-studio/resources/images/cnc/reply_icon.jpg) en un hilo existente y escriba @ para añadir un usuario. Aparece una casilla de verificación para asignar al usuario y convertir el comentario en una tarea.
2. Seleccione la casilla de verificación Asignar al usuario @.
3. Seleccione Asignar.

El usuario recibe una notificación por correo electrónico informándole de que se le ha asignado una tarea.

Resolver una tarea

Sólo los asignatarios o los asignadores pueden resolver una tarea.

Para resolver una tarea, realice una de las siguientes acciones:

Si usted es el asignado, seleccione Resolver ![garrapata](../../../../03-media/apptio-tbm-studio/resources/images/cnc/resolve_icon.jpg) al principio del hilo.

Si usted es el asignador, seleccione ![más](../../../../03-media/apptio-tbm-studio/resources/images/cnc/cnc_more_options_icon.jpg) al principio de la conversación y, a continuación, seleccione Marcar como resuelto.

Eliminar una tarea

Para eliminar una tarea, seleccione ![más](../../../../03-media/apptio-tbm-studio/resources/images/cnc/cnc_more_options_icon.jpg) al principio del hilo.

Seleccione Borrar hilo.

Los usuarios que no sean el asignatario o el asignador también pueden eliminar sus comentarios en el hilo de la tarea seleccionando ![más](../../../../03-media/apptio-tbm-studio/resources/images/cnc/cnc_more_options_icon.jpg) y luego Eliminar comentario.
