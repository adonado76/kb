## Visión general

Las tablas editables te permiten introducir y gestionar datos manualmente directamente en TBM Studio. Sirven como capas de introducción de datos editables por el usuario que se incorporan a las tablas de transformación y, en última instancia, a tu modelo de costes. Utilice tablas editables para situaciones como la asignación de mano de obra, los atributos de los centros de coste, las clasificaciones de proveedores, las explicaciones de las desviaciones presupuestarias y cualquier dato de referencia que requiera intervención humana o enriquecimiento.

**Diferencia clave:** TBM Studio admite dos tipos de tablas editables con fines fundamentalmente distintos:

- [Tablas en blanco editables](create-blank-et.html) : puedes crear todas las filas y columnas desde cero. Ideal para conjuntos de datos de referencia pequeños en los que cada dato procede de la información introducida por el usuario.
- [Tablas editables enriquecidas](create-eet.html) : el sistema genera filas a partir de una tabla de transformación existente, y los usuarios añaden anotaciones o clasificaciones. Ideal para complementar los datos generados por el sistema con el criterio humano.

Saber cuándo utilizar cada tipo es fundamental para crear flujos de datos fáciles de mantener.
