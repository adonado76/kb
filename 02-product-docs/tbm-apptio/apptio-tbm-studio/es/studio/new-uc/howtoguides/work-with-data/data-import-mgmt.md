## Importación y gestión de datos

En esta sección se ofrecen instrucciones paso a paso para importar y gestionar datos en TBM Studio. Tanto si subes archivos manualmente, te conectas a sistemas externos o gestionas versiones mensuales de datos, estas guías te ayudarán a crear una base de datos fiable para tus modelos de costes.

- [Cargar datos desde archivos (Excel, CSV )](upload-data-files.html) : importar datos desde archivos locales o de red
- [Conectarse a fuentes de datos externas ( DataLink )](connect-external-dl.html) : automatizar la integración de datos desde ServiceNow y otros sistemas
- [Tablas de versiones para datos mensuales](version-tables-monthly-data.html) : gestiona datos de series temporales con versiones mensuales
