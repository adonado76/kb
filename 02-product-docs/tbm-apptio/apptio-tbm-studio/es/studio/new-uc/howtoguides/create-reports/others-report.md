# Poniendo todo en práctica: tu primer informe

Ahora que ya conoces los conceptos básicos, a continuación te mostramos un flujo de trabajo típico para crear un informe:

- [Crea el informe](report-basic.html) con un nombre y una descripción claros
- [Añade una tabla](add-tables-report.html) para mostrar datos detallados: elige el objeto de modelo adecuado, añade dimensiones a «Filas» y añade métricas a «Valores»
- [Añade un gráfico](add-chart-visual.html) para visualizar las tendencias clave: elige el tipo de gráfico más adecuado para tu mensaje
- [Añade filtros](use-slicers.html) para mejorar la interactividad: incluye filtros para las dimensiones por las que los usuarios suelen filtrar
- [Revisa y guarda](others-report.html) : comprueba que el informe se muestre correctamente, prueba cómo funcionan los filtros y guarda los cambios

## Referencia rápida: Componentes del informe

|  |  |  |  |
| --- | --- | --- | --- |
| **Componente** | **Añadir desde** | **Áreas de configuración** | **Ideal para** |
| Tabla | Pestaña «Informe» > Tabla | Filas, columnas, valores, filtros | Datos detallados, exportaciones |
| Gráfico | Pestaña «Informe» > Gráfico | Leyenda, Ejes, Valores, Filtros | Patrones visuales, comparaciones |
| Selector de filas | Pestaña «Informe» > Filtro de filas | Filtrar por, Filtros | Filtrado interactivo |
| Cortadora compacta | Pestaña «Informes» > Compact Slicer | Cortar por (varios) | Filtrado que ahorra espacio |
