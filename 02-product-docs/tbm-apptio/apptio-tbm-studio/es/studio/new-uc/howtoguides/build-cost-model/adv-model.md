# Modelización avanzada

|  |  |
| --- | --- |
| **Tipo de contenido** | Guías prácticas |
| **Destinatarios** | Equipos de finanzas de TI, analistas de negocios (nivel intermedio a avanzado) |
| **Requisitos previos** | Ha completado los cursos « 3.2.1 » (Conceptos básicos de modelos) y « 3.2.2 » (Crear asignaciones); conoce las asignaciones de valor ponderado simple y de distribución uniforme |

Esta sección aborda técnicas avanzadas de modelización dirigidas a usuarios que ya dominan las asignaciones básicas y necesitan un control más preciso sobre cómo se distribuyen los costes en su modelo. Aprenderás a utilizar la lógica de correspondencias para vincular dinámicamente filas de origen y destino, configurar columnas de ponderación para una distribución proporcional, rastrear flujos de costes para la resolución de problemas y crear informes de modelos (diagramas de Sankey) para la comunicación con las partes interesadas.

- **[Utilizar la lógica de correspondencia para las asignaciones](../../../../studio/new-uc/howtoguides/build-cost-model/use-logic.html)**
- **[Configurar columnas de ponderación](../../../../studio/new-uc/howtoguides/build-cost-model/config-wc.html)**
- **[Seguimiento de los flujos de costes a lo largo del modelo](../../../../studio/new-uc/howtoguides/build-cost-model/trace-cost-flow.html)**
- **[Crear informes de modelos (diagramas de Sankey)](../../../../studio/new-uc/howtoguides/build-cost-model/create-model-rep.html)**
