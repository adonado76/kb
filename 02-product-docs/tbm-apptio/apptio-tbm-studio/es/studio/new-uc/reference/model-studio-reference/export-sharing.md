# Exportación y uso compartido

**Opciones para compartir**

|  |  |
| --- | --- |
| **Método** | **Descripción** |
| Informe de modelo | Crear un informe modelo para uso de los usuarios finales (se puede añadir a colecciones de informes) |
| Captura de pantalla | Diagrama de captura para documentación o presentaciones |
| Recopilación de informes | Añade el informe del modelo a una colección para facilitar el acceso ( v12.2.2+ ) |

**Consejo:** Utiliza diagramas de modelos para incorporar a nuevos miembros del equipo, resolver problemas de asignación y explicar la estructura del modelo a las partes interesadas.

*Consulte «Crear [informes» para conocer los procedimientos de creación de informes](../../howtoguides/create-reports/create-basic-report.html)*
