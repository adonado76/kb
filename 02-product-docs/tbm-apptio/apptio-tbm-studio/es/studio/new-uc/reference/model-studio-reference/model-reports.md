# Informes de modelos

Los informes de modelos permiten a los usuarios finales acceder a visualizaciones de los modelos, mostrando los flujos de costes en un diagrama de Sankey configurable y organizado por niveles.

## Configuración del diagrama de Sankey

**Creación de un informe modelo**

1. En la pestaña Inicio, haz clic en Nuevo > Informe de modelo
2. Haz clic en «Editar métricas» para seleccionar las métricas que deseas incluir
3. Añade niveles con el Editor de niveles
4. Arrastra tablas o columnas a los niveles desde el Explorador de proyectos
5. Ponle un nombre a cada nivel
6. Guarda y revisa el informe

**Operaciones del editor de niveles**

|  |  |
| --- | --- |
| **Operación** | **Cómo** |
| Editor de niveles de visualización | Haz clic en el menú Ver > Mostrar editor de niveles |
| Añadir un nivel | Haz clic en el icono «Añadir nivel» (+) |
| Eliminar un nivel | Haz clic en el icono de eliminar nivel (x) |
| Reordenar niveles | Haz clic en el icono «Reorganizar» en el encabezado de la fila y arrastra |
| Añadir una tabla al nivel | Arrastra la tabla desde el Explorador de proyectos al nivel |
| Añadir una columna al nivel | Despliega la tabla en el Explorador de proyectos y arrastra la columna al nivel correspondiente |
| Nombra un nivel | Haz clic en el campo «Nombre» situado en la parte superior del nivel e introduce el nombre |
| Volver a la vista del informe | Haz clic en el menú Ver > Mostrar documento |

**Normas sobre el contenido por niveles**

- Un nivel puede contener tablas O una sola columna, pero no ambas cosas
- Se pueden añadir varias tablas a un único nivel
- Si se añade una columna, no se pueden añadir otros elementos a ese nivel
- Varios niveles pueden utilizar columnas de la misma tabla
- Solo se pueden añadir tablas modeladas a los niveles
