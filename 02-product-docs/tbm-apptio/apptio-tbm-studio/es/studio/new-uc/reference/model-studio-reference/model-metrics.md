# Métricas de modelo

Las métricas son cálculos que definen las medidas para los informes. Representan valores como el coste, el presupuesto, las previsiones o los cálculos personalizados que se incorporan al modelo y se muestran en los informes.

## Métricas estándar

TBM Studio incluye varios tipos de métricas estándar que se utilizan habitualmente en los modelos de costes.

**Tipos métricos**

|  |  |  |
| --- | --- | --- |
| **Métrica** | **Finalidad** | **Fuentes habituales** |
| Coste | Gastos realmente incurridos | Datos reales de GL, facturas, amortización de activos |
| Presupuesto | Gastos previstos | Archivos presupuestarios, sistemas de planificación |
| Previsión | Gastos previstos para el futuro | Archivos de previsiones, modelos de proyección |
| Varianza | Diferencia entre métricas | Cálculo: Presupuesto - Coste |
| Volumen/Cantidad | Recuentos no monetarios | Número de usuarios, volumen de transacciones |
