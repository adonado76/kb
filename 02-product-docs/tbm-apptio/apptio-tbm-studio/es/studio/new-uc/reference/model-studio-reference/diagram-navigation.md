# Navegación por el diagrama

**Acciones de navegación**

|  |  |  |
| --- | --- | --- |
| **Acción** | **Cómo** | **Resultado** |
| Ver detalles del objeto | Haz clic en un nodo | Abre la vista de una sola tabla para ese objeto |
| Ir al destino | Haz clic en la flecha del controlador de asignación | Vuelve al objeto de origen |
| Seguimiento del flujo de costes | Haz clic en una línea de asignación | Resalta la trayectoria de los costes a lo largo del modelo |
| Detallar más | Haz clic en el menú > Desglose | Despliega el nodo para mostrar los valores de las columnas |
| Volver a la vista de tabla | Haz clic en «Cerrar» o en la flecha de retorno | Oculta el desglose y vuelve a la vista de objetos |
| Acercar | Rueda del ratón o controles de zoom | Acercar o alejar el diagrama |
| Desplazar | Haz clic y arrastra el fondo | Mueve la vista del diagrama |
