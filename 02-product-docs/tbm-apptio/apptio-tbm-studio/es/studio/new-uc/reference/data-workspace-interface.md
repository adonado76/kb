# Interfaz del espacio de trabajo de datos

## Explorador de proyectos

**Objetivo:** Navegar y gestionar todos los documentos del proyecto

**Tipos de documentos:**

- Tablas
- Tablas editables
- Métricas
- Perspectivas
- Tablas publicadas
- Informes
- Hora

## Panel «Pasos de transformación»

**Objetivo:** Ver y modificar el proceso de transformación de tablas

**Características:**

- Visualización paso a paso
- Incorporación de un escalón entre los escalones existentes
- Reordenación mediante arrastrar y soltar (con restricciones)
- Indicadores de error con recuento
