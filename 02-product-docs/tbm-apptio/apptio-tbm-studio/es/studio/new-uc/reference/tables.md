# Tablas

Las tablas son los contenedores de datos fundamentales en Data Studio. Cada tabla cuenta con un proceso de transformación que define cómo se procesan los datos desde la fuente hasta el resultado final.

## Tipos de tablas

|  |  |  |
| --- | --- | --- |
| **Tipo de tabla** | **Descripción** | **Caso de uso** |
| Tabla de transformaciones | Tabla estándar creada a partir de un archivo cargado o de una tabla existente | Tratamiento general de datos |
| Tabla editable | Tabla que admite la introducción manual de datos por parte del usuario | Ajustes presupuestarios, modificaciones manuales |
| Tabla editable con formato avanzado | Tabla editable basada en una tabla de transformación de origen | Modificaciones de superposición en los datos de origen |
| Tabla generada | Tabla creada a partir de una tabla existente con actualizaciones vinculadas | Conjuntos de datos derivados con actualización automática |
| Conjunto de datos maestros | Tabla de esquemas canónicos para el cumplimiento normativo de las aplicaciones | Maestro de fuentes de costes, Plan de cuentas |
| Tabla publicada | Tabla configurada para la exportación externa | Salida de datos hacia otros sistemas |
