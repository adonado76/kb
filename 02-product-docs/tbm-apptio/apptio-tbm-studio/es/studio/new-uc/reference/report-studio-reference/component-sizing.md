# Dimensionamiento de componentes

**Métodos para cambiar el tamaño**

|  |  |
| --- | --- |
| **Método** | **Cómo** |
| **Arrastrar los bordes** | Haz clic y arrastra cualquier borde o esquina del componente |
| **Cuadro de diálogo «Posición y tamaño»** | Haz clic con el botón derecho del ratón en el encabezado → Posición y tamaño → introduce los valores de ancho y alto |
| **Ajustar el tamaño del componente a pantalla completa** | Disponible en el modo de diseño de impresión para tablas |

**Ajuste automático del tamaño**

Algunos componentes pueden cambiar de tamaño automáticamente en función de su contenido:

**En el caso de las tablas:** las tablas pueden ajustarse automáticamente para mostrar todos los datos sin que aparezcan barras de desplazamiento. Configura esto en el cuadro de diálogo «Propiedades» de la tabla, en las opciones de tamaño.

**En el caso de los cuadros de grupo:** los cuadros de grupo pueden cambiar de tamaño dinámicamente para adaptarse a los componentes que contienen:

1. Selecciona el cuadro de grupo.
2. En la pestaña «Grupo», selecciona una de las opciones de ajuste automático: «Ajustar ancho automáticamente», «Ajustar alto automáticamente» o «Ajustar ambos automáticamente».

Nota: El redimensionamiento automático de los cuadros de grupo solo funciona en el modo Vista. En el modo de edición, el cuadro de grupo mantiene un tamaño fijo para facilitar la edición.
