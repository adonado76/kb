# Gráficos superpuestos (combinados)

Los gráficos superpuestos muestran dos series de datos en un solo gráfico, normalmente combinando columnas o barras con líneas. Esto permite comparar directamente indicadores relacionados, como el presupuesto frente a los datos reales o la capacidad frente a la utilización.

**Opciones de visualización de capas**

Cada capa de un gráfico superpuesto se puede mostrar de la siguiente manera:

- Columnas: barras individuales para cada valor.
- Columnas apiladas: barras segmentadas que muestran la composición.
- Líneas: puntos de datos conectados para representar tendencias.

**Creación de un gráfico superpuesto**

1. En la pestaña «Informe», haz clic en «Superponer».
2. Haz clic en la pestaña «Superposición» del panel de configuración.
3. Haz clic en «Editar capa base» y, a continuación, arrastra los campos a las áreas «Ejes» y «Valores».
4. Selecciona un tipo de visualización para la capa base.
5. Haz clic en «Editar capa 1» y repite la configuración para la segunda serie.
6. Añade filtros si es necesario para simplificar el gráfico.

**Eje compartido:** disponible en las propiedades del gráfico, esta opción hace que las escalas de los ejes izquierdo y derecho sean iguales, lo cual resulta útil a la hora de comparar valores de magnitud similar.
