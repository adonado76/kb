# Configuración de idioma

Los ajustes de localización se configuran principalmente a nivel de proyecto y determinan cómo se muestran los números, las fechas y las divisas en los informes.

|  |  |
| --- | --- |
| **Valor** | **Descripción** |
| Compactación de números | Controla la visualización de números grandes (miles, millones, miles de millones, billones) |
| Símbolos de divisas | Establece el símbolo monetario predeterminado para la visualización de datos financieros |
| Agrupación de números | Formato regional para los separadores de números (por ejemplo, la separación de unidades de 10 000 en japonés) |
| Tamaño de papel predeterminado | Establece el tamaño de papel predeterminado para las operaciones de impresión y exportación |

**Ubicación:** pestaña «Proyecto» > «Configuración del proyecto»
