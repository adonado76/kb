# Opciones de tamaño de pantalla

Al crear un nuevo informe, se selecciona el ancho de pantalla de destino que determina el área de diseño del informe.

|  |  |  |
| --- | --- | --- |
| **Tamaño de la pantalla** | **Ancho** | **Ideal para** |
| **Estándar** | 1024 píxeles | Monitores tradicionales, pantallas más pequeñas |
| **Pantalla panorámica** | 1440 píxeles | Monitores panorámicos, pantallas del salpicadero |

**Para configurar el tamaño de la pantalla durante la creación del informe:**

1. En la pestaña Inicio, haz clic en Nuevo y, a continuación, en Informe.
2. En el cuadro de diálogo «Crear nuevo informe», selecciona «Estándar» o «Pantalla panorámica» en las opciones de diseño.

Nota: Puede cambiar el tamaño de la pantalla después de crear el informe accediendo a las propiedades del mismo.
