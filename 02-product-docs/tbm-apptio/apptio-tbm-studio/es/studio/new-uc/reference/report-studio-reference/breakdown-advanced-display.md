# Desglose y opciones de visualización avanzadas

Las opciones avanzadas permiten controlar el comportamiento de los informes en contextos específicos, como la navegación por niveles, las vistas filtradas y las visualizaciones específicas de cada unidad.

|  |  |
| --- | --- |
| **Tipo de informe** | **Opciones de presentación** |
| Informe de ausencia de averías | Incluso cuando se muestra un fallo: se muestra incluso si se reciben datos sobre el fallo. Solo cuando no se muestra un desglose: se muestra únicamente con datos sin desglose. |
| Informe de averías | Incluso cuando no se muestra un desglose: se muestra incluso con datos que no incluyen un desglose. Solo cuando se muestra un desglose: se muestra únicamente cuando se introducen datos de desglose. |
| Informe filtrado | Solo cuando se filtra de esta manera: solo se muestra si el filtro coincide con la fuente. Incluso si se filtra de otra manera: se muestra independientemente de las diferencias entre los filtros. |
| Informe de la unidad | Para todas las unidades: Se aplica a todas las unidades. Solo para [unidad]: se aplica únicamente a una unidad específica del filtro. |
