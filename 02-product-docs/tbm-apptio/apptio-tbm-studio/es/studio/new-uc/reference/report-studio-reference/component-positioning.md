# Colocación de los componentes

**Posicionamiento manual (arrastrar y soltar)**

El método principal para colocar los componentes es arrastrar y soltar:

- **Mover un componente:** haz clic en el encabezado del componente y arrástralo a una nueva ubicación.
- **Cambiar el tamaño de un componente:** arrastra los bordes y las esquinas de la izquierda, la derecha o la parte inferior.
- **Posicionamiento preciso:** haz clic con el botón derecho del ratón en el encabezado del componente y selecciona «Posición y tamaño» para abrir el cuadro de diálogo «Establecer posición y tamaño».

El cuadro de diálogo «Establecer posición y tamaño» te permite especificar las coordenadas exactas:

- **Coordenada X:** posición horizontal desde el borde izquierdo (en píxeles)
- **Coordenada Y:** posición vertical desde el borde superior (en píxeles)
- **Ancho:** Ancho del componente
- **Altura:** Altura del componente

**Opciones de posicionamiento dinámico**

Para diseños adaptables, utiliza el menú «Dinámico» de la pestaña «Formato». Estas opciones determinan cómo se colocan los componentes con respecto a su contenedor (la superficie del informe, un cuadro de grupo o un componente con pestañas).

|  |  |
| --- | --- |
| **Opción** | **Descripción** |
| **Dinámico desactivado** | Desactiva el redimensionamiento y el posicionamiento dinámicos; el componente permanece en la posición en la que se ha colocado |
| **Centro** | Coloca el componente en el centro de su contenedor sin modificar su tamaño |
| **Centro y Crecimiento** | Centra el componente y permite que se amplíe simétricamente desde el centro |
| **Llenar correctamente** | Amplía el componente hasta el borde derecho de su contenedor |
| **Rellenar a la derecha y abajo** | Amplía el componente tanto hacia el borde derecho como hacia el inferior |
| **Atraca a la derecha** | Mueve el componente de modo que su borde derecho toque el borde derecho del contenedor |
| **Parte inferior del muelle** | Mueve el componente de modo que su borde inferior toque el borde inferior del contenedor |

**Para aplicar el posicionamiento dinámico:**

1. Selecciona el componente.
2. Abre el menú «Dinámico» en la pestaña «Formato».
3. Selecciona una de las opciones de posicionamiento.

Nota: Solo se puede aplicar una opción de posicionamiento dinámico a un componente en un momento dado.
