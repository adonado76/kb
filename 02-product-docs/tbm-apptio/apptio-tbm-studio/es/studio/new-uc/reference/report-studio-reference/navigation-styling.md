# Estilo de navegación

Un estilo coherente ayuda a los usuarios a reconocer los elementos de navegación y a comprender su finalidad. Aplica estas directrices de diseño para crear una experiencia de navegación profesional e intuitiva.

**Prácticas recomendadas para el diseño de botones**

- Utiliza colores coherentes para acciones similares (por ejemplo, azul para la navegación, verde para la aprobación)
- Utiliza tamaños de texto adecuados según la importancia de los botones (más grandes para las acciones principales)
- Añade descripciones emergentes útiles para explicar las funciones de los botones
- Agrupa los botones relacionados mediante cuadros de grupo

**Estilo de los enlaces**

Los enlaces de los informes en las tablas se muestran automáticamente en azul, siguiendo las convenciones estándar de la web. En los cuadros de texto HTML, puedes aplicar estilos CSS personalizados a los enlaces, aunque para ello es necesario tener conocimientos de HTML y CSS.

**Contenedores de navegación de cuadros de grupo**

Utiliza los cuadros de grupo para organizar varios elementos de navegación juntos. Los cuadros de grupo pueden incluir títulos, bordes y colores de fondo que separen visualmente la navegación del contenido del informe.
