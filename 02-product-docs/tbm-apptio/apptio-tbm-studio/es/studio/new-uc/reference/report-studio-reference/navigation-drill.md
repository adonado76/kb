# Navegación y análisis detallado

Los gráficos se pueden configurar para que los usuarios accedan a informes relacionados al hacer clic en los elementos del gráfico. Esta función de desglose permite a los usuarios explorar los datos desde el resumen hasta el nivel de detalle.

**Activar la navegación por gráficos**

1. Selecciona el gráfico.
2. En la pestaña «Ad Hoc», haz clic en «Drill» en la sección «Navegación».
3. Habilitar navegación: marca esta opción para que los elementos del gráfico sean interactivos.
4. Ir a: Selecciona el informe de destino en el menú desplegable.

**Opciones de navegación**

|  |  |
| --- | --- |
| **Opción** | **Descripción** |
| Activar navegación | Crea elementos de gráficos (barras, sectores, etc.) que funcionan como enlaces de navegación. |
| Abrir en ventana emergente | Muestra el informe de destino en una ventana emergente. Los usuarios pueden cerrarlo para volver al informe original. |
| Ir a | Selecciona el informe de destino de cualquier informe del proyecto. |
| Ejercicios por métrica | Activa el desglose de los valores que aparecen en los sectores del gráfico circular y en las barras del gráfico de barras. |
| Filtrar por la fila seleccionada | Pasa el valor seleccionado como filtro al informe de destino. |
| Filtrar según las selecciones del segmentador aplicado | Pasa las selecciones actuales del segmentador al informe de destino. |
| Incluir los filtros del informe actual | Aplica los filtros del informe de origen al informe de destino. |

Nota: Para que la navegación funcione con filtros, los datos de ambos informes deben estar vinculados a través del motor de inferencia.
