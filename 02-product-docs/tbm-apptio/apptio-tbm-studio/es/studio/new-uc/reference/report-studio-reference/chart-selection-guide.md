# Guía para la selección de gráficos

Utilice la siguiente matriz de decisión para seleccionar el tipo de gráfico más adecuado para sus datos y sus objetivos analíticos:

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| **Finalidad** | **Pocas categorías** | **Muchas categorías** | **Basado en el tiempo** | **Jerárquico** |
| Comparar precios | Columna | Barra | Columna, Línea | Mapa de árbol |
| Mostrar tendencia | Línea | Línea | Tendencia | N/D |
| Parte del todo | Tarta, Donut | Barras apiladas | Columna apilada | Mapa de árbol |
| Explicar la varianza | Cascada | Cascada | Cascada | N/D |
| Comparación multimétrica | Radial | Preformato | Preformato | N/D |
| Evolución de los KPI | Indicador | N/D | Componente de KPI | N/D |

**Buenas prácticas para la selección de gráficos**

- Utiliza gráficos circulares solo cuando quieras mostrar partes de un todo y cuando tengas seis categorías o menos.
- Es preferible utilizar gráficos de barras en lugar de gráficos circulares cuando sea importante realizar comparaciones precisas.
- Utiliza gráficos de líneas para datos de series temporales en los que la tendencia sea más importante que los valores individuales.
- Utiliza gráficos en cascada para explicar las diferencias o los cambios entre dos valores.
- Utiliza mapas de árbol cuando tengas datos jerárquicos y quieras mostrar el tamaño relativo y las proporciones.
- Utiliza gráficos superpuestos cuando compares dos métricas relacionadas que puedan tener escalas diferentes.
- Utiliza los gráficos de radar con moderación y solo cuando compares varios atributos entre unos pocos elementos.
