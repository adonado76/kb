# Filtros jerárquicos para el desglose

Los filtros jerárquicos ofrecen una experiencia similar a la navegación dentro de un mismo informe, lo que permite a los usuarios profundizar en las jerarquías de datos. A diferencia de la navegación de profundización hacia otros informes, los filtros jerárquicos filtran el informe actual con niveles de detalle cada vez mayores.

**Creación de un filtro jerárquico**

1. Crea un grupo de perspectivas personalizado con los campos ordenados de mayor a menor nivel
2. Cambia el nombre de los campos añadiendo prefijos numéricos para controlar el orden (por ejemplo, 01\_Type, 02\_SubType, 03\_OS )
3. En la pestaña Informe, haz clic en «Filtro de filas»
4. Arrastra el grupo de perspectiva al área «Dividir por»
