# Componentes del informe: Gráficos

Los gráficos convierten tus datos de costes en información visual que transmite datos financieros complejos de forma rápida y eficaz. TBM Studio ofrece un completo conjunto de tipos de gráficos optimizados para el análisis financiero en el ámbito de las tecnologías de la información, desde sencillos gráficos de barras que comparan los costes de los departamentos hasta sofisticados gráficos en cascada que explican las desviaciones presupuestarias.

## Descripción general de los tipos de gráficos

TBM Studio ofrece los siguientes tipos de gráficos, a los que se puede acceder desde la pestaña «Ad Hoc» tras crear un componente de gráfico:

|  |  |  |
| --- | --- | --- |
| **Tipo de gráfico** | **Variantes** | **Ideal para** |
| Barra | Barra horizontal, barra horizontal apilada | Comparación de categorías en paralelo |
| Columna | Columna vertical, columna apilada | Comparación de valores entre distintos periodos |
| Línea | Línea estándar, línea de tendencia | Mostrar tendencias a lo largo del tiempo |
| Gráfico circular | Estándar, Medio, Donut, Medio donut | Representación de las proporciones de un todo |
| Tendencia | Gráfico de tendencia lineal con proyecciones | Previsiones y análisis histórico |
| Indicador | Indicador de un solo valor | Indicadores de progreso de los KPI |
| Mapa de árbol | Rectángulos anidados | Datos jerárquicos y proporciones relativas |
| Cascada | Tabla de puentes con pasos | Explicación de las diferencias entre los valores |
| Radial | Radar relleno, radar con marcadores | Comparaciones multidimensionales |
| Preformato | Columna y fila combinadas | Comparación de dos series de datos relacionadas |
