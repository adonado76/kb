# Informes principales

Los informes maestros proporcionan una capa de plantilla (como un «fondo») que se puede aplicar a varios informes para mantener un diseño uniforme.

**Características principales:**

- Los componentes del informe maestro no se pueden editar desde el informe en el que se aplican.
- Los componentes añadidos a un informe se sitúan sobre la capa del informe maestro.
- Los cuadros de grupo de los informes maestros pierden su función de agrupación; utilícelos como bordes visuales.

**Para crear un informe maestro:**

1. Echa un vistazo al informe.
2. En la pestaña Informe, en el grupo Avanzadas, haz clic en Informe maestro.
3. Guarda el informe.
