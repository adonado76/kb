# Estilo de las tablas editables

El diseño visual ayuda a los usuarios a identificar las celdas editables y a comprender el estado de los datos.

**Indicadores visuales**

- **Nuevas filas:** se muestran resaltadas en naranja hasta que se guarden
- **Celdas modificadas:** visibles en la vista «Mostrar cambios»
- **Errores de validación:** celdas resaltadas en rojo con mensajes de error
- **Campos obligatorios:** indicador visual para las columnas obligatorias
- **Celdas bloqueadas:** las celdas que están siendo editadas por otros usuarios aparecen bloqueadas

**Formato de tablas**

Las opciones estándar de formato de tablas se aplican a las tablas editables:

- Ancho de las columnas, alineación y formato de los números
- Estilos y bordes de los encabezados
- Colores alternos en las filas
- Controles de ordenación y filtrado
