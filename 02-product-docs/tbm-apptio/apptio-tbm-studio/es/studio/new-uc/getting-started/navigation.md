# Áreas principales de navegación

## Barra de herramientas

La barra de herramientas situada en la parte superior de TBM Studio permite acceder rápidamente a las funciones principales y a la navegación.

*[Captura de pantalla: barra de herramientas con secciones etiquetadas]*

**Elementos clave de la barra de herramientas:**

|  |  |  |
| --- | --- | --- |
| **Elemento** | **Finalidad** | **Uso** |
| Menú **«Todos los productos»** | Accede a otros productos de Apptio | Haz clic para cambiar entre las aplicaciones de TBM Suite |
| Selector **de entorno** | Alternar entre los entornos de desarrollo, staging y producción | Selecciona en el menú desplegable para cambiar de entorno |
| Menú desplegable **de proyectos** | Acceder a la configuración actual del proyecto | Haz clic para ver los detalles del proyecto, la configuración y los permisos |
| Botón **de inicio** | Volver al panel de control principal | Acceso rápido al resumen del proyecto y a las recomendaciones |
| Menú **de ayuda** | Acceder a la documentación y al servicio de asistencia | Haga clic aquí para obtener ayuda contextual y recursos |
| Menú **de usuario** | Configuración y preferencias de la cuenta | Acceder al perfil, a las preferencias y cerrar sesión |

Consejo: Pasa el cursor por encima de los iconos de la barra de herramientas para ver las descripciones emergentes que explican su función.

## Panel de navegación

El panel de navegación de la izquierda permite acceder a todos los estudios principales y a los componentes del proyecto.

**Secciones principales de navegación:**

- **Data Studio** — Carga, transforma y gestiona tus fuentes de datos
- **Model Studio** : crea modelos de costes y configura las asignaciones
- **Report Studio** : crea y personaliza informes
- **Administración** : gestionar usuarios, permisos y configuración de proyectos

**Para navegar:**

1. Haz clic en el nombre de un estudio para ampliar su sección
2. Haz clic en un elemento concreto para abrirlo en el espacio de trabajo principal
3. Utiliza la barra de búsqueda de la parte superior para encontrar rápidamente los artículos

Nota: El panel de navegación se puede contraer para disponer de más espacio de trabajo haciendo clic en el icono de contraer (◀).

## Panel de proyectos

El panel «Proyectos» muestra todos los proyectos a los que tienes acceso dentro de tu dominio.

**Lo que ves:**

- Proyecto activo actual (resaltado)
- Otros proyectos disponibles
- Indicadores del estado del proyecto

**Para cambiar de proyecto:**

1. Haz clic en el menú desplegable **«Proyectos»** de la barra de herramientas
2. Selecciona el proyecto con el que quieras trabajar
3. La interfaz se actualiza para mostrar el proyecto seleccionado

Advertencia: los cambios no guardados en el proyecto actual podrían perderse al cambiar de proyecto. Guarda siempre tu trabajo antes de cambiar de pantalla.

## Panel del Explorador de proyectos

El Explorador de proyectos muestra la estructura del proyecto actual con todos los documentos organizados por tipo.

**Tipos de documentos habituales:**

- **Tablas** : tablas de datos y tablas de transformación
- **Modelos** : modelos de costes y configuraciones de asignación
- **Métricas** : métricas calculadas y modeladas
- **Informes** : informes publicados y paneles de control
- **Tablas editables** — Formularios de introducción de datos

**Iconos de estado del documento:**

|  |  |  |
| --- | --- | --- |
| **Icono** | **Estado** | **Significado** |
| 🔒 | Bloqueada | El documento está en tu poder |
| 🔓 | Desbloqueado | El documento está disponible para su edición |
| 🔄 | Actualizado | Hay una versión más reciente disponible (haz clic en «Actualizar espacio de trabajo») |
| ⏳ | Calculando | El documento se está calculando tras el registro |
| ⚠️ | Error | El documento contiene errores o advertencias |

Consejo: Haz clic con el botón derecho del ratón en cualquier documento del Explorador de proyectos para acceder a acciones rápidas como «Extraer», «Incorporar», «Renombrar» o «Eliminar».
