# Recomendaciones sobre recursos no utilizados

TBM Studio identifica los recursos no utilizados que se pueden eliminar para mejorar el rendimiento. Accede a ellas a través de la pestaña «Recomendaciones» y sigue el proceso de resolución de problemas guiado.

## Métricas no utilizadas

Métricas calculadas que no se utilizan en otras métricas ni en informes. Ve a TBM Studio > Recomendaciones > Métricas no utilizadas. Selecciona «Solucionar todos los problemas detectados» y sigue el proceso guiado para eliminar las métricas que no se utilizan.

## Asignaciones no utilizadas (asignaciones de cero unidades)

Asignaciones que generan un flujo de costes nulo. Esto puede indicar errores de configuración o una lógica de asignación obsoleta. Siga el flujo de trabajo «Recomendaciones» para revisar y eliminar las asignaciones no utilizadas por objeto de modelo.

## Informes no utilizados (BETA)

Informes a los que no se ha accedido recientemente. Desactiva el cálculo automático antes de utilizar este flujo de trabajo para realizar cambios masivos. Desactivar los informes que no se utilizan reduce la carga de cálculo.
