# Informar de problemas con Report Studio

## El informe no se carga o tarda en cargarse

**Síntomas:**

- El informe aparece en blanco o tarda mucho en cargarse
- El navegador agota el tiempo de espera al abrir el informe
- El informe se carga, pero tarda varios minutos

**Soluciones:**

1. **Comprueba el estado del cálculo** : ve a «Build» > «Cola de cálculo». Comprueba que el entorno haya terminado de realizar los cálculos.
2. **Actualiza el navegador y borra la caché** : actualiza la página del navegador. Si el problema persiste, prueba a borrar la caché del navegador.
3. **Verificar los datos subyacentes** : comprueba que las tablas de origen y las transformaciones contengan datos válidos para el periodo de tiempo seleccionado.
4. **Prueba en el entorno de desarrollo** : cambia al espacio de trabajo de desarrollo y comprueba si el mismo informe se carga correctamente allí.
