# Fórmulas personalizadas

Las fórmulas personalizadas (también denominadas dimensiones de fórmula) le permiten definir nuevas dimensiones calculadas utilizando campos existentes en su modelo de datos. Esto permite realizar análisis más profundos y obtener información más detallada sin necesidad de realizar cambios en el conjunto de datos o el esquema subyacentes.

## ¿Por qué utilizar Formula Dimensions?

Las dimensiones de la fórmula te ayudan a:

- Derivar nuevas dimensiones a partir de datos existentes
- Realice cálculos directamente dentro de la experiencia del informe
- Explora los datos con mayor flexibilidad sin depender de los cambios en el modelo de datos

## Cómo funcionan las dimensiones de Fórmula

- En lugar de seleccionar solo dimensiones predefinidas del esquema subyacente, puede
  - Defina una fórmula personalizada utilizando los campos y funciones compatibles
  - Ahora puedes añadir descripciones a las fórmulas personalizadas para proporcionar más información sobre la lógica de la fórmula o su uso previsto
  - Guarde la fórmula como una dimensión de fórmula reutilizable
- Una vez creadas, las dimensiones de fórmula se comportan como dimensiones estándar y se pueden utilizar para agrupar, segmentar, filtrar y visualizar

## Disponibilidad en todos los componentes y visualizaciones

Las columnas de fórmulas son compatibles con todos los componentes y visualizaciones.
