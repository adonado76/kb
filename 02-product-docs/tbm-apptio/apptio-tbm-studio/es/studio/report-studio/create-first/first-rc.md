# Crea tu primera colección de informes

Este tema le ayuda a crear una colección de informes para organizar informes relacionados y controlar el acceso de los usuarios.

1. Vaya a la pestaña Colecciones de informes
   1. En la página de inicio, seleccione la pestaña Colecciones de informes.
   2. Haga clic en el botón **Nuevo** y seleccione **Recopilación de informes.**
2. Introducir detalles de la colección
   - Mostrar colección a los usuarios finales: active esta opción para que esté disponible para los usuarios finales en el nuevo visor de informes.
   - Nombre de la colección: introduzca un nombre significativo para la colección
   - Descripción (opcional): añada una breve descripción para explicar el propósito de la colección.
   - Visible para: esta propiedad controla quién puede ver y acceder al informe.
     - Todos: el informe es visible para todos los usuarios con acceso al área de informes.
     - Solo yo: el informe solo es visible para usted. Útil para borradores, experimentación o análisis personal.
     - Seleccionar roles: el informe solo es visible para los usuarios asignados a los roles seleccionados. Los roles se seleccionan de una lista desplegable. Habilita el control de acceso basado en roles para los informes.

   ![Imagen de la nueva pantalla de recopilación de informes](../../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/nrs-rc.png)
3. Añadir informes a la colección
   - Seleccione uno o más informes para incluir en la colección
   - Puede añadir informes durante la creación o actualizar la colección más tarde.
   - Puede controlar la visibilidad del usuario final en informes individuales
   - Haga clic en Crear

   ![Imagen de la lista de informes en una colección de informes](../../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/nrs-rc1.png)
