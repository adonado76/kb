# Exportar informe a PDF

La función de exportación a PDF del visor de informes te permite exportar el informe que estás viendo actualmente a un archivo PDF. Esto facilita la descarga y el intercambio.

**Cuándo se utiliza**

Utiliza esta opción cuando quieras:

- Compartir informes con las partes interesadas
- Haz una captura de pantalla del informe para revisiones o presentaciones
- Archivar informes con fines de mantenimiento de registros o de cumplimiento normativo

**Exportar un informe a PDF**

Para exportar un informe desde el visor de informes:

1. Abre el informe en el visor de informes
2. Haz clic en **«Exportar a PDF»** en la barra de acciones situada en la parte superior del informe.
3. El sistema genera y descarga un archivo PDF de la vista actual del informe.
