# Crear un informe

Siga estos pasos para crear y elaborar su informe:

1. **Crear un informe o colección**
   1. Haga clic en **Nuevo** y elija crear un **Informe** o una **Colección de Informes**.

      ![nueva ventana emergente del estudio de informes](../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/new-rs-popup.png)
   2. Introduzca el nombre y la descripción del informe y haga clic en el botón **Crear**. Se abrirá un lienzo de informe en blanco

      ![lienzo en blanco](../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/rs-blank-canvas.png)

      Nota: También puede crear un nuevo informe utilizando el atajo de teclado *Ctrl+Mayús+O*.
2. **Explorar el diseño del lienzo**
   1. **Barra de herramientas (arriba):** Contiene visualizaciones y componentes de informe que puede añadir.
   2. **Panel de capas (izquierda)** : Muestra la jerarquía de los componentes de su informe. Desde aquí, puede copiar, pegar, reordenar, eliminar o cambiar la pertenencia a un grupo.
   3. **Explorador de dimensiones (derecha)** : Tablas de listas, Tablas editables, Métricas y Tiempo. Utilícelo para arrastrar y soltar dimensiones en configuraciones de componentes.
3. **Añadir componentes o visualizaciones**
   1. Arrastre un componente/visualización de la barra de herramientas al lienzo.
   2. Cada componente añadido aparece en el Panel de **capas**.
4. **Configure su componente**
   1. Una vez añadido un componente, aparecen 2 paneles en la parte derecha, junto al Explorador de dimensiones: ![paneles de datos y formatos](../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/rs-df-panels.png)
      1. **Panel de datos:** Seleccione un objeto modelo y añada dimensiones de datos.
      2. **Panel de formato** : Personaliza propiedades como el color, la fuente, el tamaño y otras opciones de estilo.
      3. Propiedades generales

         |  |  |
         | --- | --- |
         | Título | Deslice el conmutador si desea añadir un título al componente. En caso afirmativo, puede editar lo siguiente: - Título - Tamaño y estilo de letra (negrita, cursiva, subrayado) - Color del texto del título (con opción para restablecer el color) |
         | Subtítulo | Deslice el conmutador si desea añadir un subtítulo al componente. En caso afirmativo, puede editar lo siguiente: - Subtítulo - Tamaño y estilo de letra (negrita, cursiva, subrayado) - Color del texto de los subtítulos (con opción para restablecer el color) |
         | Ayuda | Proporcionar un consejo de herramienta para el componente. Al proporcionar esta información sobre herramientas, el icono de ayuda aparecerá junto al componente. |
         | Alt text | Proporcionar el texto Alt, que es el texto descriptivo de las imágenes y gráficos de la interfaz de usuario, proporcionando una alternativa de texto para los usuarios con lectores de pantalla o si la imagen no se carga. |
         | Posición y tamaño | Edite las siguientes propiedades, según sea necesario: - Ancho - Altura - Posición (ejes x e y) |
         | Estilo | Edite las siguientes propiedades, según sea necesario: - Color de texto - Color de fondo - Color de borde - Tamaño del borde |
5. **Guarda tu trabajo**
   1. **Guardado automático:** Tus cambios se guardan automáticamente mientras trabajas.
   2. **Guardar explícitamente:** También puedes hacer clic en el botón **Guardar** de la parte superior (junto a las migas de pan) para guardar manualmente en cualquier momento.
