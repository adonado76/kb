# Recursos de soporte

## Acerca de esta tarea

El nuevo estudio de informes ofrece una variedad de recursos de asistencia para ayudarle a aprender, resolver problemas y mantenerse informado. Estos recursos le conectan con la documentación, la formación y la comunidad en Apptio general para que pueda sacar el máximo partido al producto.

- [Notas de la versión](../whats-new-nrs.html) : Manténgase al día con las últimas mejoras. Ver todas las publicaciones *de Novedades* para estar al tanto de los cambios en los productos y las nuevas funciones.
- [Vídeos y formación:](https://education.apptio.com "(se abre en una pestaña o una ventana nueva)") acceda a recursos de formación a su propio ritmo, incluyendo tutoriales en vídeo, grabaciones de seminarios web y guías paso a paso para desarrollar sus habilidades.
- [Preguntas y](https://community.ibm.com/community/user/groups/community-home?communitykey=4100dfb8-fc23-4203-83c7-019253cf7c0b "(se abre en una pestaña o una ventana nueva)") respuestas de la comunidad: explore las preguntas y respuestas de otros Apptio usuarios o publique las suyas propias para obtener ayuda de compañeros y expertos.
- [Ideas para nuevas](https://ideas.ibm.com/ "(se abre en una pestaña o una ventana nueva)") funciones: ayude a dar forma al producto enviando solicitudes de nuevas funciones o votando las ideas compartidas por la comunidad.
- [IBM Soporte técnico:](https://www.ibm.com/mysupport/s/?language=en_US "(se abre en una pestaña o una ventana nueva)") Utilice el portal de soporte técnico para enviar una solicitud de asistencia.
