# Nuevo informe: Descripción general de Studio

El nuevo **IBM Apptio Report Studio**, diseñado con tecnología Carbon, ofrece una experiencia de generación de informes modernizada, intuitiva y más potente. Está diseñado para ayudar a los TBMA y a los usuarios empresariales a crear informes más rápidamente, con menos complejidad y una curva de aprendizaje más corta. En comparación con Classic Report Studio, esta versión ofrece:

- Más rapidez en la obtención de valor: Cree y visualice informes más rápidamente, ayudando a los equipos a tomar decisiones más rápido.
- Interfaz moderna e intuitiva: Un diseño de arrastrar y soltar fácil de aprender y usar.
- Accesibilidad mejorada: Optimizado para varios tamaños de pantalla y dispositivos.
- Creación de informes racionalizada: Reducción del tiempo de configuración con valores predeterminados inteligentes y componentes reutilizables.
- Herramientas flexibles de diseño y visualización: Elabore informes perfectos con numerosas opciones de gráficos y controles de diseño.

## Qué se puede hacer con el nuevo Report Studio

En la nueva experiencia, puedes:

- Cree informes utilizando componentes y visualizaciones configurables
- Organiza los informes en colecciones para facilitar su búsqueda
- Interactúa con los informes utilizando filtros y selectores
- Ver datos en varias monedas
- Exportar informes a PDF
- Exportar datos de la tabla para su posterior análisis

## ¿A quién va dirigido New Report Studio?

Report Studio está diseñado para dos funciones principales de usuario:

**Administradores (autores de informes)**

Los administradores crean y gestionan informes. Configurar componentes, visualizaciones y diseños para ofrecer información significativa a su público.

**Usuarios finales (lectores de informes)**

Los usuarios finales consumen los informes creados por los administradores. Se centran en comprender los datos y exportarlos cuando es necesario.

Para obtener más información, consulte [Funciones y permisos de usuario](user-role-perm.html "El nuevo Report Studio admite diferentes roles de usuario para garantizar el nivel adecuado de acceso para crear, gestionar y ver informes. Esta página explica las funciones disponibles y lo que puede hacer cada una de ellas.").
