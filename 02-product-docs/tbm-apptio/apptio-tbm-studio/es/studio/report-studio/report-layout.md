# Nuevo diseño de Report Studio

A continuación se muestra el diseño de un informe nuevo/en blanco:

![](../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/rs-report-layout.png)

|  |  |  |
| --- | --- | --- |
| 1 | Acercar | Ajusta la vista del lienzo hacia dentro o hacia fuera. |
| 2 | Informar del estado del check-in/check-out | Muestra el estado de entrada/salida del informe en contexto. |
| 3 | Guardar | Guarda el informe |
| 4 | Panel de datos | Seleccione un objeto modelo y añada dimensiones de datos. |
| 5 | Panel de formato | Personalice propiedades como el color, la fuente, el tamaño y otras opciones de estilo. |
| 6 | Tablas | Represente las fuentes de datos estructurados de las que puede extraer dimensiones. |
| 7 | Tablas editables | Represente las fuentes de datos estructurados de las que puede extraer dimensiones. |
| 8 | Métricas | Representan valores numéricos o medidas que pueden utilizarse para el análisis. |
| 9 | Dimensiones temporales | Dimensiones de fecha y hora que permiten analizar los datos por periodos (diario, mensual, trimestral, anual) |
| 10 | Navegación con migas de pan | Le permite saber dónde se encuentra exactamente en la jerarquía del informe. |
| 5 | Menú de acciones del informe | Proporciona acciones clave como Registrar, Revertir cambios, Deshacer, Rehacer, Copiar y Pegar, junto con controles de diseño como cuadrículas, reglas, alineación y opciones de distribución. |
| 6 | Componentes | Alberga los diferentes componentes del informe utilizados para construirlo. Incluye Compact Slicers, Quick Pivots, Column Pickers, HTML, Group, Tabs, etc. |
| 13 | Visualizaciones | Enumera todas las visualizaciones de datos disponibles. Incluye tablas, tablas editables, gráficos de barras, gráficos de columnas, gráficos de líneas, gráficos circulares, etc. |
| 14 | Panel Capas | Muestra todos los componentes del informe en la jerarquía del informe, permitiéndole copiarlos, pegarlos, reordenarlos, eliminarlos o agruparlos. |
