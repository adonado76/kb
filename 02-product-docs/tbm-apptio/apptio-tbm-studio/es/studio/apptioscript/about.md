# Acerca de ApptioScript

ApptioScript es el lenguaje de scripting que se utiliza para crear las aplicaciones empresariales interactivas sobre la plataforma Apptio.

ApptioScript se puede utilizar para realizar muchas operaciones, como editar tablas, enviar correo electrónico, estados de transición en el flujo de trabajo o programar lo que sucede cuando el usuario hace clic en un botón.
