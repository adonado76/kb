# Validación de datos para tablas editables

Para activar la validación de datos en theUI,, vaya a la pestaña **Proyecto** > **[**Activar funciones**](../../admin/enable-features.htm "(se abre en una pestaña o una ventana nueva)")** > y seleccione la opción **Activar paso de validación para tabla editable (Beta)**.

![](../../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/enable-validation-step.png)

Cuando introduzca datos corruptos o erróneos en una tabla editable, aparecerá un mensaje de advertencia o error cerca del valor correspondiente.

![](../../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/data-validation.png)

![](../../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/data-vld-msg.png)

A medida que rectifique los errores, las filas validadas se desplazarán hacia abajo, y las filas que aún tengan errores aparecerán en la parte superior.

Puede guardar y registrar el informe a pesar de estos errores.
