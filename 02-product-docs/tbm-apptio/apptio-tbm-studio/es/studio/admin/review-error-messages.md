# Revisar los mensajes de error

**Se aplica a** : TBM Studio 12.0 y posteriores.

Para ver los mensajes de error, debe tener la función predeterminada de **usuario avanzado** o **administrador**, o una función personalizada con los permisos adecuados. Para ver los mensajes de error, haga clic en **Errores** en la pestaña Proyecto. La aplicación muestra los errores en la ventana que aparece en la siguiente imagen:
