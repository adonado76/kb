# Cambiar el proyecto por defecto

◆ Se aplica a: Apptio TBM Studio v12.3.1 y posteriores.

Especifique el proyecto por defecto de su dominio.

1. En la pestaña **Proyecto**, en el grupo **Dominio**, haga clic en **Configuración del dominio**. Aparece el cuadro de diálogo Configuración de inquilinos.
2. En el campo **Proyecto por defecto**, seleccione el proyecto que desee y haga clic en **Aceptar**.
