# Restaurar proyectos

**Se aplica a** : TBM Studio 12.0 y posteriores

Para eliminar o restaurar un proyecto, póngase en contacto con el servicio de asistencia Apptio indicando el nombre del proyecto. Los nombres de proyecto distinguen entre mayúsculas y minúsculas, así que asegúrese de que utiliza las mayúsculas y minúsculas correctas para el nombre de su proyecto.
