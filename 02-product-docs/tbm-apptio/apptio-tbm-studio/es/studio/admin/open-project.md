# Abrir un proyecto

**Se aplica a** : TBM Studio 12.0 y posteriores. Puedes crear muchos proyectos en la aplicación, pero sólo puedes tener abierto un proyecto a la vez. Para abrir un proyecto, abra el menú Proyecto en la cabecera global y haga clic en el nombre del proyecto.

![](../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/studio-open-project.jpg)

Todos los usuarios con funciones distintas de Sólo ver pueden seleccionar un proyecto en el menú Proyecto.

Nota: Si se le asigna un rol de Sólo Ver, tendrá acceso a un único proyecto.
