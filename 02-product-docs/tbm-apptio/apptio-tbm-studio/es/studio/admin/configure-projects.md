# Configurar proyectos

**Se aplica a** : TBM Studio 12.0 y posteriores. Este artículo describe las tareas esenciales que puede realizar para configurar proyectos.

Hay dos tipos de proyectos:

- **Personalizado (Plataforma)** : Utilice un proyecto en blanco para construir los conjuntos de datos, métricas, modelos e informes.
- **Costing Standard**: Utilice un proyecto de Costing Standard para proporcionar un servicio de TI con modelización de costes, análisis de optimización de costes y capacidades de presupuestación y previsión. Los resultados de un proyecto de cálculo del coste de los servicios suelen utilizarse para calcular el coste de los servicios en la aplicación Facturación.
