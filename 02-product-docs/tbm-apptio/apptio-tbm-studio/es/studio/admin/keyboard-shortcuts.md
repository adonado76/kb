# Atajos del teclado

◆ Se aplica a: TBM Studio 12.0 y posteriores.

Están disponibles los siguientes atajos de teclado.

- **Ctrl+Alt+T** : entrar y salir del modo Estudio ([más información](navigate-tbm-studio.html "(se abre en una pestaña o una ventana nueva)") )
- **Ctrl+F1** - Conmutar el modo pestañas/cinta de TBM Studio
- **Alt+N** - Crear un nuevo documento
- **Alt+O** - Consultar un documento ([más información](bp-check-out.html "(se abre en una pestaña o una ventana nueva)") )
- **Ctrl+S, Meta+S** - Guardar
- **Alt+K** - Registrar un documento ([más información](bp-check-out.html "(se abre en una pestaña o una ventana nueva)") )
- **Ctrl+Alt+Z** : revertir un documento ([más información](bp-check-out.html "(se abre en una pestaña o una ventana nueva)") )
- **Shift+F9** - Actualizar un proyecto
- **F9** - Actualizar un documento
- **Alt+Escape, Meta+Escape** - Cerrar pestaña
- **Ctrl+Z** - Deshace todos los cambios desde el último guardado ([más información](bp-check-out.html "(se abre en una pestaña o una ventana nueva)") )
- **Ctrl+Y** - Rehacer ([más información](bp-check-out.html "(se abre en una pestaña o una ventana nueva)") )
