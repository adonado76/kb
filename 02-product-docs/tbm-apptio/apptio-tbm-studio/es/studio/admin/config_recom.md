# Recomendaciones de configuración

TBM Studio proporciona recomendaciones de configuración para que sepa si hay un problema potencial con las tablas de datos.

Para ver las recomendaciones, vaya a **Inicio** > **Recomendaciones**.

![](../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/config-recom.png)

Aparecerá el panel Recomendaciones de configuración. Si hay un gran número de recomendaciones, puede utilizar las opciones de filtro para restringir qué recomendaciones se muestran. El panel Recomendaciones de configuración muestra el número de instancias detectadas de cada problema junto a las recomendaciones. En el ejemplo siguiente, hay 2 instancias de una tabla en las que se ha utilizado una operación de unión de modelo heredado, por lo que el panel Recomendaciones de configuración muestra Operación de unión de modelo heredado utilizada. (2) seguido de los lugares donde se detecta el problema.

![](../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/config_rec_pane.png)

Consejo: Consulte la lista de recomendaciones y acciones para [Resolver los problemas identificados](../troubleshooting/studio-troubleshooting-about.htm "(se abre en una pestaña o una ventana nueva)").

El botón Solucionar todos los problemas identificados ahora se llama **Solucionar todos los problemas identificados**. Al hacer clic en el botón se iniciará un flujo de trabajo de varios pasos que involucra al administrador en la resolución de problemas y el proceso automatizado de "solución".

![](../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/fixit-button.png)
