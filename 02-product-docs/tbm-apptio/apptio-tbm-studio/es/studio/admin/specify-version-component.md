# Especifique una versión para las actualizaciones de componentes

◆ Se aplica a: TBM Studio 12.3.1 y posteriores. Al actualizar aplicaciones dentro de TBM Studio, puede especificar qué versión utilizar para las actualizaciones de componentes. Esta configuración se aplica a todas las actualizaciones de componentes y a las nuevas instalaciones de componentes.

Para configurar la versión de su componente:

1. En la pestaña **Proyecto**, haga clic en **Configuración del proyecto.** Se abrirá el cuadro de diálogo **Editar configuración del proyecto**.
2. En el campo **Versión del** componente, seleccione la versión del componente que desee. El valor por defecto es la versión del componente que se aplicó cuando se creó el proyecto o la última vez que se modificó la **Configuración del proyecto**.

## Trucos y consejos

Los iconos de los componentes instalados te lo indican:

- Si un componente instalado es anterior a la versión de la aplicación del proyecto actual y existe una actualización disponible.![imagen](../../resources/images/studio_images/studioimages/studio_specify%20version%20comp_installed.png)
- Si un componente instalado es más reciente que la versión de la aplicación del proyecto actual.![imagen](../../resources/images/studio_images/studioimages/studio_version%20component_upgrade.png)
- Puede señalar cualquier componente para ver su número de versión, que le indica su versión de contenido:![imagen](../../resources/images/studio_images/studioimages/studio_component%20config_3.png)
