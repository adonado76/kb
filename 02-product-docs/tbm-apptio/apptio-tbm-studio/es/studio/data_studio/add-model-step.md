# Datos del modelo

**Se aplica a** : TBM Studio 12.0 y posteriores

Cuando desee añadir una tabla a un modelo, añada un paso de modelo a la cadena de transformación de tablas. Al hacer clic en el paso del modelo, aparece una vista de tabla única del modelo, como se muestra a continuación. En esta vista, puede añadir controladores a la tabla y asignar valor de la tabla a otras tablas. Para más información sobre el modelado de datos, consulte [Model Studio: Calcular costes y otras métricas](../model%20studio/models-chapter-title-page.htm "(se abre en una pestaña o una ventana nueva)").
