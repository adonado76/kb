# Cargar datos

**Se aplica a** : TBM Studio 12.0 y posteriores

Cuando se cargan datos, se cargan los datos en una tabla. Puede cargar datos utilizando uno de los siguientes métodos:

- **Carga de archivos** : Seleccione un archivo de una ubicación local o de red en uno de los siguientes formatos: Excel (xls, xlsx),.csv, XML, Zip ([¿Cómo?](upload-data-file.html "Se aplica a: TBM Studio 12.0 y posteriores. TBM Studio acepta datos de archivos planos en formatos separados por comas, tabulaciones, tuberías, tildes, dos puntos y punto y coma. Antes de poder cargar datos, debe crear una tabla en la que se cargarán los datos, si no existe ya.") ).
- **Introducir manualmente** : Introducir los datos manualmente en una tabla ([¿Cómo?](enter-data-manually.html "Se aplica a: TBM Studio R12.0 y posteriores") ).
- **Tabla generada** : Crear una nueva tabla a partir de una tabla existente en el proyecto actual. La nueva tabla NO estará vinculada a la tabla existente.
- **Tabla existente** : Crear una nueva tabla a partir de una tabla existente en el proyecto actual. La nueva tabla se vinculará a la tabla existente.
- **Datalink (Classic) Conector** : Importe una tabla utilizando un conector predefinido de Datalink (Classic) .
- **Tabla importada** : Importe los datos de un proyecto Costing Standard existente.

Antes de poder cargar datos, debe crear una tabla que los reciba ([¿Cómo?](create-table.html "Se aplica a: TBM Studio 12.0 y posteriores") ).
