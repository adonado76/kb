# Visualizações salvas

Aplica-se a : 12.11.9 e posterior para todos os produtos.

Esse recurso permite selecionar um conjunto de parâmetros ad hoc e salvá-los com um nome específico, para que você possa reutilizar facilmente os relatórios com os parâmetros selecionados anteriormente. A opção Saved views (Visualizações salvas) é ativada por padrão e aparecerá no lado direito da interface do usuário.

![img](../../../../03-media/apptio-tbm-studio/resources/images/studio_images/sv-1.jpg)

No primeiro login, as exibições salvas não terão nenhuma exibição selecionada.

Salvando uma visualização

Abra qualquer relatório e personalize-o com qualquer uma das opções abaixo:

- Fatiador compacto
- Fatiador de fileiras
- Seletor de coluna - Seletor de valor, Seletor de métrica, Seletor de tempo
- Pivô rápido
- Guias
- Tabelas - Barra de pesquisa, ordem das colunas

![img](../../../../03-media/apptio-tbm-studio/resources/images/studio_images/sv-main.jpg)

No canto superior direito, expanda o ícone Exibir e escolha a opção Salvar exibição. A seguinte janela pop-up é exibida:

![img](../../../../03-media/apptio-tbm-studio/resources/images/studio_images/sv-2.jpg)

Insira valores nos campos View Name e View Description. Para definir isso como sua página de destino padrão, marque a opção Definir isso como exibição de destino para este relatório.

![img](../../../../03-media/apptio-tbm-studio/resources/images/studio_images/sv-3.jpg)

Selecione Salvar. Uma mensagem de confirmação é exibida na parte inferior da página " *<nome da visualização salva> criada com sucesso!* ".

Expanda o ícone Visualizar - a visualização salva aparece na parte inferior, conforme mostrado:

![img](../../../../03-media/apptio-tbm-studio/resources/images/studio_images/sv-4.jpg)

O ícone ![img](../../../../03-media/apptio-tbm-studio/resources/images/studio_images/homeicon.jpg) indica que a visualização salva está definida como a página de destino padrão. Se você sair e fizer login novamente na página de relatórios posteriormente, a visualização salva aparecerá como a página de destino.

O ícone ![img](../../../../03-media/apptio-tbm-studio/resources/images/studio_images/current-view.jpg) indica uma "visualização atual".

O ícone de ponto azul ![img](../../../../03-media/apptio-tbm-studio/resources/images/studio_images/blue-dot-icon.jpg) indica que estamos atualmente em uma visualização.

Observação: Você só pode criar até 5 visualizações por relatório.

Editar uma visualização

No menu Views, selecione ![img](../../../../03-media/apptio-tbm-studio/resources/images/studio_images/more.jpg) para uma visualização salva e, em seguida, selecione Edit view.

![img](../../../../03-media/apptio-tbm-studio/resources/images/studio_images/sv-edit-delete.jpg)

O pop-up Editar exibição é exibido como mostrado

![img](../../../../03-media/apptio-tbm-studio/resources/images/studio_images/sv-edit.jpg)

Faça as alterações apropriadas e selecione Salvar. Uma mensagem de confirmação " *<nome da exibição salva> atualizada com sucesso!* " aparece.

Observação: Para marcar outra visualização salva como a página de destino padrão, marque a caixa de seleção Definir esta como visualização de destino para este relatório.

Excluir uma visualização

No menu Views, selecione ![img](../../../../03-media/apptio-tbm-studio/resources/images/studio_images/more.jpg) para uma visualização salva e, em seguida, selecione Delete View.

Uma mensagem de confirmação " *<nome da exibição salva> excluída com sucesso!* " aparece.

Salvar como

Essa opção serve para criar uma nova exibição salva, com base nas modificações de uma existente.

Abra uma exibição salva e personalize alguns filtros. No menu Visualizações, selecione a opção Salvar como.

![img](../../../../03-media/apptio-tbm-studio/resources/images/studio_images/sv-svas-2.jpg)

Digite um nome exclusivo para a nova exibição e selecione Salvar.

Observação: Se você inserir um nome existente, será exibida a mensagem de erro " *Já existe uma visualização com esse nome* ".

![img](../../../../03-media/apptio-tbm-studio/resources/images/studio_images/sv-svas-3.jpg)

Uma mensagem de confirmação " *<nome da exibição salva> criada com sucesso!* " aparece. A visualização recém-criada aparece no menu, conforme mostrado.

![img](../../../../03-media/apptio-tbm-studio/resources/images/studio_images/sv-svas-4.jpg)

Reconfigurar

Você pode redefinir usando qualquer uma das opções abaixo:

- Selecione Reset All Slicers (Redefinir todos os fatiadores) para limpar todos os fatiadores selecionados.
- Selecione Reset All Global Filters (Redefinir todos os filtros globais) para limpar todos os filtros globais selecionados.
- Selecione Reset to Default Report (Redefinir para relatório padrão ) para limpar a visualização atual e substituí-la pela visualização padrão.

Observação: Os seguintes parâmetros não são considerados nas configurações de exibições salvas - Classificação de linhas, Gráficos, Filtros globais, Intervalo de datas (usa o "Período de tempo padrão" definido em Configurações de tempo), Ambiente e Filial.
