# Recomendações de configuração

TBM Studio fornece recomendações de configuração para que você saiba se há um possível problema com as tabelas de dados.

Para ver recomendações, acesse **Início** > **Recomendações**.

![](../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/config-recom.png)

O painel Recomendações de configuração é exibido. Se houver um grande número de recomendações, você poderá usar as opções de filtro para restringir quais recomendações serão exibidas. O painel Configuration Recommendations (Recomendações de configuração) exibe o número de instâncias detectadas de cada problema ao lado das recomendações. No exemplo abaixo, há duas instâncias de uma tabela em que foi usada uma operação de união de modelos herdada, de modo que o painel Recomendações de configuração exibe Operação de união de modelos herdada usada. (2) seguido dos locais onde o problema foi detectado.

![](../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/config_rec_pane.png)

Dica: consulte a lista de recomendações e ações para [resolver os problemas identificados](../troubleshooting/studio-troubleshooting-about.htm "(Abre em uma nova guia ou janela)").

O botão Fix All Identified Problems (Corrigir todos os problemas identificados) agora foi renomeado para **Troubleshoot All Identified Problems (Solucionar todos os problemas identificados** ). Ao clicar no botão, será iniciado um fluxo de trabalho de várias etapas que envolve o administrador na solução de problemas e no processo automatizado de "conserto".

![](../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/fixit-button.png)
