# Revisar mensagens de erro

**Aplica-se a** : TBM Studio 12.0 e posterior.

Você deve ter a função padrão de **Power User** ou **Admin**, ou uma função personalizada com as permissões apropriadas, para ver as mensagens de erro. Para visualizar as mensagens de erro, clique em **Errors (Erros** ) na guia Project (Projeto). O aplicativo exibe os erros na janela mostrada na imagem a seguir:
