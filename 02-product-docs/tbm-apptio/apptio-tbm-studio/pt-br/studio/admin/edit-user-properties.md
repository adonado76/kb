# Editar propriedades do usuário

aplica-se a: TBM Studio 12.0 e posterior.

Quando o administrador do Apptio criou sua conta Apptio, ele inseriu as propriedades do usuário, incluindo:

- **E-mail** - Seu nome de usuário do aplicativo. Use esse nome para fazer login no aplicativo.
- **Nome completo** - Seu primeiro e último nome. Esse nome é exibido na lista de usuários na guia Administração.
- **Senha** - Sua senha do aplicativo para fazer login no aplicativo.
- **Função atual** - Sua função determina as ações que você pode executar no aplicativo. Se o administrador do Apptio ativou as funções disponíveis para você, é possível atribuir-se uma função diferente.
- **Moeda** - A moeda padrão usada nos relatórios.
- **Locale** - A localidade padrão usada para determinar o formato dos números nos relatórios.

## Exibir propriedades do usuário

Para exibir suas propriedades de usuário, clique no seu nome de usuário no canto superior direito do cabeçalho Global e clique em **My Account (Minha conta** ).

Alterar sua senha

1. Na caixa de diálogo **Change User Properties (Alterar propriedades do usuário** ), no campo **Old Password (Senha antiga** ), digite sua senha atual.
2. Digite sua nova senha no campo **New Password (Nova senha)** e no campo **Confirm (Confirmar** ).
3. Clique em **OK**.

## Mudar sua função

Se o administrador do Apptio tiver ativado as funções disponíveis para você, será possível alterar sua função clicando na seta para baixo no campo **Função atual** e, em seguida, selecionando uma função. Para obter uma descrição das funções, consulte Definição de funções no *Guia do administrador*.

Observação: Ao alterar as funções, você deve atualizar o navegador para que a função entre em vigor.

## Selecione uma moeda

(Aplica-se a: v12.1, v12.2+.4 ). Se a sua empresa trabalha com mais de uma moeda, você pode selecionar uma moeda padrão para os relatórios. A moeda controla o símbolo da moeda (por exemplo, g.: €/$) e o código internacional de três letras da moeda (por exemplo, g.: EUR/USD) exibido com valores.

## Selecione uma localidade

A localidade determina o formato dos números. Você pode escolher entre uma ampla variedade de locais. A localidade é independente da moeda. Alguns exemplos de locais:

- Estados Unidos: 12, 345.67
- França: 12 345,67
- Alemanha: 12.345,67
