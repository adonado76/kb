# Abrir um projeto

**Aplica-se a** : TBM Studio 12.0 e posterior. Você pode criar muitos projetos no aplicativo, mas só pode abrir um projeto por vez. Para abrir um projeto, abra o menu Projeto no cabeçalho Global e clique no nome do projeto.

![](../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/studio-open-project.jpg)

Todos os usuários com funções diferentes de View Only podem selecionar um projeto no menu Project (Projeto).

Observação: se lhe for atribuída uma função View Only, você terá acesso a um único projeto.
