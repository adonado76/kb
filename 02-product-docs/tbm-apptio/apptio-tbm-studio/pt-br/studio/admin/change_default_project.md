# Alterar o projeto padrão

aplica-se a: Apptio TBM Studio v12.3.1 e posterior.

Especifique o projeto padrão de seu domínio.

1. Na guia **Projeto**, no grupo **Domínio**, clique em **Configurações de domínio**. A caixa de diálogo Configurações do locatário é exibida.
2. No campo **Default Project (Projeto padrão** ), selecione o projeto desejado e clique em **OK**.
