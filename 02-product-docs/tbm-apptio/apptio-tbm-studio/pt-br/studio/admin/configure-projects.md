# Configurar projetos

**Aplica-se a** : TBM Studio 12.0 e posterior. Este artigo descreve as tarefas essenciais que você pode executar para configurar projetos.

Há dois tipos de projetos:

- **Personalizado (plataforma)** : Use um projeto em branco para criar conjuntos de dados, métricas, modelos e relatórios.
- **Costing Standard**: Use um projeto Costing Standard para fornecer um serviço de TI com modelagem de custos, análise de otimização de custos e recursos de orçamento e previsão. Os resultados de um projeto de cálculo de custos de serviços são frequentemente usados para calcular o custo dos serviços no aplicativo de faturamento.
