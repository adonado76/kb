# Especificar uma versão para upgrades de componentes

aplica-se a: TBM Studio 12.3.1 e posterior. Ao fazer upgrade de aplicativos em TBM Studio, você pode especificar qual versão usar para upgrades de componentes. Essa configuração se aplica a todos os upgrades de componentes e novas instalações de componentes.

Para definir a versão de seu componente:

1. Na guia **Projeto**, clique em **Configurações do projeto e** a caixa de diálogo **Editar configurações do projeto** será aberta.
2. No campo **Versão do componente**, selecione a versão do componente desejada. O valor padrão é a versão do componente que foi aplicada quando o projeto foi criado ou na última vez em que as **configurações do projeto** foram alteradas.

## Dicas e truques

Os ícones dos componentes instalados informam você:

- Se um componente instalado for mais antigo do que a versão do aplicativo do projeto atual e houver uma atualização disponível.![imagem](../../resources/images/studio_images/studioimages/studio_specify%20version%20comp_installed.png)
- Se um componente instalado for mais recente do que a versão do aplicativo do projeto atual.![imagem](../../resources/images/studio_images/studioimages/studio_version%20component_upgrade.png)
- Você pode apontar para qualquer componente para ver seu número de versão, que informa a versão do conteúdo:![imagem](../../resources/images/studio_images/studioimages/studio_component%20config_3.png)
