# Sobre Data Studio

**Aplica-se a** : TBM Studio 12.0 e posterior

Quando você traz dados para um dos aplicativos Apptio, eles podem ou não estar no formato correto para serem mapeados para os conjuntos de dados mestres ou para gerar seus próprios relatórios personalizados. Se os dados não estiverem no formato correto, você precisará transformá-los usando as ferramentas de transformação do site Data Studio.
