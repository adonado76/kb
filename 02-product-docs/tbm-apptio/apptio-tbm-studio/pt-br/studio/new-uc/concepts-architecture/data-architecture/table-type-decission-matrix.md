# Matriz de decisão em forma de tabela

Use este guia para escolher o tipo de mesa adequado para a sua situação:

|  |  |  |
| --- | --- | --- |
| **Sua necessidade** | **Tipo de tabela recomendado** | **Por que escolher o ?** |
| Importar arquivos de dados externos (contabilidade geral, orçamento, previsão) | Fonte / Tabela de arquivos | Projetado especificamente para a importação de arquivos com detecção e validação automáticas |
| Limpar, unir ou reformular dados | Transformar tabela | Fluxo de etapas configuráveis para a preparação de dados |
| Entrada manual de dados a partir do zero | Tabela em branco editável | Os usuários definem todas as linhas e valores; não há dependência de fontes externas |
| Anotar ou enriquecer dados gerados por computador | Tabela editável com recursos avançados | Combina dados automatizados com o julgamento humano |
| Exportar dados do modelo para sistemas externos | Tabela publicada | Esquema estável, acessível via API, projetado para a saída de dados |
