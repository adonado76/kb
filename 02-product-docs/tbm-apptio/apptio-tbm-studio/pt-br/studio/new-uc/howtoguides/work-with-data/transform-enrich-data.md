# Transformar e enriquecer dados

|  |  |
| --- | --- |
| **Tipo de Conteúdo** | Guias práticos |
| **Público-alvo** | Analistas de negócios, equipes de TI e finanças |
| **Pré-requisitos** | Conclusão da seção “Importação e gerenciamento de dados” e compreensão das transformações de tabelas |

Esta seção aborda as principais tarefas de transformação de dados no TBM Studio. Esses guias ajudam você a limpar, enriquecer e combinar dados para preparar a modelagem de custos e a elaboração de relatórios. Cada transformação é realizada por meio de etapas de pipeline que são executadas sequencialmente, de cima para baixo.

**Nesta seção:**

- [Limpar e mapear dados usando as colunas do mapa](clean-map.html) (15 a 20 minutos)
- [Juntar dados de várias fontes](join-data.html) (15 a 20 minutos)
- [Aplicar fórmulas para transformar dados](apply-formula.html) (10 a 15 minutos)
- [Filtrar dados por intervalo de data](filter-data-date.html) (10 a 15 minutos)
- [Adicionar várias fontes de dados](append-multiple.html) (15 a 20 minutos)
