## Importação e gerenciamento de dados

Esta seção fornece instruções passo a passo para importar e gerenciar dados no TBM Studio. Quer você esteja enviando arquivos manualmente, conectando-se a sistemas externos ou gerenciando versões mensais de dados, estes guias o ajudarão a estabelecer uma base de dados confiável para seus modelos de custos.

- [Carregar dados de arquivos (Excel, CSV )](upload-data-files.html) – Importar dados de arquivos locais ou da rede
- [Conectar-se a fontes de dados externas ( DataLink )](connect-external-dl.html) – Automatizar a integração de dados do ServiceNow e de outros sistemas
- [Tabelas de versões para dados mensais](version-tables-monthly-data.html) – Gerencie dados de séries temporais com controle de versões mensal
