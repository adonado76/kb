# Criar um relatório básico

**Objetivo:** Criar um novo relatório vazio, pronto para receber tabelas e gráficos

**Quando usar:** Quando você precisar de um novo relatório para apresentar dados do modelo às partes interessadas, ou quando os relatórios existentes não atenderem às suas necessidades

**Tempo estimado:** 5 minutos

## Passos

1. Mude para o TBM Studio, caso ainda não o tenha feito.
2. Clique na guia **Início**, clique **em Novo** e, em seguida, clique em **Relatório**. A caixa de diálogo “Criar relatório” é exibida com as opções de configuração.
3. Digite um **nome** para o seu relatório. Escolha um nome descritivo que ajude os usuários a encontrar o relatório. Se você pretende exportar para o Excel, mantenha os nomes com menos de 31 caracteres para evitar que sejam cortados.
4. Insira uma **descrição** do relatório. Essa descrição aparece no Localizador de Relatórios; portanto, escreva algo que ajude os usuários finais a entender quais dados o relatório contém.
5. Selecione quem pode **visualizar** o relatório. **Todos** podem visualizar o relatório; selecione funções específicas para restringir o acesso.
6. Escolha um **layout** para o seu relatório. Selecione se deseja incluir um cabeçalho e escolha entre **Padrão** (1024 pixels de largura) ou **Widescreen** (1440 pixels de largura).
7. Clique em **OK** para criar o relatório.

## Resultados esperados

- Um novo relatório em branco é aberto na área de trabalho
- Uma guia aparece na parte inferior da área de trabalho com o nome do seu relatório
- O relatório é salvo automaticamente e está pronto para você adicionar componentes

## Armadilhas comuns

- **Esquecer de salvar:** Após fazer alterações, lembre-se de salvar seu relatório para que outras pessoas possam ver suas atualizações e o sistema possa calcular os dados.
- **Nomes pouco claros:** nomes vagos como “ Report1 ” dificultam a localização dos relatórios. Use nomes que descrevam o conteúdo, como "Resumo de custos do data center Q1."
