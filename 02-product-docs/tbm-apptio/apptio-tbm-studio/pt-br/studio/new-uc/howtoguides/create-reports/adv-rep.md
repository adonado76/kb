# Relatórios avançados

**Tipo de conteúdo:** Coleção de guias práticos

**Público-alvo:** Usuários avançados, criadores de relatórios, analistas de negócios

**Pré-requisitos:** Conclusão do curso básico do Report Studio (Seção 3.3.1 ), familiaridade com modelos de dados

**Aplicável a:** TBM Studio 12.0 e versões posteriores

Esta seção aborda técnicas avançadas de geração de relatórios destinadas a usuários avançados que precisam criar soluções complexas e interativas de geração de relatórios. Você aprenderá a criar uma navegação detalhada para explorar hierarquias de dados, adicionar componentes editáveis para fluxos de trabalho de entrada de dados, configurar layouts responsivos e gerar relatórios para diversos canais de saída.

**Nesta seção**

- Crie relatórios com funcionalidade de detalhamento para navegar do resumo aos detalhes
- Adicione componentes editáveis aos relatórios para fluxos de trabalho de entrada de dados
- Configure o dimensionamento e o posicionamento dinâmicos para layouts responsivos
- Exportar e imprimir relatórios em vários canais (PDF, e-mail, impressão)
