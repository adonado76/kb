# Noções básicas sobre relatórios

**Tipo de conteúdo:** Guia prático

**Público-alvo:** Analistas de negócios, equipes de TI e finanças, todos os usuários

**Pré-requisitos:** Acesso a um projeto do TBM Studio com dados do modelo; familiaridade com a interface do TBM Studio

**Visão geral**

Os relatórios do TBM Studio transformam os dados do seu modelo em painéis visuais e interativos que as partes interessadas podem usar para analisar custos, acompanhar orçamentos e tomar decisões fundamentadas. Esta seção aborda as habilidades fundamentais que todo criador de relatórios precisa: criar relatórios, adicionar tabelas de dados, criar gráficos e habilitar a interatividade por meio de filtros.

**Modelo de Componentes do Relatório**

Os relatórios são contêineres que contêm componentes. Pense em um relatório como uma tela na qual você coloca blocos de construção:

- **As tabelas** apresentam os dados em linhas e colunas, sendo ideais para análises detalhadas
- **Os gráficos** permitem visualizar padrões e comparações de dados rapidamente
- **Os filtros** permitem que os usuários selecionem o que querem ver sem precisar editar o relatório

Ao dominar esses três tipos de componentes, você poderá criar relatórios que atendam a uma ampla gama de necessidades comerciais.
