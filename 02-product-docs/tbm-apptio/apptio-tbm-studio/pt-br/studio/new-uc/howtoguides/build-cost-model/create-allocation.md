# Criar alocações

|  |  |
| --- | --- |
| **Tipo de Conteúdo** | Guia prático |
| **Público-alvo** | Analistas de negócios, equipes de TI e finanças |
| **Tempo de conclusão** | 30 a 60 minutos |
| **Pré-requisitos** | Tabelas de origem e de destino com etapas do modelo; Dados transformados preparados no Data Studio; Projeto baixado |

## Visão geral

As alocações são o cerne da modelagem de custos no TBM Studio. Elas determinam como os custos são transferidos de uma tabela para outra — distribuindo despesas como os custos do data center para as aplicações, os custos de mão de obra para os serviços ou os custos de TI para as unidades de negócios. Esta seção oferece orientações passo a passo para a criação de alocações, desde distribuições simples de um para um até fluxos de custos complexos em vários níveis.
