# Próximos passos para todas as funções

## Para todas as funções

Depois de concluir seu percurso de aprendizagem, considere os seguintes passos:

1. **Coloque suas habilidades em prática:** aplique o que aprendeu em projetos e fluxos de trabalho reais
2. **Explore outras perspectivas:** compreender outras funções melhora a colaboração
3. **Junte-se à comunidade:** interaja com outros usuários do TBM Studio
4. **Mantenha-se atualizado:** consulte regularmente *a seção "Novidades* " para conhecer os novos recursos
5. **Envie seus comentários:** ajude a melhorar esses percursos de aprendizagem para futuros usuários

**Oportunidades em diferentes funções**

- **Analistas de negócios → Administradores:** Se você precisar gerenciar o acesso ou as permissões dos usuários
- **Finanças de TI → Desenvolvedores:** Se você deseja automatizar o carregamento de dados ou criar integrações
- **Administradores → Finanças de TI:** Para compreender melhor os modelos que vocês estão apoiando
- **Desenvolvedores → TI Financeira:** Para compreender o contexto de negócios das suas integrações
