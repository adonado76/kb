# Componentes do relatório - Componentes de entrada

## Visão geral

Os componentes de entrada do TBM Studio permitem que os usuários insiram ou modifiquem dados diretamente nos relatórios. Ao contrário dos componentes de filtro, que se limitam a restringir os dados exibidos, os componentes de entrada gravam os dados em tabelas editáveis, que, por sua vez, passam pelo fluxo de dados e, por fim, afetam os cálculos do modelo de custos.

Os componentes de entrada são configurados principalmente por meio das propriedades editáveis das colunas da tabela e apresentados aos usuários finais por meio de componentes de relatório. Compreender a diferença entre filtrar dados (filtros) e modificar dados (componentes de entrada) é fundamental para criar relatórios eficazes que permitam a inserção de dados.

**Princípio fundamental: Os componentes de entrada funcionam com tabelas editáveis. Para habilitar a inserção de dados nos relatórios, é necessário primeiro criar uma tabela editável e configurar suas colunas com os tipos de entrada, regras de validação e restrições adequados.**
