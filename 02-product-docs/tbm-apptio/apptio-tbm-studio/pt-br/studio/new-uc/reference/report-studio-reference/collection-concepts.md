# Conceitos de cobrança

**O que é uma coleção de relatórios**

Uma coleção de relatórios é um contêiner com nome que agrupa relatórios relacionados. Quando os relatórios pertencem à mesma coleção, os usuários podem navegar facilmente entre eles usando um componente de navegação da coleção de relatórios que aparece em cada relatório. As coleções oferecem uma experiência de visualização integrada para relatórios que compartilham um objetivo comum, como análises financeiras mensais, painéis executivos ou análises de custos departamentais.

Principais características das coleções de relatórios:

- Cada relatório pode pertencer a apenas uma coleção por vez
- Uma vez criadas, as coleções ficam disponíveis em todo o projeto
- As coleções oferecem suporte ao controle de acesso baseado em funções
- As paletas de cores podem ser aplicadas no nível da coleção para garantir a consistência da identidade visual

**Organização por coleções versus organização por pastas**

As coleções de relatórios diferem da organização baseada em pastas no Explorador de Projetos. As pastas no Explorador de Projetos ajudam os criadores de conteúdo a organizar relatórios durante o desenvolvimento, enquanto as coleções são projetadas para facilitar a navegação e a consulta pelos usuários finais.

|  |  |  |
| --- | --- | --- |
| **Funcionalidade** | **Coleções de relatórios** | **Pastas do Project Explorer** |
| Propósito | Navegação do usuário final | Organização do conteúdo durante o desenvolvimento |
| Visibilidade | Visível para os consumidores que enviam relatórios | Visível para criadores de conteúdo |
| Navegação | Componente de navegação integrado | Navegação manual pelo Explorador de Projetos |
| Permissões | Controle de acesso baseado em funções | Segue as permissões no nível do relatório |
| Estilo | Suporta paletas de cores no nível da coleção | Sem recursos de estilo |

**Casos de uso para coleções**

As coleções de relatórios são particularmente úteis nos seguintes cenários:

**Pacotes de relatórios mensais**

- Agrupe todos os relatórios necessários para uma análise financeira mensal
- Inclua resumos de custos, análises de variação e relatórios de tendências em uma única coleção

**Pacotes de painéis executivos**

- Combine painéis de KPIs de alto nível com relatórios detalhados
- Oferecer aos executivos uma visão abrangente da saúde financeira da área de TI

**Visões departamentais**

- Crie coleções para cada unidade de negócios ou centro de custo
- Permitir que os chefes de departamento naveguem facilmente pelos relatórios relevantes

**Organização por tópicos**

- Relatórios agrupados por tema (por exemplo, custos de infraestrutura, custos de aplicativos, custos de mão de obra)
- Ajudar os usuários a encontrar relatórios relacionados a categorias específicas da taxonomia do TBM
