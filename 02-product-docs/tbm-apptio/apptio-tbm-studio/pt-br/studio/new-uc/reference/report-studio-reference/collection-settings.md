# Configurações da coleção

**Configurações da paleta de cores**

A partir de R12.10.10, os administradores podem aplicar paletas de cores personalizadas às coleções de relatórios. Quando uma paleta de cores é selecionada para uma coleção, todos os relatórios dessa coleção passam a usar automaticamente a mesma paleta de cores nos gráficos, garantindo a consistência visual em toda a coleção.

**Para aplicar uma paleta de cores a uma coleção:**

1. Na guia **Projeto**, selecione **Coleções de relatórios**.
2. Selecione a coleção na lista **de coleções disponíveis**.
3. Escolha uma paleta de cores entre as paletas personalizadas ou predefinidas disponíveis.
4. Clique em **Fechar** para salvar.

Observação: Se for selecionada uma paleta de cores diferente para um relatório específico dentro da coleção, a paleta desse relatório substituirá a paleta definida no nível da coleção apenas para esse relatório. Os novos relatórios adicionados à coleção não herdam automaticamente a paleta de cores da coleção; os administradores devem aplicá-la manualmente no nível do relatório, se necessário.

**Âmbito da paleta de cores**

As paletas de cores no nível da coleção se aplicam a:

- Gráficos de barras, gráficos de linhas, gráficos de pizza e a maioria dos outros tipos de gráficos
- Apenas as cores dos gráficos (não tabelas, fontes, filtros, planos de fundo ou KPIs)

As paletas de cores não se aplicam aos tipos de gráfico Treemap e Waterfall.
