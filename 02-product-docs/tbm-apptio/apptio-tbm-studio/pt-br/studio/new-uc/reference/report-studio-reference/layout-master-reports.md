# Relatórios principais

Os relatórios-mestre fornecem uma camada de modelo (como um “papel de parede”) que pode ser aplicada a vários relatórios para garantir um layout consistente.

**Principais características:**

- Os componentes do relatório mestre não podem ser editados a partir do relatório em que estão aplicados.
- Os componentes adicionados a um relatório ficam sobre a camada do relatório mestre.
- As caixas de agrupamento dos relatórios principais perdem sua funcionalidade de agrupamento; utilize-as como bordas visuais.

**Para criar um relatório mestre:**

1. Dê uma olhada no relatório.
2. Na guia Relatório, no grupo Avançado, clique em Relatório mestre.
3. Salve o relatório.
