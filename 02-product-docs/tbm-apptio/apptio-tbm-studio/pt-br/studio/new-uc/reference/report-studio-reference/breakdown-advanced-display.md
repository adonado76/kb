# Opções de detalhamento e exibição avançadas

As opções avançadas controlam o comportamento dos relatórios em contextos específicos, como navegação detalhada, visualizações filtradas e exibições específicas por unidade.

|  |  |
| --- | --- |
| **Relatar tipo** | **Exibir opções** |
| Relatório de ausências por motivo não relacionado a avarias | Mesmo ao exibir uma avaria: é exibido mesmo que sejam recebidos dados de avaria. Somente quando não houver detalhamento: exibe apenas dados sem detalhamento. |
| Relatório detalhado | Mesmo quando não há detalhamento: exibe os dados mesmo sem detalhamento. Apenas ao exibir uma discriminação: exibe-se somente quando são fornecidos dados de discriminação. |
| Relatório filtrado | Somente quando filtrado desta forma: exibe apenas se a filtragem corresponder à fonte. Mesmo que seja filtrado de outra forma: é exibido independentemente das diferenças entre os filtros. |
| Relatório da unidade | Para todas as unidades: Aplica-se a todas as unidades. Apenas para [unidade]: Aplica-se apenas a uma unidade específica no filtro. |
