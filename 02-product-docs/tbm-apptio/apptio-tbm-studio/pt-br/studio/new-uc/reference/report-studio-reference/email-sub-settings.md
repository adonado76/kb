# Configurações de assinatura por e-mail

As assinaturas por e-mail permitem que os relatórios sejam distribuídos automaticamente aos usuários em horários programados.

**Configuração da assinatura**

|  |  |
| --- | --- |
| **Configuração** | **Descrição** |
| Nome da Assinatura | Preenchido automaticamente como <Coleção de relatórios - Nome do relatório>. Pode ser personalizado. |
| Recipientes | Endereços de e-mail dos usuários que receberão a assinatura. Devem ser endereços de e-mail válidos de domínios autorizados. |
| Período de referência | O "Período atual" refere-se ao período ativo em que a assinatura está em vigor. Também é possível especificar um período fixo. |
| Incluir o PDF como anexo | Anexa uma versão em PDF do relatório ao e-mail |
| Visualizações filtradas | Permite enviar relatórios com seleções de filtro já aplicadas |
| Upload em massa de arquivos XLS | Permite a distribuição em massa com visualizações filtradas exclusivas para cada destinatário |

***Nota de segurança:*** *os e-mails só podem ser enviados para domínios autorizados configurados no ambiente do cliente. A segurança em nível de linha (RLS) não é aplicada nas assinaturas de e-mail.*
