# Gráficos de colunas

Os gráficos de colunas apresentam barras verticais e são particularmente eficazes para mostrar variações ao longo do tempo ou comparar itens quando há períodos de tempo envolvidos. Eles compartilham as mesmas opções de configuração de dados dos gráficos de barras, mas apresentam os dados na vertical.

**Gráfico de colunas verticais**

O gráfico de colunas padrão apresenta os valores na forma de barras verticais. Essa é a opção mais comum para comparações de séries temporais, nas quais o eixo X representa períodos de tempo, como meses, trimestres ou anos.

**Quando usar: para** comparar valores entre períodos, mostrar variações de um período para outro ou exibir dados cuja ordem natural de leitura seja da esquerda para a direita, em ordem cronológica.

**Gráfico de colunas empilhadas**

Os gráficos de colunas empilhadas segmentam cada coluna para mostrar a composição dos totais. Isso permite que os espectadores vejam tanto a tendência dos valores totais quanto a forma como os componentes de cada período contribuem para esse total.

**Quando usar: Para** mostrar como os componentes de custo variam ao longo do tempo, ao mesmo tempo em que se apresenta a tendência geral, como, por exemplo, os gastos mensais com TI detalhados por infraestrutura, aplicativos e serviços.

**Como criar um gráfico de colunas**

1. Na guia Relatório, clique em Gráfico. O gráfico é exibido por padrão no formato de barras horizontais.
2. No painel "Configuração do componente", selecione um objeto modelo na lista suspensa.
3. Arraste um campo de dimensão para a área da legenda para definir os agrupamentos das barras.
4. Arraste um campo métrico para a área Valores para definir o que as colunas medem.
5. Arraste um campo de tempo (Meses, Trimestres, Anos) para a área do eixo.
6. Selecione a guia Ad Hoc e clique no ícone Coluna para alternar entre barras horizontais e verticais.
