# Colunas baseadas no calendário

Adicione colunas que agreguem ou comparem valores entre períodos. Acesse essas opções na guia Fórmulas > menu Datas.

|  |  |  |
| --- | --- | --- |
| **Abreviação** | **Nome** | **Descrição** |
| YTD | No acumulado do ano | Soma os valores desde o início do exercício fiscal até o período selecionado. |
| Citação do trimestre | No acumulado do trimestre | Soma os valores desde o início do trimestre até o período selecionado. |
| FY | Ano fiscal | Total para todo o exercício fiscal, independentemente do período selecionado. |
| FQ | Trimestre fiscal | Total para todo o trimestre fiscal, independentemente do período selecionado. |
| ANN | Anualizado | Valor médio mensal acumulado multiplicado por 12 para calcular o total anual. |
| LFY | Último ano fiscal | Valor do exercício anterior. Útil para comparações ano a ano. |
| LFQ | Último trimestre fiscal | Total do trimestre fiscal anterior. |
| LTM | Últimos 12 meses | Total acumulado dos últimos 12 meses, sem incluir o mês atual. Não se baseia no calendário fiscal. |
| LM | Último mês | Valor do mês anterior. |
| LY | No mesmo período do ano passado | Período correspondente do ano anterior, para comparação ano a ano. |
| TTM | Últimos 12 meses | Soma dos últimos 12 meses (por exemplo, agosto de 2024 = agosto de 2023 a julho de 2024). Calcula o valor anualizado se houver menos de 12 meses disponíveis. |
