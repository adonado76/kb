# Configuração do layout de impressão

O modo de layout de impressão oferece controles específicos para definir como os relatórios serão exibidos ao serem exportados para PDF ou impressos.

**Acessando o modo de layout de impressão**

1. Confira o relatório para edição.
2. Na guia Página inicial, clique em Exibir e selecione Layout de impressão.
3. A guia “Layout de impressão” é exibida com os controles de layout.

Para sair: na guia “Layout de impressão”, clique em “Sair do modo de layout de impressão”.

**Opções de configuração da página**

|  |  |
| --- | --- |
| **Opção** | **Descrição** |
| **Tamanho da página** | Selecione o tamanho do papel no menu suspenso (Letter, A4, Legal, etc.) |
| **Fluxo** | Controla a disposição dos componentes (Manual, Vertical ou Horizontal) |
| **Margens** | Definir margens da página para impressão |
| **Cabeçalho** | Adicione o texto do cabeçalho; exibido em todas as páginas ou apenas na primeira página. Suporta formatação HTML e texto dinâmico |
| **Rodapé** | Personalizar o texto do rodapé; não suporta texto dinâmico |
| **Retrato** | Orientação vertical da página |
| **Paisagem** | Orientação horizontal da página |
| **Escala** | Nível de zoom para a impressão |

**Opções de fluxo**

O fluxo controla a forma como os componentes são dispostos na página impressa:

|  |  |
| --- | --- |
| **Configuração do fluxo** | **Comportamento** |
| **Manual** | Você posiciona os componentes manualmente; eles permanecem onde foram colocados |
| **Vertical** | Os componentes são dispostos automaticamente em uma única coluna vertical |
| **Horizontal** | Os componentes são organizados automaticamente em linhas, com um número máximo de componentes por linha |

**Opções de impressão de componentes**

**Fixar/Desfixar componentes:** fixe os componentes no final de um relatório para impressão. Isso é útil para mover os cortadores para o final da tela, a fim de mostrar os filtros aplicados, ou para fixar todos os componentes de um tipo específico.

**Mostrar/ocultar componentes:** Controle quais componentes aparecem na impressão:

1. Na guia “Layout de impressão”, clique em “Mostrar/ocultar componentes do relatório”.
2. Desmarque as caixas de seleção dos componentes que deseja ocultar.
3. Use o campo "Filtro" para localizar componentes específicos em listas extensas.

**Configuração de tabelas para impressão em várias páginas**

Para imprimir uma tabela extensa em várias páginas:

1. Entre no modo de layout de impressão.
2. Mova a tabela para uma página separada.
3. Selecione a tabela e, na guia “Layout de impressão”, clique em “Redimensionar componente para página inteira”.
4. Com a tabela selecionada, na subguia Tabela, selecione Imprimir todas as páginas.
5. Salve e saia do modo de layout de impressão.
