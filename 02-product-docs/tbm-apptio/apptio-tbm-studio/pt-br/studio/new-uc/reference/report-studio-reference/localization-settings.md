# Configurações de localização

As configurações de localização são definidas principalmente no nível do projeto e afetam a forma como números, datas e moedas são exibidos nos relatórios.

|  |  |
| --- | --- |
| **Configuração** | **Descrição** |
| Compactação de números | Controla a exibição de números grandes (milhares, milhões, bilhões, trilhões) |
| Símbolos monetários | Define o símbolo monetário padrão para a exibição de dados financeiros |
| Agrupamento de números | Formatação regional para separadores numéricos (por exemplo, separação de milhares no japonês) |
| Tamanho padrão do papel | Define o tamanho padrão do papel para operações de impressão e exportação |

**Localização:** Guia Projeto > Configurações do projeto
