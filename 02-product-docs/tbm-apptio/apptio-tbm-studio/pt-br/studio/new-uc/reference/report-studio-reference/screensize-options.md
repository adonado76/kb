# Opções de tamanho da tela

Ao criar um novo relatório, você seleciona a largura da tela de destino que determina a área de layout do relatório.

|  |  |  |
| --- | --- | --- |
| **Tamanho da tela** | **Largura** | **Ideal para** |
| **Padrão** | 1024 pixels | Monitores tradicionais, telas menores |
| **Tela panorâmica** | 1440 pixels | Monitores amplos, visores no painel |

**Para definir o tamanho da tela durante a criação do relatório:**

1. Na guia Início, clique em Novo e, em seguida, clique em Relatório.
2. Na caixa de diálogo “Criar novo relatório”, selecione “Padrão” ou “Widescreen” nas opções de layout.

Observação: é possível alterar o tamanho da tela após criar o relatório, acessando as propriedades do relatório.
