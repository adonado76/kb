# Casos de uso comuns

**Mapeamento de custos de mão de obra: os usuários selecionam Solução → Categoria → Subcategoria nos menus suspensos em cascata para classificar os custos de mão de obra para alocação.**

**Explicações sobre variações orçamentárias: as colunas de entrada de texto registram as explicações para as variações orçamentárias juntamente com as métricas calculadas.**

**Atualização de atributos de centros de custo: os gerentes responsáveis atualizam os atributos de seus próprios centros de custo, sob o controle da segurança por linha.**

**Correções diretas de dados: os usuários corrigem valores de dados específicos quando os dados do sistema de origem estão incompletos ou incorretos.**
