# Componentes do relatório: Navegação

## Visão geral

Os componentes de navegação permitem que os usuários naveguem entre relatórios, páginas e recursos externos no TBM Studio. Um design de navegação eficaz melhora a experiência do usuário, oferecendo acesso intuitivo a informações detalhadas e relatórios relacionados. Este documento de referência descreve todos os componentes de navegação, suas opções de configuração e as melhores práticas para implementação.

O TBM Studio oferece vários mecanismos de navegação: componentes de botão para controles de navegação explícitos, links de relatório para navegação detalhada a partir de tabelas e gráficos, links codificados usando sintaxe no estilo Wiki para navegação flexível, navegadores de coleções de relatórios para organizar relatórios relacionados e filtros hierárquicos para filtragem detalhada.
