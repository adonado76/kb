# Dimensionamento de componentes

**Métodos de redimensionamento**

|  |  |
| --- | --- |
| **Método** | **Instruções** |
| **Arraste as bordas** | Clique e arraste qualquer borda ou canto do componente |
| **Caixa de diálogo "Posição e tamanho"** | Clique com o botão direito do mouse no cabeçalho → Posição e tamanho → insira os valores de largura/altura |
| **Redimensionar o componente para ocupar a tela inteira** | Disponível no modo de layout de impressão para tabelas |

**Ajuste automático do tamanho**

Alguns componentes podem ajustar automaticamente o tamanho de acordo com o conteúdo:

**Para tabelas:** as tabelas podem ajustar automaticamente o tamanho para exibir todos os dados sem barras de rolagem. Configure isso na caixa de diálogo Propriedades da tabela, na seção de opções de dimensionamento.

**Para caixas de grupo:** as caixas de grupo podem ser redimensionadas dinamicamente para acomodar os componentes que contêm:

1. Selecione a caixa de grupo.
2. Na guia "Grupo", selecione uma das opções de dimensionamento automático: "Dimensionar largura automaticamente", "Dimensionar altura automaticamente" ou "Dimensionar ambos automaticamente".

Observação: o redimensionamento automático das caixas de grupo funciona apenas no modo Exibir. No modo de edição, a caixa de grupo mantém um tamanho fixo para facilitar a edição.
