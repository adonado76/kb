# Componentes do relatório: Filtros e segmentadores

## Visão geral

Os filtros e os segmentadores são componentes interativos que permitem aos usuários dos relatórios filtrar dinamicamente os dados exibidos em tabelas e gráficos. Eles transformam relatórios estáticos em poderosas ferramentas de análise de dados, permitindo que os usuários se concentrem em elementos específicos e obtenham insights por meio da exploração interativa.

**Principais vantagens dos filtros e segmentadores:**

- Permita que os usuários personalizem as visualizações dos relatórios sem alterar o layout do relatório
- Suporta várias combinações de critérios de filtragem utilizando a lógica AND/OR
- Fornecer feedback visual sobre valores de dados relacionados e não relacionados
- Permitir que os administradores definam configurações padrão, ao mesmo tempo em que oferecem flexibilidade aos usuários
- Oferecer suporte à exploração hierárquica por meio de dimensões de dados

Observação: os filtros funcionam apenas com tabelas e gráficos de consultas ad hoc criados por meio do painel de configuração de componentes. Eles não funcionam com tabelas e gráficos antigos.
