# Ciclos de atualização de dados

|  |  |  |
| --- | --- | --- |
| **Ciclo** | **Descrição** | **Verificação da validade** |
| Atualizações pontuais | Atualizações conforme necessário; acrescentar ou substituir | Nenhum |
| Versão 1; Sem atualizações programadas | Versão única, substituída ao fazer o upload | Nenhum |
| Versão 1; Atualização mensal | Atualização mensal, substitui o existente | Mensalmente |
| Versão 1; Atualização anual | Atualização anual, substitui o existente | Anual |
| Versões mensais; Atualização mensal | Nova versão a cada mês | Mensalmente |
| Versões anuais; Atualização mensal | Versões dos últimos 12 meses | Mensalmente |
| Versões anuais; Atualização no início do ano | Versão anual no início do exercício fiscal | Anual |
| Versões anuais com elementos mantidos | Anual com transição do exercício anterior | Anual |
