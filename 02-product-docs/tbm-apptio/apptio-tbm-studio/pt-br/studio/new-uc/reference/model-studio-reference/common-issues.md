# Problemas comuns

|  |  |  |
| --- | --- | --- |
| **Problema** | **Causa** | **Resolução** |
| Não é possível adicionar a tabela ao nível | A tabela não contém a etapa "Modelo" | Adicionar etapa do modelo à transformação da tabela |
| Não é possível adicionar uma coluna a um nível com tabelas | Conteúdo misto não permitido | Remova as mesas primeiro ou utilize um outro nível |
| Carregamento lento do relatório | Foram selecionados muitos indicadores | Limitar métricas usando a opção “Editar métricas” |
| Alocações em falta | Mesas não registradas | Verificar todas as alterações no modelo |
| Relatório não consta da coleção | Não atribuído a nenhuma coleção | Use a opção “Atribuir à coleção” |
