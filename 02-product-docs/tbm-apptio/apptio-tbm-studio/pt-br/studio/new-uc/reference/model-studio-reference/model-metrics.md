# Métricas de modelo

Métricas são cálculos que definem as medidas utilizadas nos relatórios. Eles representam valores como custo, orçamento, previsão ou cálculos personalizados que circulam pelo modelo e são exibidos nos relatórios.

## Métricas padrão

O TBM Studio inclui vários tipos de métricas padrão comumente utilizadas em modelos de custos.

**Tipos métricos**

|  |  |  |
| --- | --- | --- |
| **Métrica** | **Propósito** | **Fontes comuns** |
| Custo | Despesas efetivamente incorridas | Valores reais do GL, faturas, depreciação de ativos |
| Orçamento | Despesas previstas | Arquivos orçamentários, sistemas de planejamento |
| Previsão | Despesas futuras previstas | Arquivos de previsão, modelos de projeção |
| Variância | Diferença entre métricas | Calculado: Orçamento - Custo |
| Volume/Quantidade | Contagens não monetárias | Número de usuários, volume de transações |
