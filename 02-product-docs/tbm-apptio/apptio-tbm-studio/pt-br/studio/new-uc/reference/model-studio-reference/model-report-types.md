# Tipos de relatórios modelo

**Relatórios baseados em tabelas**

Exibir tabelas inteiras em níveis. Os usuários finais podem detalhar os dados para ver os valores das colunas.

**Casos de uso**

- Permitir que os usuários explorem diferentes dimensões (centros de custo, fornecedores, etc.)
- Exibir a visualização completa de uma camada do modelo
- Oferecer flexibilidade para análises pontuais

**Relatórios baseados em colunas**

Exibir colunas específicas, oferecendo uma visão focada em determinadas dimensões.

**Casos de uso**

- Relatório focado em elementos específicos (por exemplo, apenas o grupo de custos)
- Fornecer uma visão consistente para os relatórios executivos
- Simplificar para públicos específicos
