# Exportação e compartilhamento

**Opções de compartilhamento**

|  |  |
| --- | --- |
| **Método** | **Descrição** |
| Modelo de relatório | Criar um modelo de relatório para uso dos usuários finais (pode ser adicionado a coleções de relatórios) |
| Captura de tela | Diagrama de captura para documentação ou apresentações |
| Coleta de relatórios | Adicione o relatório modelo a uma coleção para facilitar o acesso ( v12.2.2+ ) |

**Dica:** Use diagramas de modelos para integrar novos membros da equipe, solucionar problemas de alocação e comunicar a estrutura do modelo às partes interessadas.

*Consulte “Criar [relatórios” para obter informações sobre os procedimentos de criação de relatórios](../../howtoguides/create-reports/create-basic-report.html)*
