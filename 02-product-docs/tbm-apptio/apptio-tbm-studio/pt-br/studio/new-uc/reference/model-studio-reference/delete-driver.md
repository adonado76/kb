# Excluir um motorista

Para remover um driver de um objeto modelo:

1. Clique no fator no diagrama de Sankey
2. Clique em “Excluir” na parte inferior do painel
3. Salvar alterações

Aviso: A exclusão de um driver remove todos os valores associados do objeto modelo. Verifique o impacto nas alocações posteriores antes de excluir.
