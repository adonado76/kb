# Parâmetros do relatório

**Opções de configuração**

|  |  |
| --- | --- |
| **Parâmetro** | **Descrição** |
| Nome do relatório | Nome de exibição do relatório (visível no Explorador de Projetos) |
| Descrição | Documentação opcional sobre o objetivo do relatório |
| Métricas selecionadas | Quais métricas calcular e exibir (isso afeta o desempenho) |
| Estrutura de níveis | Número, ordem e conteúdo dos níveis |
| Coleta de relatórios | Atribuição opcional a uma coleção para facilitar o acesso ( v12.2.2+ ) |

**Otimização de desempenho**

Limite as métricas selecionadas para melhorar a velocidade de cálculo nos ambientes de teste e produção.

**Limitar métricas**

1. Abra o relatório e dê uma olhada
2. Clique em Exibir > Mostrar documento > Editar métricas
3. Selecione apenas as métricas de que você precisa
4. Clique em OK e salve
5. Confira o relatório

**Dica:** selecionar menos métricas reduz significativamente o tempo de cálculo, especialmente em modelos de grande porte.

**Atribuir à coleção de relatórios**

1. Abrir o modelo de relatório
2. Clique na guia Modelagem
3. Clique em “Atribuir à coleção”

1. Selecione a coleção de relatórios no menu suspenso
2. Salvar o relatório
