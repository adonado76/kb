# Relatórios de modelos

Os relatórios de modelo oferecem ao usuário final acesso a visualizações do modelo, mostrando os fluxos de custos em um diagrama de Sankey configurável e organizado por níveis.

## Configuração do diagrama de Sankey

**Criação de um relatório modelo**

1. Na guia Página inicial, clique em Novo > Modelo de relatório
2. Clique em “Editar métricas” para selecionar quais métricas incluir
3. Adicione níveis usando o Editor de Níveis
4. Arraste tabelas ou colunas para os níveis no Explorador de Projetos
5. Dê um nome a cada nível
6. Salve e verifique o relatório

**Operações do Editor de Níveis**

|  |  |
| --- | --- |
| **Operação** | **Instruções** |
| Exibir Editor de Níveis | Clique no menu Exibir > Mostrar Editor de Níveis |
| Adicionar um nível | Clique no ícone “Adicionar nível” (+) |
| Excluir um nível | Clique no ícone “Excluir nível” (x) |
| Reordenar níveis | Clique no ícone “Reorganizar” no cabeçalho da linha e arraste |
| Adicionar tabela ao nível | Arraste a tabela do Explorador de Projetos para o nível |
| Adicionar coluna ao nível | Expanda a tabela no Explorador de Projetos e arraste a coluna para o nível |
| Dê um nome a um nível | Clique no campo “Nome” na parte superior do nível e digite o nome |
| Voltar à visualização do relatório | Clique no menu Exibir > Mostrar documento |

**Regras de conteúdo por nível**

- Um nível pode conter tabelas OU uma única coluna, mas não ambos
- É possível adicionar várias tabelas a um único nível
- Se uma coluna for adicionada, nenhum outro elemento poderá ser adicionado a essa camada
- Várias camadas podem usar colunas da mesma tabela
- Somente tabelas modeladas podem ser adicionadas aos níveis
