# Opções de configuração do driver

**Parâmetros completos do driver**

|  |  |  |  |
| --- | --- | --- | --- |
| **Parâmetro** | **Necessário** | **Valores** | **Descrição** |
| Adicionar a (Métricas) | Sim | Custo, Orçamento, Previsão, Personalizado | Quais métricas recebem o valor deste motorista |
| Utilização (Tipo) | Sim | Coluna, Métrico, Fórmula | Como o valor é determinado |
| Coluna | Condicional | Nomes das colunas da tabela | Obrigatório ao usar = Coluna |
| Métrica | Condicional | Métricas disponíveis | Obrigatório ao usar = Métrico |
| Fórmula | Condicional | Expressão de fórmula válida | Obrigatório ao usar a fórmula |
| Notas | Não | Texto livre | Documentação para facilidade de manutenção |
