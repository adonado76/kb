# Operações com tabelas

## Operações com colunas

|  |  |
| --- | --- |
| **Operação** | **Processo** |
| Adicionar uma linha | Clique no botão “Adicionar linha” na parte inferior da tabela |
| Filtrar por valor | Clique com o botão direito do mouse na célula > Filtrar por valor |
| Congelar coluna | Clique com o botão direito do mouse no cabeçalho > Formatação > Fixar coluna |
| Definir cor de fundo | Clique com o botão direito do mouse no cabeçalho > Formatação > Definir cor de fundo |
| Definir largura da coluna | Arraste a borda ou clique com o botão direito do mouse > Formatação > Definir largura |
| Mostrar valores (duplicados) | Clique com o botão direito do mouse no cabeçalho > Mostrar valores |

## Truncar (Tabelas editáveis)

***Aplicável a:****12.11.4 e versões posteriores*

|  |  |
| --- | --- |
| **Tipo de tabela** | **Efeito de truncamento** |
| Tabela editável com recursos avançados | Reverte aos valores originais da fonte |
| Tabela geral editável | Exclui todas as linhas |

**Aviso:** o truncamento é definitivo após o check-in. Se necessário, reverta antes do check-in.
