# Recomendações de recursos não utilizados

O TBM Studio identifica recursos não utilizados que podem ser removidos para melhorar o desempenho. Acesse essas opções na guia “Recomendações” e utilize o fluxo de trabalho de solução de problemas guiado.

## Métricas não utilizadas

Métricas calculadas que não são utilizadas em outras métricas ou relatórios. Acesse o TBM Studio > Recomendações > Métricas não utilizadas. Selecione “Resolver todos os problemas identificados” e siga o fluxo de trabalho guiado para excluir métricas não utilizadas.

## Alocações não utilizadas (alocações com zero unidades)

Alocações que geram fluxo de custo zero. Isso pode indicar erros de configuração ou uma lógica de alocação desatualizada. Siga o fluxo de trabalho "Recomendações" para revisar e excluir alocações não utilizadas por objeto de modelo.

## Relatórios não utilizados (BETA)

Relatórios que não foram acessados recentemente. Desative o cálculo automático antes de usar este fluxo de trabalho para alterações em massa. Desativar relatórios não utilizados reduz a sobrecarga de cálculos.
