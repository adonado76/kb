# Relatar problemas do Report Studio

## O relatório não carrega ou carrega lentamente

**Sintomas:**

- O relatório aparece em branco ou não termina de carregar
- O navegador expira ao abrir o relatório
- O relatório carrega, mas demora vários minutos

**Soluções:**

1. **Verificar o status do cálculo** - Vá para Compilação > Fila de cálculos. Verifique se o ambiente concluiu o cálculo.
2. **Atualize o navegador e limpe o cache** - Atualize a página do navegador. Se o problema persistir, tente limpar o cache do navegador.
3. **Verificar os dados subjacentes** - Verifique se as tabelas de origem e as transformações contêm dados válidos para o período selecionado.
4. **Teste no ambiente de desenvolvimento** — Mude para o espaço de trabalho de desenvolvimento e verifique se o mesmo relatório é carregado corretamente nesse ambiente.
