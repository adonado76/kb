# Atalhos de teclado

Domine esses atalhos para trabalhar com mais eficiência no TBM Studio.

## Atalhos essenciais

|  |  |  |
| --- | --- | --- |
| **Atalho** | **Ação** | **Descrição** |
| Ctrl+Alt+T | Ativar/desativar o modo Estúdio | Ativar e desativar o modo de edição do Studio |
| Ctrl+F1 | Alternar faixa de opções | Mostrar/ocultar as guias e a faixa de opções do TBM Studio |
| Alt+N | Novo documento | Criar um novo documento no contexto atual |
| Alt+O | Check-out | Selecione o documento a ser editado |
| Ctrl+S / Meta+S | Salvar | Salvar o documento atual |
| Alt+K | Efetuar check-in | Enviar o documento atual para o Staging |
| Ctrl+Alt+Z | Reverter | Desfazer todas as alterações feitas desde a última retirada |
| F9 | Atualizar documento | Atualizar o documento atual com as alterações mais recentes |
| Shift+F9 | Atualizar área de trabalho | Atualizar todos os documentos na área de trabalho |
| Alt+Esc | Fechar aba | Fechar a guia do documento atual |
| Ctrl+Z | Desfazer | Desfazer as alterações feitas desde o último salvamento |
| Ctrl+Y | Refazer | Reverter as alterações que foram desfeitas anteriormente |

**Notas sobre a plataforma:**

- Tecla **Meta** = Command (⌘) no Mac, tecla Windows no Windows
- Os atalhos **do Ctrl** funcionam tanto no Windows quanto no Mac
