# Trabalhando com documentos

## Guias do documento

Os documentos abertos aparecem como guias na área principal da área de trabalho.

**Recursos da guia:**

- **Guia ativa** — Destacada para indicar o documento atual
- **Botão Fechar** — Clique em ✕ para fechar uma guia
- **Indicador** de alterações não salvas — O asterisco (\*) indica que há alterações não salvas
- **Ícone do tipo de documento** — Indicador visual do tipo de documento

**Para gerenciar as guias:**

- **Alternar entre guias:** clique na guia que deseja visualizar
- **Fechar aba:** Clique no ✕ ou pressione Alt+Esc (Meta+Esc no Mac)
- **Fechar todas as guias:** clique com o botão direito do mouse em qualquer guia e selecione “Fechar tudo”
- **Reordenar guias:** arraste e solte as guias para reorganizá-las

## Check-out e check-in

**A função "Check Out"** bloqueia um documento para edição e impede que outros usuários façam alterações.

**Para retirar um documento:**

1. Encontre o documento no Explorador de Projetos
2. Clique com o botão direito do mouse e selecione **“Check Out”** (ou pressione Alt+O)
3. O documento abre com um ícone de cadeado
4. Faça suas edições

**O Check In** salva suas alterações e as disponibiliza para outros usuários no Staging.

**Para registrar um documento:**

1. Salve suas alterações (Ctrl+S ou Meta+S)
2. Clique em **“Check-in”** na barra de ferramentas (ou pressione Alt+K)
3. Adicione um comentário ao fazer o check-in descrevendo suas alterações (opcional, mas recomendado)
4. Clique **em OK**

Aviso: Salve frequentemente para evitar conflitos com outros usuários e garantir que seu trabalho seja salvo no servidor.
