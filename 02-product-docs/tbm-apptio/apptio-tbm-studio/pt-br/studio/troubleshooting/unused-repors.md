# Relatórios não utilizados (BETA)

Nota:

É necessário **desativar** o recurso Cálculo automático antes de usar esse fluxo de trabalho para desativar vários relatórios de uma só vez.

.

![Cálculo automático](../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/auto-calc.png)

Se você receber uma recomendação de relatórios não utilizados, navegue até o **TBM Studio** > guia **Recommendations** > **Unused Reports** e selecione **Troubleshoot All Identified Problems**.

![Relatórios não utilizados](../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/unused-reports.png)

Mude para o espaço de trabalho **de desenvolvimento** e selecione ![Próxima seta](../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/next-arrow.png) .

![Meio ambiente](../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/unused-reports-development-environment.png)

Você pode ver a descrição do problema e quais ações serão tomadas para resolvê-lo. Você pode visualizar a lista de relatórios que não foram usados recentemente. Recomenda-se revisar e desativar os relatórios que não são mais necessários para esse fluxo de trabalho para evitar cálculos desnecessários.

![Desativar relatórios não utilizados](../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/deactivate-unused-reports.png)

1. Selecione o botão **Fixar** para desativar individualmente.
2. Para desativar vários relatórios simultaneamente, selecione os relatórios desejados e, em seguida, selecione **Deactivate All Selected Reports (Desativar todos os relatórios selecionados** ).
3. Selecione a opção **Também ocultar em coleções de relatórios** para ocultar os relatórios desativados da exibição de coleções de relatórios.

   ![Relatórios desativados](../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/deactivated-reports.png)
4. Selecione ![Próxima seta](../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/next-arrow.png) e, em seguida, **Checkin**.

![](../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/checin-unused-reports.png)
