**Carregamento da tabela de assinaturas de e-mail filtradas**

A distribuição de e-mails foi projetada para otimizar o processo de gerenciamento de várias visualizações filtradas com destinatários exclusivos. Esta atualização é particularmente benéfica para administradores que precisam enviar visualizações personalizadas e filtradas de relatórios por e-mail a muitos usuários sem esforço manual.

Para usar esse recurso, selecione o ícone **Exportar** no modo “Exibir” e selecione **Assinatura por e-mail**. Ative a opção **“Upload em massa XLS”** para baixar um modelo. Este modelo lista todos os filtros disponíveis para o relatório em contexto e fornece um espaço para adicionar destinatários em cada linha. Adicione combinações de filtros exclusivas para cada conjunto de destinatários, salve o arquivo e faça o upload novamente. Ao fazer o upload de um arquivo válido, você observará duas mudanças importantes: o Arquivo de Regras de Assinatura é ativado, indicando que você fez o upload de um arquivo que pode ser baixado para visualização ou edição, e uma mensagem de upload bem-sucedido é exibida. O recurso de upload de tabelas suporta um AND lógico dos filtros aplicados para cada linha. O sistema de validação de dados verifica se as colunas foram adicionadas corretamente, se as colunas de destinatários de e-mail não estão em branco e outros campos essenciais para garantir a integridade dos dados. O campo *Destinatários* é obrigatório, enquanto apenas um dos outros campos é necessário para filtrar o conjunto de dados. A validação de dados concentra-se principalmente nos metadados da tabela (por exemplo: colunas corretas) e não necessariamente nos valores dentro das colunas.
