# Adicionar subtotais a uma tabela

**Aplica-se a** : TBM Studio 12.0 e posterior

Se você arrastar mais de um campo para a área **Rows (Linhas** ) da caixa de diálogo **Configuration (Configuração** ), o aplicativo agrupará automaticamente os valores e apresentará subtotais para cada grupo. Se você classificar a tabela pela coluna **Value (Valor** ), a classificação será feita para os valores de subtotal.

Por exemplo, suponha que você tenha a tabela mostrada na imagem a seguir:

![](../../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/studioimages/reports/rep205.jpg)

Você adiciona uma tabela a um relatório com base na tabela usando as configurações mostradas na imagem a seguir:

![](../../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/studioimages/studio/stu560.png)

O resultado é a tabela mostrada na imagem a seguir:

![](../../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/studioimages/reports/rep206.png)

## Adicionar subtotais

Controle os subtotais usando a caixa de diálogo **Subtotais** na guia **Dados**, mostrada na imagem a seguir. Você pode criar subtotais para até quatro colunas. O campo **Colocar rótulos de subtotal em** controla a coluna em que o rótulo de subtotal é colocado.

![](../../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/studioimages/studio/stu561.png)

## Entradas de grupo

Você pode agrupar entradas de tabela por qualquer um dos campos arrastados para a área **Linhas** do painel **Configuração de componentes**. Para agrupar uma tabela, selecione **Group (Grupo** ) na guia **Data (Dados** ).

Observação: se houver mais de um campo na área **Rows (Linhas** ), você deverá agrupar por pelo menos um dos campos.
