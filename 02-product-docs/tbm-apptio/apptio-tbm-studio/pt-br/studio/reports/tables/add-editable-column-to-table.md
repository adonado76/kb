# Adicionar uma coluna editável a uma tabela

## Sobre esta tarefa

**Aplica-se a** : TBM Studio 12.0 e posterior

As colunas editáveis são colunas em branco nas quais os usuários podem inserir dados. Eles podem ser adicionados somente a tabelas editáveis.

Para adicionar uma coluna editável:

## Procedimento

1. Selecione a tabela editável onde você deseja adicionar a coluna.
2. Abra o menu suspenso do ícone **Inserir** ![](../../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/studioimages/icons/table_column_add.png) na guia **Dados**.
3. Selecione a opção **Insert Editable Column (Inserir coluna editável** ).
4. Quando solicitado, digite um nome para a nova coluna e clique em **OK**.
