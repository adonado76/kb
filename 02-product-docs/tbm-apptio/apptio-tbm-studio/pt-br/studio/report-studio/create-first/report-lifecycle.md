# Ciclo de vida do relatório

As ações do ciclo de vida dos relatórios ajudam os administradores a gerenciar os relatórios após sua criação. Essas ações controlam como os relatórios são criados, atualizados, reutilizados e removidos.

## Criar um relatório

Os administradores podem criar relatórios usando o novo Report Studio.

- Os relatórios são criados a partir da guia Novos relatórios na página inicial.
- O relatório abre diretamente na tela do relatório, onde é possível adicionar componentes e visualizações.
- O salvamento automático está ativado por padrão, e as alterações são salvas à medida que você trabalha.
- Também está disponível uma opção de salvamento manual.

Para obter instruções passo a passo, consulte [Criar seu primeiro relatório](first-report.html "Este guia orienta você na criação de um novo relatório no novo Report Studio. Ao final deste guia, você terá um relatório com componentes e visualizações prontos para visualizar e compartilhar.").

## Editar um relatório

Os administradores podem editar relatórios existentes para atualizar dados, layout ou configuração.

- Você precisa verificar o relatório para visualizar e editar.
- A edição abre o relatório na tela do relatório.
- As alterações são salvas automaticamente usando o recurso de salvamento automático.

## Excluir um relatório

Os administradores podem excluir relatórios que não são mais necessários.

- Os relatórios excluídos são removidos de todas as coleções.
- Os usuários não poderão mais acessar o relatório excluído.

## Configuração do relatório de exportação

Os administradores podem exportar a configuração de um relatório para reutilização.

- A exportação captura a definição do relatório, incluindo componentes e visualizações.
- A configuração exportada pode ser importada para outros projetos ou instâncias.
- Isso é útil para migração, reutilização ou padronização entre ambientes.

## Configuração do relatório de importação

Os administradores podem importar uma configuração de relatório exportada anteriormente.

- Os relatórios importados mantêm sua estrutura e configuração originais.

## Exportar relatório para PDF (Report Studio)

Agora você pode exportar relatórios para PDF diretamente do Novo Report Studio. Isso permite que você gere uma versão compartilhável e pronta para apresentação do seu relatório sem precisar alternar para o Visualizador de Relatórios.

**Como exportar um relatório para PDF?**

1. Abra o relatório no **Novo Estúdio de Relatórios**.
2. Na barra de navegação superior, clique em **Arquivo**.
3. Selecione **Baixar como**.
4. Selecione **PDF**.
5. Selecione “**Baixar PDF diretamente** ”.
6. Clique em **Baixar**.

O arquivo PDF será gerado e baixado para o seu sistema.

**O que é exportado?**

O PDF exportado inclui:

- Todos os componentes visíveis no relatório
- Layout e formatação atuais
- Filtros aplicados refletidos no momento da exportação

Observação: O PDF reflete o estado atual do relatório no momento do download.
