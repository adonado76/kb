# Novo layout do Report Studio

O layout de um relatório novo/em branco é mostrado abaixo:

![](../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/rs-report-layout.png)

|  |  |  |
| --- | --- | --- |
| 1 | Zoom | Ajuste a visualização da tela para dentro ou para fora. |
| 2 | Relatar o status de check-in/check-out | Exibe o status de check-in/check-out do relatório no contexto. |
| 3 | Salvar | Salva o relatório |
| 4 | Painel de dados | Selecione um objeto de modelo e adicione dimensões de dados. |
| 5 | Painel de formato | Personalize propriedades como cor, fonte, tamanho e outras opções de estilo. |
| 6 | Tabelas | Representar fontes de dados estruturados das quais você pode extrair dimensões. |
| 7 | Tabelas editáveis | Representar fontes de dados estruturados das quais você pode extrair dimensões. |
| 8 | Métricas | Representam valores numéricos ou medidas que podem ser usados para análise. |
| 9 | Dimensões do tempo | Dimensões de data e hora que permitem analisar dados entre períodos (diários, mensais, trimestrais, anuais) |
| 22 | Navegação Breadcrumbs | Permite que você saiba exatamente onde está na hierarquia do relatório. |
| 11 | Menu de ações do relatório | Oferece ações importantes como Check-in, Reverter alterações, Desfazer, Refazer, Copiar e Colar, além de controles de layout como grades, réguas, alinhamento e opções de distribuição. |
| 12 | Componentes | Abriga os diferentes componentes de relatório usados para criar o relatório. Inclui Compact Slicers, Quick Pivots, Column Pickers, HTML, Group, Tabs etc. |
| 13 | Visualizações | Lista todas as diferentes visualizações de dados disponíveis. Inclui tabelas, tabelas editáveis, gráficos de barras, gráficos de colunas, gráficos de linhas, gráficos de pizza etc. |
| 14 | Painel de camadas | Mostra todos os componentes do relatório na hierarquia do relatório, permitindo que você os copie, cole, reordene, exclua ou agrupe. |
