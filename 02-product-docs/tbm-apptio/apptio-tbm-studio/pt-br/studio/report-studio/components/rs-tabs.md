# Guias

O componente Guias permite organizar o conteúdo do relatório em várias guias dentro de um único relatório. Cada guia pode conter seu próprio layout de componentes e visualizações, ajudando a estruturar relatórios complexos em seções claras e navegáveis.

## Quando usar guias

Use guias quando desejar:

- Divida um relatório grande ou complexo em seções lógicas
- Separe diferentes visualizações de análise
- Reduza a rolagem e melhore a legibilidade do relatório
- Apresente múltiplas perspectivas sem criar múltiplos relatórios

## Adicionar guias a um relatório

1. Adicione guias a partir do painel Componentes na barra de ferramentas
2. Clique nas guias para ativar os painéis Formato.
3. Painel de formatação
   1. Propriedades gerais – Veja Propriedades do componente
   2. Propriedades específicas da guia
      1. Tipo de controle – Escolha entre guias e botões (em breve)
      2. Estilo da guia global
         1. Selecione a cor da barra de guias
         2. Selecione a cor de fundo da guia
