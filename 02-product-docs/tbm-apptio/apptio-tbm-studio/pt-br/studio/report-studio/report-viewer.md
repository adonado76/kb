# Visão geral do Visualizador de Relatórios

O Visualizador de Relatórios foi projetado para que os usuários finais visualizem, interajam e consumam relatórios criados no Report Studio.

## Acessando o Visualizador de Relatórios

Você pode navegar até o **Novo Visualizador de Relatórios** das seguintes maneiras:

- Na página inicial **Nova página inicial do Report Studio**, use o menu “”**Leve-me para** na barra de navegação superior e selecione **Custeio IBM moderno**.
- No **Visualizador de Relatórios Clássico**, clique no botão azul “**Abrir Novo Visualizador de Rel** atórios” para mudar para a experiência do visualizador moderno.

## Página de destino

![novo visualizador de relatórios](../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/new-rep-viewer.png)

A página inicial do Visualizador de Relatórios exibe todos os relatórios e coleções disponíveis de forma estruturada:

- A guia Relatórios lista todas as coleções de relatórios que foram disponibilizadas aos usuários finais.
- A guia Relatórios clássicos lista os relatórios criados usando a experiência clássica de relatórios.

Para tornar um relatório visível no Visualizador de Relatórios, ele deve ser agrupado em uma coleção de relatórios. Ao selecionar uma coleção, é exibida a lista de relatórios disponíveis nessa coleção.

## Visualizando relatórios e interagindo com os dados

Os usuários finais podem abrir relatórios no Visualizador de Relatórios e interagir com os dados das seguintes maneiras:

- Aplicar filtros usando segmentadores
- Pesquisar e filtrar registros em tabelas
- Interaja com gráficos para explorar os dados visualmente

Todas as interações no Visualizador de Relatórios são não persistentes. Quaisquer alterações feitas durante a visualização de um relatório são baseadas na sessão e não modificam nem salvam a configuração do relatório.
