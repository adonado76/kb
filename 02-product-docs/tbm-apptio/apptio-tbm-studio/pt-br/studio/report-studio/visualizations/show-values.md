# Mostrar valores

O recurso **"Mostrar valores"** ajuda você a entender rapidamente como os dados estão distribuídos em uma coluna. Ele exibe todos **os valores únicos** da coluna selecionada, juntamente com o **número de vezes que cada valor aparece** na tabela.

![imagem do valor de exibição](../../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/sv-1.png)

![imagem da tabela de valores de exibição](../../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/sv-2.png)

**Use isto para**

- Explore um conjunto de dados antes de criar visualizações
- Verificar a qualidade dos dados (duplicatas, valores em branco, valores inesperados)
- Compreender campos categóricos

**Como usar a função "Mostrar valores"**

1. Abrir uma mesa
2. Passe o mouse sobre a coluna que deseja analisar
3. Clique no menu de opções adicionais no cabeçalho da coluna
4. Selecionar "Mostrar valores"

É exibida uma visualização em forma de tabela mostrando:

- Valor – cada valor único na coluna
- Contagem – número de linhas que contêm esse valor

**Tópico principal:** [Tabela](../../../studio/report-studio/visualizations/rs-table.html "O componente de tabela exibe dados em um formato tabular estruturado. É ideal para apresentar informações detalhadas, resumir métricas e permitir a filtragem interativa dentro de um relatório.")
