# Um guia para a navegação no novo Report Studio

Para acessar o novo Report Studio, vá para o Classic TBM Studio em seu ambiente principal.

**Procure o botão azul “Abrir o IBM Apptio Report Studio”** – Assim que sua instância for atualizada para a versão 12.11.21 ou superior, você verá um botão azul na parte superior do seu Report Studio atual. Ao clicar aqui, será aberta uma lista de projetos disponíveis.

Observação: os clientes que utilizam as versões 12.11.19 devem estar inscritos na pré-visualização pública para ver o botão azul.

![imagem do novo botão do estúdio de relatórios](../../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/nrs-button.png)

**Selecione um projeto para abri-lo no novo Report Studio** – Escolha um projeto no menu suspenso para iniciá-lo automaticamente na nova experiência.

O novo Report Studio abre na página inicial, conforme mostrado abaixo.

**Página inicial de relatórios**

![navegação da página inicial no novo estúdio de relatórios](../../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/nrs-home.png)

1. Novo botão – Clique aqui para criar um novo relatório ou uma nova coleção de relatórios
2. Pesquisar – Pesquise um relatório existente
3. Novos relatórios – Lista todos os relatórios criados usando o novo Report Studio.
4. Coleções de relatórios – Exibe grupos de relatórios relacionados organizados em coleções.
5. Menu de excesso de relatórios (menu de três pontos) - Clique aqui para acessar uma série de operações que podem ser realizadas em um relatório, como excluir um relatório ou exportar a definição e a configuração do relatório.
6. Configurações do projeto – Clique aqui para verificar um ou mais relatórios, reverter alterações neles ou importar uma configuração de relatório para o projeto em contexto.

**Painel de navegação**

![navegação do painel superior no novo estúdio de relatórios](../../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/nrs-navpan.png)

1. App Switcher – Use isso para alternar para outro produto IBM Apptio
2. Configurações do perfil - Use isso para gerenciar seu perfil, se passar por outro usuário (Observação: os administradores podem querer fazer isso para verificar as permissões de um usuário ou solucionar problemas) e sair da conta
3. Ajuda – Use isso para acessar a nova documentação de ajuda do Report Studio
4. Configurações - Use isso para alterar o acesso do cliente, atualizar permissões e acessar configurações de várias moedas
5. Menu suspenso Ambiente – Use isso para alternar entre os ambientes e as áreas de trabalho do usuário.
6. Seletor de data – Use isso para alternar entre períodos de datas
7. Leve-me para – Use este menu de navegação rápida para alternar para o TBM Studio Clássico e/ou o Novo Visualizador de Relatórios.
