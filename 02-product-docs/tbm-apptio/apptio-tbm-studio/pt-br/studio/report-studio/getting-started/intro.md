# Visão geral do novo Report Studio

O novo **IBM Apptio Report Studio**, com design em carbono, oferece uma experiência de relatórios modernizada, intuitiva e mais poderosa. Ele foi projetado para ajudar os TBMA e usuários empresariais a criar relatórios mais rapidamente, com menos complexidade e uma curva de aprendizado mais curta. Em comparação com o Classic Report Studio, esta versão traz:

- Tempo mais rápido para obter valor: Crie e visualize relatórios com mais rapidez, ajudando as equipes a tomar decisões mais rapidamente.
- Interface moderna e intuitiva: Um design de arrastar e soltar que é fácil de aprender e usar.
- Acessibilidade aprimorada: Otimizado para vários tamanhos de tela e dispositivos.
- Criação simplificada de relatórios: Redução do tempo de configuração com padrões inteligentes e componentes reutilizáveis.
- Ferramentas flexíveis de layout e visualização: Crie relatórios perfeitos com opções avançadas de gráficos e controles de layout.

## O que você pode fazer com o Novo Report Studio

Na nova experiência, você pode:

- Crie relatórios usando componentes e visualizações configuráveis
- Organize relatórios em coleções para facilitar a localização
- Interaja com relatórios usando filtros e seletores
- Visualizar dados em várias moedas
- Exportar relatórios para PDF
- Exportar dados da tabela para análise posterior

## Para quem é o New Report Studio?

O Report Studio foi projetado para duas funções principais do usuário:

**Administradores (autores de relatórios)**

Os administradores criam e gerenciam relatórios. Eles configuram componentes, visualizações e layouts para fornecer insights significativos ao seu público.

**Usuários finais (visualizadores de relatórios)**

Os usuários finais consomem relatórios criados pelos administradores. Eles se concentram em compreender os dados e exportá-los quando necessário.

Para obter mais detalhes, consulte [Funções e permissões do usuário](user-role-perm.html "O New Report Studio suporta diferentes funções de usuário para garantir o nível certo de acesso para criar, gerenciar e visualizar relatórios. Esta página explica as funções disponíveis e o que cada função pode fazer.").
