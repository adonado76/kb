# Funções e permissões do usuário

O New Report Studio suporta diferentes funções de usuário para garantir o nível certo de acesso para criar, gerenciar e visualizar relatórios. Esta página explica as funções disponíveis e o que cada função pode fazer.

Compreender as funções ajuda você a saber **quem pode criar e configurar relatórios e quem pode visualizá-los e interagir com eles.**

Existem duas funções principais no Report Studio:

- Admin – Cria e gerencia relatórios e coleções de relatórios
- Usuário final – Visualiza e interage com relatórios

**Função de administrador**

Os administradores são responsáveis pela **criação e manutenção** dos relatórios. Eles também podem

- Criar, editar, duplicar e excluir relatórios
- Criar e gerenciar coleções de relatórios
- Adicione e configure componentes (tabelas, segmentadores, KPIs, etc.)
- Configurar visualizações e definições de gráficos
- Defina e gerencie fontes de dados, métricas e filtros
- Configurar layout e formatação
- Visualizar relatórios no Visualizador de Relatórios
- Exportar relatórios para PDF
- Visualizar dados em várias moedas

**Função do usuário final**

Os usuários finais consomem relatórios criados pelos administradores. Eles se concentram em **visualizar, interagir e exportar** dados de relatórios. Eles também podem

- Visualizar relatórios e coleções de relatórios
- Navegue entre relatórios
- Interaja com filtros e seletores
- Exportar relatórios para PDF
- Visualizar dados em várias moedas
