# Como funciona o novo Report Studio

O Novo Estúdio de Relatórios segue um fluxo de trabalho simples e estruturado que separa a criação de relatórios do consumo de relatórios. Isso garante que os relatórios sejam fáceis de criar, manter e usar em grande escala.

Esta página explica o fluxo completo e como diferentes usuários interagem com o Report Studio.

**Fluxo de trabalho de alto nível**

O Report Studio funciona em quatro etapas principais:

1. Criar…
2. Configurar
3. Organizar
4. Visualizar e interagir

Cada etapa se baseia na anterior.

**Criar relatórios**

Os administradores começam criando um relatório.

Um relatório define:

- Os dados a serem exibidos
- Os componentes utilizados (tabelas, KPIs, segmentadores, etc.)
- As visualizações que representam os dados

**Configurar componentes e visualizações**

Depois que um relatório é criado, os administradores configuram como os dados aparecem e se comportam.

Isso inclui:

- Seleção de fontes de dados, métricas e dimensões
- Configurando as propriedades dos componentes
- Configurando visualizações (gráficos, tabelas, layouts)
- Definindo interações entre componentes (Ex.: filtros que afetam gráficos)

As opções de configuração comuns que se aplicam a todos os componentes e visualizações são documentadas separadamente para evitar duplicação.

**Organize relatórios em coleções**

Após a criação dos relatórios, os administradores podem agrupar relatórios relacionados em coleções de relatórios.

Ajuda para relatórios coletivos:

- Organize relatórios por tema, público-alvo ou caso de uso
- Melhorar a visibilidade para os usuários finais
- Defina permissões em uma coleção para controlar quais usuários podem ver os relatórios nela contidos

**Visualizar e interagir com relatórios**

Os usuários finais acessam os relatórios por meio do Visualizador de Relatórios. No Visualizador de Relatórios, os usuários podem:

- Navegue entre relatórios e coleções de relatórios
- Interaja com filtros e seletores
- Visualizar dados em várias moedas
- Exportar relatórios para PDF

**Funções no fluxo de trabalho**

Diferentes funções participam em diferentes etapas:

- Administradores – Envolvidos principalmente na criação, configuração e organização de relatórios.
- Usuários finais – Envolvidos principalmente na visualização, interação e exportação de relatórios.
