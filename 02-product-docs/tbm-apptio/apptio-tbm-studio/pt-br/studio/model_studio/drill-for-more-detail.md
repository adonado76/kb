# Perfurar para obter mais detalhes

**Aplica-se a** : TBM Studio 12.0 e posterior

Ao trabalhar em um modelo, você pode obter mais detalhes sobre a origem do valor de uma tabela, pesquisando uma coluna específica em uma tabela de origem. Por exemplo, suponha que você tenha o modelo mostrado na imagem a seguir. Você quer descobrir quanto valor está sendo contribuído para os US$ 195.750 por cada um dos datacenters.

![](../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/studioimages/studio/stu567.png)

Para detalhar:

1. Clique em **≡** e clique em **Drill Down**.
2. Selecione uma coluna em uma tabela. Em nosso exemplo, podemos selecionar a coluna **Data Center** na tabela **Data Centers Info.** O modelo agora exibe os data centers, conforme mostrado na imagem a seguir:![](../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/studioimages/studio/stu568.png)
3. Clicar em um data center mostrará a alocação do GL para esse data center e a alocação do data center para o serviço.
