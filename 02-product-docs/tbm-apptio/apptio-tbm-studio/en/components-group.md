# Group

*Grouping feature allows you to easily organize multiple components in your report layout by grouping or ungrouping them. This helps manage related components as a single unit.*

## When to use Groups

- Keep related content visually together
- Move multiple components together
- Simplify complex layouts

## Add a Group to the Report

1. Create group by multi-selecting components Press and hold the Shift key, then click to select multiple components on the canvas. Press Ctrl + G (Cmd + G on Mac) to group the selected components. (You’ll see the grouped components appear as a nested hierarchy in the Layer panel.)
1. A Group icon is also available on the toolbar – clicking it performs the same action.
1. Create a framed group Add a Group from the Components pane on the toolbar Format Panel General Properties – See Component Properties
1. Ungroup To ungroup, select the grouped component and click the Group icon again (it acts as Ungroup when a group is selected). The hierarchy updates in the Layer panel, showing individual components again.
