# Table Column Overflow Menu

Each column in a table has an overflow menu that provides additional actions to manage column behaviour and layout:

- Sort Ascending – Sorts the column values in ascending order.
- Sort Descending – Sorts the column values in descending order.
- Pin Column – Pins the column to the left so that it stays visible while scrolling.
- Auto size Column – Automatically adjusts the column width to fit its content.
- Choose Columns – Allows selection of which columns are visible in the table.
- Reset Columns – Restores the table columns to their default order and visibility.
- Group Columns – Groups one or more columns
- Insert Spacer Column – Adds an empty column to the right to help with layout spacing
