# Pie Charts

*A pie chart displays data as slices of a circle, showing the relative proportions of each category to the whole. It is ideal for illustrating percentage or part-to-whole relationships.*

## When to use a Pie Chart

- Show proportions of a total for a small number of categories
- Highlight the largest or smallest segments
- Provide a quick visual comparison of parts to the whole

## Add a Pie Chart to a Report

1. Add a Pie Chart from the Visualizations pane on the toolbar
1. Click on the Pie Chart to enable the Data and Format panels.
1. Data Panel Select the model object from the drop-down menu Categories – Defines the slices of the pie using a dimension. Click here or drag to add dimensions from the Dimension Explorer Values – Specifies the metric(s) that determine slice size Filters – Limits the data displayed in the chart based on selected conditions
1. Format Panel General Properties – See Component Properties Pie chart specific Properties Total Toggle to show the Total count. Font size and style for the legend (bold, italics, underline) Color of the total text (with option to reset the color) Data Labels Toggle to show the Label position – the options are Outside or Inside. Label content – Choose from the options - Category, Value, and Percentage. Choose the Font size, style (bold, italics, underline) and color Set colors automatically – automatically assigns colors to bars based on the selected data and theme. Define a label outline and color Legend Toggle to show the legend Font size and style for the legend (bold, italics, underline) Color of the legend text (with option to reset the color)
