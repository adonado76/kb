# Diagram Navigation

Navigation Actions

| Action | How To | Result |
| --- | --- | --- |
| Action | How To | Result |
| View object details | Click a node | Opens Single-Table view for that object |
| Navigate to target | Click arrow on allocation driver | Jumps to the source object |
| Trace cost flow | Click an allocation line | Highlights the cost path through the model |
| Drill down | Click menu > Drill Down | Expands node to show column values |
| Return to table view | Click Close or return arrow | Collapses drill-down and returns to object view |
| Zoom | Mouse wheel or zoom controls | Zoom in/out on diagram |
| Pan | Click and drag background | Moves the diagram view |
