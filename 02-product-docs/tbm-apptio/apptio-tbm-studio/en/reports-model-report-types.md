# Model Report Types

Table-Based Reports

Show entire tables in tiers. End users can drill down to see column values.

Use Cases

- Allow users to explore different dimensions (cost pools, vendors, etc.)
- Show comprehensive view of a model layer
- Provide flexibility for ad-hoc analysis

Column-Based Reports

Show specific columns, providing a focused view of particular dimensions.

Use Cases

- Focus report on specific elements (e.g., Cost Pool only)
- Provide consistent view for executive reporting
- Reduce complexity for specific audiences
