# Table Operations

## Column Operations

| Operation | Process |
| --- | --- |
| Operation | Process |
| Add a Row | Click Add Row button at bottom of table |
| Filter by Value | Right-click cell > Filter by Value |
| Freeze Column | Right-click header > Formatting > Freeze Column |
| Set Background Color | Right-click header > Formatting > Set Background Color |
| Set Column Width | Drag border or right-click > Formatting > Set Width |
| Show Values (Duplicates) | Right-click header > Show Values |

## Truncate (Editable Tables)

Applies to: 12.11.4 and later

| Table Type | Truncate Effect |
| --- | --- |
| Table Type | Truncate Effect |
| Enriched Editable Table | Reverts to original source values |
| General Editable Table | Deletes all rows |

Warning: Truncation is permanent after check-in. Revert before check-in if needed.
