# Adding Transform Steps

- Select a table from Project Explorer
- Move cursor to the dividing line between two existing steps
- Click the icon at the right edge of the Transform Steps panel
- Select the step type to add

Reordering: Drag steps to new positions (except Source, Upload, Import, Table, Model)
