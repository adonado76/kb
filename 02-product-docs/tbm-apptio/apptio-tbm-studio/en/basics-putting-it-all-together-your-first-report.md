# Putting It All Together: Your First Report

Now that you understand the basics, here's a typical workflow for creating a report:

- Create the report with a clear name and description
- Add a table to show detailed data—choose the right model object, add dimensions to Rows, add metrics to Values
- Add a chart to visualize key patterns—choose the right chart type for your message
- Add slicers for interactivity—include slicers for dimensions users commonly filter by
- Review and check in —verify the report displays correctly, test slicer interactions, check in to save your changes
