# Create Your First Report

*This guide walks through creating a new report in the new Report Studio. By the end of this guide, you will have a report with components and visualizations ready to view and share.*

1. Navigate to the New Reports Tab Go to the New Reports tab on the landing page Click the New button and select Report
1. Enter Report Details Report Name – Give your report a meaningful name. Description (Optional) - Add a short description to help users understand the report’s purpose. Viewable by – This property controls who can see and access the report. Everyone – The report is visible to all users with access to the reporting area. Only Me – The report is visible only to you. Useful for drafts, experimentation, or personal analysis. Select Roles – The report is visible only to users assigned to the selected roles. Roles are selected from a drop-down list. Enables role-based access control for reports.
1. Add Components On the toolbar, select a component from the Components menu. Choose from the available component types, such as: Slicer Column Picker Quick Pivot HTML Tabs Group Pick a Slicer When the slicer is added, the Data and Format panels open on the right. Data panel is used to configure the data that drives the slicer. Format panel is used to customize the appearance and formatting of the slicer. Configure data In the data panel, select a data model object from the drop-down. Click “Click here or drag to add data” to open the Dimension Explorer. The Dimension Explorer displays Tables, Editable Tables, Metrics and Time. Choose the dimension you want to slice the data by. Drag a dimension from Tables into the Data panel. Use the filters section to apply additional filtering criteria to the slicer. Once configured, the slicer filters the report based on the selected dimension.
1. Add Visualizations On the toolbar, select a visualization from the Visualizations menu Choose from the available visualizations: KPI Table Editable Table Bar chart Stacked Bar chart Column chart Stacked Column chart Line chart Pie chart Select Table Configure data In the data panel, select a data model object from the drop-down. Configure the Rows, Columns, Values sections of the data panel by adding dimensions directly from the Dimension Explorer. Use the filters section to apply additional filtering criteria to the table.
1. Save your Report Since Autosave is enabled, your changes will be saved automatically as you build the report. You can also manually click Save at any time to ensure your work is saved immediately.
