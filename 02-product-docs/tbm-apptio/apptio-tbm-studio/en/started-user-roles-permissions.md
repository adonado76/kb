# User Roles & Permissions

*The New Report Studio supports different user roles to ensure the right level of access for creating, managing, and viewing reports. This page explains the roles available and what each role can do.*

Understanding roles help you know who can create and configure reports and who can view and interact with them.

- Admin – Creates and manages reports and report collections
- End User – Views and interacts with reports

- Create, edit, duplicate, and delete reports
- Create and manage report collections
- Add and configure components (tables, slicers, KPIs etc.)
- Configure visualizations and chart settings
- Define and manage data sources, metrics and filters
- Configure layout and formatting
- Preview reports in the Report Viewer
- Export reports to PDF
- View data in multiple currencies

- View reports and report collections
- Navigate between reports
- Interact with filters and pickers
- Export reports to PDF
- View data in multiple currencies
