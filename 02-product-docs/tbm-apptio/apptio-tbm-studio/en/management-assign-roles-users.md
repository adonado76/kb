# Assign Roles to Users

Objective: Assign appropriate roles to users to control their access level in TBM Studio.

When to use: When creating new users, when user responsibilities change, or when adjusting access levels.

Time estimate: 2-5 minutes per user

Prerequisites

- Admin role or custom role with role assignment permissions.
- The user account must already exist (see Create and Manage Users ).
- Understanding the user's job responsibilities and required access level.

## Steps

1. Access Enhanced Access Administration as described in the Create Users section.
1. Locate the user. Use the search function or scroll through the Users list to find the user.
1. Open user details. Click on the user's name or the edit icon to access their profile.
1. Navigate to Role assignments. Find the section showing the user's current roles and available roles.
1. Assign roles: Select the role(s) appropriate for the user's job function. Users can have multiple roles assigned simultaneously. Permissions are additive—if a user has multiple roles, they receive the combined permissions of all assigned roles.
1. Save changes. Click Save to apply the role assignments.

## Expected Results

The user's access level changes immediately. The next time they log in (or after refreshing their browser if already logged in), they will see the interface and features corresponding to their assigned roles.

## Role Assignment Best Practices

- Principle of least privilege: Assign the minimum role needed for users to perform their job functions.
- Document role assignments: Maintain a record of who has which roles and why for audit purposes.
- Regular reviews: Periodically review role assignments to ensure they still match job responsibilities.
- Limit Admin roles: Only assign the Admin role to users who genuinely need full administrative access.

## Enabling Available Roles for Users

Administrators can enable the Available Roles feature for specific users, allowing them to switch between assigned roles themselves. This is useful for users who need different access levels for different tasks.

To enable Available Roles for a user:

1. Assign multiple roles to the user.
1. Enable the Available Roles option in their user profile.
1. The user can then change their active role by clicking their username in the upper-right corner, selecting My Account , and choosing from the Current Role dropdown.
