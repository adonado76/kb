# ProjectExists function

*Applies to : TBM Studio 12.0 and later*

If the specified project exists, the function returns "true." If the name project does not exist, the function returns "false."

## Where to use

- Data sets
- Calculated metrics and reports with metric columns
- Formula columns in report tables
- Dynamic text

## Syntax

ProjectExists(domain:project)

## Return type

Text
