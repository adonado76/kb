# Hybrid IT TCO Impact – Admin

This is a replacement of the monthly Apptio Datalink Connector used to "freeze" any
migrated apps with their TCO, Volume and Unit Cost. Instructions are available in the admin
report.

Hybrid IT TCO Impact Admin report has been created with below tabs and added to the Hybrid IT
Report Collection :

- Applications Migrated - Summary

  ![img](../../../../../03-media/apptio-tbm-studio/resources/images/ct_images/hyb-am-sum.png)
- Applications Migrated - Cost Pool Drivers

  ![img](../../../../../03-media/apptio-tbm-studio/resources/images/ct_images/hyb-amcpd.png)
- Applications Migrated - IT Tower Drivers

  ![img](../../../../../03-media/apptio-tbm-studio/resources/images/ct_images/hyb-am-itd.png)

Once copy table button has been clicked below input table appears and the values are copied from
the report to the respective tables, Values are same in both table and report after copy
successful
