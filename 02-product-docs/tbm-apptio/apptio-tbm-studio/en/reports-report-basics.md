# Report Basics

Content Type: How-To Guide

Target Audience: Business Analysts, IT Finance Teams, All Users

Prerequisites: Access to a TBM Studio project with model data; familiarity with the TBM Studio interface

Overview

Reports in TBM Studio transform your model data into visual, interactive dashboards that stakeholders can use to explore costs, track budgets, and make informed decisions. This section covers the fundamental skills every report creator needs: creating reports, adding data tables, building charts, and enabling interactivity through slicers.

Report Component Model

Reports are containers that hold components. Think of a report as a canvas where you place building blocks:

- Tables display data in rows and columns, ideal for detailed analysis
- Charts visualize data patterns and comparisons at a glance
- Slicers let users filter what they see without editing the report

When you master these three component types, you can build reports that serve a wide range of business needs.
