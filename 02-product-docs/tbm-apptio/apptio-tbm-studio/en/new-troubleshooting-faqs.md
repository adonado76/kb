# Troubleshooting & FAQs

## General FAQs

What is the context?

- CT Report Studio (one of the components of TBM Studio) needs to be “modernized” to deliver a simple and easy user experience.
- The intended outcome with this modernization project is to alleviate the challenges experienced by our users in learning the tool, navigating within the tool, and gleaning insights from the information presented by the tool.

- Simplifies Onboarding and TTV
- Improves admin and end-user onboarding times by providing a smart, intuitive platform that anticipates user needs, automatically presenting essential tools and configuration options based on their immediate actions
- Reduces number of clicks and guides report generation
- Leads to quicker report creation; creates polished and consistent reports with less effort
- Uses a diverse library of visual components to quickly and effectively communicate information and insights

Why hasn’t Model or Data Studio been implemented, and when will it be implemented?

Given the need to tackle the most critical aspects of our current user pain points, Report Studio and End User capabilities have been prioritized. We will implement a seamless experience for both Model and Data Studio in the upcoming phases.

- Perspectives - Functionality will be replaced by Published Formulas
- Global Slicers – Functionality will be replaced by the Saved Views feature.
- Legacy Tables in Tables and Editable Tables
- Editable Tables - "Set All Rows To" & "Add Row Dialog functions"
- Report Start Date
- Refresh Settings - Option to set manual refresh or auto-refresh every 3 seconds
- Notes Component – Previously allowed users to create notes on reports and control visibility (Ex: "Only me" or specific roles).
- Blueprints – Template-based report creation

## Report FAQs

Why is my report taking longer to load in the new Report Studio?

During the Public Preview, V2 reports are not pre-calculated. This means the system retrieves and processes the data on demand each time you open or refresh a report. As a result, you may experience slower load times compared to the Classic experience.
