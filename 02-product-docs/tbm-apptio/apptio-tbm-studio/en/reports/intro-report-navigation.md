# Report navigation components

Applies to: TBM Studio 12.0 and later

To enable users to navigate between reports, you can add buttons, coded links, report
collections, and report links to reports.

## Buttons

When clicked, a button takes the user to a specific report. See [Button](button.htm "(Opens in a new tab or window)").

## Coded links

When clicked, coded links display a specific report. The links use Wiki-style syntax and must be
entered in an HTML text box or a button. See [Coding links to other reports](coding-links-to-other-reports.htm "(Opens in a new tab or window)").

## Report collection navigator

Report collections group related reports. A report collection navigator is used to display the
reports in the collection. The following image shows an IT Finance report collection navigator. See
[Create report
collections.](creating-report-collections.htm "(Opens in a new tab or window)")

![](../../resources/images/studio_images/studioimages/studio/stu610.png)

## Report links

Links can be added to tables and charts in reports. When a user clicks an entry in a table or an
element on a chart, they are taken to a designated report. Links can include filters. For
information on links, see [Link to
reports](link-to-reports.htm "(Opens in a new tab or window)").
