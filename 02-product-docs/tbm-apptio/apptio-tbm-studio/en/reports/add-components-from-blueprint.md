# Add components from a blueprint

## About this task

**Applies to**: TBM Studio 12.0 and later

Blueprints are a complete set of report components. You can insert the blueprint components in a
report. The advantage of a blueprint is that you can use it to create a series of reports that all
have the same components. Blueprints have been created for several of the Costing Standard
modules.

To insert a blueprint:

## Procedure

1. Open a new report.
2. On the Insert tab, click **Blueprint**.
3. Select a blueprint and click **OK**.
