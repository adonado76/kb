# Add a total column to a table

Applies to: TBM Studio 12.0 and later

You can add a table total column and total columns for groups to a table. A table total column
shows the total for all visible columns in the table. Hidden columns are not included in the total.
In the following image, a total column is displayed for each quarterly group, and a total column is
displayed for the rows:

![](../../../resources/images/studio_images/studioimages/reports/rep270.png)

*Figure A. A table displaying total columns for each group and a total column for an entire
row*

Note: If a table has total columns, and you convert the table to a bar chart, the total columns will
not be included in the chart.

## Add a total row column

1. Select the table.
2. From the Data tab, click Sum.
3. Click Show Total Column.

## Add group total columns

1. Select the table.
2. From the Data tab, click Sum.
3. Click Show Group Total Columns.
