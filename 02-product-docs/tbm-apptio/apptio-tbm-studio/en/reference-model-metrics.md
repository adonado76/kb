# Model Metrics

Metrics are calculations that define measurements for reports. They represent values like cost, budget, forecast, or custom calculations that flow through the model and are displayed in reports.

## Standard Metrics

TBM Studio includes several standard metric types commonly used in cost models.

Metric Types

| Metric | Purpose | Common Sources |
| --- | --- | --- |
| Metric | Purpose | Common Sources |
| Cost | Actual incurred expenses | GL actuals, invoices, asset depreciation |
| Budget | Planned expenses | Budget files, planning systems |
| Forecast | Projected future expenses | Forecast files, projection models |
| Variance | Difference between metrics | Calculated: Budget - Cost |
| Volume/Quantity | Non-monetary counts | User counts, transaction volumes |
