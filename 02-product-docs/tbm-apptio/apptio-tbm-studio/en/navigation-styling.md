# Navigation Styling

Consistent styling helps users recognize navigation elements and understand their purpose. Apply these styling guidelines to create a professional, intuitive navigation experience.

Button Styling Best Practices

- Use consistent colors for similar actions (e.g., blue for navigation, green for approval)
- Apply appropriate text sizes for button importance (larger for primary actions)
- Add meaningful tooltips to explain button actions
- Group related buttons together using group boxes

Link Styling

Report links in tables automatically display as blue text, following standard web conventions. In HTML text boxes, you can apply custom CSS styling to links, though this requires knowledge of HTML/CSS.

Group Box Navigation Containers

Use group boxes to organize multiple navigation elements together. Group boxes can have titles, borders, and background colors that visually separate navigation from report content.
