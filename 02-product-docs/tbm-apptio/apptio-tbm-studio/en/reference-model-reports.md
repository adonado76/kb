# Model Reports

Model reports provide end-user access to model visualizations, showing cost flows in a configurable tier-based Sankey diagram.

## Sankey Diagram Configuration

Creating a Model Report

1. On the Home tab, click New > Model Report
1. Click Edit Metrics to select which metrics to include
1. Add tiers using the Tier Editor
1. Drag tables or columns into tiers from Project Explorer
1. Name each tier
1. Save and check in the report

Tier Editor Operations

| Operation | How To |
| --- | --- |
| Operation | How To |
| Display Tier Editor | Click View menu > Show Tier Editor |
| Add a tier | Click the Add Tier icon (+) |
| Delete a tier | Click the Delete Tier icon (x) |
| Reorder tiers | Click Rearrange icon in tier header and drag |
| Add table to tier | Drag table from Project Explorer to tier |
| Add column to tier | Expand table in Project Explorer, drag column to tier |
| Name a tier | Click in Name field at top of tier and enter name |
| Return to report view | Click View menu > Show Document |

Tier Content Rules

- A tier can contain either tables OR a single column, not both
- Multiple tables can be added to a single tier
- If a column is added, no other elements can be added to that tier
- Multiple tiers can use columns from the same table
- Only modeled tables can be added to tiers
