# Tabs

*The Tabs component lets you organize report content into multiple tabs within a single report. Each tab can contain its own layout of components and visualizations, helping structure complex reports into clear, navigable sections.*

## When to use Tabs

Use tabs when you want to:

- Break a large or complex report into logical sections
- Separate different analysis views
- Reduce scrolling and improve report readability
- Present multiple perspectives without creating multiple reports

## Add Tabs to a Report

1. Add Tabs from the Components pane on the toolbar
1. Click on the Tabs to enable the Format panels.
1. Format Panel General Properties – See Component Properties Tab-specific Properties Control Type – Choose between tabs and buttons (coming soon) Global tab styling Select the tab bar color Select the tab background color
