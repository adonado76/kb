# Data Studio Reference

## Overview

Data Studio is the data management environment in TBM Studio where you import, transform, and prepare data for cost modeling and reporting. This reference provides comprehensive documentation of all Data Studio features, organized for quick lookup.

Data Studio enables you to:

- Import data from files, existing tables, editable tables, and external systems
- Transform data through a configurable pipeline of steps
- Map data to master data sets for application compliance
- Publish tables for export to external systems
- Apply row-level security for data governance
