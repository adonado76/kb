# Creating Master Reports

Creating a New Master Report

To create a new master report:

1. Create a new report or check out an existing report that you want to convert to a master.
1. On the Report tab, in the Advanced group, click Master Report to enable the master report option.
1. Add the components you want to appear in all child reports (group boxes, navigation buttons, branding elements, etc.).
1. Click Save on the Home tab.
1. Check in the report to make it available to other reports.

Converting an Existing Report to Master

Any existing report can be converted to a master report:

1. Check out the report you want to use as a master.
1. Navigate to the Report tab in the ribbon.
1. In the Advanced group, click the Master Report option.
1. Save and check in the report.

Standard Master Reports

TBM Studio includes several standard master reports that ship with the product. These pre-built masters are available in the Masters drop-down menu on the Report tab. You can use these standard master reports as-is or modify them and save them with a new name using the Save As option.
