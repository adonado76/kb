# Show Values

The Show Values feature helps you quickly understand how data is distributed within a column. It displays all unique values in the selected column along with the number of times each value appears in the table.

Use this to

- Explore a dataset before creating visualisations
- Check data cleanliness (duplicates, blanks, unexpected values)
- Understand categorical fields

How to use Show Values

1. Open a table
1. Hover over the column you want to analyse
1. Click the overflow menu on the column header
1. Select Show Values

A tabular view opens showing:

- Value – each unique value in the column
- Count – number of rows containing that value
