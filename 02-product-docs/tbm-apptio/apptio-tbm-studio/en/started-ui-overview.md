## Overview

TBM Studio provides a streamlined interface designed to help you move efficiently between Data Studio, Model Studio, and Report Studio. This reference guide shows you where to find key features and how to navigate the interface.

What you'll learn:

- Main navigation areas and their purpose
- Different environments
- How to work with projects and documents
- Essential keyboard shortcuts
- Interface customization options

## What’s Next

Once you understand the interfaces, you can:

- Complete the Quick Start Guide to build your first project
- Explore Data Studio to learn about data management
- Try Keyboard Shortcuts to improve your efficiency
