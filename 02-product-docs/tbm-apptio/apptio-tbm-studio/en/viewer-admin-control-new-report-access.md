# Admin Control for New Report Viewer Access

*Admins can control whether end users have access to the New Report Viewer within the Report Studio. This setting allows teams to manage visibility of reports based on their organization’s needs.*

By default, the New Report Viewer is enabled, allowing end users to view reports seamlessly.

## When to Use

Use this setting when:

- You want to restrict report access for end users
- You are testing or configuring reports before making them visible
- You want tighter control over who can view reports in your project

## Enable or Disable New Report Viewer for End Users

To manage this setting:

1. Navigate to the Project tab in the Classic TBM Studio
1. Open the Enable Features dropdown
1. Locate the option:
1. “Enable New Report Viewer for End User”
1. Use the checkbox to: Enable (default): End users can view reports Disable: End users will not have access to the Report Viewer

- This setting is enabled by default for all projects
- Disabling this setting only affects end user access, not admin capabilities
- Changes take effect immediately
