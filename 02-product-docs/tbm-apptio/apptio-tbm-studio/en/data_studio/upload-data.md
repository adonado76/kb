# Upload data

**Applies to**: TBM Studio 12.0 and later

When you upload data, you upload the data into a table. You can upload data using one of the
following methods:

- **File Upload**: Select a file from a local or network location in one of the following
  formats: Excel (xls, xlsx), .csv, XML, Zip ([How?](upload-data-file.dita "(Opens in a new tab or window)")).
- **Enter Manually**: Enter the data manually in a table ([How?](enter-data-manually.dita "(Opens in a new tab or window)")).
- **Generated Table**: Create a new table from an existing table in the current project. The
  new table will NOT be linked to the existing table.
- **Existing Table**: Create a new table from an existing table in the current project. The new
  table WILL be linked to the existing table.
- **Datalink (Classic) Connector**: Import a table using a predefined Datalink (Classic)
  connector.
- **Imported Table**: Import the data from an existing Costing Standard project.

Before you can upload data, you must create a table that will receive the data ([How?](create-table.dita "(Opens in a new tab or window)")).
