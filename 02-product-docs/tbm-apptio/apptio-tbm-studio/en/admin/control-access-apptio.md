# Control access to Apptio applications

Applies to: TBM Studio v12.1, v12.2 and later

To control access to Apptio applications, use Enhanced Access Administration, the Apptio Access
Administration page. The parts of the system a user can access and the actions a user can perform
are controlled by roles. By default, during maintenance periods users are prevented from logging
into the system. You can override this setting. For information on using Enhanced Access
Administration, see [About
Frontdoor](../../frontdoor/home.htm "(Opens in a new tab or window)").
