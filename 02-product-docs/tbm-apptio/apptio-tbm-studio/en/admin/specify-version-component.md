# Specify a version for component upgrades

◆ Applies to: TBM Studio 12.3.1 and later. When upgrading applications within TBM Studio,
you can specify which version to use for component upgrades. This setting applies to all component
upgrades and new component installations.

To set your component version:

1. On the Project tab, click Project Settings.The Edit Project Settings dialog
   box opens.
2. In the Component Version field, select the component version you want. The default value
   is the component version that was applied when the project was created or the last time the
   Project Settings was changed.
