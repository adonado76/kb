# Advanced Modeling

| Content Type | How-To Guides |
| --- | --- |
| Content Type | How-To Guides |
| Target Audience | IT Finance Teams, Business Analysts (intermediate to advanced) |
| Prerequisites | Completed 3.2.1 (Model Basics) and 3.2.2 (Create Allocations); familiarity with simple weighted value and even distribution allocations |

This section covers advanced modeling techniques for users who have mastered basic allocations and need more precise control over how costs flow through their model. You will learn how to use matching logic to dynamically connect source and target rows, configure weighting columns for proportional distribution, trace cost flows for troubleshooting, and create model reports (Sankey diagrams) for stakeholder communication.
