# Transform & Enrich Data

| Content Type | How-To Guides |
| --- | --- |
| Content Type | How-To Guides |
| Target Audience | Business Analysts, IT Finance Teams |
| Prerequisites | Completion of Section Data Import & Management, understanding of table transforms |

This section covers essential data transformation tasks in TBM Studio. These guides help you clean, enrich, and combine data to prepare for cost modeling and reporting. Each transformation is performed using pipeline steps that are executed sequentially from top to bottom.

In this section:

- Clean and map data using Map Columns (15-20 minutes)
- Join data from multiple sources (15-20 minutes)
- Apply formulas to transform data (10-15 minutes)
- Filter data by date partition (10-15 minutes)
- Append multiple data sources (15-20 minutes)
