# Recent Releases

## Current Release (12.11.23) - July 2026

Heatmaps: Introduced a new Heatmap visualization in the New Report Studio to display data intensity across two dimensions using color gradients. With configurable color scales and aggregated measure values, heatmaps make it easy to identify patterns, trends, anomalies, and hotspots in large datasets.

Bubble charts: Introduced a new Bubble Chart visualization in the New Report Studio to analyse relationships across multiple measures by plotting data points on X and Y axes while representing a third measure through bubble size. Bubble charts enable users to identify patterns, correlations, clusters, and outliers across entities in a single, interactive view.

Hierarchical Slicers: Introduced a new Hierarchical Slicer component that supports multiple dimensions in a single slicer, enabling progressive filtering across hierarchical data. Users can navigate and filter through parent-child relationships within one intuitive control, reducing report clutter and simplifying data exploration.

Autocomplete in Formula Columns: The custom formula editor now provides intelligent autocomplete, context-aware suggestions and guided syntax assistance to simplify formula creation, reduce syntax errors, and improve authoring efficiency.

Labels: Introduced Labels to help users organize, discover, and manage reports and report collections more effectively. The release includes label assignment, search and filtering, and centralized label management capabilities.

Ghost Dimensions for Drill-through Configuration: Display column picker dimensions as virtual ("ghost") entries in the table/chart data panel, enabling users to configure drill-through actions on dynamically selected columns without adding them to the visualization.
