# Screen Size Options

When creating a new report, you select the target screen width that determines the report's layout area.

| Screen Size | Width | Best For |
| --- | --- | --- |
| Screen Size | Width | Best For |
| Standard | 1024 pixels | Traditional monitors, smaller displays |
| Widescreen | 1440 pixels | Wide monitors, dashboard displays |

To set screen size during report creation:

1. On the Home tab, click New, then click Report.
1. In the Create New Report dialog, select either Standard or Widescreen under the layout options.
