# Renaming Columns in Table

Rename column headers directly from the data configuration panel, allowing you to present data in a more meaningful or user-friendly way without changing the underlying dimension name.

Steps to use

1. Open the Data Configuration Panel of your table Go to the Data tab. Dimensions added under the Rows section define the table’s columns.
1. Rename a Dimension (Column Header) In the Rows section, click the overflow menu next to the dimension name. Select Edit Display Name . In the Edit Display Name dialog, enter the new display name for the column. The original dimension name is shown below for reference.
1. Save Changes Click Ok to confirm your changes. The updated name appears as the column header in the table.
