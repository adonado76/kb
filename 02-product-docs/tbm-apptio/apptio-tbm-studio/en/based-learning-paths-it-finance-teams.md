# Learning Paths for IT Finance Teams

*Objective: Model IT costs accurately, create allocation rules, and drive financial transparency for IT services*

Who this is for: IT finance managers, financial analysts responsible for IT cost models, and anyone who builds and maintains cost allocation models

## Foundation (30 minutes)

What you 'll learn: TBM methodology and core modeling concepts

Learning objectives:

- Understand TBM methodology and framework
- Learn key concepts: cost pools, allocations, showback vs. chargeback
- Understand the data-to-model-to-report workflow
- Navigate Model Studio interface
- Recognize when to use different allocation methods

Topics to complete:

1. TBM Methodology Overview (10 min)
1. Core Modeling Concepts (10 min)
1. Quick Model Tour (10 min)

Prerequisites: Basic understanding of IT cost structures and financial concepts

## Essential Skills (4 hours)

What you 'll learn: Upload data, build cost models, create allocations, and validate accuracy

Learning objectives:

- Upload and prepare cost data for modeling
- Build basic cost allocation models
- Create allocation rules using appropriate drivers
- Validate model accuracy and reconcile totals
- Understand model calculations and timing

Topics to complete:

1. Upload and Prepare Data (45 min)
1. Transform Data for Modeling (45 min)
1. Build Basic Cost Models (60 min)
1. Create Simple Allocations (45 min)
1. Validate Model Accuracy (30 min) → Section 3.2.4
1. Understand Build and Calculation (15 min) → Section 4.5

Prerequisites: Foundation completed, access to cost data

Estimated practice time: Add 3-4 hours working with sample data or a pilot model

## Advanced (6 hours)

What you'll learn : Master complex allocations, multi-tier models, budgeting, and advanced model optimization

Learning objectives:

- Design and implement complex multi-tier allocation flows
- Handle sophisticated allocation scenarios (cascade, weighted, conditional)
- Build budget and forecast models alongside actuals
- Perform what-if analysis and scenario modeling
- Optimize model performance and build times
- Implement allocation best practices

Topics to complete:

1. Multi-Tier Cost Flows (90 min) → Section 3.2.3
1. Advanced Allocation Patterns (60 min) → Section 3.2.3
1. Budget and Forecast Modeling (60 min) → Section 3.2.5
1. What-If Analysis (45 min) → Section 3.2.6
1. Model Performance Optimization (45 min)
1. Model Governance and Best Practices (30 min) → Section 6.1

Prerequisites: Essential Skills completed, experience building simple models

## Common challenges

- Unallocated costs: Review driver data completeness and matching logic
- Circular allocations: Redesign model flow or use reciprocal method
- Build performance: Simplify transforms, reduce calculation scope
- Data quality: Implement validation rules early in the pipeline
