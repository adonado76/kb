# Localization Settings

Localization settings are primarily configured at the project level and affect how numbers, dates, and currencies are displayed in reports.

| Setting | Description |
| --- | --- |
| Setting | Description |
| Number Compaction | Controls display of large numbers (thousands, millions, billions, trillions) |
| Currency Symbols | Sets the default currency symbol for financial data display |
| Number Grouping | Regional formatting for number separators (e.g., Japanese 10,000-unit separation) |
| Default Paper Size | Sets default paper size for print and export operations |

Location: Project tab > Project Settings
