# Collection Navigation

Report Collection Navigator Component

The report collection navigator is a report component that gives users an easy way to move between reports in a collection. When you add a report to a collection, the navigator component is automatically added to the report.

The navigator typically appears as a horizontal bar at the top of the report, displaying the names of all visible reports in the collection. Users can click any report name to navigate directly to that report.

Navigation Features

The report collection navigator provides the following navigation capabilities:

| Feature | Description |
| --- | --- |
| Feature | Description |
| Direct Navigation | Click any report name in the navigator to jump directly to that report. |
| Current Report Indicator | The currently viewed report is highlighted in the navigator, helping users maintain context. |
| Filter Context | Slicer selections and filters can optionally be passed between reports when navigating. |
| Responsive Display | The navigator adapts to the available screen width, wrapping, or condensing as needed. |

Alternative Navigation Methods

In addition to the collection navigator, TBM Studio supports other navigation methods between reports:

- Buttons: Clickable buttons that navigate to specific reports
- Coded Links: Wiki-style syntax links in HTML text boxes or buttons
- Report Links: Links from table values or chart elements to related reports
