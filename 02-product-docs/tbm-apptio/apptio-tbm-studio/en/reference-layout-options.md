# Layout Options

## Overview

TBM Studio Report Studio provides a flexible layout system for positioning and sizing report components. This section covers the complete layout system, including screen sizing, component positioning, container components, and print layout configuration.

Reports are built within the Report Editing workspace, where you add and arrange tables, charts, slicers, buttons, and other components. The layout system supports both manual positioning for precise control and automatic flow layouts for responsive designs.
