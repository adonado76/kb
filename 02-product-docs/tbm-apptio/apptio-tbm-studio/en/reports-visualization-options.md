# Visualization Options

Report Viewing Features

| Feature | Action | Description |
| --- | --- | --- |
| Feature | Action | Description |
| View drivers/allocations | Click a table in the report | Shows drivers and allocations for that table |
| Trace value flow | Click table elements sequentially | Shows cost path through the model (v12.1+) |
| Drill to column | Click menu > Drill Down > select column | Expands table to show column values |
| View specific item | Click item after drilling down | Shows allocations for that specific item |
| Return to table view | Click return arrow in tier corner | Collapses drill-down |
| Switch metric | Use metric picker dropdown | Changes displayed metric |

Within-Tier Allocations

Beginning with v12.2.2, allocations between tables in the same tier display with offset positioning rather than looping lines.
