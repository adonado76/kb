# Legacy Charts vs. Ad Hoc Charts

TBM Studio supports two chart creation methods: the modern Ad Hoc approach (recommended) and legacy charts from earlier versions.

Ad Hoc Charts (Recommended)

- Created using the Component Configuration panel with drag-and-drop fields.
- Support slicers, column pickers, and quick pivots.
- Full access to all visualization types on the Ad Hoc tab.
- Interactive features like legend filtering and zoom.
- Access to custom color palettes.

Legacy Charts

- Created using older tools and edited via data path configuration.
- Not compatible with slicers or other interactive components.
- Some properties marked with (*) in the Properties dialog apply only to legacy charts.
- Can be converted from ad-hoc charts, but the conversion cannot be reversed.

Converting: To convert an ad-hoc chart to a legacy chart, open the Chart menu in the upper-left corner of the chart frame and select Convert to Legacy Chart. This action is permanent and cannot be undone.
