# Model Basics

| Content Type | How-To Guide Collection |
| --- | --- |
| Content Type | How-To Guide Collection |
| Target Audience | Business Analysts, IT Finance Teams |
| Time Estimate | 30-45 minutes total |
| Prerequisites | Basic familiarity with TBM Studio interface; at least one transform table created in Data Studio |

This section introduces the fundamental tasks for working with the cost model in TBM Studio. Before you can allocate costs or create model reports, you need to understand how to add tables to the model, what metrics flow through it, and how to visualize the overall cost flow structure.

The cost model is the heart of TBM Studio . It takes your prepared data from Data Studio and allows you to define how costs, budgets, and other values flow from source systems through to business units and services. Every allocation, every cost flow, and every financial insight begins with the model.

This section contains three foundational how-to guides:

- Add a Table to the Model — How to bring your transform tables into the modeling layer
- Understand Model Metrics (Cost, Budget, Forecast) — What Cost, Budget, and Forecast mean and how they flow
- View the Model Diagram — How to visualize and interpret the overall cost flow

Once you complete these tasks, you will be ready to create allocations .
