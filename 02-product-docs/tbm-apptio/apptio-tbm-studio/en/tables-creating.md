# Creating Tables

Create a New Table

Navigation: Home tab > New > Table

Parameters:

| Parameter | Required | Description |
| --- | --- | --- |
| Parameter | Required | Description |
| Name | Yes | Unique identifier for the table |
| Category | Yes | Folder organization in Project Explorer; can use existing or create new |

- Click Home tab
- Click New > Table
- Enter table name
- Enter or select category
- Click OK

The system creates the table and displays the Source step in the transform pipeline.
