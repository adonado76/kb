# Styling Editable Tables

Visual styling helps users identify editable cells and understand data state.

Visual Indicators

- New rows: Highlighted in orange until saved
- Modified cells: Visible in Show Changes view
- Validation errors: Cells highlighted in red with error messages
- Required fields: Visual indicator for mandatory columns
- Locked cells: Cells being edited by other users appear locked

Table Formatting

Standard table formatting options apply to editable tables:

- Column widths, alignment, and number formatting
- Header styles and borders
- Alternating row colors
- Sort and filter controls
