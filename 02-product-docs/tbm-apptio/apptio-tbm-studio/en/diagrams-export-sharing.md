# Export and Sharing

Sharing Options

| Method | Description |
| --- | --- |
| Method | Description |
| Model Report | Create a model report for end-user consumption (can be added to report collections) |
| Screenshot | Capture diagram for documentation or presentations |
| Report Collection | Add model report to a collection for easy access (v12.2.2+) |

Tip: Use model diagrams for onboarding new team members, debugging allocation issues, and communicating model structure to stakeholders.

See Create Reports for report creation procedures
