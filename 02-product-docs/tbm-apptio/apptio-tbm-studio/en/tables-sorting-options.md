# Sorting Options

Tables can be sorted by one or more columns in ascending or descending order. Access sorting controls on the Data tab.

Single Column Sort

1. Select the column to sort by clicking its header.
1. Click the Ascending or Descending icon on the Data tab.

Multi-Column Sort

1. Click the Sort icon on the Data tab.
1. In the Sort dialog, select up to four columns and their sort direction.
1. Click OK to apply the sort.

| Sort Option | Description |
| --- | --- |
| Sort Option | Description |
| Ascending | A to Z for text, smallest to largest for numbers, earliest to latest for dates. |
| Descending | Z to A for text, largest to smallest for numbers, latest to earliest for dates. |
| Maximum columns | Up to 4 columns can be used for multi-column sorting. |
| Subtotal sorting | When sorted by a Value column in a subtotaled table, sorting applies to subtotal values. |
