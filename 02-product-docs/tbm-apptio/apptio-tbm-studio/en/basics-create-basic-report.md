# Create a Basic Report

Objective: Create a new, empty report that's ready for tables and charts

When to use: When you need a new report to present model data to stakeholders, or when existing reports don't meet your requirements

Time estimate: 5 minutes

## Steps

1. Switch to TBM Studio if you're not already there.
1. Click the Home tab, click New , and then click Report . The Create Report dialog appears with configuration options.
1. Enter a name for your report. Choose a descriptive name that helps users find the report. If you plan to export to Excel, keep names under 31 characters to avoid truncation.
1. Enter a description of the report. This description appears in the Report Finder, so write something that helps end users understand what data the report contains.
1. Select who can view the report. Everyone makes the report visible to all users; select specific roles to restrict access.
1. Choose a layout for your report. Select whether to include a heading and choose between Standard (1024 pixels wide) or Widescreen (1440 pixels wide).
1. Click OK to create the report.

## Expected Results

- A new, blank report opens in the workspace
- A tab appears at the bottom of the workspace with your report name
- The report is automatically saved and ready for you to add components

## Common Pitfalls

- Forgetting to check in: After making changes, remember to check in your report so others can see your updates and the system can calculate data.
- Unclear naming: Vague names like "Report1" make reports hard to find. Use names that describe the content, such as "Data Center Cost Summary Q1."
