# A Guide to New Report Studio’s Navigation

*To access the new Report Studio, go to the Classic TBM Studio in your main environment.*

Select a project to open it in the new report studio – Choose a project from the drop-down to automatically launch it in the new experience.

The new Report Studio opens on the landing page, as shown below.

1. New button – Click here to create a new report or a new report collection
1. Search – Search for an existing report
1. New Reports – Lists all reports created using the new Report Studio.
1. Report Collections – Displays groups of related reports organized into collections.
1. Report Overflow Menu (3 dots menu) - Click here to access a host of operations that can performed on a report, like deleting a report, or exporting the report’s definition and configuration.
1. Project Settings – Click here to check in one or more reports, revert changes to them or to import a report configuration into the project in context.

1. App Switcher – Use this to switch to another IBM Apptio product
1. Profile Settings - Use this to manage your profile, impersonate another user (Note: Admins may want to do this to check a user's permissions or to troubleshoot problems), and Logout
1. Help – Use this to access the new Report Studio help documentation
1. Settings - Use this to change the customer access, update permissions, and access multi-currency settings
1. Environment Dropdown – Use this to switch between the environments and user workspaces.
1. Date Picker – Use this to switch between date periods
1. Take me to – Use this quick navigation menu to switch to the Classic TBM Studio and/or the New Report Viewer.
