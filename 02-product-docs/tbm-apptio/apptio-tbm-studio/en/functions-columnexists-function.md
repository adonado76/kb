# ColumnExists function

*Applies to : TBM Studio 12.0 and later*

Use this function to determine if a column exists in a table.

## Where to use

- Tables

## Syntax

=ColumnExists(column_name)

## Arguments

column_name

The name of the column that you want to check.

## Return type

True or false
