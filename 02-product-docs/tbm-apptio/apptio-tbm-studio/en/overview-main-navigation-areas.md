# Main Navigation Areas

## Toolbar

The toolbar at the top of TBM Studio provides quick access to core functions and navigation.

[Screenshot: Toolbar with labeled sections]

Key toolbar elements:

| Element | Purpose | Usage |
| --- | --- | --- |
| Element | Purpose | Usage |
| All Products menu | Navigate to other Apptio products | Click to switch between TBM Suite applications |
| Environment selector | Switch between Development, Staging, and Production | Select from dropdown to change environments |
| Project dropdown | Access current project settings | Click to view project details, settings, and permissions |
| Home button | Return to main dashboard | Quick access to project overview and recommendations |
| Help menu | Access documentation and support | Click for context-sensitive help and resources |
| User menu | Account settings and preferences | Access profile, preferences, and sign out |

## Navigation Pane

The left navigation pane provides access to all major studios and project components.

Main navigation sections:

- Data Studio — Upload, transform, and manage your data sources
- Model Studio — Build cost models and configure allocations
- Report Studio — Create and customize reports
- Administration — Manage users, permissions, and project settings

To navigate:

1. Click a studio name to expand its section
1. Click a specific item to open it in the main workspace
1. Use the search bar at the top to quickly find items

## Projects Pane

The Projects pane shows all projects you have access to within your domain.

What you see:

- Current active project (highlighted)
- Other available projects
- Project status indicators

To switch projects:

1. Click the Projects dropdown in the toolbar
1. Select the project you want to work with
1. The interface refreshes to show the selected project

## Project Explorer Pane

The Project Explorer shows the structure of your current project with all documents organized by type.

Common document types:

- Tables — Data tables and transform tables
- Models — Cost models and allocation configurations
- Metrics — Calculated and modeled metrics
- Reports — Published reports and dashboards
- Editable Tables — Data entry forms

Document status icons:

| Icon | Status | Meaning |
| --- | --- | --- |
| Icon | Status | Meaning |
| 🔒 | Locked | Document is checked out by you |
| 🔓 | Unlocked | Document is available for editing |
| 🔄 | Updated | A newer version is available (click Update Workspace) |
| ⏳ | Calculating | Document is being calculated after check-in |
| ⚠️ | Error | Document has errors or warnings |
