# New Report Studio Overview

*The new IBM Apptio Report Studio , which is carbon-designed, delivers a modernized, intuitive, and more powerful reporting experience. It is designed to help TBMAs, and business users build reports faster, with less complexity and a shorter learning curve. Compared to the Classic Report Studio, this version brings:*

- Faster time-to-value: Create and view reports more quickly, helping teams make decisions faster.
- Modern, intuitive interface: A drag-and-drop design that’s easy to learn and use.
- Improved accessibility: Optimized for various screen sizes and devices.
- Streamlined report creation: Reduced setup time with smart defaults and reusable components.
- Flexible layout and visualization tools: Build pixel-perfect reports with rich chart options and layout controls.

## What you can do with the New Report Studio

- Create reports using configurable components and visualizations
- Organize reports into collections for easier discovery
- Interact with reports using filters and pickers
- View data in multiple currencies
- Export reports to PDF
- Export table data for further analysis

## Who is New Report Studio for?

Report Studio is designed for two primary user roles:

Admins (Report Authors)

Admins create and manage reports. They configure components, visualizations, and layouts to deliver meaningful insights to their audience.

End Users (Report Viewers)

End users consume reports created by Admins. They focus on understanding data and exporting it when needed.

For more details, see User Roles & Permissions .
