# Export Report to PDF

The export to PDF feature in the report viewer allows you to export the report currently
being viewed into a PDF file. This makes it easy to download and share.

**When to use**

Use this option when you want to:

- Share reports with stakeholders
- Capture a snapshot of the report for reviews or presentations
- Archive reports for record-keeping or compliance purposes

**Export a report to PDF**

To export a report from the report viewer:

1. Open the report in the report viewer
2. Click **Export to PDF** on the action bar at the top of the report.
3. The system generates and downloads a PDF of the current report view.
