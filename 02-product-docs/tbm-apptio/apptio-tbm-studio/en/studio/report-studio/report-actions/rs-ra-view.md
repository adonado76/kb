# View Menu

This menu helps you to format the look and feel of the report canvas. The View menu in
the report canvas is:

![view menu](../../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/rs-viewmenu.png)

***Actual Size***

Resize a report component by dragging the bottom, sides, or corners of a component's border.

- Hover the mouse pointer over the bottom, side, or corner of a component's border. The
  mouse pointer will change to a Resize icon. If the border is hidden, position the mouse
  pointer anywhere over the component to see its border.
- When the Resize, icon will appear, click and drag the border to change the height or
  width of the component.

*Fit To – Options are Fit to page and Fit to width*

These options will auto fit the component/visualization as per the page or width size. You can
also change the size from the feature.

*Show Grids*

This option shows the grids in the report canvas.

![show grids](../../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/rs-showgrid.png)

*Show rulers*

This option shows the rulers in the report canvas.

![show rulers](../../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/rs-shwruler.png)

*Snap-to-grid*

When **Snap to Grid** is selected, components will snap to the nearest point on an
invisible grid.

*Show layers*

Select this option to view the position of different components and visualizations in the
report. Select the Menu option in each component to

- Move it backward or forward
- Copy, paste, or delete the component

![show layers](../../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/rs-show-layers.png)
