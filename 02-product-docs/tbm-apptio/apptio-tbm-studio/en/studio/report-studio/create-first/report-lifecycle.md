# Report Lifecycle

Report lifecycle actions help admins manage reports after they are created. These
actions control how reports are created, updated, reused, and removed.

## Create a Report

Admins can create reports using the new Report Studio.

- Reports are created from the New Reports tab on the landing page.
- The report opens directly in the report canvas, where components and visualizations
  can be added.
- Autosave is enabled by default, and changes are saved as you work.
- A manual save option is also available.

For step-by-step instructions, see [Create Your First
Report](first-report.html "This guide walks through creating a new report in the new Report Studio. By the end of this guide, you will have a report with components and visualizations ready to view and share.").

## Edit a Report

Admins can edit existing reports to update data, layout, or configuration.

- You need to check out the report to view and edit.
- Editing opens the report in the report canvas.
- Changes are saved automatically using autosave.

## Delete a Report

Admins can delete reports that are no longer needed.

- Deleted reports are removed from all collections.
- Users will no longer be able to access the deleted report.

## Export Report Configuration

Admins can export a report’s configuration for reuse.

- Exporting captures the report definition, including components and visualizations.
- The exported configuration can be imported into other projects or instances.
- This is useful for migration, reuse, or standardization across environments.

## Import Report Configuration

Admins can import a previously exported report configuration.

- Imported reports retain their original structure and configuration.

## Export Report to PDF (Report Studio)

You can now export reports to PDF directly from the New Report Studio. This allows you to
generate a shareable, presentation-ready version of your report without switching to the
Report Viewer.

**How to export a report to PDF?**

1. Open the report in the **New Report Studio**.
2. From the top navigation bar, click **File**.
3. Select **Download as**.
4. Choose **PDF**.
5. Select **Download PDF directly**.
6. Click **Download**.

The PDF file will be generated and downloaded to your system.

**What gets exported?**

The exported PDF includes:

- All visible components in the report
- Current layout and formatting
- Applied filters reflected at the time of export

Note: The PDF reflects the current state of the report at the time of download.
