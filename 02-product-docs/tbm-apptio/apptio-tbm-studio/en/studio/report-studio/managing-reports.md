# Managing reports

Once you have created a report, you can perform the following actions:

1. View a Report
   1. Goto the New Reports listing on the landing page.
   2. Select the report you want to open.
2. Modify a Report
   1. Check out the report to your workspace.
   2. Make edits.
   3. Check in the report to move changes back to the Development environment.
3. Delete a Report
   1. From the report list on the landing page, select the Delete option in the report
      overflow menu.
   2. Confirm the deletion when prompted.
