# Email Report

The Email Report feature in the report viewer allows you to send the report currently
being viewed via email. You can share the report either by attaching the PDF to the email or by
including a link that allows recipients to view or download the report as a PDF.

**When to use**

Use this option when you want to:

- Share reports directly with stakeholders from the report viewer
- Send a PDF version of the report without manually downloading and attaching it
- Provide recipients with a link to view or download the report as a PDF

**Email a report**

![email report popup](../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/er1.png)

![email report window](../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/er2.png)

To email a report from the report viewer:

1. Open the report in the report viewer
2. Click the **Email icon** on the action bar at the top of the report.
3. The Email Report dialog opens.
4. Choose how you want to share the report:
   1. Include a link to the PDF in the email
   2. Attach the PDF to the email
5. Enter the recipient email address(es).
6. The email subject is pre-populated by default, but you can edit it if needed.
7. Optionally, add a message to the email.
8. Select **Send a copy to self** if you want to receive a copy of the email.
9. Click Send Email.

Once the email is sent, recipients can view the report or download the report as a PDF.
