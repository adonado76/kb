# Show Values

The **Show Values** feature helps you quickly understand how data is distributed within
a column. It displays all **unique values** in the selected column along with the
**number of times each value appears** in the table.

![image of show value](../../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/sv-1.png)

![image of show value table](../../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/sv-2.png)

**Use this to**

- Explore a dataset before creating visualisations
- Check data cleanliness (duplicates, blanks, unexpected values)
- Understand categorical fields

**How to use Show Values**

1. Open a table
2. Hover over the column you want to analyse
3. Click the overflow menu on the column header
4. Select Show Values

A tabular view opens showing:

- Value – each unique value in the column
- Count – number of rows containing that value
