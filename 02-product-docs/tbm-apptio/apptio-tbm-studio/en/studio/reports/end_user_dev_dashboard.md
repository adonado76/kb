# End User Devices reports

End User Devices (EUD) displays the following reports.

[Learn how to configure
End User Devices](component%20end%20user%20devices.htm "(Opens in a new tab or window)")

## Summary Report

View an executive summary of the entire end user device fleet, including:

- assignments and consumption profiles
- makes, models and locations
- end-of-life aging and non-compliance, including estimated refresh costs

![Summary Report](../../resources/images/end%20user%20devices%20-%20configuration%20guide/component%20end%20user%20devices_10.png)

## Age Analysis

Monitor the aging and estimated refresh cost of devices by end of life. This allows you to plan
and forecast effectively for device updates or replacements.

![Age Analysis](../../resources/images/end%20user%20devices%20-%20configuration%20guide/component%20end%20user%20devices_11.png)

## Warranty Analysis

Monitor the aging and estimated refresh cost of devices by end of warranty. This allows you to
plan and forecast effectively for device updates or replacements.

![Warranty Analysis](../../resources/images/end%20user%20devices%20-%20configuration%20guide/component%20end%20user%20devices_12.png)

## Multiple Devices

Monitor users with multiple devices. This allows you to effectively retrieve and reallocate
excess devices to other users.

![Multiple Devices](../../resources/images/end%20user%20devices%20-%20configuration%20guide/component%20end%20user%20devices_13.png)

## Orphaned Devices

Monitor Potential Orphaned Devices (deployed devices with matched user Employee ID against an
inactive employee ID in HR Data). This report monitors devices with inactive users or users that
have not accessed the device in the last 90 days, allowing you to effectively retrieve and
reallocate unused devices to other users.

![Orphaned Devices](../../resources/images/end%20user%20devices%20-%20configuration%20guide/component%20end%20user%20devices_14.png)

## Spare Capacity

Monitor in-stock device levels across locations. This allows you to effectively re-distribute
inventory and assign them for new employees or to refresh existing user devices with expired life or
warranty.

![Spare Capacity](../../resources/images/end%20user%20devices%20-%20configuration%20guide/component%20end%20user%20devices_15.png)

## Compliance

Monitor devices to ensure compliance to latest operating systems. When you know which devices are
no longer compliant, you can plan and forecast effectively for device updates or replacements.

![Compliance](../../resources/images/end%20user%20devices%20-%20configuration%20guide/component%20end%20user%20devices_16.png)

## Maintenance

This report maintains end user exemptions, refresh models and estimated costs, stock status
definitions, and latest OS version details.

![Maintenance](../../resources/images/end%20user%20devices%20-%20configuration%20guide/component%20end%20user%20devices_17.png)
