# Add an editable column to a table

## About this task

**Applies to**: TBM Studio 12.0 and later

Editable columns are blank columns where users can enter data. They can be added only to editable
tables.

To add an editable column:

## Procedure

1. Select the editable table where you want to add the column.
2. Open the drop-down menu for the **Insert** icon ![](../../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/studioimages/icons/table_column_add.png) on the
   **Data** tab.
3. Select the **Insert Editable Column** option.
4. When prompted, enter a name for the new column and click **OK**.
