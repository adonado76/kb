# Report layout

**Applies to**: TBM Studio 12.0 and later

After adding components to a report, you can arrange them using the report layout tools on the
**Format** tab.

## Format tab

The **Format** tab includes tools to align components, distribute them evenly both
horizontally and vertically, and make the components uniform in size.

## Group boxes

Group boxes are cosmetic additions to reports and are useful for visually grouping components in
a report. They also physically group components together so they can be moved as a group. To add a
grouping box, click **Group** on the **Report** tab.

## Tabbed components

To provide users with quick access to a series of report components without having to scroll, you
can place the components in a tabbed group. All components except grouping boxes can be placed on a
tab. To add a tabbed group, click **Tabs** on the **Report** tab.

## Dynamic sizing and positioning

Dynamic sizing and positioning adjusts components based on the contents of tables, charts, group
boxes, and tabbed components. To use dynamic sizing, click **Dynamic** on the **Format**
tab.

## Master reports

To ensure a set of reports has the same layout, you can create a master report and apply it to
the set of reports. A master report can include all the components of a regular report such as
buttons and group boxes. To apply a master report, click **Masters** on the **Report**
tab.

## Report blueprints

Some applications come with blueprint reports. A blueprint report is a layout template based on a
standard report that ships with the application. If you want to create a custom report that has the
same layout as one of the standard reports, you can use a blueprint. For more information, see [Add components from a
blueprint](add-components-from-blueprint.html).
