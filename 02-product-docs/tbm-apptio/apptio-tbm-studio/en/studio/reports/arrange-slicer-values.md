# Arrange slicer values

**Applies to**: TBM Studio 12.0 and later

You can arrange the values in a slicer vertically (the default) or horizontally by selecting the
**Layout** options from the **Slicer** tab. Also, you can select the number of columns or rows
from the drop-down menu under each option. The same slicer arranged vertically, horizontally, with
one and two columns, and with one and two rows is shown in the following image:
