# Create Allocations

|  |  |
| --- | --- |
| **Content Type** | How-To Guide |
| **Target Audience** | Business Analysts, IT Finance Teams |
| **Time to Complete** | 30-60 minutes |
| **Prerequisites** | Source and target tables with Model steps; Transform data prepared in Data Studio; Project checked out |

## Overview

Allocations are the heart of cost modeling in TBM Studio. They determine how costs flow
from one table to another—distributing expenses like datacenter costs to applications, labor
costs to services, or IT costs to business units. This section provides step-by-step
guidance for creating allocations, from simple one-to-one distributions to complex
multi-tier cost flows.
