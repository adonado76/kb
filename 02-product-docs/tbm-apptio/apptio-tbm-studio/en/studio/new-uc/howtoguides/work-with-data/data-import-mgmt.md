## Data Import & Management

This section provides step-by-step instructions for importing and managing data in TBM
Studio. Whether you are uploading files manually, connecting to external systems, or
managing monthly data versions, these guides will help you establish a reliable data
foundation for your cost models.

- [Upload Data from Files (Excel, CSV)](upload-data-files.html) – Import data from local or network files
- [Connect to External Data Sources (DataLink)](connect-external-dl.html)– Automate data integration from ServiceNow and
  other systems
- [Version Tables for Monthly Data](version-tables-monthly-data.html) – Manage time-series data with monthly
  versioning
