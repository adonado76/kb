# Introduction

This section helps you identify, diagnose, and resolve common issues in TBM Studio. Content
is organized by symptom and functional area to help you quickly find solutions. When
self-service troubleshooting isn't sufficient, guidance on using diagnostic tools and
getting help is also provided.

**How to use this section:**

- **If you know the error message:** Go to 7.2 Error Messages Explained
- **If you're experiencing a symptom:** Go to 7.1 Common Issues & Solutions
- **If you need to analyze performance:** Go to 7.3 Diagnostic Tools
- **If you need additional assistance:** Go to 7.4 Getting Help
