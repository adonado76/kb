|  |  |

## Overview

TBM Studio provides a streamlined interface designed to help you move efficiently between
Data Studio, Model Studio, and Report Studio. This reference guide shows you where to find
key features and how to navigate the interface.

**What you'll learn:**

- Main [navigation areas](navigation.html) and their purpose
- Different [environments](environments.html)
- How to [work with projects and documents](work-with-docs.html)
- Essential [keyboard shortcuts](shortcuts.html)
- Interface [customization options](custom-options.html)

## What’s Next

Once you understand the interfaces, you can:

- Complete the [Quick Start Guide](qsg.html)to build your first project
- Explore Data Studio to learn about [data management](../howtoguides/work-with-data/data-import-mgmt.html)
- Try [Keyboard Shortcuts](shortcuts.html) to improve your efficiency
- Browse Learning Paths for role-specific guidance
