# Learning Paths for IT Finance Teams

**Objective:** Model IT costs accurately, create allocation rules, and drive
financial transparency for IT services

**Who this is for:** IT finance managers, financial analysts responsible for IT cost
models, and anyone who builds and maintains cost allocation models

## Foundation (30 minutes)

**What you**'ll learn: TBM methodology and core modeling concepts

**Learning objectives:**

- Understand TBM methodology and framework
- Learn key concepts: cost pools, allocations, showback vs. chargeback
- Understand the data-to-model-to-report workflow
- Navigate Model Studio interface
- Recognize when to use different allocation methods

**Topics to complete:**

1. TBM Methodology Overview (10 min)
2. [Core Modeling Concepts](../getting-started/core-concepts.html) (10 min)
3. [Quick Model Tour](../getting-started/qsg.html) (10 min)

**Prerequisites:** Basic understanding of IT cost structures and financial concepts

## Essential Skills (4 hours)

**What you**'ll learn: Upload data, build cost models, create allocations, and validate
accuracy

**Learning objectives:**

- Upload and prepare cost data for modeling
- Build basic cost allocation models
- Create allocation rules using appropriate drivers
- Validate model accuracy and reconcile totals
- Understand model calculations and timing

**Topics to complete:**

1. [Upload and Prepare Data](../howtoguides/work-with-data/data-import-mgmt.html)(45 min)
2. [Transform Data for
   Modeling (45 min)](../howtoguides/work-with-data/transform-enrich-data.html)
3. [Build Basic Cost Models](../howtoguides/build-cost-model/model-basics.html)(60 min)
4. [Create Simple Allocations](../howtoguides/build-cost-model/setup-sa.html)
   (45 min)
5. Validate Model Accuracy (30 min) *→ Section 3.2.4*
6. Understand Build and Calculation (15 min) *→ Section 4.5*

**Prerequisites:** Foundation completed, access to cost data

**Estimated practice time:** Add 3-4 hours working with sample data or a pilot model

## Advanced (6 hours)

**What you'll learn**: Master complex allocations, multi-tier models, budgeting, and
advanced model optimization

**Learning objectives:**

- Design and implement complex multi-tier allocation flows
- Handle sophisticated allocation scenarios (cascade, weighted, conditional)
- Build budget and forecast models alongside actuals
- Perform what-if analysis and scenario modeling
- Optimize model performance and build times
- Implement allocation best practices

**Topics to complete:**

1. [Multi-Tier Cost
   Flows](../howtoguides/build-cost-model/create-multi-tier-allo.html) (90 min) *→ Section 3.2.3*
2. [Advanced
   Allocation Patterns](../howtoguides/build-cost-model/create-multi-tier-allo.html) (60 min) *→ Section 3.2.3*
3. [Budget and
   Forecast Modeling](../howtoguides/build-cost-model/understand-model-metrics.html) (60 min) *→ Section 3.2.5*
4. **What-If Analysis** (45 min) *→ Section 3.2.6*
5. [Model Performance Optimization](../admin/performance-optimize.html)
   (45 min)
6. [Model Governance and Best Practices](../admin/project-admin.html) (30
   min) *→ Section 6.1*

**Prerequisites:** Essential Skills completed, experience building simple models

## Common challenges

- **Unallocated costs:** Review driver data completeness and matching logic
- **Circular allocations:** Redesign model flow or use reciprocal method
- **Build performance:** Simplify transforms, reduce calculation scope
- **Data quality:** Implement validation rules early in the pipeline
