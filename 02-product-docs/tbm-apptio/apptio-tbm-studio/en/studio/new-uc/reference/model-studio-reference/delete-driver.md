# Deleting a Driver

To remove a driver from a model object:

1. Click the driver in the Sankey diagram
2. Click Delete at the bottom of the panel
3. Save changes

Warning: Deleting a driver removes all associated value from the model object. Verify
the impact on downstream allocations before deleting.
