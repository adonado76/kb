# Component Sizing

**Resize Methods**

|  |  |
| --- | --- |
| **Method** | **How To** |
| **Drag borders** | Click and drag any edge or corner of the component |
| **Position and Size dialog** | Right-click header → Position and Size → enter width/height values |
| **Resize Component to Full Page** | Available in Print Layout mode for tables |

**Auto-Sizing**

Some components can automatically resize based on their content:

**For tables:** Tables can auto-size to show all data without scrollbars. Configure this in
the table's Properties dialog under sizing options.

**For group boxes:** Group boxes can resize dynamically to accommodate their contained
components:

1. Select the group box.
2. On the Group tab, select one of the Auto Sizing options: Auto-size width, Auto-size height, or
   Auto-size both.

Note: Auto-resizing for group boxes works only in View mode. In Edit mode, the group box maintains a
fixed size for easier editing.
