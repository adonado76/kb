# Creating Master Reports

**Creating a New Master Report**

To create a new master report:

1. Create a new report or check out an existing report that you want to convert to a master.
2. On the Report tab, in the Advanced group, click Master Report to enable the master report
   option.
3. Add the components you want to appear in all child reports (group boxes, navigation buttons,
   branding elements, etc.).
4. **Click Save** on the **Home** tab.
5. **Check in** the report to make it available to other reports.

**Converting an Existing Report to Master**

Any existing report can be converted to a master report:

1. **Check out** the report you want to use as a master.
2. Navigate to the **Report** tab in the ribbon.
3. In the **Advanced** group, click the **Master Report** option.
4. Save and check in the report.

Note: Once a report is designated as a master report, it will appear in the Masters drop-down menu
on the Report tab for all other reports in the project.

**Standard Master Reports**

TBM Studio includes several standard master reports that ship with the product. These pre-built
masters are available in the **Masters** drop-down menu on the **Report** tab. You can use
these standard master reports as-is or modify them and save them with a new name using the **Save
As** option.

Warning: You cannot delete standard master reports that ship with the product. You can
only delete custom master reports that you have created.
