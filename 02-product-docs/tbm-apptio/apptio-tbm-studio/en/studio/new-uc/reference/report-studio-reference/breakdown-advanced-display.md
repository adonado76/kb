# Breakdown and Advanced Display Options

Advanced options control how reports behave in specific contexts such as drill-down navigation,
filtered views, and unit-specific displays.

|  |  |
| --- | --- |
| **Report Type** | **Display Options** |
| Non-breakdown report | Even when showing a breakdown: Displays even if receiving breakdown data. Only when not showing a breakdown: Displays only with non-breakdown data. |
| Breakdown report | Even when not showing a breakdown: Displays even with non-breakdown data. Only when showing a breakdown: Displays only when fed breakdown data. |
| Filtered report | Only when filtered like this: Displays only if filtering matches the source. Even if filtered some other way: Displays regardless of filter differences. |
| Unit report | For All Units: Applies to all units. For [unit] only: Applies only to a specific unit in the filter. |
