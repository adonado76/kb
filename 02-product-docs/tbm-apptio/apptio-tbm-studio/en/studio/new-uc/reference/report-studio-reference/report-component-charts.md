# Report Components: Charts

Charts transform your cost data into visual insights that communicate complex financial
information quickly and effectively. TBM Studio provides a comprehensive suite of chart types
optimized for IT financial analysis, from simple bar charts comparing department costs to
sophisticated waterfall charts explaining budget variances.

## Chart Type Overview

TBM Studio offers the following chart types, accessible from the Ad Hoc tab after creating a
chart component:

|  |  |  |
| --- | --- | --- |
| **Chart Type** | **Variants** | **Best Used For** |
| Bar | Horizontal Bar, Stacked Horizontal Bar | Comparing categories side by side |
| Column | Vertical Column, Stacked Column | Comparing values across time periods |
| Line | Standard Line, Trend Line | Showing trends over time |
| Pie | Standard, Half, Donut, Donut Half | Showing proportions of a whole |
| Trend | Line-based trend with projections | Forecasting and historical analysis |
| Gauge | Single value gauge | KPI progress indicators |
| Tree Map | Nested rectangles | Hierarchical data and relative proportions |
| Waterfall | Bridge chart with steps | Explaining variances between values |
| Radar | Filled Radar, Radar with Markers | Multi-dimensional comparisons |
| Overlay | Combined Column/Line | Comparing two related data series |
