# Report Components: Filters and Slicers

## Overview

Filters and slicers are interactive components that enable report users to dynamically filter
data displayed in tables and charts. They transform static reports into powerful data discovery
tools, allowing users to focus on specific elements and find insights through interactive
exploration.

**Key benefits of filters and slicers:**

- Enable users to customize report views without modifying the report design
- Support multiple combinations of filter criteria using AND/OR logic
- Provide visual feedback on related and unrelated data values
- Allow administrators to set defaults while giving users flexibility
- Support hierarchical drill-down through data dimensions

Note: Slicers only work with Ad Hoc Query tables and charts created using the Component
Configuration panel. They do not work with legacy tables and charts.
