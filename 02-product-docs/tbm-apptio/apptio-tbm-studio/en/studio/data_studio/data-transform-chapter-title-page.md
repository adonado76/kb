# About Data Studio

**Applies to**: TBM Studio 12.0 and later

When you bring data into one of the Apptio applications, it may or may not be in the correct
format to map to the master data sets or to generate your own custom reports. If the data is not in
the correct format, you will need to transform the data using the Data Studio transform tools.
