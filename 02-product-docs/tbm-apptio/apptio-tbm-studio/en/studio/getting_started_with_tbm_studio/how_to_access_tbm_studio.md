# How to access TBM Studio

Apptio tests and supports the following web browsers for use with the Apptio Technology
Business Management (TBM) Suite.

## Supported browsers

The Apptio applications are delivered as a service through the Internet. The interface is
web-based and available through most web browsers. While there is nothing inherent in the Apptio
application user interface that prevents it from rendering in any given web browser, certain
features may not render correctly in all browsers.

## Google Chrome

Chrome versions 31 and higher are supported.

## Microsoft Edge

Microsoft Edge 41.16299.547.0 and higher are supported.

## Mozilla Firefox

Firefox versions 20.x and higher are supported. New Apptio features are tested on the latest
version of Firefox when the feature ships. We recommend upgrading to the latest version of Firefox.
If it isn’t possible to use the latest Firefox version, we recommend using a different browser.

## General browser recommendations and screen resolution

While Apptio does not specifically test with browsers other than those listed above, there is
nothing inherent in the Apptio application interface that would prevent it from rendering in other
browsers. However, if you use a browser other than the browsers listed above and experience issues
with the interface, Apptio support may ask you to switch to a supported browser as a resolution to
the problem.

For all browsers, we recommend the following:

- Enable JavaScript, cookies, and SSL 3.0
- Set your screen resolution to a minimum 1024 x 768 for the best experience. Screen resolutions
  less than 1024 x 768 may require additional scrolling to work with an entire page. You should not
  resize a non-maximized browser to under 1024 x 768 on a higher resolution monitor.
- Some third-party Web browser plug-ins and extensions can interfere with Apptio software
  functionality. If you experience malfunctions or inconsistent behavior, disable all of the Web
  browser's plug-ins and extensions and try again.

## Supported screen resolution

The recommended minimum screen resolution (assuming you have maximized your browser window) is
1024 x 768.
