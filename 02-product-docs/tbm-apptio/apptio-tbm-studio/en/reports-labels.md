# Labels

## Overview

Labels help you organize, categorize, and quickly find reports and report collections by assigning descriptive tags. Labels provide a flexible way to group content based on business function, purpose, project, team, or any custom category that fits your organization's needs.

Unlike folders or collections, labels allow the same report to belong to multiple categories, making it easier to discover related content as your report library grows.

## Why Use Labels?

As the number of reports increases, finding the right report can become challenging. Labels help you:

- Organize reports using meaningful categories.
- Assign multiple categories to a single report.
- Quickly locate reports using search or filters.
- Maintain consistent categorization using predefined labels.
- Improve report discoverability across your organization.

## Working with Labels

You can assign one or more labels while creating a new report.

1. Create a new report.
1. In the Labels field, either: Select one or more existing labels. Search for an existing label. Create a new label
1. Save the report.

The selected labels are associated with the report and appear wherever the report is listed.

Labels can be modified at any time.

To update labels:

1. On the Report Listing page, click “Manage Labels”
1. Add, remove, or modify the assigned labels
1. Save your changes

## Labels for Report Collections

Report Collections can also have labels to help organize groups of reports.

While creating a report collection, assign one or more labels to categorize the collection.

You can add, edit, or remove labels from an existing report collection on the Report Collections listing page at any time.

## Search Using Labels

Labels make reports easier to discover.

You can search for reports by entering:

- A label name

Reports matching the entered label are displayed in the search results.

## Manage Labels

Only admins can create and manage labels.

Create labels – Create reusable labels to promote consistent categorization across reports.

Examples include:

- Finance
- Operations
- Quarterly Review
- Cost Optimization

Edit Labels - Update the name of an existing label to keep your labelling system current and meaningful.

Any updates are reflected wherever that label is used.

Delete Labels - Remove labels that are no longer needed to keep the label library organized.

Deleting a label removes it from the system and from reports or collections where it was previously assigned.

## Label Inheritance

Labels assigned to report collections and reports are managed independently.

- Labels applied to a report collection are not automatically assigned to the reports inside the collection.
- Labels applied to individual reports do not automatically appear on the parent report collection.
- Reports and collections can each have their own set of labels.
- You can search and filter reports and report collections independently using their respective labels.

## Best Practices

To keep your report library organized:

- Use predefined labels whenever possible to maintain consistency.
- Apply multiple labels when a report belongs to more than one category.
- Use descriptive, business-friendly label names.
- Periodically remove labels that are no longer relevant.
- Establish a common labelling convention across your organization to improve discoverability.
