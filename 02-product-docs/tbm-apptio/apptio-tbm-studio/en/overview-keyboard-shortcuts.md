# Keyboard Shortcuts

Master these shortcuts to work more efficiently in TBM Studio.

## Essential Shortcuts

| Shortcut | Action | Description |
| --- | --- | --- |
| Shortcut | Action | Description |
| Ctrl+Alt+T | Toggle Studio mode | Switch in and out of Studio editing mode |
| Ctrl+F1 | Toggle ribbon | Show/hide the TBM Studio tabs and ribbon |
| Alt+N | New document | Create a new document in current context |
| Alt+O | Check out | Check out selected document for editing |
| Ctrl+S / Meta+S | Save | Save current document |
| Alt+K | Check in | Check in current document to Staging |
| Ctrl+Alt+Z | Revert | Undo all changes since last check out |
| F9 | Update document | Refresh current document with latest changes |
| Shift+F9 | Update workspace | Refresh all documents in workspace |
| Alt+Escape | Close tab | Close current document tab |
| Ctrl+Z | Undo | Undo changes since last save |
| Ctrl+Y | Redo | Redo previously undone changes |

- Meta key = Command (⌘) on Mac, Windows key on Windows
- Ctrl shortcuts work on both Windows and Mac
