# Permissions

Control who can perform input actions through row-level permissions:

| Permission | Description | Default |
| --- | --- | --- |
| Permission | Description | Default |
| Add Row Permission | Who can add new rows | Everyone |
| Duplicate Row Permission | Who can duplicate rows | Everyone |
| Delete Row Permission | Who can delete individual rows | Everyone |
| Delete All Rows Permission | Who can delete all rows | Admin and Partner |
| Edit Row Permission | Who can edit cell values | Everyone |
| Publish Row Permission | Who can publish changes | Everyone |
| Download Permission | Who can download table data | Everyone |
| Upload Permission | Who can upload table data | Everyone |
