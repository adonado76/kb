# Allowable Metric count exceeded

The allowable metric count is the total number of metrics which your project can have.
This includes calculated and modeled metrics.

If you have reached this limit and try to create a new metric, the following error message is
displayed:

![](../../resources/images/troubleshooting_images/allowable-metric-count-exceeded.png)

## Configuration recommendation for Allowable Metric Count exceeded error

To resolve this error:

1. In the Project Explorer view > select Metrics
2. Select the metric you want to remove
3. In the Home tab, select Check Out
4. Select the highlighted metric again
5. In the Home tab, select Delete
6. elect OK to continue

If you need more help to reduce the total number of metrics, contact Apptio support.
