# KPI

*The KPI (Key Performance Indicator) component is used to highlight important metrics in a report. It helps users quickly understand key metrics at a glance*

## When to use a KPI

Use a KPI component when you want to:

- Display metrics prominently
- Highlight performance, totals, or key outcomes
- Draw attention to critical values in a report

## Add a KPI to the Report

1. Add a KPI from the Visualizations pane on the toolbar
1. Click on the KPI to enable the Data and Format panels.
1. Data Panel Select the model object from the dropdown list Primary Metric - Drag or add a primary metric Comparison Type - The primary metric can be compared to the following: None – No comparison. Comparison Metric – Choose a secondary metric to compare with the primary metric Previous Period – Choose to compare with the previous period of the primary metric Comparison Metric - Drag or add a secondary metric for comparison Filters - Drag or add the filter criteria
1. Format Panel General Properties – See Component Properties KPI-specific Properties Trend Choose the color for positive, negative and neutral trends Choose the indicator to represent the trend (arrow and value) Pick the trend indicator size Primary/ Comparison Metric Edit the following properties, as required for the primary metric’s name and value Font size and style (bold, italics, underline) Color of the title text (with option to reset the color) Shorthand notations - Shorten values for the primary and secondary metrics.
