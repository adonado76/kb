# Recents

This feature allows you to see the reports that you have used most recently. The Recents
option is enabled by default and will appear in the left navigation menu. It will display the list
of recently accessed reports.

![img](../../../../03-media/apptio-tbm-studio/resources/images/studio_images/recent-2.png)

The Recents option will show a maximum of five reports with the most recently accessed report at
the top.

Note: If an admin removes or revokes a report for a user, and if it was stored in the Preference
service data, it will be automatically removed from Recents, after it is first loaded in the CT
view.
