# GetReportName function

*Applies to : TBM Studio 12.0 and later*

Use this function to dynamically get the name of the report that you are using.

## Where to use

- Dynamic text

Use to dynamically get the name of the report that you are using.

For example, you can combine with evalWiki() to define a standard header in a data set and use the GetReportName function to get the name of the current report.

## Syntax

GetReportName()

## Arguments

There are no arguments.

## Return type

Text

## Example

GetReportName()

1. GetReportPath
1. EvalWiki
