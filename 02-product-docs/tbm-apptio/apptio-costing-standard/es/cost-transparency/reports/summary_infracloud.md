# Resumen

El informe Resumen de Infraestructura y Nube contiene los siguientes elementos:

![](../../resources/images/ct_images/ct%20report%20collections/summary_1.jpg)

| Elemento clave | Descripción |
| --- | --- |
| (1) Recogida de informes | Esta colección de informes proporciona los detalles que necesita para revisar el gasto, las tendencias y el uso de sus recursos (mano de obra y torre de TI):   - Infraestructura y nube, TCO de servidor y TCO de almacenamiento (descritos en esta página) |
| (2) & (3) Resumen de costes YTD por perfil de virtualización | Seleccione el corchete angular TCO de servidor para saltar a la pestaña TCO de servidor.    Utilice el gráfico para ver un resumen de sus costes interanuales por uso de nubes públicas y privadas, servidores físicos y servidores virtuales.  La tabla ofrece detalles sobre el coste interanual, el porcentaje de cambio, las horas de instancia y el coste por hora. Seleccione un elemento de la columna Perfil de virtualización para abrir un cuadro de diálogo con más detalles sobre el elemento seleccionado.     - Indicadores clave de rendimiento (KPI): proporcionan una visión de alto nivel de su gasto en activos virtuales. - Gráfico y tabla de tendencias: El gráfico de tendencias muestra el coste por hora de instancia frente a la media de 12 meses mes a mes. La tabla muestra el coste, el número de servidores, el número de instancias y la capacidad de memoria para cada entorno y clase. |
| (4) & (5) Resumen de costes interanuales por tipo | Seleccione el soporte angular TCO de almacenamiento para saltar a la pestaña TCO de almacenamiento.    Utilice el gráfico para ver un resumen de sus costes interanuales desglosados por cada tipo de sistema de almacenamiento (como redes de área de almacenamiento [SAN], almacenamiento conectado a la red [NAS] y sistemas de cinta).  La tabla proporciona detalles sobre el coste YTD, el porcentaje de cambio, el espacio, el porcentaje de cambio GB y el coste por GB. Haga clic en un elemento de la columna Tipo para abrir un cuadro de diálogo con más detalles     - Indicadores clave de rendimiento: Proporcione una visión de alto nivel de su gasto en almacenamiento. - Gráfico y tabla de tendencias: El gráfico de tendencias muestra los costes por dirección no asignada mes a mes. La tabla muestra la ubicación y el fabricante de cada activo de almacenamiento, y desglosa el coste, el coste YTD, el espacio instalado, el espacio direccionable y el espacio asignado para cada ubicación y fabricante. |
| (6) & (7) YTD Public Cloud OpEx Resumen por categoría de servicio | Seleccione el Public Cloud TCO soporte angular para saltar a la página de aterrizaje Public Cloud . |
