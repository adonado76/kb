# Gestión informática - Unidades de negocio - Informe directo/indirecto ( v103 )

Utilice este informe para revisar los costes directos e indirectos por unidad de negocio.

Se aplica a: Costing Standard 11.8.x que se ejecuta en TBM Studio v12 o TBM Studio v11.

![](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/itmbudirectindirect103_1.png)

## Navegación

Gestión informática > Unidades de negocio > Directa/Indirecta

## Funciones

Este informe está destinado a:

- Propietarios de unidades de negocio
- directores de informática
- directores financieros

## Objetivos

Utilice este informe para revisar los costes directos e indirectos por unidad de negocio.
