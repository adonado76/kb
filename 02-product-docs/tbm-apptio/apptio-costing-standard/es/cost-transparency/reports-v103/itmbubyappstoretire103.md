# Gestión informática - Unidades de negocio - Informe By Apps to Retire ( v103 )

Utilice este informe para ahorrar costes si se retira una aplicación.

Se aplica a: Costing Standard 11.8.x que se ejecuta en TBM Studio v12 o TBM Studio v11.

![](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/itmbubyappstoretire103_1.png)

## Navegación

Gestión informática > Unidades de negocio > Por aplicaciones a retirar

## Funciones

Este informe está destinado a:

- Propietarios de unidades de negocio
- directores de informática
- directores financieros

## Objetivos

Utilice este informe para:

- Revisar los costes mensuales y anuales de las aplicaciones.
- Compare los costes por inversión en aplicaciones, unidad de negocio y familia de aplicaciones.

## Preguntas contestadas

La información presentada en este informe puede utilizarse para responder a las siguientes preguntas:

- ¿Existen aplicaciones cuyo coste no justifique su mantenimiento?
- ¿Qué aplicaciones son las más costosas? ¿Está justificado ese coste?

## Próximas acciones

Haga un seguimiento de los usuarios de la aplicación para ver si siguen teniendo una razón válida para seguir utilizándola.
