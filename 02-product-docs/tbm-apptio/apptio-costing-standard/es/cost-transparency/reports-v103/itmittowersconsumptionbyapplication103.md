# IT Management - IT Tower Details - Informe de consumo por aplicación ( v103 )

◆ Se aplica a: Costing Standard 11.8.x que se ejecuta en TBM Studio v12 o TBM Studio v11.

## Introducción

Utilice este informe para ver los gastos por aplicación.

![](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/itmittowerconsumptionbyapplication103_1.png)

## Navegación

Gestión TI > Torres TI > Nombre de la torre TI > Consumo por aplicación

## Funciones

Este informe está destinado a:

- Gestión de TI
- Propietario de torre de TI

## Objetivos

Utilice este informe para:

- Vea los 10 informes más caros.
- Revisar los costes de aplicación por torre de TI, subtorre y propietario de torre.

## Preguntas contestadas

La información presentada en este informe puede utilizarse para responder a las siguientes preguntas:

- ¿Mi gasto va a respaldar las aplicaciones más críticas de la organización?
- ¿Qué aplicaciones se retirarán?
- Cuál es la tendencia de costes de una aplicación.

## Próximas acciones

- Revisar los planes para eliminar las aplicaciones retiradas.
- Ajustar el gasto para apoyar las aplicaciones más críticas.
