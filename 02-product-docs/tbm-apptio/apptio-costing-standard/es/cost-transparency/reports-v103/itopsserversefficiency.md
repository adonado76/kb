# Operaciones de TI - Informe de objetivos de eficiencia de servidores

◆ Se aplica a: Costing Standard 11.8.x que se ejecuta en TBM Studio v12 o TBM Studio v11.

## Introducción

Utilice este informe para comparar el coste de las asignaciones de memoria y CPU con los objetivos corporativos.

![](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/itopsserversefficiency_1.png)

## Navegación

Infraestructura y operaciones de TI > Informe sobre la eficiencia de los servidores

## Funciones

Este informe está destinado a:

- Líderes/gestores de servidores

## Objetivos

Utilice este informe para:

- Comparar la memoria asignada con los objetivos corporativos.
- Comparar la CPU asignada con el objetivo corporativo.

## Preguntas contestadas

La información presentada en este informe puede utilizarse para responder a las siguientes preguntas:

- ¿Se ajusta el coste de la memoria asignada a los objetivos corporativos?
- ¿Cuál es la tasa efectiva por GB de memoria asignado?
- ¿Se ajusta el coste por CPU asignada a los objetivos corporativos?
- ¿Cuál es la tasa efectiva por núcleo de CPU?
