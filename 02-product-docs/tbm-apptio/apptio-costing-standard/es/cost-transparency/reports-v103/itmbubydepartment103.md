# Gestión informática - Unidades de negocio - Informe por departamentos ( v103 )

Utilice este informe para ver los detalles de los departamentos de una unidad de negocio.

Se aplica a: Costing Standard 11.8.x que se ejecuta en TBM Studio v12 o TBM Studio v11.

![](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/itmbubydepartment103_1.png)

## Navegación

Gestión de TI > Unidades de negocio > Por departamento

## Funciones

Este informe está destinado a:

- Propietarios de unidades de negocio
- directores de informática
- directores financieros

## Objetivos

Utilice este informe para ver los detalles de los departamentos de una unidad de negocio.

## Preguntas contestadas

La información presentada en este informe puede utilizarse para responder a las siguientes preguntas:

¿Es necesario tomar medidas para mitigar el riesgo?

## Próximas acciones

Utilice los divisores para filtrar el informe por órgano de gobierno, tipo de unidad de negocio o gestor de relaciones (de negocio).
