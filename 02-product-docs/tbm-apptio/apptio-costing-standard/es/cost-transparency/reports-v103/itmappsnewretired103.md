# Gestión informática - Aplicaciones - Informe nuevo y retirado ( v103 )

Utilice este informe para revisar los recuentos de solicitudes nuevas y retiradas.

Se aplica a: Costing Standard 11.8.x que se ejecuta en TBM Studio v12 o TBM Studio v11.

![](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/itmappsnewretired103_1.png)

## Navegación

Gestión TI > Aplicaciones > Nuevas y retiradas

## Funciones

Este informe está destinado a:

- Propietarios de aplicaciones
- Propietarios de la cartera de aplicaciones / Vicepresidente de Desarrollo y Soporte de Aplicaciones
- Arquitectos de empresa

## Objetivos

Utilice este informe para:

- Revisar las tendencias del recuento de solicitudes en periodos de 3 meses.
- Aplicaciones introducidas en los últimos 12 meses.
- Solicitudes retiradas en los últimos 12 meses.

## Preguntas contestadas

La información presentada en este informe puede utilizarse para responder a las siguientes preguntas:

- ¿Coincide el número de solicitudes con las jubilaciones previstas?
- ¿Deben retirarse otras aplicaciones?
- ¿Se introducen las aplicaciones con demasiada rapidez?
- ¿Es necesario tomar medidas para mitigar el riesgo?

## Próximas acciones

Haga clic en una aplicación para ver un informe detallado.
