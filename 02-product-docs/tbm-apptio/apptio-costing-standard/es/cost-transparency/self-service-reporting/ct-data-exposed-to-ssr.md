# Datos estándar de cálculo de costes disponibles en Apptio BI

Apptio BI informará con respecto a su entorno de producción Costing Standard .

Se aplican los siguientes requisitos mínimos de la versión Costing Standard :

- Informes para datos de fuentes de costes y torres: Costing Standard versión 12.8.1 o posterior
- Informes de datos adicionales listos para usar: Costing Standard versión 12.8.3 o posterior
- Informes para campos personalizados en datos out-of-the-box, conjuntos de datos modelados personalizados, métricas modeladas personalizadas: versión 12.9.7
- Compatibilidad multidivisa: Costing Standard versión 12.9.1 o posterior
- Soporte de seguridad a nivel de fila: versión 12.9.1. o posterior
