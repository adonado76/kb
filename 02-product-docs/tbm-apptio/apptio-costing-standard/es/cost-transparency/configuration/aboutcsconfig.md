# Acerca de la configuración de los servicios en la nube en Costing Standard

Utilice el componente CTF - Cloud Service Provider para importar datos de uso de servicios web. El componente es opcional.

## Introducción

Utilice el componente CTF - Cloud Service Provider para importar datos de uso de servicios web. El componente es opcional.

Si utiliza Amazon Web Services ( AWS ), Microsoft Azure, SoftLayer, o CenturyLink, puede importar los datos de uso de los informes mensuales de uso y facturación. Los datos se utilizan en los informes CTF y en los informes de aplicaciones y servicios. Los datos pueden ayudarle a comparar el coste de los servicios en nube con el de los servicios internos. Para utilizar los datos de los servicios web, debe instalar el componente CTF - Cloud Service Provider.

## Dos procedimientos de configuración diferentes

Existen dos procedimientos de configuración diferentes:

- [Amazon Web Services configuración](aws.html)
- [Microsoft Azure y la configuración de otros proveedores](azure.dita)

Cada procedimiento se documenta por separado.

## Vinculado a proveedores

Para que el coste de la nube se calcule correctamente, los datos del proveedor deben introducirse en la aplicación Costing Standard . Los datos del proveedor identifican los costes procedentes de los servicios web y los asignan a los servicios en nube. El modelo que figura a continuación muestra la relación.
