# Introducción a la aplicación Costing Standard

- [Acerca de la aplicación Costing Standard](aboutct.html "Utilice la aplicación Costing Standard para recopilar los datos que necesita para tomar decisiones más rápidas alineadas con el negocio, mejorar la eficiencia y comunicar valor.")
- [Acerca de los componentes de Costing Standard](aboutcomponent.html "Un componente de la aplicación Costing Standard agrupa informes y métricas para proporcionar información sobre su empresa. Un componente también puede instalar una o más tablas modelo en su proyecto. La figura A muestra los componentes disponibles en la aplicación Costing Standard.")
- [Acerca de las personalizaciones de su proyecto](aboutcustom.html "Si ha personalizado el contenido de Apptio e instala un componente, el proceso de actualización no actualizará los elementos personalizados del componente. La actualización mantendrá intactos los elementos personalizados (no se aplicará ningún cambio) y actualizará los elementos que no se hayan personalizado.")
