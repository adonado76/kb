# Microsoft Azure y la configuración de otros proveedores

Para configurar la aplicación para Microsoft Azure u otros proveedores, debe instalar el componente CTF-Cloud Service.

## Paso 1: Instalar el componente CTF - Cloud para Azure

El componente Proveedor de servicios en la nube debe estar instalado antes de instalar este componente.

Para instalar el componente CTF - Azure :

### Procedimiento

1. Abra el proyecto Costing Standard .
2. Haga clic en la pestaña **Proyectos**.
3. Haga clic en **Componentes** en la cinta.
4. Haga clic en el componente **CTF - Azure**.
5. Haga clic en **Instalar**.

### Resultados

Al instalar el componente Azure se crean los siguientes conjuntos de datos en la categoría Proveedor de servicios en la nube:

- Azure Datos maestros de facturas EA
- Azure Datos maestros de factura detallados de EA

Se crea el siguiente elemento personalizado:

- Azure EA Factura detallada en bruto
