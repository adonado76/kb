# Taller de comunicación

## Preparación de los datos de comunicación

1. Si aún no lo has hecho, carga tus datos de comunicación incluyendo:
   - Comunicación
2. Siguiendo las prácticas habituales de Apptio las prácticas habituales, clasifique cada conjunto de datos brutos en la categoría de Comunicaciones
3. Si procede, aplique un filtro de fechas a sus datos de comunicación.

## Acerca de los identificadores de comunicación

Por defecto, el identificador del conjunto de datos maestros de comunicaciones es el identificador de objeto. Debe tener alguna forma de identificar cada dispositivo de comunicación y utilizarlo como identificador de objeto.

## Acerca de las claves de comunicación

Algunas de las claves del conjunto de datos maestros de comunicaciones son generadas por el sistema y otras son definidas por el usuario. Cualquier clave generada por el sistema no debe ser alterada. Las teclas predefinidas son:

- Tecla ITRT\_Comunicaciones
- Communications\_End User Devices Clave
- Clave\_Servidores\_de\_Comunicación
- Communications\_Services Clave

## Asignar datos a los datos maestros de comunicación

Asigne sus datos de Comunicaciones al conjunto de datos de Datos Maestros de Comunicaciones. La mayor parte de la cartografía se explica por sí misma en este punto. En 12.7 puede utilizar la función de asignación de columnas en su conjunto de datos brutos para asignarlos a los datos maestros de comunicaciones ([Map Columns](https://community.apptio.com/docs/DOC-10561 "(se abre en una pestaña o una ventana nueva)") )

## Informes de comunicación

Actualmente no hay informes que se iluminen después de añadir datos al conjunto de datos Datos maestros de comunicaciones.
