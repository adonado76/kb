# Fixed Assets Reports

The **Fixed Assets report collection** provides visibility into IT asset
depreciation, aging, and lifecycle status across the organization. This collection includes:

**Fixed Assets**

The Fixed Assets report provides detailed visibility into depreciated and depreciating IT
assets over time. It shows asset information such as class, age, location, IT resource tower,
vendor, asset type, and category, along with depreciation metrics.

Use this report to review asset aging, track depreciation status, and understand how asset
costs evolve throughout their lifecycle.

This report is designed for the following roles:

- IT Finance
- Service Owners

## Insights Provided

- Identify assets that are fully depreciated and nearing end-of-life.
- Understand remaining depreciation for active assets across different age ranges.
- Review asset distribution by age, vendor, location, IT tower, asset type, and category.
- Monitor asset OpEx depreciation trends year to date.
- Analyze depreciation costs and remaining asset value at a detailed asset level.

For more details on how to use the Fixed Assets report go [here](https://www.ibm.com/docs/en/apptio-commercial/costing-standard/saas?topic=reports-fixed-assets "(Opens in a new tab or window)")
