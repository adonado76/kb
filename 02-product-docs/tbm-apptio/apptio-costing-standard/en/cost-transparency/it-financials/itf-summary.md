# Summary

**Financial Usecases**

This section provides the foundational financial capabilities required to understand, govern,
and manage IT spend. It brings together actuals, budgets, forecasts, and allocations to
establish a consistent financial view across IT costs, enabling transparency, accountability,
and informed financial decision-making. This financial foundation supports all downstream
cost, vendor, asset, project, and cloud analyses across the platform.

**IT Financials**

IT Financials manage the core financial data used across Costing, including OpEx and CapEx
actuals, budgets, and forecasts. This capability ensures financial data is standardized,
governed, and consistently mapped to master data, forming the backbone for all IT cost
reporting and analysis.

**Vendors**

Vendor reporting delivers financial visibility into IT supplier spend, helping organizations
understand spend concentration, fragmentation, and variance across vendors. It supports
budgeting, cost control, and vendor optimization decisions using consistent financial data.
Vendor Insights extends vendor financial analysis into invoices, purchase orders, and
contracts, enabling tighter financial governance and risk management. It helps ensure spend
accuracy, contract compliance, and proactive oversight of vendor financial commitments.

**Labor & Time Tracking**

Labor and Time Tracking provides financial visibility into internal labor costs by capturing
how effort translates into spend. This capability improves cost allocation accuracy and
ensures labor is reflected correctly in service, project, and total cost views.

**Fixed Assets**

Fixed Assets provides financial governance over capitalized IT investments by tracking asset
values, depreciation, and allocations. This ensures CapEx is accurately represented in
financial reporting and aligned with ongoing IT costs.

**Public Cloud**

Public Cloud costing delivers financial visibility into cloud spend, supporting both fully
burdened cost analysis and IT-finance-focused reporting. It enables accurate tracking,
allocation, and control of cloud expenses as part of the broader IT financial picture.

**Projects**

Project costing tracks financial investments across IT initiatives, enabling comparison of
actual spend against budget and forecast. It helps organizations understand how
project-related costs contribute to overall IT financial performance.

**Resource Towers**

Resource Towers provide a financial view of IT spend organized by functional areas, enabling
analysis of cost distribution, trends, and budget alignment. This view supports ongoing
financial reviews by highlighting where costs are increasing, stabilizing, or deviating from
plan.
