# IT Management - Applications - By IT Tower report (v103)

Use this report to Review application spending across IT towers and by application
family.

Applies to: Costing Standard 11.8.x running on either TBM Studio v12
or TBM Studio v11.

![image](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/itmappsbyittower103_1.png)

## Navigation

IT Management > Applications > By IT Tower

## Roles

This report is designed for:

- Application Owners
- Application Portfolio Owners / VP of Application Dev & Support
- Enterprise Architects

## Objectives

Use this report to review application spending across IT Towers and by application family.

## Questions answered

The information presented on this report can be used to answer the following questions:

- For a given application, is the consumption of data centers, servers, and storage costs correct?
- Is action required to mitigate budget risk?

## Next actions

application.
