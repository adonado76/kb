# IT Finance - Outliers report (v103)

Applies to: Costing Standard 11.8.x running on either TBM Studio v12
or TBM Studio v11.

## Introduction

Use this report to review budget variance by cost pool and cost center owner. This report was
updated in R11.8.1.

![](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/itfoutliers103.png)

## Navigation

IT Finance > Outliers

## Roles

This report is designed for IT Finance personnel.

## Objectives

Use the charts in the report to:

- Budget variance by cost pool
- Budget variance by cost center owner

Use the tables in the report to:

- Identify the top over and under budgets by account
- Identify the top over and under budgets by cost center

## Questions answered

Use the information presented on this report to answer the following questions:

- Which cost pools have the largest variance?
- What are the top over-budget cost center owners?
- Where do I prioritize investigation to understand and help explain any unexpected variance?

Use the tables in the report to answer the following questions:

- Which specific accounts do I need to investigate to explain deviations?
- Which specific cost centers do I need to investigate to explain deviations?
- Are the budget variances significant enough to warrant further investigation?

## Next actions

- If variance is immaterial, no concern or action is needed.
- Click on a specific account to see transaction level details.
- Click on a specific cost center to see transaction level details.
