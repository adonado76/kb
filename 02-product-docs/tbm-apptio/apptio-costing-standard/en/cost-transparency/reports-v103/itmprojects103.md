# IT Management - Projects report (v103)

◆ Applies to: Costing Standard 11.8.x running on either TBM Studio v12 or TBM Studio
v11.

## Introduction

Use this report to review project CapEx and OpEx spend and status from a variety of perspectives.
To change the perspective, select a tab: At Risk, By Portfolio, By Investment Type, By Project
Manager, By IT Owner, By Business Sponsor, and Project Details.

![](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/itmprojects_intro1.png)

## Navigation

IT Management > Projects

## Roles

This report is designed for:

- CIO
- IT Management

## Questions answered

Use the information presented in the Project reports to answer the following questions:

- What percent of my IT spend is going towards growth or innovation?
- How has my project spend been progressing against the overall budget?
- Is project spend trending as planned for the current year?
- How are my top projects doing against budget and other project status?
- Who is the project manager if I need more information?
- Who is the project owner if I need to communicate or manage expectations?
- Which projects are over-budget and require further analysis to determine why and if additional
  funding is warranted?

## Objectives

The objectives for the reports on the tabs are described below.

## Summary tab

Use to:

- Review the CapEx and OpEx spend and project status.
- Review project spend by status, priority, and project manager.

See the figure above for a screen capture of the report tab.

See the figure above for a screen capture of the report tab.

## At Risk tab

Use to:

- Review project status based
- Drill into a project to see a project summary, drivers by cost pool, and spend by vendor.

![](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/itmprojects_atrisk2.png)

## By Business Initiative tab

Use to:

- Review spend by business initiative.
- Drill into a business initiative to see details.

![](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/itmprojects_bybusinessinitiative3.png)

## By Investment Type tab

Use to:

- Review spend by RTB (Run-the-Business), GTB (Grow-the-Business), and TTB
  (Transform-the-Business).
- Drill into an investment type to see details.

![](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/itmprojects_byinvestmenttype4.png)

## By Business Sponsor tab

Use to:

- Review spend by business sponsor and sponsor business unit.
- Drill into a business sponsor to see details.

![](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/itmprojects_bybusinesssponsor5.png)

## By Portfolio tab

Use to:

- Review spend by project portfolio and business initiative.
- Drill into a portfolio to see details.

![](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/itmprojects_byportfolio6.png)

## Labor Analysis tab

Use to:

- Review the staffing by roles.
- Filter by employee type, project name, and role type.

![](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/itmprojects_laboranalysis7.png)

## Applications Impacted tab

Use to:

- Review the applications impacted by each project.
- Filter by project status, business initiative, and project portfolio.

![](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/itmprojects_applicationsimpacted8.png)

## Project Details tab

Use to:

- Review details for all projects by months, quarters, and years.
- Filter by business unit and business sponsor.
- Sort by budget, budget variance, status, and priority.
- Drill into a project to see details about the project drivers, vendors, labor, and applications
  impacted.

![](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/itmprojects_projectdetails9.png)
