# IT Management - Applications - Infrastructure Details report (v103)

Use this report to review the Compute IT Towers consumption by the
applications.

Applies to: Costing Standard 11.8.x running on either TBM Studio v12
or TBM Studio v11.

![](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/itmappsinfrastructuredetails103_1.png)

## Navigation

IT Management > Applications > Infrastructure Details

## Roles

This report is designed for:

Review the Compute IT tower’s consumption by the applications.

## Objectives

Use this report to review the Compute IT tower’s consumption by the applications.

## Questions answered

The information presented on this report can be used to answer the following questions:

- For a given application, is the consumption of Compute (server) resources appropriate?
- Must anything be done to mitigate risk?

## Next actions

- Click the Storage and Service Desk tabs to obtain similar reports.
- Click an application name to obtain an Application Portfolio – Detail report for the application.
