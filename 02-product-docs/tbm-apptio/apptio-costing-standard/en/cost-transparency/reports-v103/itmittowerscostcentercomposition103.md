# IT Management - IT Towers Details - Cost Center Composition report (v103)

◆ Applies to: Costing Standard 11.8.x running on either TBM Studio v12 or TBM Studio
v11.

## Introduction

Use this report to identify spend by cost pool and cost center for each subtower.

![](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/itmittowerscostcentercomposition103_1.png)

## Navigation

IT Management > IT Towers > IT Tower Name > Cost Center Composition

## Roles

This report is designed for:

- IT Management
- IT Tower Owner

## Objectives

Use this report to:

- Identify spend by cost pool and cost center for each sub-tower.
- Focus on a specific sub-tower or tower owner using the filters.

## Questions answered

The information presented on this report can be used to answer the following questions:

- In which cost centers is spending occurring?
- Are there costs coming from unexpected cost centers?
- Is action required to further investigate spend in a particular cost Pool?

## Next actions

- Look at cost center and cost pool spend for previous periods to identify trends (increasing,
  decreasing or staying consistent).
- Look at account transactions for the period using the Budget Variance tab.
- Contact an IT Finance analyst for further investigation.
