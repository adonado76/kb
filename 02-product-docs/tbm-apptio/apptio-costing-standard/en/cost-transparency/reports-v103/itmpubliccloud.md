# IT Management - Public Cloud report

◆ Applies to: Costing Standard 11.8.x running on either TBM Studio v12 or TBM Studio
v11.

## Introduction

Use this report to review cloud costs by category, IaaS/PaaS, SaaS, and labor.

![public cloud report](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/itmpubliccloud_1.png)

## Navigation

IT Management > Cloud Provider

## Roles

This report is designed for:

- CIO
- IT Operations Leadership

## Objectives

Use this report to:

- Review public cloud spend KPIs.
- View public cloud spend and products by category, IaaS\PaaS, SaaS, Labor.
- View public cloud spend by cost center, cost pool, IT resource tower and sub-tower for the
  current period, quarter, and YTD.

## Questions answered

The information presented on this report can be used to answer the following questions:

- What percent of our technology spend (as a percent of our total modeled tower spend) is in
  public cloud services?
- How is that spend trending over past 6 months?
- What products are providing our Compute Linux service? Compute Windows?

## Next actions

- Click a tab to view the data by category, IaaS\PaaS, SaaS, and Labor.
- Review the details for each vendor by clicking the Details tab.
- Review the transaction details for a vendor by clicking View in the Tx column.
