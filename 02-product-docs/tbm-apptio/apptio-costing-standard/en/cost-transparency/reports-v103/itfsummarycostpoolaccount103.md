# IT Finance - Summary - Cost Pool - Account report (v103)

Applies to: Costing Standard 11.8.x running on either TBM Studio v12
or TBM Studio v11.

## Introduction

Use this report to identify and verify the accounts mapped to a specific cost pool.

![](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/itfsummarycostpoolaccount103_1.png)

## Navigation

IT Finance > Summary > Click on a cost pool in the table

## Roles

This report is designed for IT Finance Personnel.

## Objectives

Use this report to:

- Identify and verify the accounts mapped to a specific cost pool (e.g. cost category).
- View the spend and variance for the current period and YTD.

## Questions answered

Use the information presented on this report to answer the following questions:

- Do all these accounts belong in this cost pool category?
- If an account does not belong, where should it be moved?
- By account, where is my biggest spend & variance -- by amount and by percent?
- Is action required to mitigate budget risk?

## Next actions

- If variance is immaterial, no concern or action is needed.
- To see the transaction details for an account, click an account.
