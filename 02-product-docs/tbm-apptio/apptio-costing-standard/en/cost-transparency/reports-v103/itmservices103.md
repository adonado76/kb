# IT Management - Services reports (v103)

◆ Applies to: Costing Standard 11.8.x running on either TBM Studio v12 or TBM Studio
v11.

## Introduction

Use this report to review the service costs by owner, category, and service type.

![](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/itmservices103_1.png)

## Navigation

IT Management > Services

## Roles

This report is designed for:

- Service owners
- CIO

## Objectives

Use this report to:

- Review the services by owner, service category, service type, and cost pool.
- Review the services costs for the current period and year-to-date.

## Questions answered

The information presented on this report can be used to answer the following questions:

- Which services have the greatest spend?
- Does the spend match the priorities assigned to the investment objectives?

consumption, and cost pool.
