# IT Management - Financials - By Cost Pool report (v103)

The IT Management - Financials - By Cost Pool report shows all IT spend by the standard
cost pool structure.

Applies to: Costing Standard 11.8.x running on either TBM Studio v12
or TBM Studio v11.

![](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/itmfinancialsbycostpool103_1.png)

## Navigation

IT Management > Financials > By Cost Pool

## Roles

This report is designed for:

- CIO
- IT Management

## Objectives

Use this report to:

- CIO view of all IT spend by the Apptio standard Cost Pool structure.
- Filter by IT Owner (CIO-1) or specific Cost Center Owner to review spend by account sub-group.
- Add Cost Sub-Pool category detail.

## Questions answered

The information presented on this report can be used to answer the following questions:

- Where do we have the largest category spend?
- Are we spending more in this cost category than we planned?
- Are there opportunities to better control or reduce spend in an area?

## Next actions

Click an entry in the Cost Pool column to analyze spend in the category by Cost Center and to
view the detailed transactions.
