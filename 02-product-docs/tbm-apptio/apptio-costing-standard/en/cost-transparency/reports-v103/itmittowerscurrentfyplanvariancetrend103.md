# IT Management - IT Towers Details - Current FY Plan Variance - Trend report (v103)

◆ Applies to: Costing Standard 11.8.x running on either TBM Studio v12 or TBM Studio
v11.

## Introduction

Use this report to see the volumes and unit costs by month for the past 13 months and to review
the volume, total costs, unit costs and percent change by month.

![](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/itmittowerscurrentfyplanvariancetrend103_1.png)

## Navigation

IT Management > IT Towers > IT Tower Name >Trend View

## Roles

This report is designed for:

- IT Management
- IT Tower Owner

## Objectives

Use this report to:

- Quickly see the spend and budget by month for the past 13 months.
- Review supporting data to see the budget variance and percent variance by month.

## Questions answered

The information presented on this report can be used to answer the following questions:

- How is my spend for my IT sub-tower fluctuating over time?
- Is the spend trending up, down, or remaining constant?
- Is my spend tracking to the planned budget or am I seeing spikes in over/under budget
  variance?
- Is action required to mitigate budget risk?

## Next actions

- View account transactions for current or previous months using the Transaction report.
- Investigate volume and unit costs using the Transaction report.
