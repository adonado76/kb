# IT Management - Applications reports

Applies to: Costing Standard 11.8.x running on either TBM Studio v12
or TBM Studio v11.

## Introduction

The IT Management Applications reports focus on application spend from a wide range of
perspectives.

Use the IT Management Applications reports to view application spend by:

- Top spend
- Category
- IT Tower
- Infrastructure details
- Vendor
- Projects
- New and retired applications
- Application details
