# IT Finance - Account Variance report (v103)

Applies to: Costing Standard 11.8.x running on either TBM Studio v12
or TBM Studio v11.

## Introduction

Use this report to look at all accounts by cost category to see where total spend is going.

![image](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/itfaccountvariance103_1.png)

## Navigation

IT Finance > Acct Variance

## Roles

This report is designed for IT Finance personnel.

## Objectives

Use this report to:

- Look at all accounts by cost category to see where total spend is going.
- Search/slice by account groups to find and view specific category spend.
- Include other account hierarchy filters to organize and view information.

## Questions answered

You can use the information presented on this report to answer the following questions:

- Where do we have the largest category spend?
- Are we spending more in this cost category than we expected?
- Are there opportunities to better control or reduce spend in an area?

## Next actions

- Click on View in the Variance Trend column to view the monthly variance YTD.
- Click on an account in the Account Description column to view the detailed transactions.
