# IT Operations - Storage Efficiency Target report

◆ Applies to: Costing Standard 11.8.x running on either TBM Studio v12 or TBM Studio
v11.

Use this report to compare storage allocation and utilization statistics to corporate
targets.

![](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/itopsstorageefficiency_1.png)

## Navigation

IT Infra & Operations > Storage Efficiency Target report

## Roles

This report is designed for:

- Storage leaders/managers

## Objectives

Use this report to:

- Compare the allocated storage to corporate targets.
- Compare the cost of utilization to corporate targets.

## Questions answered

The information presented on this report can be used to answer the following questions:

- Does the cost of allocated memory align with the corporate targets?
- What is the effective rate per allocated GB of memory?
- Does the cost per CPU allocated align with the corporate targets?
- What is the effective rate per CPU core?
