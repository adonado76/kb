# IT Management - Applications - New and Retired report (v103)

Use this report to review the counts for new and retired applications.

Applies to: Costing Standard 11.8.x running on either TBM Studio v12
or TBM Studio v11.

![](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/itmappsnewretired103_1.png)

## Navigation

IT Management > Applications > New & Retired

## Roles

This report is designed for:

- Application Owners
- Application Portfolio Owners / VP of Application Dev & Support
- Enterprise Architects

## Objectives

Use this report to:

- Review application count trends over 3-month spans.
- Applications introduced in the most recent 12 months.
- Applications retired in the most recent 12 months.

## Questions answered

The information presented on this report can be used to answer the following questions:

- Is our application count trending with our planned retirements?
- Should other applications be retired?
- Are applications being introduced too quickly?
- Is action required to mitigate risk?
