# Labor Review - Labor Details - Cost Composition Report

Applies to: Costing Standard R12

Use this report to understand which accounts contribute to a labor spend.

![](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/laborreviewlabordetailscostcomposition_1.png)

## Path

Labor – Labor Review tab – Labor Details by Cost Center and Name (Details view) – drill down to a
Cost Center – Cost Composition tab

## Roles

This report is designed for:

- IT Finance Personnel
- Cost Center Owners

## Objectives

Use this report to:

- Understand which accounts contribute to the labor spend for a cost center.
- Review and manage labor spend.
- Make informed staffing decisions.

## Questions Answered

You can use the data on this report to answer the following questions:

- Which GL accounts contribute to the labor spend for a cost center?
- Which accounts are causing the variance from the current FY plan?
- Does the amount tied to a specific account make sense for my cost center?
- Is action required to further investigate spend for any account?

## Next Actions

View the transactions for each account by clicking **View** in the **Tx** column.
