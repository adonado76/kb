# IT Management - Applications - By Projects report (v103)

Use this report to identify application spend by project and application
family.

Applies to: Costing Standard 11.8.x running on either TBM Studio v12
or TBM Studio v11.

![](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/itmappsbyprojects103_1.png)

## Navigation

IT Management > Applications > By Projects

## Roles

This report is designed for:

- Application Owners
- Application Portfolio Owners / VP of Application Dev & Support
- Enterprise Architects

## Objectives

Use this report to identify application spend by project and application family.

## Questions answered

The information presented on this report can be used to answer the following questions:

- How much have we spent by application related projects?
- Does the spend match the investment objectives of the applications?

## Next actions

Use the IT Finance – Projects report to follow up with the project managers.
