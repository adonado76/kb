# Application Review

Use this report to monitor spending across your application portfolio, comparing
costs across operational and development environments to identify trends and high-cost applications.
Leverage filters to analyze spend by application, type, owner, or family for targeted cost
management decisions.

This report is designed for use by the following personas:

- CIO
- IT Financial Controllers
- Application Portfolio Managers
- CFO
- Business Unit Leaders

## Key Elements

![](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/nx-apprev-1.png)

![](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/nx-apprev2.png)

| Element | Description |
| --- | --- |
| Filter controls (1) | Four filters let you narrow the report by application name, application type, application IT owner, and application family. |
| Summary Cards (2) | Three summary cards show total application operating expense year to date, application run year to date, and application count. |
| Year To Date Application OpEx Spend by Category (3) | This section shows year-to-date application operating expense by the selected dimension. |
| Top 5 Total App OpEx Spend YTD by Category (4) | A horizontal bar chart shows the five applications with the highest year-to-date operating expense. |
| Top 5 Total App Spend Trend by Category (4) | A line chart shows the spending trend over time for the top five applications. |
| Application Spend Details Tables (5) | This table includes columns such as application name, application family, application business owner, application investment objective, application run year to date, application development year to date, and total application operating expense year to date. |

## Questions Answered

- What is our total application spending and how does it compare to overall IT spending?
- Which applications are the most expensive to run?
- How is application spending trending over time?
- What percentage of application spending is for running apps versus developing new features?
- How many applications do we have compared to last year?
- Which application families (Finance, Sales, HR, etc.) consume the most budget?
- What are the investment objectives for our most expensive applications?

## Recommended Actions

- Review the top 5 most expensive applications and assess if their spending levels are justified
  by business value.
- Check the App Run versus App Dev split to see if you're spending too much on maintenance versus
  innovation.
- Use the pivot selector to view spending by Application Family or Application Type to identify
  which categories need cost reduction.
- Look at the trend chart to identify applications with increasing costs and investigate the
  reasons for growth.
- Filter by Application Investment Objective to see how much you're spending on apps marked for
  "Migrate" or "Eliminate" that should have declining costs.
- Compare your Application Count (145) to previous year to understand if portfolio rationalization
  efforts are working.
- Click on application names in the details table to drill into specific cost drivers and
  optimization opportunities.
