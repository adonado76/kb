# Infra & Cloud Report Collection

This collection features automated data feeds from public cloud service providers,
aggregating and translating billing detail into standard IT categories. You can view public cloud
TCO with fully burdened cost in the same report as public cloud (IaaS/SaaS) and on-premises
assets.
