# IT Towers Report Collection

This collection provides cost-per-unit so you can see total cost and efficiency trends
according to the standard ATUM taxonomy, aligned with peer benchmarks. Intuitive analytics start
with a summarized view that allow you to drill down to details when desired.

The automated model shows how GL budget variance impacts towers/functions:

- [IT Towers Performance](ittowersperformance.html)
