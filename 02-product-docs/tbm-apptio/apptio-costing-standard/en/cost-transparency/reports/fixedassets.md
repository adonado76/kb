# Fixed Assets

This report shows you the details of your deprecated assets (such as class, age,
location, IT tower, vendor, asset type, and category) over time.

![](../../resources/images/ct_images/ct%20report%20collections/fixed%20assets.jpg)

| Key element | Description |
| --- | --- |
| (1) Report Collection | This report collection provides the details you need to review your resource (labor and IT tower) spend, trends, and usage:   - Fixed Assets report (described on this page) - [ITP – Asset Review report](itp-asset-review.html "This report allows you to view the details of your deprecated assets (such as class, age, location, IT tower, vendor, asset type, and category) over time.") |
| (2) Slicers | Two sets of slicers in this report let you see your asset data limited by age range, vendor, then by asset type, category, and IT resource tower.  The following roles can use the slicers in this report for a more personalized view:   - IT Financial Controller or CIO: Without setting any slicers, you can see the overview of the   asset spend by class, age range, location, IT tower, vendor, asset type, or category. You can also   see data about life in months, accumulated/remaining deprecation, and deprecation costs of specific   assets. - Cost Center Owner or CIO -1: Set the filters to see specific areas of responsibility and manage   your deprecation budget and timing. - Financial Analyst: Set the slicers for areas you support and see asset deprecation across the   company. |
| (3) KPIs | KPIs provide a high-level view of your asset OpEx deprecation:   - Asset OpEx YTD Under 1 Yr: This KPI expresses the portion of your asset OpEx YTD (cost and   count) that is less than one year old. - Asset OpEx YTD Under 1 to 3 Yrs: This KPI expresses the portion of your asset OpEx YTD (cost and   count) that is between 1-3 years old. - Asset OpEx YTD Under 3 to 5 Yrs: This KPI expresses the portion of your asset OpEx YTD (cost and   count) that is between 3-5 years old. - Asset OpEx YTD Greater Than 5 Yrs : This KPI expresses the portion of your asset OpEx YTD (cost   and count) that is over 5 years old. |
| (4) Charts and table | View your asset count and spend by age range across cost centers, roles, and locations.  • Summary: Use this tab for an overview of your asset deprecation:  • OpEx YTD and Asset Count by Age Range: These charts provide a quick view of the age range of your assets based on the slicers you select.  • Asset Depreciation Details: Use this table to see the purchase price, accumulated depreciation, and depreciation for the current month/YTD for the pivot you select above the table. Select the options above the table to add columns.  • Depreciation LIM: Use this tab to view the current deprecation of each asset in terms of life in months, accumulated/remaining deprecation, and deprecation costs. Use the options above the table to add columns.  A screenshot of a social media post Description automatically generated  • Fully Deprecated: Use this tab to view a chart and table with the count of fully deprecated assets per tower and details about each class of assets, including accumulated/remaining deprecation and deprecation costs. Use the Select Pivot fields to change the view to IT tower, sub-tower, or category. Use the options above the table to add columns. |

Questions answered:

You can use this report to answer the following questions:

- Which assets are fully depreciated?
- What depreciation remains for existing fixed assets by time period?
