# Application Cost Take-Out

![Application Cost](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/application-cost-report.png)

| Key Benefits | Details |
| --- | --- |
| - View application costs, unit costs, and per-user costs - Drill-down into the monthly spend for each application - See the cost savings you've achieved by retiring applications - Identify areas which can rationalize your application portfolio   **Questions Answered**   - What is the average cost per user of each application? - Are we paying for redundant or duplicate applications that are serving the same purpose? - We have this application's contract expiring soon – should we renew it? | **For** :  Solution Leaders, Application Owners, Service Owners  **How to Navigate**: Go to **Reports** > **Application (Collection)** > **Application Cost Take-Out** |

**Insights:**

Application Cost Take-Out Report has been designed to answer the following questions:

**Application Rationalization**:  
This shows the Annual Application Run Spend along
with the Business Owners. The following view can be used to identify overlap within a given
application function, enabling the elimination of redundant applications as part of the Application
Portfolio Rationalization activity.

![Application Rationalization](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/application-rationalization.png)

**Retired Applications** :  
This shows applications flagged for retirement or
elimination,with remaining run spend and target dates. This information is used to schedule
decommissioning and capture savings.

![Retired Application](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/retired-application.png)

**Expiring Contracts**:  
This shows contracts by applications and, based on their
end date, allows us to decide whether to renew the contract or retire it to reduce spend.

![Expiring Contracts](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/expiring-contract.png)

**Application Optimization**:  
This shows the consolidated financial and usage
metrics for top 10 applications. This is used to understand total cost of ownership and target the
highest-value optimization actions.

![Application Optimization](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/application-optimization.png)

By drilling into each application, we can understand the various OpEx Spend drivers, as well as
the Labor, Vendor, and Contracts Detail for that application. This detailed information can be
viewed in the Application Cost Take-Out Detail Report.

![Application Drill](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/application-drill.png)
