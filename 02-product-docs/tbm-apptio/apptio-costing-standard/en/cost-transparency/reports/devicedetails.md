# Device Details

Use this report to view your spending for on-premises servers and physical storage
devices (network-attached storage [NAS] and storage area networks [SAN]). The report provides the
cost, cost YTD, and metrics related to space allocation and use.

This report has the following elements:

![](../../resources/images/ct_images/ct%20report%20collections/device%20details.png)

(1) KPIs: The KPIs in this report are the same as in the Storage Utilization Summary. See element
1 in the Utilization Summary report.

(2) Filters: The filters in this report are the same as in the Storage Operational Summary. See
element 2 in the **Storage - Spend Profile** report.

(3) Storage Device Additional Details:

- Use the options at the top of this chart to add columns for Location and Manufacturer to the
  chart.
- Use the table to view details about specific storage devices and quickly understand installed
  space versus addressable space versus allocated space.
- Select any link in the Supported Apps column to view information about the applications that
  use the storage device.

![](../../resources/images/ct_images/ct%20report%20collections/device%20details_1.png)

Questions answered:

- How is our Storage strategy for devices evolving?
- Where are we incurring Storage device costs?
