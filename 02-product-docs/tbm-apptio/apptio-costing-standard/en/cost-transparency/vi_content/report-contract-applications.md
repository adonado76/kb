# Contract to Applications

Applies to: Vendor Insights on TBM Studio 12.8
and later (v107)

Use the  **Contract to Applications**  report to analyze contract renewals within the context
of related applications.

This report is designed for:

- CIO and Senior IT leadership
- Application Owners
- Services Owners
- IT Finance Managers
- Vendor Managers

**Display the Contract to Applications report**

In the  Application  menu, select  Vendor Insights  .

1. Navigate to  Report Collections > Contracts  .
2. From the bar at the top of the page, select  Contract to Applications  .
3. Optionally, filter the report using the options at the top of the report.
4. To export or email your data, select  Export  ( ![image](../../../../../03-media/apptio-costing-standard/resources/images/vi_images/vi/export.jpg) ) at the top right of
   the page and select an export format.
5. To create an alert to notify you if a contract is expiring, select  Alert  ( ![image](../../../../../03-media/apptio-costing-standard/resources/images/vi_images/vi/alertsicon.jpg) ) on the top
   right of the page. To learn more, see  [Create alerts for expiring vendor
   contracts](alerts.html)  .

![image](../../../../../03-media/apptio-costing-standard/resources/images/vi_images/vi/vi-contracttoapps.jpg)

- Click any item in the  **Application**  column of a table in the  **Applications Contract
  Consolidation**  report component to view the details for that application.   
  ![image](../../../../../03-media/apptio-costing-standard/resources/images/vi_images/vi/contracts-applications-report_next1.jpg)
- Click any item in the  **Contract**  column of a table in the  **Applications Contract
  Consolidation**  report component to view the details for that contract.   
  ![image](../../../../../03-media/apptio-costing-standard/resources/images/vi_images/vi/contracts-applications-report_next2.jpg)

Questions answered

Use the information in the report to answer the following
questions:

- Which contracts are related to applications that are targeted for retirement?
- Which contracts are supporting business-critical applications?
- Where do application contracts overlap that could be consolidated?
