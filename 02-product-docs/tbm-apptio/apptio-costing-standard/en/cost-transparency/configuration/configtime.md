# Configure time for the project

The project time defines the start and end dates of a project, and the type of periods
that will be used. To configure the project time, click Time Settings on the Ribbon.**Set the
project time**

## About this task

In a time-enabled project, start and end dates are set for the data and the fiscal year is
defined. In a time-enabled project, you can enter data on a regular basis, allocate it to a specific
time period, and see trends across the duration of the project. You can view report data for the
current month or period, or for quarter, half, or full year time periods. The application supports
Gregorian, 445, 454, 544, and 13 period calendars.

## Procedure

1. Switch to Studio mode by clicking the Studio Mode icon at the right end of the Ribbon. The icon
   will turn orange and the Studio Ribbon will be displayed.
2. Click the Project tab in the Ribbon.
3. Click Time Settings. The Configure Project Time Settings dialog displays as shown below.

   ![](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/6839_2.png)
4. Select a Start of Project time period and an End of Project time period. Select dates that
   include historical data that you will be importing into the project. 

   NOTE: Once you set
   the Start of Project date, you cannot change it. However, you can change the End of Project
   date.
5. Configure any other project time settings appropriate to your project. For descriptions of the
   fields, see the Apptio Community.
6. Click Configure Time.
