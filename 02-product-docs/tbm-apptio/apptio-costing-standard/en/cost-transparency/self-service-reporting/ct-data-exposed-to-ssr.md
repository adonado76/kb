# Costing Standard data available in Apptio BI

Apptio BI will report against your Costing Standard production environment.

The following minimum Costing Standard version requirements apply:

- Reporting for Cost Source and Tower data: Costing Standard version 12.8.1 or later
- Reporting additional out-of-the-box data: Costing Standard version 12.8.3 or later
- Reporting for custom fields in out-of-the-box data, custom modeled data sets, custom modeled metrics: version 12.9.7
- Multi-currency support: Costing Standard version 12.9.1 or later
- Row level security support: version 12.9.1. or later
