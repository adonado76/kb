# AWS Deprecation of Billing Formats

AWS has indicated that they will be deprecating specific versions of billing reports including
the Detailed Billing Report with resources and tags as well as the Cost Allocation Report, both of
which are sources of billing data for Cloud Cost Management. AWS has not provided a specific date by
which they will end support for those billing formats, but has mentioned that when they do announce
a date, they will provide at least a 6-month grace period.

Apptio is currently working on an integration with the AWS Cost and Usage report (aka CUR) which
will be the go forward billing format for AWS. Apptiowill complete that integration well in advance
of AWS ending support for the other billing formats.
