# Vendor Insights - Coups Integration Getting Started

Coupa is a procure-to-pay software that contains data that can be used to deploy Vendor Insights . Follow the below steps to deploy Vendor Insights using Coupa data from the Coupa Datalink (Classic) Connector.

1. [Configure Coupa for Apptio](config-coupa.html)
2. [Configure the Coupa DataLink
   Connector](configure_the_datalink_coupa_connector.html)
3. [Configure Coupa data for Apptio](coupa-configure-data.html)
4. [Install and configure Vendor
   Insights using Coupa data](coupa-install.html)

**Subject matter experts needed**

The following subject matter experts are needed to perform the steps listed above.

- Coupa Administrator   
    
   This person will need access to log in to the Coupa administrative settings to
  register Datalink (Classic) as an approved application. During this setup process, the Coupa
  Administrator will need to collect several configuration settings and provide them to the
  TBMA to complete the Datalink (Classic) 's connector configuration.
- Technology Business Management Administrator (TBMA)   
    
   Configuring Vendor Insights requires a TBMA who can
  access TBM Studio to configure the Vendor
  Insights component. Also, the TBMA will need to be able to log in to the Datalink
  (Classic) administrator console which was registered by the Coupa Administrator as an
  authorized application in the Azure administrator console. There is guidance provided on TBM
  Connect to walk the TBMA through the configuration process.
