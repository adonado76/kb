# Summary of Cloud Configuration Steps

This document will provide a high-level summary of the steps required to light up cloud
use cases in Costing Standard.

For the more detailed, step-by-step configuration instructions please see:

[Costing Standard Cloud Services
Configuration Guide](../configuration-additional/ctcsconfigguide.html)

The configuration guide for cloud in CT Apps and Services is currently underway.

## Procedure

1. Install the CTF - Cloud component
2. Upload your cloud billing files
3. Create a transform of the billing files
4. Distribute Reserved Instance costs (AWS only, not applicable if you do not utilize Reserved
   Instances)
5. Append the billing file transforms to the Cloud Service Provider object
6. Fill any cloud service mapping gaps
7. If necessary, configure Cost source to Cloud Service Provider allocation.
