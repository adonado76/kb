## Hybrid IT TCO Impact for IBM Apptio Costing Standard – February 28, 2025

Summary

The Hybrid IT TCO Impact solution, now available with the release of R12.11.12, is a significant
addition to the IBM Apptio portfolio. This solution is designed to address the challenges of
managing Hybrid IT environments, where applications are constantly being moved between on-premise,
private cloud, and public cloud environments. In such dynamic environments, it is crucial to have a
clear understanding of the costs, priorities, and targets associated with a data-driven Hybrid IT
strategy. Without this visibility, decision-making can become speculative, and organizations may
struggle to optimize their IT investments.

The Hybrid IT TCO Impact solution provides organizations with the necessary tools to measure and
understand the financial impact of their evolving hybrid IT environments. By offering visibility
into unit economics, Total Cost of Ownership (TCO), and consumption of applications before and after
migrations, this solution empowers stakeholders, including C-Suite executives and Application
Owners, to make informed decisions. With Hybrid IT TCO Impact, organizations can ensure that their
migrations deliver the intended financial value, and optimize their IT investments to achieve better
business outcomes. By leveraging this solution, organizations can navigate the complexities of
Hybrid IT with confidence, and make data-driven decisions that drive business success.

Compare Hybrid IT Estate Against Targets

The Hybrid IT TCO Impact Summary report provides a comprehensive overview of an organization's
Hybrid IT estate, encompassing on-premise, private cloud, and public cloud environments, allowing
users to track key metrics such as Total Cost of Ownership (TCO), unit cost, and number of
applications for current and past Hybrid states. This report enables users to measure how their
Hybrid IT environment has changed over time, and compare their current state against predefined
targets, providing valuable insights to inform decision-making and optimize their Hybrid IT
footprint.

![](../resources/images/ct_images/cs-1.jpg)

Understand the Financial Impact of Application Migrations

As application migrations take place, users will be able to see the financial impact of each
migration type and migration strategy. Migration types include On-Premise to Public Cloud, Public
Cloud to Private Cloud, and any other direction and combination of these. Migration strategies could
include application rehosting, refactoring, rebuilding etc.

This allows users to quickly obtain insights into any “bad actors” such as applications with an
increased unit cost after migration. It further helps understand which migrations have delivered the
greatest financial benefits to date across TCO, Unit Cost and Usage Volumes.

![](../resources/images/ct_images/cs-2.jpg)

Unlock the Drivers Behind Changes to TCO

Analyze migration results at an individual application level, incorporating metadata such as
business criticality and migration date. This helps identify specific Cost pools and IT towers
driving financial benefits or drawbacks, informing data-driven decisions to optimize future
migrations.

## Public Cloud TCO for Costing Standard (v120) – February 28, 2025

The Public Cloud TCO release for Costing Standard (v120) and Costing Essentials (v200) offers
multiple benefits for primary users, mainly IT finance teams. This version provides a simple and
transparent financial lens across monthly public cloud costs, helping to avoid unexpected bill
shock. It also enables teams to drive accountability and ensure optimal efficiency of public cloud
services, minimizing waste.

The new release includes the installation of new components, such as Public Cloud TCO in v120
and Public Cloud TCO and Public Cloud TCO Reporting in v200. Users can connect to cloud data and
utilize features like Account Name, Business Unit, and Cloud Tower Mapping Table to tailor reports
and gain insights into their organization's cloud practices. With this release, users can assess if
their organization is applying "Best in Class" cloud practices, including RI Coverage and
Utilization Rates, to optimize their cloud services.
