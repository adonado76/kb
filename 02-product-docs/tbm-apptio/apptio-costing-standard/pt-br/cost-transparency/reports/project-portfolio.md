# Portfólio de projetos

## Gastos do projeto por categoria

![](../../resources/images/ct_images/ct%20report%20collections/project%20portfolio.jpg)

| Elementos-chave | Perguntas respondidas |
| --- | --- |
| - Analisar os KPIs relacionados aos projetos. - Revisar as despesas do projeto por categoria - Compreender a composição e o consumo do projeto | - Qual a porcentagem de projetos iniciados no prazo? Quantos projetos estão ativos? - Qual é o gasto mais alto do projeto com base na Iniciativa de Negócios? - Quantos recursos um projeto precisou para ser concluído? |

## Detalhamento da iniciativa de negócios do projeto

![](../../resources/images/ct_images/ct%20report%20collections/project%20portfolio_1.jpg)

| Elementos-chave | Perguntas respondidas |
| --- | --- |
| - Analisar todos os projetos relacionados à Iniciativa de Negócios - Compreender o estilo de vida do projeto para cada projeto - Compreender o status do projeto | - Quantos projetos fazem parte dessa iniciativa de negócios? - Em que fase estão os projetos? Temos projetos que estão prestes a ser concluídos? - Qual é o status dos projetos? Precisamos entender por que o cronograma está atrasado? |

## Consumo do projeto por aplicativo

| - Revisar o consumo e a composição do projeto - Entenda os aplicativos afetados pelos projetos | - Qual nome de projeto tem o maior gasto? - Quantos aplicativos são afetados por um projeto? |

| --- | --- |
| - Revisar os KPIs do projeto - Entenda o resumo do projeto, o gasto total do projeto por pool de custos e o desempenho do projeto. - Entenda as tendências de gastos com projetos. | - Qual é o gasto total do projeto? - Quais aplicativos são afetados por esse projeto? - Há alguma tendência nos gastos com OpEx vs. CapEx? |
