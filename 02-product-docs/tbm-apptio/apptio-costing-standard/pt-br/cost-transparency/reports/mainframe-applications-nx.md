# Aplicativos de mainframe

Utilize este relatório para compreender os custos das aplicações, avaliar como os recursos se alinham às prioridades de negócios e acompanhar as tendências de custos em todos os portfólios de produtos e ciclos de vida das aplicações. Aplique filtros e visualizações disponíveis para se concentrar nos dados relevantes para sua análise e apoiar decisões informadas sobre investimentos e orçamentos.

Este relatório foi elaborado para ser utilizado pelos seguintes perfis de usuários:

- Gerentes de portfólio de aplicativos
- Líderes de Unidades de Negócios
- Analistas financeiros de TI
- Responsáveis pelas aplicações

## Elementos-chave

![](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/nx-mf-apps.png)

| Elemento | Descrição |
| --- | --- |
| Controles de filtro (1) | Dois filtros permitem filtrar o relatório por família de aplicativos e objetivo de investimento do aplicativo. |
| Gráfico de despesas por portfólio de produtos (2) | Este gráfico de barras horizontais mostra os custos por portfólio de produtos, tais como cartões de crédito, engajamento do cliente, gestão de patrimônio, empréstimos e finanças e atuária. |
| Gráfico de gastos por objetivo de investimento (3) | Este gráfico de barras verticais mostra os custos por objetivo de investimento em aplicativos, como “Tolerar”, “Migrar” e “Investir”. |
| Gráfico de gastos por ciclo de vida da aplicação (4) | Este gráfico de barras horizontais mostra os custos por fase do ciclo de vida da aplicação, sendo “Em serviço” a categoria principal. |
| Painel de seleção de colunas (5) | Este painel permite mostrar ou ocultar colunas na tabela, tais como: Responsável comercial pela aplicação, Responsável de TI pela aplicação, Família da aplicação, Objetivo de investimento da aplicação, Ciclo de vida da aplicação, Estratégia de migração da aplicação e ID da aplicação. |
| Tabela de detalhes da aplicação de mainframe (6) | Esta tabela apresenta dados de aplicativos de mainframe com colunas como nome do aplicativo, unidade de medida comercial, volumes, custo unitário, família do aplicativo, responsável comercial pelo aplicativo, responsável de TI pelo aplicativo e custo. Os nomes dos aplicativos levam a informações mais detalhadas. |

## Perguntas respondidas

- Quais portfólios de produtos consomem a maior parte do orçamento destinado a aplicações de mainframe?
- Quanto você está gastando com aplicativos marcados para migração em comparação com aqueles nos quais planeja investir?
- Qual é o custo unitário de cada transação, e quais transações apresentam o maior custo por transação?
- Como os custos das aplicações são distribuídos entre os diferentes responsáveis pelas áreas de negócios e de TI?
- Quais aplicativos estão em operação em comparação com outras fases do ciclo de vida, e quais são os custos associados a eles?
- Como identificar os aplicativos que poderiam se beneficiar de uma modernização ou desativação?
- Qual é a relação entre os volumes de pedidos e os custos em todo o seu portfólio?

## Ações recomendadas

- Filtre por família de aplicações ou objetivo de investimento para se concentrar em áreas específicas da sua carteira.
- Classifique os aplicativos por custo ou custo unitário para identificar os aplicativos mais caros.
- Analise as solicitações com altos custos unitários para determinar se é possível otimizá-las.
- Compare os gastos entre diferentes portfólios de produtos para garantir um investimento equilibrado.
- Identifique os aplicativos marcados para migração e planeje iniciativas de modernização.
- Exporte os dados para compartilhá-los com as partes interessadas ou realizar análises adicionais.
- Agende reuniões periódicas com os responsáveis pelas aplicações para discutir as tendências de custos e estratégias de otimização.
