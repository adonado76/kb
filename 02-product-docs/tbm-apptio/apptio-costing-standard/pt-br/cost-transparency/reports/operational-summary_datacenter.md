# Resumo operacional

Use esse relatório para visualizar seus gastos operacionais mensais com recursos de data center, como capacidade de energia e custo por pé quadrado.

Esse relatório tem os seguintes elementos:

![](../../resources/images/ct_images/ct%20report%20collections/operational%20summary_1.png)

(1) KPIs: Os KPIs deste relatório são os mesmos do relatório Resumo. Consulte o elemento 1 no relatório resumido.

(2) Filtros: Os filtros desse relatório são os mesmos do relatório Resumo. Consulte o elemento 2 no relatório resumido.

(3) Métricas operacionais por categoria: Use as opções na parte superior desse gráfico para exibir seletivamente várias métricas. Métricas adicionais estão disponíveis na lista Resumido por. Use o gráfico para visualizar as despesas atuais do Data Center com base na métrica que você selecionar.

(4) Tabela: Use essa tabela para obter uma visão mais detalhada dos dados em Operational Metrics by Category (Métricas operacionais por categoria). A lista Summarized By (Resumido por) altera os dados exibidos nessa tabela. Selecione qualquer link na primeira coluna para ver os detalhes do pivô para esse item.

Perguntas respondidas:

- Quais dos meus data centers são mais caros e por quê?
- Há oportunidades de redução de custos?
- Qual aspecto dos gastos do data center é mais eficiente ou ineficiente?
