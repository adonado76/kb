# Etapa 13: Fechar o tronco de mesclagem

1. Selecione o ramo da nova versão do modelo (como Upgrade da versão 104).
2. Na **faixa de opções Projeto**, clique em **Fechar ramificação**.

   A caixa de diálogo **Confirmar fechamento** é aberta.
3. Clique em **OK** para fechar a ramificação.

   ![](../../../../../../03-media/apptio-costing-standard/resources/images/ct_images/upgrade-from-v3-29.png)
4. Confirme que Tronco não é mais exibido na barra de navegação.

   ![](../../../../../../03-media/apptio-costing-standard/resources/images/ct_images/upgrade-from-v3-30.png)

   Dica: Feche a ramificação de upgrade o mais rápido possível. A ramificação consome a mesma quantidade de recursos que o projeto principal do tronco. O fechamento da ramificação de upgrade liberará recursos e melhorará o desempenho geral.
