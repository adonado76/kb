# Etapa 2: Abra a nova filial

Atualize o projeto em uma ramificação separada ([Saiba mais](https://community.apptio.com/docs/DOC-5472 "(Abre em uma nova guia ou janela)") )

1. Na guia **Project (Projeto** ), clique no menu suspenso **Trunk (Tronco** ).
2. Selecione a ramificação que você deseja, por exemplo, "Upgrade da versão 104"

   ![](../../../../../../03-media/apptio-costing-standard/resources/images/ct_images/ct-open-trunk.png)

   Depois de selecionar uma ramificação, você verá a ramificação ativa na barra de menus.
3. Sempre que retornar ao site TBM Studio, verifique se você está no ramo de upgrade correto antes de continuar. Caso contrário, e **Trunk** for exibido, você precisará selecionar novamente a ramificação "Version 104 Upgrade".

Observação: Não faça alterações no projeto principal (por exemplo, Tronco) durante o curso das atividades de atualização na ramificação de atualização separada. Se você fizer isso, todas as alterações feitas no tronco principal serão perdidas após a mesclagem da ramificação de atualização.
