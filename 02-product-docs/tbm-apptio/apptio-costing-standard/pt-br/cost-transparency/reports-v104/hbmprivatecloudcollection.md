# Relatório Private Cloud VMWare - HBM

- Aplica-se a: Hybrid Business Management em TBM Studio TBM Studio e posterior

Use o relatório Private Cloud VMware para obter uma compreensão detalhada de sua infraestrutura de nuvem privada. Os relatórios permitem que você analise seus gastos com a nuvem privada, entenda melhor sua nuvem privada com as principais métricas e acompanhe a eficiência mês a mês. Use este relatório para entender a saúde financeira geral de sua implantação de nuvem privada, com KPIs perspicazes e análises de alto nível.

Esta coleção de relatórios foi projetada para as seguintes funções:

- Diretor de TI
- Chefe de Operações

Essa coleção de relatórios está alinhada às seguintes metas de negócios:

- Compreender a utilização dos recursos implementados e provisionados
- Otimize os gastos com a nuvem privada

## Exibir o relatório

1. Faça login em Apptio.
2. Na página **inicial**, clique em **Hybrid Business Management**.
3. No menu Coleção de relatórios, selecione **Nuvem privada**.

Esse relatório tem os seguintes elementos:

(1) KPIs
:   Os KPIs fornecem uma visão de alto nível de seus gastos com a nuvem privada:

    - **Custo acumulado** no ano - esse KPI mostra o gasto geral com a nuvem privada implementada no acumulado do ano.
    - **Núcleos de CPU implantados** - Este KPI mostra o número total de núcleos de CPU usados para VMware. O KPI secundário mostra **os núcleos de CPU não utilizados**.
    - **GB de RAM implantada** - este KPI mostra o total de RAM usado para VMware em gigabytes. O KPI secundário mostra o total de **RAM não utilizada**.
    - **GB de armazenamento implantado** - Esse KPI mostra o armazenamento total usado para VMware em gigabytes. O KPI secundário mostra **o armazenamento não utilizado**.

(2) Gráfico de custos
:   - Clique na guia **Summary (Resumo** ) para visualizar seus gastos mensais com CPU provisionada, economia e recomendação de custo dos serviços VMware. Use as caixas de seleção à direita para mostrar e ocultar dados no gráfico.
    - Clique na guia **Details (Detalhes** ) para visualizar a tabela **Savings Analysis (Análise de economia)**, que oferece uma visão mais detalhada dos dados usados para gerar o gráfico de custos. Use os filtros à direita para filtrar a tabela Análise de economia por Host Reservation e Group Name. Observe que esses filtros não afetam os dados exibidos no gráfico de custo.

(3) Gráfico de capacidade
:   Este gráfico mostra sua capacidade mensal para VMware CPU, RAM e armazenamento. Use as caixas de seleção à direita para mostrar e ocultar dados no gráfico.

Perguntas respondidas:

- Qual é o custo da minha nuvem privada e da capacidade não utilizada?
- Quanta capacidade não utilizada pode ser recuperada para atender às solicitações de recursos recebidas das equipes de aplicativos?
- Qual é a eficiência da minha implementação de nuvem privada?
- Quantos recursos as equipes de aplicativos estão usando?
