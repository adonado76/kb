# Página inicial do cálculo de custos padrão

- Aplica-se a: Costing Standard em TBM Studio 12.3.3 e posterior, com modelo v104 e posterior

Use a página inicial Costing Standard para acessar as coleções de relatórios no aplicativo Costing Standard . Os relatórios são organizados por áreas de interesse. Clique em um título para abrir o relatório resumido de alto nível de uma coleção de relatórios.

![](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/ct_home_page.png)

## Ícone da página inicial

Clique no ícone Home ![](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/home_17x15.png) no canto superior direito para retornar à página Home de qualquer lugar do aplicativo.

## Acesso

O acesso ao site Costing Standard é controlado por funções. Você pode acessar o aplicativo se tiver sido atribuído a qualquer uma das seguintes funções padrão, exceto Gerente de Projeto e Proprietário da Unidade de Negócios. Se não conseguir acessar o aplicativo, entre em contato com o administrador do site Apptio.

- Admin
- Analista
- Administração da adaptação
- Proprietário do processo orçamentário
- Proprietário do Negócio
- Usuário corporativo
- Datalink (Classic) Administrador
- Usuário executivo
- Financeiro
- Usuário avançado
- Gerenciador de Serviços
- Somente visualizar

As seguintes funções não podem acessar Costing Standard:

- Proprietário da unidade de negócios
- Gestor de Projeto
