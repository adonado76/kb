# Contrato para Aplicações

Aplica-se a: Vendor Insights no TBM Studio 12.8 e posterior ( v107 )

Use o relatório **Contrato para Aplicações** para analisar renovações de contrato no contexto de aplicações relacionadas.

Este relatório foi elaborado para:

- CIO e liderança sênior de TI
- Proprietários de aplicativos
- Proprietários de serviços
- Gerentes de Finanças de TI
- Gerentes de fornecedores

**Exibir o relatório Contrato para aplicativos**

No menu Aplicativo, selecione Vendor Insights .

1. Navegue até Coleções de relatórios > Contratos.
2. Na barra na parte superior da página, selecione Contrato para Aplicativos.
3. Opcionalmente, filtre o relatório usando as opções na parte superior do relatório.
4. Para exportar ou enviar seus dados por e-mail, selecione Exportar ( ![imagem](../../../../../03-media/apptio-costing-standard/resources/images/vi_images/vi/export.jpg) ) no canto superior direito da página e selecione um formato de exportação.
5. Para criar um alerta para notificá-lo quando um contrato estiver expirando, selecione Alerta ( ![imagem](../../../../../03-media/apptio-costing-standard/resources/images/vi_images/vi/alertsicon.jpg) ) no canto superior direito da página. Para saber mais, consulte [Criar alertas para contratos de fornecedores que estão prestes a expirar](alerts.html).

![imagem](../../../../../03-media/apptio-costing-standard/resources/images/vi_images/vi/vi-contracttoapps.jpg)

- Clique em qualquer item na coluna **Aplicativo** de uma tabela no componente do relatório **Consolidação de contratos de aplicativos** para visualizar os detalhes desse aplicativo.   
   ![imagem](../../../../../03-media/apptio-costing-standard/resources/images/vi_images/vi/contracts-applications-report_next1.jpg)
- Clique em qualquer item na coluna **Contrato** de uma tabela no componente do relatório **Consolidação de contratos de aplicativos** para visualizar os detalhes desse contrato.   
   ![imagem](../../../../../03-media/apptio-costing-standard/resources/images/vi_images/vi/contracts-applications-report_next2.jpg)

Perguntas respondidas

Use as informações do relatório para responder às seguintes perguntas:

- Quais contratos estão relacionados a aplicativos que serão descontinuados?
- Quais contratos estão a apoiar aplicações críticas para o negócio?
- Onde os contratos de aplicação se sobrepõem e poderiam ser consolidados?
