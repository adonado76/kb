# Conceitos de nuvem

Os tópicos a seguir fornecem informações sobre recursos relacionados à nuvem:

Instâncias reservadas
:   - [AWS Instâncias reservadas - uma visão Apptio](awsreserved.html "Para configurar o aplicativo para Amazon Web Services, você deve instalar os componentes CTF-Cloud Service e CTF- Amazon Web Services.")
    - [Azure Instâncias reservadas - alocações de ingestão e custo](azurereserved.html "Microsoft Azure lançou as instâncias reservadas (RIs) no final de 2017. Os custos associados a esses RIs podem ser consideráveis e precisam ser levados em conta ao considerar seus gastos com o site Azure.")
    - [Custos de instância reservados - Alocações dentro de Apptio](reservedcosts.html "As instâncias reservadas (RIs) incluem opções de preços pelas quais todos ou alguns dos custos são pagos antecipadamente no momento da compra. Para essas opções (All Upfront ou Partial Upfront), o site Apptio distribuirá o custo inicial uniformemente a cada mês durante o prazo de compra do RI. Esse custo mensal efetivo será adicionado a quaisquer taxas recorrentes associadas ao RI (0 no caso do All Upfront) para formar o custo mensal acumulado efetivo.")

Provedor de nuvem ATUM arquivos de mapeamento
:   - [AWS Mapeamento de serviços](awsservices.html "Apptio os conectores CBM e AWS DataLink da TBM permitem mapear os itens de linha de faturamento dos itens de linha de faturamento do provedor de nuvem para os atributos do site ATUM, incluindo torres e subtorres de TI, a hierarquia de serviços da TBM, bem como uma unidade de medida normalizada.")
    - [Azure Mapeamento de serviços.](azuremap.html "Apptio os conectores CBM e AWS DataLink da TBM permitem mapear os itens de linha de faturamento dos itens de linha de faturamento do provedor de nuvem para os atributos do site ATUM, incluindo torres e subtorres de TI, a hierarquia de serviços da TBM, bem como uma unidade de medida normalizada.")
