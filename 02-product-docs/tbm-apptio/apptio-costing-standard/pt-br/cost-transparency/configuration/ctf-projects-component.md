# CTF - Componente de projetos: instalar e configurar

Use o componente CTF - Projetos para importar os custos do projeto. O componente CTF-Projects não é instalado automaticamente com o projeto padrão Costing Standard . O componente Projetos fornece insights sobre os gastos com projetos por tipo, centro de custos, projetos e torres de TI. Aplica-se a: Costing Standard em TBM Studio 12.4 e posterior

## Usa

Use os relatórios do componente CTF-Projects para fazer o seguinte:

- Gerenciar despesas de acordo com o orçamento
- Identificar alavancas de gastos
- Alinhar as despesas com os objetivos comerciais
- Alinhar as despesas à estratégia
- Confirmar o valor ou cancelar projetos

## Instalar o componente

O componente CTF - Projects não é instalado com o projeto padrão Costing Standard .

Para instalar o componente CTF - Projetos:

1. Abra o projeto Costing Standard .
2. Clique na guia **Projeto**.
3. Clique em **Components (Componentes** ) na faixa de opções.
4. Clique no componente **CTF - Projetos**.
5. Clique em **Install (Instalar** ).

## Carregar dados do projeto

Se quiser rastrear os custos do projeto no aplicativo Costing Standard , você deve carregar os dados do projeto e anexar e mapear os dados na tabela Projects Master Data (Dados mestre de projetos). Você pode usar os relatórios Projects para revisar os gastos de TI por tipo de investimento e visualizar os gastos mensais do projeto em OpEx e CapEx YTD.

## Fontes de dados típicas

Os números do projeto geralmente são extraídos de aplicativos de gerenciamento de projetos.

## Uploads

Há uma tabela de projeto: Dados mestre de projetos. Para preencher a tabela, carregue uma tabela de dados do projeto e, em seguida, anexe-a e mapeie-a para a tabela de dados mestre de projetos.
