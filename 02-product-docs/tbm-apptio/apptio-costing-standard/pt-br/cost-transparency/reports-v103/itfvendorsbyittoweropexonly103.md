# IT Finance - Vendors - By IT Tower report ( OpEx Only) ( v103 )

Aplica-se a: Costing Standard 11.8.x em execução em [TBM Studio v12](https://community.apptio.com/community/apptio/product-central/tbm-studio/studio-v12 "(Abre em uma nova guia ou janela)") ou [TBM Studio v11](https://community.apptio.com/community/apptio/product-central/tbm-studio/studio-v11 "(Abre em uma nova guia ou janela)").

## Introdução

Identificar qual torre de TI tem o maior gasto com fornecedores.

![](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/itfvendorsbyittoweropexonly103_1.png)

## Navegação

Finanças de TI > Fornecedores > Por torre de TI (somente OpEx )

## Funções

Este relatório foi elaborado para:

- Finanças de TI
- Gerente de fornecedores
- Administração da TI

## Objetivos

Use este relatório para:

- Identifique qual torre de TI tem o maior gasto com fornecedores usando o gráfico.
- Identifique os fornecedores associados a cada torre de TI usando a tabela.

## Perguntas respondidas

As informações apresentadas neste relatório podem ser usadas para responder às seguintes perguntas:

- Qual é o volume de gastos com fornecedores gerados por cada torre de TI?
- Quais fornecedores se enquadram em qual torre de TI?
- Há vários fornecedores que fornecem produtos e serviços semelhantes?
- Faz sentido que esse gasto do fornecedor esteja nessa torre de TI?
