# IT Finance - Relatório de variação de contas ( v103 )

Aplica-se a: Costing Standard 11.8.x em execução em TBM Studio v12 ou TBM Studio v11.

## Introdução

Use este relatório para examinar todas as contas por categoria de custo e ver para onde está indo o total de gastos.

![imagem](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/itfaccountvariance103_1.png)

## Navegação

IT Finance > Variação de contas

## Funções

Este relatório foi elaborado para a equipe de finanças de TI.

## Objetivos

Use este relatório para:

- Examine todas as contas por categoria de custo para ver para onde está indo o total de gastos.
- Pesquise/divida por grupos de contas para localizar e visualizar despesas de categorias específicas.
- Inclua outros filtros de hierarquia de contas para organizar e visualizar informações.

## Perguntas respondidas

Você pode usar as informações apresentadas neste relatório para responder às seguintes perguntas:

- Onde temos o maior gasto por categoria?
- Estamos gastando mais nessa categoria de custo do que esperávamos?
- Há oportunidades para controlar melhor ou reduzir as despesas em uma área?

## Próximas ações

- Clique em View (Exibir) na coluna Variance Trend (Tendência da variação) para visualizar a variação mensal YTD.
- Clique em uma conta na coluna Account Description (Descrição da conta) para visualizar as transações detalhadas.
