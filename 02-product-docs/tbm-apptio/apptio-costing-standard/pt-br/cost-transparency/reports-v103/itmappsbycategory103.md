# Gerenciamento de TI - Aplicativos - Relatório por categoria ( v103 )

Use este relatório para analisar os principais gastos com aplicativos por objetivo de investimento em aplicativos, ciclo de vida do aplicativo, importância comercial, tipo de aplicativo, categoria de usuário do aplicativo, família de aplicativos ou função do aplicativo.

Aplica-se a: Costing Standard 11.8.x em execução em TBM Studio v12 ou TBM Studio v11.

![](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/itmappsbycategory103_1.png)

## Navegação

Gerenciamento de TI > Aplicativos > Por categoria

## Funções

Este relatório foi elaborado para:

- Proprietários de aplicativos
- Proprietários do portfólio de aplicativos / VP de desenvolvimento e suporte de aplicativos
- Arquitetos corporativos

## Objetivos

Use este relatório para:

- Use a lista suspensa para analisar os principais gastos com aplicativos por objetivo de investimento em aplicativos, ciclo de vida do aplicativo, importância comercial, tipo de aplicativo, categoria de usuário do aplicativo, família de aplicativos ou função do aplicativo.
- Analise a tendência de gastos com aplicativos pela categoria selecionada.

## Perguntas respondidas

As informações apresentadas neste relatório podem ser usadas para responder às seguintes perguntas:

- Essa é a combinação correta de gastos com todas as categorias de investimento?
- Nossos investimentos em aplicativos são suficientes para apoiar as metas corporativas?
- É necessário tomar medidas para reduzir o risco?
