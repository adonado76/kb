# Finanças de TI - Fornecedores - Por projeto ( v103 )

Aplica-se a: Costing Standard 11.8.x em execução em [TBM Studio v12](https://community.apptio.com/community/apptio/product-central/tbm-studio/studio-v12 "(Abre em uma nova guia ou janela)") ou [TBM Studio v11](https://community.apptio.com/community/apptio/product-central/tbm-studio/studio-v11 "(Abre em uma nova guia ou janela)").

## Introdução

Use esse relatório para identificar os projetos que estão gerando os maiores gastos com fornecedores.

![](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/itfvendorsbyproject103_1.png)

## Navegação

Finanças de TI > Fornecedores > Por projeto

## Funções

Este relatório foi elaborado para:

- Finanças de TI
- Gerente de fornecedores
- Administração da TI

## Objetivos

Use este relatório para:

- Identifique os projetos que estão gerando mais gastos com fornecedores usando o gráfico.
- Identifique os fornecedores que estão apoiando projetos específicos usando a tabela.

## Perguntas respondidas

As informações apresentadas neste relatório podem ser usadas para responder às seguintes perguntas:

- Qual projeto está gerando os maiores gastos com fornecedores?
- Quem é o gerente de projeto que está supervisionando os gastos do projeto com fornecedores?
- Quanto das despesas do fornecedor é capitalizado?
- Faz sentido que o fornecedor esteja envolvido no projeto?

## Próximas ações

- Visualize a tendência de 13 meses por fornecedor clicando em Visualizar na coluna Tendência.
- Analise os fornecedores por projeto ou torre de TI selecionando a guia By IT Tower (somente OpEx ).
