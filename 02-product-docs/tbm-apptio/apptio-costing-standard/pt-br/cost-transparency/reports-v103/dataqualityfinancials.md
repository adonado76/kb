# Qualidade dos dados - Relatório financeiro

Aplica-se a: Costing Standard 11.8.x em execução em TBM Studio v12 ou TBM Studio v11.

## Introdução

O relatório Financials se concentra nos custos não alocados.

![](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/dataqualityfinancials_1.png)

## Navegação

Qualidade de dados > Finanças

## Funções

Este relatório foi desenvolvido para administradores de TBM.

## Objetivos

Use este relatório para analisar as alocações para a camada financeira do modelo de custo mostrado abaixo. O relatório concentra-se no conjunto de dados da Fonte de Custos e nos registros específicos que não estão fluindo para o modelo.

![](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/dataqualityfinancials_2.png)

Os objetivos específicos incluem o seguinte:

- Veja rapidamente os custos não alocados da camada financeira.
- Avalie as tendências de custos não alocados ao longo do tempo por objetos do modelo financeiro de TI.
- Identificar custos não alocados por objetos do modelo financeiro de TI.
- Revisar as estratégias de alocação para os objetos financeiros de TI no modelo (por exemplo, fonte de custos, mão de obra, ativos fixos etc.).

## Perguntas respondidas

As informações apresentadas neste relatório podem ser usadas para responder às seguintes perguntas:

- Quais registros específicos não estão sendo alocados a um dos objetos de destino (mão de obra, ativos fixos, etc.)?
- Posso ver algum padrão que possa identificar o erro?
- Todos os campos importantes que orientam minhas alocações estão completos e são válidos?

## Próximas ações

- Exporte registros específicos para revisão e correção.
- Ajustar ou corrigir as regras de alocação no Studio.
- Corrigir dados nos dados de origem ou por meio de transformações de dados em Data Studio.
