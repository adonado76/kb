# Coleções de relatórios de cálculo de custos padrão ( v103 )

Aplica-se a : [Costing Standard](https://community.apptio.com/community/apptio/product-central/cost-transparency "(Abre em uma nova guia ou janela)") em execução em TBM Studio R12+ com o modelo v103.

Observação: Este é o artigo principal de todas as coleções de relatórios Costing Standard para o modelo v103. Para determinar qual componente você está usando, consulte [os relatórios Compare v104 e v103 Costing Standard](../reports-v104/comparev103v104reports.html) .

## Navegação

1. Faça login no Apptio.
2. Na página **Ambientes**, clique em **Transparência de custos**.
3. Na página **inicial**, clique em uma das seguintes coleções de relatórios ou clique em um link para ir diretamente para um relatório específico.

A seguir, links para listas específicas de coleta de relatórios:

- [Relatórios resumidos de benchmarking ( v103 )](../reports-v104/benchmarkingsummaryreports103_psdeprecate.html)
- [Relatórios financeiros de TI](itfreports103.html)
- [Relatórios de gestão de TI](itmreports103.html)
- [Relatórios de qualidade de dados](dataqualityreports.html)
- [Relatórios de infraestrutura e operações de TI](itinfraopsreports.html)
