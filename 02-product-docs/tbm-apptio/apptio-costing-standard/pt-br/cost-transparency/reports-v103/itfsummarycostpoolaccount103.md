# IT Finance - Resumo - Pool de custos - Relatório de contas ( v103 )

Aplica-se a: Costing Standard 11.8.x em execução em TBM Studio v12 ou TBM Studio v11.

## Introdução

Use esse relatório para identificar e verificar as contas mapeadas para um pool de custos específico.

![](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/itfsummarycostpoolaccount103_1.png)

## Navegação

IT Finance > Summary > Clique em um pool de custos na tabela

## Funções

Este relatório foi desenvolvido para a equipe financeira de TI.

## Objetivos

Use este relatório para:

- Identificar e verificar as contas mapeadas para um pool de custos específico (por exemplo, categoria de custos).
- Visualizar as despesas e o desvio para o período atual e o acumulado no ano.

## Perguntas respondidas

Use as informações apresentadas neste relatório para responder às seguintes perguntas:

- Todas essas contas pertencem a essa categoria de pool de custos?
- Se uma conta não pertencer à empresa, para onde ela deve ser movida?
- Por conta, onde está meu maior gasto e variação, por valor e por porcentagem?
- São necessárias medidas para reduzir o risco orçamentário?

## Próximas ações

- Se a variação for irrelevante, nenhuma preocupação ou ação será necessária.
- Para ver os detalhes da transação de uma conta, clique em uma conta.
