# Sobre os relatórios de gerenciamento de TI ( v103 )

Aplica-se a: Costing Standard 11.8.x em execução em TBM Studio v12 ou TBM Studio v11.

## Introdução

Os relatórios de gerenciamento de TI se concentram no orçamento, na previsão e nos números de gastos por torre de TI. Os relatórios fornecem dados sobre:

- O orçamento para o ano corrente.
- O gasto até o momento.
- Variações orçamentárias por ITTower.
- As principais despesas e a variação por ITTower.

Para acessar os relatórios do IT Management, clique em IT Management na página inicial. Use as guias na página do relatório de Gerenciamento de TI para acessar dados específicos:

- Finanças
- Projetos
- Torres de TI
- Aplicativos
- Unidades de negócios
- Public Cloud
- Serviços
