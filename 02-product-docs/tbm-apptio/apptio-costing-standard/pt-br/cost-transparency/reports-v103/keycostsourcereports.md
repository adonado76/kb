# Principais relatórios de fontes de custos

A tabela abaixo identifica os principais relatórios disponíveis após a configuração do componente Fonte de custos. Use a tabela ao analisar os relatórios para identificar os objetivos dos relatórios, bem como as questões abordadas por eles.

| **Reportar** | **Público** | **Objetivos** | **Perguntas abordadas** |
| --- | --- | --- | --- |
| Costing Standard - Resumo | - CIOs - Administração da TI | - Entenda onde os gastos de TI estão ocorrendo e identifique rapidamente as exceções ou os desvios não planejados, analisando os gráficos. - Compare as despesas anuais e acumuladas com o planejado por torre, pool de custos, mão de obra interna vs. externa e principais projetos, revisando as tabelas. - Acesse um resumo de uma torre, pool de custos, tipo de mão de obra ou projeto específico clicando em um item de uma das tabelas. | - Onde estou gastando a maior parte do meu dinheiro? É razoável? - Onde minhas despesas não estão no caminho certo para cumprir o plano? - Tenho áreas abaixo do orçamento que equilibram esse gasto excessivo? |
| Costing Standard Resumo - Finanças | - CIOs - Administração da TI | - Visualizar a tendência de variação orçamentária nos últimos 13 meses por Pool de custos. - Identificar as principais despesas e variações por centro de custos para entender quem é o responsável. | - Quanto gastei em pool de custos até agora neste ano? - Estou no caminho certo para atingir o orçamento para essa categoria de custo? - Qual foi a tendência da variação nos últimos 13 meses? - Há picos ou desvios incomuns a serem analisados? |
| Finanças de TI - Resumo | - Equipe de finanças de TI | - Revisar os KPIs de gastos e orçamento. - Entenda o gasto acumulado até o momento e o gasto anual projetado. - Identificar a variação orçamentária por pool de custos. - Identifique as principais despesas e variações por pool de custos (por exemplo, g.: categoria de custos) usando o gráfico Custo por pool de custos e a tabela Custo para orçamento por pool de custos. | Use as informações apresentadas na parte superior do relatório para responder às seguintes perguntas:   - Como minhas despesas estão sendo comparadas com o orçamento deste ano? - A previsão é de que o orçamento do ano fique acima ou abaixo do previsto? - Quais pools de custos estão acima ou abaixo do orçamento?   Use as informações apresentadas na parte inferior do relatório para responder às seguintes perguntas:   - Quanto gastei por pool de custos este ano? - Onde está meu maior gasto? - Onde está minha maior variação - por valor e por porcentagem? - Que quantidade de variação ou porcentagem de variação é material em comparação com meus gastos gerais com TI? - São necessárias medidas para reduzir o risco orçamentário? |
| Finanças de TI - Pool de custos | - Equipe de finanças de TI | - Visualize as despesas fixas /var iable e OpEx/CapEx por pool de custos. | Você pode usar as informações apresentadas neste relatório para responder às seguintes perguntas:   - Se a demanda ou o consumo cair, tenho flexibilidade para reduzir os gastos? - Onde eu poderia considerar mudanças no sourcing que criariam uma estrutura de custos mais variável? - Que agilidade financeira é apropriada para nossa empresa neste momento? Devo definir uma meta variável percentual? - Como a capitalização varia de acordo com o pool de custos? - Onde está capitalizada a maior parte de meus gastos? |
| Dimensões de dados - Fonte de custos | - Administrador da TBM | - Avaliar a integridade dos dados. A integridade é medida como o número de células nos conjuntos de dados de suporte que têm valores. Por exemplo, uma pontuação de completude de 80% indica que 80% das células têm valores. Em geral, uma pontuação de 80% ou mais é aceitável. - Avaliar a validade dos dados. A validade é determinada pela comparação dos valores nos dados com uma lista de valores aceitáveis que você carregou no aplicativo. Em geral, uma pontuação de 80% ou mais é aceitável. - Avalie a exclusividade dos dados. Em geral, uma pontuação de 80% ou mais é aceitável. | - Meus dados estão completos o suficiente para tomar decisões informadas? - Os dados estão corretos? |
