# Gerenciamento de TI - Aplicativos - Relatório por fornecedor ( v103 )

Utilize este relatório para identificar as despesas por fornecedor, família de aplicativos e aplicativo.

Aplica-se a: Costing Standard 11.8.x em execução em TBM Studio v12 ou TBM Studio v11.

![](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/itmappsbyvendor103.png)

## Navegação

Gerenciamento de TI > Aplicativos > Por fornecedor

## Funções

Este relatório foi elaborado para:

- Proprietários de aplicativos
- Proprietários do portfólio de aplicativos / VP de desenvolvimento e suporte de aplicativos
- Arquitetos corporativos

## Objetivos

Utilize este relatório para identificar as despesas por fornecedor, família de aplicativos e aplicativo.

## Perguntas respondidas

As informações apresentadas neste relatório podem ser usadas para responder às seguintes perguntas:

- Os gastos com cada fornecedor estão de acordo com o orçamento?
- A combinação dos sites OpEx e CapEx é adequada para dar suporte às operações atuais e aos planos futuros?
- São necessárias ações para mitigar o risco?
