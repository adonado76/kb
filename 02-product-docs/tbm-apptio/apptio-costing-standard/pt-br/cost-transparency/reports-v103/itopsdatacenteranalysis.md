# Operações de TI - Relatório de análise de data centers

◆ Aplica-se a: Costing Standard 11.8.x em execução em TBM Studio v12 ou TBM Studio v11.

## Introdução

Use esse relatório para analisar os custos e a eficiência dos data centers.

![](../../../../../03-media/apptio-costing-standard/resources/images/ct_images/itopsdatacenteranalysis_1.png)

## Navegação

Infraestrutura e operações de TI > Análise de data centers

## Funções

Este relatório foi elaborado para:

- Líderes/gerentes de data center

## Objetivos

Use este relatório para:

- Analise os custos mensais e acumulados para os data centers.
- Analisar os custos por RU (unidade de rack), pé quadrado e taxa de unidade de energia nos data centers.
- Analise a variação orçamentária de cada data center.
- Analise os custos por região, local, finalidade, nível de certificação e nível.

## Perguntas respondidas

As informações apresentadas neste relatório podem ser usadas para responder às seguintes perguntas:

- Quais data centers são mais econômicos com base em RU, metragem quadrada e taxa de unidade de energia?
- Como o custo varia ao longo do ano civil?
- Há data centers que estão com desempenho abaixo do esperado?
