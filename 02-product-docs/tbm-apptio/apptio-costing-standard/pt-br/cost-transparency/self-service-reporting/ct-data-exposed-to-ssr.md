# Dados de cálculo de custos padrão disponíveis em Apptio BI

Apptio BI será reportado em seu ambiente de produção Costing Standard .

Aplicam-se os seguintes requisitos mínimos da versão Costing Standard :

- Relatórios para dados da fonte de custos e da torre: Costing Standard versão 12.8.1 ou posterior
- Relatório de dados adicionais prontos para uso: Costing Standard versão 12.8.3 ou posterior
- Relatórios para campos personalizados em dados prontos para uso, conjuntos de dados modelados personalizados, métricas modeladas personalizadas: versão 12.9.7
- Suporte a várias moedas: Costing Standard versão 12.9.1 ou posterior
- Suporte de segurança em nível de linha: versão 12.9.1. ou posterior
