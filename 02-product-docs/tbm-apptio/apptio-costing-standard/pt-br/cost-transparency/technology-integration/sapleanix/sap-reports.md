# SAP LeanIX Relatórios

Embora não existam relatórios específicos SAP LeanIX em IBM Apptio Costing atualmente, a integração aprimora significativamente os relatórios e recursos existentes IBM Apptio. Os usuários podem aproveitar os recursos de relatórios de custos e modelagem de custos do IBMApptio para produzir:

- **Relatórios de TCO no nível do aplicativo**

  Custo de execução, custo de desenvolvimento, manutenção, custo total ao longo do tempo; utilizando dados do SAP LeanIX como fonte para aplicações, dependências, componentes, grupos de usuários e capacidades de negócios. Adicionando visualizações adicionais aos relatórios prontos para uso, e. g.: Custo da aplicação YearToDate por adequação técnica, adequação funcional e conformidade.
- **Relatórios de TCO da unidade de negócios**

  Usar SAP LeanIX relacionamentos (aplicativos, recursos comerciais, grupos de usuários, provedores) para alocar custos adequadamente.
- **SAP LeanIX relatório de exportação**

  IBM Apptio O Costing possui um componente de integração que instala um relatório predefinido que a API SAP LeanIX usa para extrair métricas de TCO do aplicativo diretamente do IBM Apptio Costing para o SAP LeanIX,, a fim de aprimorar os relatórios e a visibilidade dos custos no SAP LeanIX.
