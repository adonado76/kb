# Product components at a glance

To set up the rest of the documentation, it helps to anchor the main components that later sections will describe.

- Interactive Benchmarking Full benchmark dataset No Costing project required to start exploring Lets you self-select peer groups by attributes such as industry, region, revenue size, and scale Shows ranges, quartiles, minimum, maximum, and archetypes for each metric Also acts as the source for benchmark profiles that can be loaded into Costing reports

- Benchmark-enabled Costing reports Consume selected benchmark profiles from Interactive Benchmarking Combine them with your modeled actuals in Costing Provide side-by-side actual vs. benchmark views with drill-through to cost sources and general ledger detail
