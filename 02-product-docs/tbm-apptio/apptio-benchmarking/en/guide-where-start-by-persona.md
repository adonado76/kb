# Where to start by Persona

You do not need the whole thing on day one! Start here based on your role.

TBMAs / ITFMAs / Admins

Start with:

- Overview & Key Concepts
- Quickstart

Use as working references:

- Implementation Checklist
- Quickstart
- Data & Methodology Reference
- Troubleshooting & FAQ

For program and stakeholder context:

- Operations & Governance

Executive or Sponsor (CIO, CTO, CFO, VP-level)

Start with:

- Overview & Key Concepts
- Using Benchmarking Reports

For how your Resource Tower will be measured and how targets are set:

- Operations & Governance

For detailed questions on metric definitions:

- Data & Methodology Reference

Resource Tower Owner / Domain Lead (Infra, End User, Network, etc)

Start with:

- Overview & Key Concepts
- Using Benchmarking Reports

For how your Resource Tower will be measured and how targets are set:

- Operations & Governance

For detailed questions on metric definitions:

- Data & Methodology Reference

When something looks wrong

Start with:

- Troubleshooting & FAQ

Then, depending on the issue:

- Configuration Guide
- Data & Methodology Reference
