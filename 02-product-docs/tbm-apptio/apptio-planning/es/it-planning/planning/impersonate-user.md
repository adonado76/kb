# Suplantación de usuarios

Nota: *Se requieren roles de administrador o responsable del proceso presupuestario para suplantar la identidad.*

La suplantación de identidad de usuario permite a los administradores actuar temporalmente como otro usuario para:

- Solucionar problemas de permisos o acceso
- Validar lo que un usuario final puede ver o editar
- Confirmar cambios en la configuración
- Apoyar a los usuarios durante la incorporación o la formación

Los administradores pueden suplantar a cualquier usuario con acceso a Apptio Planning.

**Suplantación de identidad de un usuario**

1. En la Apptio barra de herramientas, seleccione **Configuración del perfil** → **Suplantar identidad**.
2. Introduzca el **nombre de usuario** del usuario que desea suplantar.
3. Seleccione **Siguiente**.
4. Ahora estás suplantando al usuario seleccionado.

Selecciona **Configuración (icono de engranaje) → Acerca de la aplicación** para confirmar que has iniciado sesión como el usuario suplantado. El nombre de usuario que se muestra coincidirá con la cuenta suplantada e incluirá un asterisco (\*) para indicar el modo de suplantación.

**Para dejar de suplantar la identidad de un usuario:**

- En la Apptio barra de herramientas, seleccione **Configuración del perfil** → **Cerrar sesión**.
- Vuelve a iniciar sesión con tus propias credenciales.
