# Solución de problemas: Los KPI o valores de un plan muestran el error #REF

## Síntoma

Los KPI o valores de un plan muestran el error #REF.

## ¿Qué está pasando?

El error #REF suele producirse cuando faltan tipos de conversión de moneda.

[Más información sobre la importación y publicación de tipos de cambio](import-publish-currency.html "Las siguientes tareas sólo pueden ser realizadas por usuarios asignados a los roles de Administrador o Propietario de Proceso Presupuestario. Para obtener más información sobre las funciones, consulte Permisos y funciones de Frontdoor.")

## Solución

Asegúrese de que las tablas de tipos de cambio tienen una conversión de moneda válida para todos los plazos afectados.
