# Actualizar tablas de partidas

## Antes de empezar

La mayoría de las tablas con partidas en las aplicaciones Apptio Planning ofrecen las siguientes posibilidades:

- <export-populate-line.html>
- <enter-import-line.html>
- <filter-line-item.html>
- <customize-save-share.html>

## Procedimiento

- **Añadir y eliminar filas de una tabla**

  Puede añadir y eliminar filas de las tablas haciendo clic con el botón derecho en la fila y seleccionando una de las opciones disponibles:

  - Insertar fila arriba
  - Insertar fila abajo
  - Suprimir fila
  - Fila duplicada
- **Atajos de comandos de tabla**

  Los siguientes atajos de teclado están disponibles para navegar por la mayoría de las tablas de partidas:

  - **TAB** - Desplazarse al foco de la página.
  - **MAYÚS + ESPACIO** - Abrir el menú de la celda.

    Consejo : **SUGERENCIA** : Utilice este comando para abrir las distintas acciones de la página Estado. Para seleccionar la acción que prefiera, pulse **Intro**.
  - **SHIFT + F10** - Abrir el menú de columnas.
  - **MAYÚS + TAB** - Si estás en la primera celda de una tabla, sal de la tabla. De lo contrario, pase a la opción anterior del menú.
  - **TAB** - Si se encuentra en la última celda de una tabla, sale de ella. De lo contrario, pase a la siguiente opción del menú.
  - **ENTER** - Expandir o contraer una fila o hacer clic en una celda.

  Consejo: consulte también Apptio keyboard accessibility support.
