# Configure Apptio Aplicaciones de planificación

Las siguientes tareas sólo pueden ser realizadas por usuarios asignados a los roles de Administrador o Propietario de Proceso Presupuestario. Para obtener más información sobre las funciones, consulte Permisos y funciones de Frontdoor.

La configuración inicial incluye las siguientes tareas de configuración de usuarios realizadas por el Administrador, y tareas de importación y validación de datos realizadas por el Administrador o el Propietario del Proceso Presupuestario. Una vez completada la configuración inicial, el trabajo del administrador continúa con la gestión de los datos.

Entre las tareas requeridas figuran las siguientes:

- [Configurar la transparencia de costes](configure-ct-support.html) (sólo se aplica a Planning )
- [Editar el perfil de la empresa](edit-company-profile.html "El Perfil de Empresa permite a los usuarios Administradores y Propietarios de Procesos Presupuestarios configurar los parámetros de la aplicación para personalizar la visualización, activar o desactivar funciones y definir el comportamiento del flujo de trabajo en Apptio Planning.")
- [Gestión de usuarios y accesos mediante Frontdoor](../../frontdoor/home.html)
- [Gestión de datos de referencia para la planificación](manage-itfmf-reference.html "◆ Se aplica a: Planificación") (sólo en Planning )
- [Gestionar dimensiones](manage-reference-data.html)
- [Gestionar dimensiones personalizadas](manage-custom-reference.html "Las siguientes tareas sólo pueden ser realizadas por usuarios asignados a los roles de Administrador o Propietario de Proceso Presupuestario. Para obtener más información sobre las funciones, consulte Permisos y funciones de Frontdoor.")

Entre las tareas opcionales se incluyen las siguientes:

- [Personaliza, guarda y comparte diseños de tablas](customize-save-share.html)
- [Suplantar la identidad de un usuario](impersonate-user.html "Las siguientes tareas sólo pueden ser realizadas por usuarios asignados a los roles de Administrador o Propietario de Proceso Presupuestario. Para obtener más información sobre las funciones, consulte Permisos y funciones de Frontdoor.")
- [Restringir el acceso a los datos salariales de los trabajadores](restrict-access-labor.html "Las siguientes tareas sólo pueden ser realizadas por usuarios asignados al rol de Administrador. Ver permisos y roles de Frontdoor.")
- [Integrar con transparencia de costes](integrate-ct.html "Si su organización utiliza tanto Apptio Costing Standard como una aplicación de planificación Apptio, puede integrarlas para compartir datos.")
- [Active las funciones Service Demand Planning o Project Financial Planning](edit-company-profile.html "El Perfil de Empresa permite a los usuarios Administradores y Propietarios de Procesos Presupuestarios configurar los parámetros de la aplicación para personalizar la visualización, activar o desactivar funciones y definir el comportamiento del flujo de trabajo en Apptio Planning.")
- [Archivar, restaurar o eliminar planes](manage-plans.html "Con la página Planes, puede gestionar sus planes de forma fácil e intuitiva.")
- [Admite varias divisas](support-multiple-currencies.html)
- [Activar calendarios personalizados](custom-calendars.html "Apoyamos tres formas en que una organización puede estructurar sus calendarios fiscales:")

Project Financial Planning entre las tareas opcionales se incluyen las siguientes:

- [Gestionar los datos de referencia del proyecto](pfp/manage-project-configuration.html)

VER TAMBIÉN: [Project Financial Planning](pfp/gs-project-financial.html "◆ Se aplica a: Planificación con proyecto Planificación financiera.")

Service Demand Planning entre las tareas opcionales se incluyen las siguientes:

- [Gestionar los datos de referencia del Servicio](sdp/manage-service-reference.html)

VER TAMBIÉN: [Service Demand Planning](sdp/gs-service-demand.html)

Una vez completada la configuración, su organización puede iniciar el proceso de planificación o previsión. Véase [Planificación y previsión presupuestarias](budget-planning-forecasting.html).
