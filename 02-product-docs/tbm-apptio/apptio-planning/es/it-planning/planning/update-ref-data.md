# Actualizar las dimensiones de los datos de referencia

◆ Se aplica a: Planning

Las siguientes tareas sólo pueden ser realizadas por usuarios asignados a los roles de Administrador o Propietario de Proceso Presupuestario. Para obtener más información sobre las funciones, consulte Permisos y funciones de Frontdoor.

## Actualizar datos de referencia

1. En el menú de navegación del Plan Costing & Planning , seleccione **Configuración >** Datos de referencia > Cuenta.
2. Seleccione ![más icono](../../../../../03-media/apptio-planning/resources/images/icons/3-dots.png) en la esquina superior derecha **>** Seleccione **Importar >** Configure las opciones de importación para importar los datos del archivo.csv.

   ![imagen](../../resources/images/it%20planning_images/import-itp.png)
3. Siga las instrucciones que aparecen en pantalla para completar el proceso de importación de datos. Examine y verifique los datos importados.
   - Si sus datos parecen correctos y desea que estén disponibles en los planes, seleccione ![más icono](../../../../../03-media/apptio-planning/resources/images/icons/3-dots.png) **>** seleccione **Publicar**.
   - Si observa problemas con sus datos y necesita solucionarlos antes de publicarlos, seleccione ![más icono](../../../../../03-media/apptio-planning/resources/images/icons/3-dots.png) **>** seleccione **Revertir**.
