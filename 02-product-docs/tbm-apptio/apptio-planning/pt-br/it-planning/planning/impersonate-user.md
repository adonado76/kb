# Representação de usuário

Observação: *as funções de administrador ou responsável pelo processo orçamentário são necessárias para representar outra pessoa.*

A representação de usuário permite que os administradores ajam temporariamente como outro usuário para:

- Resolva problemas de permissão ou acesso
- Valide o que um usuário final pode ver ou editar
- Confirme as alterações de configuração
- Apoiar os usuários durante a integração ou o treinamento

Os administradores podem se passar por qualquer usuário com acesso ao Apptio Planning.

**Fingindo ser um usuário**

1. Na barra Apptio de ferramentas, selecione **Configurações do perfil** → **Representar**.
2. Digite o **nome** de usuário da pessoa que você deseja representar.
3. Selecione **Avançar**.
4. Você agora está representando o usuário selecionado.

Selecione **Configurações (ícone de engrenagem) → Sobre o aplicativo** para confirmar que você está conectado como o usuário representado. O nome de usuário exibido corresponderá à conta representada e incluirá um asterisco (\*) para indicar o modo de representação.

**Para parar de se passar por um usuário:**

- Na barra Apptio de ferramentas, selecione **Configurações do perfil** → **Sair**.
- Faça login novamente usando suas próprias credenciais.
