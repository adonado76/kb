# Atualizar dados reais nos planos de previsão

Observação: *as funções de administrador ou responsável pelo processo orçamentário são necessárias para atualizar os valores reais.*

Os administradores e usuários do Budget Process Owner podem atualizar os dados de despesas reais em um plano de previsão sem a necessidade de recriar o plano sempre que houver novos dados reais.

Os dados reais podem ser atualizados nos planos de previsão que estão no estado **NOVO** ou **ABERTO**.

No Plano de Previsão, os dados financeiros reais estão disponíveis na guia **Despesas > Resumo**. Os itens de linha reais são identificados pelo valor do **Tipo de item de linha** definido como “**Reais** ”.

## Como atualizar os dados reais

1. Abra seu plano de previsão e navegue até a página **Expenses (Despesas** ).
2. Clique no **menu Elipses→** **Update Actuals Data (Atualizar dados reais** ).
3. Na janela pop-up, use o menu suspenso para selecionar os **meses** para os quais você deseja atualizar os valores reais.
4. Clique em **Update (Atualizar** ) para atualizar os dados dos períodos escolhidos.
