# Gerenciar layouts

Os layouts controlam como as tabelas de planejamento são exibidas, incluindo visibilidade da coluna, classificação, agrupamento, filtros e formatação. Você pode criar visualizações pessoais para o seu trabalho diário ou compartilhar layouts entre equipes.

## Antes de começar

Tipos de layout

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| **Tipo** | **Visibilidade** | **Criado por** | **Caso de uso** | **Limite** |
| Pública | Visível para todos os usuários | Usuários com função de administrador ou proprietário do processo orçamentário | Use como layout padronizado para alinhamento da equipe | 50 layouts privados |
| Privada | Visível apenas para você | Qualquer usuário com acesso aos planos S | Adapte a estrutura a seus fluxos de trabalho ou análises | 50 layouts públicos |

## Criação e gerenciamento de layouts

**Criar um layout privado**

1. Comece em qualquer exibição de tabela de despesas.
2. Ajuste as colunas, os filtros, o agrupamento, a classificação etc. de acordo com sua visualização preferida.
3. Abra o **menu seletor de layout**.
4. Escolha **"Salvar como...".**
5. Forneça um **nome de layout**.
6. Marque a caixa de seleção para salvar as configurações de filtro.
7. Clique em **Salvar**.

**Atualização de um layout existente**

- Depois de fazer ajustes na tabela, você pode Salvar ou Salvar como em um layout existente.
- Para layouts públicos, somente os usuários com funções de Administrador ou Proprietário do processo orçamentário podem modificar ou excluir esses layouts.

**Criar um layout público**

Observação: As funções de administrador ou de proprietário do processo orçamentário são necessárias para criar layouts públicos.

Os administradores também podem modificar o **layout padrão**, que determina a visualização inicial para todos os usuários quando eles abrem um plano, garantindo uma experiência consistente em toda a organização.

**Converter um layout privado em público:**

1. Abra o **menu seletor de layout**.
2. Clique em **Gerenciar layouts...**
3. Alterne o layout desejado para **Público**.
4. O layout ficará imediatamente visível para **todos os usuários**.

Uma vez tornado público, o layout pode ser acessado e aplicado por qualquer pessoa no plano.

**Gerenciando layouts**

Na janela **Gerenciar layouts**, você pode executar as seguintes ações:

1. Renomeie ou exclua os layouts que você possui.
2. Os administradores ou proprietários de processos orçamentários podem:
   1. Gerenciar ou excluir qualquer layout público.
   2. Converter layouts privados em públicos.
