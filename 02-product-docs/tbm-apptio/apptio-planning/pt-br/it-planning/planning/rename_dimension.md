# Renomear dimensões padrão relacionadas ao projeto

## Sobre esta tarefa

Importante: *Disponível com a* *assinatura* ***do Apptio Planning Standard***

Você pode renomear as seguintes dimensões relacionadas ao projeto em Integrated Investment Planning:

- Projeto
- Grupo de projetos
- Função de trabalho de projeto

## Procedimento

- **Renomear uma dimensão**

  1. Navegue até Configuração > Esquema > Dimensões padrão.
  2. Selecione a dimensão que você deseja renomear.
  3. No nome da dimensão, selecione ![Editar ícone](../../../../../03-media/apptio-planning/resources/images/icons/icon-edit.png).
  4. Digite o novo nome.

     A dimensão é renomeada.

     O novo nome da dimensão é exibido nos Itens de linha em que a dimensão renomeada é usada como um atributo de pesquisa
- **Restaurar o nome da dimensão padrão**

  Se desejar, você pode voltar ao nome de dimensão padrão

  - Selecione a dimensão e, em seguida, selecione ![ícone de atualização](../../../../../03-media/apptio-planning/resources/images/icons/icon-reset.png)Redefinir.

  O nome da dimensão é redefinido para o nome padrão.
