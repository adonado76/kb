# Atualizar dimensões de dados de referência

aplica-se a: Planning

As tarefas abaixo só podem ser realizadas por usuários atribuídos às funções Admin ou Budget Process Owner. Para obter mais informações sobre funções, consulte Permissões e funções do Frontdoor.

## Atualizar dados de referência

1. No Costing & Planning No menu de navegação do plano, selecione **Configuração >** Dados de referência > Conta.
2. Selecione ![mais ícone](../../../../../03-media/apptio-planning/resources/images/icons/3-dots.png) no canto superior direito **>** Selecione **Importar >** Defina suas opções de importação para importar os dados do arquivo.csv.

   ![imagem](../../resources/images/it%20planning_images/import-itp.png)
3. Siga as instruções na tela para concluir o processo de importação de dados. Examine e verifique seus dados importados.
   - Se seus dados estiverem corretos e você quiser disponibilizá-los nos planos, selecione ![mais ícone](../../../../../03-media/apptio-planning/resources/images/icons/3-dots.png) **>** selecione **Publish**.
   - Se você encontrar problemas com seus dados e precisar solucioná-los antes de publicar, selecione ![mais ícone](../../../../../03-media/apptio-planning/resources/images/icons/3-dots.png) **>** selecione **Revert**.
