# Apptio Dicas e truques de planejamento

## Sobre esta tarefa

Este guia oferece dicas práticas para aprimorar sua experiência com o site Apptio Planning. Você aprenderá a navegar com eficiência na plataforma, personalizar layouts e evitar problemas comuns, como barras de rolagem ausentes ou incompatibilidade de moedas. Além disso, ele fornece conselhos sobre como otimizar o desempenho do navegador e marcar páginas para facilitar o acesso.

1. A guia Resumo do plano serve apenas para revisão e análise.

   Não é possível fazer atualizações em seus planos na guia Summary (Resumo). Para atualizar os planos, use as guias Trabalho, Contratos, Ativos e Outros.

   ![imagem](../../resources/images/it%20planning_images/tips-summary.png)

   <analyze-plan-forecast.html>
2. Marcar uma página inicial como favorita.

   Você pode configurar qualquer página que desejar para ser sua página inicial, com marcadores.

   [Saiba mais sobre como criar marcadores](bookmark.html#Create)
3. Diminua o nível de zoom do navegador se não estiver vendo a barra de rolagem inferior.

   Diminua o zoom do navegador, por exemplo, para 95%.
4. Use layouts públicos.

   Modifique o layout da tabela e salve-o como um layout pessoal para uso futuro.

   [Saiba mais sobre como personalizar layouts de tabela](customize-save-share.html#top)
5. Ao usar os relatórios Painel e Resumo para comparar orçamentos, defina a moeda como Moeda da empresa para evitar misturar tipos de moeda.

   [Saiba mais sobre a comparação de planos](compare-versions-plans.html#top)
6. Para obter o melhor desempenho, use Google Chrome como seu navegador de Internet.
