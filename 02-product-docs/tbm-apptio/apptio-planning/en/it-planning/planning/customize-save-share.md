# Manage Layouts

Layouts control how your plan tables are displayed, including column visibility,
sorting, grouping, filters, and formatting. You can create personal views for your day-to-day
work or share layouts across teams.

## Before you begin

Layout Types

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| **Type** | **Visibility** | **Created By** | **Use Case** | **Limit** |
| Public | Visible to all users | Users with Admin or Budget Process Owner role | Use as standardized layout for team alignment | 50 private layouts |
| Private | Visible only to you | Any user with access to plans S | Tailor structure to your workflows or analyses | 50 public layouts |

## Creating & Managing Layouts

**Create a Private Layout** 

1. Start from any Expenses table view.
2. Adjust columns, filters, grouping, sorting, etc. to your preferred view.
3. Open the **Layout selector menu**.
4. Choose **“Save As…”**.
5. Provide a **layout name**.
6. Select the checkbox to save filter settings.
7. Click **Save**.

**Updating an Existing Layout**

- After making table adjustments, you can Save or Save As over an existing layout.
- For public layouts, only users with Admin or Budget Process Owner roles may modify or
  delete those layouts.

**Create a Public Layout** 

Note: Admin or Budget Process Owner roles are required to
create Public Layouts.

Admins can also modify the **Default Layout**, which determines the initial view for all
users when they open a plan — ensuring a consistent experience across the organization.

**Convert a Private Layout to Public:**

1. Open the **Layout selector menu**.
2. Click **Manage Layouts...**
3. Toggle the desired layout to **Public**.
4. The layout will immediately become visible to **all users**.

Once made public, the layout can be accessed and applied by anyone in the plan.

**Managing Layouts**

In the **Manage Layouts** window, you can perform the following actions:

1. Rename or delete any layouts you own.
2. Admins or Budget Process Owners can:
   1. Manage or delete any public layout.
   2. Convert private layouts to public.
