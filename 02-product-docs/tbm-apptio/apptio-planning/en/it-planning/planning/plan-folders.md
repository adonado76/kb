# Plan Folders

To organize your plans and make them easier to find, you can use plan folders. On the
Plans page, you can create or manage folders and move plans into the folders you’ve created.

Note: Admin or Budget Process Owner roles are required to create and manage folders.

## Create a plan folder

To create plan folders:

1. In the left-hand navigation pane, navigate to **Plans.**
2. In the upper right, click the **three dot overflow menu**.
3. Click **New Plan Folder**.

   ![image of new plan folder](../../../../../03-media/apptio-planning/resources/images/it_planning_images/new-pln-folder.png)
4. In the dialog box, enter a folder name, then click **Create Folder**.

## View and manage folders

To access and manage your folders:

1. In the left-hand navigation pane, navigate to **Plans.**
2. To view the plans in a folder, click the **caret icon** next to the folder’s name.
3. To rename a folder, select the checkbox next to the folder, then select the
   **three-dot overflow menu** and click **Rename Folder**. In the dialog box,
   enter a new folder name, then click **Rename**.
4. To archive a folder, select the checkbox next to the folder, then select the
   **three-dot overflow menu** and click **Archive**. Archiving a folder will
   archive the plans within the folder.
5. To delete a folder, select the checkbox next to the folder, then select the
   **three-dot overflow menu** and click **Delete Folder**. In the confirmation
   dialog, type **DELETE** in the text box and select whether to also delete plans
   within the folder, and click **Delete**.

![image of managing folders](../../../../../03-media/apptio-planning/resources/images/it_planning_images/managr-folders.png)

## Add plans to a folder

To move plans into a folder:

1. In the left-hand navigation pane, navigate to **Plans.**
2. Select the checkboxes next to plans you want to move, then click the **three-dot
   overflow menu** and click **Move to Folder**.
3. In the dialog box, select a **folder**, then click **Move Folder**.
