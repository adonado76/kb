# Set labor headcount targets

A Budget Process Owner can set annual labor headcount targets at any level of your organization's
Department, Service, or Project hierarchy. Groups with owner permissions for Cost Objects can set
labor headcount targets for their subordinate Cost Centers, and individual users with owner
permissions for cost objects can view variance against headcount targets. Unlike financial targets,
labor targets focus on current and planned headcount (not their cost). See [Set financial targets.](set-financial-targets.html)

Enable the labor planning features. See [Edit the Company Profile](edit-company-profile.html).

1. In the Plan menu, select a plan.

   [Learn more about the plan menus](navigate-apptio-planning.html)
2. In the Plan section menu, select Department or Project.
3. In the Plan sub-section menu, select a department, project, department group, or project
   group.
4. On the Component menu, select Planning. See [Navigate in Apptio planning
   applications](navigate-apptio-planning.html). > Targets.
5. Select the Labor tab.
6. Enter headcount targets in the Targets column. The REMAINDER value in the table
   updates to display the labor headcount target of the parent Cost Center minus the sum of subordinate
   Cost Center's labor headcount targets.

Note: Any changes to the targets will automatically reflect in all the plans.

Other notable columns in the Labor Targets table include:

- Variance (Var) - The line-item's target value minus the plan's value. This and the
  variance value change as you enter target values.
- Variance Percentage (Var %) - The percentage difference between the target and
  plan values.
- Plan - The sum of labor headcount for the last month of the planning period (as
  seen in the Expenses> Labor table), regardless of any varied headcount values
  throughout the year.

## View labor KPIs

After you publish labor targets using the previous instruction, the labor KPIs appear on the
Expenses > Labor tab.

1. In the plan menus at the top right, select a plan, a Cost Object category, and a cost object or
   cost object group.

   [Learn more about
   navigating in Planning](navigate-apptio-planning.html)
2. Navigate to Planning > Expenses.
3. Select the Labor tab.
   - Select Existing to plan labor compensation adjustments for current labor.
   - Select Planned to plan labor compensation adjustments for planned labor.
