# Set financial targets

Watch this video from Apptio Education Services: [Creating and Opening
Plans](https://community.apptio.com/videos/1607 "(Opens in a new tab or window)").

Or see all [Apptio planning video and training resources](https://community.apptio.com/viewdocument/apptio-products-video-catalog?CommunityKey=1bdb1f0b-9524-43d4-b987-5d2b8595eebf "(Opens in a new tab or window)").

A Budget Process Owner can set targets for various departments, projects, and their groups. The
targets are displayed in the KPIs in the reports. In addition to financial targets, you can set and
then track variance against current and planned labor headcount. See [Delegate project
costs](pfp/delegate-project-costs.html).

A budget owner can set budget or forecast financial targets for their departments or projects.
Depending on the depth of the Department or Project hierarchy, you may need to select a lower-level
department or project from the Plan Section menu. See [Apptio planning menu](navigate-apptio-planning.dita "(Opens in a new tab or window)") and [Manage Financial Dimensions
reference data](manage-financial-dimensions.html "The tasks below can only be performed by users assigned to the Admin or Budget Process Owner roles. For additional information on roles, see Frontdoor permissions and roles.").

To set targets, utilize either of the following approaches:

- Top-down - A Budget Process Owner sets targets for the whole plan. Each budget owner or
  Project Owner then allocates targets to their individual department or project. This is the most
  common approach to setting targets.
- Bottom-up - A group budget owner bases a target on subordinate department or project
  budgets. Targets then roll up to the Budget Process Owner to determine the overall budget.

To set targets:

1. In the Plan menu, select a plan.
2. In the Plan section menu, select Department or Project.
3. In the Plan sub-section menu, select a department, project, department group, or project
   group.
4. On the Component menu, select Planning. See [Navigate in Apptio planning
   applications](navigate-apptio-planning.html). > Targets.
5. Click a tab on the upper left of the page to select a display.
6. In the Target column, type individual Cost Object targets. As you enter target values,
   the (REMAINDER) value in the first row of the table updates.

Note:

As you enter or edit target values:

- The remaining unassigned amount is displayed at the top of the table.
- As soon as you move focus off the cell or navigate to another view, changes to target values
  will take effect.
- Updated target values roll-up into the summary target displayed in the KPIs.

On the Summary page, the Tree Map chart uses the following colors to indicate if a
department's or project's budget is under or over target. See [Analyze a plan or forecast.](analyze-plan-forecast.dita "(Opens in a new tab or window)")

- Gray - A target has not been set for the department or project.
- Blue - The department or project budget is under target. The darker the color, the
  greater the percentage under target.
- Red - The department or project budget is over target. The darker the color, the greater
  the percentage over target.

Note: You must have either the OpEx or CapEx tab selected to light up the tree map as
you set the targets at that level, as shown in the following image.

Tip: If Project Financial Planning is enabled, Project Owners can delegate project
costs. See [Edit the Company
Profile](edit-company-profile.html). Delegated costs give Project Owners the flexibility to record expenses in the
project but specify the departments responsible for paying future costs. On the Targets view,
select or clear the Show Delegated Costs checkbox to show or hide the delegated costs from
the view. For more information, see [Delegate project costs](pfp/delegate-project-costs.html).
