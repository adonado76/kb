# Status

To check the status of all the data sharing activities between Planning and Costing please
navigate to Automated Data Management > Status.

![](../../../resources/images/it%20planning_images/status-adm.png)

To learn more, visit [Data Sharing Status](../../../data-sharing/data_sharing_status.html).

All the data sharing activities are executed through an asynchronous job. While ADM UI shows high
level status of the data sharing activities, the detailed job level status for each entity and any
errors during data sharing can be viewed by following the below steps:

1. On the Apptio Planning menu, select **Settings** (![](../../../resources/images/gear.png)) then select **Cost Transparency Integration**.
2. Select the **Status** tab.

   This page shows the details of Automated Data Management jobs
   and further information on each, if any.

   ![](../../../resources/images/it%20planning_images/cti-pg.png)

   ![](../../../resources/images/it%20planning_images/status-1.png)
