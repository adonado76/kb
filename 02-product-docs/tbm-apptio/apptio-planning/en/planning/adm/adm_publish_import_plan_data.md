# Publish Plan data to Costing Standard

## Publish Plan data to Costing Standard

The ApptioOne Costing data model can accommodate only one budget and one forecast for analysis.
Hence, only one of each plan type can be pushed from ApptioOne Planning to ApptioOne Costing.

To publish budgets or forecasts to Costing, follow the steps below:

1. On the left navigation menu, select Planning > Plans.
2. Under the Active Plans tab, select the checkbox corresponding to the plan you want to
   publish data for.
3. Right-click the selected plan or select the ![](../../../resources/images/icons/3-dots.png) button from the top right of
   the page.
4. Select Publish to Cost Transparency....

   Confirm Publish dialog is displayed,
   that prompts you for a confirmation on publishing the data.

   ![](../../../resources/images/it%20planning_images/publish_to_ct_dialog.png)
5. Select Publish. Select Status on the dialog window to view the status of your
   task. The Publish to Costing process does not require any follow-up task, and you can find the
   imported data in Cost Transparency once the job is complete.

At this point, you may also click Cancel and return to performing other Planning tasks.
The publish task, even if it's a sizable one, does not block you from performing other Planning and
Cost Transparency Integration tasks.

## Publish Line-Item data to Costing Standard

This feature allows the Administrator to publish only specific expense line data on-demand and
not the entire plan.

Navigate to Expenses > Labor Activity tab

Expand Actions, and then select Publish Labor Activity to Cost Transparency

![](https://help.apptio.com/en-us/resources/images/it%20planning_images/publish-1.png)

Popup appears as shown

![](https://help.apptio.com/en-us/resources/images/it%20planning_images/popup.png)

Select Status button to view the status of the published job.

![](https://help.apptio.com/en-us/resources/images/it%20planning_images/status.png)

![](https://help.apptio.com/en-us/resources/images/it%20planning_images/status-1.png)

- To publish Financials, navigate to Expenses > Summary tab > Actions, and
  then select Publish Financials to Cost Transparency...
- To publish labor details, navigate to Expenses > Summary tab > Actions, and
  then select Publish Labor to Cost Transparency...
- To publish the contract details, navigate to Expenses > Summary tab >
  Actions, and then select Publish Contracts to Cost Transparency...
- To publish asset details, navigate to Expenses > Summary tab > Actions, and
  then select Publish Assets to Cost Transparency...
