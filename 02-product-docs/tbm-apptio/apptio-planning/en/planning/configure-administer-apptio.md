# Configure Apptio planning applications

The tasks below can only be performed by users assigned to the Admin or Budget Process Owner
roles. For additional information on roles, see Frontdoor permissions and roles.

Initial configuration includes the following user setup tasks performed by the Admin, and data
import and validation tasks performed by the Admin or Budget Process Owner. Once the initial
configuration is complete, the Admin's work continues with data management.

Required tasks include the following:

- [Configure Cost
  Transparency](configure-ct-support.htm "(Opens in a new tab or window)") (applies to Planning only)
- [Edit the Company
  Profile](edit-company-profile.htm "(Opens in a new tab or window)")
- [Manage users and access
  using Frontdoor](../../frontdoor/home.htm "(Opens in a new tab or window)")
- [Manage reference data for
  Planning](manage-itfmf-reference.htm "(Opens in a new tab or window)")(applies to Planning only)
- [Manage
  dimensions](manage-reference-data.html "(Opens in a new tab or window)")
- [Manage custom
  dimensions](manage-custom-reference.htm "(Opens in a new tab or window)")

Optional tasks include the following:

- [Customize, save, and share
  table layouts](customize-save-share.htm "(Opens in a new tab or window)")
- [Impersonate a user](impersonate-user.htm "(Opens in a new tab or window)")
- [Restrict access to labor
  salary details](restrict-access-labor.htm "(Opens in a new tab or window)")
- [Integrate with Cost
  Transparency](integrate-ct.htm "(Opens in a new tab or window)")
- [Enable Service Demand
  Planning or Project Financial Planning features](edit-company-profile.htm "(Opens in a new tab or window)")
- [Archive, restore, or delete
  plans](manage-plans.htm "(Opens in a new tab or window)")
- [Support multiple
  currencies](support-multiple-currencies.htm "(Opens in a new tab or window)")
- [Enable custom
  calendars](custom-calendars.htm "(Opens in a new tab or window)")

Project Financial Planning optional tasks include the following:

- [Manage Project
  reference data](pfp/manage-project-configuration.htm "(Opens in a new tab or window)")

SEE ALSO:
[Project Financial
Planning](pfp/gs-project-financial.htm "(Opens in a new tab or window)")

Service Demand Planning optional tasks include the following:

- [Manage Service
  reference data](sdp/manage-service-reference.htm "(Opens in a new tab or window)")

SEE ALSO:
[Service Demand
Planning](sdp/gs-service-demand.htm "(Opens in a new tab or window)")

process. See [Budget
planning and forecasting](budget-planning-forecasting.htm "(Opens in a new tab or window)").
