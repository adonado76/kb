# Manage email notifications

Apptio sends email notifications to the email addresses provided as part of user
administration in Enhanced Access Administration. For Enhanced Access Administration-specific
information, see User and role management.

## About this task

Events that trigger email notifications include the following:

- When the password of a user is reset in Enhanced Access Administration
- When the status of a plan is switched from New to Open; see [Open a plan or a forecast](open-plan-forecast.html).
- When a plan owner submits their plan for approval; see [Submit a plan or forecast](submit-plan-forecast.html).
- When the Budget Process Owner or approver comments on or rejects a submitted plan

The triggers of email notifications cannot be changed. However, the Admin has full control over
the specific email address per user to which notifications are sent. The email address value does
not need to match the user name value of the user used to log in to the application

## Procedure

**Planning Process Events**

- Users receive email notifications when Planning Process Events is enabled on the Company Profile
  page. For more information, see [Edit the Company Profile](edit-company-profile.html).
- Users with edit levels of Owner or Edit & Submit receive email notifications for Planning
  Process Events. For more information, see [Cost Object Permissions](manage-cost-object.html).
