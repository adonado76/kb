# Enable Apex Line Item Table

When an Admin or Budget Process Owner enables the Apex Line Item Table feature, users with
non-administrator role can change the table view between Classic View and New View.

Note: Expenses > New view is available only for the customers who don't have PFP or
SDP.

## Enable Apex Line Item Table

Administrators only

Under ![](../../resources/images/icons/icon-settings_20x20.png) > Company Profile,
select Enable New Expenses Page Experience (Beta), and save the changes.
