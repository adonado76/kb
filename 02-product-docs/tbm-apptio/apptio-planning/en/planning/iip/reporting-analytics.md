# Reporting and Analytics

In Integrated Investment Planning, projects are exposed as dimension on Summary, Labor, Labor
Activity, Contracts, and Assets expense lines so the project information is available in Apptio BI
data sources for Planning. You can build customized project reports using Apptio BI.

Below, you can find some examples for which reports can be built:

- Report on Project and Department costs.
- Rollup and report Run and Grow cost for the entire organization or an individual project.
- Build reports for Opex and Capex spend on projects.
- Internal and external Project Labor allocation.
- Labor allocation to a project by Project Labor Role.

Separately or combined.

To learn more about Apptio BI, see [Planning in
Apptio BI](../ssr/it-planning-data-self-service-reporting.htm "(Opens in a new tab or window)") and [About
Apptio BI](../../../apptio%20bi/home.htm "(Opens in a new tab or window)").
