# Bookmark a page

All users can bookmark pages in Apptio planning applications. You can bookmark any page that you
think you'll return to. Bookmarking saves time and effort during navigation and eliminates the need
to recreate views or remember to apply layouts, allowing you to use the application more
efficiently.

Additionally, you can set any bookmarked page as your default landing page that will open
whenever you log in to the application.

## Bookmark features

The following information is saved in a bookmark:

- All selections in the top menu bar, excluding plan selection.
- Your current tab (if your view contains a tab).
- If your view contains a table, any layout changes you make to a table.
- Filter selections.

Note: Bookmarks are unique to users and cannot be shared.

## Create a bookmark

The following image indicates the bookmark icon, as well as the information captured in a
bookmark.

Tip: The bookmark icon appears filled, if the current page is bookmarked.

1. Select the icon. The Create New Bookmark dialog appears.
2. Enter the name of your bookmark, then select Save.
3. To navigate to a saved bookmark, select Bookmarks in the top menu bar,
   then select the bookmark you want to open. The Bookmarks drop-down is only
   visible when you have at least one saved bookmark.

## Set a default landing page

You can choose any bookmarked page as your default landing page. This page opens each time you
log in to the application.

1. Navigate to the page you want to use as your landing page, then select the icon.
2. Select Bookmarks in the top menu bar, and then select
   Bookmarks Manager.
3. Select to set a bookmark as the home page.

## Edit, reorder, or delete a bookmark

Use the Bookmarks Manager to edit, reorder, or delete existing
bookmarks.

To open the Bookmarks Manager, select Bookmarks in
the top menu bar, and then select Bookmarks Manager.

1. Select the name of a bookmark to edit the text.
2. Select and hold the icon to reorder your bookmarks.
3. Select the icon to delete a bookmark.
