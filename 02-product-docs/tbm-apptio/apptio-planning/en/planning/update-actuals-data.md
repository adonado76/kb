# Update Actuals Data

Administrators and BPO users can now update the actuals in the forecast plan without
having to recreate the forecast each time there is change in actuals.

## About this task

This option is visible in the old and new view, on Expenses,
Summary Charts and Dashboard pages for both
New and Open plans.

Note: This option does not appear for Archived plans.

## Procedure

1. Open the Expenses page in the New view. In
   this scenario, there is a mismatch in the actual spends for `Consulting` and a
   missing entry for `Travel & Expenses` in January 2024.

   **Add Image
   (../../resources/images/it planning\_images/ua-1.png)**
2. To update the actual data, select the **Add Image (../../resources/images/it
   planning\_images/three-dots\_39x29.png)** icon, and then select Update Actuals
   Data option.

   **Add Image (../../resources/images/it
   planning\_images/ua-2.png)**
3. The following popup appears. The drop-down will have all the months for which actual data
   is available. 

   **Add Image (../../resources/images/it
   planning\_images/upd-act-popup.png)**
4. Select the `January 2024` month from the drop-down.

   **Add
   Image (../../resources/images/it planning\_images/upd-act-popup-1.png)**
5. Select Update. The page is refreshed and the updated data for
   Consulting and Travel & Expenses in January 2024
   is shown.

   **Add Image (../../resources/images/it
   planning\_images/upd-act-popup-2.png)**

   A success message `Actuals data updated
   successfully for January 2024` appears in the top banner.
