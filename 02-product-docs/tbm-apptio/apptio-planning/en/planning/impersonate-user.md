# Impersonate a user

The tasks below can only be performed by users assigned to the Admin or Budget Process
Owner roles. For additional information on roles, see Frontdoor permissions and roles.

![](../../resources/images/it%20planning_images/icon_video.png)

Roles
are managed in Enhanced Access Administration. To learn more about Enhanced Access Administration,
watch this video series from Apptio Education Services: [Frontdoor (Admin
Console) Essentials Training](https://community.apptio.com/docs/DOC-6665 "(Opens in a new tab or window)") .

Or see all [Apptio planning video and training resources](https://community.apptio.com/viewdocument/apptio-products-video-catalog?CommunityKey=1bdb1f0b-9524-43d4-b987-5d2b8595eebf "(Opens in a new tab or window)").

An Admin
may want to impersonate another user to ensure that user has what they need available to them and to
troubleshoot problems. When enabled, an Admin can impersonate any user within your organization.

1. In the Apptio toolbar, select Profile settings > Impersonate.
2. Log in with your user name and password.
3. When prompted, enter the user name of the account to impersonate. Note that the user name at the
   top right of the page displays the impersonated user's name with an asterisk next to it to indicate
   the user is being impersonated.
4. Log out to exit the impersonation.
