# Set the common currency

A single, common currency is used for all currency conversions. The common currency is
usually the organization's currency of record. All currency conversion rates are set against this
currency, and, unless another is specified, it is the default display currency per Cost
Center.

## About this task

After you enable the support for multiple currencies, set the common currency for your
organization. See [Support
multiple currencies](support-multiple-currencies.html)

The tasks below can only be performed by users assigned to the Admin or Budget Process Owner
roles. For more information on roles, see Frontdoor permissions and roles. ![](../../resources/images/it%20planning_images/icon_gear.png)

## Procedure

1. On the Apptio Planning menu, click the
   Settings button ( ![](../../resources/images/it%20planning_images/icon_gear.png) ), then click Company
   Profile.
2. Select the common currency from the Currency list
3. Click Save and Exit.
