# Modify a visualization

You can duplicate reports in Apptio BI. This is useful if you want to view a different
version of an existing visualization (for example, using a forecast rather than a budget as the data
source) without changing the original report. The following instructions cover the use-case of
duplicating a report, then editing a visualization on the new report to use a different
plan.

## Procedure

1. Navigate to your report in Apptio BI.
2. Click the "..." icon in the top-right corner, then select Duplicate.
3. Open the copied report.
4. To avoid confusion, rename the report.
5. Click the "..." icon in the top-right corner of the visualization you want to edit, then
   select Edit.
6. In the Plan Name field, select a different plan to use in your new
   visualization.
7. (Optional) Update the visualization name.

## What to do next

After saving your visualization, you will view the report which contains the visualization. You
can also share the report with other members of your organization. For information on sharing, see
[Manage reports](../../../apptio_bi/self-service-reporting_user_guide/manage-self-service-reports.html)
