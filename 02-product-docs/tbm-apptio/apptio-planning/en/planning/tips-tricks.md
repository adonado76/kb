# Apptio planning tips and tricks

## About this task

This guide offers practical tips for enhancing your experience with Apptio planning. You'll learn
how to effectively navigate the platform, customize layouts, and avoid common issues such as missing
scroll bars or currency mismatches. Additionally, it provides advice on optimizing browser
performance and bookmarking pages for easy access.

1. The Plan Summary tab is for review and analysis only.

   You can't make
   updates to your plans in the Summary tab. To update plans, use the Labor, Contracts, Assets, and
   Other tabs.

   ![](../../resources/images/it%20planning_images/tips-summary.png)

   <analyze-plan-forecast.html>
2. Bookmark a home page.

   You can set up any page you want to be your home page, with
   bookmarks.

   [Learn more about
   creating bookmarks](bookmark.html#Create)
3. Decrease your browser zoom level if you are missing the bottom scroll bar.

   Decrease your
   browser zoom, for example to 95%.
4. Use public layouts.

   Modify your table layout and save it as a personal layout for future
   use.

   [Learn more about
   customizing table layouts](customize-save-share.html#top)
5. When using the Dashboard and Summary reports to
   compare budget, set the currency to Company Currency to avoid mixing currency
   types.

   [Learn more about
   comparing plans](compare-versions-plans.html#top)
6. For best performance, use Google Chrome as your internet browser.
