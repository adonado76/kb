# Datos de actividad laboral ETC

Actualmente, los datos disponibles en Apptio BI sólo admiten datos de horas, pero para IIP la planificación de la Actividad Laboral está habilitada utilizando tanto la unidad Horas como la unidad ETC. Esta función permite a los usuarios crear informes de asignación utilizando ETC para que puedan analizar el informe de asignación utilizando la capacidad, los detalles de utilización y los datos de asignación mensual por recurso.

Cree una visualización como se describe [aquí](create-visualisation-for-report.html "Complete los siguientes pasos para crear una visualización en Apptio BI utilizando datos de Planificación. Puede personalizar la visualización de varias formas, como seleccionar el tipo de visualización (gráfico de barras, gráfico de líneas, etc.), filtrar los datos y seleccionar un periodo de tiempo."). Seleccione la Fuente de Datos como Trabajo de Planificación.

![Añadir visualización a un nuevo informe](../../../resources/images/it%20planning_images/fte-metric_229x465.png)

Se añade una nueva opción Cantidad (ETC) en el campo Métricas.

Introduzca los datos necesarios en los demás campos. El informe de muestra aparece como se muestra.

![Ejemplo de informe](../../../resources/images/it%20planning_images/fte-bi-report.png)

Puede personalizar la visualización de varias formas, seleccionando el tipo de visualización (gráfico de barras, gráfico de líneas, etc.), filtrando los datos y seleccionando un periodo de tiempo.
