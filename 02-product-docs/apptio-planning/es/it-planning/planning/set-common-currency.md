# Fijar la moneda común

Se utiliza una única moneda común para todas las conversiones monetarias. La moneda común suele ser la moneda de registro de la organización. Todas las tasas de conversión de moneda se establecen contra esta moneda y, a menos que se especifique otra, es la moneda de visualización por defecto por Centro de Coste.

## Acerca de esta tarea

Después de habilitar el soporte para múltiples monedas, establezca la moneda común para su organización. Ver [Admitir varias monedas](support-multiple-currencies.html)

Las siguientes tareas sólo pueden ser realizadas por usuarios asignados a los roles Admin o Propietario de Proceso Presupuestario. Para obtener más información sobre las funciones, consulte Permisos y funciones de Frontdoor. ![configuración](../../resources/images/it%20planning_images/icon_gear.png)

## Procedimiento

1. En el menú Apptio Planning, haga clic en el botón Configuración ( ![configuración](../../resources/images/it%20planning_images/icon_gear.png) ) y, a continuación, en Perfil de empresa.
2. Seleccione la moneda común en la lista Moneda
3. Haga clic en Guardar y salir.
