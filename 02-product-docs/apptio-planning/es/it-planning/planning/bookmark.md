# Marcar una página

Los marcadores le permiten volver rápidamente a las páginas de uso frecuente en Apptio Planning sin necesidad de volver a aplicar filtros, diseños o configuraciones cada vez. Cualquier usuario puede crear marcadores, y también puedes establecer uno como página de destino predeterminada, para que se abra automáticamente al iniciar sesión.

## Lo que ahorra un marcapáginas

Al crear un marcador, se captura la siguiente información:

- Selecciones de Departamento, Proyecto e Intervalo de fechas.
- La pestaña actual (si la vista contiene varias pestañas).
- Cambios en el diseño de las tablas (orden de las columnas, tamaño, visibilidad).
- Filtros aplicados.

Nota: Los marcadores son personales de cada usuario y no pueden compartirse.

## Crear un marcador

1. Navegue hasta la página que desea marcar.
2. Seleccione el **icono Marcador** en la parte superior derecha**.**
   1. El icono aparece relleno si la página actual ya está marcada.

   ![imagen del icono del marcador](../../../../../03-media/apptio-planning/resources/images/it_planning_images/create-bookmrk.png)
3. En el cuadro de diálogo **Crear nuevo marcador**, introduzca un nombre para el marcador.
4. (Opcional) Seleccione la opción para establecer esta página como página de inicio.
5. Seleccione **Guardar**.

## Abrir un marcador

1. Seleccione **Marcadores** en la barra de menú superior y, a continuación, elija el marcador que desee.
2. El **desplegable Marcadores** sólo aparece cuando tienes al menos un marcador guardado.

![imagen del marcador de apertura](../../../../../03-media/apptio-planning/resources/images/it_planning_images/open-bookmrk.png)

## Establecer una página de destino predeterminada

Puedes elegir cualquier marcador guardado como página de destino predeterminada. Esta página se abrirá automáticamente cada vez que se conecte.

1. Abre **Marcadores > Administrador de marcadores** en la barra de menú superior.
2. Seleccione el marcador que desea establecer como página de inicio y haga clic en el icono **Inicio**.
3. Pulse **Guardar**.

![imagen del marcador por defecto](../../../../../03-media/apptio-planning/resources/images/it_planning_images/bookmrk-def.png)

## Gestionar favoritos

Utiliza el **Gestor de favoritos** para editar, reordenar o eliminar favoritos.

1. Seleccione **Marcadores > Administrador de marcadores**.
2. Desde aquí puedes:
   1. **Editar** - Seleccione el nombre de un marcador para renombrarlo.
   2. **Reordenar** - Seleccione y arrastre el icono de reordenar para cambiar el orden.
   3. **Borrar** - Seleccione el icono de borrar para eliminar un marcador.

   Consejo: Cree marcadores para las vistas filtradas más utilizadas (por ejemplo, un departamento específico o un periodo de previsión) para ahorrar tiempo y agilizar su flujo de trabajo.
