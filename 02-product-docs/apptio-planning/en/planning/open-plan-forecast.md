# Open a plan or a forecast

After you create a budget plan or forecast, optionally adjust the baseline values, set
the targets, and open the plan so that the budget owners can edit line items. Budget owners cannot
see a plan when it is in the New state. When you open a plan, an email
notification is sent to budget owners and all users who have the Edit &
Submit permission associated with the Cost Objects in that plan.

## Before you begin

![](../../resources/images/it%20planning_images/icon_video.png) Watch this video from
Apptio Education Services: [Creating and Opening Plans](https://community.apptio.com/videos/1607 "(Opens in a new tab or window)") or Or see all [Apptio planning video and training resources](https://community.apptio.com/viewdocument/apptio-products-video-catalog?CommunityKey=1bdb1f0b-9524-43d4-b987-5d2b8595eebf "(Opens in a new tab or window)").

## Procedure

- When you open a plan, it is assigned a version number that displays at the upper
  right.

  1. n the Plan menu, select a Plan.
  2. Select Open Plan. The plan's state indicator changes from
     New to Open.
  3. Notify the plan users with the Comments and Collaboration feature, by selecting ![](../../resources/images/icons/icon-comment.png).

  [Learn more about using
  comments](../../cnc/using-comments.htm "(Opens in a new tab or window)")
- Reopen a finalized budget plan or forecast.

  Once a plan has been finalized, changes to that plan are restricted. A Budget Process Owner can
  reopen a previously finalized plan to edit that plan.

  1. Navigate to Planning >  > Plans and
     select a finalized plan.
  2. Select **Reopen Plan**. The plan's state indicator changes from **Final** to
     **Open**.
