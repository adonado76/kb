# FTE Labor Activity Data

Currently, the data available in the Apptio BI supports only hours data, but for IIP the Labor Activity planning is enabled using both Hours and FTE unit. This feature allows the users to create allocation reports using FTE so that they can analyze the allocation report using the capacity, utilization details and monthly allocation data per resource.

Create a visualization as described [here](create-visualisation-for-report.html "Complete the following steps to create a visualization in Apptio BI using data from Planning. You can customize the visualization in a variety of ways, including selecting the visualization type (bar chart, line graph, and more), filtering data, and selecting a time period."). Select the Data Source as Planning Labor.

![Add visualization to new report](../../../resources/images/it%20planning_images/fte-metric_229x465.png)

A new option Quantity (FTE) is added in the
Metrics field.

Enter data in all the other fields as required. The
sample report appears as shown.

![Sample report](../../../resources/images/it%20planning_images/fte-bi-report.png)

You can customize the visualization in a variety of ways, by selecting the
visualization type (bar chart, line graph, and more), filtering data, and selecting a time
period.
