# Manage direct costs

The Expenses page provides visibility on expected direct costs associated with all
the Cost Objects in Apptio planning applications. Indirect costs are calculated differently,
depending on the cost object.

1. In the plan menus at the top right, select a plan, a Cost Object category, and a cost object or
   cost object group.

   [Learn more about
   navigating in Planning](navigate-apptio-planning.htm#Toolbar)
2. Navigate to Planning > Expenses.
3. The Expenses page includes the following tabs:
   - Summary - Display all direct costs. Optionally, filter costs by CapEx and OpEx using the
     links below the tabs.
   - Labor - Forecast and manage Cost Object labor (see [Create a labor plan)](create-labor-plan.htm "(Opens in a new tab or window)").
   - Contracts - Forecast and manage contracts associated with your Cost Objects (see [Manage contracts](manage-contracts.htm "(Opens in a new tab or window)")).
   - Asset - Forecast needed Cost Object assets (see [Create an asset plan and enter details)](manage-assets.htm "(Opens in a new tab or window)").
   - Other - Forecast OpEx and CapEx for your projects, departments, or services (see [Enter financial details and adjust
     baseline values)](enter-financial-details.htm "(Opens in a new tab or window)").

## View delegated project costs

If Project Financial Planning Project Financial Planningis enabled, Project Owners can delegate
project costs to department and service owners. All of these owners can show or hide delegated costs
from their financials. See [Delegate project costs](pfp/delegate-project-costs.htm "(Opens in a new tab or window)").
