# Manage Labor Activity Type reference data

The Labor Activity Type reference data represents mapping between Department Cost Center Account
and Project Cost Center Account for given Activity Type. This information is used to allocate charge
and cross-charge amount to provider cost center and consumer cost center.

1. In the navigation menu, select Reference Data > Standard Dimensions > Labor Activity Type.
2. Update the data table values as required:
   - Labor Activity Type Names - The Activity Type name.
   - Cross-Charge Account – Account code for Department or provider cost center that receives
     cross-charge or credit.
   - Charge Account – Account code for project or consumer cost center against which charge, or debit
     will be applied.
   - Is Capitalizable – Represents whether the labor activity can be capitalized or not.
   - Is Default – Default value for the Labor Activity type. When entering data on the Labor Activity
     view, the activity marked as default will be applied by default if the user doesn’t select one. Only
     one activity can be marked as default.
