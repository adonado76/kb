# Manage Project Labor Role reference data

The Project Labor Role reference data represents project specific role for the internal or
external labor resource and labor rate at which project will be charged for using the resource of
given role.

1. In the navigation menu, select Reference Data > Standard Dimensions > Project Labor Role.
2. Update the data table values as required:
   - Name - The Project Labor Role naame.
   - Parent Code - Represents your project group's hierarchy. A Parent Code value is the code value
     of its immediate higher-level or parent account (if any).
   - Currency - The common currency code for the Project Labor Role. This value is required when
     support for multiple currencies is enabled in Company Profile. The common currency value should be
     the three-letter ISO code for the currency. If no currency code is entered, the default common
     currency is used.
   - Project Labor Rate – Represents hourly rate for labor role. Supports decimal numbers.
   - Source– Represents the source of Project Labor Role. It could be internal, external, specific
     Vendor name or any other source name you would like to provide.
