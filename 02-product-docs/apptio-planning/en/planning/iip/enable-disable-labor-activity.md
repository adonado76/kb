# Enable/Disable Labor Activity

Note: Labor Activity checkbox will be visible under Company Profile only if “Enable Integrated
Investment Planning” checkbox is enabled. Disabling Integrated Investment Planning feature will
automatically hide the Labor Activity checkbox from the company profile and hide Labor Activity tab
on the Expenses page.

Labor Activity feature in Integrated Investment Planning enables customers to plan and manage the
labor resource allocation to investments/projects as hourly or FTE based assignments. In case you
are managing this assignment outside Apptio or this type of allocation is not yet part of your
planning process, you won’t be using Expenses > Labor Activity tab. In such a
scenario, you might want to hide “Labor Activity” tab for the end users.

![](../../../resources/images/it%20planning_images/e-d-lab-act.png)

It is now possible for IT Planning administrators to enable or disable “Labor Activity” tab from
the Expenses view by following below steps.

1. Navigate to Settings![](../../../resources/images/icons/icon-settings.png) >
   Company Profile> Enable Capabilitiessection.
2. Select/unselect the Labor Activity checkbox to enable/disable the Labor Activity tab on
   the Expenses page.

## Prevent over allocation

Select this option to prevent over allocating the resource beyond available capacity. To know
more, see [Prevent over
allocation](prevent-overallocation.htm "(Opens in a new tab or window)").

## Allow over allocation

This option is selected by default. It allows you to allocate labor resources without any
capacity constraints and set 'over' and 'under' allocation threshold values to flag allocations that
exceed or fall below the configured thresholds. To know more, see [Allow over allocation](allow-over-allocation.htm "(Opens in a new tab or window)").
