# Manage Integrated Investment Planning reference data

◆ Applies to: Apptio  planning applications with Integrated Investment
Planning.

To enable Integrated Investment Planning features, see [Edit the Company Profile](../edit-company-profile.htm "(Opens in a new tab or window)").

This help document does not apply to Project Financial Planning customers. For Project Financial
Planning reference data, see [Manage Project Configuration reference data.](../pfp/manage-project-configuration.htm "(Opens in a new tab or window)")

The tasks below can only be performed by users assigned to the Admin or Budget Process Owner
roles.

For additional information on roles, see [Cloudability SaaS user roles in Enhanced Access Administration](../../../cloudability-saas/administration/roles-and-permissions.htm "(Opens in a new tab or window)")

Dimensions are the essential data categories in budget line items. Many built-in dimensions have
dependencies on other dimensions. You can import dimensions reference data from a .CSV file or
from Costing Standard  and export dimensions reference data templates and table data.

The Integrated Investment Planning feature requires to configure the following reference data:

- [Project](manage-project-data-ref.htm "(Opens in a new tab or window)")
- [Project
  Group](manage-project-group-data-ref.htm "(Opens in a new tab or window)")
- [Project Labor
  Role](manage-project-labor-role-data-ref.htm "(Opens in a new tab or window)")
- [Labor
  Activity Type](manage-labor-activity-type-data-ref.htm "(Opens in a new tab or window)")
