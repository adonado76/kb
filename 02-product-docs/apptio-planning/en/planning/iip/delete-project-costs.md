# Delegate Project Costs

For projects with multiple phases (e.g., build/run), delegating expenses to specific cost centers
aids in effectively tracking the financial performance of each project phase. Project owners can
delegate the asset depreciation to a different cost center as well as delegate contract expense to a
different cost center starting from a specified date. To know more, refer the topics below:

[Delegate Asset
Costs](asset-depreciation-cost-center.htm "(Opens in a new tab or window)")

[Delegate Contract
Costs](delete-contract-costs.htm "(Opens in a new tab or window)")
