# Update line-item tables

## Before you begin

Most tables with line items in Apptio planning applications offer the following capabilities:

- <export-populate-line.html>
- <enter-import-line.html>
- <filter-line-item.html>
- <customize-save-share.html>

## Procedure

- **Adding and removing table rows**

  You can add and remove rows from tables by right-clicking on the row and selecting one of the
  available options:

  - Insert Row Above
  - Insert Row Below
  - Delete Row
  - Duplicate Row
- **Table command shortcuts**

  The following keyboard shortcuts are available to navigate most line-item tables:

  - **TAB** - Move to the focus on the page.
  - **SHIFT + SPACE** - Open the cell menu.

    Tip: **TIP**: Use this command to open the various actions in the
    Status page. To select your preferred action, press **Enter**.
  - **SHIFT + F10** - Open the column menu.
  - **SHIFT + TAB** - If you are in the first cell in a table, exit the table. Otherwise, move to
    the previous item in the menu.
  - **TAB** - If you are in the last cell in a table, exit the table. Otherwise, move to the next
    item in the menu.
  - **ENTER** - Expand or collapse a row or click on a cell.

  Tip: See also Apptio keyboard accessibility support.
