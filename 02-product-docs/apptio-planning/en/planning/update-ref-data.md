# Update reference data dimensions

◆ Applies to: ApptioOne Plan

The tasks below can only be performed by users assigned to the Admin or Budget Process Owner
roles. For additional information on roles, see Frontdoor permissions and roles.

## Update reference data

1. In the Costing & Planning Plan navigation menu, select **Configuration >** Reference Data
   > Account.
2. Select ![](../../resources/images/icons/3-dots.png) in the top right corner **>**
   Select **Import >** Set your import options to import the .csv file data.

   ![](../../resources/images/it%20planning_images/import-itp.png)
3. Follow the on-screen instructions to complete the data-import process. Examine and verify your
   imported data.
   - If your data looks correct and you want to make it available in plans, select ![](../../resources/images/icons/3-dots.png)
     **>** select **Publish**.
   - If you see problems with your data and need to troubleshoot before publishing, select ![](../../resources/images/icons/3-dots.png)
     **>** select **Revert**.
