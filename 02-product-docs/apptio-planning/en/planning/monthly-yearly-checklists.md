# Monthly and yearly checklists for administering Planning

◆ Applies to: Planning administration. The tasks below can only be performed by users
assigned to the Admin or Budget Process Owner roles. For additional information on roles, see
Frontdoor permissions and roles.

Use this checklist as a guide for maintaining your Planning solution.

Note: This checklist does not cover all operational elements for every possible scenario. It is
intended to facilitate the operational procedures for a typically configured solution. All checklist
items link to detailed instructions.

## Monthly checklist

- [Import and publish
  actuals](import-publish-actuals.dita "(Opens in a new tab or window)")
- [Create a plan or
  forecast](create-budget-plan.dita "(Opens in a new tab or window)") and notify Department Owners of entry period
- [Integrate with Cost
  Transparency](integrate-ct.dita "(Opens in a new tab or window)")

## Yearly checklist

- [Configure project time settings](../../studio/admin/configure-project-time.dita "(Opens in a new tab or window)") in Costing Standard.
- Update reference data with the new organization information. Built-in dimensions are managed by
  Admins or Budget Process Owners. Many built-in dimensions have dependencies on other dimensions (see
  [Reference data attribute and
  dimensions dependencies](manage-reference-data.html) for information).
  - Update Accounts, Cost Centers, Departments, Location, and Vendors (see [Manage Financial reference
    data](manage-financial-dimensions.dita "(Opens in a new tab or window)")).
  - Update External and Internal Labor Pools, Labor Allocation Rules, Labor Rates, and Roles (see
    [Manage Labor reference
    data](manage-labor-configuration.dita "(Opens in a new tab or window)")).
  - Update Contract types and VAT rates (see [Manage Contract reference data](manage-contract-configuration.dita "(Opens in a new tab or window)")).
  - Update Asset class (see [Manage Asset Class reference data](manage-asset-configuration.dita "(Opens in a new tab or window)")).
  - Update Cost Object Permissions (see [Manage Cost Object Permissions reference data](manage-cost-object.dita "(Opens in a new tab or window)"))
  - If you are supporting multiple currencies, update Actual and Plan Currency rates (see [Manage currency conversion rate
    tables reference data](manage-multi-currency.dita "(Opens in a new tab or window)")).
  - If [Project Financial
    Planning is enabled](pfp/gs-project-financial.dita "(Opens in a new tab or window)"), update Project and Project Group (see [Manage Project reference
    data](pfp/manage-project-configuration.dita "(Opens in a new tab or window)")).
- [Create a plan or
  forecast](create-budget-plan.dita "(Opens in a new tab or window)").
- [Enter financial details
  and adjust baseline values](enter-financial-details.dita "(Opens in a new tab or window)").
- [Open a plan or a
  forecast](open-plan-forecast.dita "(Opens in a new tab or window)").
- Notify Department Owners of a budget entry period (start and end date).
- [Create a labor
  plan](create-labor-plan.dita "(Opens in a new tab or window)").
- [Review, approve, or return
  plans or Cost Objects](review-approve-return.dita "(Opens in a new tab or window)").
- [Finalize a plan or
  forecast](finalize-plan-forecast.dita "(Opens in a new tab or window)").
- [Integrate with Cost
  Transparency](integrate-ct.dita "(Opens in a new tab or window)").
