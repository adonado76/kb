# Actual Summarization

This feature describes how the actuals line items are segregated, based on the columns that are
chosen for summarization.

To see the actuals, navigate to  **Spend Management**  >�  **Transactions**  .

![](../../resources/images/it_planning_images/as-1.jpg)

Now, let us create a Forecast plan starting February 2024 - Actuals are pulled only if the start
period is after the Actuals month.

![](../../resources/images/it_planning_images/as-2.jpg)

Select the new plan. The Expenses Summary page appears with all values from the Actuals page.

![](../../resources/images/it_planning_images/as-3.jpg)

Navigate to  **Company Profile**  .

![](../../resources/images/it_planning_images/as-4.jpg)

This section controls how the financials or the actual could be summarised on the summary page.
Select the fields that you would like to see on the Expenses page and then select  **Save and Exit** .

The expenses page will now show all the dimensions and attributes selected in the above section.
