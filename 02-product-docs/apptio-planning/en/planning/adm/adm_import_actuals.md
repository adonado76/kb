# Import Actuals

Importing of Actuals data is performed by a TBMA (usually once per month) to import
Actual spend data from Costing Standard to Planning. You can use the imported Actuals to form
forecasts, perform variance analysis, and create other comparison views.

The imported Actuals are up to date with the latest reference data changes since Automated Data
Management updates reference data in each actuals period in Spend Management before importing the
Actuals.

If automated data management is enabled in the ApptioOne Planning environment, any changes to
actuals in the Cost Transparency will get automatically imported into Spend Management in the
respective Actuals month. Reference data in the spend management will get automatically updated
before importing the actuals. The status of actuals import is available in Cost Transparency
Integration > Status page.

Since Actuals are now imported automatically, we have removed Import Actuals functionality from
![](../../../resources/images/it%20planning_images/settings-icon_30x27.png)> **Cost
Transparency Integration**.
