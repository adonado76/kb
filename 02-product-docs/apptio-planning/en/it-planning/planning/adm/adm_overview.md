# Overview

Costing & Planning is integrated with the **Automated Data Management** platform
service that enables automated sharing of data from Costing & Planning
Cost (Costing Standard) to Costing & Planning Plan ( Planning), and
brings in the ability to import plan data on-demand from Costing Standard to
a specific plan.

With Automated Data Management integration, the following capabilities are added:

- [Automated Data Management for Reference
  Data](adm_capabilities.html)
- [Automated Data Management for Actuals](adm_capabilities.html)
- [Automated Data Management for Plan Baseline
  Data](adm_capabilities.html)
- [Automated Data Management for
  Platform](../../../data-sharing/home.html)
