# User Impersonation

Note: *Admin or Budget Process Owner roles are required to impersonate.*

User Impersonation allows Admins to temporarily act as another user to:

- Troubleshoot permission or access issues
- Validate what an end user can see or edit
- Confirm configuration changes
- Support users during onboarding or training

Admins can impersonate any user with access to Apptio Planning.

**Impersonating a User**

1. In the Apptio toolbar, select **Profile Settings** → **Impersonate**.
2. Enter the **username** of the user you want to impersonate.
3. Select **Next**.
4. You are now impersonating the selected user.

Select **Settings (Gear icon) → About The Application** to confirm you are logged in
as the impersonated user. The username shown will match the impersonated account and include
an asterisk (\*) to indicate impersonation mode.

**To stop impersonating a user:**

- In the Apptio toolbar, select **Profile Settings** → **Logout**.
- Log back in using your own credentials.
