# Configure Apptio Planning applications

The tasks below can only be performed by users assigned to the Admin or Budget Process Owner
roles. For additional information on roles, see Frontdoor permissions and roles.

Initial configuration includes the following user setup tasks performed by the Admin, and data
import and validation tasks performed by the Admin or Budget Process Owner. Once the initial
configuration is complete, the Admin's work continues with data management.

Required tasks include the following:

- [Configure Cost
  Transparency](configure-ct-support.html) (applies to Planning only)
- [Edit the Company
  Profile](edit-company-profile.html "The Company Profile allows Admin and Budget Process Owner users to configure application-wide settings that customize the display, enable or disable features, and define workflow behavior across Apptio Planning.")
- [Manage users and access
  using Frontdoor](../../frontdoor/home.html)
- [Manage reference data for
  Planning](manage-itfmf-reference.html "◆ Applies to: Planning")(applies to Planning only)
- [Manage
  dimensions](manage-reference-data.html)
- [Manage custom
  dimensions](manage-custom-reference.html "The tasks below can only be performed by users assigned to the Admin or Budget Process Owner roles. For additional information on roles, see Frontdoor permissions and roles.")

Optional tasks include the following:

- [Customize, save, and share
  table layouts](customize-save-share.html)
- [Impersonate a user](impersonate-user.html "The tasks below can only be performed by users assigned to the Admin or Budget Process Owner roles. For additional information on roles, see Frontdoor permissions and roles.")
- [Restrict access to labor
  salary details](restrict-access-labor.html "The tasks below can only be performed by users assigned to the Admin role. See Frontdoor permissions and roles.")
- [Integrate with Cost
  Transparency](integrate-ct.html "If your organization runs both Apptio Costing Standard and an Apptio Planning application, you can integrate them to share data.")
- [Enable Service Demand Planning or Project Financial Planning features](edit-company-profile.html "The Company Profile allows Admin and Budget Process Owner users to configure application-wide settings that customize the display, enable or disable features, and define workflow behavior across Apptio Planning.")
- [Archive, restore, or delete
  plans](manage-plans.html "With the Plans page, you can manage your plans easily and intuitively.")
- [Support multiple
  currencies](support-multiple-currencies.html)
- [Enable custom
  calendars](custom-calendars.html "We support three ways an organization can structure their fiscal calendars:")

Project Financial Planning optional tasks include the following:

- [Manage Project
  reference data](pfp/manage-project-configuration.html)

SEE ALSO:
[Project Financial Planning](pfp/gs-project-financial.html "◆ Applies to: Planning with Project Financial Planning.")

Service Demand Planning optional tasks include the following:

- [Manage Service
  reference data](sdp/manage-service-reference.html)

SEE ALSO:
[Service Demand Planning](sdp/gs-service-demand.html)

After you complete the configuration, your organization can begin the planning or forecasting
process. See [Budget
planning and forecasting](budget-planning-forecasting.html).
