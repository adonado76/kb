# Labor Headcount Targets

Labor Headcount Targets allow you to set and monitor a quantitative goal for headcount
(number of people) across departments. Unlike financial targets (which focus on cost), labor
targets focus on the headcount itself. This ensures workforce planning aligns with
organizational objectives and helps highlight variances between target and planned headcount.

## Who Can Set Targets

- **Admins** or **Budget Process Owners** can set targets for any **department**
  or **group department** within the plan.
- **Group Cost Center Owners** can set targets for their **subordinate
  departments**, allowing for distributed ownership and accountability.

This structure ensures flexibility in how targets are managed across the
organization—either centrally or at different levels of the departmental hierarchy.

## Setting Headcount Targets

1. Navigate to the **Targets** page from the left navigation menu.
2. Select the **Labor** tab.
3. In the table, enter the headcount target for **All Departments**, then allocate it
   down to lower-level departments using the **Department** dropdown.
4. Alternatively, you can begin by entering headcount targets at the **leaf department**
   level and allocate totals upward.
5. The **Remainder** row displays any remaining unassigned amount.

Note: Targets can only be defined for the first year of the plan.

## Viewing and Comparing Headcount Targets

Once targets are established, they appear in several areas of the plan:

- **KPI Comparisons:** Display **Plan vs. Target** and **Variance** metrics,
  allowing you to quickly assess how current headcount plans align with defined headcount
  targets.
- **Summary Page** **→** **Headcount tab:** Compare plan headcount data to
  targets using the **treemap** and **waterfall** charts, which use color coding to
  indicate whether variances are positive or negative.

To view comparisons to headcount targets on the Summary page, ensure the following:

- **Group By:** Department
- **Compare To:** Targets
- **Date Range:** Set to the first year of the plan
- **View:** Headcount
