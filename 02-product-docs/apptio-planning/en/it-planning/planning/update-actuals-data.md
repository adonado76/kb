# Update Actuals in Forecast Plans

Note: *Admin or Budget Process Owner roles are required to update actuals.*

Administrators and Budget Process Owner users can update actual spend data in a forecast plan
without needing to recreate the plan each time new actuals arrive.

Actuals can be refreshed in Forecast plans that are in either the **NEW** or **OPEN**
state.

In the Forecast Plan, Financial Actuals are available on the **Expenses > Summary** tab. Actual
line items are identified by the **Line Item Type** value set to **Actuals**.

## How to Update Actuals

1. Open your forecast plan and navigate to the **Expenses** page.
2. Click the **Ellipses menu**→ **Update Actuals Data**.
3. In the popup window, use the drop-down to select the **months** for which you want to
   update actual values.
4. Click **Update** to refresh the data for the chosen periods.
