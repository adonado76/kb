# A Guide to Apptio Planning’s Navigation

This topic provides an overview of how the Apptio Planning
navigation is organized. The menu items shown in your account depend on your user permissions.

## Top Navigation Bar

The top navigation bar provides quick access to key settings, navigation tools, and plan-level
actions in Apptio Planning:

![image of top navigation menu](../../../../../03-media/apptio-planning/resources/images/it_planning_images/pln-menu.png)

1. **Apptio Logo:** Switch between Apptio
   applications available in your organization. Apptio Planning
   supports common login and user administration through Frontdoor.
2. **Currency:**If multi-currency is enabled, use the currency menu to change your
   view to a different currency. Select **Original** to view and edit values in their
   originally entered currencies.
3. **Bookmarks Menu:** Open or manage saved bookmarks. Available once you create a
   bookmark.
4. **Help:** Open the Help Center, send product feedback, view release notes, or
   access the IBM Community.
5. **Settings:** Access Company Profile, manage users, and configure system settings.
6. **Profile Settings:**Manage your user profile; Admins can impersonate users.
7. **Application Menu:** Switch between Apptio applications
   available in your organization. Apptio Planning supports common
   login and user administration through Frontdoor.
8. **Chevron Icon:**Expand or collapse the left navigation pane.
9. **Plan Menu:**Select and switch between available plans.
10. **Gear Icon:** Configure KPI display settings.
11. **Comments:** View and manage plan-level comments.
12. **Departments:**Filter your view by department hierarchy.
13. **Projects:** Filter your view by specific projects.
14. **Date Range:** Adjust the visible time range for the plan.
15. **Three-Dot Overflow Menu:** Access additional actions, including plan state
    changes (Open, Final), approvals, submissions, or other admin tasks.
16. **Bookmark Icon:** Create a new bookmark for the current page.

## Navigation Sidebar Menu

The left sidebar menu organizes your different areas in your plan and configuration
settings:

![image of side navigation menu](../../../../../03-media/apptio-planning/resources/images/it_planning_images/nav-menu.png)

- **Dashboard:** View an overview of key KPIs, trends, and plan performance metrics.
- **Summary:** Review consolidated financial and headcount data across your plan.
- **Variance Analysis:** Analyze budget vs. actual variances, identify drivers, and add
  commentary.
- **Expenses:** Enter, adjust, and review expenses at the line-item level.
- **Projects:** Manage the list of projects and associated planning details.
- **Status:** Track plan status, submissions, and approvals.
- **Targets:** Define financial and headcount targets by department.
- **Plans:** Create, organize, and manage plans. (Admin users only).
- **Settings:** Adjust plan-level settings and configurations. (Admin users only).
- **Spend Management:** Import, update, and manage actuals data.
- **Change History:** View and audit plan changes in a detailed log (Admin users only).
- **Configuration:** Manage system-wide settings such as dimensions, schemas, and user
  permissions. (Admin users only).
