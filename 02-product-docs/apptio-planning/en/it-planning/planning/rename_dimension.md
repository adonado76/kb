# Rename project-related standard dimensions

## About this task

Important: *Available with the* ***Apptio Planning Standard****subscription*

You can rename the following project-related dimensions in Integrated Investment Planning:

- Project
- Project Group
- Project Labor Role

## Procedure

- **Rename a dimension**

  1. Navigate to
     Configuration > Schema > Standard
     Dimensions.
  2. Select the dimension which you want to rename.
  3. On the dimension name, select ![Edit icon](../../../../../03-media/apptio-planning/resources/images/icons/icon-edit.png).
  4. Enter the new name.

     The dimension gets renamed.

     The new dimension name is displayed in
     the Line Items where the renamed dimension is used as a lookup attribute
- **Restore the default dimension name**

  If you want, you can go back to the default dimension name

  - Select the dimension, then select ![refresh icon](../../../../../03-media/apptio-planning/resources/images/icons/icon-reset.png)Reset.

  The dimension name is reset to the default name.
