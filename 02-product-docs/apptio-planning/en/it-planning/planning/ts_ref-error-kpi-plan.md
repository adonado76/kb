# Troubleshoot: KPI or values in a plan display the #REF error

## Symptom

The KPI or values in a plan display the #REF error.

## What's going on

The #REF error usually occurs when there are missing currency conversion rates.

[Learn more about importing
and publishing currency rates](import-publish-currency.html "The tasks below can only be performed by users assigned to the Admin or Budget Process Owner roles. For additional information on roles, see Frontdoor permissions and roles.")

## Solution

Ensure that the currency rate tables have a valid currency conversion for all affected
timeframes.
