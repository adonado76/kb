# Bookmark a page

Bookmarks allow you to quickly return to frequently used pages in Apptio Planning
without needing to reapply filters, layouts, or settings each time. Any user can create
bookmarks, and you can also set one as your default landing page, so it opens automatically when
you log in.

## What a Bookmark Saves

When you create a bookmark, the following information is captured:

- Department, Project and Date Range selections.
- The current tab (if the view contains multiple tabs).
- Table layout changes (column order, size, visibility).
- Applied filters.

Note: Bookmarks are personal to each user and cannot be shared.

## Create a Bookmark

1. Navigate to the page you want to bookmark.
2. Select the **Bookmark icon** in the upper right**.**
   1. The icon appears filled if the current page is already bookmarked.

   ![image of bookmark icon](../../../../../03-media/apptio-planning/resources/images/it_planning_images/create-bookmrk.png)
3. In the **Create New Bookmark** dialog, enter a name for the bookmark.
4. (Optional) Select the option to set this page as your home page.
5. Select **Save**.

## Open a Bookmark

1. Select **Bookmarks** in the top menu bar, then choose the bookmark you want.
2. The **Bookmarks drop-down** appears only when you have at least one saved bookmark.

![image of opening bookmark](../../../../../03-media/apptio-planning/resources/images/it_planning_images/open-bookmrk.png)

## Set a Default Landing Page

You can choose any saved bookmark as your default landing page. This page will open
automatically each time you log in.

1. Open **Bookmarks > Bookmarks Manager** from the top menu bar.
2. Select the bookmark you want to set as your home page and click the **Home** icon.
3. Click **Save**.

![image of default bookmark](../../../../../03-media/apptio-planning/resources/images/it_planning_images/bookmrk-def.png)

## Manage Bookmarks

Use the **Bookmarks Manager** to edit, reorder, or delete bookmarks.

1. Select **Bookmarks > Bookmarks Manager**.
2. From here you can:
   1. **Edit** – Select a bookmark name to rename it.
   2. **Reorder** – Select and drag the reorder icon to change the order.
   3. **Delete** – Select the delete icon to remove a bookmark.

   Tip: Create bookmarks for commonly used filtered views (e.g., a
   specific department or forecast period) to save time and streamline your workflow.
