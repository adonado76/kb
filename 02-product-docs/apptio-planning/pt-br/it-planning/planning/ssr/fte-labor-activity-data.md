# Dados de atividade de mão de obra FTE

Atualmente, os dados disponíveis no site Apptio BI suportam apenas dados de horas, mas para o IIP o planejamento da atividade de trabalho é ativado usando tanto a unidade de horas quanto a de FTE. Esse recurso permite que os usuários criem relatórios de alocação usando FTE para que possam analisar o relatório de alocação usando a capacidade, os detalhes de utilização e os dados de alocação mensal por recurso.

Crie uma visualização conforme descrito [aqui](create-visualisation-for-report.html "Conclua as etapas a seguir para criar uma visualização em Apptio BI usando dados do Planning. Você pode personalizar a visualização de várias maneiras, inclusive selecionando o tipo de visualização (gráfico de barras, gráfico de linhas e outros), filtrando dados e selecionando um período de tempo."). Selecione a Fonte de dados como Trabalho de planejamento.

![Adicionar visualização a um novo relatório](../../../resources/images/it%20planning_images/fte-metric_229x465.png)

Uma nova opção Quantidade (FTE) foi adicionada ao campo Métricas.

Insira dados em todos os outros campos, conforme necessário. O relatório de amostra aparece como mostrado.

![Relatório de amostra](../../../resources/images/it%20planning_images/fte-bi-report.png)

Você pode personalizar a visualização de várias maneiras, selecionando o tipo de visualização (gráfico de barras, gráfico de linhas e outros), filtrando dados e selecionando um período de tempo.
