# Personalize, salve e compartilhe layouts de tabela

A tabela de despesas na nova visualização oferece recursos aprimorados de personalização, incluindo a capacidade de personalizar a estrutura da coluna, o agrupamento, os filtros e salvar as alterações como um layout. Os usuários individuais podem criar até 50 layouts privados, enquanto os usuários administradores podem criar até 50 layouts públicos. Isso representa um aumento em relação ao limite anterior de 30 layouts em configurações públicas e privadas no Expenses Legacy View.

Em uma versão futura, ainda este ano, os usuários também poderão sincronizar os layouts existentes do Expenses Legacy View com o New View, garantindo a continuidade e a facilidade de transição.

## Personalizar um layout de tabela

Todos os usuários podem personalizar até 50 layouts de tabela. Além disso, os administradores e os responsáveis pelo orçamento podem personalizar o layout padrão para todos os usuários da organização.

1. Navegue até **Planning** > **Despesas** no menu de navegação esquerdo.
2. Selecione o plano apropriado no menu suspenso **Plan (Plano** ) no canto superior esquerdo.
3. Selecione a guia específica em que deseja personalizar o layout da tabela.
4. Opcionalmente, execute várias opções de personalização compatíveis para personalizar o layout da tabela, como
   - **Reordenar colunas** : Selecione e arraste um título de coluna. À medida que você arrasta a coluna, uma barra vertical aparece, indicando o local de destino.
   - **Mostrar ou ocultar colunas** : Selecione o menu Columns (Colunas) no menu direito da tabela e, em seguida, marque a caixa de seleção das colunas a serem mostradas ou desmarque a caixa de seleção das colunas a serem ocultadas.
   - **Classificar colunas** : As colunas podem ser classificadas selecionando-se o cabeçalho da coluna. Um indicador de classificação é exibido ao lado do título da coluna. Você pode alterar a direção da classificação ou remover a classificação selecionando o cabeçalho da coluna. As colunas agrupadas são classificadas por padrão.
   - **Agrupar ou desagrupar dados de coluna** : Selecione Group Column ou Group Columns e, em seguida, marque a caixa de seleção das colunas a serem agrupadas ou desmarque a caixa de seleção das colunas a serem desagrupadas. Os grupos podem ser aninhados e as informações agrupadas podem ser expandidas ou recolhidas. Se os dados tiverem sido agrupados anteriormente por outra dimensão, a seleção de Group Column removerá esse agrupamento e aplicará o novo.
   - Coluna de **pinos** no lado esquerdo ou direito da tabela.
   - **Aplicar filtros** em várias colunas e atributos.

## Salvar, aplicar, excluir ou compartilhar um layout de tabela personalizado

Salve até 50 layouts personalizados para reutilização. Cada layout é exclusivo da guia ativa (e, se disponível, da subguia). No entanto, os layouts personalizados não são específicos do plano ativo ou do centro de custos.

Criar layout
:   1. Personalize o layout da tabela de acordo com os requisitos, seguindo as instruções da seção anterior.
    2. Selecione ![ícone de mesa](../../resources/images/it%20planning_images/layout-icon_28x20.png) no menu Expenses Table (Tabela de despesas).
    3. No menu de layout, selecione **Add New Layout (Adicionar novo layout** ).
    4. Digite o nome do layout.
    5. Se você quiser salvar os filtros aplicados também no layout, marque a caixa de seleção **Apply Filter Selection (Aplicar seleção de filtro)** e selecione **Save (Salvar** ).
    6. O layout recém-criado aparecerá em **My Layouts (Meus layouts** ).
    7. Passe o cursor sobre o botão **Layout** para ver o layout selecionado no momento.

Renomear layout
:   1. Selecione ![ícone de mesa](../../resources/images/it%20planning_images/layout-icon_28x20.png) no menu Expenses Table (Tabela de despesas).
    2. No menu My Layouts, selecione o ícone de lápis do layout que deseja renomear.

       ![painel](../../resources/images/it%20planning_images/tl-edit.png)
    3. O pop-up Renomear layout é exibido.

       ![visualização resumida](../../resources/images/it%20planning_images/tl-edit-1.png)
    4. Atualize o nome do layout e selecione **Salvar**.

       Nota:

       O ícone **Salvar** permanece desativado até que seja feita uma alteração no layout.

Editar layout
:   1. Para personalizar um layout, selecione o layout que deseja alterar.
    2. Faça as alterações necessárias e selecione **Salvar**. Uma mensagem de confirmação "*Success: Save my Layouts* " é exibida e o layout atualizado é carregado.

       ![salvar no painel](../../resources/images/it%20planning_images/tl-edit-2.png)

Clonar layout
:   Observação: essa opção está disponível apenas para layouts públicos. O ícone **Clone** permanece desativado até que seja feita uma alteração no layout.

    1. Selecione ![ícone de mesa](../../resources/images/it%20planning_images/layout-icon_28x20.png) no menu Expenses Table (Tabela de despesas).
    2. No menu **Public Layouts (Layouts públicos** ), selecione o "Default Layout" (Layout padrão) e selecione o ícone **Clone** ![ícone de cópia](../../resources/images/it%20planning_images/tl-cln-icon_20x20.png).

       ![layouts de clones](../../resources/images/it%20planning_images/tl-clone.png)
    3. O pop-up Layout de cópia é exibido. Digite um nome para o layout e selecione **Salvar**.

       ![novo layout](../../resources/images/it%20planning_images/tl-copy.png)
    4. O layout clonado é adicionado à seção My Layouts, conforme mostrado.

       ![clonar](../../resources/images/it%20planning_images/tl-cln-1.png)

## Excluir layout

1. Selecione ![img](../../resources/images/it%20planning_images/layout-icon_28x20.png) no menu Expenses Table (Tabela de despesas).
2. No menu Meus layouts, selecione o ícone **Excluir** do layout que deseja excluir.

   ![excluir layout](../../resources/images/it%20planning_images/tl-del.png)
3. O layout é excluído.

   Observação: Os layouts públicos não podem ser excluídos.

## Alterações no layout da redefinição

- Para redefinir o layout completo, selecione a opção **Reset columns to default layout (Redefinir colunas para o layout padrão** ) no menu ![img](../../resources/images/it%20planning_images/layout-icon_28x20.png) .

  ![restaurar layout](../../resources/images/it%20planning_images/tl-reset-default.png)
- Para desfazer as alterações feitas em um layout específico (antes de salvá-lo), selecione o ícone ![ícone de redefinição](../../resources/images/it%20planning_images/tl-reset-icon_26x22.png) para esse layout.

  ![ícone de redefinição](../../resources/images/it%20planning_images/tl-reset.png)

  É exibida uma mensagem de erro, conforme mostrado. Selecione **Okay** para continuar com a redefinição.

  ![aviso](../../resources/images/it%20planning_images/tl-rst-error.png)

## Gerenciar o acesso ao layout

O acesso ao layout está ativado somente para administradores. Ele permite que o administrador torne um layout público, para que fique disponível a todos os usuários da organização.

1. Selecione ![img](../../resources/images/it%20planning_images/layout-icon_28x20.png) no menu Expenses Table (Tabela de despesas).
2. No menu suspenso, selecione a opção **Acesso ao layout**. A seguinte janela pop-up é exibida.

   ![layouts](../../resources/images/it%20planning_images/tl-access.png)
3. Alterne o botão de alternância para o layout que precisa ser tornado **público** e selecione **Aplicar**.

   ![layouts](../../resources/images/it%20planning_images/tl-toggle.png)
4. O layout selecionado é removido da seção **My Layouts (Meus layouts** ) e aparecerá na seção **Public Layout (Layout público** ). Quando o layout é marcado como "Público", ele não pode ser excluído.

   ![layouts de clones](../../resources/images/it%20planning_images/tl-publay.png)

   Para tornar o layout privado ou excluir um layout público, alterne o botão de alternância novamente e selecione **Aplicar**.

## Sincronizar layouts herdados

Os usuários agora podem transferir seus layouts de tabela de despesas da visualização antiga para a nova visualização com um único clique, eliminando a migração manual e economizando tempo.

Para sincronizar layouts, basta clicar no ícone Layouts na Nova visualização e selecionar "Sync Legacy Layouts" Isso copiará automaticamente seus layouts da visualização herdada para a nova visualização. Observe que você precisará sincronizar os layouts de cada guia da tabela de despesas individualmente para concluir a migração da visualização herdada para a nova visualização.

![layouts de sincronização](../../../../../03-media/apptio-planning/resources/images/it_planning_images/release-notes/3.78_rn.jpg)

Os layouts migrados do Legacy View para o New View ficarão visíveis em My Layouts (Meus layouts) e Public Layouts (Layouts públicos) para layouts privados e públicos, respectivamente. O nome do layout migrado será sufixado com "\_migrated". Veja a captura de tela de referência abaixo.

![layouts migrados](../../../../../03-media/apptio-planning/resources/images/it_planning_images/release-notes/378rn-1.jpg)

Se você continuar a fazer atualizações nos layouts da visualização legada depois de migrá-los para a nova visualização, precisará usar a opção Sincronizar layouts legados na nova visualização para trazer as atualizações para a nova visualização.

Nota:

- O layout é consistente em todos os planos e é específico para uma página.
- O layout pode salvar os filtros aplicados, o agrupamento de colunas e as colunas que estão visíveis na tabela.
- Os usuários administradores e BPO podem editar e salvar alterações no layout padrão.
- Os usuários administradores/BPO também podem compartilhar seus layouts salvos de forma privada, tornando-os "públicos"
- Os usuários que não são administradores/BPO, como os usuários CCO, podem usar o layout padrão (deve mostrar as últimas alterações de layout salvas) definido pelos administradores, mas não podem salvar suas alterações de layout nele. Em vez disso, eles podem criar um novo layout privado e salvar a personalização feita no layout público.
- Os usuários que não são administradores/BPO, como os usuários CCO, podem criar até 50 layouts privados.
- Ao escolher salvar um novo layout privado, há uma opção de caixa de seleção para salvar o layout com filtros (se aplicados) e, se essa opção for desmarcada, os filtros aplicados não serão saved.Plan comparações podem ser salvas no layout.
- As comparações de planos podem ser salvas no layout.
- Não é possível salvar o seletor de períodos.
- Quando o usuário faz algumas novas alterações no layout, mas não as salva, um símbolo de asterisco é exibido ao lado do layout ativo no momento para que o usuário saiba que foram feitas algumas alterações no layout, mas que elas não foram salvas. O usuário pode selecionar Revert para descartar as alterações recentes ou selecionar Save para salvá-las no layout em que o usuário está no momento. O usuário também pode selecionar Clone para salvar essas novas alterações como um novo Layout. Se nenhuma dessas opções for escolhida, a vida útil das alterações de layout recém-feitas durará até que o usuário atualize a página ou faça logout. Se o usuário tentar mudar para outra guia ou selecionar outro layout com alterações não salvas no layout atual, será exibida uma mensagem de aviso descrevendo a perda de alterações não salvas no layout.
- Se a tabela não estiver mostrando colunas de período (meses, trimestres, anos), siga as etapas abaixo para torná-las visíveis.
- 1. Selecione Grouping (Agrupamento) no menu da tabela.
  2. Certifique-se de que Ano, Mês(es), Trimestre(s) esteja selecionado.
  3. Se as colunas de período ainda não estiverem visíveis, selecione o botão Layout acima da tabela e selecione **Redefinir colunas para o layout padrão**.
  4. Agora, todas as colunas de período devem estar visíveis.
  5. Navegue até o botão Layout > Layout público > Layout padrão e clique no botão Salvar. Essa etapa garantirá que as alterações sejam salvas e que as colunas de período se tornem visíveis por padrão.

  Essas etapas devem tornar as colunas de período visíveis na exibição de tabela.
