# Ajustar valores de linha de base

Observação: as funções de administrador ou de proprietário do processo orçamentário são necessárias para ajustar os valores de linha de base.

Depois de criar um plano, talvez seja necessário ajustar os valores de linha de base antes de tornar o plano acessível aos proprietários do orçamento. Esses ajustes podem ser aplicados globalmente em todos os departamentos e contas ou direcionados a departamentos e contas específicos.

## Como ajustar os valores da linha de base

1. Abra o plano e navegue até a página **Expenses (Despesas** ) e selecione a guia **Other (Outros** ).
2. No menu **Ações**, selecione **Ajustar valores...**
3. Selecione um **tipo de ajuste** :
   1. **Percentage (Porcentagem** ) - Ajusta os valores em uma porcentagem do valor original.
   2. **Absoluto** - Ajustar por um valor fixo de moeda (disponível somente quando várias moedas não estiverem ativadas).
4. No **valor do ajuste** :
   1. Para **Porcentagem**, digite um número inteiro (por exemplo, 10 para +10%, -5 para -5%).
   2. Para **Absoluto**, insira um valor em moeda.
5. Use as opções **Apply To (Aplicar a** ) para escolher o escopo do ajuste por Departamento e Conta.
6. Selecione um **período** **inicial** e final no **Intervalo de Período**.
7. Clique em **Adjust (Ajustar** ) para aplicar a alteração.

## Observações importantes

- O plano deve estar no estado **Novo** para aplicar ajustes.
- Você ainda pode ajustar manualmente as linhas individuais posteriormente, navegando em objetos de custo específicos e editando diretamente os valores mensais.
- **Não há como desfazer** - os ajustes são permanentes depois de aplicados.
