# Visão geral

O Costing & Planning está integrado ao serviço da plataforma **Automated Data Management**, que permite o compartilhamento automatizado de dados do Costing & Planning
Cost (Costing Standard) para o Costing & Planning Plan ( Planning), e traz a capacidade de importar dados do plano sob demanda do Costing Standard para um plano específico.

Com a integração do Automated Data Management, os seguintes recursos são adicionados:

- [Automatizado Data Management para dados de referência](adm_capabilities.html)
- [Automatizado Data Management para reais](adm_capabilities.html)
- [Automatizado Data Management para dados de linha de base do plano](adm_capabilities.html)
- [Automatizado Data Management para plataforma](../../../data-sharing/home.html)
