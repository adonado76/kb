# Atualizar tabelas de itens de linha

## Antes de começar

A maioria das tabelas com itens de linha nos aplicativos Apptio Planning oferece os seguintes recursos:

- <export-populate-line.html>
- <enter-import-line.html>
- <filter-line-item.html>
- <customize-save-share.html>

## Procedimento

- **Adição e remoção de linhas de tabela**

  Você pode adicionar e remover linhas de tabelas clicando com o botão direito do mouse na linha e selecionando uma das opções disponíveis:

  - Inserir linha acima
  - Inserir linha abaixo
  - Excluir linha
  - Linha duplicada
- **Atalhos de comandos da tabela**

  Os seguintes atalhos de teclado estão disponíveis para navegar na maioria das tabelas de itens de linha:

  - **TAB** - Ir para o foco na página.
  - **SHIFT + ESPAÇO** - Abre o menu da célula.

    Dica: **DICA** : use esse comando para abrir as várias ações na página Status. Para selecionar sua ação preferida, pressione **Enter**.
  - **SHIFT + F10** - Abre o menu da coluna.
  - **SHIFT + TAB** - Se você estiver na primeira célula de uma tabela, saia da tabela. Caso contrário, vá para o item anterior no menu.
  - **TAB** - Se você estiver na última célula de uma tabela, saia da tabela. Caso contrário, passe para o próximo item do menu.
  - **ENTER** - Expandir ou recolher uma linha ou clicar em uma célula.

  Dica: Consulte também o suporte à acessibilidade do teclado Apptio.
