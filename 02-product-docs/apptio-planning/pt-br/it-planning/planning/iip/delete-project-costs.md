# Delegar custos do projeto

Importante: *Disponível com a* *assinatura* ***do Apptio Planning Standard***

No caso de projetos com várias fases (por exemplo, construção/execução), delegar despesas a centros de custo específicos ajuda a monitorar com eficácia o desempenho financeiro de cada fase do projeto. Os proprietários do projeto podem delegar a depreciação do ativo a um centro de custo diferente, bem como delegar a despesa do contrato a um centro de custo diferente a partir de uma data especificada. Para saber mais, consulte os tópicos abaixo:

[Custos de ativos delegados](asset-depreciation-cost-center.html)

[Custos do contrato de delegado](delegate-contract-costs.html)
