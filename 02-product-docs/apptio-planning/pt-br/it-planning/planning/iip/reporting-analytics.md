# Relatórios e análises

Importante: *Disponível com a* *assinatura* ***do Apptio Planning Standard***

Em Integrated Investment Planning, os projetos são expostos como dimensão nas linhas de despesas Resumo, Trabalho, Atividade de trabalho, Contratos e Ativos, de modo que as informações do projeto estejam disponíveis nas fontes de dados de Apptio BI para Planejamento. Você pode criar relatórios de projeto personalizados usando o site Apptio BI.

Abaixo, você pode encontrar alguns exemplos de relatórios que podem ser criados:

- Relatório sobre os custos do projeto e do departamento.
- Rollup e relatório de custo de Run and Grow para toda a organização ou para um projeto individual.
- Criar relatórios para gastos com Opex e Capex em projetos.
- Alocação interna e externa de mão de obra do projeto.
- Alocação de mão de obra para um projeto por função de mão de obra do projeto.

Separadamente ou combinados.

Para saber mais sobre Apptio BI, consulte [Planejamento em Apptio BI](../ssr/it-planning-data-self-service-reporting.html) e [Sobre Apptio BI](../../../apptio_bi/home.html).
