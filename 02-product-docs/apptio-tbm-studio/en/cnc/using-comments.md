# Using comments

Comments and Collaboration feature allows you to engage in conversations with others about
reports. These conversations might include discussions about variances analysis, project status,
possible initiatives for savings, outliers, trends, missing data, next steps, and so on.

Enter a comment

1. To add a comment to a report, click the Comments ![more](../../../../03-media/apptio-tbm-studio/resources/images/cnc/comments_icon.jpg) icon.

   ![Comments](../../../../03-media/apptio-tbm-studio/resources/images/cnc/addcomment.jpg)
2. In the unlabeled box in the Comments panel, enter your comment, then click  Submit  .

Reply to a comment

To respond to a comment, click Reply , enter your response, then click Reply .

To notify a person about the comment, mention them using the @<username> convention.

![Reply Comments](../../../../03-media/apptio-tbm-studio/resources/images/cnc/comments_reply.jpg)

If you do not wish to reply, you can start a new comment thread.

Edit and delete comments

To edit or delete a comment, click the vertical ellipsis associated with the comment, then click
Edit comment or Delete thread .

![Edit Comments](../../../../03-media/apptio-tbm-studio/resources/images/cnc/comments_edit.jpg)

You can only delete your own comments. Admins can delete anyone's comments. Notifications are
not sent when comments are edited or deleted.

Disable notifications

Email notifications are enabled by default to alert users of mentions and responses on comment
threads. In-product notifications are not supported. However, if you want to disable notifications,
contact your Apptio Support representative.
