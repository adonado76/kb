# Comments and Collaboration Using tasks

Applies to : Costing Standard from 12.11.5 and later

The tasks feature allows users to have actionable conversations by enabling users to create
comments and assign specific users tasks to complete. Tasks can be assigned to one individual at a
time and can be used on private or public threads.

Assign a task

To start a task, add a comment ![comment](../../../../03-media/apptio-tbm-studio/resources/images/cnc/comments_icon.jpg) and determine if you
want the comment private or public. For more information, see [Using comments](using-comments.html)  and  [Private Commenting Feature](cnc-private-commenting.html) .

Enter your comment and type  @  to add a user.

A checkbox appears to assign a user and convert the comment to a task.

Select the  Assign to @  user 
 checkbox.

Select  Assign  .

The user receives an email notification informing them a task is assigned.

Reassign a task 

1. To reassign a task, select  Reply 
   ![arrow](../../../../03-media/apptio-tbm-studio/resources/images/cnc/reply_icon.jpg) in an existing thread
   and type  @  to add a user. A checkbox appears to assign the user and convert the comment
   to a task.
2. Select the  Aassign to @  user 
    checkbox.
3. Select  Assign  .

The user receives an email notification informing them a task is assigned.

Resolve a task

Only assignees or assignors can resolve a task.

To resolve a task, do one of the following:

If you are the assignee, select  Resolve 
![tick](../../../../03-media/apptio-tbm-studio/resources/images/cnc/resolve_icon.jpg) at the start of the
thread.

If you are the assignor, select ![more](../../../../03-media/apptio-tbm-studio/resources/images/cnc/cnc_more_options_icon.jpg) at the start of the
thread, and then select  Mark as resolved  .

Delete a task

To delete a task, select ![more](../../../../03-media/apptio-tbm-studio/resources/images/cnc/cnc_more_options_icon.jpg) at the start of the
thread.

Select  Delete thread  .

Users who are not the assignee or the assignor can also delete their comments in the task's
thread by selecting ![more](../../../../03-media/apptio-tbm-studio/resources/images/cnc/cnc_more_options_icon.jpg) and then  Delete comment  .
