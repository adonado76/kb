# Edit user properties

◆ Applies to: TBM Studio 12.0 and later.

When your Apptio Admin created your Apptio account, they entered your user properties including:

- Email — Your application user name. Use this name to log in to the application.
- Full Name — Your first and last name. This name is displayed in the user list on the
  Administration tab.
- Password — Your application password for logging in to the application.
- Current Role — Your role determines the actions you can perform in the application. If
  your Apptio Admin has enabled Available Roles for you, you can assign yourself to a different
  role.
- Currency — The default currency used in reports.
- Locale — The default locale used to determine the format of the numbers in reports.

## Display user properties

To display your user properties, click your user name in the upper-right corner of the Global
header and click My Account.

Change your password

1. In the Change User Properties dialog, in the Old Password field, enter your
   current password.
2. Enter your new password in the New Password field and Confirm field.
3. Click OK.

## Change your role

If your Apptio Admin has enabled Available Roles for you, you can change your role by clicking
the down arrow in the Current Role field and then by selecting a role. For a description of
the roles, see Defining Roles in the *Administrator Guide*.

Note: When you change roles, you
must refresh your browser for the role to take effect.

## Select a currency

(Applies to: v12.1, v12.2+.4). If your company works with more than one currency, you can select
a default currency for reports. The currency controls the currency symbol (e.g.: €/$) and the
three-letter international currency code (e.g.: EUR/USD) displayed with values.

## Select a locale

The locale determines the format of the numbers. You can choose from a wide range of locales. The
locale is independent of the currency. Some examples of locales:

- United States: 12,345.67
- France: 12 345,67
- Germany: 12.345,67
