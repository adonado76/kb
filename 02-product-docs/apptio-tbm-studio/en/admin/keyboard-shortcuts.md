# Keyboard shortcuts

◆ Applies to: TBM Studio 12.0 and later.

The following keyboard shortcuts are available.

- Ctrl+Alt+T — Switch in and out of Studio mode ([learn more](navigate-tbm-studio.html "(Opens in a new tab or window)"))
- Ctrl+F1 — Toggle the TBM Studio tabs/ribbon mode
- Alt+N — Create a new document
- Alt+O — Check out a document ([learn more](bp-check-out.html "(Opens in a new tab or window)"))
- Ctrl+S, Meta+S — Save
- Alt+K — Check in a document ([learn more](bp-check-out.html "(Opens in a new tab or window)"))
- Ctrl+Alt+Z — Revert a document ([learn more](bp-check-out.html "(Opens in a new tab or window)"))
- Shift+F9 — Update a project
- F9 — Update a document
- Alt+Escape, Meta+Escape — Close tab
- Ctrl+Z — Undoes all changes since the last save ([learn more](bp-check-out.html "(Opens in a new tab or window)"))
- Ctrl+Y — Redo ([learn
  more](bp-check-out.html "(Opens in a new tab or window)"))
