# Review error messages

Applies to: TBM Studio 12.0 and later.

You must have the default role of Power User or
Admin, or a custom role with appropriate permissions, to see the error
messages. To view error messages, click Errors on the Project tab. The
application displays the errors in the window shown in the following image:

![](../../resources/images/studio_images/studioimages/studio_project%20errors_project%20errors.png)
