# What you can do with TBM Studio

TBM Studio includes:

- Gather the right data, intelligently transform it, and enhance it over time. See [About Data Studio](../data_studio/data-transform-chapter-title-page.html "Applies to: TBM Studio 12.0 and later")
- Visually customize allocations and configure a flexible cost model. See [About Model Studio](../model_studio/models-chapter-title-page.html "Applies to: TBM Studio 12.0 and later")
- Customize existing reports or create your own. See [Introduction to Report Studio](../reports/intro_report_studio.html "Applies to: TBM Studio 12.0 and later")
