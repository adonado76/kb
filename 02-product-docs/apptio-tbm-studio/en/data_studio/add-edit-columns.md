# Add and edit columns

Applies to: TBM Studio 12.0 and later. To add new columns or edit
existing columns in a table, add a Formulas step to the transform pipeline.
Columns you add are displayed at the top of the list of columns in the Edit pane. Columns that are
part of the original table are displayed at the bottom of the list of columns in the Edit
pane.

## Add a column

1. If you have not done so already, add a Formulas step to the data transform.
2. Click Add a new column.
3. If needed, change the column type by clicking the Type field.
4. Enter a name for the column.
5. Enter a formula for the column.
6. Click Save.

## Edit a column

1. Use the Select column to edit drop-down list to select the column to
   edit.
2. Make the edits.
3. Click Save.

## Delete a column

To delete a column, point to the first column in the Edit pane and click the Delete icon ![](../../resources/images/studio_images/studioimages/icons/delete2.png).

## Filter the Edit pane

You can filter the Edit pane using the search fields at the top of each column.

To filter by column type (key, text, number, date), click a type icon in the search field of the
Type column.
