# Model Views Overview

The Model Overview Reports provide a comprehensive view of cost flows within the model, enabling
users to visualize and understand cost allocations. This report is designed for Admin and Power
Users, offering a transparent and detailed insight into cost fallout.

![](../../../../../03-media/apptio-tbm-studio/resources/images/ct_images/ct-model-view.png)

## Use Cases

This report solves the following use cases:

- Analyzing cost allocations to understand how costs are distributed across different objects
- Providing a clear and transparent view of cost flows to stakeholders
- Identifying areas for cost reduction and optimization
- Using cost allocation data to inform budgeting and forecasting decisions

## Personas

- Admin
- Power Users

## Questions answered

- How are costs allocated across different objects?
- What is the cost fallout for each object?
- How do costs flow between different objects?
- What are the key cost drivers in the model?
- How can costs be optimized and reduced?
