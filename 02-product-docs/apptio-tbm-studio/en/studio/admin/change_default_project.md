# Change the default project

◆ Applies to: Apptio TBM Studio v12.3.1 and later.

Specify your domain's default project.

1. On the **Project** tab, in the **Domain** group, click
   **Domain Settings**. The Tenant Settings dialog box appears.
2. In the **Default Project** field, select the project you want, then click
   **OK**.
