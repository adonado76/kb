# Configure projects

**Applies to**: TBM Studio 12.0 and later. This article describes the essential tasks you can
perform to configure projects.

There are two types of projects:

- **Custom (Platform)**: Use a blank project to build the data sets, metrics, models, and
  reports.
- **Costing Standard**: Use a Costing Standard project to provide an IT service with cost
  modeling, cost optimization analysis, and budgeting and forecasting capabilities. The results from a
  service costing project are often used to cost services in the Billing application.
