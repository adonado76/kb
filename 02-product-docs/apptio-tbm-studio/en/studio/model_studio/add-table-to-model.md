# Add a table to the model

**Applies to**: TBM Studio 12.0 and later

If you want a table to be part of the model, add a **Model** step to the table transform.

![](../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/model.png)

## Add a Model step

1. Check out the table.
2. Add a new step, and select Model as the type of step you want to add.
3. Save the changes.

After you add the **Model** step, the table is displayed in a single table view:

![](../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/studioimages/cost-source-model.png)
