# Unused Metrics

This feature provides automatic recommendations on unused calculated metrics, offering
visibility into user-created metrics that are not utilized in other metrics or reports. The system
will automatically check in changes and mark issues as resolved.

Navigate to **TBM Studio** > **Recommendations** tab > **Unused Metrics** and then
select **Troubleshoot All Identified Problems**.

![img](../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/unusedmetrics-1.png)

Switch to **Development** workspace and select **Next**.

![](../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/umetrics-2.png)

Click **Next** icon.

![](../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/unused-metrics.png)

Select **Delete Metric** button. The unused metric is deleted.

![](../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/umetric-del.png)

Select **Next**, and then select **Checkin** in the last page.

![](../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/umetricdeleted.png)
