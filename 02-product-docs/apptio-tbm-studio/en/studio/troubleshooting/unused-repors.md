# Unused Reports (BETA)

Note:

It is required to **turn off** the Auto Calculate feature before using this
workflow to deactivate multiple reports at once.

.

![Auto Calculate](../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/auto-calc.png)

If you receive an Unused Reports Recommendation, navigate to  **TBM Studio**  > **Recommendations**  tab >  **Unused Reports**  , and then select  **Troubleshoot All
Identified Problems**  .

![Unused Reports](../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/unused-reports.png)

Switch to  **Development**  workspace and select ![Next Arrow](../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/next-arrow.png) .

![Environment ](../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/unused-reports-development-environment.png)

You can see the issue description and what actions will be taken to resolve it. You can view the
list of reports that have been unused recently. It is recommended to review and deactivate reports
that are no longer needed by this workflow to avoid unnecessary calculations.

![Deactivate Unused Reports](../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/deactivate-unused-reports.png)

1. Select the  **Fix**  button to deactivate individually.
2. To deactivate multiple reports simultaneously, select the desired reports and then select
   **Deactivate All Selected Reports**.
3. Select **Also hide in report collections** option to hide the deactivated reports from the
   report collections view.

   ![Deactivated Reports](../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/deactivated-reports.png)
4. Select ![Next Arrow](../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/next-arrow.png) , and then select  **Checkin** .

![](../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/checin-unused-reports.png)
