# PeriodsInYear function

**Applies to**: TBM Studio and later

Returns the number of periods in the year.

## Where to use

This function can be used in:

- Data sets
- Formula columns in report tables
- Dynamic text

## Syntax

`PeriodsInYear()`

## Arguments

None

## Return type

String

## Example

Given a 13-period project, and a period set to P10 FY2013, the following function returns the
result shown below:

| Function | Return |
| --- | --- |
| PeriodsInYear() | 13 |
