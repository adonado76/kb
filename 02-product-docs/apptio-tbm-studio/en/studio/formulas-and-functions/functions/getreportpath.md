# GetReportPath function

**Applies to**: TBM Studio 12.0 and later

Returns the path of the current report.

## Where to use

This function can be used in:

- Dynamic text

## Syntax

`GetReportPath()`

## Return type

Text

Example:

`GetReportPath()`
