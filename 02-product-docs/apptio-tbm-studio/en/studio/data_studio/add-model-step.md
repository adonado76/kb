# Model data

**Applies to**: TBM Studio 12.0 and later

When you want to add a table to a model, add a model step to the table transform pipeline.
Clicking the model step displays a single table view of the model as shown below. In this view, you
can add drivers to the table, and allocate value from the table to other tables. For information on
modeling data, see [Model Studio: Calculate cost and other metrics](../model%20studio/models-chapter-title-page.htm "(Opens in a new tab or window)").
