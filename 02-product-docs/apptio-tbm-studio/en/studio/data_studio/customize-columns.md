# Customize columns

**Applies to**: TBM Studio 12.0 and later

You can customize the columns in the table displayed when you click the **Table** step in a
transform pipeline.

## Freeze a column

In large tables, you can freeze a column to prevent it from scrolling out of view. To freeze a
column, right-click the column header, click **Formatting**, and click **Freeze Column** from
the pop-up menu. The application moves the column to the far left of the table and freezes the
column in that position. As you scroll the table, the frozen column stays in place.

To unfreeze the column, right-click the column header, click **Formatting**, and click
**Unfreeze Column** from the pop-up menu.

## Set the background color

You can set the background color for each column in a table. To set the color, right-click in the
column heading, click **Formatting**, and click **Set Background Color** from the pop-up
menu.

## Show unique and duplicate values

To see a list of unique values in a column, right-click in the column header and click **Show
Values**. The values are displayed in a pop-up window. The list includes one entry for each unique
value in the column and indicates the number of times the value occurs.

## Set the width of a column

You can set the width of a column by performing one of the following actions:

- Drag the column border in the column header.
- Right-click in the column header, click **Formatting**, click Set Width, and enter the number
  of pixels.
