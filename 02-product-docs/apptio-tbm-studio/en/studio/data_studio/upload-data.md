# Upload data

**Applies to**: TBM Studio 12.0 and later

When you upload data, you upload the data into a table. You can upload data using one of the
following methods:

- **File Upload**: Select a file from a local or network location in one of the following
  formats: Excel (xls, xlsx), .csv, XML, Zip ([How?](upload-data-file.html "Applies to: TBM Studio 12.0 and later. TBM Studio accepts data from flat files in comma-, tab-, pipe-, tilde-, colon-, and semi-colon-separated formats. Before you can upload data, you must create a table where the data will be loaded, if it does not already exist.")).
- **Enter Manually**: Enter the data manually in a table ([How?](enter-data-manually.html "Applies to: TBM Studio R12.0 and later")).
- **Generated Table**: Create a new table from an existing table in the current project. The
  new table will NOT be linked to the existing table.
- **Existing Table**: Create a new table from an existing table in the current project. The new
  table WILL be linked to the existing table.
- **Datalink (Classic) Connector**: Import a table using a predefined
  Datalink (Classic) connector.
- **Imported Table**: Import the data from an existing Costing Standard project.

Before you can upload data, you must create a table that will receive the data ([How?](create-table.html "Applies to: TBM Studio 12.0 and later")).
