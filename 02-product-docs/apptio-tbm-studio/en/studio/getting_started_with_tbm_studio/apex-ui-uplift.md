# ApEx UI Uplift

**Applies to**: 12.11.0 and later

## Introduction

The ApEx changes is the first step to modernize the look and feel of TBM
Studio and the CT platform. ​It focuses on updating the look and feel of TBM
Studio by adopting ApEx styling and colors. The ApEx UI setting will be enabled by default for
all new projects created by new or existing customers.

![](../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/r-notes/apexui-default-enable.png)​

The changes are applied to the report section, and then automatically applied to the report
collections, tables, and Editable Tables. The look and feel of the following UI components has been
updated:

Note: If the older UI is preferred for consistency with other existing projects, disable the new
ApEx design by navigating to **Project** tab > **Project Settings** > and uncheck the
**Enable Apex UI** option.

## Open Reference Projects

**Applies to**: 12.11.2 and later. The Apptio reference applications like Vendor Insights, Costing Standard, Billing, etc are also enabled with
Apex UI. The Admins and the end users can see reports in the new modern UI.

![](../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/r-notes/apxui-2.png)

![](../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/r-notes/apxui-3.png)

## Components Impacted

The following report components are impacted by the new design:

- Colors of all charts and fonts
- Borders across all components
- Icons, Context Menu, Font size
- KPIs
- Tabs
- Tables
- Editable Table Upload
- Slicers
- Dialog Box
- Column pickers
- Editable Table
- Pivot
- Notifications (Not applicable to CT view)
