# Group

Grouping feature allows you to easily organize multiple components in your report layout
by grouping or ungrouping them. This helps manage related components as a single unit.

## When to use Groups

Use Groups when you want to:

- Keep related content visually together
- Move multiple components together
- Simplify complex layouts

## Add a Group to the Report

1. Create group by multi-selecting components
   1. Press and hold the **Shift** key, then click to select multiple components on the
      canvas.
   2. Press **Ctrl + G (Cmd + G on Mac)** to group the selected components. (You’ll see
      the grouped components appear as a nested hierarchy in the **Layer panel.)**
2. A **Group icon** is also available on the toolbar – clicking it performs the same action.

   ![](../../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/grp-icon.png)
3. Create a framed group
   1. Add a Group from the Components pane on the toolbar
   2. Format Panel
      1. General Properties – See Component Properties
4. Ungroup
   1. To ungroup, select the grouped component and click the Group icon again (it acts as
      Ungroup when a group is selected).
   2. The hierarchy updates in the Layer panel, showing individual components again.
