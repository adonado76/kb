# Components Overview

Components are the building blocks used to structure reports and enable interaction with
data in Report Studio. They allow admins to present data, apply filters, and guide how users
explore reports.

Components can be added to the report canvas and configured using properties such as data
selection, filters, and formatting.

## What You Can Do with Components

Using components, you can:

- Display data in structured formats such as tables and KPIs
- Enable interaction through slicers, filters, and pickers
- Organize report content using layout components like tabs and groups
- Add contextual or explanatory content using text or HTML

## Common Component Properties

All components share a common set of properties, which are documented once and referenced
from individual component pages.

General Properties under Format Panel

|  |  |
| --- | --- |
| Title | Slide the toggle if you wish to add a title to the component. If yes, you can edit the following:  - Title - Font size and style (bold, italics, underline) - Color of the title text (with option to reset the color) |
| Subtitle | Slide the toggle if you wish to add a sub-title to the component. If yes, you can edit the following:  - Subtitle - Font size and style (bold, italics, underline) - Color of the subtitle text (with option to reset the color) |
| Help tooltip | Provide a tool tip for the component. On providing this tooltip, the help icon will appear beside the component. |
| Alt Text | Provide the Alt text, which is the descriptive text for UI images and graphics, providing a text alternative for users with screen readers or if the image fails to load. |
| Position & size | Edit the following properties, as required:  - Width - Height - Position (x and y axis) |
| Style | Edit the following properties, as required:  - Text color - Background color - Border color - Border size |
