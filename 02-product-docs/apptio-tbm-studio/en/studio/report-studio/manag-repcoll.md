# Add or Remove Reports

1. Open the collection from the Report Collections menu on the landing page.
2. You can
   - Rename the collection
   - Add or remove reports.
   - Change visibility by toggling **Show collection to end users**.
3. Delete a Report Collection
   1. On the Report Collections tab, click the **Delete** icon against the report collection.
   2. Deleting a collection does not delete the reports inside it. They will remain
      available in the project.
