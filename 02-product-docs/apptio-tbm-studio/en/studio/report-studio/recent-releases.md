# Recent Releases

## Current Release (12.11.22) - May 2026

**Theme Engine:** The Theme Engine introduces centralized report styling in the New Report
Studio, enabling consistent formatting across reports through reusable themes. Themes include color
palettes, typography, spacing, borders, and component styling. A default project-level theme is
automatically applied to newly created and migrated reports, while users can also override themes at
the individual report level. This significantly reduces manual formatting effort during migration
and helps maintain visual consistency at scale.

**Custom Formulas:** You can now add descriptions to custom formulas to provide additional
context about the formula logic or intended usage. Descriptions help improve readability,
maintainability, and collaboration by making formulas easier to understand for report authors and
other users.

**Export to Excel:** You can now export table visualizations directly to Excel from the Report
Designer. This enables users to perform further analysis, share data externally, and work with
tabular report data outside the application.

**Zero values in Charts:** Added support for zero-value filters in chart visualizations,
enabling users to include or exclude zero-value data points for more flexible and focused data
analysis.

**Drill Navigation:** Added the ability to disable drill-through interactions directly from
the Data Panel after configuration, providing greater control over report interactivity and
navigation behavior.

**Editable Tables:**

- Enhanced Editable Tables with support for configurable row limits, giving users greater control
  over the number of rows that can be added or edited within a table.
- Added support for displaying formatted data in Editable Tables, improving readability and
  ensuring consistency with configured data formats across reports.
- Added support for using custom formulas with the “Add to Filter” capability in Editable Tables,
  enabling more flexible and dynamic filtering workflows.
