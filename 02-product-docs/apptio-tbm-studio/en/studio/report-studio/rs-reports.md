# Creating a report

Follow these steps to create and build your report:

1. **Create a report or collection**
   1. Click **New** and choose to create a **Report** or a **Report Collection**.

      ![new report studio popup](../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/new-rs-popup.png)
   2. Enter the name and description for the report and click the **Create** button. A blank
      report canvas will open

      ![blank canvas](../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/rs-blank-canvas.png)

      Note: You can also create a new report using the
      *Ctrl+Shift+O* keyboard shortcut.
2. **Explore the canvas layout**
   1. **Toolbar (top):**Contains visualizations and report components you can add.
   2. **Layer Panel (left)**: Shows the hierarchy of components in your report. From
      here, you can copy, paste, reorder, delete, or change group membership.
   3. **Dimensions Explorer (right)**: Lists Tables, Editable Tables, Metrics and Time.
      Use this to drag and drop dimensions into component configurations.
3. **Add components or visualizations**
   1. Drag a component/visualization from the toolbar onto the canvas.
   2. Each added component appears in the **Layer** Panel.
4. **Configure your component**
   1. Once a component is added, 2 panels appear on the right side, next to the Dimensions
      Explorer: ![data and format panels](../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/rs-df-panels.png)
      1. **Data Panel:** Select a model object and add data dimensions.
      2. **Format Panel**: Customize properties such as color, font, size, and other
         styling options.
      3. General Properties

         |  |  |
         | --- | --- |
         | Title | Slide the toggle if you wish to add a title to the component. If yes, you can edit the following: - Title - Font size and style (bold, italics, underline) - Color of the title text (with option to reset the color) |
         | Subtitle | Slide the toggle if you wish to add a sub-title to the component. If yes, you can edit the following: - Subtitle - Font size and style (bold, italics, underline) - Color of the subtitle text (with option to reset the color) |
         | Help tooltip | Provide a tool tip for the component. On providing this tooltip, the help icon will appear beside the component. |
         | Alt Text | Provide the Alt text, which is the descriptive text for UI images and graphics, providing a text alternative for users with screen readers or if the image fails to load. |
         | Position & size | Edit the following properties, as required: - Width - Height - Position (x and y axis) |
         | Style | Edit the following properties, as required: - Text color - Background color - Border color - Border size |
5. **Save your work**
   1. **Autosave:** Your changes are automatically saved as you work.
   2. **Explicit Save:** You can also click the **Save** button at the top (next to the
      breadcrumbs) to save manually at any time.
