# Creating a Report Collection

A **Report Collection** is a container that groups related reports together for
easier access and organization.

1. **Create a Collection**

   ![new report collection](../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/rs-rep-collection.png)

   1. On the landing page, click **New** and select **Report Collection**.
   2. Give your collection a name and an optional description.
   3. Use the toggle **Show collection to end users** if you want the collection to be
      visible to end users.
      1. **On**: The collection will appear for end users in their view.
      2. **Off**: The collection is hidden from end users and only visible to the
         admins/report designers.
2. **Add Reports to a Collection**
   1. Click Add Report to add existing report(s) from the project into the collection.

      ![adding report to report collection](../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/rs-rc-report.png)
   2. The added reports will be listed under the collection.
