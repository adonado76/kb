# Export a Table to Excel

You can export individual tables from a report to Microsoft Excel (.xlsx) directly from the
table controls. This allows you to further analyse, share, or archive report data outside
the application. Currently available in the New Report Viewer.

**How to export a table to Excel?**

1. Open the report in the New Report Viewer.
2. Navigate to the table you want to export.
3. Click the overflow menu in the table’s outboard controls.
4. Select **Export to Excel.**
5. The file will download automatically to your system.

What gets exported?

- All rows currently available in the table
- Column headers as displayed
- Applied filters reflected in the exported data

Note: The export reflects the current state of the table at the time of download.

Tips:

- Apply filters before exporting if you only need a subset of the data.
- If the table includes sorting, the exported file reflects the sorted order.

**Export to Excel in Report Designer**

You can now export table visualizations directly to Excel from the Report Designer from the table’s outboard controls.
