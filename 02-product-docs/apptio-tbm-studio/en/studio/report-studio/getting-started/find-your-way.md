# A Guide to New Report Studio’s Navigation

To access the new Report Studio, go to the Classic TBM Studio in your main environment.

**Look for the blue “Open IBM Apptio Report Studio” button** – Once your instance is upgraded
to 12.11.21 and above, you will see a blue button at the top of your current report studio. Clicking
this will open a list of available projects.

Note: Customers on versions 12.11.19 should be
enrolled in the public preview to see the blue button.

![image of the new report studio button](../../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/nrs-button.png)

**Select a project to open it in the new report studio** – Choose a project from the drop-down
to automatically launch it in the new experience.

The new Report Studio opens on the landing page, as shown below.

**Reports Home** 

![navigation of landing page in new report studio](../../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/nrs-home.png)

1. New button – Click here to create a new report or a new report collection
2. Search – Search for an existing report
3. New Reports – Lists all reports created using the new Report Studio.
4. Report Collections – Displays groups of related reports organized into collections.
5. Report Overflow Menu (3 dots menu) - Click here to access a host of operations that can
   performed on a report, like deleting a report, or exporting the report’s definition and
   configuration.
6. Project Settings – Click here to check in one or more reports, revert changes to them or to
   import a report configuration into the project in context.

**Navigation Panel** 

![navigation of top panel in new report studio](../../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/nrs-navpan.png)

1. App Switcher – Use this to switch to another IBM Apptio product
2. Profile Settings - Use this to manage your profile, impersonate another user (Note: Admins may
   want to do this to check a user's permissions or to troubleshoot problems), and Logout
3. Help – Use this to access the new Report Studio help documentation
4. Settings - Use this to change the customer access, update permissions, and access multi-currency
   settings
5. Environment Dropdown – Use this to switch between the environments and user workspaces.
6. Date Picker – Use this to switch between date periods
7. Take me to – Use this quick navigation menu to switch to the Classic TBM Studio and/or the New
   Report Viewer.
