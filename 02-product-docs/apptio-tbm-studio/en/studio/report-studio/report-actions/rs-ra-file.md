# File Menu

The File menu in the report canvas is

![filemenu](../../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/rs-filemenu.png)

***Check In***

To check in a report, go to **File** > **Check In**.

![checkin menu](../../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/rs-checkinmenu.png)

This triggers a recalculation, and others can view the changes you have made to the document.
The changes will be made available in the Staging environment. You can ‘select all’ or ‘none’
of the reports.

***Check Out***

To edit a report, you must first check it out. To checkout, go to **File**> **Check
Out**.

When you check out a report, it is locked so others cannot edit the document. You can save
changes to the document without triggering a recalculation. When you are done editing a
document, save it and check it back in.

***Revert Changes***

The revert function abandons all changes you made to checked-out documents you select and then
discards the check out. To revert, follow these steps:

1. From the Create Report page, go to **File** and then click **Revert Changes**.
2. Select the checkbox next to documents you want to revert and click **Revert Check
   Out**.

   ![revert menu](../../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/rs-revert.png)

***Save***

When you first create a report, the application automatically saves the report. If you make
changes to a report and then navigate away, the report is automatically saved. You can also
save a report by clicking the **Save** icon on the report creation page.
