# Working with Documents

## Document Tabs

Open documents appear as tabs in the main workspace area.

**Tab features:**

- **Active tab** — Highlighted to show current document
- **Close button** — Click ✕ to close a tab
- **Unsaved indicator** — Asterisk (\*) shows unsaved changes
- **Document type icon** — Visual indicator of document type

**To manage tabs:**

- **Switch tabs:** Click the tab you want to view
- **Close tab:** Click the ✕ or press Alt+Escape (Meta+Escape on Mac)
- **Close all tabs:** Right-click any tab and select “Close All”
- **Reorder tabs:** Drag and drop tabs to rearrange

## Check Out and Check In

**Check Out** locks a document for editing and prevents other users from making
changes.

**To check out a document:**

1. Find the document in Project Explorer
2. Right-click and select **Check Out** (or press Alt+O)
3. The document opens with a locked icon
4. Make your edits

**Check In** saves your changes and makes them available to other users in Staging.

**To check in a document:**

1. Save your changes (Ctrl+S or Meta+S)
2. Click **Check In** in the toolbar (or press Alt+K)
3. Add a check-in comment describing your changes (optional but recommended)
4. Click **OK**

Warning: Check in frequently to avoid conflicts with other users and to ensure
your work is saved to the server.
