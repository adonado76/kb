# Customization Options

## Studio Mode Toggle

TBM Studio offers two interface modes:

**Studio Mode (default):**

- Full editing interface with all tools and options
- Ribbons and detailed controls visible
- Best for development and configuration

**Report View Mode:**

- Streamlined interface focused on viewing
- Minimal controls, maximizes report display area
- Best for presentations and end-user consumption

**To toggle:** Press Ctrl+Alt+T or click the Studio mode toggle in the toolbar.

## ApEx UI

TBM Studio includes a modernized ApEx user interface with updated styling and colors.

**What changes:**

- Report styling and colors
- Chart and table design
- Icons and context menus
- Dialog boxes and notifications
- Borders and fonts

**To enable/disable ApEx UI:**

1. Go to **Project** tab > **Project Settings**
2. Check or uncheck **Enable Apex UI**
3. Click **Save**

Note: ApEx UI is enabled by default for new projects. Disable it if you prefer consistency
with older projects or need the legacy interface.
