# Common Use Cases

**Labor Cost Mapping: Users select Solution → Category → Subcategory from cascading dropdowns to
classify labor costs for allocation.**

**Budget Variance Explanations: Text input columns capture explanations for budget variances
alongside calculated metrics.**

**Cost Center Attribute Maintenance: Distributed managers update attributes for their own cost
centers, controlled by Row-Level Security.**

**Direct Data Corrections: Users correct individual data values where source system data is
incomplete or incorrect.**
