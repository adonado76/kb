# Report Components: Navigation

## Overview

Navigation components enable users to move between reports, pages, and external resources within
TBM Studio. Effective navigation design improves user experience by providing intuitive access to
detailed information and related reports. This reference documents all navigation components, their
configuration options, and best practices for implementation.

TBM Studio provides several navigation mechanisms: button components for explicit navigation
controls, report links for drill-through from tables and charts, coded links using Wiki-style syntax
for flexible navigation, report collection navigators for organizing related reports, and
hierarchical slicers for drill-down filtering.
