# Report Collections

Report collections group related reports so users can easily navigate between reports on a
similar topic or perspective. Collections improve report discoverability by organizing reports into
logical groupings and providing a consistent navigation experience. Administrators configure report
collections to control which user roles can access reports within each collection.
