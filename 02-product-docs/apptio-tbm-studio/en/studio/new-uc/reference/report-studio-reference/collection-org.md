# Collection Organization

**Report Ordering Within Collections**

The order in which reports appear in the collection navigator is determined by their order in the
Manage Report Collections dialog. Reports are displayed from left to right in the navigator based on
their position in the collection's report list.

To reorder reports, open the Manage Report Collections dialog, select the collection, and drag
reports to rearrange them in the desired order.

**Collection Naming Conventions**

Follow these best practices when naming collections:

- Use clear, descriptive names that indicate the collection's purpose
- Keep names concise for better display in navigation elements
- Consider your audience when choosing names (technical vs. business terminology)
- Use consistent naming patterns across collections (e.g., "IT Finance - Monthly", "IT Finance -
  Quarterly")
