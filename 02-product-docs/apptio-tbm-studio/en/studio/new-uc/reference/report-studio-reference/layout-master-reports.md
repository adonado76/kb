# Master Reports

Master reports provide a template layer (like "wallpaper") that can be applied to multiple
reports for consistent layout.

**Key characteristics:**

- Master report components cannot be edited from the report where they're applied.
- Components added to a report sit on top of the master report layer.
- Group boxes from master reports lose their grouping functionality; use them as visual borders.

**To create a master report:**

1. Check out the report.
2. On the Report tab, in the Advanced group, click Master Report.
3. Save the report.
