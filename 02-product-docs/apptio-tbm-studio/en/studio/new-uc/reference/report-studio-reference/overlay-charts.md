# Overlay (Combination) Charts

Overlay charts display two data series in a single chart, typically combining columns or bars
with lines. This enables direct comparison of related metrics, such as budget versus actuals or
capacity versus utilization.

**Layer Visualization Options**

Each layer in an overlay chart can be displayed as:

- Columns: Individual bars for each value.
- Stacked Columns: Segmented bars showing composition.
- Lines: Connected data points for trends.

**Creating an Overlay Chart**

1. On the Report tab, click Overlay.
2. Click the Overlay tab in the configuration panel.
3. Click Edit Base Layer, then drag fields into the Axis and Values areas.
4. Select a visualization type for the base layer.
5. Click Edit Layer 1 and repeat the configuration for the second series.
6. Add filters if needed to simplify the chart.

**Share Axis:** Available in chart properties, this option makes the scales on the left and
right axes the same, which is useful when comparing values of similar magnitude.
