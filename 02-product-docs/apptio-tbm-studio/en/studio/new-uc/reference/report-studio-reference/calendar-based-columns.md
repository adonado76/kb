# Calendar-Based Columns

Add columns that aggregate or compare values across time periods. Access these via Formulas tab >
Dates menu.

|  |  |  |
| --- | --- | --- |
| **Abbreviation** | **Name** | **Description** |
| YTD | Year to Date | Sums values from fiscal year start through the selected period. |
| QTD | Quarter to Date | Sums values from quarter start through the selected period. |
| FY | Fiscal Year | Total for the entire fiscal year regardless of selected period. |
| FQ | Fiscal Quarter | Total for the entire fiscal quarter regardless of selected period. |
| ANN | Annualized | Average monthly value to date multiplied by 12 to project annual total. |
| LFY | Last Fiscal Year | Value from the previous fiscal year. Useful for year-over-year comparison. |
| LFQ | Last Fiscal Quarter | Total for the previous fiscal quarter. |
| LTM | Last 12 Months | Rolling 12-month total, not including current month. Not fiscal-calendar based. |
| LM | Last Month | Value from the previous month. |
| LY | This Period Last Year | Same period in the previous year for year-over-year comparison. |
| TTM | Trailing 12 Months | Sum of previous 12 months (e.g., Aug 2024 = Aug 2023-Jul 2024). Annualizes if <12 months available. |
