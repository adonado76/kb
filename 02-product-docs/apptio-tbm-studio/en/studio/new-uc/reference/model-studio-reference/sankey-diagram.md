# Sankey Diagram Concepts

The Sankey diagram is used in both Single-Table and Model Report views to visualize value flow.

**Diagram Elements**

|  |  |
| --- | --- |
| **Element** | **Description** |
| Nodes (Rectangles) | Represent model objects or table columns. Width indicates total value. |
| Flow Lines | Represent allocations between objects. Width proportional to allocated value. |
| Flow Direction | Value flows left to right (from sources to targets) |
| Value Labels | Display metric values on nodes and flows |
