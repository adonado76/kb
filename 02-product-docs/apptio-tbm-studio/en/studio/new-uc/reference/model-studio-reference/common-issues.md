# Common Issues

|  |  |  |
| --- | --- | --- |
| **Issue** | **Cause** | **Resolution** |
| Cannot add table to tier | Table lacks Model step | Add Model step to table transform |
| Cannot add column to tier with tables | Mixed content not allowed | Remove tables first, or use a different tier |
| Slow report loading | Too many metrics selected | Limit metrics using Edit Metrics |
| Missing allocations | Tables not checked in | Check in all model changes |
| Report not in collection | Not assigned to collection | Use Assign to Collection option |
