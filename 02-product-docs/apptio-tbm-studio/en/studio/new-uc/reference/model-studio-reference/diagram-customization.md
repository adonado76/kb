# Diagram Customization Options

**Config Only Mode**

When editing a modeled table, use Config Only mode for faster editing without value calculations.

**How to Enable**

1. Click View in the ribbon
2. Select Config Only
3. Numbers disappear from the model view

**How to Disable**

1. Click View in the ribbon
2. Select Show Document

**Note:** Config Only mode is toggled per object, not globally.

**Metric Selection**

Switch the metric displayed in the model view using the metric picker at the top of the model
pane.
