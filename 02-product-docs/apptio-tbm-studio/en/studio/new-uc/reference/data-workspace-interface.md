# Data Workspace Interface

## Project Explorer

**Purpose:** Navigate and manage all project documents

**Document Types:**

- Tables
- Editable Tables
- Metrics
- Perspectives
- Published Tables
- Reports
- Time

## Transform Steps Pane

**Purpose:** View and modify table transform pipeline

**Features:**

- Sequential step display
- Step addition between existing steps
- Drag-and-drop reordering (with restrictions)
- Error indicators with counts
