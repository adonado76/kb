# Transform Steps

Transform steps form a pipeline that processes data from source to output. Steps execute
sequentially from top to bottom.

## Transform Pipeline Overview

**Default Steps** (always present):

- **Source:** Determines data source type
- **Table:** Displays final output after all transforms

**File Upload Steps** (added automatically):

- **Upload:** File selection and management
- **Import:** Import configuration settings
