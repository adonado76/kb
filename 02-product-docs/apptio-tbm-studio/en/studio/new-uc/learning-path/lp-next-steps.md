# Next Steps for All Roles

## For All Roles

Once you've completed your learning path, consider these next steps:

1. **Apply your skills:** Use what you've learned in real projects and workflows
2. **Explore other paths:** Understanding other roles improves collaboration
3. **Join the community:** Engage with other TBM Studio users
4. **Stay current:** Review *"What's New*" regularly for new features
5. **Provide feedback:** Help improve these learning paths for future users

**Cross-Role Opportunities**

- **Business Analysts → Administrators:** If you need to manage user access or
  permissions
- **IT Finance → Developers:** If you want to automate data loads or build
  integrations
- **Administrators → IT Finance:** To better understand the models you're supporting
- **Developers → IT Finance:** To understand business context for your integrations
