## Whats new in Costing and Billing

- [Recent releases](whats-new.html)

- **[My Landing Page](../costing-billing/modernization-home.html)**  
  This feature allows you to select a specific page or report to be your default landing page. The Home option is enabled by default and will always show the Service Costing report.
- **[Search](../costing-billing/modernization-search.html)**  
  This feature allows you to search for a report, so that you can access them easily, instead of searching and viewing it. The Search icon is enabled by default and appears on the top navigation bar in the UI.
- **[Recents](../costing-billing/modernization-recent.html)**  
  This feature allows you to see the reports that you have used most recently. The Recents option is enabled by default and will appear in the left navigation menu. It will display the list of recently accessed reports.
- **[Favorites](../costing-billing/modernization-favorites.html)**  
  This feature allows you to select one or more reports and mark them as favorite, so that you can access them easily, without the hassle of searching and navigating for viewing it. The Favorites option is enabled by default and will appear on the left navigation menu in the UI.
