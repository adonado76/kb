# Search

This feature allows you to search for a report, so that you can access them easily,
instead of searching and viewing it. The Search icon is enabled by default and appears on the top
navigation bar in the UI.

Applies to: Costing Standard, Billing, Hybrid Business Management, Vendor Insights,
Costing Essentials and Demand.

![img](../../../../03-media/apptio-tbm-studio/resources/images/studio_images/search-1.png)

Note: The search icon is visible for an end user, and not to an Administrator.

Select the Search icon and enter the full/partial name of the report in the search
textbox. The search results show up only on entering 3 or more characters. The list of reports
meeting the search criteria will appear as you type. The search result will show Report Name (Alias)
if alias is present, other wise only Report Name.

![img](../../../../03-media/apptio-tbm-studio/resources/images/studio_images/search-new.png)

Select the report name that you want to view.

![img](../../../../03-media/apptio-tbm-studio/resources/images/studio_images/search-2.png)

The search function is applicable only for top-level reports. The drill-to reports will not be
shown in the search results.

If an Admin impersonates a user, then the user can see the reports searched by the Admin.

If there are no matching report, a message "No results found" will appear.

![img](../../../../03-media/apptio-tbm-studio/resources/images/studio_images/search-3.png)

If 'Configure Recents' is enabled, the searched report will automatically appear in the [Recents](modernization-recent.html "This feature allows you to see the reports that you have used most recently. The Recents option is enabled by default and will appear in the left navigation menu. It will display the list of recently accessed reports.") section, at the top.
