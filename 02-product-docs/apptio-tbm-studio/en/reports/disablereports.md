# Disable reports

◆ Applies to: TBM Studio 12.7 and later

An administrator can temporarily disable a report. This article addresses options and
considerations when removing or disabling a report.

## Delete reports

The first and simplest way to disable a report is to delete it. You can do so after checking out
a report.

On the Home tab, in the Document group, click Delete. The report is removed from
the project.

Note: You cannot delete an out-of-the-box (OOTB) report.

## Restrict reports by role

If you want a report to be accessible to a specific set of roles, you can adjust the report's
permissions settings. For more information on setting report permissions, see [Work with report
permissions](control-access-reports-11755.htm "(Opens in a new tab or window)").

1. On the Projects tab, in the Advanced group, click Permissions.
2. Modify the report level permissions based on the best practices described in [Work with report
   permissions](control-access-reports-11755.htm "(Opens in a new tab or window)").

## Prevent a report from calculating

1. A third option for disabling reports is to prevent a report from calculating.
2. On the Report tab, in the Advanced group, clear the Active checkbox.
3. Check in the report.
