# Introduction to Report Studio

**Applies to**: TBM Studio 12.0 and later

After you upload tables and build allocation models in TBM Studio, you can use Report Studio to
create reports. Reports can contain tables and charts, as well as components for filtering the
reports, entering data, and navigating between reports.
