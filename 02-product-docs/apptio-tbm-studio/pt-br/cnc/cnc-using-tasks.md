# Comentários e colaboração Usando tarefas

Aplica-se a: Costing Standard de 12.11.5 e posterior

O recurso de tarefas permite que os usuários tenham conversas acionáveis, possibilitando a criação de comentários e a atribuição de tarefas a usuários específicos. As tarefas podem ser atribuídas a um indivíduo de cada vez e podem ser usadas em tópicos privados ou públicos.

Atribuir uma tarefa

Para iniciar uma tarefa, adicione um comentário ![Comentário](../../../../03-media/apptio-tbm-studio/resources/images/cnc/comments_icon.jpg) e determine se você deseja que o comentário seja privado ou público. Para obter mais informações, consulte [Uso de comentários](using-comments.html) e [recurso de comentários privados](cnc-private-commenting.html).

Digite seu comentário e digite @ para adicionar um usuário.

É exibida uma caixa de seleção para atribuir um usuário e converter o comentário em uma tarefa.

Marque a caixa de seleção Atribuir a @ usuário.

Selecione Assign.

O usuário recebe uma notificação por e-mail informando que uma tarefa foi atribuída.

Reatribuir uma tarefa

1. Para reatribuir uma tarefa, selecione Reply ![seta](../../../../03-media/apptio-tbm-studio/resources/images/cnc/reply_icon.jpg) em um tópico existente e digite @ para adicionar um usuário. É exibida uma caixa de seleção para atribuir o usuário e converter o comentário em uma tarefa.
2. Marque a caixa de seleção Atribuir a @ usuário.
3. Selecione Assign.

O usuário recebe uma notificação por e-mail informando que uma tarefa foi atribuída.

Resolver uma tarefa

Somente os designados ou os responsáveis podem resolver uma tarefa.

Para resolver uma tarefa, siga um destes procedimentos:

Se você for o responsável, selecione Resolver ![marcação](../../../../03-media/apptio-tbm-studio/resources/images/cnc/resolve_icon.jpg) no início do tópico.

Se você for o responsável pela atribuição, selecione ![mais…](../../../../03-media/apptio-tbm-studio/resources/images/cnc/cnc_more_options_icon.jpg) no início do tópico e, em seguida, selecione Marcar como resolvido.

Excluir uma tarefa

Para excluir uma tarefa, selecione ![mais…](../../../../03-media/apptio-tbm-studio/resources/images/cnc/cnc_more_options_icon.jpg) no início do tópico.

Selecione Excluir thread.

Os usuários que não são o responsável ou o responsável também podem excluir seus comentários no tópico da tarefa selecionando ![mais…](../../../../03-media/apptio-tbm-studio/resources/images/cnc/cnc_more_options_icon.jpg) e, em seguida, Excluir comentário.
