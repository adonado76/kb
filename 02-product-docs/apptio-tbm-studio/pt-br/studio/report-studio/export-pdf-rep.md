# Exportar relatório para PDF

A função de exportação para PDF no visualizador de relatórios permite exportar o relatório que está sendo visualizado para um arquivo PDF. Isso facilita o download e o compartilhamento.

**Quando usar**

Use esta opção quando quiser:

- Compartilhar relatórios com as partes interessadas
- Crie uma captura de tela do relatório para revisões ou apresentações
- Arquivar relatórios para fins de manutenção de registros ou conformidade

**Exportar um relatório para PDF**

Para exportar um relatório a partir do visualizador de relatórios:

1. Abra o relatório no visualizador de relatórios
2. Clique em “**Exportar para PDF** ” na barra de ações na parte superior do relatório.
3. O sistema gera e baixa um PDF da visualização atual do relatório.
