# Menu Organizar

O novo estúdio de relatórios oferece ferramentas de layout flexíveis para colocar e organizar componentes em um relatório. A maioria dessas ferramentas de layout está no menu **Organizar**. Pelo menos um componente deve ser selecionado para ativar os comandos no menu **Organizar**. Para começar, clique no componente que deseja modificar.

O menu Organizar na tela do relatório é

![organizar o menu](../../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/rs-arrangemenu.png)

*Alinhar*

Use os comandos do grupo Align (Alinhar) para controlar como dois ou mais componentes aparecem em relação uns aos outros. Esses comandos só ficam ativos depois que você seleciona dois ou mais componentes em um relatório.

Selecione os componentes que deseja alinhar. Vá para **Arrange** > **Align** e selecione a opção de alinhamento desejada. As opções são

![alinhar menu](../../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/rs-align.png)

Para alinhamentos horizontais, use as opções **Esquerda**, **Centro** e **Direita**, enquanto para alinhamento vertical, escolha as opções **Superior**, **Meio** e **Inferior**.

*Mover componentes do relatório*

Em um relatório, você pode reposicionar os componentes, se houver mais de um.

Clique e mantenha pressionado o cabeçalho do componente do relatório e, em seguida, arraste o componente para um novo local. Se você mover um componente para uma posição em que ele se sobreponha a outro componente, poderá selecionar quais componentes devem ficar na frente do outro.

No menu **Organizar**, selecione o componente e escolha uma das seguintes opções:

- Trazer para a frente - move o componente totalmente para a frente
- Bring forward - avança o componente em uma etapa
- Enviar para trás - move o componente totalmente para trás
- Enviar para trás - move o componente uma etapa para trás

***Distribuir***

Use a opção Distribute (Distribuir) para espaçar uniformemente os componentes na horizontal ou na vertical, ajustando o layout automaticamente, mesmo quando alguns componentes estiverem ocultos para determinados usuários. Os atalhos para distribuição são:

- Shift+Alt+H - Distribuir horizontalmente
- Shift+Alt+V - Distribuir verticalmente
