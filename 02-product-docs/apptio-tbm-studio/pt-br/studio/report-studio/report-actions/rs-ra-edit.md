# Menu Editar

O menu Editar na tela do relatório é

![menu de edição](../../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/rs-editmenu.png)

*Desfazer/Refazer*

Essas opções são usadas para reverter alterações pequenas e discretas durante a edição de documentos em seu espaço de trabalho. Quando os documentos são verificados, você pode retornar a uma versão anterior ou refazê-la. Alguns cenários para desfazer ou refazer as alterações são:

- Posicionamento de elementos de relatório em um relatório
- Edições discretas na configuração dos elementos de relatório (por exemplo: texto do botão, conteúdo HTML, etc.)
- Edição de fórmulas para colunas na etapa de fórmula de uma tabela

*Selecionar tudo*

Essa opção seleciona todos os componentes presentes na tela do relatório.
