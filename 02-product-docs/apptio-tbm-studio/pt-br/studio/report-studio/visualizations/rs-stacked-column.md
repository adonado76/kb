# Coluna empilhada

Para criar um gráfico de colunas,

1. Abra um relatório novo ou existente.
2. Vá para **Visualizações** e selecione o bloco **Coluna**.

   ![ladrilho de coluna empilhada](../../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/rs-stkdcol.png)
3. Clique no componente Stacked Column para ativar os painéis **Data** e **Format**.
4. Expanda os painéis para configurar os dados e as opções de formato:

   |  |  |
   | --- | --- |
   | **Dados** | |
   | Selecionar objeto modelo | Selecione o objeto modelo na lista suspensa |
   | Legenda | Arraste ou adicione um dado de legenda |
   | Eixo Y | Arraste ou adicione os dados para o eixo Y |
   | Eixo X | Arraste ou adicione os dados para o eixo Y |
   | Filtros | Arraste ou adicione os critérios de filtro |
   | **Formato - Propriedades** | |
   | Eixo X | Edite as seguintes propriedades, conforme necessário - Alternar para mostrar o título do eixo X - Alternar para mostrar os valores do eixo X - Tamanho e estilo da fonte (negrito, itálico, sublinhado) - Cor do texto e dos valores do título do eixo X (com a opção de redefinir a cor) - Alternar para mudar a posição do eixo X - Alternar para mostrar as linhas de grade do eixo X |
   | Eixo Y | Edite as seguintes propriedades, conforme necessário - Alternar para mostrar o título do eixo Y - Alternar para mostrar os valores do eixo Y - Tamanho e estilo da fonte (negrito, itálico, sublinhado) - Cor do texto e dos valores do título do eixo Y (com a opção de redefinir a cor) - Alternar para inverter a faixa - Alternar para mostrar as linhas de grade do eixo Y |
   | Legenda | Edite as seguintes propriedades, conforme necessário - Alternar para mostrar a legenda - Tamanho e estilo da fonte da legenda (negrito, itálico, sublinhado) - Cor do texto da legenda (com opção para redefinir a cor) |
   | Barras | Edite as seguintes propriedades, conforme necessário - Número de resultados a serem exibidos - Acolchoamento entre as barras (espessura das barras) - Preenchimento entre grupos (espaçamento entre grupos) - Cor da barra para todos os parâmetros - Legenda, eixo X, eixo Y |
   | Etiquetas de dados | Edite as seguintes propriedades, conforme necessário - Alternar para mostrar os rótulos de dados - as opções são **dentro** ou **fora da** barra. |
