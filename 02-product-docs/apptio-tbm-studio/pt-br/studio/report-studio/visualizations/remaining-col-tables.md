# Renomear colunas em uma tabela

Renomeie os cabeçalhos das colunas diretamente no painel de configuração de dados, permitindo que você apresente os dados de forma mais significativa ou intuitiva sem alterar o nome da dimensão subjacente.

Passos para usar

1. **Abra o Painel de Configuração de Dados da sua tabela**
   1. Vá para a guia **Dados**.
   2. As dimensões adicionadas na seção **“Linhas”** definem as colunas da tabela.
2. **Renomear uma dimensão (cabeçalho da coluna)**
   1. Na seção **“Linhas”**, clique no menu de opções ao lado do nome da dimensão.

      ![editar nome exibido](../../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/rename1.png)
   2. Selecione “**Editar nome de exibição** ”.
   3. Na caixa de diálogo “**Editar nome de exibição** ”, insira o novo nome de exibição para a coluna.
   4. **O nome original da dimensão** é apresentado abaixo para referência.

      ![imagem da edição do nome de exibição](../../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/rename2d.png)
3. **Salvar alterações**
   1. Clique em **OK** para confirmar as alterações.
   2. O nome atualizado aparece como cabeçalho da coluna na tabela.

Observação: a renomeação afeta apenas o nome exibido na visualização da tabela. O nome original da dimensão no conjunto de dados permanece inalterado.
