# Precisão numérica para gráficos cartesianos

É possível controlar o número de casas decimais exibidas para os valores nos gráficos cartesianos usando a configuração **“Precisão numérica”,** disponível no **painel** “Formato” do gráfico.

A configuração aceita padrões de formatação como:

- #.00 → Exibe valores com 2 casas decimais
- Exemplo: 10 → 10.00

Essa melhoria contribui para aumentar a legibilidade e a consistência dos valores numéricos exibidos nos gráficos.

Observação: a formatação de precisão numérica é compatível, no momento, apenas com **gráficos cartesianos**.
