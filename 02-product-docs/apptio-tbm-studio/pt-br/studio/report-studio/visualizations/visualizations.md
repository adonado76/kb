# Visão geral das visualizações

As visualizações ajudam a representar os dados visualmente, para que as tendências, os padrões e as comparações sejam mais fáceis de entender rapidamente. O Report Studio oferece uma variedade de tipos de visualização que funcionam perfeitamente com filtros, segmentadores e navegação detalhada para dar suporte à análise interativa.

## O que você pode fazer com visualizações

Com as visualizações, você pode:

- Compare valores entre categorias
- Identifique tendências e mudanças ao longo do tempo
- Destaque padrões e valores atípicos
- Permita a exploração interativa por meio de filtragem e navegação detalhada

## Tipos de visualização suportados

O Report Studio suporta vários tipos de visualização, incluindo:

- KPIs, tabelas e tabelas editáveis
- Gráficos de barras e colunas
- Gráficos de barras empilhadas e colunas empilhadas
- Gráficos de linhas
- Gráficos circulares
- Gráficos de colunas + linhas e colunas empilhadas + linhas

## Visualizações compatíveis

O recurso Compatible Visualizations permite que você alterne rapidamente entre tipos de gráficos compatíveis sem precisar reconstruir a visualização. Isso o ajuda a experimentar diferentes formatos visuais e a encontrar aquele que melhor representa seus dados.

**Etapas para uso**

1. Selecione um gráfico
   1. Clique em um gráfico (por exemplo, um gráfico de barras) em seu relatório.
2. Use a opção de conversão
   1. No cabeçalho do widget, localize o ícone Visualizações compatíveis.
   2. Selecione o ícone para exibir uma lista de visualizações compatíveis com o tipo de gráfico selecionado.

      ![imagem de visualizações compatíveis](../../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/cv-2b.png)
3. Selecione um tipo de visualização
   1. Selecione entre opções como:
      1. Gráfico de colunas
      2. Gráfico de barras empilhadas
      3. Gráfico de colunas empilhadas
   2. O gráfico é atualizado instantaneamente para o tipo visual escolhido, mantendo os dados, a formatação e a configuração existentes.
