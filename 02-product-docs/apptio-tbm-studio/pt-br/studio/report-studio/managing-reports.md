# Gerenciamento de relatórios

Depois de criar um relatório, você pode executar as seguintes ações:

1. Exibir um relatório
   1. Acesse a listagem New Reports (Novos relatórios) na página de destino.
   2. Selecione o relatório que deseja abrir.
2. Modificar um relatório
   1. Confira o relatório em seu espaço de trabalho.
   2. Faça edições.
   3. Verifique o relatório para mover as alterações de volta para o ambiente de desenvolvimento.
3. Excluir um relatório
   1. Na lista de relatórios da página inicial, selecione a opção Excluir no menu de estouro do relatório.
   2. Confirme a exclusão quando solicitado.
