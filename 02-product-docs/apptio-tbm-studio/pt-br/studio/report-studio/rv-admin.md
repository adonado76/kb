# Controle administrativo para o acesso ao novo visualizador de relatórios

Os administradores podem definir se os usuários finais têm acesso ao Novo Visualizador de Relatórios no Report Studio. Essa configuração permite que as equipes gerenciem a visibilidade dos relatórios de acordo com as necessidades da organização.

Por padrão, o Novo Visualizador de Relatórios está ativado, permitindo que os usuários finais visualizem os relatórios sem dificuldades.

## Quando usar

Use esta configuração quando:

- Você deseja restringir o acesso aos relatórios para os usuários finais
- Você está testando ou configurando relatórios antes de torná-los visíveis
- Você deseja ter um controle mais rigoroso sobre quem pode visualizar os relatórios do seu projeto

## Ativar ou desativar o novo visualizador de relatórios para usuários finais

![](../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/ed-new.png)

Para gerenciar essa configuração:

1. Acesse a guia Projeto no TBM Studio Clássico
2. Abra o menu suspenso **“Ativar recursos”**
3. Encontre a opção:
4. “Ativar o novo visualizador de relatórios para o usuário final”
5. Use a caixa de seleção para:
   1. Ativar (padrão): Os usuários finais podem visualizar relatórios
   2. Desativar: Os usuários finais não terão acesso ao Visualizador de Relatórios

Nota:

- Essa configuração está ativada por padrão em todos os projetos
- Desativar essa configuração afeta apenas o acesso do usuário final, não as funções de administrador
- As alterações entram em vigor imediatamente
