# Crie sua primeira coleção de relatórios

Este tópico ajuda você a criar uma coleção de relatórios para organizar relatórios relacionados e controlar o acesso dos usuários.

1. Navegue até a guia Coleções de relatórios
   1. Na página inicial, selecione a guia Coleções de relatórios.
   2. Clique no botão **Novo** e selecione **Coleção de relatórios.**
2. Insira os detalhes da coleção
   - Mostrar coleção aos usuários finais – Ative esta opção para disponibilizá-la aos usuários finais no Novo Visualizador de Relatórios.
   - Nome da coleção – Insira um nome significativo para a coleção
   - Descrição (opcional) - Adicione uma breve descrição para explicar o objetivo da coleção.
   - Visível por – Esta propriedade controla quem pode ver e acessar o relatório.
     - Todos – O relatório fica visível para todos os usuários com acesso à área de relatórios.
     - Somente eu – O relatório é visível apenas para você. Útil para rascunhos, experimentação ou análise pessoal.
     - Selecionar funções – O relatório fica visível apenas para usuários atribuídos às funções selecionadas. As funções são selecionadas a partir de uma lista suspensa. Permite o controle de acesso baseado em funções para relatórios.

   ![imagem da nova tela de coleta de relatórios](../../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/nrs-rc.png)
3. Adicionar relatórios à coleção
   - Selecione um ou mais relatórios para incluir na coleção
   - Você pode adicionar relatórios durante a criação ou atualizar a coleção posteriormente.
   - Você pode controlar a visibilidade do usuário final em relatórios individuais
   - Clique em Criar

   ![imagem da lista de relatórios em uma coleção de relatórios](../../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/nrs-rc1.png)
