# Fórmulas personalizadas

As fórmulas personalizadas (também conhecidas como dimensões de fórmula) permitem definir novas dimensões calculadas utilizando campos existentes no seu modelo de dados. Isso permite uma análise mais profunda e insights mais ricos, sem a necessidade de alterações no conjunto de dados ou esquema subjacente.

## Por que usar o Formula Dimensions?

As dimensões da fórmula ajudam você a:

- Derive novas dimensões a partir dos dados existentes
- Realize cálculos diretamente na experiência do relatório
- Explore os dados com mais flexibilidade, sem depender de alterações no modelo de dados

## Como funcionam as dimensões da fórmula

- Em vez de selecionar apenas dimensões predefinidas do esquema subjacente, você pode
  - Defina uma fórmula personalizada usando campos e funções compatíveis
  - Agora você pode adicionar descrições às fórmulas personalizadas para fornecer mais contexto sobre a lógica da fórmula ou sua finalidade
  - Salve a fórmula como uma dimensão de fórmula reutilizável
- Uma vez criadas, as dimensões de fórmula se comportam como dimensões padrão e podem ser usadas para agrupamento, segmentação, filtragem e visualização

## Disponibilidade em componentes e visualizações

As colunas de fórmula são compatíveis com todos os componentes e visualizações.
