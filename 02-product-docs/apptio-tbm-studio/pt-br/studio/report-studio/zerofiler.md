# Sem filtros

Agora, os filtros de valor zero são compatíveis com visualizações de gráficos, permitindo que os usuários incluam ou excluam pontos de dados com valores zero durante a análise. Isso proporciona maior flexibilidade para se concentrar nas tendências relevantes, reduzir o ruído visual e melhorar a legibilidade geral e a interpretação dos dados dos gráficos.
