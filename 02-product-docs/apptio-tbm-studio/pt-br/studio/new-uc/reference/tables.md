# Tabelas

As tabelas são os principais contêineres de dados no ` Data Studio `. Cada tabela possui um fluxo de transformação que define como os dados são processados, desde a fonte até a saída.

## Tipos de tabelas

|  |  |  |
| --- | --- | --- |
| **Tipo de tabela** | **Descrição** | **Caso de uso** |
| Transformar tabela | Tabela padrão criada a partir do upload de um arquivo ou de uma tabela existente | Tratamento geral de dados |
| Tabela editável | Tabela que aceita entradas manuais do usuário | Ajustes orçamentários, intervenções manuais |
| Tabela editável com recursos avançados | Tabela editável baseada em uma tabela de transformação de origem | Modificações de sobreposição nos dados de origem |
| Tabela gerada | Tabela criada a partir de uma tabela existente com atualizações vinculadas | Conjuntos de dados derivados com atualização automática |
| Conjunto de dados mestre | Tabela de esquema canônico para conformidade da aplicação | Mestre de fontes de custo, Plano de contas |
| Tabela publicada | Tabela configurada para exportação externa | Saída de dados para outros sistemas |
