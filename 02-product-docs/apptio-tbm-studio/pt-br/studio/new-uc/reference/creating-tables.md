# Criação de tabelas

**Criar uma nova tabela**

**Navegação:** Guia Início > Novo > Tabela

**Parâmetros:**

|  |  |  |
| --- | --- | --- |
| **Parâmetro** | **Necessário** | **Descrição** |
| Nome | Sim | Identificador único da tabela |
| Categoria | Sim | Organização de pastas no Explorador de Projetos; é possível usar as existentes ou criar novas |

- Clique na guia **"Página inicial"**
- Clique em **Novo** > **Tabela**
- Digite o nome da tabela
- Digite ou selecione **uma categoria**
- Clique **em OK**

O sistema cria a tabela e exibe a etapa “Fonte” no pipeline de transformação.
