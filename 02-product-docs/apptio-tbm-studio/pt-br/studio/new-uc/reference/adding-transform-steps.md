# Adicionando etapas de transformação

- Selecione uma tabela no Explorador de Projetos
- Mova o cursor até a linha divisória entre duas etapas existentes
- Clique no

  ![ícone de mais](../../../../../../03-media/apptio-tbm-studio/resources/images/icons/icon-add.jpg)

  ícone na extremidade direita do painel “Etapas de transformação”
- Selecione o tipo de etapa a ser adicionada

**Reordenação:** Arraste as etapas para novas posições (exceto Origem, Carregar, Importar, Tabela, Modelo)

**Tópico principal:** [Etapas da transformação](../../../studio/new-uc/reference/transform-steps.html)
