# Data Studio Referência

## Visão geral

Data Studio é o ambiente de gerenciamento de dados do TBM Studio onde você importa, transforma e prepara dados para modelagem de custos e geração de relatórios. Esta referência oferece uma documentação completa de todos os recursos do Data Studio, organizada para facilitar a consulta.

Data Studio permite que você:

- Importar dados de arquivos, tabelas existentes, tabelas editáveis e sistemas externos
- Transforme os dados por meio de um fluxo de etapas configurável
- Mapeamento de dados para conjuntos de dados mestres para fins de conformidade da aplicação
- Publicar tabelas para exportação para sistemas externos
- Aplicar segurança no nível da linha para a governança de dados
