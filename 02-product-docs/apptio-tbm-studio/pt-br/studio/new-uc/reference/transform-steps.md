# Etapas da transformação

As etapas de transformação formam um fluxo de trabalho que processa os dados desde a fonte até a saída. As etapas são executadas sequencialmente, de cima para baixo.

## Visão geral do pipeline de transformação

**Etapas padrão** (sempre presentes):

- **Fonte:** Determina o tipo de fonte de dados
- **Tabela:** Exibe o resultado final após todas as transformações

**Etapas para o envio de arquivos** (adicionadas automaticamente):

- **Upload:** Seleção e gerenciamento de arquivos
- **Importar:** Importar configurações
