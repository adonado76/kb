# Permissões no nível do componente

**Ocultar componentes por função**

Componentes individuais do relatório podem ser ocultados para determinadas funções, mantendo-se visíveis para outras. Isso permite criar relatórios multifuncionais que exibem informações diferentes de acordo com a função do usuário.

**Configurando a visibilidade dos componentes:**

1. Selecione o componente no relatório
2. Na guia Relatório, no grupo Avançado, clique em Visibilidade
3. A função atual do usuário é a opção
4. Escolha as funções que devem ter acesso ao componente
5. Salve e verifique o relatório

**Opções de visibilidade**

A caixa de diálogo “Visibilidade dos componentes do relatório” oferece várias condições de visibilidade:

|  |  |
| --- | --- |
| **Opção** | **Comportamento** |
| **O componente contém dados** | Ocultar o componente se ele não tiver dados |
| **A função atual do usuário é** | Mostrar apenas para as funções selecionadas |
| **O texto dinâmico não resulta em "oculto"** | Visibilidade condicional com base em fórmula |
