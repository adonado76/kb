# Opções de layout

## Visão geral

O TBM Studio Report Studio oferece um sistema de layout flexível para posicionar e dimensionar os componentes dos relatórios. Esta seção aborda o sistema de layout completo, incluindo o dimensionamento da tela, o posicionamento dos componentes, os componentes contêineres e a configuração do layout de impressão.

Os relatórios são criados na área de trabalho de edição de relatórios, onde você adiciona e organiza tabelas, gráficos, filtros, botões e outros componentes. O sistema de layout oferece suporte tanto ao posicionamento manual, para um controle preciso, quanto a layouts de fluxo automático, para designs responsivos.
