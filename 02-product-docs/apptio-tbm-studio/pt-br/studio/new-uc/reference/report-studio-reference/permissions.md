# Permissões

Controle quem pode realizar ações de entrada por meio de permissões no nível da linha:

|  |  |  |
| --- | --- | --- |
| **Permissão** | **Descrição** | **Padrão** |
| Permissão para adicionar linha | Quem pode adicionar novas linhas | Todos |
| Permissão para duplicar linha | Quem pode duplicar linhas | Todos |
| Permissão para excluir linha | Quem pode excluir linhas individuais | Todos |
| Permissão para excluir todas as linhas | Quem pode excluir todas as linhas | Administrador e Parceiro |
| Permissão para editar linha | Quem pode editar os valores das células | Todos |
| Permissão para publicar linha | Quem pode publicar alterações | Todos |
| Permissão para baixar | Quem pode baixar os dados da tabela | Todos |
| Permissão para fazer upload | Quem pode enviar dados de tabelas | Todos |
