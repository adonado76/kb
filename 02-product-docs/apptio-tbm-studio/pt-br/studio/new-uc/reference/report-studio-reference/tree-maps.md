# Mapas em árvore

Os mapas em árvore exibem dados hierárquicos na forma de retângulos aninhados, em que o tamanho de cada retângulo representa a magnitude relativa do seu valor. Essa visualização é excelente para mostrar proporções e facilitar a identificação de valores atípicos ou categorias dominantes à primeira vista.

**Mapas de árvores de valor único**

Um mapa em árvore básico exibe uma única métrica (como o custo), com os tamanhos dos retângulos representando os valores relativos de cada categoria. Isso é útil para identificar rapidamente quais itens mais contribuem para o total.

**Mapas de árvores de variância**

Os mapas em árvore podem exibir dois valores (como Custo e Orçamento) com cores que indicam a diferença entre eles. Uma escala de cores ajuda os usuários a identificar rapidamente quais itens estão acima ou abaixo do orçamento.

**Cálculo da variação:** (Primeiro valor - Segundo valor) / Segundo valor

**Criando um mapa em árvore**

1. Crie uma tabela com os dados que você deseja visualizar.
2. Na guia Ad Hoc, selecione Mapa em árvore.
3. Use as opções da guia "Mapa em árvore" para configurar a visualização.

\

Observação: os mapas em árvore suportam até 4 colunas na legenda. Não é possível aplicar paletas de cores personalizadas a mapas em árvore.
