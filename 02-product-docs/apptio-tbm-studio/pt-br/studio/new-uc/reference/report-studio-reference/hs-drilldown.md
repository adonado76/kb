# Filtros hierárquicos para detalhamento

Os filtros hierárquicos oferecem uma experiência semelhante à navegação dentro de um único relatório, permitindo que os usuários explorem as hierarquias de dados. Ao contrário da navegação por detalhamento para outros relatórios, os filtros hierárquicos filtram o relatório atual em níveis crescentes de detalhe.

**Criação de um filtro hierárquico**

1. Crie um grupo de perspectivas personalizado com os campos ordenados do nível mais alto para o mais baixo
2. Renomeie os campos com prefixos numéricos para controlar a ordem (por exemplo, 01\_Type, 02\_SubType, 03\_OS )
3. Na guia Relatório, clique em Segmentador de linhas
4. Arraste o grupo de perspectiva para a área “Dividir por”
