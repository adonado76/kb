# Opções de classificação

As tabelas podem ser classificadas por uma ou mais colunas, em ordem crescente ou decrescente. Acesse os controles de classificação na guia Dados.

**Classificação em uma coluna**

1. Selecione a coluna pela qual deseja ordenar clicando no cabeçalho dela.
2. Clique no ícone "Ascendente" ou "Descendente" na guia "Dados".

**Classificação por várias colunas**

1. Clique no ícone "Ordenar" na guia "Dados".
2. Na caixa de diálogo "Ordenar", selecione até quatro colunas e a direção da ordenação.
3. Clique em OK para aplicar a ordenação.

|  |  |
| --- | --- |
| **Opção de classificação** | **Descrição** |
| Crescente | Por ordem alfabética para texto, do menor para o maior para números, da mais antiga para a mais recente para datas. |
| Decrescente | De Z a A para texto, do maior para o menor para números, do mais recente para o mais antigo para datas. |
| Número máximo de colunas | É possível usar até 4 colunas para a classificação por várias colunas. |
| Classificação por subtotal | Quando a classificação é feita por uma coluna de valor em uma tabela com subtotais, a classificação se aplica aos valores dos subtotais. |
