# Formatação de tabelas editáveis

O estilo visual ajuda os usuários a identificar células editáveis e a compreender o estado dos dados.

**Indicadores visuais**

- **Novas linhas:** destacadas em laranja até serem salvas
- **Células modificadas:** visíveis na exibição “Mostrar alterações”
- **Erros de validação:** células destacadas em vermelho com mensagens de erro
- **Campos obrigatórios:** Indicador visual para colunas obrigatórias
- **Células bloqueadas:** as células que estão sendo editadas por outros usuários aparecem bloqueadas

**Formatação de tabelas**

As opções padrão de formatação de tabelas se aplicam às tabelas editáveis:

- Largura das colunas, alinhamento e formatação de números
- Estilos e bordas de cabeçalho
- Cores alternadas nas linhas
- Controles de classificação e filtragem
