# Criação de coleções

**Criando uma nova coleção**

Somente administradores podem criar coleções de relatórios, e essas coleções só podem ser criadas no projeto "Costing Standard". Depois de criadas, as coleções ficam disponíveis em todo o projeto, e outros usuários com as permissões adequadas podem adicionar e remover relatórios da coleção.

**Para criar uma coleção de relatórios personalizada:**

1. Na guia **Projeto**, no grupo **Dados do projeto**, clique em **Coleções de relatórios**.
2. Na caixa de diálogo “**Gerenciar coleções de relatórios** ”, clique em “**Adicionar coleção** ”.
3. Uma nova entrada chamada "Nova coleção de relatórios" é adicionada à lista de coleções.

1. Para alterar o nome, clique na entrada “Nova coleção de relatórios” na lista e, em seguida, clique no nome da coleção no campo **“Coleção selecionada”** para editá-lo.
2. Clique em **Fechar** para salvar a coleção.

**Observação:** não é possível excluir coleções de relatórios padrão (OOTB). Somente coleções personalizadas podem ser excluídas.

**Metadados da coleção**

Cada coleção de relatórios possui os seguintes metadados configuráveis:

|  |  |
| --- | --- |
| **Propriedade** | **Descrição** |
| Nome | O nome de exibição mostrado no navegador de coleções e nas caixas de diálogo de gerenciamento. Escolha um nome claro e descritivo que ajude os usuários a entender o objetivo da coleção. |
| Paleta de cores | (A partir do R12.10.10 ) Uma paleta de cores personalizada ou predefinida, aplicada a todos os gráficos nos relatórios da coleção. Relatórios individuais podem substituir a paleta de coleta. |
| Relatórios | A lista de relatórios atribuídos à coleção. Os relatórios são exibidos na ordem em que aparecem nesta lista. |

**Excluindo uma coleção**

**Para excluir uma coleção de relatórios personalizados:**

1. Na guia **Projeto**, no grupo **Dados do projeto**, clique em **Coleções de relatórios**.
2. Na caixa de diálogo “**Gerenciar coleções de relatórios** ”, selecione a coleção que deseja excluir.
3. Clique no botão “**Excluir coleção** ”.

Aviso: ao excluir uma coleção, o agrupamento é removido, mas os relatórios em si não são excluídos. Os relatórios que estavam na coleção excluída não terão mais nenhuma atribuição de coleção.
