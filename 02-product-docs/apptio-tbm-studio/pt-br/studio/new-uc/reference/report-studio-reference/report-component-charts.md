# Componentes do relatório: Gráficos

Os gráficos transformam seus dados de custos em insights visuais que transmitem informações financeiras complexas de forma rápida e eficaz. O TBM Studio oferece um conjunto abrangente de tipos de gráficos otimizados para a análise financeira de TI, desde simples gráficos de barras que comparam os custos dos departamentos até sofisticados gráficos em cascata que explicam os desvios orçamentários.

## Visão geral dos tipos de gráficos

O TBM Studio oferece os seguintes tipos de gráficos, acessíveis na guia Ad Hoc após a criação de um componente de gráfico:

|  |  |  |
| --- | --- | --- |
| **Tipo de gráfico** | **Variantes** | **Ideal para** |
| Barras | Barra horizontal, barra horizontal empilhada | Comparando categorias lado a lado |
| Coluna | Coluna vertical, coluna empilhada | Comparando valores entre diferentes períodos |
| Linha | Linha padrão, linha de tendência | Mostrando tendências ao longo do tempo |
| Gráfico de pizza | Padrão, Meia, Rosca, Meia Rosca | Mostrando proporções de um todo |
| Tendência | Gráfico de tendência com linhas e projeções | Previsão e análise histórica |
| Medidor | Medidor de valor único | Indicadores de progresso de KPI |
| Mapa em árvore | Retângulos aninhados | Dados hierárquicos e proporções relativas |
| Queda de água | Tabela de transição com etapas | Explicando as variações entre os valores |
| Radar | Radar preenchido, Radar com marcadores | Comparações multidimensionais |
| Sobreposição | Coluna/Linha combinadas | Comparando duas séries de dados relacionadas |
