# Aplicar e remover relatórios mestre

**Aplicar um relatório mestre**

Para aplicar um relatório mestre a um relatório personalizado:

1. **Confira** o relatório personalizado no modo de edição.
2. Vá até a guia **Relatório** na faixa de opções.
3. Abra a lista suspensa “**Masters** ”.
4. Selecione o relatório mestre que deseja aplicar.
5. Salve e verifique o relatório.

**Excluir um relatório mestre**

Para remover um relatório mestre de um relatório personalizado:

1. **Confira** o relatório personalizado.
2. Vá até a guia **Relatório**.
3. Abra a lista suspensa “**Masters** ”.
4. Selecione **“Nenhum”** na parte inferior da lista.
5. Salve e verifique o relatório.

**Exclusão de relatórios mestre**

Para excluir um relatório mestre personalizado:

1. No modo Edição de relatórios, clique na seção **Relatórios** no **Explorador de** projetos.
2. Filtre a lista de relatórios por **Relatórios Mestres** para exibir apenas os relatórios mestres.
3. **Selecione** o relatório principal que deseja excluir.
4. Clique em **“Excluir”** na guia “**Página inicial** ”.
5. **Confirme** para concluir a exclusão.

Aviso: a exclusão de um relatório mestre pode afetar todos os relatórios filhos que o utilizam. Antes de excluir, verifique quais relatórios dependem do mestre e remova-o desses relatórios ou aplique um mestre diferente.
