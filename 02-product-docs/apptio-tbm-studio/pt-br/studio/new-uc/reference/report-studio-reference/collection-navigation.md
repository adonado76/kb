# Navegação pela coleção

**Componente de coleta de relatórios do Navigator**

O navegador de coleções de relatórios é um componente de relatórios que oferece aos usuários uma maneira fácil de navegar entre os relatórios de uma coleção. Quando você adiciona um relatório a uma coleção, o componente de navegação é automaticamente adicionado ao relatório.

O navegador geralmente aparece como uma barra horizontal na parte superior do relatório, exibindo os nomes de todos os relatórios visíveis na coleção. Os usuários podem clicar no nome de qualquer relatório para acessá-lo diretamente.

**Recursos de navegação**

O navegador de coleta de relatórios oferece os seguintes recursos de navegação:

|  |  |
| --- | --- |
| **Funcionalidade** | **Descrição** |
| Navegação direta | Clique no nome de qualquer relatório no navegador para acessar diretamente esse relatório. |
| Indicador de relatório atual | O relatório atualmente visualizado é destacado no navegador, ajudando os usuários a manter o contexto. |
| Contexto do filtro | As seleções e os filtros do Slicer podem, opcionalmente, ser transferidos entre relatórios durante a navegação. |
| Exibição responsiva | O navegador se adapta à largura disponível da tela, ajustando-se ou compactando-se conforme necessário. |

**Métodos alternativos de navegação**

Além do navegador de coleções, o TBM Studio oferece outros métodos de navegação entre relatórios:

- **Botões:** Botões clicáveis que levam a relatórios específicos
- **Links codificados:** links com sintaxe do tipo wiki em caixas de texto ou botões HTML
- **Links para relatórios:** Links a partir de valores de tabelas ou elementos de gráficos para relatórios relacionados
