# Gráficos sobrepostos (combinados)

Os gráficos sobrepostos exibem duas séries de dados em um único gráfico, geralmente combinando colunas ou barras com linhas. Isso permite a comparação direta de métricas relacionadas, como o orçamento em relação aos resultados reais ou a capacidade em relação à utilização.

**Opções de visualização de camadas**

Cada camada em um gráfico sobreposto pode ser exibida como:

- Colunas: Barras individuais para cada valor.
- Colunas empilhadas: Barras segmentadas que mostram a composição.
- Linhas: Pontos de dados interligados para indicar tendências.

**Criação de um gráfico sobreposto**

1. Na guia Relatório, clique em Sobreposição.
2. Clique na guia “Sobreposição” no painel de configuração.
3. Clique em “Editar camada base” e, em seguida, arraste os campos para as áreas “Eixo” e “Valores”.
4. Selecione um tipo de visualização para a camada base.
5. Clique em “Editar Camada 1” e repita a configuração para a segunda série.
6. Adicione filtros, se necessário, para simplificar o gráfico.

**Eixo compartilhado:** Disponível nas propriedades do gráfico, essa opção torna as escalas dos eixos esquerdo e direito iguais, o que é útil ao comparar valores de magnitude semelhante.
