# Estilo da navegação

Um estilo consistente ajuda os usuários a reconhecer os elementos de navegação e a compreender sua finalidade. Aplique estas diretrizes de estilo para criar uma experiência de navegação profissional e intuitiva.

**Melhores práticas para o design de botões**

- Use cores consistentes para ações semelhantes (por exemplo, azul para navegação, verde para aprovação)
- Utilize tamanhos de texto adequados de acordo com a importância dos botões (maiores para ações principais)
- Adicione dicas de ferramentas informativas para explicar as funções dos botões
- Agrupe botões relacionados usando caixas de grupo

**Estilo dos links**

Os links nas tabelas do relatório são exibidos automaticamente em azul, seguindo as convenções padrão da web. Nas caixas de texto HTML, é possível aplicar estilos CSS personalizados aos links, embora isso exija conhecimentos de HTML/CSS.

**Contêineres de navegação em caixas de grupo**

Use caixas de grupo para organizar vários elementos de navegação juntos. As caixas de grupo podem ter títulos, bordas e cores de fundo que separam visualmente a navegação do conteúdo do relatório.
