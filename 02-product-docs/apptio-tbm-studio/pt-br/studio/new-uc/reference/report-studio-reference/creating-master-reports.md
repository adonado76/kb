# Criação de relatórios mestre

**Criação de um novo relatório mestre**

Para criar um novo relatório mestre:

1. Crie um novo relatório ou selecione um relatório existente que você deseja converter em um relatório mestre.
2. Na guia Relatório, no grupo Avançado, clique em Relatório mestre para ativar a opção de relatório mestre.
3. Adicione os componentes que deseja que apareçam em todos os relatórios filhos (caixas de grupo, botões de navegação, elementos de identidade visual, etc.).
4. **Clique em “Salvar”** na guia “**Página inicial** ”.
5. **Marque** o relatório para disponibilizá-lo para outros relatórios.

**Conversão de um relatório existente em relatório mestre**

Qualquer relatório existente pode ser convertido em um relatório mestre:

1. **Selecione** o relatório que você deseja usar como modelo.
2. Vá até a guia **Relatório** na faixa de opções.
3. No grupo “**Avançado** ”, clique na opção “**Relatório mestre** ”.
4. Salve e verifique o relatório.

Observação: assim que um relatório for designado como relatório mestre, ele aparecerá no menu suspenso “Mestres” na guia “Relatórios” para todos os outros relatórios do projeto.

**Relatórios padrão**

O TBM Studio inclui vários relatórios padrão que vêm junto com o produto. Esses modelos pré-definidos estão disponíveis no menu suspenso **“Modelos”,** na guia “**Relatórios** ”. Você pode usar esses relatórios-padrão tal como estão ou modificá-los e salvá-los com um novo nome usando a opção “**Salvar como** ”.

Aviso: Não é possível excluir relatórios padrão que acompanham o produto. Você só pode excluir relatórios-mestre personalizados que tenha criado.
