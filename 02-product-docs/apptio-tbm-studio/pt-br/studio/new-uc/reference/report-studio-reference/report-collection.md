# Coleções de relatórios

Agrupe relatórios relacionados para que os usuários possam navegar facilmente entre relatórios sobre um tema ou perspectiva semelhante. As coleções melhoram a visibilidade dos relatórios, organizando-os em grupos lógicos e proporcionando uma experiência de navegação consistente. Os administradores configuram coleções de relatórios para controlar quais funções de usuário podem acessar os relatórios dentro de cada coleção.
