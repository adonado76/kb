# Configurações de ativação e pré-carregamento

Essas configurações determinam se um relatório é incluído nos cálculos do sistema e se ele é pré-calculado para agilizar o acesso do usuário.

|  |  |
| --- | --- |
| **Configuração** | **Descrição e impacto** |
| Ativo | Quando marcada, o relatório é incluído nos cálculos do sistema e seus dados são pré-carregados. Quando a caixa de seleção não estiver marcada, o relatório fica desativado e não é gerado. Use esta opção para desativar temporariamente um relatório sem excluí-lo ou para impedir que relatórios incompletos sejam calculados. |
| Pré-carregar | Os relatórios com a opção “Ativo” ativada têm seus dados pré-calculados durante a geração. Se o seu domínio estiver configurado para exigir o pré-carregamento, os usuários não poderão visualizar relatórios que não tenham sido pré-carregados. O pré-carregamento melhora a experiência do usuário em relatórios com grande volume de dados, garantindo que os cálculos estejam concluídos antes que os usuários acessem o relatório. |

**Localização:** guia Relatório > grupo Avançado > caixa de seleção Ativo
