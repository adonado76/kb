# Navegação e detalhamento

Os gráficos podem ser configurados para direcionar os usuários a relatórios relacionados quando eles clicam em elementos do gráfico. Essa funcionalidade de detalhamento permite que os usuários explorem os dados, desde o nível de resumo até os detalhes.

**Ativando a navegação no gráfico**

1. Selecione o gráfico.
2. Na guia Ad Hoc, clique em Drill na seção Navegação.
3. Ativar navegação: marque esta opção para tornar os elementos do gráfico clicáveis.
4. Ir para: Selecione o relatório desejado no menu suspenso.

**Opções de navegação**

|  |  |
| --- | --- |
| **Opção** | **Descrição** |
| Ativar navegação | Cria elementos de gráfico (barras, fatias, etc.) funcionam como links de navegação. |
| Abrir como janela modal | Exibe o relatório de destino em uma janela pop-up. Os usuários podem fechá-lo para voltar ao relatório original. |
| Ir para | Selecione o relatório de destino a partir de qualquer relatório do projeto. |
| Exercícios por métrica | Ativa a detalhamento dos valores exibidos nas fatias do gráfico de pizza e nas barras do gráfico de barras. |
| Filtrar na linha selecionada | Envia o valor selecionado como filtro para o relatório de destino. |
| Filtrar com base nas seleções do segmentador aplicado | Transfere as seleções atuais do filtro para o relatório de destino. |
| Incluir os filtros do relatório atual | Aplica os filtros do relatório de origem ao relatório de destino. |

Observação: Para que a navegação funcione com filtros, os dados de ambos os relatórios devem estar vinculados por meio do mecanismo de inferência.
