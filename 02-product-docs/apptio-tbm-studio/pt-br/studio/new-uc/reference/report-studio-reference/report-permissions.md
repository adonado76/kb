# Permissões do relatório

## Visão geral

As permissões de relatórios permitem controlar quem pode visualizar, editar e administrar relatórios no TBM Studio. O sistema de permissões opera em vários níveis — desde atribuições de funções em todo o domínio até a visibilidade de componentes individuais —, permitindo um controle granular sobre o acesso aos relatórios, ao mesmo tempo em que mantém os requisitos de segurança e governança.

Esta seção aborda a arquitetura completa de permissões para relatórios, incluindo a relação entre as permissões de relatórios e a segurança no nível da linha (RLS), as melhores práticas para o projeto de permissões e a resolução de problemas comuns de acesso.
