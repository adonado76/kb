# Relatórios principais

## Visão geral

Os relatórios-modelo funcionam como modelos reutilizáveis que garantem um layout, uma identidade visual e elementos de design consistentes em vários relatórios no TBM Studio. Ao criar um relatório mestre e aplicá-lo aos seus relatórios personalizados, você garante a consistência visual e reduz a carga de manutenção associada à atualização de elementos comuns em todo o seu conjunto de relatórios.
