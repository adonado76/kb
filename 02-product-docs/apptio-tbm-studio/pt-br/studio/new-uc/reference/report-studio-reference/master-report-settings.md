# Configuração do Relatório Principal

Os relatórios-modelo funcionam como modelos que podem ser aplicados a vários relatórios, garantindo um layout e uma aparência consistentes em todo um conjunto de relatórios.

**Principais características:**

- Os relatórios principais são sempre relatórios de nível superior
- Os componentes de um relatório mestre aparecem como "fundo" — visíveis, mas não editáveis nos relatórios derivados
- Os componentes adicionados aos relatórios derivados são sobrepostos ao layout mestre
- Os relatórios padrão são fornecidos pelo sistema e não podem ser excluídos

**Para criar um relatório mestre:**

1. Confira o relatório
2. Vá até a guia Relatório > grupo Avançado
3. Clique na opção Relatório principal
4. Salvar e fazer o check-in
