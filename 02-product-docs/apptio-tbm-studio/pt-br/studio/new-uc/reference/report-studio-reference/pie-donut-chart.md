# Gráficos de pizza e de rosca

Os gráficos de pizza apresentam os dados como fatias proporcionais de um círculo, o que os torna excelentes para mostrar como as partes contribuem para o todo. O TBM Studio oferece quatro variantes de gráficos de pizza para atender a diferentes necessidades de apresentação.

**Variantes do gráfico de pizza**

|  |  |
| --- | --- |
| **Variante** | **Descrição** |
| Padrão | Gráfico circular completo, apresentando todas as fatias como segmentos proporcionais. |
| Meia-final | Gráfico de pizza semicircular que exibe dados em um arco de 180 graus. Útil quando o espaço vertical é limitado. |
| Rosca | Círculo completo com o centro oco, deixando espaço para KPIs ou um texto resumido. |
| Meia rosquinha | Gráfico de rosca semicircular que combina o formato de semicírculo com um centro oco. |

**Comportamento do gráfico de pizza**

**Importante:** Os gráficos de pizza só funcionam com números positivos. Se seus dados contiverem valores negativos, será exibida uma mensagem informando que há dados inadequados no lugar do gráfico.

Um gráfico de pizza sempre mostra fatias que representam o total (100%) do valor representado no gráfico. Se você limitar a exibição para mostrar apenas os itens principais (usando a opção “Máximo de linhas”), uma fatia adicional chamada “Outros” será adicionada automaticamente para representar a porcentagem restante. A outra fatia não pode ser ocultada.

**Corte explodido**

Você pode destacar uma fatia específica "expandindo-a", o que faz com que ela se destaque ligeiramente do resto do gráfico circular. Existem duas maneiras de criar uma seção explodida:

- Clique com o botão direito do mouse em uma fatia do gráfico e selecione “Explodir” no menu de contexto.
- Digite o nome da fatia no campo “Fatia Descomposta” na caixa de diálogo “Propriedades do gráfico”.

**Propriedades do gráfico de pizza**

As propriedades do gráfico de pizza estão disponíveis na guia "Pie" da caixa de diálogo "Propriedades". As principais opções incluem o posicionamento das etiquetas, as porcentagens das fatias e o estilo visual. Experimente as opções para obter a apresentação desejada.
