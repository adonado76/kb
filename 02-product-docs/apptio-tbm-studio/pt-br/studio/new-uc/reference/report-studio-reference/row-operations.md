# Operações em linhas

Os componentes de tabela editáveis suportam várias operações no nível da linha, controladas por permissões.

**Adicionar novas linhas**

Os usuários com permissões adequadas podem adicionar linhas:

1. Clique no botão “Adicionar linha” na parte inferior da tabela
2. Uma nova linha aparece na parte inferior, destacada em laranja
3. Insira os valores nas colunas editáveis
4. Clique em Salvar para gravar a nova linha

Ative a opção “Adicionar linha” na faixa de opções “Tabelas editáveis” para exibir uma caixa de diálogo de formulário para inserir dados em uma nova linha.

**Excluir linhas**

Para excluir linhas:

1. Clique com o botão direito do mouse na linha para excluir
2. Selecione “Excluir linha” no menu de contexto
3. A linha está marcada para exclusão (exibida em “Mostrar alterações”)
4. Clique em Salvar para confirmar a exclusão

Para exclusões em massa, ative a opção “Ativar coluna de caixa de seleção” em Propriedades avançadas. Selecione várias linhas usando as caixas de seleção, clique com o botão direito do mouse e selecione “Excluir linha”.

**Linhas duplicadas**

O recurso "Linha duplicada" simplifica os cenários de mapeamento dividido:

- Clique com o botão direito do mouse em uma linha e selecione “Duplicar linha”
- Uma cópia aparece abaixo da linha original
- Todas as colunas visíveis e ocultas são duplicadas
- As colunas com restrições de exclusividade são desmarcadas e destacadas conforme necessário

Aviso: *Em tabelas editáveis avançadas, os usuários não podem excluir linhas provenientes da tabela de transformação; apenas as linhas adicionadas podem ser excluídas.*
