# Gráficos de linhas

Os gráficos de linhas conectam pontos de dados com linhas contínuas, tornando-os ideais para mostrar tendências e padrões ao longo do tempo. Eles são excelentes para mostrar a taxa de variação e podem exibir várias séries simultaneamente para comparação.

**Gráfico de linhas padrão**

O gráfico de linhas padrão representa pontos de dados e os conecta por meio de segmentos de linha reta. Cada série aparece em uma linha separada, permitindo que os espectadores comparem tendências entre várias categorias.

**Quando usar:** Acompanhar tendências de custos ao longo do tempo, comparar várias métricas ou unidades de negócios nos mesmos períodos, ou identificar padrões e anomalias em dados de séries temporais.

**Interatividade do gráfico de linhas**

**Recurso de zoom:** Nos gráficos de linha, é possível ampliar períodos específicos mantendo o botão do mouse pressionado e arrastando horizontalmente pelo gráfico. Uma opção "Redefinir zoom" aparece no canto superior direito para voltar à visualização completa. Os níveis de zoom são reiniciados quando se sai do relatório.

**Filtragem da legenda:** Clique em qualquer item da legenda para ocultar essa série. O item aparece desativado. Clique novamente para restaurar a visibilidade. Isso permite que os usuários se concentrem em séries de dados específicas sem alterar a configuração do gráfico.

**Propriedades específicas do gráfico de linhas**

- Incluir zero no eixo Y: força o eixo Y a começar em 0. Quando desmarcada, o eixo Y pode começar com um valor mais alto para exibir melhor pequenas variações.
- Remover zeros à esquerda e à direita: Remove quaisquer zeros no início ou no final do gráfico para evitar que a linha caia até o eixo X no início e no final.
