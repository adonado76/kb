# Gráficos de radar

Os gráficos de radar (também chamados de gráficos em teia de aranha) comparam várias características entre duas ou mais entidades por meio de eixos que se irradiam a partir de um ponto central. São úteis para comparar os valores orçados com os reais em vários departamentos ou para avaliar itens com base em diversos critérios.

**Tipos de gráficos radiais**

- Radar preenchido: exibe áreas poligonais preenchidas que conectam pontos de dados. Ideal para comparar três características ou menos, a fim de evitar sobrecarregamento visual.
- Radar com marcadores: mostra apenas os pontos de dados e as linhas de conexão, sem preenchimento. Adequado para qualquer número de características.

**Criação de um gráfico radial**

1. Na guia Relatório, clique em Gráfico.
2. Arraste um campo para a área Eixo (isso define as características comparadas).
3. Arraste um ou mais campos para a área Valores.
4. Na guia Ad Hoc, clique em Radar.
5. No menu suspenso "Radar", selecione "Radar preenchido" ou "Radar com marcadores".

**Comportamento do pivô:** Quando um pivô é aplicado a um gráfico de radar, ele afeta o eixo, e não as linhas (ao contrário de outros tipos de gráfico).

**Número máximo de linhas:** controla o número de eixos exibidos. O valor padrão é seis. Ao contrário de outros gráficos, os gráficos de radar não exibem uma categoria "Outros" para itens excedentes.
