# Posicionamento de componentes

**Posicionamento manual (arrastar e soltar)**

O principal método para posicionar componentes é o arrastar e soltar:

- **Mover um componente:** clique no cabeçalho do componente e arraste-o para um novo local.
- **Redimensionar um componente:** Arraste as bordas e os cantos à esquerda, à direita ou na parte inferior.
- **Posicionamento preciso:** clique com o botão direito do mouse no cabeçalho do componente e selecione “Posição e tamanho” para abrir a caixa de diálogo “Definir posição e tamanho”.

A caixa de diálogo “Definir posição e tamanho” permite especificar coordenadas exatas:

- **Coordenada X:** Posição horizontal a partir da borda esquerda (em pixels)
- **Coordenada Y:** posição vertical a partir da borda superior (em pixels)
- **Largura:** Largura do componente
- **Altura:** Altura do componente

**Opções de posicionamento dinâmico**

Para layouts responsivos, use o menu Dinâmico na guia Formatar. Essas opções controlam a forma como os componentes se posicionam em relação ao seu contêiner (a área do relatório, uma caixa de grupo ou um componente com abas).

|  |  |
| --- | --- |
| **Opção** | **Descrição** |
| **Dinâmico desativado** | Desativa o redimensionamento e o posicionamento dinâmicos; o componente permanece onde foi colocado |
| **Centro** | Alinha o componente ao centro do seu contêiner sem alterar o tamanho |
| **Centrar e Crescer** | Centraliza o componente e permite que ele se expanda simetricamente a partir do centro |
| **Preencha corretamente** | Estende o componente até a borda direita do seu contêiner |
| **Preencher à direita e na parte inferior** | Estende o componente até as bordas direita e inferior |
| **Atracar à direita** | Mova o componente de forma que sua borda direita toque na borda direita do contêiner |
| **Fundo do cais** | Mova o componente de forma que sua borda inferior toque a borda inferior do contêiner |

**Para aplicar o posicionamento dinâmico:**

1. Selecione o componente.
2. Abra o menu Dinâmico na guia Formatar.
3. Selecione uma das opções de posicionamento.

Observação: Apenas uma opção de posicionamento dinâmico pode ser aplicada a um componente por vez.
