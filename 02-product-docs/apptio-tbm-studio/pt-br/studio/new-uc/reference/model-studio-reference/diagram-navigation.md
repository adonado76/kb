# Navegação pelo diagrama

**Ações de navegação**

|  |  |  |
| --- | --- | --- |
| **Ação** | **Instruções** | **Resultado** |
| Ver detalhes do objeto | Clique em um nó | Abre a visualização de tabela única para esse objeto |
| Navegue até o destino | Clique na seta no driver de alocação | Leva ao objeto de origem |
| Rastrear fluxo de custos | Clique em uma linha de alocação | Destaca a trajetória de custos no modelo |
| Realizar drill down | Clique no menu > Detalhar | Expande o nó para mostrar os valores das colunas |
| Voltar à visualização em tabela | Clique em Fechar ou na seta de retorno | Recolhe a expansão e volta à visualização do objeto |
| Zoom | Roda do mouse ou controles de zoom | Aumentar/diminuir o zoom no diagrama |
| Percorrer | Clique e arraste o plano de fundo | Mova a visualização do diagrama |
