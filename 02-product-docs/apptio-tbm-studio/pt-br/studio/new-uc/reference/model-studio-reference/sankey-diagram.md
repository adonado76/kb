# Conceitos do diagrama de Sankey

O diagrama de Sankey é utilizado tanto na visualização de tabela única quanto na visualização de relatório de modelo para visualizar o fluxo de valores.

**Elementos do diagrama**

|  |  |
| --- | --- |
| **Elemento** | **Descrição** |
| Nós (retângulos) | Representam objetos do modelo ou colunas da tabela. A largura indica o valor total. |
| Linhas de fluxo | Representar alocações entre objetos. Largura proporcional ao valor atribuído. |
| Direção do fluxo | O valor flui da esquerda para a direita (das fontes para os destinos) |
| Etiquetas de valor | Exibir valores métricos em nós e fluxos |
