# Opções de visualização

**Recursos de visualização de relatórios**

|  |  |  |
| --- | --- | --- |
| **Funcionalidade** | **Ação** | **Descrição** |
| Ver motoristas/alocações | Clique em uma tabela no relatório | Mostra os drivers e as alocações dessa tabela |
| Rastrear o fluxo de valores | Clique nos elementos da tabela em sequência | Mostra o caminho do custo pelo modelo ( v12.1+ ) |
| Perfurar até a coluna | Clique no menu > Detalhar > selecione a coluna | Expande a tabela para mostrar os valores das colunas |
| Ver item específico | Clique no item após a filtragem | Mostra as alocações para esse item específico |
| Voltar à visualização em tabela | Clique na seta de retorno no canto do nível | Reconfigura a visualização detalhada |
| Alterar métrica | Usar o menu suspenso de seleção de unidades métricas | Alterações na métrica exibida |

**Alocações dentro do nível**

A partir da versão v12.2.2, as alocações entre tabelas no mesmo nível são exibidas com posicionamento deslocado, em vez de linhas em loop.
