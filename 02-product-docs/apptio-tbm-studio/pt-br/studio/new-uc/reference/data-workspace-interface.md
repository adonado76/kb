# Interface do Data Workspace

## Explorador de projetos

**Objetivo:** Navegar e gerenciar todos os documentos do projeto

**Tipos de documentos:**

- Tabelas
- Tabelas editáveis
- Métricas
- Perspectivas
- Tabelas publicadas
- Relatórios
- Tempo

## Painel de etapas de transformação

**Objetivo:** Visualizar e modificar o pipeline de transformação de tabelas

**Recursos:**

- Exibição sequencial das etapas
- Inserção de um degrau entre os degraus existentes
- Reordenação por arrastar e soltar (com restrições)
- Indicadores de erro com contagem
