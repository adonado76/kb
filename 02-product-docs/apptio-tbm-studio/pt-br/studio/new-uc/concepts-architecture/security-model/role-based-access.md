# Controle de acesso baseado em funções (RBAC)

A camada de autorização no TBM Studio utiliza um modelo de controle de acesso baseado em funções (RBAC). Em vez de atribuir permissões individuais a cada usuário, as permissões são agrupadas em funções, e os usuários são atribuídos a uma ou mais funções. Isso simplifica a administração e garante a consistência — quando for necessário alterar as permissões de um grupo de usuários, basta modificar a função, em vez de atualizar cada usuário individualmente.
