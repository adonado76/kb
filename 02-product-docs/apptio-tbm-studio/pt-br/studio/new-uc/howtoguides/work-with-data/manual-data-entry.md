# Entrada manual de dados

|  |  |
| --- | --- |
| **Tipo de Conteúdo** | Guias práticos |
| **Público-alvo** | Analistas de negócios, equipes de TI e finanças, administradores |
| **Pré-requisitos** | Data Studio acesso e compreensão das tabelas do TBM Studio e dos pipelines de transformação |

## Visão geral

As tabelas editáveis permitem que você insira e atualize dados manualmente diretamente no TBM Studio. Elas funcionam como camadas de entrada de dados editáveis pelo usuário, que alimentam as tabelas de transformação e, por fim, o seu modelo de custos. Utilize tabelas editáveis para cenários como mapeamentos de mão de obra, atributos de centros de custo, classificações de fornecedores, explicações sobre variações orçamentárias e quaisquer dados de referência que exijam intervenção humana ou enriquecimento.

**Diferencial principal:** O TBM Studio oferece suporte a dois tipos de tabelas editáveis com finalidades fundamentalmente diferentes:

- [Tabelas em branco editáveis](create-blank-et.html) : você cria todas as linhas e colunas do zero. Ideal para pequenos conjuntos de dados de referência em que cada dado provém da entrada do usuário.
- [Tabelas editáveis enriquecidas](create-eet.html) : o sistema gera linhas a partir de uma tabela de transformação existente, e os usuários adicionam anotações ou classificações. Ideal para complementar os dados gerados pelo sistema com o julgamento humano.

Entender quando usar cada tipo é fundamental para criar pipelines de dados fáceis de manter.
