# Modelagem avançada

|  |  |
| --- | --- |
| **Tipo de Conteúdo** | Guias práticos |
| **Público-alvo** | Equipes de finanças de TI, analistas de negócios (nível intermediário a avançado) |
| **Pré-requisitos** | Concluiu os cursos 3.2.1 (Noções básicas de modelos) e 3.2.2 (Criação de alocações); familiaridade com alocações de valor ponderado simples e de distribuição uniforme |

Esta seção aborda técnicas avançadas de modelagem para usuários que já dominam as alocações básicas e precisam de um controle mais preciso sobre a forma como os custos são distribuídos em seu modelo. Você aprenderá a usar a lógica de correspondência para conectar dinamicamente linhas de origem e de destino, configurar colunas de ponderação para distribuição proporcional, rastrear fluxos de custos para resolução de problemas e criar relatórios de modelo (diagramas de Sankey) para comunicação com as partes interessadas.

- **[Configurar colunas de ponderação](../../../../studio/new-uc/howtoguides/build-cost-model/config-wc.html)**
- **[Acompanhar os fluxos de custos ao longo do modelo](../../../../studio/new-uc/howtoguides/build-cost-model/trace-cost-flow.html)**
