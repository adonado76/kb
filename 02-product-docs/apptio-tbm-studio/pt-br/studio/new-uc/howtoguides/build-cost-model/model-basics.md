# Noções básicas sobre modelos

|  |  |
| --- | --- |
| **Tipo de Conteúdo** | Coleção de guias práticos |
| **Público-alvo** | Analistas de negócios, equipes de TI e finanças |
| **Tempo estimado** | 30 a 45 minutos no total |
| **Pré-requisitos** | Conhecimento básico da interface do TBM Studio; pelo menos uma tabela de transformação criada no Data Studio |

Esta seção apresenta as tarefas fundamentais para trabalhar com o modelo de custos no TBM Studio. Antes de poder alocar custos ou criar relatórios do modelo, é necessário entender como adicionar tabelas ao modelo, quais métricas circulam por ele e como visualizar a estrutura geral do fluxo de custos.

**O modelo de custos é o coração do TBM Studio**. Ele utiliza os dados preparados no Data Studio e permite que você defina como os custos, orçamentos e outros valores fluem dos sistemas de origem para as unidades de negócios e serviços. Cada alocação, cada fluxo de custos e cada análise financeira começa com o modelo.

Esta seção contém três guias práticos básicos:

- [Adicionar uma tabela ao modelo](add-tables.html) — Como incorporar suas tabelas de transformação à camada de modelagem
- [Compreender as métricas do modelo (custo, orçamento, previsão)](understand-model-metrics.html) — O que significam custo, orçamento e previsão e como se relacionam
- [Veja o diagrama do modelo](view-model-diagram.html) — Como visualizar e interpretar o fluxo geral de custos

Depois de concluir essas tarefas, você estará pronto para [criar alocações](create-allocation.html).
