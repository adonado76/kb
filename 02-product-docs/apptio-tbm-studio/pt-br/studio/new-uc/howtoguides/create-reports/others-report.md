# Reunindo tudo: seu primeiro relatório

Agora que você já entendeu o básico, veja a seguir um fluxo de trabalho típico para a criação de um relatório:

- [Crie o relatório](report-basic.html) com um nome e uma descrição claros
- [Adicione uma tabela](add-tables-report.html) para mostrar dados detalhados — escolha o objeto de modelo adequado, adicione dimensões às Linhas e adicione métricas aos Valores
- [Insira um gráfico](add-chart-visual.html) para visualizar os principais padrões — escolha o tipo de gráfico adequado para a sua mensagem
- [Adicione filtros](use-slicers.html) para aumentar a interatividade — inclua filtros para as dimensões pelas quais os usuários costumam filtrar
- [Revisar e salvar](others-report.html) — verifique se o relatório é exibido corretamente, teste as interações com os filtros, salve para gravar suas alterações

## Referência rápida: Componentes do relatório

|  |  |  |  |
| --- | --- | --- | --- |
| **Componente** | **Adicionar de** | **Áreas de configuração** | **Ideal para** |
| Tabela | Guia Relatório > Tabela | Linhas, colunas, valores, filtros | Dados detalhados, exportações |
| Gráfico | Guia Relatório > Gráfico | Legenda, Eixos, Valores, Filtros | Padrões visuais, comparações |
| Seletor de linhas | Guia Relatório > Filtro de linhas | Filtrar por, Filtros | Filtragem interativa |
| Fatiador compacto | Guia Relatório > Compact Slicer | Dividir por (vários) | Filtragem com economia de espaço |
