# Resumo

A personalização de relatórios transforma relatórios individuais em um ambiente de relatórios coeso e profissional. As quatro técnicas desta seção funcionam em conjunto:

|  |  |  |
| --- | --- | --- |
| **Técnica** | **Propósito** | **Quem pode configurar** |
| Perspectivas personalizadas | Organize os campos para facilitar a criação de relatórios | Usuário avançado, Administrador |
| Coleções de relatórios | Relatórios relacionados ao grupo para navegação | Admin (criar/excluir), Usuários (adicionar relatórios) |
| Relatórios principais | Manter um layout e uma identidade visual consistentes | Usuário avançado, Administrador |
| Permissões do relatório | Controle quem vê o quê | Administrador (nível de relatório), Usuário avançado (componente) |

**Fluxo de trabalho recomendado:**

1. Crie suas próprias perspectivas com os campos mais úteis para os analistas
2. Crie modelos de relatórios para garantir a consistência da identidade visual
3. Crie relatórios usando perspectivas e modelos
4. Organize os relatórios em coleções lógicas
5. Configure permissões nos níveis de coleção, relatório e componente

Essa base permite que você mantenha um ambiente de relatórios escalável e seguro, capaz de atender a públicos diversos sem deixar de ser fácil de gerenciar.
