# Atribuir funções aos usuários

**Objetivo:** Atribuir funções adequadas aos usuários para controlar seu nível de acesso no TBM Studio.

**Quando usar:** Ao criar novos usuários, quando as responsabilidades dos usuários forem alteradas ou ao ajustar os níveis de acesso.

**Tempo estimado:** 2 a 5 minutos por usuário

**Pré-requisitos**

- Função de administrador ou função personalizada com permissões de atribuição de funções.
- A conta de usuário já deve existir (consulte *Criar e gerenciar usuários* ).
- Compreender as responsabilidades profissionais do usuário e o nível de acesso necessário.

## Passos

1. **Acesse a Administração de Acesso Avançado** conforme descrito na seção “Criar usuários”.
2. **Localize o usuário.** Use a função de pesquisa ou percorra a lista de usuários para encontrar o usuário.
3. **Abrir detalhes do usuário.** Clique no nome do usuário ou no ícone de edição para acessar o perfil dele.
4. **Acesse “Atribuições de funções”.** Encontre a seção que mostra as funções atuais do usuário e as funções disponíveis.
5. **Atribuir funções:**
   - Selecione a(s) função(ões) adequada(s) à função do usuário.
   - Os usuários podem ter várias funções atribuídas simultaneamente.
   - As permissões são cumulativas — se um usuário tiver várias funções, ele receberá as permissões combinadas de todas as funções atribuídas.
6. **Salve as alterações.** Clique em **Salvar** para aplicar as atribuições de funções.

## Resultados esperados

O nível de acesso do usuário é alterado imediatamente. Na próxima vez que fizerem login (ou após atualizar o navegador, caso já estejam conectados), eles verão a interface e os recursos correspondentes às funções que lhes foram atribuídas.

Observação: quando os usuários alteram suas funções, é necessário atualizar o navegador para que a nova função entre em vigor. Informe os usuários quando alterar a atribuição de funções deles.

## Melhores práticas para atribuição de funções

- **Princípio do privilégio mínimo:** atribua aos usuários a função mínima necessária para que possam desempenhar suas atribuições.
- **Atribuições de funções em documentos:** Manter um registro de quem desempenha quais funções e por quê, para fins de auditoria.
- **Revisões periódicas:** revise periodicamente a atribuição de funções para garantir que elas ainda correspondam às responsabilidades do cargo.
- **Limitar as funções de administrador:** atribua a função de administrador apenas aos usuários que realmente precisam de acesso administrativo total.

## Ativação das funções disponíveis para os usuários

Os administradores podem ativar o recurso **“Funções disponíveis”** para usuários específicos, permitindo que eles próprios alternem entre as funções atribuídas. Isso é útil para usuários que precisam de diferentes níveis de acesso para diferentes tarefas.

Para ativar as funções disponíveis para um usuário:

1. Atribua várias funções ao usuário.
2. Ative a opção **“Funções disponíveis”** no perfil do usuário.
3. O usuário pode então alterar sua função atual clicando no nome de usuário no canto superior direito, selecionando “**Minha conta** ” e escolhendo uma opção no menu suspenso “**Função atual** ”.
