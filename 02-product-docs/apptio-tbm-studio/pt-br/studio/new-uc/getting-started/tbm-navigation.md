## Visão geral

O TBM Studio oferece uma interface simplificada, projetada para ajudá-lo a alternar com eficiência entre o Data Studio, o Model Studio e o Report Studio. Este guia de referência mostra onde encontrar os principais recursos e como navegar pela interface.

**O que você aprenderá:**

- Principais [áreas de navegação](navigation.html) e sua finalidade
- [Ambientes](environments.html) diferentes
- Como [trabalhar com projetos e documentos](work-with-docs.html)
- [Atalhos de teclado](shortcuts.html) essenciais
- [Opções de personalização](custom-options.html) da interface

## O que vem a seguir

Depois de entender as interfaces, você poderá:

- Siga o [Guia de Início Rápido](qsg.html) para criar seu primeiro projeto
- Acesse Data Studio para saber mais sobre [gerenciamento de dados](../howtoguides/work-with-data/data-import-mgmt.html)
- Experimente [usar atalhos de teclado](shortcuts.html) para aumentar sua eficiência
- Explore os percursos de aprendizagem para obter orientações específicas para cada função
