# Principais áreas de navegação

## Barra de ferramentas

A barra de ferramentas na parte superior do TBM Studio oferece acesso rápido às principais funções e à navegação.

*[Captura de tela: Barra de ferramentas com seções identificadas]*

**Elementos principais da barra de ferramentas:**

|  |  |  |
| --- | --- | --- |
| **Elemento** | **Propósito** | **Uso** |
| Menu **"Todos os produtos"** | Acesse outros produtos da Apptio | Clique para alternar entre os aplicativos do TBM Suite |
| Seletor **de ambiente** | Alternar entre Desenvolvimento, Pré-produção e Produção | Selecione no menu suspenso para alterar os ambientes |
| Menu suspenso **do projeto** | Acessar as configurações atuais do projeto | Clique para ver os detalhes, as configurações e as permissões do projeto |
| Botão **Início** | Voltar ao painel principal | Acesso rápido à visão geral do projeto e às recomendações |
| Menu **de ajuda** | Acesse a documentação e o suporte | Clique aqui para obter ajuda contextual e recursos |
| Menu **do usuário** | Configurações e preferências da conta | Acessar perfil, preferências e sair |

Dica: Passe o mouse sobre os ícones da barra de ferramentas para ver dicas de ajuda que explicam suas funções.

## Área de janela de navegação

O painel de navegação à esquerda dá acesso a todos os principais estúdios e componentes do projeto.

**Seções principais da navegação:**

- **Data Studio** — Carregue, transforme e gerencie suas fontes de dados
- **Model Studio** — Crie modelos de custos e configure alocações
- **Report Studio** — Crie e personalize relatórios
- **Administração** — Gerenciar usuários, permissões e configurações do projeto

**Para navegar:**

1. Clique no nome de um estúdio para expandir sua seção
2. Clique em um item específico para abri-lo na área de trabalho principal
3. Use a barra de pesquisa na parte superior para encontrar itens rapidamente

Observação: O painel de navegação pode ser recolhido para liberar mais espaço na tela clicando no ícone de recolhimento (◀).

## Painel de Projetos

O painel "Projetos" exibe todos os projetos aos quais você tem acesso dentro do seu domínio.

**O que você vê:**

- Projeto ativo no momento (destacado)
- Outros projetos disponíveis
- Indicadores de status do projeto

**Para alternar entre projetos:**

1. Clique no menu suspenso **“Projetos”** na barra de ferramentas
2. Selecione o projeto com o qual deseja trabalhar
3. A interface é atualizada para mostrar o projeto selecionado

Aviso: as alterações não salvas no projeto atual podem ser perdidas ao alternar entre projetos. Sempre salve seu trabalho antes de mudar de tela.

## Painel do Explorador de Projetos

O Explorador de Projetos mostra a estrutura do seu projeto atual, com todos os documentos organizados por tipo.

**Tipos comuns de documentos:**

- **Tabelas** — Tabelas de dados e tabelas de transformação
- **Modelos** — Modelos de custos e configurações de alocação
- **Métricas** — Métricas calculadas e modeladas
- **Relatórios** — Relatórios publicados e painéis
- **Tabelas editáveis** — Formulários de inserção de dados

**Ícones de status do documento:**

|  |  |  |
| --- | --- | --- |
| **Ícone** | **Estado** | **Significado** |
| 🔒 | Bloqueado | O documento está sob sua responsabilidade |
| 🔓 | Desbloqueado | O documento está disponível para edição |
| 🔄 | Atualizado | Está disponível uma versão mais recente (clique em “Atualizar área de trabalho”) |
| ⏳ | Calculando | O documento está sendo calculado após o check-in |
| ⚠️ | Erro | O documento contém erros ou avisos |

Dica: clique com o botão direito do mouse em qualquer documento no Explorador de Projetos para acessar ações rápidas, como Retirar, Arquivar, Renomear ou Excluir.
