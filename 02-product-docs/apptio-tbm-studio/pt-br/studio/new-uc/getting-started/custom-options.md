# Opções de personalização

## Alternar o modo de estúdio

O TBM Studio oferece dois modos de interface:

**Modo Estúdio (padrão):**

- Interface de edição completa com todas as ferramentas e opções
- Faixas de opções e controles detalhados visíveis
- Ideal para desenvolvimento e configuração

**Modo de visualização do relatório:**

- Interface simplificada e voltada para a visualização
- Controles mínimos, maximiza a área de exibição do relatório
- Ideal para apresentações e consumo pelo usuário final

**Para ativar ou desativar:** Pressione Ctrl+Alt+T ou clique no botão de alternância do modo Studio na barra de ferramentas.

## ApEx Interface do usuário

O TBM Studio inclui uma interface de usuário modernizada do tipo “ ApEx ”, com estilo e cores atualizados.

**O que muda:**

- Estilo e cores do relatório
- Criação de gráficos e tabelas
- Ícones e menus de contexto
- Caixas de diálogo e notificações
- Bordas e fontes

**Para ativar/desativar a interface do usuário do ` ApEx `:**

1. Vá para a guia **Projeto** > **Configurações do projeto**
2. Marque ou desmarque **a opção “Ativar interface do usuário do Apex”**
3. Clique **em Salvar**

Observação: a interface do usuário do ApEx está ativada por padrão em novos projetos. Desative essa opção se preferir manter a consistência com projetos anteriores ou se precisar da interface antiga.
