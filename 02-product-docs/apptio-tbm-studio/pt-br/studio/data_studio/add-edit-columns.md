# Adicionar e editar colunas

**Aplica-se a** : TBM Studio 12.0 e posterior. Para adicionar novas colunas ou editar colunas existentes em uma tabela, adicione uma etapa **de Fórmulas** ao pipeline de transformação. As colunas que você adiciona são exibidas na parte superior da lista de colunas no painel Editar. As colunas que fazem parte da tabela original são exibidas na parte inferior da lista de colunas no painel Editar.

## Adicionar uma coluna

1. Se ainda não tiver feito isso, adicione uma etapa de Fórmulas à transformação de dados.
2. Clique em **Adicionar uma nova coluna**.
3. Se necessário, altere o tipo de coluna clicando no campo **Type (Tipo** ).
4. Digite um nome para a coluna.
5. Digite uma fórmula para a coluna.
6. Clique em **Salvar**.

## Editar uma coluna

1. Use a lista suspensa **Selecionar coluna para editar** para selecionar a coluna a ser editada.
2. Faça as edições.
3. Clique em **Salvar**.

## Excluir uma coluna

Para excluir uma coluna, aponte para a primeira coluna no painel Editar e clique no ícone Excluir ![](../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/studioimages/icons/delete2.png).

## Filtrar o painel Editar

Você pode filtrar o painel Editar usando os campos de pesquisa na parte superior de cada coluna.

Para filtrar por tipo de coluna (chave, texto, número, data), clique em um ícone de tipo no campo de pesquisa da coluna **Tipo**.
