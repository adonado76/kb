# Fazer upload de dados

**Aplica-se a** : TBM Studio 12.0 e posterior

Ao fazer upload de dados, você faz upload dos dados em uma tabela. Você pode fazer upload de dados usando um dos seguintes métodos:

- **Upload de arquivo** : Selecione um arquivo de um local local ou de rede em um dos seguintes formatos: Excel (xls, xlsx),.csv, XML, Zip ([Como?](upload-data-file.html "Aplica-se a: TBM Studio 12.0 e posterior. O TBM Studio aceita dados de arquivos simples nos formatos separados por vírgula, tabulação, pipe, til, dois pontos e ponto e vírgula. Antes de fazer upload de dados, você deve criar uma tabela na qual os dados serão carregados, caso ela ainda não exista.") ).
- **Enter Manually (Inserir manualmente** ): Insira os dados manualmente em uma tabela ([Como?](enter-data-manually.html "Aplica-se a: TBM Studio R12.0 e posterior") ).
- **Tabela gerada** : Criar uma nova tabela a partir de uma tabela existente no projeto atual. A nova tabela NÃO será vinculada à tabela existente.
- **Tabela existente** : Criar uma nova tabela a partir de uma tabela existente no projeto atual. A nova tabela SERÁ vinculada à tabela existente.
- **Datalink (Classic) Conector** : Importar uma tabela usando um conector Datalink (Classic) predefinido.
- **Tabela importada** : Importar os dados de um projeto existente do site Costing Standard .

Antes de fazer upload de dados, você deve criar uma tabela que receberá os dados ([Como?](create-table.html "Aplica-se a: TBM Studio 12.0 e posterior") ).
