# Carregar dados no projeto Row-level Security usando DataLink

O projeto de segurança em nível de linha é usado para limitar a visibilidade dos dados em nível de linha.

## Sobre esta tarefa

Se desejar, os dados de segurança em nível de linha podem ser carregados diretamente no projeto usando Datalink (Classic).

[Saiba mais](apply-row-level-security.html "(Abre em uma nova guia ou janela)")

Carregar dados na segurança em nível de linha

1. Navegue até sua instância Datalink (Classic) .
2. Selecione o agente e o conector desejados.

   A página **Configuration** é aberta.
3. Na seção **Project (Projeto** ), digite "Users" (Usuários).

   ![](../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/studioimages/lowlevel_security.png)
4. Conclua o restante do processo como faria em qualquer outra execução do Datalink
   (Classic) .
