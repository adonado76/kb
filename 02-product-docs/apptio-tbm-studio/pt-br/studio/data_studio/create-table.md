# Criar uma tabela

**Aplica-se a** : TBM Studio 12.0 e posterior

Antes de fazer upload de dados, você deve criar uma tabela que receberá os dados. Quando você cria uma tabela, um pipeline de transformação básico é criado ao mesmo tempo. Usando o pipeline, você pode carregar dados na tabela e realizar transformações nos dados, se necessário.

Criar uma tabela
:   1. Clique na guia **Home**.
    2. Clique em **New** e depois em **Table**.
    3. Digite um nome para a tabela.
    4. Insira uma categoria. Quando você começar a digitar, serão exibidos os nomes correspondentes das categorias que já existem. Além disso, você pode inserir um novo nome de categoria. As categorias são usadas para organizar as tabelas no **Project Explorer**.
    5. Clique em **OK**. O aplicativo cria a tabela e exibe a etapa **Source** no pipeline de transformação.

Excluir uma tabela
:   1. Dê uma olhada na tabela.
    2. Na guia **Início**, no grupo **Documento**, clique em **Excluir**.
    3. Verifique na tabela.
