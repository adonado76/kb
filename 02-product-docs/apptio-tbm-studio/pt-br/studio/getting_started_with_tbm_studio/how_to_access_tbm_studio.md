# Como acessar o TBM Studio

Apptio testa e oferece suporte aos seguintes navegadores da Web para uso com o Apptio Technology Business Management (TBM) Suite.

## Navegadores Suportados

Os aplicativos Apptio são fornecidos como um serviço pela Internet. A interface é baseada na Web e está disponível na maioria dos navegadores da Web. Embora não haja nada inerente à interface de usuário do aplicativo Apptio que impeça sua renderização em qualquer navegador da Web, alguns recursos podem não ser renderizados corretamente em todos os navegadores.

## Google Chrome

As versões 31 e superiores do Chrome são compatíveis.

## Microsoft Edge

Microsoft Edge 41.16299.547.0 e superiores são compatíveis.

## Mozilla Firefox

Firefox são compatíveis com as versões 20.x e superiores. Os novos recursos do Apptio são testados na versão mais recente do Firefox quando o recurso é lançado. Recomendamos a atualização para a versão mais recente do site Firefox. Se não for possível usar a versão mais recente do Firefox, recomendamos usar um navegador diferente.

## Recomendações gerais de navegador e resolução de tela

Embora o site Apptio não teste especificamente com outros navegadores além dos listados acima, não há nada inerente à interface do aplicativo Apptio que o impeça de ser renderizado em outros navegadores. No entanto, se você usar um navegador diferente dos listados acima e tiver problemas com a interface, o suporte do Apptio poderá solicitar que você mude para um navegador compatível como solução para o problema.

Para todos os navegadores, recomendamos o seguinte:

- Habilite JavaScript, cookies e SSL 3.0
- Defina a resolução da tela para um mínimo de 1024 x 768 para obter a melhor experiência. Resoluções de tela inferiores a 1024 x 768 podem exigir rolagem adicional para trabalhar com uma página inteira. Você não deve redimensionar um navegador não maximizado para menos de 1024 x 768 em um monitor de resolução mais alta.
- Alguns plug-ins e extensões de navegadores da Web de terceiros podem interferir na funcionalidade do software Apptio. Se você tiver problemas de funcionamento ou comportamento inconsistente, desative todos os plug-ins e extensões do navegador da Web e tente novamente.

## Resolução de tela suportada

A resolução de tela mínima recomendada (supondo que você tenha maximizado a janela do navegador) é de 1024 x 768.
