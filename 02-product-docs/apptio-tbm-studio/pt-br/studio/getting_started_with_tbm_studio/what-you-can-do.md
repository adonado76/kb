# Introdução ao TBM Studio

TBM Studio é um conjunto de ferramentas que o ajuda a gerenciar seus gastos comerciais com tecnologia. Ele inclui três componentes principais:

- Data Studio
- Estúdio Modelo
- Report Studio

- **Data Studio** : Permite que você [reúna os dados corretos, transforme-os de forma inteligente](../data_studio/data-transform-chapter-title-page.html "Aplica-se a: TBM Studio 12.0 e posterior") e aprimore-os ao longo do tempo.
- **Model Studio** : Permite que você [personalize visualmente as alocações e configure um modelo de custo flexível](../model_studio/models-chapter-title-page.html "Aplica-se a: TBM Studio 12.0 e posterior") para refletir com precisão os custos e serviços de TI.
- **Report Studio** : Permite que você [personalize os relatórios existentes ou crie novos relatórios](../reports/intro_report_studio.html "Aplica-se a: TBM Studio 12.0 e posterior") que incluam tabelas, gráficos e componentes para filtrar os relatórios, inserir dados e navegar entre os relatórios.

Para obter mais informações, consulte [Conceitos essenciais do TBM Studio](../admin/essential-tbmstudio-concepts.html "Aplica-se a: TBM Studio 12.0 e posterior. No TBM Studio, você organiza seu trabalho em projetos. Em um projeto, você coleta e transforma dados em conjuntos de dados, cria modelos, adiciona objetos aos modelos, atribui conjuntos de dados aos objetos, aloca custos de um objeto para outro e resume os resultados em um ou mais relatórios. Instâncias, domínios e projetos organizam seu trabalho:")
