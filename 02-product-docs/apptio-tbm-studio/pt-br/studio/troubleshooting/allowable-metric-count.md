# Contagem de métricas permitida excedida

A contagem de métricas permitida é o número total de métricas que seu projeto pode ter. Isso inclui métricas calculadas e modeladas.

Se você tiver atingido esse limite e tentar criar uma nova métrica, a seguinte mensagem de erro será exibida:

![aviso](../../../../../03-media/apptio-tbm-studio/resources/images/troubleshooting_images/allowable-metric-count-exceeded.png)

## Recomendação de configuração para o erro Allowable Metric Count exceeded

Para resolver esse erro:

1. Na exibição **Project Explorer** > selecione **Metrics**
2. Selecione a métrica que você deseja remover
3. Na guia **Home**, selecione **Check Out**
4. Selecione novamente a métrica destacada
5. Na guia **Home**, selecione **Excluir**
6. selecione **OK** para continuar

Se precisar de mais ajuda para reduzir o número total de métricas, entre em contato com o suporte Apptio.
