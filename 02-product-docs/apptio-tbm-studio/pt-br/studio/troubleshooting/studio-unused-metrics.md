# Métricas não utilizadas

Esse recurso fornece recomendações automáticas sobre métricas calculadas não utilizadas, oferecendo visibilidade das métricas criadas pelo usuário que não são utilizadas em outras métricas ou relatórios. O sistema verificará automaticamente as alterações e marcará os problemas como resolvidos.

Navegue até **TBM Studio** > guia **Recommendations (Recomendações** ) > **Unused Metrics (Métricas não utilizadas** ) e selecione **Troubleshoot All Identified Problems (Solucionar todos os problemas identificados** ).

![img](../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/unusedmetrics-1.png)

Alterne para o espaço de trabalho **Development** e selecione **Next**.

![](../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/umetrics-2.png)

Clique no ícone **Next**.

![](../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/unused-metrics.png)

Selecione o botão **Excluir métrica**. A métrica não utilizada é excluída.

![](../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/umetric-del.png)

Selecione **Next** e, em seguida, selecione **Checkin** na última página.

![](../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/umetricdeleted.png)
