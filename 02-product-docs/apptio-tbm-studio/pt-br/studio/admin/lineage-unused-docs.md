# Como encontrar documentos não utilizados

Na guia Project (Projeto), Enable Feature (Ativar recurso), clique em Enable Show Unused Documents (Ativar mostrar documentos não utilizados). Agora, esse recurso estará disponível para uso na própria guia Projeto. Vamos selecioná-lo e ele abrirá uma lista de tabelas que contém qualquer métrica calculada, nome de tabela ou relatório que tenha sido introduzido pelo usuário, mas não usado por nada. Portanto, isso realmente lhe dá a oportunidade de remover essas entidades.

![imagem](../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/unused-doc.png)

Observação: os resultados para documentos não utilizados são mostrados somente durante o uso do Lineage, onde não estamos considerando nenhuma dependência de projeto externo e há um escopo futuro prospectivo para isso. Nesses casos, podemos tender a remover qualquer entidade, mas certifique-se de verificar se ela não está sendo usada em nenhum outro projeto.
