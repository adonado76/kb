# Restaurar projetos

**Aplica-se a** : TBM Studio 12.0 e posterior

Para excluir ou restaurar um projeto, entre em contato com o suporte Apptio informando o nome do projeto. Os nomes de projetos diferenciam maiúsculas de minúsculas, portanto, certifique-se de usar a caixa correta para o nome do projeto.
