# Configurar a moeda comum

**Aplica-se a** : TBM Studio 12.0 e posterior. Você pode alterar a moeda comum para seu perfil individual ou para projetos inteiros do TBM Studio .

## Alterar a moeda comum de seu perfil

É possível alterar a moeda comum apenas para a sua conta de usuário sem afetar o projeto de outros usuários.

1. Navegue até ![](../../../../../03-media/apptio-tbm-studio/resources/images/icons/icon-user-account_18x20.png).
2. Em **Currency (Moeda** ), selecione a moeda de sua preferência.
3. Clique em **OK**.

   ![](../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/studio-change-curencies.png)

   Observação: As preferências de moeda do seu perfil terão prioridade sobre as Configurações do projeto na maioria das páginas do site TBM Studio.

## Alterar a moeda base de todo o projeto

Com as permissões corretas, você pode alterar a moeda comum usada em todo o projeto, o que afetará todos os usuários que podem acessar o projeto.

1. Na guia **Projeto**, no grupo **Configuração de projeto**, clique em **Configurações de projeto**.

   ![](../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/studio_admin_base_currency_sui.png)
2. Em **Moeda base**, selecione a moeda de sua preferência.
3. Em **Compactação de números**, defina a compactação de sua preferência para milhares, milhões, bilhões e trilhões.
4. Em **Currency Symbols (Símbolos de moeda** ), defina o símbolo de sua moeda preferida.
5. Clique em **Save (Salvar** ) para aplicar as alterações.

   Observação: as Configurações do projeto só podem ser alteradas se não estiverem sendo verificadas por outros usuários no momento.
