# Atalhos de teclado

aplica-se a: TBM Studio 12.0 e posterior.

Os seguintes atalhos de teclado estão disponíveis.

- **Ctrl+Alt+T** - Entra e sai do modo Studio ([saiba mais](navigate-tbm-studio.html "(Abre em uma nova guia ou janela)") )
- **Ctrl+F1** - Alterne o modo de abas/fitas do site TBM Studio
- **Alt+N** - Criar um novo documento
- **Alt+O** - Verificar um documento ([saiba mais](bp-check-out.html "(Abre em uma nova guia ou janela)") )
- **Ctrl+S, Meta+S** - Salvar
- **Alt+K** - Fazer check-in em um documento ([saiba mais](bp-check-out.html "(Abre em uma nova guia ou janela)") )
- **Ctrl+Alt+Z** - Reverter um documento ([saiba mais](bp-check-out.html "(Abre em uma nova guia ou janela)") )
- **Shift+F9** - Atualizar um projeto
- **F9** - Atualizar um documento
- **Alt+Escape, Meta+Escape** - Fechar a guia
- **Ctrl+Z** - Desfaz todas as alterações desde o último salvamento ([saiba mais](bp-check-out.html "(Abre em uma nova guia ou janela)") )
- **Ctrl+Y** - Refazer ([saiba mais](bp-check-out.html "(Abre em uma nova guia ou janela)") )
