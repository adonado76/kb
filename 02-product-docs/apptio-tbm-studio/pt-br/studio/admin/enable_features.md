# Recursos da versão beta

TBM Studio às vezes contém recursos que ainda não foram liberados para todos os clientes. Esses são recursos da versão beta e podem não funcionar exatamente como foram projetados.

## Ativar os recursos da versão beta

Como TBMA, você pode usar a opção Enable Features (Ativar recursos) para controlar quais recursos da versão beta seus usuários podem acessar. Quando você ativa um recurso Beta, ele se torna visível para todos os usuários do seu ambiente.

1. Na guia Project (Projeto), selecione Enable Features (Ativar recursos).
2. Se você não estiver conectado como TBMA, a opção Enable Features estará desativada. ![](../../../../../03-media/apptio-tbm-studio/studio/admin/enable_features.png)
3. Use a caixa de seleção para ativar ou desativar os recursos da versão beta.
