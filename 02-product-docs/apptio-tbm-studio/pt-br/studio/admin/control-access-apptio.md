# Controle o acesso aos aplicativos Apptio

**Aplica-se a** : TBM Studio v12.1, v12.2 e posterior

Para controlar o acesso aos aplicativos Apptio, use Enhanced Access
Administration, a página Apptio Access Administration . As partes do sistema que um usuário pode acessar e as ações que ele pode executar são controladas por funções. Por padrão, durante os períodos de manutenção, os usuários são impedidos de fazer login no sistema. Você pode substituir essa configuração. Para obter informações sobre como usar o site Enhanced Access Administration, consulte [Sobre o Frontdoor](/docs/SSFG2DK/frontdoor/home.htm).
