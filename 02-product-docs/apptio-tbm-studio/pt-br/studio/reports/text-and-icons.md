# Componentes de texto em relatórios

**Aplica-se a** : TBM Studio 12.0 e posterior

Ao criar um relatório, você pode adicionar componentes que exibem texto. Esses componentes são:

|  |  |
| --- | --- |
| **Caixa de texto HTML** | Use para adicionar texto a um relatório que seja visível para todos os usuários. |
| **KPI** | Use para adicionar indicadores-chave de desempenho a um relatório. |

## Editar propriedades do componente

Há uma grande variedade de propriedades disponíveis para componentes. As propriedades disponíveis diferem de acordo com cada tipo de componente. **As propriedades** são definidas usando a faixa de opções e uma caixa de diálogo Propriedades.

Para editar as propriedades de um componente, exiba a caixa de diálogo **Propriedades** seguindo um destes procedimentos:

- No canto superior esquerdo do gráfico ou da tabela, clique no pequeno triângulo ![imagem](../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/studioimages/icons/arrow-down-greyicon.jpg) à esquerda do nome do componente para exibir o menu **Action (Ação** ), conforme mostrado na imagem a seguir. No menu **Ação**, selecione **Propriedades**.

  ![imagem](../../resources/images/studio_images/studioimages/studio_html%20comp_properties.png)
- Clique com o botão direito do mouse em qualquer lugar dentro das bordas do componente e clique em **Properties (Propriedades** ) no menu pop-up.

- [KPIs: indicadores-chave de desempenho](kpi.html "Aplica-se a: TBM Studio 12.0 e posterior")
