# Adicionar colunas a uma tabela

**Aplica-se a** : TBM Studio 12.0 e posterior

Assista a este vídeo de demonstração do Apptio Education Services: [Adicionar uma coluna de taxa](https://community.apptio.com/videos/1893 "(Abre em uma nova guia ou janela)"). Ou navegue por [todos os vídeos do site Apptio](https://community.apptio.com/docs/DOC-7714 "(Abre em uma nova guia ou janela)").

Depois de adicionar uma tabela a um relatório, você pode adicionar colunas à tabela. Os principais tipos de colunas são descritos a seguir:

- **Total** - Exibe os totais de cada linha em uma tabela ([Como?](add-total-column.htm "(Abre em uma nova guia ou janela)") ).
- **Fórmula** - A coluna pode se basear em uma fórmula personalizada, nas colunas do conjunto de dados que fazem o backup da tabela ou em um valor modelado [(How?)](add-formula-column-to-table.htm "(Abre em uma nova guia ou janela)").
- **Calculada** - A coluna pode exibir valores para uma única coluna, como contagem, porcentagem e soma. Ele também pode comparar duas colunas na tabela ou comparar valores entre projetos ([Como?](add-calculated-column.htm "(Abre em uma nova guia ou janela)") ).
- **Calendar** - Exibe valores para os dados com base em um intervalo de tempo maior que um mês ou período ([Como?](add-calendar-based-column.htm "(Abre em uma nova guia ou janela)") ).
- **Sparkline** - Exibe uma pequena linha de tendência ([Como?](add-sparkline-column-to-tables.htm "(Abre em uma nova guia ou janela)") ).
- **Editável** - Uma coluna em branco adicionada a uma tabela editável na qual os usuários podem inserir dados.

A tabela abaixo mostra onde os tipos de colunas podem ser usados.

| **Tipo de coluna** | **Tabelas baseadas em objetos** | **Transformar tabelas** | **Tabelas editáveis** |
| --- | --- | --- | --- |
| Fórmula | X | X | X |
| Calculado | X | X |  |
| Agenda | X | X |  |
| Linha de faísca | X | X |  |
| Total | X | X | X |
| Editável |  |  | X |

A quebra de linha não é aplicável à coluna de fonte primária de uma tabela editável enriquecida.
