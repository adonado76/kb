# Dispositivos do usuário final

O End User Devices (EUD) fornece insights para que os gerentes de ativos de TI e os gerentes de suporte a desktops tomem decisões baseadas em dados sobre consolidação de frota, otimização e custos esperados. Atualmente, há três componentes relacionados aos dispositivos do usuário final.

- CT Apps-Dispositivos de usuário final: Este é um componente antigo e não possui relatórios
- BoIT–End dispositivos de usuário: Esse é um componente antigo e não tem relatórios
- **Dispositivos do usuário final** : Esse é o componente mais recente com relatórios que os clientes precisam instalar para ver os relatórios com 7 soluções e 27 casos de uso. Abaixo está a captura de tela do componente para instalação:
