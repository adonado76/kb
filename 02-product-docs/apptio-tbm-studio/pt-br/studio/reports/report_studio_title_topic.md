# TBM Studio Sobre o Report Studio

12.0 e posteriormente

Depois de criar tabelas e modelos de alocação em TBM Studio , você pode usar o Report Studio para criar relatórios. Os relatórios podem conter tabelas e gráficos, bem como componentes para filtrar relatórios, inserir dados e navegar entre relatórios.
