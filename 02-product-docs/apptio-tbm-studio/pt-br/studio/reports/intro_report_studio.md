# Introdução ao Report Studio

**Aplica-se a** : TBM Studio 12.0 e posterior

Depois de carregar tabelas e criar modelos de alocação em TBM Studio, você pode usar o Report Studio para criar relatórios. Os relatórios podem conter tabelas e gráficos, bem como componentes para filtrar os relatórios, inserir dados e navegar entre os relatórios.

- [Representar um código de hierarquia em perspectivas](hierarchy-codes-perspectives.html "Aplica-se a: TBM Studio 12.0 e posterior")
- [Criar coleções de relatórios](creating-report-collections.html "Aplica-se a: TBM Studio 12.0 e posterior")
