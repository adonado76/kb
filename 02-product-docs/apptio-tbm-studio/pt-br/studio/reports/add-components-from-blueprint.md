# Adicionar componentes de um blueprint

## Sobre esta tarefa

**Aplica-se a** : TBM Studio 12.0 e posterior

Os blueprints são um conjunto completo de componentes de relatório. Você pode inserir os componentes do blueprint em um relatório. A vantagem de um blueprint é que você pode usá-lo para criar uma série de relatórios que tenham os mesmos componentes. Foram criados projetos para vários dos módulos do site Costing Standard .

Para inserir um blueprint:

## Procedimento

1. Abra um novo relatório.
2. Na guia Inserir, clique em **Blueprint**.
3. Selecione um blueprint e clique em **OK**.
