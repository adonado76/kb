# GetReportName função

**Aplica-se a** : TBM Studio 12.0 e posterior

Use essa função para obter dinamicamente o nome do relatório que você está usando.

## Onde usar

Essa função pode ser usada em:

- Texto Dinâmico

Use para obter dinamicamente o nome do relatório que você está usando.

Por exemplo, você pode combinar com `evalWiki()` para definir um cabeçalho padrão em um conjunto de dados e usar a função GetReportName para obter o nome do relatório atual.

## Sintaxe

`GetReportName()`

## Argumentos

Não há argumentos.

## Tipo de retorno

Texto
