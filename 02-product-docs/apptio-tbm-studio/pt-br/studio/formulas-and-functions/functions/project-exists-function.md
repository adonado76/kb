# ProjectExists função

**Aplica-se a** : TBM Studio 12.0 e posterior

Se o projeto especificado existir, a função retornará "true" (verdadeiro) Se o nome do projeto não existir, a função retornará "false"

## Onde usar

Essa função pode ser usada em:

- Conjuntos de dados
- Métricas calculadas e relatórios com colunas de métricas
- Colunas de fórmula em tabelas de relatórios
- Texto Dinâmico

## Sintaxe

`ProjectExists(domain:project)`

## Tipo de retorno

Texto
