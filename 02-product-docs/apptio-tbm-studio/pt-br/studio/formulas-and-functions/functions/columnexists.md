# ColumnExists função

**Aplica-se a** : TBM Studio 12.0 e posterior

Use essa função para determinar se uma coluna existe em uma tabela.

## Onde usar

Essa função pode ser usada em:

- Tabelas

## Sintaxe

`=ColumnExists(column_name)`

## Argumentos

*nome\_da\_coluna*

O nome da coluna que você deseja verificar.

## Tipo de retorno

Verdadeiro ou falso
