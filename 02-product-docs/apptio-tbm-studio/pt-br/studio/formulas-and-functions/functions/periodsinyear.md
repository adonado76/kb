# PeriodsInYear função

**Aplica-se a** : TBM Studio e posterior

Retorna o número de períodos no ano.

## Onde usar

Essa função pode ser usada em:

- Conjuntos de dados
- Colunas de fórmula em tabelas de relatórios
- Texto Dinâmico

## Sintaxe

`PeriodsInYear()`

## Argumentos

Nenhum

## Tipo de retorno

Sequência

## Exemplo

Dado um projeto de 13 períodos e um período definido como P10 FY2013, a função a seguir retorna o resultado mostrado abaixo:

| Função | Voltar |
| --- | --- |
| PeriodsInYear() | 13 |
