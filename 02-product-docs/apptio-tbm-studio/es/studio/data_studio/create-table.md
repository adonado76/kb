# Crear una tabla

**Se aplica a** : TBM Studio 12.0 y posteriores

Antes de poder cargar datos, debe crear una tabla que los reciba. Al crear una tabla, se crea al mismo tiempo un canal de transformación básico. Mediante la canalización, puede cargar datos en la tabla y realizar transformaciones en los datos si es necesario.

Crear una tabla
:   1. Haz clic en la pestaña **Inicio**.
    2. Haga clic en **Nuevo** y, a continuación, en **Tabla**.
    3. Introduzca un nombre para la tabla.
    4. Introduzca una categoría. Cuando empiece a escribir, se mostrarán los nombres coincidentes de las categorías que ya existen. También puede introducir un nuevo nombre de categoría. Las categorías se utilizan para organizar las tablas en el **Explorador de proyectos**.
    5. Pulse **Aceptar**. La aplicación crea la tabla y muestra el paso **Fuente** en la cadena de transformación.

Borrar una tabla
:   1. Echa un vistazo a la tabla.
    2. En la pestaña **Inicio** del grupo **Documento**, haga clic en **Eliminar**.
    3. Compruébalo en la tabla.
