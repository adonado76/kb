# Acerca de Data Studio

**Se aplica a** : TBM Studio 12.0 y posteriores

Cuando se introducen datos en una de las aplicaciones de Apptio, puede que estén o no en el formato correcto para asignarlos a los conjuntos de datos maestros o para generar sus propios informes personalizados. Si los datos no están en el formato correcto, tendrá que transformarlos utilizando las herramientas de transformación de Data Studio.
