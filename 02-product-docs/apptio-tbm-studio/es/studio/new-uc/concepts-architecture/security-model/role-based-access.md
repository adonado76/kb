# Control de acceso basado en roles (RBAC)

La capa de autorización de TBM Studio utiliza un modelo de control de acceso basado en roles (RBAC). En lugar de asignar permisos individuales a cada usuario, los permisos se agrupan en roles, y a los usuarios se les asigna uno o varios roles. Esto simplifica la gestión y garantiza la coherencia: cuando sea necesario modificar los permisos de un grupo de usuarios, basta con modificar el rol en lugar de actualizar cada usuario por separado.
