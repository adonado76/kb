# Resumen

La personalización de informes convierte los informes individuales en un entorno de presentación de informes coherente y profesional. Las cuatro técnicas de esta sección se complementan entre sí:

|  |  |  |
| --- | --- | --- |
| **Técnica** | **Finalidad** | **¿Quién puede configurarlo?** |
| Perspectivas personalizadas | Organiza los campos para facilitar la creación de informes | Usuario avanzado, administrador |
| Colecciones de informes | Informes relacionados con el grupo para la navegación | Administrador (crear/eliminar), Usuarios (añadir informes) |
| Informes principales | Asegurar la coherencia en el diseño y la imagen de marca | Usuario avanzado, administrador |
| Permisos del informe | Controla quién ve qué | Administrador (nivel de informes), Usuario avanzado (componente) |

**Flujo de trabajo recomendado:**

1. Diseña tus vistas personalizadas con los campos que más necesitan los analistas
2. Crea plantillas de informes para mantener la coherencia de la imagen de marca
3. Crear informes utilizando perspectivas y plantillas
4. Organiza los informes en colecciones lógicas
5. Configurar los permisos a nivel de colección, informe y componente

Esta plataforma te permite mantener un entorno de generación de informes escalable y seguro, capaz de atender a públicos diversos sin dejar de ser fácil de gestionar.
