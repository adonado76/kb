# Crear un informe básico

**Objetivo:** Crear un nuevo informe vacío listo para incluir tablas y gráficos

**Cuándo utilizarlo:** cuando necesites un nuevo informe para presentar los datos del modelo a las partes interesadas, o cuando los informes existentes no se ajusten a tus necesidades

**Tiempo estimado:** 5 minutos

## Pasos

1. Pásate a TBM Studio si aún no lo has hecho.
2. Haz clic en la pestaña **Inicio**, haz clic en **Nuevo** y, a continuación, haz clic en **Informe**. Aparece el cuadro de diálogo «Crear informe» con las opciones de configuración.
3. Introduce un **nombre** para tu informe. Elige un nombre descriptivo que ayude a los usuarios a encontrar el informe. Si tienes pensado exportar los datos a Excel, utiliza nombres de menos de 31 caracteres para evitar que se trunquen.
4. Introduce una **descripción** del informe. Esta descripción aparecerá en el Buscador de informes, así que escribe algo que ayude a los usuarios finales a comprender qué datos contiene el informe.
5. Selecciona quién puede **ver** el informe. **De forma predeterminada,** el informe es visible para todos los usuarios; selecciona roles específicos para restringir el acceso.
6. Elige un **diseño** para tu informe. Selecciona si deseas incluir un encabezado y elige entre **«Estándar»** (1024 píxeles de ancho) o **«Pantalla panorámica»** (1440 píxeles de ancho).
7. Haz clic en **Aceptar** para crear el informe.

## Resultados previstos

- Se abre un nuevo informe en blanco en el espacio de trabajo
- En la parte inferior del área de trabajo aparece una pestaña con el nombre de tu informe
- El informe se guarda automáticamente y ya está listo para que añadas componentes

## Errores habituales

- **Olvidarse de guardar los cambios:** después de realizar cambios, recuerda guardar el informe para que los demás puedan ver tus actualizaciones y el sistema pueda procesar los datos.
- **Nombres poco claros:** los nombres imprecisos como « Report1 » dificultan la búsqueda de los informes. Utiliza nombres que describan el contenido, como «Resumen de costes del centro de datos Q1 »
