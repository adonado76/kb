# Informes avanzados

**Tipo de contenido:** Colección de guías prácticas

**Destinatarios:** usuarios avanzados, creadores de informes, analistas de negocios

**Requisitos previos:** Haber completado el curso «Conceptos básicos de Report Studio» (Sección 3.3.1 ) y estar familiarizado con los modelos de datos

**Aplicable a:** TBM Studio 12.0 y versiones posteriores

Esta sección aborda técnicas avanzadas de generación de informes dirigidas a usuarios avanzados que necesitan crear soluciones de informes complejas e interactivas. Aprenderás a crear una navegación de desglose para explorar jerarquías de datos, añadir componentes editables para los flujos de trabajo de introducción de datos, configurar diseños adaptativos y generar informes para múltiples canales de salida.

**En esta sección**

- Crea informes de desglose para pasar del resumen a los detalles
- Añadir componentes editables a los informes para los flujos de trabajo de introducción de datos
- Configurar el tamaño y la posición dinámicos para diseños adaptativos
- Exportar e imprimir informes a través de múltiples canales (PDF, correo electrónico, impresión)
