# Transformar y enriquecer los datos

|  |  |
| --- | --- |
| **Tipo de contenido** | Guías prácticas |
| **Destinatarios** | Analistas de negocios, equipos de TI y finanzas |
| **Requisitos previos** | Finalización de la sección «Importación y gestión de datos»; comprensión de las transformaciones de tablas |

En esta sección se tratan las tareas esenciales de transformación de datos en TBM Studio. Estas guías te ayudan a limpiar, enriquecer y combinar datos para preparar la elaboración de modelos de costes y la generación de informes. Cada transformación se lleva a cabo mediante pasos de la cadena de procesamiento que se ejecutan secuencialmente de arriba abajo.

**En esta sección:**

- [Limpiar y mapear los datos utilizando las columnas del mapa](clean-map.html) (15-20 minutos)
- [Combinar datos de varias fuentes](join-data.html) (15-20 minutos)
- [Aplicar fórmulas para transformar datos](apply-formula.html) (10-15 minutos)
- [Filtrar los datos por partición de fecha](filter-data-date.html) (10-15 minutos)
- [Añadir varias fuentes de datos](append-multiple.html) (15-20 minutos)
