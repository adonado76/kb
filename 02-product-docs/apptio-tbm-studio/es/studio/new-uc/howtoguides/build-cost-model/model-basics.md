# Conceptos básicos sobre modelos

|  |  |
| --- | --- |
| **Tipo de contenido** | Colección de guías prácticas |
| **Destinatarios** | Analistas de negocios, equipos de TI y finanzas |
| **Tiempo estimado** | Entre 30 y 45 minutos en total |
| **Requisitos previos** | Conocimientos básicos de la interfaz de TBM Studio; haber creado al menos una tabla de transformación en Data Studio |

En esta sección se describen las tareas básicas para trabajar con el modelo de costes en TBM Studio. Antes de poder asignar costes o crear informes del modelo, es necesario comprender cómo añadir tablas al modelo, qué métricas se procesan en él y cómo visualizar la estructura general del flujo de costes.

**El modelo de costes es el núcleo de TBM Studio**. Toma los datos que hayas preparado en Data Studio y te permite definir cómo fluyen los costes, los presupuestos y otros valores desde los sistemas de origen hasta las unidades de negocio y los servicios. Cada asignación, cada flujo de costes y cada análisis financiero parte del modelo.

Esta sección contiene tres guías prácticas básicas:

- [Añadir una tabla al modelo](add-tables.html) : cómo incorporar tus tablas de transformación a la capa de modelado
- [Comprender las métricas del modelo (coste, presupuesto, previsión)](understand-model-metrics.html) : qué significan el coste, el presupuesto y la previsión, y cómo se relacionan entre sí
- [Ver el diagrama del modelo](view-model-diagram.html) : cómo visualizar e interpretar el flujo general de costes

Una vez que hayas completado estas tareas, estarás listo para [crear asignaciones](create-allocation.html).
