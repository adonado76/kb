# Crear asignaciones

|  |  |
| --- | --- |
| **Tipo de contenido** | Guía práctica |
| **Destinatarios** | Analistas de negocios, equipos de TI y finanzas |
| **Tiempo de finalización** | 30-60 minutos |
| **Requisitos previos** | Tablas de origen y destino con los pasos del modelo; datos transformados preparados en Data Studio; proyecto descargado |

## Visión general

Las asignaciones son el núcleo de la modelización de costes en TBM Studio. Determinan cómo se distribuyen los costes de una tabla a otra, asignando gastos como los costes del centro de datos a las aplicaciones, los costes de personal a los servicios o los costes de TI a las unidades de negocio. En esta sección se ofrece una guía paso a paso para crear asignaciones, desde simples distribuciones uno a uno hasta complejos flujos de costes de varios niveles.
