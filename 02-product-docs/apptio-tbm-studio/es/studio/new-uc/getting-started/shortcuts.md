# Atajos de teclado

Domina estos atajos para trabajar de forma más eficiente en TBM Studio.

## Atajos imprescindibles

|  |  |  |
| --- | --- | --- |
| **Atajo** | **Acción** | **Descripción** |
| Ctrl+Alt+T | Activar el modo Estudio | Activar y desactivar el modo de edición de Studio |
| Ctrl+F1 | Alternar la cinta | Mostrar u ocultar las pestañas y la cinta de TBM Studio |
| Alt+N | Nuevo documento | Crear un nuevo documento en el contexto actual |
| Alt+O | Extracción | Abre el documento seleccionado para editarlo |
| Ctrl+S / Meta+S | Guardar | Guardar el documento actual |
| Alt+K | Incorporar | Guardar el documento actual en el entorno de prueba |
| Ctrl+Alt+Z | Revertir | Deshacer todos los cambios realizados desde la última salida |
| F9 | Actualizar documento | Actualizar el documento actual con los últimos cambios |
| Shift+F9 | Actualizar el espacio de trabajo | Actualizar todos los documentos del espacio de trabajo |
| Alt+Esc | Cerrar pestaña | Cerrar la pestaña del documento actual |
| Ctrl+Z | Deshacer | Deshacer los cambios realizados desde el último guardado |
| Ctrl+Y | Rehacer | Rehacer los cambios que se han deshecho anteriormente |

**Notas sobre la plataforma:**

- Tecla **Meta** = Comando (⌘) en Mac, tecla Windows en Windows
- Los atajos de teclado **con la tecla Ctrl** funcionan tanto en Windows como en Mac
