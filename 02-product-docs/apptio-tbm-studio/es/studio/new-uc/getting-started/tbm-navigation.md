# Descripción general de la interfaz de usuario

|  |  |
| --- | --- |
| **Tipo de contenido** | Referencia |
| **Destinatarios** | Usuarios nuevos, todos los usuarios |
| **Requisitos previos** | Acceso a TBM Studio, comprensión de los conceptos básicos (Sección 1.2 ) |

## Visión general

TBM Studio ofrece una interfaz optimizada diseñada para ayudarte a moverte con eficacia entre « Data Studio », «Model Studio» y «Report Studio». Esta guía de referencia te muestra dónde encontrar las funciones principales y cómo navegar por la interfaz.

**Lo que aprenderás:**

- [Áreas](navigation.html) principales de navegación y su finalidad
- Diferentes [entornos](environments.html)
- Cómo [trabajar con proyectos y documentos](work-with-docs.html)
- [Atajos de teclado](shortcuts.html) imprescindibles
- [Opciones de personalización](custom-options.html) de la interfaz

## ¿Qué viene ahora?

Una vez que comprendas las interfaces, podrás:

- Sigue la [guía de inicio rápido](qsg.html) para crear tu primer proyecto
- Visita Data Studio para obtener más información sobre [la gestión de datos](../howtoguides/work-with-data/data-import-mgmt.html)
- Prueba [los atajos de teclado](shortcuts.html) para mejorar tu eficiencia
- Explora las rutas de aprendizaje para obtener orientación específica para cada puesto
