# Casos de uso habituales

**Asignación de costes de mano de obra: los usuarios seleccionan «Solución» → «Categoría» → «Subcategoría» en los menús desplegables para clasificar los costes de mano de obra con vistas a su asignación.**

**Explicaciones de las variaciones presupuestarias: Las columnas de texto permiten introducir explicaciones sobre las variaciones presupuestarias junto con las métricas calculadas.**

**Mantenimiento de atributos de centros de coste: los responsables distribuidos actualizan los atributos de sus propios centros de coste, bajo el control de la seguridad a nivel de fila.**

**Correcciones directas de datos: los usuarios corrigen valores de datos concretos cuando los datos del sistema de origen están incompletos o son incorrectos.**
