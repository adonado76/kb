# Prácticas recomendadas para el diseño

**Alineación y espaciado**

- Utiliza cuadros de grupo con disposición vertical u horizontal para mantener un espaciado uniforme.
- Establezca valores de espaciado explícitos en el cuadro de diálogo «Espaciado» para que los espacios entre los componentes sean uniformes.
- Alinee los componentes relacionados (por ejemplo, un gráfico y su tabla correspondiente) utilizando cuadros de grupo.

**Diseños optimizados para impresión**

- Prueba los diseños en el modo «Diseño de impresión» antes de dar por terminado el trabajo.
- Utiliza el flujo vertical en los informes con muchos componentes para garantizar una paginación coherente.
- Añade filtros de segmentación al final para que los informes impresos muestren el contexto de filtrado.
- Evita colocar información importante cerca de los márgenes de la página.

**Patrones de diseño habituales**

|  |  |
| --- | --- |
| **Patrón** | **Implementación** |
| **Diseño del panel de control** | Varios cuadros de grupo con gráficos e indicadores clave de rendimiento; filtros en un grupo de la barra lateral |
| **Informe detallado** | Tabla grande con cortadoras asociadas; utiliza «Flujo vertical» para imprimir |
| **Análisis por pestañas** | Componente con pestañas que muestra vistas diferentes en cada pestaña; filtros compartidos fuera de las pestañas |
| **Maestro-detalle** | Tabla resumen con botones de desglose para consultar los informes detallados |
