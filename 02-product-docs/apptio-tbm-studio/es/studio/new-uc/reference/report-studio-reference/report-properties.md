# Propiedades de informe

Las propiedades de los informes controlan el comportamiento, el aspecto y la accesibilidad de los informes en TBM Studio. Estos ajustes influyen en cómo se muestran los datos, quién puede acceder a los informes, cómo se gestionan los periodos de tiempo y cómo se ven los informes al imprimirlos o exportarlos. Esta guía ofrece documentación detallada sobre todas las opciones de configuración a nivel de informe.

## Metadatos del informe

Los metadatos del informe definen las propiedades básicas de identificación y organización de un informe. Estos ajustes se configuran al crear un informe y se pueden modificar posteriormente.

**Propiedades básicas**

|  |  |
| --- | --- |
| **Propiedad** | **Descripción** |
| Nombre del informe | El nombre que se muestra del informe. Aparece en el Explorador de proyectos, en las pestañas de informes y en la barra de navegación. Los nombres deben ser descriptivos y ajustarse a las convenciones de nomenclatura de la organización. |
| Descripción | Texto opcional en el que se describe el objetivo y el contenido del informe. Se muestra como una ventana emergente cuando el usuario pasa el cursor por encima del nombre del informe. Utilice este campo para explicar qué datos contiene el informe y cuál es su finalidad. |
| Recopilación de informes | El contenedor organizativo del informe. Los informes de grupo recogen informes relacionados y pueden tener su propia configuración de permisos, que se aplica a los informes guardados. |
| Visible para | Determina qué roles pueden ver el informe. Las opciones incluyen «Todos» o roles específicos. Esta configuración determina el acceso inicial al crear el informe. |
| Crear como nivel superior | Al guardar un informe creado mediante navegación descendente, esta opción coloca el informe en la parte superior de la jerarquía de informes, en lugar de anidarlo bajo su elemento principal. |

**Configuración del tamaño de la pantalla**

Al crear un informe, se selecciona el tamaño de pantalla que determina el ancho predeterminado del diseño del informe:

|  |  |  |
| --- | --- | --- |
| **Opción** | **Ancho** | **Ideal para** |
| Estándar | 1024 píxeles | Monitores tradicionales, pantallas de tabletas, informes con tablas menos anchas |
| Pantalla panorámica | 1440 píxeles | Monitores panorámicos modernos, paneles de control con varias columnas, diseños complejos |

Nota: *El tamaño de la pantalla influye en el área de edición predeterminada. Los componentes pueden sobrepasar estos límites, pero es posible que sea necesario desplazarse horizontalmente en pantallas más pequeñas.*
