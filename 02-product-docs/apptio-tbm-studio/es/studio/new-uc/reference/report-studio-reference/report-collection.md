# Colecciones de informes

Agrupa los informes relacionados para que los usuarios puedan navegar fácilmente entre informes sobre un tema o perspectiva similares. Las colecciones mejoran la visibilidad de los informes al organizarlos en grupos lógicos y ofrecer una experiencia de navegación coherente. Los administradores configuran las colecciones de informes para controlar qué roles de usuario pueden acceder a los informes de cada colección.
