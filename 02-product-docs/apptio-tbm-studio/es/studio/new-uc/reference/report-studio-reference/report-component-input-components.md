# Componentes del informe - Componentes de entrada

## Visión general

Los componentes de entrada de TBM Studio permiten a los usuarios introducir o modificar datos directamente en los informes. A diferencia de los componentes de filtro, que simplemente restringen los datos mostrados, los componentes de entrada vuelven a introducir datos en tablas editables, que luego pasan por el flujo de datos y, en última instancia, influyen en los cálculos del modelo de costes.

Los componentes de entrada se configuran principalmente a través de las propiedades editables de las columnas de las tablas y se ponen a disposición de los usuarios finales mediante componentes de informe. Comprender la diferencia entre filtrar datos (segmentadores) y modificarlos (componentes de entrada) es fundamental para crear informes eficaces que permitan introducir datos.

**Principio clave: Los componentes de entrada funcionan con tablas editables. Para habilitar la introducción de datos en los informes, primero debe crear una tabla editable y configurar sus columnas con los tipos de entrada, las reglas de validación y las restricciones adecuados.**
