# Informes principales

## Visión general

Los informes maestros sirven como plantillas reutilizables que proporcionan un diseño, una imagen de marca y elementos de diseño uniformes en múltiples informes de TBM Studio. Al crear un informe maestro y aplicarlo a tus informes personalizados, garantizas la coherencia visual y reduces la carga de mantenimiento que supone actualizar los elementos comunes en todo tu conjunto de informes.
