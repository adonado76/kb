# Configuración del informe maestro

Los informes maestros sirven como plantillas que se pueden aplicar a varios informes, lo que garantiza un diseño y una apariencia uniformes en todo un conjunto de informes.

**Características principales:**

- Los informes maestros son siempre informes de primer nivel
- Los componentes de un informe maestro aparecen como «fondo»: son visibles, pero no se pueden editar en los informes derivados
- Los componentes añadidos a los informes derivados se superponen al diseño maestro
- Los informes maestros estándar son proporcionados por el sistema y no se pueden eliminar

**Para crear un informe maestro:**

1. Echa un vistazo al informe
2. Ve a la pestaña «Informe» > grupo «Avanzado»
3. Haz clic en la opción «Informe maestro»
4. Guardar y registrarse
