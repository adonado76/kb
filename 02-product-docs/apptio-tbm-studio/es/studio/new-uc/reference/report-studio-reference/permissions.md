# Permisos

Controle quién puede realizar acciones de introducción de datos mediante permisos a nivel de fila:

|  |  |  |
| --- | --- | --- |
| **Permiso** | **Descripción** | **Valor predeterminado** |
| Permiso para añadir filas | ¿Quién puede añadir nuevas filas? | Todo el mundo |
| Permiso para duplicar filas | ¿Quién puede duplicar filas? | Todo el mundo |
| Permiso para eliminar filas | ¿Quién puede eliminar filas individuales? | Todo el mundo |
| Permiso para eliminar todas las filas | ¿Quién puede eliminar todas las filas? | Administrador y socio |
| Permiso para editar fila | ¿Quién puede editar los valores de las celdas? | Todo el mundo |
| Permiso para publicar filas | ¿Quién puede publicar cambios? | Todo el mundo |
| Permiso de descarga | ¿Quién puede descargar los datos de la tabla? | Todo el mundo |
| Permiso para subir archivos | ¿Quién puede subir datos de tablas? | Todo el mundo |
