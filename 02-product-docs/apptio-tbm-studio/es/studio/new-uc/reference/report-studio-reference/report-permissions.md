# Permisos del informe

## Visión general

Los permisos de los informes permiten controlar quién puede ver, editar y administrar los informes en TBM Studio. El sistema de permisos funciona en varios niveles —desde la asignación de roles a nivel de dominio hasta la visibilidad de componentes individuales—, lo que permite un control preciso del acceso a los informes sin dejar de cumplir los requisitos de seguridad y gobernanza.

En esta sección se describe la arquitectura completa de permisos para los informes, incluida la relación entre los permisos de los informes y la seguridad a nivel de fila (RLS), las prácticas recomendadas para el diseño de permisos y la resolución de problemas comunes de acceso.
