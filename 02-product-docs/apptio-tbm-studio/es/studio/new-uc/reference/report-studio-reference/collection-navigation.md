# Navegación por la colección

**Componente de recopilación de informes de Navigator**

El navegador de colecciones de informes es un componente de informes que ofrece a los usuarios una forma sencilla de desplazarse entre los informes de una colección. Cuando añades un informe a una colección, el componente de navegación se añade automáticamente al informe.

El navegador suele aparecer como una barra horizontal en la parte superior del informe, en la que se muestran los nombres de todos los informes visibles de la colección. Los usuarios pueden hacer clic en el nombre de cualquier informe para acceder directamente a él.

**Funciones de navegación**

El navegador de recopilación de informes ofrece las siguientes funciones de navegación:

|  |  |
| --- | --- |
| **Característica** | **Descripción** |
| Navegación directa | Haz clic en el nombre de cualquier informe en el navegador para ir directamente a ese informe. |
| Indicador de informe actual | El informe que se está visualizando aparece resaltado en el navegador, lo que ayuda a los usuarios a mantener el contexto. |
| Contexto del filtro | Las selecciones y los filtros del segmentador se pueden transferir, si se desea, de un informe a otro al navegar. |
| Pantalla adaptativa | El navegador se adapta al ancho de pantalla disponible, ajustándose o reduciéndose según sea necesario. |

**Métodos alternativos de navegación**

Además del navegador de colecciones, TBM Studio admite otros métodos de navegación entre informes:

- **Botones:** Botones en los que se puede hacer clic para acceder a informes específicos
- **Enlaces codificados:** enlaces con sintaxis de tipo wiki en cuadros de texto o botones HTML
- **Enlaces a informes:** enlaces desde los valores de las tablas o los elementos de los gráficos a informes relacionados
