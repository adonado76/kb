# Gráficos de radar

Los gráficos de radar (también conocidos como gráficos de araña) comparan varias características entre dos o más entidades mediante ejes que parten de un punto central. Son útiles para comparar los datos presupuestados con los reales en distintos departamentos o para evaluar elementos según múltiples criterios.

**Tipos de gráficos de radar**

- Radar relleno: muestra áreas poligonales rellenas que conectan los puntos de datos. Ideal para comparar tres características o menos, a fin de evitar la saturación visual.
- Radar con marcadores: muestra únicamente los puntos de datos y las líneas que los unen, sin relleno. Apto para cualquier número de características.

**Cómo crear un gráfico de radar**

1. En la pestaña Informe, haz clic en Gráfico.
2. Arrastra un campo al área «Eje» (esto define las características que se comparan).
3. Arrastra uno o varios campos al área «Valores».
4. En la pestaña «Ad Hoc», haz clic en «Radar».
5. En el menú desplegable «Radar», selecciona «Radar con relleno» o «Radar con marcadores».

**Comportamiento del eje de pivote:** cuando se aplica un eje de pivote a un gráfico de radar, este afecta al eje en lugar de a las filas (a diferencia de otros tipos de gráficos).

**Número máximo de filas:** controla el número de ejes que se muestran. El valor predeterminado es seis. A diferencia de otros gráficos, los gráficos de radar no incluyen una categoría «Otros» para los elementos sobrantes.
