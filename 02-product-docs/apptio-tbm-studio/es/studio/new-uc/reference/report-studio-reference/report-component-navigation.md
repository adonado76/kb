# Componentes del informe: Navegación

## Visión general

Los componentes de navegación permiten a los usuarios desplazarse entre informes, páginas y recursos externos dentro de TBM Studio. Un diseño de navegación eficaz mejora la experiencia del usuario al ofrecer un acceso intuitivo a información detallada e informes relacionados. Este documento de referencia recoge todos los componentes de navegación, sus opciones de configuración y las mejores prácticas para su implementación.

TBM Studio ofrece varios mecanismos de navegación: componentes de botones para controles de navegación explícitos, enlaces a informes para profundizar en los datos desde tablas y gráficos, enlaces codificados con sintaxis de estilo wiki para una navegación flexible, navegadores de colecciones de informes para organizar informes relacionados y filtros jerárquicos para el filtrado detallado.
