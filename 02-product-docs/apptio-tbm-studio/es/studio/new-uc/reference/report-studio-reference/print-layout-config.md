# Configuración del diseño de impresión

El modo de diseño de impresión ofrece controles independientes para configurar el aspecto de los informes al exportarlos a PDF o al imprimirlos.

**Acceder al modo de diseño de impresión**

1. Echa un vistazo al informe para revisarlo.
2. En la pestaña «Inicio», haz clic en «Ver» y, a continuación, selecciona «Diseño de impresión».
3. Aparece la pestaña «Diseño de impresión» con los controles de diseño.

Para salir: en la pestaña «Diseño de impresión», haz clic en «Salir del modo de diseño de impresión».

**Opciones de configuración de página**

|  |  |
| --- | --- |
| **Opción** | **Descripción** |
| **Tamaño de página** | Selecciona el tamaño de papel en el menú desplegable (Carta, A4, Legal, etc.) |
| **Flujo** | Controla la ubicación de los componentes (manual, vertical u horizontal) |
| **Márgenes** | Configurar los márgenes de la página para la impresión |
| **Cabecera** | Añadir texto de encabezado; se mostrará en todas las páginas o solo en la primera página. Admite formato HTML y texto dinámico |
| **Pie de página** | Personalizar el texto del pie de página; no admite texto dinámico |
| **Vertical** | Orientación vertical de la página |
| **Horizontal** | Orientación horizontal de la página |
| **Escala** | Nivel de ampliación para la impresión |

**Opciones de flujo**

El flujo determina cómo se distribuyen los componentes en la página impresa:

|  |  |
| --- | --- |
| **Configuración del caudal** | **Comportamiento** |
| **Manuales** | Los componentes se colocan manualmente; permanecen donde se han colocado |
| **Vertical** | Los componentes se disponen automáticamente en una sola columna vertical |
| **Horizontal** | Los componentes se disponen automáticamente en filas, con un número máximo de componentes por fila |

**Opciones de impresión de componentes**

**Fijar/Desfijar componentes:** Fija los componentes al final de un informe para su impresión. Esto resulta útil para desplazar los cortadores hasta el final y mostrar los filtros aplicados, o para fijar todos los componentes de un tipo específico.

**Mostrar/ocultar componentes:** controla qué componentes aparecen en la impresión:

1. En la pestaña «Diseño de impresión», haz clic en «Mostrar/ocultar componentes del informe».
2. Desmarca las casillas de los componentes que quieras ocultar.
3. Utiliza el campo «Filtro» para buscar componentes específicos en listas largas.

**Configuración de tablas para la impresión de varias páginas**

Para imprimir una tabla extensa en varias páginas:

1. Accede al modo de diseño de impresión.
2. Coloca la tabla en una página aparte.
3. Selecciona la tabla y, a continuación, en la pestaña «Diseño de impresión», haz clic en «Ajustar el componente a página completa».
4. Con la tabla seleccionada, en la subpestaña «Tabla», selecciona «Imprimir todas las páginas».
5. Guarda y sal del modo de diseño de impresión.
