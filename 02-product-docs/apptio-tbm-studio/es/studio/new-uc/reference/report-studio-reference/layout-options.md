# Opciones de diseño

## Visión general

TBM Studio Report Studio ofrece un sistema de diseño flexible para colocar y ajustar el tamaño de los componentes de los informes. En esta sección se describe el sistema de maquetación completo, incluyendo el tamaño de las pantallas, la ubicación de los componentes, los componentes contenedores y la configuración de la maquetación para impresión.

Los informes se crean en el área de trabajo «Edición de informes», donde se añaden y organizan tablas, gráficos, filtros, botones y otros componentes. El sistema de maquetación admite tanto el posicionamiento manual, para un control preciso, como las maquetaciones automáticas con flujo continuo, para diseños adaptativos.
