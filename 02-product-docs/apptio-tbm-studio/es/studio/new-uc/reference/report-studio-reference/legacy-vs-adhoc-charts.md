# Gráficos tradicionales frente a gráficos ad hoc

TBM Studio admite dos métodos de creación de gráficos: el enfoque moderno «ad hoc» (recomendado) y los gráficos heredados de versiones anteriores.

**Gráficos personalizados (recomendados)**

- Creado mediante el panel de configuración de componentes con campos de arrastrar y soltar.
- Compatible con herramientas de corte, selección de columnas y tablas dinámicas rápidas.
- Acceso completo a todos los tipos de visualización en la pestaña «Ad Hoc».
- Funciones interactivas como el filtrado de leyendas y el zoom.
- Acceso a paletas de colores personalizadas.

**Gráficos antiguos**

- Creado con herramientas antiguas y editado mediante la configuración de la ruta de datos.
- No es compatible con los filtros ni con otros componentes interactivos.
- Algunas propiedades marcadas con (\*) en el cuadro de diálogo «Propiedades» solo se aplican a los gráficos antiguos.
- Se puede convertir a partir de gráficos ad hoc, pero la conversión no se puede revertir.

**Conversión:** Para convertir un gráfico ad hoc en un gráfico heredado, abre el menú Gráfico situado en la esquina superior izquierda del marco del gráfico y selecciona «Convertir en gráfico heredado». Esta acción es definitiva y no se puede deshacer.
