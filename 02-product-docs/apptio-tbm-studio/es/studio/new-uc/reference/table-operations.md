# Operaciones con tablas

## Operaciones con columnas

|  |  |
| --- | --- |
| **Operación** | **Proceso** |
| Añadir una fila | Haz clic en el botón «Añadir fila» situado en la parte inferior de la tabla |
| Filtrar por valor | Haz clic con el botón derecho del ratón en la celda > Filtrar por valor |
| Congelar columna | Haz clic con el botón derecho del ratón en el encabezado > Formato > Congelar columna |
| Establecer el color de fondo | Haz clic con el botón derecho del ratón en el encabezado > Formato > Establecer color de fondo |
| Establecer el ancho de columna | Arrastra el borde o haz clic con el botón derecho > Formato > Establecer ancho |
| Mostrar valores (duplicados) | Haz clic con el botón derecho del ratón en el encabezado > Mostrar valores |

## Truncar (tablas editables)

***Aplicable a:****12.11.4 y versiones posteriores*

|  |  |
| --- | --- |
| **Tipo de tabla** | **Efecto de truncamiento** |
| Tabla editable con formato avanzado | Vuelve a los valores originales de la fuente |
| Tabla general editable | Elimina todas las filas |

**Advertencia:** el truncamiento es definitivo una vez finalizado el check-in. Si es necesario, revierte los cambios antes de guardar.
