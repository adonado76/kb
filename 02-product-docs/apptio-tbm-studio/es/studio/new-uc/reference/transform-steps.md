# Pasos de la transformación

Los pasos de transformación conforman un flujo de trabajo que procesa los datos desde la fuente hasta el resultado final. Los pasos se ejecutan de forma secuencial, de arriba abajo.

## Descripción general del proceso de transformación

**Pasos predeterminados** (siempre presentes):

- **Fuente:** Determina el tipo de fuente de datos
- **Tabla:** Muestra el resultado final tras todas las transformaciones

**Pasos para subir archivos** (añadidos automáticamente):

- **Carga:** selección y gestión de archivos
- **Importar:** Ajustes de configuración de importación
