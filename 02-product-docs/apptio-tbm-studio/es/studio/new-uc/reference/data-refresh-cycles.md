# Ciclos de actualización de datos

|  |  |  |
| --- | --- | --- |
| **Ciclo** | **Descripción** | **Comprobación de la fecha de caducidad** |
| Actualizaciones puntuales | Actualizar según sea necesario; añadir o sustituir | Ninguna |
| Versión 1; No hay actualizaciones programadas | Versión única, sustituida al subirla | Ninguna |
| Versión 1; Actualización mensual | Actualización mensual, sustituye a la versión actual | Mensualmente |
| 1 versión; Actualización anual | Actualización anual, sustituye a la versión actual | Anual |
| Versiones mensuales; Actualización cada mes | Una nueva versión cada mes | Mensualmente |
| Versiones anuales; Actualización mensual | Versiones de los últimos 12 meses | Mensualmente |
| Versiones anuales; actualización a principios de año | Versión anual al inicio del ejercicio fiscal | Anual |
| Versiones anuales con elementos heredados | Anual con el remanente del año anterior | Anual |
