# Opciones de configuración del controlador

**Parámetros completos del controlador**

|  |  |  |  |
| --- | --- | --- | --- |
| **Parámetro** | **Obligatorio** | **Valores** | **Descripción** |
| Añadir a (Métricas) | Sí | Coste, presupuesto, previsión, personalizado | ¿Qué métricas recogen el valor de este conductor? |
| Uso de (Tipo) | Sí | Columna, Métrico, Fórmula | Cómo se determina el valor |
| Columna | Condicional | Nombres de las columnas de la tabla | Obligatorio al utilizar = Columna |
| Métrica | Condicional | Métricas disponibles | Obligatorio al utilizar = Sistema métrico |
| Fórmula | Condicional | Expresión de fórmula válida | Obligatorio al utilizar la fórmula = |
| Notas | Nee | Texto libre | Documentación para facilitar el mantenimiento |
