# Opciones de visualización

**Funciones de visualización de informes**

|  |  |  |
| --- | --- | --- |
| **Característica** | **Acción** | **Descripción** |
| Ver pilotos/asignaciones | Haz clic en una tabla del informe | Muestra los controladores y las asignaciones de esa tabla |
| Seguimiento del flujo de valores | Haz clic en los elementos de la tabla de forma secuencial | Muestra la ruta de costes a través del modelo ( v12.1+ ) |
| Taladrar en la columna | Haz clic en el menú > Desglose > selecciona la columna | Expande la tabla para mostrar los valores de las columnas |
| Ver artículo concreto | Haz clic en el elemento tras profundizar en el análisis | Muestra las asignaciones correspondientes a ese artículo concreto |
| Volver a la vista de tabla | Haz clic en la flecha de retorno situada en la esquina superior | Oculta el desglose |
| Cambiar métrica | Utilizar el menú desplegable del selector métrico | Cambios en la métrica mostrada |

**Asignaciones dentro de cada nivel**

A partir de la versión v12.2.2, las asignaciones entre tablas del mismo nivel se muestran con un desplazamiento en lugar de líneas en bucle.
