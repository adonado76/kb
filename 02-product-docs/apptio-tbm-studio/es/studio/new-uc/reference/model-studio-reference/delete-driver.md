# Eliminar un controlador

Para eliminar un controlador de un objeto de modelo:

1. Haz clic en el factor determinante del diagrama de Sankey
2. Haz clic en «Eliminar» en la parte inferior del panel
3. Guardar cambios

Advertencia: Al eliminar un controlador, se eliminan todos los valores asociados del objeto del modelo. Comprueba el impacto en las asignaciones posteriores antes de eliminar.
