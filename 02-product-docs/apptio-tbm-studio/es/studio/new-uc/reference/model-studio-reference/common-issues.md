# Problemas habituales

|  |  |  |
| --- | --- | --- |
| **Emitir** | **Causa** | **Resolución** |
| No se puede añadir la tabla al nivel | A la tabla le falta el paso «Modelo» | Añadir el paso «Modelo» a la transformación de la tabla |
| No se puede añadir una columna a un nivel con tablas | No se permite contenido mixto | Elimina primero las tablas o utiliza un nivel diferente |
| La carga del informe es lenta | Se han seleccionado demasiadas métricas | Limitar métricas mediante «Editar métricas» |
| Asignaciones pendientes | Mesas sin registrar | Registrar todos los cambios de modelo |
| El informe no se encuentra en la colección | No asignado a ninguna colección | Utiliza la opción «Asignar a colección» |
