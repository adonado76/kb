# Reglas de agregación de métricas

**Cómo se agregan las métricas modeladas**

Al visualizar métricas modeladas en diferentes niveles de agrupación:

- Nivel de detalle: muestra los valores de cada fila
- Nivel agrupado: suma todos los valores del grupo
- En todos los periodos: sigue la configuración de «Comportamiento temporal»

**Cómo se agregan las métricas calculadas**

Las métricas calculadas vuelven a calcular su fórmula en cada nivel de agrupación, en lugar de sumarlas:

- Nivel de detalle: Fórmulas por fila
- Nivel agrupado: la fórmula se aplica a los valores agrupados
- Excepción: si la opción «Sumar» está marcada, los valores se suman
