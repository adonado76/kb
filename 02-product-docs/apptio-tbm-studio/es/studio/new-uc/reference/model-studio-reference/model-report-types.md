# Tipos de informes modelo

**Informes basados en tablas**

Mostrar las tablas completas en filas. Los usuarios finales pueden profundizar en los datos para ver los valores de las columnas.

**Casos de uso**

- Permitir a los usuarios explorar diferentes dimensiones (grupos de costes, proveedores, etc.)
- Mostrar vista completa de una capa del modelo
- Ofrecer flexibilidad para realizar análisis puntuales

**Informes basados en columnas**

Mostrar columnas específicas, lo que permite obtener una visión centrada en determinadas dimensiones.

**Casos de uso**

- Informe centrado en elementos específicos (por ejemplo, solo el grupo de costes)
- Proporcionar una visión coherente para la elaboración de informes ejecutivos
- Simplificar el contenido para públicos específicos
