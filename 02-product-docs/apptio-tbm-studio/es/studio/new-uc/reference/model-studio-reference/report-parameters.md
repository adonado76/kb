# Parámetros del informe

**Opciones de configuración**

|  |  |
| --- | --- |
| **Parámetro** | **Descripción** |
| Nombre del informe | Nombre que se mostrará para el informe (visible en el Explorador de proyectos) |
| Descripción | Documentación opcional sobre la finalidad del informe |
| Métricas seleccionadas | Qué métricas calcular y mostrar (afecta al rendimiento) |
| Estructura de niveles | Número, orden y contenido de los niveles |
| Recopilación de informes | Asignación opcional a una colección para facilitar el acceso ( v12.2.2+ ) |

**Optimización del rendimiento**

Limita las métricas seleccionadas para mejorar la velocidad de cálculo en los entornos de prueba y producción.

**Limitar métricas**

1. Abre el informe y échale un vistazo
2. Haz clic en Ver > Mostrar documento > Editar métricas
3. Selecciona solo las métricas que necesites
4. Haz clic en «Aceptar» y guarda
5. Consulta el informe

**Consejo:** Seleccionar menos métricas reduce considerablemente el tiempo de cálculo, sobre todo en modelos de gran tamaño.

**Asignación a la recopilación de informes**

1. Abrir el informe del modelo
2. Haz clic en la pestaña «Modelado»
3. Haz clic en «Asignar a la colección»

1. Selecciona la colección de informes en el menú desplegable
2. Guardar el informe

*→ Véase la sección 3.3.5: Organizar los informes en colecciones para la gestión de colecciones*
