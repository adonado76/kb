# Conceptos del diagrama de Sankey

El diagrama de Sankey se utiliza tanto en la vista de tabla única como en la vista de informe de modelo para visualizar el flujo de valores.

**Elementos del diagrama**

|  |  |
| --- | --- |
| **Elemento** | **Descripción** |
| Nodos (rectángulos) | Representan objetos del modelo o columnas de una tabla. La columna «Ancho» indica el valor total. |
| Líneas de flujo | Representar las asignaciones entre objetos. Ancho proporcional al valor asignado. |
| Dirección del flujo | El valor fluye de izquierda a derecha (desde las fuentes hacia los destinos) |
| Etiquetas de precios | Mostrar valores métricos en nodos y flujos |
