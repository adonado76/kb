# Creación de tablas

**Crear una nueva tabla**

**Navegación:** pestaña Inicio > Nuevo > Tabla

**Parámetros:**

|  |  |  |
| --- | --- | --- |
| **Parámetro** | **Obligatorio** | **Descripción** |
| Nombre | Sí | Identificador único de la tabla |
| Categoría | Sí | Organización de carpetas en el Explorador de proyectos; se pueden utilizar las existentes o crear otras nuevas |

- Haz clic en la pestaña **«Inicio»**
- Haz clic en **Nuevo** > **Tabla**
- Introduce el nombre de la tabla
- Introduce o selecciona **una categoría**
- Haz clic **en Aceptar**

El sistema crea la tabla y muestra el paso «Fuente» en el proceso de transformación.

**Tema principal:** [Tablas](../../../studio/new-uc/reference/tables.html)
