# Data Studio Referencia

## Visión general

Data Studio es el entorno de gestión de datos de TBM Studio en el que se importan, transforman y preparan los datos para la modelización de costes y la elaboración de informes. Esta guía ofrece una documentación completa de todas las funciones de Data Studio, organizada para facilitar la consulta.

Data Studio te permite:

- Importar datos desde archivos, tablas existentes, tablas editables y sistemas externos
- Transforma los datos mediante un flujo de trabajo configurable
- Asignar datos a conjuntos de datos maestros para garantizar el cumplimiento normativo de las aplicaciones
- Publicar tablas para exportarlas a sistemas externos
- Aplicar la seguridad a nivel de fila para la gobernanza de datos
