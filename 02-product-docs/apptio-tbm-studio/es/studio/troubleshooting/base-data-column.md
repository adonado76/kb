# Columna de datos básicos Recomendación no utilizada

Vaya a **TBM Studio** >� pestaña **Recomendaciones** > **Columnas de datos base no utilizadas** y, a continuación, seleccione **Solucionar todos los problemas identificados**.

![recomendación de datos de base](../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/bd-1.jpg)

Cambie al espacio de trabajo **Desarrollo** y seleccione **Siguiente**.

![flujo de recomendación](../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/workflow-common.jpg)

Seleccione individualmente el botón **Fijar**.

![siguiente](../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/bd-2.jpg)

Una vez completado, el estado de las columnas cambia.

![](../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/bd-4.jpg)

Seleccione **Siguiente** y, a continuación, **Checkin** en la última página.

![registro](../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/bd-3.jpg)
