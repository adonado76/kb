# Introducción a TBM Studio

TBM Studio es un conjunto de herramientas que le ayuda a gestionar su gasto empresarial en tecnología. Incluye tres componentes principales:

- Data Studio
- Estudio de modelos
- Estudio de informes

- **Data Studio** : Le permite [recopilar los datos adecuados, transformarlos de forma inteligente](../data_studio/data-transform-chapter-title-page.html "Se aplica a: TBM Studio 12.0 y posteriores") y mejorarlos con el tiempo.
- **Model Studio** : Le permite [personalizar visualmente las asignaciones y configurar un modelo de costes flexible](../model_studio/models-chapter-title-page.html "Se aplica a: TBM Studio 12.0 y posteriores") para reflejar con precisión los costes y servicios de TI.
- **Estudio de informes** : Permite [personalizar los informes existentes o crear otros nuevos](../reports/intro_report_studio.html "Se aplica a: TBM Studio 12.0 y posteriores") que incluyan tablas, gráficos, así como componentes para filtrar los informes, introducir datos y navegar entre ellos.

Para más información, consulte [Conceptos esenciales de TBM Studio](../admin/essential-tbmstudio-concepts.html "Se aplica a: TBM Studio 12.0 y posteriores. En TBM Studio, organizas tu trabajo en proyectos. Dentro de un proyecto, se recopilan y transforman datos en conjuntos de datos, se crean modelos, se añaden objetos a los modelos, se asignan conjuntos de datos a los objetos, se imputan costes de un objeto a otro y se resumen los resultados en uno o varios informes. Instancias, dominios y proyectos organizan tu trabajo:")
