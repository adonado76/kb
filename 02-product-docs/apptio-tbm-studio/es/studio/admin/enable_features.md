# Características de la versión beta

TBM Studio a veces contiene funciones que aún no se han puesto a disposición de todos los clientes. Se trata de funciones de la versión beta y es posible que no funcionen exactamente como se diseñaron.

## Activar las funciones de la versión beta

Como TBMA, puede utilizar la opción Habilitar funciones para controlar a qué funciones de la versión beta pueden acceder sus usuarios. Cuando activas una función Beta, se hace visible para todos los usuarios de tu Entorno.

1. En la pestaña Proyecto, seleccione Activar funciones.
2. Si no ha iniciado sesión como TBMA, la opción Activar funciones estará desactivada. ![](../../../../../03-media/apptio-tbm-studio/studio/admin/enable_features.png)
3. Utilice la casilla de verificación para activar o desactivar las funciones de la versión beta.
