# Controle el acceso a las aplicaciones Apptio

**Se aplica a** : TBM Studio v12.1, v12.2 y posteriores

Para controlar el acceso a las aplicaciones Apptio, utilice Enhanced Access
Administration, la página Apptio Access Administration . Las partes del sistema a las que puede acceder un usuario y las acciones que puede realizar se controlan mediante roles. Por defecto, durante los periodos de mantenimiento se impide a los usuarios acceder al sistema. Puede anular esta configuración. Para obtener información sobre el uso de Enhanced Access Administration, consulte [Acerca de Frontdoor](/docs/SSFG2DK/frontdoor/home.htm).
