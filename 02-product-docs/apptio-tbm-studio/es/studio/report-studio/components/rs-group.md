# Grupo

La función de agrupación le permite organizar fácilmente varios componentes en el diseño de su informe agrupándolos o desagrupándolos. Esto ayuda a gestionar los componentes relacionados como una sola unidad.

## Cuándo utilizar los grupos

Utiliza los grupos cuando desees:

- Mantenga el contenido relacionado visualmente junto
- Mover varios componentes juntos
- Simplifica diseños complejos

## Añadir un grupo al informe

1. Crear grupo seleccionando varios componentes
   1. Mantenga pulsada la tecla **Mayús** y haga clic para seleccionar varios componentes en el lienzo.
   2. Pulse **Ctrl + G (Cmd + G en Mac)** para agrupar los componentes seleccionados. (Verás que los componentes agrupados aparecen como una jerarquía anidada en el **panel Capas).**
2. En la barra de herramientas también hay un **icono de Grupo** : al hacer clic en él se realiza la misma acción.

   ![](../../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/grp-icon.png)
3. Crear un grupo enmarcado
   1. Añadir un grupo desde el panel Componentes de la barra de herramientas
   2. Panel de formato
      1. Propiedades generales: consulte Propiedades de los componentes
4. Desagrupar
   1. Para desagrupar, seleccione el componente agrupado y vuelva a hacer clic en el icono Agrupar (que actúa como Desagrupar cuando se selecciona un grupo).
   2. La jerarquía se actualiza en el panel Capas, mostrando de nuevo los componentes individuales.
