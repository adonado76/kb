# Descripción general de las visualizaciones

Las visualizaciones te ayudan a representar los datos de forma visual, de modo que las tendencias, los patrones y las comparaciones sean más fáciles de comprender de un vistazo. Report Studio ofrece una amplia gama de tipos de visualización que funcionan a la perfección con filtros, segmentadores y navegación detallada para facilitar el análisis interactivo.

## Qué se puede hacer con las visualizaciones

Con las visualizaciones, puedes:

- Comparar valores entre categorías
- Identificar tendencias y cambios a lo largo del tiempo
- Resaltar patrones y valores atípicos
- Habilite la exploración interactiva mediante el filtrado y la navegación por niveles

## Tipos de visualización compatibles

Report Studio admite múltiples tipos de visualización, entre los que se incluyen:

- KPI, tablas y tablas editables
- Gráficos de barras y columnas
- Gráficos de barras apiladas y columnas apiladas
- Gráficos de líneas
- Gráficos circulares
- Gráficos de columnas + líneas y columnas apiladas + líneas

## Visualizaciones compatibles

La función Visualizaciones compatibles permite cambiar rápidamente entre tipos de gráficos compatibles sin tener que reconstruir la visualización. Esto le ayudará a experimentar con distintos formatos visuales y a encontrar el que mejor represente sus datos.

**Pasos a seguir**

1. Seleccione un gráfico
   1. Haga clic en un gráfico (por ejemplo, un gráfico de barras) de su informe.
2. Utilizar la opción de conversión
   1. En la cabecera del widget, localice el icono Visualizaciones compatibles.
   2. Seleccione el icono para ver una lista de visualizaciones compatibles para el tipo de gráfico seleccionado.

      ![imagen de visualizaciones compatibles](../../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/cv-2b.png)
3. Seleccione un tipo de visualización
   1. Seleccione entre opciones como:
      1. Gráfico de columnas
      2. Gráfico de barras apiladas
      3. Gráfico de columnas apiladas
   2. El gráfico se actualiza instantáneamente al tipo visual elegido, conservando los datos, el formato y la configuración existentes.
