# Menú Edición

El menú Edición del lienzo del informe es

![menú de edición](../../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/rs-editmenu.png)

*Deshacer/Rehacer*

Estas opciones se utilizan para deshacer cambios pequeños y discretos mientras se editan documentos en el espacio de trabajo. Cuando se retiran documentos, se puede volver a una versión anterior o rehacerla. Algunos escenarios para deshacer o rehacer los cambios son:

- Posicionamiento de los elementos de un informe
- Modificaciones discretas de la configuración de los elementos del informe (por ejemplo: texto de los botones, contenido HTML, etc.)
- Edición de fórmulas para columnas en el paso de fórmulas de una tabla

*Seleccionar todo*

Esta opción selecciona todos los componentes presentes en el lienzo del informe.
