# Organizar menú

El nuevo estudio de informes ofrece herramientas de diseño flexibles para colocar y organizar los componentes de un informe. La mayoría de estas herramientas de diseño se encuentran en el menú **Organizar**. Al menos un componente debe estar seleccionado para activar los comandos del menú **Organizar**. Para empezar, haga clic en el componente que desea modificar.

El menú Organizar del lienzo del informe es

![organizar menú](../../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/rs-arrangemenu.png)

*Alinear*

Utilice los comandos del grupo Alinear para controlar el aspecto relativo de dos o más componentes. Estos comandos sólo se activan después de seleccionar dos o más componentes en un informe.

Seleccione los componentes que desea alinear. Vaya a **Organizar** > **Alinear** y seleccione la opción de alineación que desee. Las opciones son

![alinear menú](../../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/rs-align.png)

Para alineaciones horizontales, utilice las opciones **Izquierda**, **Centro** y **Derecha**, mientras que para la alineación vertical, elija las opciones **Superior**, **Medio** e **Inferior**.

*Desplazar componentes del informe*

En un informe, puede reposicionar los componentes, hay más de uno.

Haga clic y mantenga pulsada la cabecera del componente del informe y, a continuación, arrastre el componente a una nueva ubicación. Si desplaza un componente a una posición en la que se solapa con otro, puede seleccionar qué componentes deben estar delante del otro.

En el menú **Organizar**, seleccione el componente y elija cualquiera de las siguientes opciones:

- Traer al frente - mueve el componente hasta el frente
- Adelantar - adelanta un paso el componente
- Enviar al fondo: desplaza el componente hasta el fondo
- Enviar hacia atrás - desplaza el componente un paso hacia atrás

***Distribuya***

Utilice la opción Distribuir para espaciar uniformemente los componentes horizontal o verticalmente, ajustando el diseño automáticamente, incluso cuando algunos componentes estén ocultos para determinados usuarios. Los atajos para distribuir son:

- Mayús+Alt+H - Distribuir horizontalmente
- Mayús+Alt+V - Distribuir verticalmente
