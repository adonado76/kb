# Cómo funciona el nuevo Report Studio

El nuevo Report Studio sigue un flujo de trabajo sencillo y estructurado que separa la creación de informes del consumo de informes. Esto garantiza que los informes sean fáciles de crear, mantener y utilizar a gran escala.

Esta página explica el flujo de principio a fin y cómo interactúan los diferentes usuarios con Report Studio.

**Flujo de trabajo de alto nivel**

Report Studio funciona en cuatro etapas principales:

1. Crear
2. Configurar
3. Organizar
4. Ver e interactuar

Cada etapa se basa en la anterior.

**Crear informes**

Los administradores comienzan creando un informe.

Un informe define:

- Los datos que se mostrarán
- Los componentes utilizados (tablas, KPI, segmentadores, etc.)
- Las visualizaciones que representan los datos

**Configurar componentes y visualizaciones**

Una vez creado un informe, los administradores configuran cómo se muestran y se comportan los datos.

Esto incluye:

- Selección de fuentes de datos, métricas y dimensiones
- Configuración de las propiedades de los componentes
- Configuración de visualizaciones (gráficos, tablas, diseños)
- Definición de interacciones entre componentes (por ejemplo, segmentadores que afectan a los gráficos)

Las opciones de configuración comunes que se aplican a todos los componentes y visualizaciones se documentan por separado para evitar duplicaciones.

**Organizar informes en colecciones**

Una vez creados los informes, los administradores pueden agrupar los informes relacionados en colecciones de informes.

Ayuda para recopilar informes:

- Organiza los informes por tema, público o caso de uso
- Mejorar la visibilidad para los usuarios finales
- Establezca permisos en una colección para controlar qué usuarios pueden ver los informes que contiene

**Ver e interactuar con informes**

Los usuarios finales acceden a los informes a través del visor de informes. En el visor de informes, los usuarios pueden:

- Navegar entre informes y colecciones de informes
- Interactuar con filtros y selectores
- Ver datos en varias monedas
- Exportar informes a PDF

**Funciones en el flujo de trabajo**

En cada etapa participan diferentes roles:

- Administradores: se dedican principalmente a crear, configurar y organizar informes.
- Usuarios finales: se dedican principalmente a ver, interactuar y exportar informes.
