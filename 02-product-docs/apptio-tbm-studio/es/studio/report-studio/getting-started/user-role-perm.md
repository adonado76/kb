# Roles y permisos de usuario

El nuevo Report Studio admite diferentes roles de usuario para garantizar el nivel adecuado de acceso para crear, gestionar y ver informes. Esta página explica las funciones disponibles y lo que puede hacer cada una de ellas.

Comprender las funciones te ayuda a saber **quién puede crear y configurar informes y quién puede verlos e interactuar con ellos.**

Hay dos funciones principales en Report Studio:

- Administrador: crea y gestiona informes y colecciones de informes
- Usuario final: visualiza e interactúa con los informes

**Función de administrador**

Los administradores son responsables de **crear y mantener** los informes. También pueden

- Crear, editar, duplicar y eliminar informes
- Crear y gestionar colecciones de informes
- Añadir y configurar componentes (tablas, segmentadores, KPI, etc.)
- Configurar visualizaciones y ajustes de gráficos
- Definir y gestionar fuentes de datos, métricas y filtros
- Configurar el diseño y el formato
- Vista previa de informes en el visor de informes
- Exportar informes a PDF
- Ver datos en varias monedas

**Función del usuario final**

Los usuarios finales consumen los informes creados por los administradores. Se centran en **visualizar, interactuar y exportar** datos de informes. También pueden

- Ver informes y colecciones de informes
- Navegar entre informes
- Interactuar con filtros y selectores
- Exportar informes a PDF
- Ver datos en varias monedas
