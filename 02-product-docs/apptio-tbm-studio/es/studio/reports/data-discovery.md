# Utilizar informes para descubrir datos

**Se aplica a** : TBM Studio 12.0 y posteriores

Vea este vídeo de demostración de Apptio Education Services: [Utilización de informes modelo](https://community.apptio.com/videos/1583 "(se abre en una pestaña o una ventana nueva)"). O consulte [todos los vídeos de Apptio](https://community.apptio.com/docs/DOC-7714 "(se abre en una pestaña o una ventana nueva)").

Los informes estáticos pueden proporcionar información valiosa mediante tablas y gráficos. Y, si las tablas y los gráficos contienen enlaces, los usuarios pueden profundizar para obtener más detalles sobre elementos individuales. Puede hacer que los informes sean aún más útiles ofreciendo a los usuarios la posibilidad de filtrar los informes y encontrar relaciones mediante el uso de rebanadores. Con los troceadores, el usuario puede centrarse en elementos concretos de las tablas y los gráficos y obtener información que de otro modo no vería.

En la siguiente imagen se muestra un informe de Transacción GL con varios cortadores. Utilizando las rebanadoras, el usuario puede filtrar la tabla de **Transacciones GL** y el gráfico de **Categorías** por categoría, subcategoría, fecha y descripción. Además, pueden utilizar el control deslizante de la parte inferior del informe para filtrar por **Coste**.

![](../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/studioimages/reports/rep050_541x479.png)

Por ejemplo, en la siguiente imagen, el informe se ha filtrado por Categoría (**Centros de Datos** y **Redes** ), Subcategoría (**Dallas** y **LAN** ) y Descripción (**Equipamiento** y **Alquiler** ), así como por costes superiores a 10.000 dólares. La ventaja de las cortadoras es que permiten muchas combinaciones de filtros.
