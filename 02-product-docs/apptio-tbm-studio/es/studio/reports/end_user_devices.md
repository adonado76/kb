# Dispositivos de usuario final

End User Devices (EUD) proporciona información para que los gestores de activos de TI y los gestores de soporte de escritorio tomen decisiones basadas en datos sobre la consolidación de flotas, la optimización y los costes previstos. A día de hoy, existen 3 componentes relacionados con los dispositivos de usuario final.

- CT Apps-Dispositivos de usuario final: Este es un componente antiguo, y no tiene informes
- BoIT–End dispositivos de usuario: Este es un componente antiguo, y no tiene informes
- **Dispositivos de usuario final** : Este es el último componente con informes que los clientes tienen que instalar para ver los informes con 7 soluciones y 27 casos de uso. A continuación se muestra la captura de pantalla del componente para la instalación:
