# Añadir una columna editable a una tabla

## Acerca de esta tarea

**Se aplica a** : TBM Studio 12.0 y posteriores

Las columnas editables son columnas en blanco en las que los usuarios pueden introducir datos. Sólo pueden añadirse a tablas editables.

Para añadir una columna editable:

## Procedimiento

1. Seleccione la tabla editable en la que desea añadir la columna.
2. Abra el menú desplegable del icono **Insertar** ![](../../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/studioimages/icons/table_column_add.png) en la pestaña **Datos**.
3. Seleccione la opción **Insertar columna editable**.
4. Cuando se le solicite, introduzca un nombre para la nueva columna y haga clic en **Aceptar**.
