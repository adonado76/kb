# Añadir columnas a una tabla

**Se aplica a** : TBM Studio 12.0 y posteriores

Vea este vídeo de demostración de Apptio Education Services: [Añadir una columna de tarifas](https://community.apptio.com/videos/1893 "(se abre en una pestaña o una ventana nueva)"). O consulte [todos los vídeos de Apptio](https://community.apptio.com/docs/DOC-7714 "(se abre en una pestaña o una ventana nueva)").

Después de añadir una tabla a un informe, puede añadir columnas a la tabla. A continuación se describen los principales tipos de columnas:

- **Total** - Muestra los totales de cada fila de una tabla ([¿Cómo?](add-total-column.htm "(se abre en una pestaña o una ventana nueva)") ).
- **Fórmula** - La columna puede basarse en una fórmula personalizada, en las columnas del conjunto de datos que respaldan la tabla o en un valor modelado [(¿Cómo?)](add-formula-column-to-table.htm "(se abre en una pestaña o una ventana nueva)").
- **Calculada** - La columna puede mostrar valores para una sola columna como recuento, porcentaje y suma. También puede comparar dos columnas de la tabla o comparar valores entre proyectos ([¿Cómo?](add-calculated-column.htm "(se abre en una pestaña o una ventana nueva)") ).
- **Calendario** - Muestra los valores de los datos basados en un lapso de tiempo superior a un mes o periodo ([¿Cómo?](add-calendar-based-column.htm "(se abre en una pestaña o una ventana nueva)") ).
- **Sparkline** - Muestra una pequeña línea de tendencia ([¿Cómo?](add-sparkline-column-to-tables.htm "(se abre en una pestaña o una ventana nueva)") ).
- **Editable** - Una columna en blanco añadida a una tabla editable donde los usuarios pueden introducir datos.

La tabla siguiente muestra dónde pueden utilizarse los tipos de columnas.

| **Tipo de columna** | **Tablas basadas en objetos** | **Transformar tablas** | **Tablas editables** |
| --- | --- | --- | --- |
| Fórmula | X | X | X |
| Calculado | X | X |  |
| Calendario | X | X |  |
| Gráfico de línea | X | X |  |
| Total | X | X | X |
| Editable |  |  | X |

El ajuste de palabras no es aplicable a la columna de origen principal de una tabla editable enriquecida.
