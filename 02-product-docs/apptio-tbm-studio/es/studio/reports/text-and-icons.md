# Componentes de texto en los informes

**Se aplica a** : TBM Studio 12.0 y posteriores

Al crear un informe, puede añadir componentes que muestren texto. Estos componentes son:

|  |  |
| --- | --- |
| **Cuadro de texto HTML** | Se utiliza para añadir texto a un informe que es visible para todos los usuarios. |
| **ICR** | Permite añadir indicadores clave de rendimiento a un informe. |

## Editar las propiedades de los componentes

Existe una amplia gama de propiedades disponibles para los componentes. Las propiedades disponibles difieren según el tipo de componente. **Las propiedades** se establecen mediante la cinta de opciones y un cuadro de diálogo Propiedades.

Para editar las propiedades de un componente, muestre el cuadro de diálogo **Propiedades** realizando una de las siguientes acciones:

- En la esquina superior izquierda del gráfico o tabla, haga clic en el pequeño triángulo ![imagen](../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/studioimages/icons/arrow-down-greyicon.jpg) situado a la izquierda del nombre del componente para mostrar el menú **Acción**, como se muestra en la imagen siguiente. En el menú **Acción**, seleccione **Propiedades**.

  ![imagen](../../resources/images/studio_images/studioimages/studio_html%20comp_properties.png)
- Haga clic con el botón derecho del ratón en cualquier lugar dentro de los bordes del componente y haga clic en **Propiedades** en el menú emergente.

Los componentes del informe se describen en los siguientes temas:

- [HTML](html.html "Se aplica a: TBM Studio 12.0 y posteriores")
- [KPI: indicadores clave de rendimiento](kpi.html "Se aplica a: TBM Studio 12.0 y posteriores")
