# Introducción a Report Studio

**Se aplica a** : TBM Studio 12.0 y posteriores

Después de cargar tablas y construir modelos de asignación en TBM Studio, puede utilizar Report Studio para crear informes. Los informes pueden contener tablas y gráficos, así como componentes para filtrar los informes, introducir datos y navegar entre ellos.
