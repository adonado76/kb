# Representar un código jerárquico en perspectivas

**Se aplica a** : TBM Studio 12.0 y posteriores

Al publicar un campo en una perspectiva personalizada en la sección **Perspectivas** del **Explorador de proyectos**, una de las opciones del cuadro de diálogo **Publicar campo** es **Representa un código de jerarquía**. Si marca esta opción, cambia el comportamiento del campo cuando se utiliza con un slicer. Cuando un usuario introduce texto en el campo de búsqueda de la cortadora, la opción busca todos los valores que empiecen por el texto introducido más un carácter adicional. A continuación, crea una entrada de grupo para cada uno en la cortadora.
