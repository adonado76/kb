# TBM Studio Acerca de Report Studio

12.0 y más tarde

Después de crear tablas y modelos de asignación en TBM Studio , puede utilizar Report Studio para crear informes. Los informes pueden contener tablas y gráficos, así como componentes para filtrar informes, introducir datos y navegar entre informes.
