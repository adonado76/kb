# Acerca de Model Studio

**Se aplica a** : TBM Studio 12.0 y posteriores

Un modelo de datos es una representación lógica y gráfica del flujo de datos relevantes para el coste, el presupuesto, la utilización, los niveles de servicio, el consumo u otras métricas de sus servicios de TI. En Studio, un modelo se considera una métrica, donde una métrica es un valor. Un modelo de datos muestra cómo una amplia gama de datos financieros u operativos están relacionados con el coste y el rendimiento de los productos y servicios que TI proporciona a la empresa.
