# Añadir una tabla al modelo

**Se aplica a** : TBM Studio 12.0 y posteriores

Si desea que una tabla forme parte del modelo, añada un paso **Modelo** a la transformación de la tabla.

![](../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/model.png)

## Añadir un paso Modelo

1. Echa un vistazo a la tabla.
2. Añada un nuevo paso y seleccione Modelo como el tipo de paso que desea añadir.
3. Guarda los cambios.

Después de añadir el paso **Modelo**, la tabla se muestra en una vista de tabla única:

![](../../../../../03-media/apptio-tbm-studio/resources/images/studio_images/studioimages/cost-source-model.png)
