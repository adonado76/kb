# GetReportName función

**Se aplica a** : TBM Studio 12.0 y posteriores

Utilice esta función para obtener dinámicamente el nombre del informe que está utilizando.

## Dónde utilizarlo

Esta función puede utilizarse en:

- Texto dinámico

Se utiliza para obtener dinámicamente el nombre del informe que se está utilizando.

Por ejemplo, se puede combinar con `evalWiki()` para definir una cabecera estándar en un conjunto de datos y utilizar la función GetReportName para obtener el nombre del informe actual.

## Sintaxis

`GetReportName()`

## Argumentos

No hay argumentos.

## Tipo de retorno

Texto

## Ejemplo

`GetReportName()`
