# PeriodsInYear función

**Se aplica a** : TBM Studio y posteriores

Devuelve el número de periodos del año.

## Dónde utilizarlo

Esta función puede utilizarse en:

- Conjuntos de datos
- Columnas de fórmulas en tablas de informes
- Texto dinámico

## Sintaxis

`PeriodsInYear()`

## Argumentos

Ninguna

## Tipo de retorno

Serie

## Ejemplo

Dado un proyecto de 13 periodos, y un periodo fijado en P10 FY2013, la siguiente función devuelve el resultado que se muestra a continuación:

| Función | Volver |
| --- | --- |
| PeriodsInYear() | 13 |
