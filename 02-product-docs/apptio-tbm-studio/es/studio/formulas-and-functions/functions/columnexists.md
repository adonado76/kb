# ColumnExists función

**Se aplica a** : TBM Studio 12.0 y posteriores

Utilice esta función para determinar si una columna existe en una tabla.

## Dónde utilizarlo

Esta función puede utilizarse en:

- Tablas

## Sintaxis

`=ColumnExists(column_name)`

## Argumentos

*nombre\_columna*

El nombre de la columna que desea comprobar.

## Tipo de retorno

Verdadero o falso
