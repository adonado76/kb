# ProjectExists función

**Se aplica a** : TBM Studio 12.0 y posteriores

Si el proyecto especificado existe, la función devuelve "true" Si el nombre del proyecto no existe, la función devuelve "false"

## Dónde utilizarlo

Esta función puede utilizarse en:

- Conjuntos de datos
- Métricas calculadas e informes con columnas de métricas
- Columnas de fórmulas en tablas de informes
- Texto dinámico

## Sintaxis

`ProjectExists(domain:project)`

## Tipo de retorno

Texto
