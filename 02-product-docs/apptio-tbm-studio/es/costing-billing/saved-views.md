# Vistas guardadas

Se aplica a : 12.11.9 y posteriores para todos los productos.

Esta función le permite seleccionar un conjunto de parámetros ad hoc y guardarlos con un nombre específico, de modo que pueda reutilizar fácilmente los informes con los parámetros previamente seleccionados. La opción Vistas guardadas está activada por defecto y aparecerá en la parte derecha de la interfaz de usuario.

![img](../../../../03-media/apptio-tbm-studio/resources/images/studio_images/sv-1.jpg)

En el primer inicio de sesión, Vistas guardadas no tendrá ninguna vista seleccionada.

Guardar una vista

Abra cualquier informe y personalícelo con cualquiera de las opciones siguientes:

- Cortadora compacta
- Cortadora de filas
- Selector de columnas - Selector de valores, selector de métricas, selector de horas
- Pivote rápido
- Tabuladores
- Tablas - Barra de búsqueda, orden de las columnas

![img](../../../../03-media/apptio-tbm-studio/resources/images/studio_images/sv-main.jpg)

En la esquina superior derecha, despliegue el icono Vista y elija la opción Guardar vista. Aparecerá la siguiente ventana emergente:

![img](../../../../03-media/apptio-tbm-studio/resources/images/studio_images/sv-2.jpg)

Introduzca los valores en los campos Nombre de la vista y Descripción de la vista. Para establecerla como página de destino predeterminada, marque la opción Establecerla como vista de destino para este informe.

![img](../../../../03-media/apptio-tbm-studio/resources/images/studio_images/sv-3.jpg)

Seleccione Guardar. Aparece un mensaje de confirmación en la parte inferior de la página " *<nombre de la vista guardada> ¡creada con éxito!* ".

Expanda el icono Vista : la vista guardada aparece en la parte inferior, como se muestra:

![img](../../../../03-media/apptio-tbm-studio/resources/images/studio_images/sv-4.jpg)

El icono ![img](../../../../03-media/apptio-tbm-studio/resources/images/studio_images/homeicon.jpg) indica que la vista guardada está configurada como página de destino predeterminada. Si cierra la sesión y vuelve a acceder a la página de informes más adelante, la vista guardada aparecerá como página de destino.

El icono ![img](../../../../03-media/apptio-tbm-studio/resources/images/studio_images/current-view.jpg) indica una "vista actual".

El icono del punto azul ![img](../../../../03-media/apptio-tbm-studio/resources/images/studio_images/blue-dot-icon.jpg) indica que nos encontramos en una vista.

Nota: Sólo puede crear hasta 5 vistas por informe.

Editar una vista

En el menú Vistas, seleccione ![img](../../../../03-media/apptio-tbm-studio/resources/images/studio_images/more.jpg) para una vista guardada y, a continuación, seleccione Editar vista.

![img](../../../../03-media/apptio-tbm-studio/resources/images/studio_images/sv-edit-delete.jpg)

La ventana emergente Editar vista aparece como se muestra

![img](../../../../03-media/apptio-tbm-studio/resources/images/studio_images/sv-edit.jpg)

Realice los cambios oportunos y seleccione Guardar. Un mensaje de confirmación " *<nombre de la vista guardada> ¡actualizada correctamente!* " aparece.

Nota: Para marcar otra vista guardada como página de destino predeterminada, seleccione la casilla de verificación Establecer esta como vista de destino para este informe.

Borrar una vista

En el menú Vistas, seleccione ![img](../../../../03-media/apptio-tbm-studio/resources/images/studio_images/more.jpg) para una vista guardada y, a continuación, seleccione Eliminar vista.

Un mensaje de confirmación " *<nombre de la vista guardada> ¡borrada con éxito!* " aparece.

Guardar como

Esta opción permite crear una nueva vista guardada, a partir de las modificaciones de una ya existente.

Abre una vista guardada y personaliza algunos filtros. En el menú Vistas, seleccione la opción Guardar como.

![img](../../../../03-media/apptio-tbm-studio/resources/images/studio_images/sv-svas-2.jpg)

Introduzca un nombre único para la nueva vista y seleccione Guardar.

Nota: Si introduce un nombre existente, aparecerá un mensaje de error " *Ya existe una vista con este nombre* ".

![img](../../../../03-media/apptio-tbm-studio/resources/images/studio_images/sv-svas-3.jpg)

Un mensaje de confirmación " *<nombre de la vista guardada> ¡creada con éxito!* " aparece. La vista recién creada aparece en el menú como se muestra.

![img](../../../../03-media/apptio-tbm-studio/resources/images/studio_images/sv-svas-4.jpg)

Restablecer

Puede reiniciar utilizando cualquiera de las opciones siguientes:

- Seleccione Restablecer todas las cortadoras para borrar todas las cortadoras seleccionadas.
- Seleccione Restablecer todos los filtros globales para borrar todos los filtros globales seleccionados.
- Seleccione Restablecer informe por defecto para borrar la vista actual y sustituirla por la vista por defecto.

Nota: Los siguientes parámetros no se tienen en cuenta para la configuración de las vistas guardadas: Ordenación por filas, Gráficos, Filtros globales, Intervalo de fechas (utiliza el "Periodo de tiempo predeterminado" establecido en Configuración de tiempo), Entorno y Rama.
