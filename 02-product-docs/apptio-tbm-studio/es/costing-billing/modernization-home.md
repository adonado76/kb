# Mi página de aterrizaje

Esta función le permite seleccionar una página o informe específico para que sea su página de destino predeterminada. La opción Inicio está activada por defecto y siempre mostrará el informe Cálculo del coste del servicio.

![img](../../../../03-media/apptio-tbm-studio/resources/images/studio_images/home-1.png)

## Añadir la página de destino

Para configurar su página de destino preferida, vaya a Configuración y, a continuación, seleccione la opción Configurar página de destino/Favoritas.

![img](../../../../03-media/apptio-tbm-studio/resources/images/studio_images/home-2.png)

Seleccione los valores apropiados en la colección Informe y en el desplegable Informe.

Nota: Los valores del panel Configurar página de destino/Favoritos se ordenarán alfabéticamente. Puede restablecer la página de aterrizaje seleccionando el valor "ninguno seleccionado". Si la opción Configurar favoritos no está seleccionada en las características del proyecto, la pestaña Favoritos estará desactivada en la ventana emergente Configurar página de destino/Favoritos.

Los valores del panel Configurar página de destino/Favoritos se ordenarán alfabéticamente. Puede restablecer la página de aterrizaje seleccionando el valor "ninguno seleccionado". Si la opción Configurar favoritos no está seleccionada en las características del proyecto, la pestaña Favoritos estará desactivada en la ventana emergente Configurar página de destino/Favoritos.

![img](../../../../03-media/apptio-tbm-studio/resources/images/studio_images/home-4.png)

Seleccione Guardar. Un mensaje de confirmación "*¡Página de inicio configurada correctamente!* " aparece. Si actualiza Mi página de destino o abre la aplicación la próxima vez, se mostrará la página seleccionada como página de destino predeterminada.

![img](../../../../03-media/apptio-tbm-studio/resources/images/studio_images/home-6.png)

- Cualquier informe OOTB o personalizado de nivel superior puede seleccionarse como página de destino.
- Si un administrador elimina o revoca un informe para un usuario, y si estaba almacenado en los datos del servicio de preferencias, se eliminará automáticamente de Mi página de destino después de que se cargue por primera vez en la vista CT.
- Si un administrador suplanta a un usuario, el usuario puede ver los cambios realizados en la página de destino por el administrador.
- No se pueden seleccionar como página de destino predeterminada los informes de perforación u otros informes ocultos (a través de la recopilación de informes).

Nota: No puede crear ninguna página de destino personalizada. La página de destino seleccionada se guarda en todas las sesiones, navegadores y dispositivos, pero no en la caché del navegador.
